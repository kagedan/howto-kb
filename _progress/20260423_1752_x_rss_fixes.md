## 補足情報

### SocialData API メモ
- Base URL: `https://api.socialdata.tools`
- Article 取得: `/twitter/article/{tweet_id}` — tweet_id を渡す（URL中の article_id ではない）
- ツイート取得: `/twitter/statuses/show?id={tweet_id}`
- スレッド取得: `/twitter/thread/{conversation_id}`
- レスポンスが `article` を持つのは Xの長文記事投稿のみ。通常ツイートに叩くと HTTP 500

### Draft.js content_state の構造（SocialData返却）
- `blocks[]`: type は `unstyled / header-two / unordered-list-item / ordered-list-item / blockquote / atomic / code-block`
- `inlineStyleRanges[]`: style は主に `Bold`
- `entityMap[]`: `{"key":"<n>", "value":{"type":"DIVIDER|TWEMOJI|MEDIA|LINK|MARKDOWN", "data":...}}`
  - MEDIA: `data.mediaItems[].mediaId` を記事トップの `media_entities[]` から引く
  - LINK: `data.url`
  - MARKDOWN: `data.markdown`（既にMD文字列）
  - TWEMOJI: 元テキストの絵文字をそのまま残す
  - DIVIDER: `---`
