Here’s how to install the Pika Skills Plugin:

1. Make sure you’ve connected Claude to the Pika MCP
2. Open Cowork or Code → Customize
3. Click the + next to Personal plugins → Create plugin → Add marketplace
4. Paste Pika-Labs/Pika-Plugins and click Sync
5. Open Pika → Connectors and click Connect
6. Authorize Claude in the browser OAuth screen
7. Done — the plugin shows version, author, and skills, with the toggle ON
