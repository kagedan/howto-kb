## はじめに

Claude Coworkは、Anthropicが提供するデスクトップ向けエージェント機能です。Claude Codeがエンジニア向けに「ターミナル上で自律的に動くエージェント」であるのに対し、Coworkは**非エンジニアを含む知識労働者向け**に、ローカルファイルや日常アプリを横断して作業を代行することを目的としています。

2026年2月にリサーチプレビュー、4月に有料プラン（Pro / Max / Team / Enterprise）で一般提供開始となりました。

## 一言で言うと

> 「ゴールを伝えて離席 → 完成物を受け取る」 を、デスクトップ上のファイル・アプリに対して実現するエージェント。

Claudeが作業計画を提示し、ユーザーの承認を得てから実行します。人間の判断が必要な箇所は必ずユーザーに残す設計です。

## Coworkでできること（代表例）

### 1. ローカルファイルの読み書き・整理

### 2. ドキュメント・資料の生成

### 3. リサーチ・情報統合

* Web + ローカル資料を横断した調査研究
* 複数の情報源を参照した比較分析
* 調査結果のドキュメント化まで一気通貫

### 4. 複数タスクの並列処理

### 5. 外部サービス連携（エンタープライズ機能含む）

### 6. スケジュール実行・プロジェクト管理

* 定期実行のスケジュール設定
* 関連タスクをプロジェクト単位でまとめて管理

## 対象ユーザー

「時間はかかるが技術的に複雑でない業務」を扱う知識労働者が想定されています。

* リサーチャー / アナリスト
* 法務・財務チーム
* 営業・人事などバックオフィス担当
* エンジニア以外でAIエージェントの力を使いたい人

## 使い方（セットアップ〜実行）

1. Claude Desktopを最新版にアップデート
2. アプリを開き、上部の「Chat / Cowork」モード切替で **Cowork** を選択
3. やりたいタスクを自然言語で記述（例: 「`デスクトップに散らばったスクリーンショットをフォルダにまとめて」）
4. Claudeが **計画** を提示
5. 計画を確認し、問題なければ実行承認
6. 実行中はアプリを開いたまま待機（閉じるとセッション終了）
7. 完成物を受け取る

## エンタープライズ向け機能（2026年4月追加）

大組織での運用を想定した管理機能が追加されました。

* ロールベースのアクセス制御
* グループ単位の利用量上限
* 使用状況アナリティクス
* OpenTelemetryサポート拡張
* Zoom MCPコネクタ
* ツール/コネクタごとの権限設定

## 注意点

* **セッション中はアプリを閉じない**: 閉じると実行が中断する
* **消費量が多い**: 通常チャットより利用枠を消費するため、Proプランでは長時間タスクで頭打ちになる可能性あり
* **まだリサーチプレビューの色合いが残る**: 破壊的変更や予期しない挙動を許容できる業務から試すのが無難
* **権限設計はユーザー責任**: ローカルファイルに直接アクセスするため、読み書き対象を絞る運用が重要

## Claude Codeとの使い分け

| 観点 | Claude Code | Claude Cowork |  
| 対象 | エンジニア | 非エンジニア含む知識労働者 |  
| UI | ターミナル / IDE | デスクトップアプリ |  
| 主な作業 | コード編集・実行 | ファイル整理・文書生成・リサーチ  
| 強み | コマンド実行・Git操作 | Office系ファイル・業務アプリ連携

同じエージェント基盤を共有しつつ、**「コードを書く人向け」と「ドキュメントを扱う人向け」** で分かれているイメージです。

## 所感

* 「AIエージェントをコーディング以外に使う」領域で、最も具体的に製品化されたものの一つ
* ローカルファイルを直接触れる点は、ChatGPT Agentモードとの大きな差別化ポイント
* 非エンジニアに「AIに仕事を任せる」体験を提供する意味で、社内展開のインパクトが大きい
* 処理に時間はかかる印象で、時間はかかるが技術的に複雑でない業務を任せるのは相性が良さそう
* Microsoftとの連携はまだそこまで強くなさそうで、そちらはCodexの方が相性が良いかもしれない(試したい)
