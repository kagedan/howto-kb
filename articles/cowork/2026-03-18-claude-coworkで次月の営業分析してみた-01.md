## Excelに営業データはあるのに、活かせていなかった

今回使ったのは、2ヶ月分の営業データです。

10人分の  
・商談数  
・成約数  
・成約率  
・総売上

いわゆる基本的なKPIは、一通りExcelにまとまっていました。

数字自体はちゃんと見ているし、  
「誰が調子いいか」くらいはなんとなく分かる。

でも、本当にやりたかったのはそこじゃありませんでした。

知りたかったのは、  
「じゃあ次月はどうなるのか？」という予測です。

営業データとプロンプトはこんな感じ・・・

![](https://assets.st-note.com/img/1773810240-ya0g7Gi2fwBYL41NqZT9pxOV.png?width=1200)

営業データ(ダミー)

```
# 🎯ミッション
あなたは一流のデータアナリストです。
与えられた営業データをもとに、翌月（2025年4月）の売上を最大化する戦略を設計してください。

# 🧠思考ステップ（必ず順番に実行）
STEP1：データ理解
・各営業マンの「商談数・成約数・売上」を整理
・成約率（成約数 ÷ 商談数）を算出

STEP2：パフォーマンス分析
・営業マンごとの強み・弱みを特定
・成約率と売上の相関を分析
・安定型／爆発型などのタイプ分類を行う

STEP3：予測
・2025年4月の各営業マンの成約率を予測
・予測根拠（トレンド、直近成績など）を簡潔に説明

STEP4：最適化設計
・売上最大化を目的とした商談数の最適配分を決定
・高効率営業マンへ重点配分するロジックを明確にする

STEP5：配分順序設計
・商談を割り当てる順序を設計
・「誰に→何件→次に誰」という具体的な順序を提示

# 📊入力データ
・期間：2025年2月〜3月
・対象：営業マン10名
・項目：商談数、成約数、売上高（その他必要データ含む）

# 📈出力要件（必須）
以下を必ず数値付きで出力する：

①営業マン別 成約率予測ランキング
・順位
・営業マン名
・予測成約率（%）

②営業マン別 商談数最適配分
・営業マン名
・割り当て商談数
・期待売上

③商談割り当て順序（実行手順）
・ステップ形式で具体的に提示

④分析ロジック（簡潔）
・なぜその配分になるのかを説明

# ⚠️制約条件
・分析は2月〜3月データのみ使用
・推測は論理的根拠に基づくこと
・数値は必ず具体的に提示
・売上最大化を最優先とする

# 📤Excel出力指示（最重要）
最終結果はExcel形式で出力すること。
以下の3シート構成で作成：

【Sheet1：成約率ランキング】
・順位 / 営業マン名 / 予測成約率

【Sheet2：商談数配分】
・営業マン名 / 割当商談数 / 期待売上

【Sheet3：配分順序】
・順番 / 営業マン名 / 割当数 / 理由

【Sheet4：説明】
・4月売上最大化 戦略方針を表にして箇条書きで書く。
・ 期待売上合計（4月）を表にして箇条書きで書く。
・Sheet1から3がの理由をシート毎に箇条書きで細かく全て説明する。

※表形式（CSVまたはExcel貼付可能形式）で出力すること
※列名を必ず含めること

# 🧭最終ゴール
「誰にどれだけ商談を振れば最も売上が伸びるか」が一目で分かる状態にする
```

## Coworkに投げたら、想像以上の分析が返ってきた

やったことはシンプルでした。

2ヶ月分の営業データを、そのままClaude Coworkに渡して、  
「このデータをもとに分析と次月の予測をしてほしい」とプロンプト投げただけです。

![](https://assets.st-note.com/img/1773816736-Y1WSOrfdIBHDKlta29jQibnT.png?width=1200)

営業データのファイルを選択して、上記のスクリプトを入力します。

正直、最初はそこまで期待していませんでした。

まず驚いたのが、  
**各メンバーごとの特徴をちゃんと捉えていたこと**です。

商談数は多いけど成約率が低い人。  
逆に、件数は少ないけど確度が高い人。

そういった違いをしっかり分けた上で、  
「なぜそうなっているのか」まで踏み込んで説明してくる。

その瞬間、思わずこう思いました。

「これ、自分でやるより精度高くないか？」

今までなんとなく見ていた数字が、

## Claude Coworkは便利

ここまでやってみて、率直に思ったことがあります。

Claude Coworkは、普通に便利です。

特別な分析スキルがあるわけでもなく、  
ただデータを持っているだけの状態でも、

ここまで整理されて、意味が見えて、  
次に何を考えるべきかのヒントが得られる。

正直、「もっと早く使えばよかった」と思いました。

特に、営業のように数字が日々積み上がっていく仕事では、  
データをどう扱うかで結果が大きく変わるはずです。

それなのに、これまではどこかで「難しそう」と思って、  
ちゃんと向き合えていなかった。

でも、今回試してみて分かりました。

完璧な分析じゃなくてもいい。  
まずは、持っているデータを“投げてみる”だけでいい。

そこから見えてくるものが、確実にある。

もしあなたの手元にも、  
Excelに眠っている営業データがあるなら、

一度、Claude Coworkに投げてみてください。

きっと、今までとは違う見え方がするはずです。
