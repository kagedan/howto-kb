AIでスライドを生成する方法はいろいろありますが、たいてい有料だったり無料枠をすぐ使い果たしたりします。

今回は、無料枠が大きい `Gemini API` と、 `Marp` という仕組みを使ってスライドを作成する方法を解説します。

---

## 💡 前提条件：AIエージェント環境の準備

本記事で紹介するスライド生成には、前回の記事で紹介した **VSCode + Gemini API + Cline** の開発環境を使用します。

まだAIエージェント環境を作成していない方は、まず以下の記事を参考にセットアップを完了させてください。（Googleアカウントがあれば数分で完全無料で構築できます）

👉 [【初学者向け】VSCode + Gemini API + Clineで無料のAIエージェント開発環境を作る](https://zenn.dev/kanaru/articles/d610b07cb10c0e)

別のAIエージェントをすでに使っている場合はそちらを使ってください。

## 【ステップ1】Marp拡張機能をインストールしよう

まずは、Markdownで書いたスライドをVSCode上でリアルタイムにプレビュー・出力するために、VSCode拡張機能 **「Marp for VS Code」** をインストールします。

1. VSCodeの左端メニューから **「拡張機能」アイコン** をクリックします。
2. 検索窓に **「Marp for VS Code」** と入力します。
3. 表示された拡張機能の **インストール** ボタンをクリックします。

![Marp for VS Codeの検索とインストール画面](https://static.zenn.studio/user-upload/deployed-images/b15cba2801fff65a6bd398fe.png?sha=59ffba826e4a09e7ec1703d23a26fc954f4d62c9)

これでスライド表示・書き出しの準備は完了です！

---

## 【ステップ2】Marpの基本構造を理解しよう

AIエージェントに指示を出す前に、Marpがどのような構造でスライドを作るのか、基本を抑えておきましょう。

まずは、VSCodeでテスト用の新規ファイルを作成して、実際にスライドを表示してみます。

### 空のフォルダを用意してVSCodeで開く

ファイルを保存して作業を進めるために、まずは作業用の空のフォルダを作成してVSCodeで開きます。

1. デスクトップや書類フォルダなど、パソコンの分かりやすい場所に新しく空のフォルダ（例：`marp`）を作成します。
2. 作成したフォルダを、VSCodeのウィンドウへ直接ドラッグ＆ドロップして開きます（または、VSCodeのメニュー「ファイル」＞「フォルダーを開く...」からフォルダを選択して開きます）。

### 新規ファイルの作成方法

1. VSCodeの上部メニューから **Explore** (下のキャプチャで⭕️で囲った場所)をクリック
2. **New File** (下のキャプチャで⭕️で囲った場所)をクリック
3. ファイル名を **`test.md`** と入力し、Enterを押すと `test.me` ファイルが作成されます。

![VS Codeで新規ファイル作成方法](https://static.zenn.studio/user-upload/deployed-images/3915068f368adad60e1dd221.png?sha=f372ec0242dbc2b3de966d95dd2d69fd2b4c9b63)

### コードをコピペする

1. 作成した `test.md` に、以下のコードをコピー＆ペーストします。
2. キーボードの **`Ctrl + S`**（Macは **`Cmd + S`**）を押してファイルを保存します。

```
---
marp: true
theme: default
size: 16:9
paginate: true
header: "Marpの基本"
footer: "サンプルスライド"
---

# 1枚目のスライド 🚀

ここにスライドのメインテキストを書きます。
- 箇条書きその1
- 箇条書きその2

---

# 2枚目のスライド 🎨

3本のハイフン `---` で区切ることで、新しいスライドになります。
- テーマを変えたいときは `theme: gaia` などに書き換えます。
- ページ番号は `paginate: true` で自動表示されます。
- アスペクト比は `size: 16:9` でワイド画面に指定しています。
- `header` や `footer` を指定すると、すべてのスライドに共通の文字が表示されます。
```

### プレビューの開き方

コードを貼り付けたら、スライドがどのように表示されるかプレビュー画面を開いてみましょう。

* VSCodeのエディタ上の方にある **「プレビューボタン」**（下のキャプチャで⭕️で囲った場所）をクリックします。
* または、ショートカットキー **`Ctrl + Shift + V`**（Macは **`Cmd + Shift + V`**）を押します。

デザインが適用されたスライドプレビューが表示されれば成功です！

![VS CodeのMarpプレビュー画面](https://static.zenn.studio/user-upload/deployed-images/3d7c17623133ede68b6d233d.png?sha=4558aa853c979abc171fe3e3b26e98de3363810a)

### Marpの構造の3大ポイント

1. **Markdownのルール**:  
   `Marp` は `Markdown` という記法を拡張したものです。  
   スライド内の見出し（タイトル）は `#` や `##`、本文は箇条書き（`-`）など、通常のMarkdown記法でそのまま記述できます。
2. **設定（フロントマター）**:  
   ファイルの先頭にある `---` と `---` で囲まれた部分は「フロントマター」と呼ばれ、Marpの設定を記述します。

   * `marp: true`: このファイルをMarpスライドとして認識させます（必須）。
   * `theme: default`: スライドのデザインテーマを設定します。
   * `size: 16:9`: スライドの画面比率（アスペクト比）を設定します（16:9のほか、4:3も指定可能）。
   * `paginate: true`: スライドの右下にページ番号を表示します。
   * `header: "..."`: スライド上部に共通のヘッダーを表示します。
   * `footer: "..."`: スライド下部に共通のフッターを表示します。
3. **スライドの区切り (`---`)**:  
   半角ハイフン3つ `---` だけの行を書くと、そこがスライドの区切り（改ページ）になります。

## 【ステップ3】AIエージェントにスライド作成を指示しよう

それでは、さっそくAIエージェント（Cline）にスライドを作ってもらいましょう。

以下のようなプロンプトをClineのチャットに入力して送信してみましょう。

```
@/test.md
以下のテーマで、Marp形式のスライドを作成してください。
「AIエージェントがもたらす業務効率化の未来」について、大学1年生向けに分かりやすく解説するスライド（5〜6ページ構成）
```

指示を送信すると、Clineが自動的にスライドのMarkdownコードを書き込んでくれます。

![AIが生成したMarpのプレビュー画面](https://static.zenn.studio/user-upload/deployed-images/200c5bf29380d2f70875649c.png?sha=ac0622715b6d25c58313364e5bb7bf46b81bc41e)

!

**💡 Tips：`@/ファイル名` の記述について**  
Clineでは、プロンプトの先頭などに `@/ファイル名`（例：`@/test.md`）と入力すると、AIエージェントに**そのファイルを明示的に読み込ませて参照させる**ことができます。

これにより、先ほどステップ2で作成した基本スライドの内容や設定をAIにインプットした状態で、「このスライドの内容をベースにして、今回のテーマのスライドを新しく作って」といった指示がスムーズに行えるようになります。

## 【ステップ4】スライドをPDFやPowerPoint形式に書き出そう

スライドの作成が完了したら、発表や共有のためにPDFやPowerPoint（PPTX）形式のファイルとして保存（書き出し）しましょう。

1. 書き出したいスライドのMarkdownファイル（例：`test.md` や `slides.md`）をエディタで開きます。
2. 三角形のMarpのメニューボタン(下のキャプチャで⭕️で囲った場所)をクリックして、 **`Export Slide Deck...`** をクリックします。

![Marpの書き出しコマンドを選択する画面](https://static.zenn.studio/user-upload/deployed-images/81a0f7668cd4a2fc4638b915.png?sha=07f02f539087d35ffffa22f5510269b939e5e98a)

4. 保存ダイアログが表示されるので、画面下の **「ファイルの種類」** から保存したい形式を選択します。
   * **`pdf`**: 印刷や配布に最適なPDF形式
   * **`pptx`**: PowerPointでさらに編集可能なスライド形式
   * **`html`**: ブラウザでそのままプレゼンテーションができるHTML形式

![保存ダイアログでファイル形式を選択する様子](https://static.zenn.studio/user-upload/deployed-images/bac4691988014b042a4542c8.png?sha=7d06b298c992a1d84c0744e1dd5e0a29c64d8683)

5. **「Export」** をクリックして保存します。

これで、AIエージェントとMarpを使って作ったスライドが、実際のプレゼンテーションファイルとして出力されます！
