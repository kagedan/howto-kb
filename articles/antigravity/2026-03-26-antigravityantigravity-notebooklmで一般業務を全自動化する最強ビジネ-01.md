## 0. 事前準備：AntigravityとNotebookLMを連携しよう

本編に入る前に、まずはAIに「あなたの会社の知識」を教えるための連携設定（MCP設定）を済ませておきましょう。一度設定すれば、あとはずっと使えます。

### 準備する手順

1. **NotebookLMにログイン** ブラウザでGoogleの [NotebookLM](https://notebooklm.google.com/) にアクセスし、Googleアカウントでログインしておきます。
2. **連携ツールのインストール** PCのターミナル（Windowsなら「PowerShell」、Macなら「ターミナル」アプリ）を開き、以下のコマンドをコピー＆ペーストしてEnterを押します。 uv tool install notebooklm-mcp
3. **認証の実行** 続けて、ターミナルで nlm login と入力してEnter。ブラウザが開くので、アクセスを許可します。
4. **Antigravityへ接続** 最後にターミナルで nlm setup add antigravity と入力してEnter。 これで、AntigravityというAIエディタの裏側に、あなたのNotebookLMが直結されました！

準備はこれだけです。それでは、最強のビジネスハック10選を見ていきましょう！

## アイデア1：【営業向け】自社商材と顧客情報を掛け合わせた「神・提案書」の自動生成

過去の優秀な提案書や商品カタログをAIに覚えさせ、ターゲット企業に合わせた完璧な提案書（PowerPointやPDFの元となる構成と文章）を一瞬で出力させます。

### 📝 実践手順

1. **NotebookLMで「営業提案ナレッジ」ノートブックを作成** NotebookLMの画面で「新しいノートブック」を作り、自社の製品カタログ、料金表、過去にコンペで勝った優秀な提案書（PDFやWord）をアップロードします。
2. **Antigravityで作業フォルダを開く** PCのデスクトップなどに「提案書作成」という空のフォルダを作ります。Antigravityを起動し、メニューの「File」＞「Open Folder」から、いま作ったフォルダを開きます。
3. **AIエージェントに指示を出す** Antigravityの右側にあるAIチャット欄（Agentパネル）を開き、以下のプロンプトを送信します。

   1. 「MCP経由でNotebookLMの『営業提案ナレッジ』を読み込んでください。〇〇株式会社（ITベンチャー、従業員50名、コスト削減が課題）に向けて、当社の『クラウド管理ツール』を提案するための提案書を作成します。ノートブック内の過去の成功事例を参考に、表紙からクロージングまでのストーリー構成を作り、美しいMarkdown形式のテキストファイル（proposal.md）として保存してください。」
4. **結果を確認する** 数秒後、フォルダの中に proposal.md というファイルが自動で作成されます。クリックして開くと、自社の強みと価格体系を正確に反映した提案書の下書きが完成しています。これをWordやパワポにコピペして整えれば完了です！

4. **結果を確認する** 作成された press\_release.txt を開くと、ガイドラインを遵守し、NGワードを一切含まない、そのまま配信できるレベルのプレスリリース文案が生成されています。

4. **結果を確認する** AIが勝手にプログラムを書き、実行までしてくれます。フォルダ内に新しくできた checked\_expenses.csv をExcelで開くと、ルール違反の行にだけエラー内容が記載されています。

## アイデア9：【すべての事務職へ】マニュアルから作る「古い社内システムの自動入力ツール」

「Aの画面を見て、Bの古いシステムに手入力でコピペする」…そんな苦痛な単純作業を消し去るChrome拡張機能を作ります。

### 📝 実践手順

1. **NotebookLMで「社内システム入力マニュアル」ノートブックを作成** 「この受注番号をコピーして、別タブの検索窓に貼り付け、登録ボタンを押す」といった画面操作の手順書をテキストで入れます。
2. **Antigravityで作業フォルダを開く** PC内に「コピペ自動化ツール」という空のフォルダを作り、Antigravityで開きます。
3. **AIエージェントに指示を出す** AIチャット欄に以下のように指示します。

   1. 「NotebookLMの『社内システム入力マニュアル』に書かれた手作業を自動化したいです。ブラウザ（Chrome）でこの作業を自動でやってくれる『Chrome拡張機能』のプログラムコード一式（manifest.jsonやJavaScriptファイル）をこのフォルダ内に作成してください。」
4. **結果を確認する（Chromeへの導入）** ファイル群が生成されたら、Google Chromeを開き、URLバーに chrome://extensions/ と入力して拡張機能の管理画面を開きます。 右上の「デベロッパーモード」をONにし、「パッケージ化されていない拡張機能を読み込む」ボタンをクリックして、先ほどの「コピペ自動化ツール」フォルダを選択します。これだけで、あなた専用の自動化ツールがChromeにインストールされ、手作業とおさらばできます！
