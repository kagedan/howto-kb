## アイデア出し

ハルシネーション率が極端に高い、古いモデルを使ってください。とりわけGemini 2.5は良いモデルです。そのモデルに対して、「〇〇を解決してくれるアプリ名を20個挙げて」のように聞いてみると、うち半数程度は存在しない嘘のアプリです。嘘のアプリの詳細について特徴や欠点などを詳しく聞いていると、稀に結構良いアプリを語ってくれる場合があるのでそれを実際に作るようにすれば良いです。

もっとターゲットなどを絞るなら、ある分野の大手アプリ上位5つに対してユーザーがなぜ大手を使うのかという理由の言語化と、ユーザーが感じている不満や欠点を列挙させるのが良いです。このタスクの場合はGrokあたりが良いです。Claude SonnetやDeepSeekでも良いですが、AIを使わずに自分で調査するのが案外一番時短です。

とりわけ最新のAIはユーザーに対して危害を加えないことが徹底されているため、あまり良いアイデアは出しづらいです。より古いモデルを使用するのがおすすめですが、そもそもアイデア出しをAIに任せていると脳みその創造力を養う筋肉が衰えてそのうちAI弱者に転落する危険性があるのであまり推奨はしません。自分で考えてください。

また、無作為に単語を列挙してくれるwebサイトがあるのでそこを眺めるのも良いと思います。あとはXでキレ散らかしてるユーザーの観察だったり、AppStoreやGooglePlayのレビュー欄だったり。

## 仕様書

仕様書をAIに作らせるのは非推奨です。UIも同様。  
ここは自分で作らないと大量にAI Slop Appを量産している業者に埋もれます

具体的な方法ですが、以下について考えてください。

* 何を解決するのか？
* ユーザーにどのような体験を提供するのか？
* どの機能を盛り込み、どの機能を盛り込まないか？　その意図は？
* 誰にこのアプリを使わせたいか？　ターゲットではなく1〜3人

そのうえで、収益化したいならこれらについても。

* 自分がユーザーだったとして、何円だったら買う？
* サブスクを採用するかどうか？
* アマプラの 600円/月 を前提に、金額設定は妥当か？

特にアマプラの 600円/月 について考えてみるのはかなり良いと思います。よく 500円/月 や 300円/月 のつまらないミニアプリなんかを見かけますが、とても払いにくいです。一方でアマプラは 5900/年 なので、年1900円や買い切り1500円とかであれば気軽に買えますね。Prodraftsなどの、実際にユーザーが多い有料製品なんかを参考にしてみてください。

仕様書のフォーマットなどは一切気にしないでください。  
とにかく思いついたことなどを全て書き散らして下さい。読みづらくて結構。その内容を最新のシゴデキAIに読み込ませて、内容を整理してもらって下さい。批判的なレビューとかをもらいつつ、読みやすい良い感じにまとめてもらったものをもとにして、次に進みます。

## 画面設計 / UI

まず画面数と画面遷移、各画面に含める機能を列挙します。iPadか紙が良いです。手書きして下さい。  
画面は仕様書に書き散らした機能を分配する先だと考えて下さい。使用頻度が高い機能は遷移数を最小限に抑えたりしましょう。

それができたら、各画面を順番に設計します。私の場合、PowerPointで試作します。  
なんなら手書きでも良いです。俗に言うワイヤーフレームというものを作って下さい。画面の見た目や、ボタンや要素の配置を考えて、位置と大きさだけ決めて下さい。

そしたらAntigravity 2.0を開いて、Gemini 3.5 Flashに画像を渡してアプリを作ってもらって下さい。画面遷移と最小限のロジックだけ実装させて、細かいロジックの修正とかは放置してください。  
そこから色々文句を言って、Geminiに修正してもらう、というループを繰り返しまくってください。とにかく納得できるまで、細部まで何度も繰り返して下さい。

目安として、私の場合はここに3日かけます。

## 実装

Antigravityに作ってもらったアプリケーションをもとに、Opus 4.8かGPT 5.5に仕様書の修正をしてもらって下さい。デザインとUIをSSOTとして、仕様書の方の修正を行う感じです。

そうしたらあとは簡単で、仕様書とデザイン/UIを、全部<アプリ名>/spec/というディレクトリにぶち込んでください。

お好きなAI Agent （おすすめはCursor / Kilo / Codex）を<アプリ名>/ディレクトリで開いて、お好きな方法で実装をしてもらって下さい。ここはいろんな人がnoteやZennやTwitterで言いまくっているので、そっちに任せます。

## おすすめのAI

### デザイン

下記8種が全部独特の個性を持ってるので、実際に全部使ってみて良さげに思ったものを使ってみるのが良いと思う。安さから行くならOpus抜きの6種でも良いかも。

* **Gemini 3.5 Flash**  
  再現して欲しいアニメーションやUIを動画で渡せるのが良い
* **Gemini 3 Pro** （3.1 Proじゃないよ）  
  同上。3.1 Proよりもハルシネーションが高いので、その分個性的
* **Kimi K2.6**  
  一発で80点を作ってくれる
* **Composer 2.5**  
  爆速なOpus 4.5
* **Qwen 3.7 Max**  
  Qwenにしか作れない、独特なセンスがある
* **Qwen 3.6 Plus**  
  動画渡せるのが強い。センスは相変わらず独特
* **Claude Opus 4.7** （Claude Design）  
  値段以外文句なし
* **Claude Opus 4.5**　（API）  
  APIでしか使えないけど、個人的満足度No.1
