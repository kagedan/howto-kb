## はじめに

2026年6月、Anthropic（Claude Fable 5）、OpenAI（GPT-5.6）、Google（Gemini Omni / 3.5）が相次いで「自律エージェント」機能を実装しました。とくにGoogleは Gemini API 上で **Managed Agents**（隔離されたLinuxサンドボックスで動く、状態を保持する自律エージェント）をパブリックプレビュー公開しています。

本記事では、「エージェント」を自前で組むときの最小構成を、Pythonの擬似コードで整理します。各社SDKはバージョンで差異があるため、ここでは**プロバイダ非依存の設計パターン**として読んでください。

## エージェントの最小ループ

自律エージェントの本質は「観測→思考→行動→観測…」のループです。チャットとの違いは、**ツール実行を人間を介さず回す**点にあります。

```
from dataclasses import dataclass, field
from typing import Callable

@dataclass
class Tool:
    name: str
        description: str
            run: Callable[[dict], str]

            class MiniAgent:
                def __init__(self, llm, tools: list[Tool], max_steps: int = 8):
                        self.llm = llm
                                self.tools = {t.name: t for t in tools}
                                        self.max_steps = max_steps

                                            def run(self, task: str) -> str:
                                                    history = [{"role": "user", "content": task}]
                                                            for _ in range(self.max_steps):
                                                                        decision = self.llm.decide(history, list(self.tools.values()))
                                                                                    if decision["type"] == "final":
                                                                                                    return decision["content"]
                                                                                                                tool = self.tools[decision["tool"]]
                                                                                                                            observation = tool.run(decision["args"])  # 人間の承認なしで自走
                                                                                                                                        history.append({"role": "tool", "name": tool.name,
                                                                                                                                                                    "content": observation})
                                                                                                                                                                            return "max_steps に到達しました"
                                                                                                                                                                            ```
                                                                                                                                                                            
                                                                                                                                                                            ## ツール定義の例
                                                                                                                                                                            
                                                                                                                                                                            エージェントに渡すツールは「名前・説明・実行関数」の3点セットで十分です。説明文がそのままLLMへのプロンプトになるため、**簡潔かつ曖昧さのない記述**が品質を左右します。
                                                                                                                                                                            
                                                                                                                                                                            ```python
                                                                                                                                                                            def web_search(args: dict) -> str:
                                                                                                                                                                                q = args.get("query", "")
                                                                                                                                                                                    return f"[検索結果のダミー] query={q}"
                                                                                                                                                                                    
                                                                                                                                                                                    tools = [
                                                                                                                                                                                        Tool("web_search", "Webを検索する。args: {query: str}", web_search),
                                                                                                                                                                                        ]
                                                                                                                                                                                        ```
                                                                                                                                                                                        
                                                                                                                                                                                        ## 安全に自走させるための3原則
                                                                                                                                                                                        
                                                                                                                                                                                        Claude Fable 5 が公開3日で輸出規制を受けた事実が示すように、エージェントの「自律性」は強力ゆえにリスクも伴います。個人開発でも以下は最低限守りましょう。
                                                                                                                                                                                        
                                                                                                                                                                                        1. **max_steps と timeout を必ず設定する**（暴走・コスト爆発の防止）
                                                                                                                                                                                        2. **書き込み系ツール（送信・課金・削除）は承認ゲートを挟む**
                                                                                                                                                                                        3. **サンドボックス内で実行する**（GeminiのManaged Agentsが隔離Linux環境を採用しているのと同じ思想）
                                                                                                                                                                                        
                                                                                                                                                                                        ```python
                                                                                                                                                                                        WRITE_TOOLS = {"send_email", "post_tweet", "delete_file"}
                                                                                                                                                                                        
                                                                                                                                                                                        def guarded_run(tool, args, approve):
                                                                                                                                                                                            if tool.name in WRITE_TOOLS and not approve(tool.name, args):
                                                                                                                                                                                                    return "ユーザー承認が得られなかったため中止"
                                                                                                                                                                                                        return tool.run(args)
                                                                                                                                                                                                        ```
