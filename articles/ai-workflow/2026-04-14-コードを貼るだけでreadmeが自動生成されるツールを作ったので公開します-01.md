## はじめに

みなさん、ドキュメント書いてますか？

正直に言います。**私はドキュメントを書くのが嫌いです。**

コードは書ける。でもREADMEを書こうとすると手が止まる。API仕様書？後でいいや。関数の説明？コード読めばわかるでしょ。

そんな「ドキュメント書かない病」を根本から解決するツールを作りました。

その名も **DocForge**。

---

## DocForgeとは

コードを貼り付けるだけで、以下を自動生成するWebサービスです。

* 📄 **README**（プロジェクト概要・インストール方法・使い方）
* 🔍 **関数仕様書**（引数・戻り値・使用例）
* 📚 **API仕様書**（エンドポイント一覧・リクエスト/レスポンス）

日本語対応・ブラウザだけで動きます。インストール不要。

👉 **<https://docforge-300l.onrender.com>**

---

## デモ

試しにこのコードを貼り付けてみます。

```
class UserService {
  constructor(db) { this.db = db; }

  async findById(id) {
    return this.db.query('SELECT * FROM users WHERE id = ?', [id]);
  }

  async create(email, password) {
    const hashed = await bcrypt.hash(password, 10);
    return this.db.query(
      'INSERT INTO users (email, password) VALUES (?, ?)',
      [email, hashed]
    );
  }

  async delete(id) {
    return this.db.query('DELETE FROM users WHERE id = ?', [id]);
  }
}
```

生成ボタンを押すと…

---

**生成されたREADME（抜粋）:**

> ### UserService
>
> UserServiceは、データベースを通じたユーザー管理機能を提供するサービスクラスです。ユーザーの検索・作成・削除といった基本的なCRUD操作をカプセル化し、パスワードのセキュアなハッシュ化にも対応しています。
>
> #### 主な機能
>
> * ユーザーID検索
> * 新規ユーザー作成（パスワードハッシュ化済み）
> * ユーザー削除

**約10秒でここまで生成されます。** 日本語で、コンテキストを理解した上で。

## 技術構成

Claude APIを使っています。

```
コード入力（ブラウザ）
    ↓
Node.js + Hono（バックエンド）
    ↓
Claude API（claude-sonnet-4-6）
    ↓
ドキュメント生成・表示
```

プロンプトを工夫することで、単純な要約ではなく「使い方がわかるドキュメント」を生成するようにしています。
