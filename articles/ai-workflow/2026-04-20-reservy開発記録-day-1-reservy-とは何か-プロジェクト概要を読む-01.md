## 学んだこと

### 1. 【概要】

Reservyは、ネイルサロンを中心とした、小規模サロン向けの予約管理SaaS。  
サロンの顧客はLINEや公開ページから予約し、店舗は管理画面で予約・顧客・分析などを一元管理することができる。

### 2. 【主な機能】

#### 予約管理

| 機能 | 説明 |
| --- | --- |
| 公開予約 | LIFF や公開ページからの予約作成 |
| LINE 会話型予約 | リッチメニュー起点の段階入力型予約 |
| 手動予約 | 管理者による電話 / 来店予約登録 |
| 空き枠計算 | 営業時間、ブロック、スタッフ、オプション時間を加味 |
| 予約ステータス管理 | 仮予約、確定、完了、キャンセル |
| 顧客キャンセル | 専用 URL からのキャンセル |

#### 顧客・店舗運用

| 機能 | 説明 |
| --- | --- |
| 顧客台帳 | 顧客情報・来店履歴・累計売上の管理 |
| 施術記録（カルテ） | 画像付き施術履歴の保存（爪の状態・デザイン・満足度など構造化フィールド対応） |
| 顧客統合・削除 | 重複顧客の統合（予約・カルテを移動）、ソフトデリート |
| メニュー / オプション管理 | 料金・所要時間・説明文の管理 |
| スタッフ管理 | 指名予約・対応メニュー・表示順管理 |
| 営業時間 / ブロック設定 | 予約可能時間と予約不可枠の制御（TimeTree 風カレンダー UI） |
| 予約可能期間制御 | 当月のみ / 翌月自動開放 / 手動開放 の月単位制御 |
| クーポン管理 | 割引コード発行・LINE 一括配布・予約時適用・利用記録 |

#### 分析・連携

| 機能 | 説明 |
| --- | --- |
| 月次分析 | 売上・予約・曜日別・メニュー別の集計 |
| Instagram 分析（Pro） | 自社 / 競合アカウントの投稿分析 |
| LINE 友だち管理 | 友だち一覧・チャット履歴・Welcome Message |
| LINE ネイルデザイン推薦 | Instagram Hashtag Search + AI でパーソナライズされたデザイン提案 |
| Web Push 通知 | 管理者向けの予約 / 障害通知（RFC 8291/8292 準拠） |
| 週報自動生成 | 内部データ + 外部データをもとにした AI レポート（Cron 生成） |
| LP アクセス分析 | PV・CTA クリック・スクロール深度・コンバージョンファネル |

#### 事業運用

| 機能 | 説明 |
| --- | --- |
| 課金管理 | Stripe Checkout / Portal / プラン変更 |
| 紹介コード | 紹介施策と利用実績の管理 |
| モニター招待 | 招待コード発行 → 無料体験 → 自動ダウングレード |
| ポートフォリオ（Pro） | 公開作品ページとテーマ設定 |
| 広告バナー | Free プラン向け広告表示（マネタイズ） |
| アフィリエイト用品紹介 | 全プラン対応の商品紹介・購入記録・経費 CSV 出力 |
| マニュアル管理 | PDF マニュアルの版管理 |

#### データ移行

| 機能 | 説明 |
| --- | --- |
| CSV データインポート（Plus/Pro） | 主要7サービスからの顧客・予約・メニュー一括取込 |

#### システム管理

| 機能 | 説明 |
| --- | --- |
| テナント管理 | テナント一覧・プラン変更・ユーザー管理 |
| 障害監視 | API エラーログ・Cloudflare 使用量 |
| Reservy AI | コードベース / ドキュメントに対するナレッジ QA |
| IG DM 営業ツール | 営業先リスト・AI トーク生成・DM 会話ビューア・アカウント自動発見 |
| 全セッション無効化 | セキュリティインシデント時の強制再ログイン |

#### 認証・セキュリティ

| 機能 | 説明 |
| --- | --- |
| パスキー（WebAuthn） | Face ID / Touch ID によるパスワードレスログイン |
| 2FA（TOTP） | ワンタイムパスワードによる二要素認証 |
| iCalendar フィード | webcal:// による予約カレンダー連携 |
| 監査ログ | 全管理者操作を `audit_log` に記録 |

### 3. 【技術スタック】

| レイヤー | 主な技術 |
| --- | --- |
| フロントエンド | React 19, TypeScript, Vite 6, Tailwind CSS 4, react-router-dom 7 |
| バックエンド | Cloudflare Pages Functions, JavaScript ES Modules, Web Crypto API |
| データ | Cloudflare D1, Cloudflare R2 |
| 通知 / 外部連携 | LINE Messaging API, LINE Login / LIFF, Web Push, Stripe, Instagram Graph API |
| AI / 検索 | Cloudflare Workers AI, Gemini, Claude, Tavily, Brave Search |
| 運用 | Wrangler, Vitest, Cloudflare GraphQL Analytics API |
