## 最小の呼び出し例

```
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

resp = client.messages.create(
    model="claude-fable-5",
        max_tokens=1024,
            messages=[
                    {"role": "user", "content": "FizzBuzzをPythonで、テスト付きで書いて"}
                        ],
                        )

                        print(resp.content[0].text)
                        ```

                        ## 長文脈を活かす：複数ファイルを丸ごと渡す

                        長文脈が強化されたモデルでは、複数ファイルをまとめて渡して横断的なリファクタリングを依頼できます。

                        ```python
                        import glob
                        from pathlib import Path

                        def collect_sources(root, ext="py"):
                            chunks = []
                                for path in sorted(glob.glob(f"{root}/**/*.{ext}", recursive=True)):
                                        code = Path(path).read_text(encoding="utf-8")
                                                chunks.append(f"### FILE: {path}\n{code}")
                                                    return "\n\n".join(chunks)

                                                    context = collect_sources("./src")

                                                    resp = client.messages.create(
                                                        model="claude-fable-5",
                                                            max_tokens=4096,
                                                                messages=[{"role": "user", "content": "重複ロジックと潜在バグを指摘し修正案を:\n\n" + context}],
                                                                )
                                                                print(resp.content[0].text)
                                                                ```

                                                                ポイントは、ファイルごとに見出しを付けて構造化すること。モデルがファイル境界を認識しやすくなり、出力の精度が上がります。

                                                                ## ストリーミングで長い出力を受け取る

                                                                ```python
                                                                with client.messages.stream(
                                                                    model="claude-fable-5",
                                                                        max_tokens=2048,
                                                                            messages=[{"role": "user", "content": "FastAPIでCRUDを一式書いて"}],
                                                                            ) as stream:
                                                                                for text in stream.text_stream:
                                                                                        print(text, end="", flush=True)
                                                                                        ```
