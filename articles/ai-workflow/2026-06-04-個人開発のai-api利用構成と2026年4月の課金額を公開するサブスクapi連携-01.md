こんにちは、個人開発者のttokunagaです。

AIモデルを開発や日常の作業に組み込む際、すべてをAPI（従量課金）で賄うとコストが読めず不安になることはありませんか？  
私は現在、\*\*「基本は月額固定のサブスクリプション（ChatGPT PlusやClaude Proなど）を使い倒し、メッセージ制限が来た時や特定の自動化タスクでのみAPIを叩く」\*\*というハイブリッド構成で運用しています。

今回は2026年4月のAPI課金実績を、先月との比較やダッシュボードのスクリーンショットとともに振り返ります。

---

## 📊 利用総額・コスト推移と合計トークン数

* **今月のAPI使用総額**: **$4.48**（約670円）
* **先月のAPI使用総額**: **$426.59**（前月比: **-$422.11**）
* **API累計使用金額**: **$1,282.57**（2025年10月から）
* **今月の合計トークン数**: **約1,325万トークン**
* **今月のリクエスト数**: **120件**

4月も引き続き、大学の大型プロジェクトでAIを使っていました。ただし3月の反省を踏まえて、APIに直接投げるのではなく、ClaudeとChatGPTの月額200ドルプランを中心に使う運用へ切り替えています。

その結果、API利用は一気に落ち着きました。OpenAI APIの軽い利用だけで済んでおり、サブスク経由に寄せた効果がそのまま数字に出ています。

---

## 🥇 OpenAI API

* **主な用途**: 大学の大型プロジェクトの補助、軽量なAPI処理、短めの実験
* **今月の消費トークン数**: **13,253,322トークン**
* **今月のAPIコスト**: **$4.48**

**💡 使用感・感想**  
3月と比べると別物のように安く収まりました。大学の大型プロジェクト自体は続いていましたが、ClaudeとChatGPTの200ドルプランを中心に使うことで、API課金としてはかなり抑えられています。

APIを使う場面を限定すると、サブスク中心の運用でも十分に作業を回せることが分かりました。自動化や連携が必要な場面だけAPIに残すのが、今の自分には一番現実的です。

## 💡 まとめ

2026年4月のAPI課金は\*\*$4.48\*\*でした。前月の$426.59から大きく下がり、ClaudeとChatGPTの200ドルプランを中心にした効果がはっきり出ています。

APIは強力ですが、毎月必ず大きく使う必要はありません。必要なときだけ使う運用にすると、個人開発のランニングコストとしてかなり扱いやすくなります。
