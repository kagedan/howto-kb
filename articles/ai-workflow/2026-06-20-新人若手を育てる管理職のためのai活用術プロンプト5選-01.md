新人や若手の育成は、管理職の大切な仕事のひとつです。でも「どう教えるか」「どんなフィードバックをするか」を毎回考えるのは時間がかかります。この記事では、AIを使って育成の準備・フィードバック・振り返りを効率化する5つの方法を紹介します。

## 管理職が育成に使う時間は意外と多い

あなたは、新人・若手の育成に毎週どのくらいの時間を使っていますか？

指導計画を立てる・フィードバックの言葉を考える・1on1の準備をする。これらをひとつひとつ丁寧にやろうとすると、自分の業務時間が削られていきます。

かといって「雑な育成」はしたくない。この板挟みが、管理職の育成における一番のストレスです。

AIを使うと、育成の「準備」にかかる時間を短縮できます。「何を指導するか・どう評価するか」という判断はあなたがしますが、「準備の構造を整える・言葉を整える」ことはAIに手伝ってもらえます。

5つのプロンプトを紹介します。

## 育成を効率化する5つのプロンプト

### プロンプト①：育成計画・指導スケジュールを作る

新人が入ってきたとき、最初の1〜3ヶ月の指導計画を作ります。

```
以下の条件で、新人・若手向けの育成計画を作成してください。

対象：【例：入社1ヶ月目の新入社員・営業職】
期間：【例：入社から3ヶ月間】
習得してほしいスキル：【例：商品知識・電話応対・日報の書き方・小口案件の担当】
現状のレベル：【例：社会人経験なし・PC操作は問題なし】
週の指導時間：【例：朝礼10分＋週1回の振り返り30分】
```

このプロンプトで何が得られるか：月ごとのマイルストーン・週ごとのタスク・確認ポイントが整った指導計画が出てきます。一から考える時間が半分以下になります。

**なぜ効くのか**：育成計画は「全体像」があるかどうかで指導の質が大きく変わります。行き当たりばったりの指導は本人の成長を遅らせます。AIが骨格を作ってくれるので、あなたはその内容を確認・調整するだけで済みます。

### プロンプト②：「自分で考えさせる」問いかけを設計する

部下から「どうすればいいですか？」と聞かれたとき、毎回答えを教えていませんか？

```
以下の状況で、部下が自分で考えるための「問いかけ」を5つ作成してください。

状況：【例：新人が「顧客からクレームを受けたがどう対応するかわからない」と相談してきた】
部下のレベル：【例：入社4ヶ月目・基本業務は習得済み】
目的：【例：自分で考えて行動できるようになってほしい・答えを教えずに気づかせたい】
```

このプロンプトで何が得られるか：「あなたはどう思う？」だけでなく、具体的で答えやすい問いかけが複数出てきます。部下の考える力を引き出す会話が設計できます。

**なぜ効くのか**：答えを教え続けると、部下は「答えをもらう」ことに慣れていきます。自分で考えるクセをつけさせる問いかけは管理職の重要なスキルですが、毎回考えるのは大変です。AIに構成を作ってもらうと、引き出しが増えます。

### プロンプト③：フィードバックの文章を整える

成長を感じてもらえるフィードバックは、言葉の選び方が大切です。

```
以下の状況で、部下へのフィードバック文を作成してください。

対象：【例：入社6ヶ月目の若手スタッフ】
良かった点：【例：顧客への報告書を自発的に改善してきた・読みやすくなっていた】
改善してほしい点：【例：報告のタイミングが遅い・完成してから報告するのではなく、途中経過を共有してほしい】
フィードバックの目的：【例：自信を持たせながら次の課題を伝えたい】
```

このプロンプトで何が得られるか：ポジティブな面を先に伝え、改善点を前向きに伝える構成のフィードバック文ができます。「厳しすぎる・優しすぎる」のバランスをAIが整えてくれます。

**なぜ効くのか**：フィードバックは「何を言うか」より「どう言うか」で受け取り方が大きく変わります。特に改善点の伝え方は難しいです。AIに言葉を整えてもらうことで、伝わる確率が上がります。

Before：「報告が遅いです。次からはもっと早くしてください。」

After：「報告書の改善、しっかり気づけていましたね。次のステップとして、完成を待たずに途中経過を共有する習慣をつけていきましょう。」

### プロンプト④：振り返りシートを作る

定期的な振り返りの場を設けると、部下の成長速度が変わります。

```
以下の条件で、若手スタッフ向けの振り返りシートを作成してください。

対象：【例：入社3ヶ月目の若手・月次振り返りで使用する】
振り返りのポイント：
【例：①今月の目標達成度 ②うまくいったこと ③うまくいかなかったこと ④原因の考察 ⑤来月の目標】
所要時間：【例：記入15分・上長との確認15分】
```

このプロンプトで何が得られるか：記入しやすく、成長を可視化できる振り返りシートが出てきます。そのままExcelやGoogleフォームに転記して使えます。

**なぜ効くのか**：振り返りがないと、部下は「なんとなく仕事をこなす」だけになりがちです。シートがあると「何を振り返ればいいか」が明確になり、本人の気づきが増えます。

### プロンプト⑤：1on1前の話すことを整理する

1on1（上長と部下の定期的な対話）を実りある時間にするための準備です。

```
以下の状況で、1on1で話すべきことを整理してください。

対象：【例：入社8ヶ月目の若手・最近元気がない様子・業務量は多くない】
前回の1on1から気づいたこと：【例：先週、小さなミスが続いた・報告が少なくなった】
確認したいこと：【例：最近どんな状態か・業務に不安はないか・今後のキャリアに希望はあるか】
目的：【例：本音を引き出したい・安心感を与えたい】
```

このプロンプトで何が得られるか：話の切り出し方・確認するポイント・締めの言葉の流れが整います。1on1の準備が5分でできます。

**なぜ効くのか**：1on1は「何を話せばいいかわからない」という管理職も多いです。特に部下の様子が変わったときは、切り出し方に悩みます。AIが流れを整えてくれると、当日の会話に集中できます。

## よくある失敗例

**失敗1：指導計画を作って、そのまま忘れる**

AIで育成計画を作っても、月次で進捗を確認しなければ意味がありません。「作って終わり」になりがちです。計画を作ったら、毎月の振り返りとセットで使うことが大切です。

**失敗2：フィードバックをAIに丸投げする**

AIが作ったフィードバック文は、あくまで「叩き台」です。実際の部下のキャラクター・これまでの関係性・文脈を踏まえた修正が必要です。言葉をそのまま使わず、自分の言葉に変換してから伝えてください。

## よくある質問

**Q：AIに部下の情報を入力しても大丈夫ですか？**

氏名・社員番号など個人を特定できる情報は入力しないようにしてください。「入社6ヶ月目の若手」など属性情報に留めれば問題ありません。社内のセキュリティポリシーも事前に確認しておくと安心です。

**Q：部下が少ない（1〜2人）場合でも使えますか？**

むしろ少人数こそ丁寧な育成ができる機会です。人数が少ないほどプロンプトに具体的な情報を入れやすく、より実情に合った内容が出てきます。

**Q：テレワーク環境での育成にも使えますか？**

使えます。特にプロンプト②（問いかけ設計）とプロンプト⑤（1on1準備）はオンライン1on1でそのまま活用できます。

## まとめ

育成の「準備・フィードバック・振り返り」をAIで整えると、管理職自身の負担が減り、部下への指導の質が上がります。

今日できる最初の一歩は、次の1on1の前にプロンプト⑤を試してみることです。準備が整うだけで、当日の会話の質が変わります。
