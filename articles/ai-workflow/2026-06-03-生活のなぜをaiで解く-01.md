「また同じことをやってしまった…」

そう思いながら、  
毎日を過ごしていませんか？

毎朝の準備でバタバタする。  
買い物に行くたびに  
買い忘れが出る。

やることに追われて  
「今日も何もできなかった」で  
一日が終わる。

その「面倒くさい」の積み重ねが、  
毎日の疲れの正体です。

でも、それは  
あなたがだらしないからでも、  
要領が悪いからでもありません。

**「当たり前」だと思って  
疑っていなかっただけです。**

---

この記事を読むと、  
こんな変化が起きます。

✅ **致命的ミスの回避**  
　→ 「またやってしまった」という  
　　同じミスの繰り返しを  
　　仕組みで断ち切れる

✅ **冷静な判断力**  
　→ 毎日の「考えること」が減り、  
　　本当に大切なことだけに  
　　エネルギーを使えるようになる

✅ **機会損失の解消**  
　→「面倒だからいいや」が消え、  
　　やりたいことに  
　　時間と心が向くようになる

---

AIは「あなたの日常に潜む無駄を  
一緒に探してくれる相棒」であり、  
「当たり前を疑う思考のパートナー」です。

今日からあなたの日常は、  
少しずつ、確実に楽になっていきます。

### ◆日常の無駄を発見するプロンプト

**✅目的：**  
　自分では気づいていない  
　日常の「無駄な行動」を特定し、  
　すぐに改善できる方法を得る。

**✅いつ使うか：**  
　「なんとなく毎日忙しい」  
　「時間が足りない気がする」と  
　感じているとき。

```
# ロール設定
あなたは日常生活の
ライフハック専門家です。
相談者の1日の行動を分析し、
無意識に繰り返している
「時間・エネルギーの無駄」を
特定してください。

# 背景・目的・対象
忙しい日常を送る大人が対象です。
大きな生活改革ではなく、
「今日から変えられる小さな無駄」を
優先して見つけてください。

# 指示内容
以下の【入力データ】をもとに、
無駄を「時間の無駄」
「エネルギーの無駄」
「お金の無駄」の
3カテゴリで分類し、
各カテゴリから
改善できるものを1つずつ
提案してください。

# 制約条件
・今日から始められるものを優先する
・特別な道具や費用が
　不要なものを優先する
・「やめるだけでいいこと」も
　必ず1つ含める

# 出力形式
【発見された無駄】
・時間の無駄：〜
・エネルギーの無駄：〜
・お金の無駄：〜

【今日から変えられること】
1. 〜
2. 〜
3. 〜

【やめるだけでいいこと】
・〜

# 入力データ
・平日の起床〜就寝までの
　大まかな流れ：
　（例：6時起床→準備→9時仕事開始
　　→17時終了→家事→23時就寝）
・毎日「面倒だな」と感じる場面：
　（例：夕食の献立を考えること）
・「なぜかいつも時間を取られる」
　こと：
　（例：鍵や財布を毎朝探している）
・最近「また無駄にした」と
　感じたこと：
　（例：使い切れなかった食材を捨てた）
```

**✅具体的な使用方法：**  
　「入力データ」に  
　今日の1日をそのまま書くだけ。

　うまくまとめなくて大丈夫です。

　思いつくままに書いた言葉から、  
　AIが無駄を見つけてくれます。

**✅読者への一言：**  
　「自分の無駄」は  
　自分では見えにくいものです。

　他人の目線で整理してもらうだけで、  
　「そこか！」という発見が  
　必ずあります。

### ◆「当たり前」を疑うプロンプト

**✅目的：**  
　長年続けてきた習慣の中から  
　「実は必要なかった行動」を  
　洗い出し、生活をシンプルにする。

**✅いつ使うか：**  
　「なんとなくずっとこうしてきた」  
　という習慣を見直したいとき。

```
# ロール設定
あなたは行動経済学と
ライフデザインの専門家です。
「当たり前」と思われている行動の中から、
実は不要・非効率なものを
洗い出してください。

# 背景・目的・対象
長年の習慣を持つ大人が対象です。
「やめることへの罪悪感」を
取り除きながら、
シンプルな生活への
一歩を後押しする提案をしてください。

# 指示内容
以下の【入力データ】をもとに、
「本当に必要か？」を
問い直すべき習慣を3つ特定し、
それぞれに
①やめた場合のメリット
②やめる際の心理的ハードルと
　その乗り越え方
を提示してください。

# 制約条件
・「やめましょう」と
　断言するのではなく
　「試しにやめてみる」
　という提案にする
・1週間だけ試せる
　小さな実験として提案する
・やめた後の「代替案」も
　セットで提示する

# 出力形式
【見直すべき習慣①】
習慣：〜
やめた場合のメリット：〜
心理的ハードルと乗り越え方：〜
1週間の実験：〜

（②③も同様）

# 入力データ
・毎日欠かさずやっていること：
　（例：朝のニュースを1時間見る）
・「本当に必要かな」と
　薄々感じていること：
　（例：週1の大掃除）
・やめたいけどやめられないこと：
　（例：SNSのチェック）
・「これをやめたら楽になりそう」
　と感じること：
　（例：毎日のお弁当作り）
```

**✅具体的な使用方法：**  
　「やめたいけどやめられないこと」に  
　正直に書いてください。

　AIは責めるのではなく、  
　「1週間だけ試す」という  
　小さな実験として  
　背中を押してくれます。

**✅読者への一言：**  
　「当たり前」をやめることに  
　罪悪感は不要です。

　1週間試してみて、  
　困らなければやめていい。  
　それだけのことです。

## ビフォー・アフター比較

**同じ悩みでも、プロンプトの作り方で  
結果は大きく変わります。**

想定シナリオ：  
「毎日の家事と仕事に追われ、  
『なぜかいつも時間が足りない』と  
感じている会社員」

---

### Before（簡易プロンプト）

```
毎日を効率よく過ごすコツを教えて。
```

**AIの回答（例）**：

> 優先順位をつけてタスクを管理しましょう。  
> 朝の時間を有効活用することも効果的です。  
> 不要なことを減らすことで  
> 時間が生まれます。

---

👉 **どこが足りないか？**

---

### After（本記事のプロンプトを使用）

```
# 入力データ
・平日の流れ：
　7時起床→準備30分
　→9時仕事開始→18時終了
　→買い物・夕食・片付け
　→23時就寝
・面倒な場面：
　夕食の献立を毎日考えること
・時間を取られること：
　スーパーで何を買うか
　毎回その場で考えている
・無駄にしたと感じたこと：
　週3回は食材を使い切れず捨てている
```

**AIの回答（例）**：

>  【発見された無駄】  
>  ・時間の無駄：  
>  献立を毎日その場で考えることで、  
>  1回あたり平均15〜20分消費。  
>  週7日で約2時間の思考コストが  
>  発生しています。  
>   
>  ・お金の無駄：  
>  食材の使い切れないロスは、  
>  月換算で3,000〜5,000円相当に  
>  なっている可能性があります。  
>   
>  【今日から変えられること】  
>  1. 週1回だけ「今週の献立7日分」を  
>  AIに作ってもらい、  
>  買い物リストも同時に作成する  
>   
>  2. スーパーに行く前に  
>  リストをスマホで確認する習慣をつける  
>   
>  【やめるだけでいいこと】  
>  ・毎日スーパーに行くこと  
> （週2回まとめ買いに変えるだけで  
>  往復時間と衝動買いが激減します）

---

👉 **何が変わったか？**

「ロール設定」で  
AIをライフハック専門家として起動し、

「背景指定」で  
1日の流れと具体的な悩みを渡したことで、

**数字と具体的な行動が  
セットで出てきました。**

情報を渡すほど、  
AIの回答は精度が上がります。

これがプロンプトの核心です。

## 1. 献立と買い物を一括自動化する

「今日の夕食、何にしよう」

この一言が、  
毎日どれだけのエネルギーを  
奪っているか、  
考えたことはありますか？

献立を考えて、  
冷蔵庫を確認して、  
買い物リストを作って、  
スーパーを回る。

この一連の流れを、  
週1回のAI作業に  
まとめてしまいましょう。

---

### ◆週間献立＆買い物リスト自動化プロンプト

**✅目的：**  
　毎日の「何を作ろう」を週1回の  
　AI作業に集約し、  
　食費と時間の両方を削減する。

**✅いつ使うか：**  
　週の始め（日曜夜か月曜朝）に  
　1回だけ使う。

```
# ロール設定
あなたは家庭料理と
食費管理の専門家です。
家族の好みと冷蔵庫の食材を踏まえた、
無駄のない1週間の献立と
買い物リストを作成してください。

# 背景・目的・対象
忙しい家庭が対象です。
「作るのが大変すぎない」
「食材を使い切れる」
「食費が予算内に収まる」の
3点を最優先にしてください。

# 指示内容
以下の【入力データ】をもとに、
月〜日の夕食7日分の献立と、
それに必要な買い物リストを
同時に作成してください。
食材の使い回しを意識して
ロスが出ない構成にすること。

# 制約条件
・調理時間は1食30分以内を目安にする
・同じ食材を複数の料理で
　使い回す構成にする
・週の後半は「作り置きの活用」を
　組み込む

# 出力形式
【今週の献立】
月曜：〜（調理時間：〜分）
火曜：〜
…（日曜まで）

【今週の買い物リスト】
野菜：〜
肉・魚：〜
その他：〜

【作り置きのすすめ】
〜

# 入力データ
・家族構成：（例：大人2人・子ども1人）
・苦手な食材・アレルギー：
　（例：魚介類が苦手）
・今冷蔵庫にあるもの：
　（例：豚肉・じゃがいも・玉ねぎ）
・今週の予算：（例：夕食のみで1万円）
・特に楽にしたい曜日：
　（例：水・金は時間がない）
```

**✅具体的な使用方法：**  
　「今冷蔵庫にあるもの」を  
　書き出してから送信してください。

　冷蔵庫の写真を  
　一緒に送る方法も有効です。

**✅読者への一言：**  
　「献立を考える」という行為は、  
　想像以上に脳を消耗させます。

　週1回のAI作業に変えるだけで、  
　毎日の夕方の気持ちが  
　驚くほど楽になります。

## 2. 「探す時間」をゼロにする

鍵、スマホ、書類、充電器。

毎日どこかで何かを  
探していませんか？

**約21時間**になります。

**✅目的：**  
　「探す行動」を仕組みで消し、  
　毎日の小さなストレスを  
　一掃する。

**✅いつ使うか：**  
　「また探してしまった」と  
　感じたとき。  
　または引っ越し・模様替えの  
　タイミングに。

```
# ロール設定
あなたは整理収納と
行動動線の専門家です。
「探す行動がゼロになる」
モノの定位置と収納ルールを
設計してください。

# 背景・目的・対象
片付けが苦手・続かないと感じる
大人が対象です。
「ルールが複雑すぎて続かない」を
避けるため、
できるだけシンプルな仕組みを
作ってください。

# 指示内容
以下の【入力データ】をもとに、
よく探してしまうモノの
「定位置ルール」を設計し、
「戻す行動が自然に起きる仕掛け」を
提案してください。

# 制約条件
・ルールは3つ以内に絞る
・特別な収納グッズを
　買わなくてもできるものを優先する
・「なぜそこに置くのか」の
　理由も添える

# 出力形式
【定位置ルール】
モノ①：場所・理由〜
モノ②：〜
モノ③：〜

【戻す行動が自然に起きる仕掛け】
〜

【今すぐできる配置変更】
〜

# 入力データ
・よく探してしまうもの：
　（例：鍵・スマホ・リモコン）
・家の間取りのざっくりした説明：
　（例：1LDK・玄関から
　　リビングがすぐ隣）
・よく使う動線：
　（例：帰宅→リビング→
　　キッチン→寝室）
・過去に試して続かなかったこと：
　（例：専用トレーを置いたが
　　使わなくなった）
```

**✅具体的な使用方法：**  
　「過去に試して続かなかったこと」を  
　正直に書いてください。

　AIはその失敗を踏まえて、  
　「今度こそ続く仕組み」を  
　提案してくれます。

**✅読者への一言：**  
　「探す5分」を  
　笑えないのは、  
　毎日積み上がるからです。

　定位置を決めるのに  
　時間はかかりません。

　今日中に終わります。

### ◆未処理タスク一掃プロンプト

**✅目的：**  
　頭の中に溜まっている  
　「やらなきゃリスト」を  
　全部吐き出して整理し、  
　今週中に終わらせる順番を決める。

**✅いつ使うか：**  
　「やらなきゃいけないことが  
　多すぎて頭がパンクしそう」  
　なとき。

```
# ロール設定
あなたはタスク整理と
優先順位付けの専門家です。
相談者の「やらなきゃリスト」を
整理し、
最小の労力で最大の
スッキリ感を得られる
順番を設計してください。

# 背景・目的・対象
先送りが癖になっている大人が対象です。
「全部やる」ではなく、
「本当に必要なものだけを
正しい順番でやる」ことを
目指してください。

# 指示内容
以下の【入力データ】をもとに、
タスクを
「今週必ずやる」
「来月までにやる」
「実はやらなくていい」の
3つに仕分けし、
「今週必ずやる」の
実行順序を決めてください。

# 制約条件
・「実はやらなくていい」を
　必ず1つ以上見つける
・各タスクに
　「所要時間の目安」を添える
・一番最初にやるべきタスクに
　理由をつける

# 出力形式
【今週必ずやる（優先順）】
1. 〜（所要時間：〜分）理由：〜
2. 〜
…

【来月までにやる】
・〜

【実はやらなくていい】
・〜（理由：〜）

# 入力データ
・頭の中にある「やらなきゃ」を
　思いつく限り全部書く：
　（例：区役所で住民票を取る・
　　スマホのプランを見直す・
　　実家に電話する・
　　車の車検の予約・
　　クローゼットの整理）
・期限があるもの：
　（例：車検は来月まで）
・一番気が重いもの：
　（例：スマホのプランの見直し）
```

**✅具体的な使用方法：**  
　「やらなきゃ」を  
　箇条書きで全部書き出してから  
　送信してください。

　整理する前に書き出すのが  
　重要です。

　頭から出すだけで、  
　半分くらいスッキリします。

**✅読者への一言：**  
　「やらなきゃ」が頭の中にある限り、  
　ずっとじわじわと  
　エネルギーを消耗します。

　紙に書いてAIに渡す。

　たったそれだけで、  
　頭が一気に軽くなります。

## 4. 固定費を見直して毎月を楽にする

「お金が足りない」と感じるとき、  
収入を増やすことより先に  
やるべきことがあります。

**「出ていくお金の無駄を止める」こと。**

サブスクリプション、  
保険、スマホのプラン、  
光熱費。

毎月自動で引き落とされている  
「忘れていたお金」が、  
必ずあります。

### ◆固定費最適化プロンプト

**✅目的：**  
　毎月自動で出ていく固定費を見直し、  
　生活の質を下げずに支出を減らす。

**✅いつ使うか：**  
　「なんとなくお金が貯まらない」  
　と感じているとき。  
　または年に1回の  
　定期メンテナンスとして。

```
# ロール設定
あなたは家計最適化の
ファイナンシャルアドバイザーです。
生活の質を落とさずに
固定費を削減する方法を
提案してください。

# 背景・目的・対象
固定費の見直しに
なかなか手が回らない
忙しい大人が対象です。
「調べる手間を最小化しながら
確実に削減できる」方法を
優先してください。

# 指示内容
以下の【入力データ】をもとに、
見直すべき固定費を特定し、
削減できる金額の目安と
具体的なアクションを
提案してください。

# 制約条件
・「解約しましょう」ではなく
　「プラン変更で済むもの」を
　優先する
・各項目の見直しに
　かかる時間の目安を添える
・「触らなくていいもの」も
　明示する

# 出力形式
【見直し優先度：高】

アクション：〜（所要時間：〜分）

【見直し優先度：中】
〜

【触らなくていいもの】
〜

【合計削減見込み】
月〜円（年〜円）

# 入力データ

　（例：スマホ代9,000円・

　（例：2年以上前）
```

## 5. 「考えること」自体を減らす

毎日の疲れの多くは、  
「動いた疲れ」ではなく  
「考えた疲れ」です。

今日の献立、  
今週のタスク、  
返信するメール、  
週末の予定。

判断や決断は、  
回数が増えるほど質が下がります。

**「考えなくていい仕組み」を作ることが、  
最高のライフハックです。**

---

### ◆思考コスト削減プロンプト

**✅目的：**  
　毎日繰り返している  
　「小さな判断」を仕組み化し、  
　脳のエネルギーを本当に大切なことだけに  
　使えるようにする。

**✅いつ使うか：**  
　「なんか毎日疲れている気がする」  
　という漠然とした疲れを  
　感じているとき。

```
# ロール設定
あなたは行動設計と
認知負荷削減の専門家です。
毎日無意識に繰り返している
「小さな判断」を特定し、
それを仕組みに変える方法を
提案してください。

# 背景・目的・対象
毎日の判断疲れを感じている
大人が対象です。
「考えなくてもいい状態」を
作ることで、
重要な判断に
エネルギーを集中できるよう
設計してください。

# 指示内容
以下の【入力データ】をもとに、
毎日繰り返している「判断」の中から
仕組み化・ルール化できるものを
3つ特定し、
それぞれの「考えなくなる方法」を
提案してください。

# 制約条件
・「決める手間」が少ない
　シンプルなルールにする
・1週間で習慣化できる
　レベルの変化にする
・「これを決めたら考えなくていい」
　という明快なルールにする

# 出力形式
【思考コストを削減できる判断①】
今やっていること：〜
仕組み化の方法：〜
ルール：「〜と決める」

（②③も同様）

【仕組み化後に空く時間・エネルギー】
〜

# 入力データ
・毎日「どうしようかな」と
　考えていること：
　（例：今日の服・
　　昼食の注文・
　　返信の順番）
・毎週「また考えた」と感じること：
　（例：週末の過ごし方・
　　掃除のタイミング）
・「もうルール化したい」と
　感じていること：
　（例：何でもいいから
　　朝食を固定したい）
```

**✅具体的な使用方法：**  
　「毎日『どうしようかな』と  
　　考えていること」に  
　思いつく限り書いてください。

　些細なことほど、  
　仕組み化の効果が大きいです。

**✅読者への一言：**  
　スティーブ・ジョブズが  
　毎日同じ服を着ていたのは、  
　「服を選ぶ判断」を  
　なくすためだったと言われています。

　大げさに聞こえるかもしれませんが、  
　小さな判断を減らすだけで、  
　毎日の疲れ方は確実に変わります。

## まとめ

「面倒くさい」を  
「仕方ない」で終わらせないでください。

献立を考える時間、  
モノを探す時間、  
雑務を先送りする時間、  
固定費を払い続ける時間、  
毎日繰り返す判断の時間。

そのすべてをAIと一緒に見直せば、  
気づかないうちに毎日1〜2時間が  
手元に戻ってきます。

「当たり前」を疑うことが、  
豊かな日常への一番の近道です。

今日、一つだけ  
「本当に必要か？」と  
問い直してみてください。
