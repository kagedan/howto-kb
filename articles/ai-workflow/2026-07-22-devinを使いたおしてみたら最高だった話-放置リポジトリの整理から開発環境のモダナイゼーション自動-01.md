今回 [Oikonさん](https://x.com/oikon48) からDevinの一ヶ月クーポンを頂き、  
Devinを試す機会があったので色々触ってみました。

実際に試したことや各機能の感想を素直にまとめてみました。

---

## まずやったこと（いきなり週制限を飛ばす）

### 1. 放置リポジトリの整理（README / AGENTS.md の更新）

最初はちょっとアホな使い方（？）からスタート。昔からコツコツ積み上げてきた自分が持っている複数リポジトリに対して、`README.md` や `AGENTS.md` を更新するタスクをブランチを切って一括で割り振ってみました。

* **複数リポジトリまとめて指示できる体験にまず感動！**（オンボーディング直後のこの衝撃はすごかった）
* ずっと放置・メンテしていなかったリポジトリが綺麗に整理された。
* 「これなんだっけ……？」状態の過去作も、`AGENTS.md` を読めば一瞬で思い出せるようになって大満足。

### 2. メインで更新しているVitePlugin系の依存環境アップデート

手動でやるとAIを使っても地味に時間がかかるのが依存ライブラリの更新作業。扱っているリポジトリもそこそこあるので億劫だったんですが、これを一括指示！

### 3. 放置していたOSSのPRレビュー ＆ Slack通知

放置気味のリポジトリに残っていたPRの処理。「今更見ながらマージするのめんどいな…」という作業をすべてDevinにお任せしました。

* 最終決定権は自分が持ちたかったので、**完了したらSlackに報告が飛ぶように設定**したらめちゃくちゃ快適に。

### 4. 5年熟成された「秘伝のタレ」制作環境のモダナイゼーション

Web屋をやっているとサイト制作のテンプレートを作るんですが、これが5年前から積み重ねた秘伝のタレコード。ぶっちゃけ誰もメンテしたくないやつです。

これをDevinに「モダナイズして」と投げてみたら、**ものの1時間で機能はそのまま、中身だけが最高のモダン構成に生まれ変わりました。** さよならgulp, webpack……！

## 各機能のリアルな感想・推しポイント

### 1. Security Swarm

<https://docs.devin.ai/ja/work-with-devin/security-swarm>

**これ、めちゃめちゃ良かった！！**

複数リポジトリをまとめてセキュリティチェックしてくれる機能なんですが、ちゃんと優先度（高, 中...）をつけて対応してくれます。

![](https://static.zenn.studio/user-upload/6cf54f7b9258-20260722.png)

昔作ったMCPのリポジトリを通したら、目も当てられない惨状（このザマ）であることが判明して震えました……笑

### 2. Data Analyst Agent

<https://docs.devin.ai/ja/work-with-devin/data-analyst>

ざっくり言うとDB分析ができるモード。個人開発レベルだと一見ユースケースに困りそうですが、個人管理しているDiscordサーバーのログを読ませて分析させてみました。

* 「直近3ヶ月のユーザー発言ランキングを作って」
* 「直近3ヶ月のアクティブ時間帯別まとめ表を作って」

こんな雑な指示でもサクッと分析してくれました（※鯖民の許可を取ってないのでスクショはお見せできませんが！）。対応データベースがやけに潤沢なのも助かるポイント。

### 3. DeepWiki

<https://docs.devin.ai/ja/work-with-devin/deepwiki>

リポジトリを自動インデックス化して、アーキテクチャ図・情報源リンク・コードベース要約を含むWikiを自動生成してくれる機能。

以前からあった機能ですが、**さまざまな言語（特に日本語！）に対応したのがデカい！**  
自分のツールのWikiが日本語で自動生成される体験、それだけで胸アツすぎます。

![](https://static.zenn.studio/user-upload/7d20b7be8535-20260722.png)

### 4. Devin Review

<https://docs.devin.ai/ja/work-with-devin/devin-review>

一言でいうとPR Reviewツール。

過去のガバガバコードをAIエージェントに改修させてPRを出させるものの、当の本人がポンコツなのでPRのセルフレビューすらまともにできない……そんな私に降り注いだ一本の光！

* `REVIEW.md` を用意しておけば「注意すべき点」や「ここは無視してOK」などを指定できる。
* 変更量が大量なときに**変更サマリー（Summary）を出してくれるのが本当に助かる。**

![](https://static.zenn.studio/user-upload/ba366f799866-20260722.png)

### 5. ComputerUse / テストと動画録画

<https://docs.devin.ai/ja/work-with-devin/testing-and-recordings>

正直、DevinってUI系のテストが弱いイメージがありました。  
でもログを眺めていたら、**実際にブラウザを操作してテスト＆録画までしてくれていた！** しかも精度がすごい上に爆速。イメージが変わりました。

![](https://static.zenn.studio/user-upload/8cec294a265b-20260722.webp)

### 6. Slack連携

<https://docs.devin.ai/ja/integrations/slack>

普段ずっとSlackを開いているので、Slackから雑に指示を出せるのが超快適。試しにめちゃめちゃしょうもない会話もしてみました（笑）。

![](https://static.zenn.studio/user-upload/f35fce1b42b8-20260722.png)

最初 Slack 上での機能呼び出し方が分からなかったんですが、Wikiを見たら「`!` から始まるコマンド」で呼び出せるとのこと。昔のDiscord bot感があって分かる人には刺さるやつです。

![](https://static.zenn.studio/user-upload/a9268f2b53cb-20260722.png)

### 7. 自動化（Triggers）

GitHub Webhook、Linear、スケジュール設定などをトリガーにしてDevinを自動起動できます。

* 頻繁に使う小規模ツールの依存関係アップデートを週1回自動化。
* マイナーアプデは勝手に更新。
* 破壊的変更があった時は勝手に自動修正。

なにこれ便利……感動。他にもGitHub Issueを投げておけばやってくれるし、Slackでリアクションを付けたら動いてくれる。**感覚としては「優秀な開発秘書」が1人ついたイメージです。**

## 今回試せなかった機能 & 今後注目したいポイント

### Devin CLI

<https://docs.devin.ai/ja/cli>

試したかったものの、普段GUI派の自分としてはCLIで動かす明確な動機が見つからず後回しに。気づいたら週制限に達して終了＼(^o^)／

### Devin MCP

<https://docs.devin.ai/ja/work-with-devin/devin-mcp>

Devin単体で完結していたため、他のAIエージェントからDevinを呼び出す具体的なユースケースが思い浮かばず。ヘビーユーザーになるとこのへんも活用できるのかな？

### Devin Desktop / Local

<https://docs.devin.ai/ja/desktop/getting-started>

今回はクラウド前提で動かしていたため見送ったのですが、実はCognitionの方からアツい補足情報をいただきました！

裏でこうした役割特化のサブエージェントが自動連携しているからこそ、あの高い精度が出せるんですね。次回は `.devin/agents/` で自分好みのカスタムエージェントを作る沼にもハマってみたいと思います！

## まとめ

今回Devinをがっつり触ってみて感じたのは、単なる「コードを書いてくれるAI」ではなく、**「開発プロセス全体を隣で支えてくれるパートナー（専属秘書）」** だということです。

特に、放置されがちなリポジトリの整理や「秘伝のタレ」化したビルド環境のモダナイゼーション、週次の自動メンテナンスといった、**「面倒だけどやらなきゃいけない作業」を丸投げできる価値**は計り知れません。
