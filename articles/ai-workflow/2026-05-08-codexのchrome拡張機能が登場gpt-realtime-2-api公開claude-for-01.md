## 🚀 CodexがChrome拡張機能として独立動作！ブラウザ操作の常識が変わる

OpenAIのAIエージェント「Codex」が、ついにChrome拡張機能として単独で動けるようになりました！

これまでのCodexデスクトップアプリは、AIがブラウザを操作している最中に自分もブラウザを触ると処理が止まってしまうという地味なストレスがあったんです。でも今回の拡張機能では、**専用の独立したタブで動作する**ため、その問題がまるっと解消されました！

実際の動作は速くて正確。Xのポストをリサーチさせたり、スプレッドシートを操作させたり、さらにはページの内部構造（DOM）を分析させることも可能っぽいんです。夢が広がるのは「CodexとChatGPT Proを会話させて要件定義を自動化する」みたいな応用技。Codexのデスクトップアプリ → プラグイン設定 → Chromeを選択、で簡単に入れられますよ。

## 🔊 GPT-Realtime-2 API公開！ChatGPTボイスモードが進化を予告

OpenAIがリアルタイム音声エージェント「GPT-Realtime-2」のAPIを公開しました。前バージョンから**リアルタイム性が大幅向上**し、会話の途中で割り込めるようにもなったとのこと。デモ動画を見るとそのなめらかさに思わず「おぉ」となるはずです。

ただし現時点ではAPIのみの公開で、ChatGPTアプリへの搭載はもう少し先。OpenAI公式も「ボイスの更新を心待ちにしていることは分かっている。続報をお楽しみに」とコメントしています。

料金は入力100万トークンあたり$32、出力は$64と、精度の高さに見合った設定。一般ユーザーはChatGPTのボイスモードのアップデートを待つのがベストで、開発者向けの機能として今後どんなアプリが生まれるかが楽しみですね！

## 🏢 Claude for Excel・PowerPoint・Wordが正式リリース！Office連携がついに本番へ

AnthropicがClaude for Excel、PowerPoint、Wordの**一般提供（GA）開始**を発表しました！これまでパブリックベータとして試験提供されていたものが、ついに正式版として誰でも使えるようになりましたよ。

Outlookは引き続きパブリックベータ版ではありますが、主要なOfficeアプリではClaudeをフル活用できる環境が整ったことになります。日常的にExcelでデータ整理したり、PowerPointで資料を作ったりしている方には、直接的な業務効率化につながる嬉しいニュースですね。

企業でMicrosoft 365を使っているビジネスパーソンはぜひチェックしてみてください！

## 🛠️ Cursor 3がPRレビューとマルチスタッキングを強化！開発ワークフローが一本化

AIコードエディタのCursorが大型アップデートで、**プルリクエストの作成からマージまでをCursor内で完結**できるようになりました。GitHubとブラウザを行き来する手間がなくなるので、エンジニアさんにはかなりありがたい進化のはず。

さらに注目は「マルチスタッキング」機能で、プランを並列サブタスクに分割して同時進行できるように。スラッシュコマンドから呼び出せるサブエージェントの並列実行がより使いやすくなり、複雑な実装作業を一気に効率化できそうです。
