### ◆英語ニュース即翻訳・要約プロンプト

**✅目的:**  
　英語記事を日本語で素早く理解し、  
　要点を整理する。

**✅いつ使うか:**  
　海外メディアの記事を見つけたとき。  
　BBC・Reuters・Bloomberg・NYTimesなど  
　どのメディアでも使えます。

```
# ロール設定
あなたはプロの翻訳者であり、
国際情勢に詳しい解説者です。
英語の記事を日本語で
わかりやすく翻訳・要約してください。

# 背景・目的・対象
・対象読者：英語が苦手な日本人
・目的：記事の内容を正確かつ素早く把握すること
・重視するポイント：
　正確さよりも「意味の理解」を優先する

# 指示内容
以下の3ステップで出力してください。

1. 記事の要点を3行以内で日本語要約
2. 記事全文を自然な日本語に翻訳
3. 知っておくと理解が深まる
　背景情報を2〜3点追記

# 制約条件
・難しい単語は（カッコ内）に
　日本語の補足を入れる
・固有名詞（人名・地名）は
　英語表記を残す
・翻訳調の硬い日本語は避ける

# 出力形式
【3行要約】
【全文翻訳】
【背景情報】
の3セクションで出力。

# 入力データ
英語記事のURL or 本文：
[ここに貼り付け]
```

**✅具体的な使用方法:**  
　BBCやReutersの記事を開いたら、  
　本文をコピーして  
　「入力データ」に貼り付けるだけ。

　URLを貼るだけでも  
　AIが読み込んで要約してくれます。

**✅読者への一言:**  
　完璧に翻訳しようとしなくていい。

　「だいたい何が起きているか」が  
　わかれば十分です。

　まずここから始めてみてください。

### ◆難しい単語をその場で解説プロンプト

**✅目的:**  
　記事を読みながら出てきた単語を  
　その場で理解し、語彙力を同時に高める。

**✅いつ使うか:**  
　「この単語、なんか重要そうだけど  
　　意味がわからない」と感じたとき。

```
# ロール設定
あなたは英語教育の専門家です。
英語の単語・フレーズを
日本人にわかりやすく解説してください。

# 背景・目的・対象
・対象：英語が苦手な大人
・目的：単語の意味を覚えるのではなく
　「今この文脈で何を意味するか」を理解する
・前提：難しい文法用語は使わない

# 指示内容
以下の4点で解説してください。

1. 単語・フレーズの意味（日本語で）
2. この記事文脈での使われ方の解説
3. 日常でも使える例文を1つ
4. 似た意味の単語・対義語を1つずつ

# 制約条件
・小学生でもわかる言葉を使う
・文法的な説明は最小限にする
・「へぇ、そういう意味か」と
　思えるような面白い説明を心がける

# 出力形式
単語ごとに上記4点を
番号付きで整理。

# 入力データ
単語・フレーズ：[ここに入力]
前後の文脈（あれば）：[ここに入力]
```

**✅具体的な使用方法:**  
　気になった単語を見つけたら  
　コピペするだけ。

　前後の文も一緒に貼ると、  
　より正確な解説が得られます。

**✅読者への一言:**  
　単語帳を作る必要はありません。

　記事を読みながら  
　「この単語、何？」と聞き続けるだけで、  
　語彙力は自然と積み重なります。

## ビフォー・アフター比較

**同じ英語記事でも、  
プロンプトの作り方で  
理解度は大きく変わります。**

想定シナリオ：  
「アメリカの最新経済ニュースを  
　日本語で理解したい」

---

### Before（簡易プロンプト）

```
この英語記事を翻訳してください。
```

**AIの回答（例）**：

> The Federal Reserve decided to maintain  
> interest rates at current levels.  
> （連邦準備制度は、  
> 現在の水準で金利を維持することを決定しました。）

---

👉 **どこが足りないか？**

* 直訳調で読みにくい
* 「連邦準備制度」って何かわからない
* なぜ重要なのか背景がない
* 自分に関係あるのかわからない

---

### After（本記事のプロンプトを使用）

```
# ロール設定
あなたはプロの翻訳者であり、
国際情勢に詳しい解説者です。

# 背景・目的・対象
・対象読者：英語が苦手な日本人
・目的：記事の内容を正確かつ素早く把握すること

# 入力データ
The Federal Reserve decided to
maintain interest rates at current levels,
citing persistent inflation concerns.
```

**AIの回答（例）**：

>  【3行要約】  
>  アメリカの中央銀行（FRB）が  
>  金利を現状維持すると発表。  
>  物価の高止まりを懸念しているため。  
>  利下げはまだ先になる見通し。  
>   
>  【全文翻訳】  
>  アメリカの中央銀行にあたる  
>  連邦準備制度（FRB）は、  
>  根強いインフレへの懸念を理由に、  
>  政策金利を現在の水準で  
>  据え置くことを決定しました。  
>   
>  【背景情報】  
>  ・FRBとは：日本の日銀にあたる機関。  
>  　金利を上げると物価が下がりやすくなる。  
>  ・日本への影響：  
>  　円安が続く要因のひとつ。  
>  ・投資家への影響：  
>  　株式市場に短期的な影響が出やすい。

---

👉 **何が変わったか？**

「ロール設定」で  
AIを翻訳者＋解説者として起動し、

「対象読者」と「目的」を明示したことで、

**自分に必要な情報を  
自分の言葉で届けてもらえました。**

「誰のために」「何のために」を渡すほど、  
AIの回答は使いやすくなります。

これがプロンプトの核心です。

## 1. 複数ソースを比較して事実を見極める

「この記事、本当のことを  
　書いているの？」

そう疑ったことはありませんか？

ひとつのメディアだけを読んでいると、  
そのメディアの「色」に  
染まってしまいます。

まるでテレビのチャンネルを  
1チャンネルしか観ないようなもの。

複数のチャンネルを比べて、  
初めて「本当のこと」が見えてきます。

---

### ◆多国籍ソース比較・客観分析プロンプト

**✅目的:**  
　同じニュースを複数の視点から比較し、  
　偏りのない事実認識を養う。

**✅いつ使うか:**  
　「このニュース、信じていいの？」  
　と感じたとき。  
　または政治・経済・国際情勢など  
　立場によって解釈が異なるニュース全般。

```
# ロール設定
あなたは国際メディアリテラシー
（情報を正しく読み解く力）の専門家です。
複数のニュースソースを比較分析し、
客観的な事実と各メディアの
立場・偏りを明確に整理してください。

# 背景・目的・対象
・対象読者：ニュースを批判的に
　読む習慣をつけたい日本人
・目的：
　特定のニューストピックについて、
　複数のメディアがどのように報じているかを
　客観的に把握する
・比較したいメディアの例：
　BBC（英国）・Reuters（国際）・
　AP通信（米国）・NHK（日本）など

# 指示内容
以下の5点で整理してください。

1. 各メディアが共通して報じている「事実」
2. メディアによって表現や強調点が
　異なる部分
3. 各メディアの立場・傾向の解説
4. 「最も信頼性が高い事実」として
　判断できる内容のまとめ
5. このニュースを読む際の
　注意点・見落としやすい視点

# 制約条件
・特定のメディアを
　一方的に批判しない
・「どのメディアも正確」という
　前提は置かない
・客観的な言葉で分析する

# 出力形式
5つの項目を見出しつきで整理。
表形式で比較できる部分は表にする。

# 入力データ
ニューストピック：[ここに入力]
各メディアの記事URL or 本文：
[ここに貼り付け]
```

**✅具体的な使用方法:**  
　Googleニュースで  
　同じトピックを英語検索して、  
　3〜4記事のURLをまとめて貼るだけ。

　「貿易戦争」「金融危機」「選挙結果」など  
　どのテーマにも対応できます。

**✅読者への一言:**  
　「偏っている」と気づくことが、  
　情報リテラシーの第一歩です。

　一つのメディアを信じ込むのではなく、  
　「比べる習慣」をAIと一緒に作りましょう。

## 2. 英語ニュースを毎日の習慣にする

「毎日続けるのが難しい」

それが一番の壁ですよね。

歯磨きと同じで、  
習慣になってしまえば  
苦にならなくなります。

問題は、最初の「仕組み」を  
作れていないだけです。

---

### ◆英語ニュース習慣化・朝5分ルーティンプロンプト

**✅目的:**  
　毎日5分で英語ニュースを読み解く  
　ルーティンを設計し、無理なく続ける。

**✅いつ使うか:**  
　「英語ニュースを続けたいけど  
　　続かない」と感じているとき。  
　または英語学習を習慣化したいとき。

```
# ロール設定
あなたは習慣化コーチ兼
英語学習アドバイザーです。
無理なく続けられる
「英語ニュース習慣」を設計してください。

# 背景・目的・対象
・対象：英語が苦手で
　続けることが苦手な日本人
・使える時間：
　[例：朝5分 / 通勤中10分 / 寝る前5分]
・現在の英語レベル：
　[例：中学英語レベル]
・使いたいデバイス：
　[例：スマホのみ]

# 指示内容
以下を設計してください。

1. 毎日5〜10分でできる
　「英語ニュース習慣ルーティン」の手順
2. 初心者でも読みやすい
　おすすめ英語ニュースサイト3つ
　（理由つき）
3. 続けるための
　「挫折しないコツ」を3点
4. 1ヶ月後にどんな変化が
　期待できるかの具体的な目安

# 制約条件
・「完璧にやろう」というプレッシャーを与えない
・スマホ一台で完結する方法にする
・英語の勉強感を出しすぎない

# 出力形式
ルーティン手順はステップ形式。
その他は箇条書きで整理。

# 入力データ
生活スタイル・使える時間：[ここに入力]
現在の英語への苦手意識：[ここに入力]
```

**✅具体的な使用方法:**  
　「使える時間」と  
　「現在の英語レベル」を入れるだけ。

　自分専用の習慣ルーティンが  
　5分で完成します。

**✅読者への一言:**  
　英語ニュースは、  
　「勉強」じゃなくて「情報収集」です。

　その意識で始めると、  
　不思議と続けやすくなります。

## 3. 投資・経済ニュースを即解読する

「為替が動いた」  
「利上げ決定」  
「貿易摩擦が激化」

こういう経済ニュース、  
「なんか重要らしい」と思いながら  
スルーしていませんか？

でもこれ、  
自分の資産や仕事に直結する  
情報かもしれません。

専門用語という「壁」を  
AIで取り除きましょう。

---

### ◆経済・投資ニュース解読プロンプト

**✅目的:**  
　難解な経済ニュースを  
　「自分への影響」に落とし込んで理解する。

**✅いつ使うか:**  
　Bloomberg・Financial Times・  
　WSJなどの経済記事を読んだとき。  
　または「これ、どういう意味？」と  
　感じた経済用語に出会ったとき。

```
# ロール設定
あなたは経済ジャーナリストであり、
ファイナンシャルプランナーでもあります。
難しい経済ニュースを
一般の人にわかりやすく解説し、
「自分の生活・資産への影響」まで
落とし込んでください。

# 背景・目的・対象
・対象：経済に詳しくない日本の個人投資家
　または会社員
・目的：記事の内容を理解した上で
　「自分はどう動くべきか」を考えるヒントを得る
・資産状況（任意）：
　[例：日本株と米国ETFを保有 / 預金のみ]

# 指示内容
以下の4点で解説してください。

1. 記事の要点を
　中学生でもわかる言葉で3行要約
2. 出てきた専門用語の解説
　（1つにつき2〜3行以内）
3. この出来事が
　日本の個人にどう影響するか
　（仕事・資産・物価の3軸で）
4. 「知っておくと役立つ」
　関連する経済の基礎知識を1点

# 制約条件
・投資アドバイスは行わない
　（「〜すべき」は言わない）
・あくまで「理解のサポート」にとどめる
・難しいグラフや数式は使わない

# 出力形式
4つの項目を見出しつきで
わかりやすく整理。

# 入力データ
英語記事のURL or 本文：[ここに貼り付け]
自分の状況（任意）：[ここに入力]
```

**✅具体的な使用方法:**  
　記事URLを貼るだけでもOK。

　「自分の状況」を加えると、  
　より自分ごとに落とし込んだ  
　解説が得られます。

**✅読者への一言:**  
　経済ニュースは「難しい」のではなく、  
　「専門用語が多いだけ」です。

　壁を取り除くと、  
　世界の動きがリアルに見えてきます。

## 4. 英語記事から「使える英語表現」を学ぶ

「英語を学びたいけど  
　教材を買う気にならない」

そんな人に朗報があります。

英語ニュースそのものが、  
最高の教材です。

実際のネイティブが使う表現を、  
リアルな文脈の中で学べる。

学習アプリより、  
よほど実践的です。

---

### ◆英語ニュースから表現を抽出・学習プロンプト

**✅目的:**  
　英語記事から「使える表現」を抜き出し、  
　実際の会話・ライティングに応用できる形で学ぶ。

**✅いつ使うか:**  
　英語記事を読んでいて  
　「この言い回し、かっこいいな」  
　「ビジネスでも使えそう」と感じたとき。

```
# ロール設定
あなたは実践的な英語教育の専門家です。
ニュース記事から
「ネイティブが実際に使う表現」を抽出し、
日本人がすぐに使えるよう
丁寧に解説してください。

# 背景・目的・対象
・対象：英語を「読む」だけでなく
　「話す・書く」に活かしたい日本人
・目的：
　記事を読みながら
　実用的な英語表現を自然に吸収する
・レベル：TOEIC600点程度を想定

# 指示内容
以下の形式で抽出・解説してください。

1. 記事から「使える表現」を5つ抽出
2. 各表現の意味・ニュアンスを
　日本語で解説
3. ビジネスや日常会話での
　使用例を1文ずつ
4. 似た表現・言い換え表現を1つずつ
5. 「この表現を覚えると
　どんな場面で使えるか」を一言で

# 制約条件
・フォーマルすぎる表現より
　実際に使われる自然な表現を優先
・文法的な説明は最小限
・「覚えよう」ではなく
　「使ってみたい」と思える解説にする

# 出力形式
表現ごとに番号付きで
5点セットを整理。

# 入力データ
英語記事の本文：[ここに貼り付け]
特に気になった表現（あれば）：[ここに入力]
```

**✅具体的な使用方法:**  
　記事を読んで  
　「いいな」と思った表現をメモしながら  
　このプロンプトに貼り付けるだけ。

　記事を読む→表現を学ぶ、  
　一石二鳥の習慣になります。

**✅読者への一言:**  
　英語は  
　「勉強するもの」ではなく、  
　「使うもの」です。

　記事を読みながら表現を拾うだけで、  
　語彙力は確実に積み上がっていきます。

## 5. 世界のトレンドをAIでまとめて把握する

「世界のニュース、  
　全部チェックする時間なんてない」

もちろん、その通りです。

でも「世界で何が起きているか」の  
大まかな流れは、  
把握しておきたい。

AIを使えば、  
10分で「今週の世界」を  
俯瞰できます。

---

### ◆週次・世界トレンドまとめプロンプト

**✅目的:**  
　1週間の主要な国際ニュースを  
　カテゴリ別に整理し、  
　短時間で世界の動きを把握する。

**✅いつ使うか:**  
　週末や週明けに  
　「今週の世界の動き」を  
　まとめてキャッチアップしたいとき。

```
# ロール設定
あなたは国際ニュースの
キュレーター（編集者）です。
今週の主要な国際ニュースを
日本人の視点で整理し、
わかりやすく届けてください。

# 背景・目的・対象
・対象：忙しくて毎日ニュースを
　追えない日本の社会人
・目的：
　短時間で世界の動きを把握し、
　「知っているべき情報」をおさえる
・カバーしたいジャンル：
　[例：政治・経済・テクノロジー・気候]

# 指示内容
以下の形式でまとめてください。

1. 今週の主要トピックを
　カテゴリ別に5〜7件
2. 各トピックを3行以内で要約
3. 「日本への影響・関連性」を
　各トピックに1行で添える
4. 今週のキーワード（重要用語）を
　3〜5個抽出して解説

# 制約条件
・特定の国や立場に
　偏らない中立的な視点で書く
・専門用語には必ず補足を入れる
・読むのに10分かからない
　ボリュームにする

# 出力形式
カテゴリ別の見出しつきで整理。
キーワードは最後にまとめて掲載。

# 入力データ
期間：[例：2025年4月第1週]
特に気になるジャンル：[ここに入力]
参考にする記事URL（あれば）：
[ここに貼り付け]
```

**✅具体的な使用方法:**  
　週末にこのプロンプトを実行するだけで、  
　月曜の会話・仕事・投資判断に  
　すぐ活かせる情報が手に入ります。

　参考記事がなくても、  
　AIの学習データから整理してくれます。

**✅読者への一言:**  
　「世界を知っている人」は、  
　特別な才能があるわけではありません。

　ただ、情報を仕入れる仕組みを  
　持っているだけです。

　そのひとつを、  
　今日手に入れてください。

---

## まとめ

英語ニュースを読むことは、  
「英語の勉強」ではありません。

「世界の一次情報に、  
　自分でアクセスする力」を持つことです。

AIを使えば、  
英語の壁は今日からなくなります。  
翻訳・要約・単語解説・ソース比較。  
すべてがプロンプト一つで動き出します。

情報格差は、  
努力の差ではなく「仕組みの差」です。

今日から、あなたも  
世界の情報を直接掴んでください。

参考になったら、  
ぜひ**スキ・フォロー**をいただけると  
励みになります。

次の記事でもお役に立てるよう、  
精一杯書き続けます。
