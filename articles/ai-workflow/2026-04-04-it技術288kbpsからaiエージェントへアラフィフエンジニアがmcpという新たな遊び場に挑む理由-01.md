## 🛠️ 「よきにはからう」共通規格、MCPの衝撃

**MCP（Model Context Protocol）。**

Claudeを開発するAnthropic社が提唱した、LLMが外部データと対話するための共通規格だ。

> MCP（Model Context Protocol）とは、LLMやAIエージェントが外部のツールやデータベースと対話するために開発された**共通規格（プロトコル）です**。これは、「Claude」の開発で知られるAnthropic社によって提唱されました。

[MCPとは？AIの「連携コスト」を削減する新標準。仕組みやメリットを解説 | クラウドエース株式会社](https://cloud-ace.jp/column/detail519/)

これまでは、AIに何かをさせようとするたびに、連携先ごとに異なる『お作法（API仕様）』を個別に読み解き、泥臭く繋ぎ込むプログラミングが必要だった。

**MCPは、その「連携コスト」を抜本的に解決する。**

極端に言えば、MCPが媒介となって「よきにはからって」くれる仕組みが整いつつあるのだ。

## 🎨 Geminiに「図解」を頼むという遊び心

検証の構成はこうだ。

1. MacOS上に Claude Desktop をインストール
2. ローカルファイル操作用のMCPサーバを作成
3. MCPサーバをClaude Desktopに登録
4. チャットから実際にファイルを操作させる

この全体像を整理するために、僕は**Geminiを「技術に詳しいプリンシパルエンジニア」に見立て、構成図を描くためのプロンプト作成を依頼した。**

（作成依頼のプロンプト）

```
#CONTEXT
MCPに関する技術を把握したいため、MacPCにClaudeをインストールしてファイル操作用のMCPサーバを作成したいと思っている。
#ROLE
あなたは、生成AIの技術に詳しいプリンシパルエンジニアです。
#Objective
**MCP技術の関係性を示すための絵を描くプロンプトを作成してください。**
#SCOPE
・MacPCを全体像として以下の関係を描きたい
　- Claude
  - MCP サーバー
  - MCP クライアント
　- ファイルシステム
・図の雰囲気はイラストレーションを使った2次元でお願いします。
```

自分より詳しい存在（AI）に相談し、自分自身の理解を深める。

これこそが、AI時代の「遊び方」だと思う。

結果として、MacBookという「遊び場」の中で、**Claude（クライアント）がMCPサーバーを通じてファイルシステムと対話する、美しいデータパイプラインのイメージ**が出来上がった。
