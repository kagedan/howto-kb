Claude Code 2.0.70(抜粋)
 
【機能】
・Enter キーでプロンプト候補を即座に送信
・`mcp__server__*` MCPツール権限のワイルドカード構文を追加
・プラグインマーケットプレイスの自動更新トグルを追加
・`plan_mode_required` パラメータを追加。Subagentsが実装前にPlan承認を必須にできる
・`current_usage` をStatuslineに追加。正確なコンテキストウィンドウの計算が可能
 
【修正】
・`/config`でthinkingモードを切り替えても、設定が永続化されなかった問題を修正
・メモリ使用量のパフォーマンスを3倍改善
・`#` のクイックメモリショートカットを削除

【プラグインマーケットプレイスの自動更新トグルを追加】

/plugin で追加したマーケットプレイスを表示すると、"Enable/Disable auto-update" のトグルが追加されている。

自動アップデートはClaude Code起動時に確認される模様。 https://t.co/u6vjos95Ow

【`current_usage` をStatuslineに追加】

`input_tokens`、`output_tokens`、`cache_creation_input_tokens`、`cache_read_input_tokens`が追加され、正確にコンテキスト%を計算可能に。

公式ドキュメントにサンプルもある:
https://t.co/ggcpCkGrtM https://t.co/i5Plx3V6PW

【`mcp__server__*` MCPツール権限のワイルドカード構文を追加】

`*`のワイルドカードを使って、Claude CodeのMCPサーバーのツール権限を設定できるように。

e.g.)

```json
{
    "permissions": {
        "allow": [
            "mcp__filesystem__*", 
            "mcp__context7__*"
         ],
        "deny": [
             "mcp__xxxxxxxx__*"
         ]
    }
}
```
