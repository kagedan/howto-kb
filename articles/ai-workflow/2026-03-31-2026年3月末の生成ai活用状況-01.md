## AI活用

### 会社サイト

ノーコードではなく、PayloadCMSを使うのが生成AI時代の最適解ではないか？そう思った私は、前に進みだしたというのが２月。

ウェブサイトの更新については、Claude Codeを使ってサイト更新に成功。でもデプロイにもたついて、仕事も忙しいしーーーーー、ってなっていましたが、最終的に Railway + Cloudflare というインフラ使いで落ち着きました。  
ココからどんな感じでAIにデータを回しながら、コンテンツ作成などもしていけるかというのを実験していきたいと思います。  
公開は今週末に粛々と作業をすすめられればと思っております。(無事できて次のようなサイトになりました)

### Claude Cowork

Claude Cowork が楽しい。そう思えた３月でした。なんかいろいろとこなせるやんって気になってきました。  
また、それに従いデータ取得などの必要も出てきて、Claude Code も使うという感じに。同じディレクトリで、Claude Code, Cowork が両方働く環境になりつつありますが、面白いなと思っております。  
まぁもしかしたら Claude Code だけで済むかもしれんので、今後はもう少しこの二つの使い分けを研究できたらと思います。

### 知識を付ける

３月はGeminiを使う量が減り（といってもプロダクトの開発で使っているのはGemini APIですが）、Claude 比率が高まりました。  
Claudeも新機能出まくリングな昨今なので、まとまった知識が欲しいと思っていたところ、Anthropic Academyというのがあるのを知りました。  
Claudeなどあれこれを無料で学べるコースです。

最高か！ということで、とりあえず「Claude 101」と「Introduction to Claude Cowork」を修了。  
色々良かったけど、CoworkでProjectsを使うとメモリーが効くようになるってのを知ったのが一番の収穫よねぇ。

ついでに、はんなりプログラミンを再起動し、いろいろと触ったあと、LT会をしました。ローカルLLM、IOT、スキルズとあり、盛り上がりました。よかった。
