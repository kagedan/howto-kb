同じ要件を3つのAIに渡して、そのまま送れる文面はどれ？ビジネスメールの使い分けを検証。様々なシーンで使い分けられるプロンプトも。

「取引先に納期を延ばしてもらう」「ミスをお詫びする」——こういう“言いにくいメール”ほど、書き出しで手が止まりますよね。

そこで今回は、まったく同じ要件を GPT・Gemini・Claude の3つのAIに渡し、出てきたメールを並べて比べてみました。読み終えるころには、「この場面はこのAI」という自分なりの使い分けが見えてくるはずです。

※本記事の実例は、各社で標準的に使われるモデル（GPT-5.5・Gemini 3.5 Flash・Claude Sonnet 5／2026年7月上旬に検証）で試したものです。同じプロンプトでも実行するたびに文面は変わるため、あくまで一例としてご覧ください。

## 30秒でわかる結論

先に結論からお伝えします。メール作成では、3つとも十分に実用レベルです。そのうえで、こう使い分けると迷いません。

- **要点だけ手早くまとめたい定型メール → GPT**
- **経緯や想定される質問まで丁寧に説明したいメール → Gemini**
- **簡潔にまとめつつ、結びで相手に気遣いを添えたいメール → Claude**
「効率優先のGPT」「丁寧で説明重視のGemini」「簡潔＋気遣いのClaude」。この3語だけ覚えて帰ってもらえれば十分です。以下は、その根拠を実際の文面で見ていきます。

なお、3つを見比べる時間がない場合は、後半で紹介する「メール指示に入れたい5要素」と「お願いメール用プロンプト」をそのまま使えば、手元のAIが1つだけでも近い仕上がりになります。

## 1. まず「同じ要件」を3つのAIに渡す

比べるときのコツは、全部のAIにまったく同じ材料と、同じ指示を渡すことです。天秤AIなら、1つの画面で同じ指示を3つのAIに一度に送れるので、この“書き比べ”がそのまま実現できます。

今回は、次のような箇条書きのメモを用意しました。きれいな文章にする必要はありません。伝えたいことを書き出すだけでOKです。

**渡した指示（プロンプト）**

以下の要件で、取引先への丁寧なお願いメールを作成してください。件名も付けてください。

```
以下の要件で、取引先への丁寧なお願いメールを作成してください。件名も付けてください。
・部品の入荷が遅れている
・納期を1週間延ばしてほしい（当初3/10 → 3/17）
・迷惑をかけて申し訳ない気持ちは伝えたい
・次回から余裕を持って発注する、とフォローも入れたい

```

このプロンプトを、3つのAIそれぞれに投げます。では、結果を見ていきましょう。

## 2. 3つのAIが書いたメール（全文）

**▼GPT の場合 — 要素を凝縮した、効率重視の一通**

```
件名：納期延長のお願い（3/10 → 3/17）

株式会社〇〇 〇〇様

いつもお世話になっております。株式会社△△の□□です。

現在手配しております部品につきまして、入荷に遅れが発生しており、当初予定しておりました
納期での対応が難しい状況となっております。

つきましては、誠に恐縮ではございますが、納期を下記のとおり1週間延長いただくことは
可能でしょうか。

　当初納期：3月10日
　変更希望納期：3月17日

ご迷惑をおかけすることとなり、大変申し訳ございません。今後は同様のことがないよう、
次回以降はより余裕を持って部品を発注し、納期管理を徹底してまいります。

お忙しいところ恐れ入りますが、ご確認のほど何卒よろしくお願いいたします。

株式会社△△ □□
TEL：xxx-xxxx-xxxx　Mail：xxxx@example.com

```

件名も端的で、遅延理由→依頼→日付（箇条書き）→お詫び→再発防止、という要素を無駄なく詰め込んでおり、読みやすさも十分です。
**「まず要点だけ、手早く仕上げたい」ときに最速**です。

**▼Gemini の場合 — 丁寧だが、情報量とお詫びが多め**

件名：【重要】ご注文部品の納期変更に関するお願い（株式会社△△）

株式会社〇〇
〇〇部　〇〇様

いつも大変お世話になっております。株式会社△△の□□でございます。
平素は格別のご愛顧を賜り、厚く御礼申し上げます。

さて、現在ご注文いただいております部品の納期につきまして、大変重要なお知らせと
お詫びがございます。実は、対象部品の入荷に遅れが発生しており、当初予定しておりました
