副業で年間7桁。

この数字だけ見ると  
「すごい人なんだろうな」と思うかもしれない。

でも3年前の僕は、  
副業の「ふ」の字も知らない  
ただのサラリーマンだった。

最初の月収は0円。

2ヶ月目も3,000円。

「これ本当に稼げるようになるの？」

毎晩そう思いながら  
パソコンに向かってた。

でも、あるタイミングで  
「やり方」を変えたら  
収入が一気に跳ね上がった。

3ヶ月目 → 月3万円  
6ヶ月目 → 月8万円  
9ヶ月目 → 月15万円  
12ヶ月目 → 月25万円

年間にすると7桁。

この記事では、  
0円から月25万円に至るまでに

「具体的に何をやったか」  
「何が転機になったか」  
「どんな失敗をしたか」

すべて公開する。

キラキラした成功談じゃない。

泥臭くて、地味で、  
でもリアルな記録。

これから副業を始める人、  
月1万〜3万で伸び悩んでる人に  
読んでほしい。

■ 僕のスペック（副業開始時）

・30代サラリーマン  
・文章を書く仕事の経験なし  
・特別な資格なし  
・副業の知識ゼロ  
・パソコンは持ってた（それだけ）

つまり、何の強みもない普通の会社員。

「こんな自分でも稼げた」  
というのが一番伝えたいこと。

■ 月収推移（12ヶ月分）

1ヶ月目 → 0円（準備期間）  
2ヶ月目 → 3,000円  
3ヶ月目 → 30,000円 ← ここで最初の壁を超えた  
4ヶ月目 → 45,000円  
5ヶ月目 → 60,000円  
6ヶ月目 → 80,000円 ← ジャンル特化した  
7ヶ月目 → 100,000円  
8ヶ月目 → 120,000円 ← 直接契約を開始  
9ヶ月目 → 150,000円  
10ヶ月目 → 180,000円 ← AI活用を本格化  
11ヶ月目 → 220,000円  
12ヶ月目 → 250,000円

合計：約150万円（最初の1年間）  
2年目以降、年間300万円ペースに。

この数字を見て  
「嘘だろ」と思う人もいるかもしれない。

でも、これから書く内容を読めば  
「あ、こういうことか」と納得してもらえると思う。

■ 転機は3回あった

月収が大きく跳ねたタイミングが3回ある。

・3ヶ月目：最初の「月3万の壁」を超えた  
・6ヶ月目：ジャンル特化で単価が2倍になった  
・10ヶ月目：AI活用で作業時間が半分になった

この3つの転機について  
具体的に何をしたかを書いていく。

ここから先が、この記事の核心。

テンプレートもプロンプトも  
全部載せてる。
