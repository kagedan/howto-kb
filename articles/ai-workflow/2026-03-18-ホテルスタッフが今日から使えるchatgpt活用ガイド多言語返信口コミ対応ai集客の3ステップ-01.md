フロントの外国語対応で困っていませんか？ChatGPTを使えば、英語・中国語・韓国語のメール返信が30秒で完成します。設定不要・無料・スマホでも使えます。今日から試せる具体的なステップを紹介します。

## ステップ別・今日からできる実践ガイド

### ステップ1：ChatGPTアカウントを作る（5分・無料）

まだアカウントがない方は、chat.openai.com にアクセスしてメールアドレスで登録するだけ。スマホのブラウザからでもそのまま使えます。無料版でも今日紹介するプロンプトは全て動作します。

有料プラン（月額約3,000円）にすると回数制限がなくなり、より長文の文書も扱えるようになりますが、まずは無料版で十分です。

### ステップ2：「プロンプト」を貼り付けて使う

ChatGPTは「指示文（プロンプト）」を入力するだけでOK。難しい設定は一切不要です。以下のプロンプト集をそのままコピーして使ってください。

### ステップ3：返信文を確認してからゲストに送る

AIが生成した文章は、必ず一度読んでから送信してください。基本的な内容は正確ですが、施設固有の情報（チェックイン時刻・料金など）の最終確認は人間が行います。

## そのまま使えるプロンプト集（観光業務向け）

**プロンプト①：英語・中国語・韓国語の同時翻訳**

```
以下の日本語の返信文を、英語・繁体字中国語・韓国語の3言語で
ホテルらしい丁寧なトーンで翻訳してください。

【返信文をここに貼り付ける】
```

**プロンプト②：英語口コミへの返信文を作る**

```
あなたは日本のホテルのカスタマーサービス担当です。
以下の英語口コミに対して、感謝・改善への言及・
再訪への誘導を含む丁寧な英語返信を80〜100語で書いてください。

【口コミをここに貼り付ける】
```

**プロンプト③：自施設のAI紹介文を作る**

```
あなたは旅行情報を提供するAIです。
以下の宿泊施設情報をもとに、外国人旅行者向けの
魅力的な英語紹介文を150語程度で書いてください。
「温泉」「食事」「アクセス」の3点を必ず含めてください。

【施設名・特徴・所在地・アクセスをここに書く】
```

**プロンプト④：日本語の館内案内を多言語化する**

```
以下の日本語の館内案内文を英語・タイ語・ベトナム語に翻訳してください。
わかりやすくシンプルな表現でお願いします。

【案内文をここに貼り付ける】
```

## よくある失敗と解決策

**失敗①：「なんか不自然な日本語が混じってる」** → 解決策：プロンプトの冒頭に「出力は英語のみにしてください」と言語を指定する。翻訳ミスが減ります。

**失敗②：「施設名や固有名詞が英語で変に表記された」** → 解決策：プロンプトに「施設名は『〇〇』のまま使用してください」と補足する。AIは固有名詞を翻訳しようとする場合がある。

**失敗③：「毎回プロンプトを入力するのが面倒」** → 解決策：よく使うプロンプトをメモ帳やスマホのメモアプリに保存しておく。ChatGPTには「カスタム指示」機能もあり、毎回「あなたはホテルスタッフです」と書かなくて済む設定ができます。
