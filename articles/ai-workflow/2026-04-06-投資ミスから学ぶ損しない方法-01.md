「なんであの時、買ってしまったんだろう」

チャートを眺めながら、  
そう後悔したことはありませんか？

損切りできずに塩漬けになった株。  
SNSに流されて飛びついた仮想通貨。  
「絶対上がる」と信じて握り続けたあの銘柄。

投資の失敗には、  
不思議なほど「共通点」があります。

それは才能でも、運でもありません。

**行動パターンの問題です。**

そして行動パターンは、  
あらかじめ知っておけば、  
回避することができます。

---

この記事を読むと、

✅ **致命的ミスを事前に回避できる**  
　→過去の失敗パターンを知るだけで  
　　同じ罠に落ちなくなる

✅ **感情ではなく冷静な判断ができる**  
　→AIが思考の歪みを指摘してくれるから  
　　熱くなった頭を冷やせる

✅ **機会損失をなくし資産を守れる**  
　→「やらかし」の構造がわかると  
　　本当に必要な行動だけに集中できる

---

AIは「過去の膨大な失敗事例」を  
知っています。

あなたの投資行動が  
そのパターンに当てはまっていないか、  
一緒にチェックしてもらいましょう。

「次は失敗しない」と思いながら  
また失敗してしまう、その循環を、  
今日から断ち切ることができます。

## 基本のプロンプト

まず、例え話からはじめます。

「AIに投資の失敗を分析させる」とは、  
いったいどういうことでしょうか。

---

**「ゲームのリプレイ機能」に例えてみます。**

ゲームで負けたとき、  
強いプレイヤーは  
「どこで失敗したか」を  
リプレイで確認します。

「あのとき前に出すぎた」  
「ここで回復を使えばよかった」

それを次のゲームに活かすから、  
どんどん強くなれる。

投資の失敗を  
AIに分析させるのも、同じです。

「どの判断がミスだったか」  
「何がそうさせたのか」

それを言語化するだけで、  
同じ負けパターンを繰り返さなくなります。

一人で振り返ると感情が邪魔をします。  
でもAIは、感情なしに事実だけを整理してくれます。

### ◆投資失敗パターン診断プロンプト

**✅目的：**  
　自分の投資行動が、  
　典型的な失敗パターンに  
　当てはまっていないか診断する。

**✅いつ使うか：**  
　損失が出たとき、  
　または「なんとなく不安」な  
　ポジションを持っているとき。

```
# ロール設定
あなたは行動ファイナンス（投資家の心理的バイアスを研究する学問）
の専門家です。
投資家が陥りがちな失敗パターンを熟知しており、
依頼者の投資行動を客観的に分析し、
改善点を具体的に伝えることができます。

# 背景・目的・対象
・対象者：個人投資家（初心者〜中級者）
・目的：過去の投資行動を振り返り、
　失敗の根本原因を特定する
・分析対象：以下の入力データに記載の
　取引・行動

# 指示内容
以下の4点を分析してください。

1. この取引で見られる典型的な失敗パターン
　（例：損切りできない、利確が早すぎる、
　FOMO（乗り遅れ恐怖）による衝動買いなど）

2. その行動を引き起こした
　心理的バイアス（思考の歪み）の名前と説明

3. 同じ失敗を繰り返さないための
　具体的なルール（3点）

4. 次回の投資前に自問すべき
　チェックリスト（5項目）

# 制約条件
・責めるのではなく、改善に向けた
　前向きな分析をする
・専門用語には必ず一言補足をつける
・「気をつけよう」ではなく
　「こうすれば防げる」という
　行動レベルの提案にする

# 出力形式
見出しつきの箇条書きで
わかりやすく整理してください。

# 入力データ
・いつ：[例：2024年11月]
・何を：[例：○○株を1株3,000円で100株購入]
・なぜ買ったか：[例：SNSで話題だったから]
・結果：[例：2,000円まで下落、損切りできず保有中]
・そのとき何を考えていたか：
　[例：「いつか戻る」と思って売れなかった]
```

**✅具体的な使用方法：**  
　「入力データ」の部分に  
　自分の取引をそのまま書くだけ。

　細かく書けば書くほど、  
　AIの分析精度が上がります。

　「なぜ買ったか」「そのとき何を考えていたか」  
　この2項目が特に重要です。

**✅読者への一言：**  
　失敗を振り返るのは、つらいことです。

　でも「なぜそうしたか」を  
　言語化するだけで、  
　次の行動は確実に変わります。

　AIが感情なしに付き合ってくれます。

### ◆投資ルール設計プロンプト

**✅目的：**  
　自分の投資スタイルに合った  
　「感情に振り回されないルール」を  
　AIと一緒に作る。

**✅いつ使うか：**  
　これから投資を始めるとき、  
　または「ルールを守れていない」と  
　感じているとき。

```
# ロール設定
あなたは個人投資家向けの
ポートフォリオ・コーチです。
投資家が感情的な判断ではなく、
あらかじめ決めたルールに従って
行動できるよう支援することが専門です。

# 背景・目的・対象
・投資経験：[例：投資歴2年、株式メイン]
・主な失敗パターン（わかる範囲で）：
　[例：損切りが遅い、急騰銘柄に飛びつく]
・現在の投資スタイル：
　[例：長期保有希望だが
　値動きが気になってすぐ売ってしまう]
・月に使える投資資金：[例：3万円]

# 指示内容
以下を設計してください。

1. 買う前に必ず確認する
　「買いルール」（5項目）

2. 売る判断を迷わないための
　「売りルール」（損切り・利確それぞれ）

3. 感情的になったときに
　「一晩置く」ための自問リスト（3項目）

4. 週1回の振り返りに使える
　セルフチェックシート（簡易版）

# 制約条件
・「完璧なルール」ではなく
　「続けられるシンプルなルール」にする
・初心者でも理解できる言葉で書く
・数字・具体的な基準を入れる
　（例：「下がったら売る」ではなく
　「-10%で必ず売る」）

# 出力形式
各ルールを箇条書きで整理し、
すぐ使えるシートの形で出力。

# 入力データ
自分の投資スタイル・失敗パターンの詳細：
[ここに入力]
```

**✅具体的な使用方法：**  
　「主な失敗パターン」には  
　正直に書いてください。

　「損切りが遅い」  
　「利確が早すぎる」  
　「SNSに流される」

　何でも大丈夫です。  
　そこからルールが生まれます。

**✅読者への一言：**  
　ルールとは「未来の自分を守るための約束」です。

　感情が高ぶったとき、  
　あなたを守ってくれるのは  
　冷静なときに作ったルールだけです。

**同じ状況でも、プロンプトの作り方で  
結果は大きく変わります。**

想定シナリオ：  
「急騰した銘柄を衝動買いして損をした。  
なぜ失敗したか知りたい。」

```
投資で失敗しました。
なぜ失敗するのか教えてください。
```

**AIの回答（例）**：

> 感情的な判断を避けましょう。  
> リスク管理が大切です。  
> 分散投資を心がけてください。

---

👉 **どこが足りないか？**

---

### After（本記事のプロンプトを使用）

```
# ロール設定
あなたは行動ファイナンスの専門家です。

# 入力データ
・いつ：2024年10月
・何を：話題のAI銘柄を高値でつかみ
・なぜ買ったか：
　XのポストでバズっていたのでFOMOで購入
・結果：翌週に20%下落、
　「戻るはず」と思い保有継続中
・そのとき何を考えていたか：
　「自分だけ乗り遅れたくなかった」
```

**AIの回答（例）**：

>  【失敗パターン】  
>  FOMO（乗り遅れ恐怖）による  
>  衝動的な高値買い  
>   
>  【働いていたバイアス】  
>  「ハーディング効果」：  
>  周囲の行動に引っ張られて  
>  自分の判断を失う心理  
>   
>  「現状維持バイアス」：  
>  損失を確定したくないため  
>  保有し続けてしまう心理  
>   
>  【次回への具体ルール】  
>  ・SNSで話題の銘柄は  
> 　48時間ルールを設ける（即買いしない）  
>  ・-10%の時点で自動的に損切りを実行  
>  ・「なぜ買うか」を3行で書けなければ買わない

## 実践プロンプト集

## 1. 典型的な失敗パターンを全部知る

「知らないから罠にはまる」

投資の失敗には、  
実は決まった「型」があります。

まずは「どんな失敗パターンがあるか」を  
まるごと把握してしまいましょう。

知っているだけで、  
罠を避けられる確率が格段に上がります。

---

### ◆投資失敗パターン全網羅プロンプト

**✅目的：**  
　投資家が陥りがちな失敗パターンを  
　一覧で把握し、  
　自分に当てはまるものをチェックする。

**✅いつ使うか：**  
　投資を始める前・再開する前、  
　または「なんとなく不安」なとき。

```
# ロール設定
あなたは投資教育の専門家です。
個人投資家が過去に繰り返してきた
典型的な失敗パターンを熟知しており、
初心者にもわかりやすく伝えることが得意です。

# 背景・目的・対象
・対象：個人投資家（初心者〜中級者）
・目的：投資の典型的な失敗パターンを
　体系的に理解し、自分の行動を点検する
・投資対象：株式・投資信託・仮想通貨など
　（幅広く対応）

# 指示内容
以下を整理してください。

1. 投資初心者が特に陥りやすい
　失敗パターン（TOP5）と
　それぞれの「やってしまいがちな行動」

2. 中級者になっても繰り返しやすい
　失敗パターン（TOP3）

3. 各パターンに対して
　「一言チェック質問」を作る
　（自分が当てはまるか確認できる問い）

4. 「これだけは守れ」という
　投資の鉄則（3つ）

# 制約条件
・専門用語には必ず補足をつける
・「知識」ではなく「行動」レベルの
　説明にする
・怖がらせるのではなく
　「知ることで守れる」トーンで

# 出力形式
パターン名＋行動例＋チェック質問の
セットで見出しつき箇条書きで整理。

# 入力データ
自分の投資スタイル（任意）：
[例：個別株メイン、投資歴1年、
損切りが苦手]
```

**✅具体的な使用方法：**  
　「入力データ」は空白でも動きます。

　でも「自分の投資スタイル」を  
　一言入れると、  
　自分に刺さるパターンを  
　優先的に出してくれます。

**✅読者への一言：**  
　失敗パターンを「知識」として  
　持っているだけで、  
　頭の中に「警告灯」がつきます。

　まずは「こんな罠があるんだ」と  
　全体像を知るところから始めましょう。

## 2. 自分の失敗トレードを深く分析する

「あのとき、なぜ売れなかったのか」

損失が出た後に  
一人で振り返ると、  
どうしても感情が邪魔をします。

「あのニュースが悪かった」  
「タイミングが悪かっただけだ」

でも本当の原因は、  
もっと深いところにあることが多いです。

---

### ◆失敗トレード深掘り分析プロンプト

**✅目的：**  
　損失が出たトレードの  
　根本原因をAIと一緒に特定し、  
　二度と同じ失敗をしないための  
　仕組みを作る。

**✅いつ使うか：**  
　損失が確定した直後、  
　または「なんで売らなかったんだろう」  
　と後悔しているとき。

```
# ロール設定
あなたはトレード心理学と
行動ファイナンスの専門家です。
投資家が感情的に判断を誤るパターンを
深く理解しており、
具体的なトレードの振り返りを通じて
根本原因を特定し、
再発防止の仕組みづくりを支援します。

# 背景・目的・対象
・分析目的：失敗トレードの根本原因特定
・対象：以下の入力データに記載のトレード

# 指示内容
以下の5ステップで分析してください。

【STEP1】事実の整理
　取引の経緯を時系列で整理する

【STEP2】判断ミスの特定
　「どの瞬間に判断を誤ったか」を
　具体的に指摘する

【STEP3】バイアスの特定
　そのミスを引き起こした
　心理的バイアスの名前と説明

【STEP4】根本原因
　「なぜそのバイアスが働いたのか」
　もう一段深い原因を掘り下げる

【STEP5】再発防止ルール
　このトレードから学べる
　具体的なルールを3つ作る

# 制約条件
・「運が悪かった」という結論にしない
・感情・行動・判断の3つの視点から分析する
・ルールは「〇〇したら△△する」という
　行動レベルで具体的に書く

# 出力形式
STEP1〜5の見出しつきで
順番に整理。

# 入力データ
・取引した銘柄・商品：[例：○○株]
・買った理由：[ここに入力]
・売った（または売れなかった）理由：
　[ここに入力]
・損益結果：[例：-50,000円]
・そのとき感じていたこと：
　[ここに入力]
・今思うと何が間違いだったか：
　[ここに入力]
```

**✅具体的な使用方法：**  
　「そのとき感じていたこと」に  
　正直に書くのが最大のコツです。

　「焦っていた」  
　「絶対上がると思っていた」  
　「損を認めたくなかった」

　感情を書けば書くほど、  
　AIの分析が核心に迫ります。

**✅読者への一言：**  
　失敗を分析するのは、  
　自分を責めるためではありません。

　「なぜそうなったか」を知ることで、  
　同じ罠から抜け出せます。

　AIは責めません。  
　ただ、一緒に考えてくれます。

## 3. バイアスチェックで冷静さを取り戻す

「今の判断、本当に正しいのか？」

投資で怖いのは、  
「自分が間違っている」と  
気づけないことです。

思い込みや感情が  
判断を歪めていても、  
当の本人には見えません。

これを「認知バイアス」といいます。

---

### ◆投資判断バイアスチェックプロンプト

**✅目的：**  
　今まさに下そうとしている投資判断が、  
　心理的バイアスに歪められていないか  
　AIに客観的にチェックしてもらう。

**✅いつ使うか：**  
　「これ、買うべきか迷っている」  
　「今すぐ売るべきか」と  
　判断に迷っているまさにそのとき。

```
# ロール設定
あなたは認知バイアスの専門家であり、
投資判断の客観的なレビュアーです。
感情や思い込みに気づいていない投資家に対し、
「今の判断は本当に合理的か」を
中立的にチェックする役割を担います。

# 背景・目的・対象
・現在の状況：[例：○○株を3ヶ月保有中、
　現在-15%だが保有継続を検討している]
・今しようとしている判断：
　[例：「いつか戻る」と思って売らないでいる]
・その根拠：
　[例：企業の業績は悪くないと思う。
　ただ詳しく調べてはいない]

# 指示内容
以下の4点をチェックしてください。

1. この判断に疑われるバイアスを3つ挙げ、
　それぞれ「どう歪んでいるか」を説明する

2. 逆の立場（売るべきという立場）から
　見た場合の論拠を3つ挙げる

3. 「感情を排除したとき」に
　この判断はどう見えるか、
　客観的に評価する

4. 判断を下す前に確認すべき
　「3つの問い」を提示する

# 制約条件
・「売れ」「買え」などの指示はしない
　（判断するのは本人）
・バイアス名には必ず補足を入れる
・中立的な立場で分析する

# 出力形式
1〜4の見出しつき箇条書き。

# 入力データ
現在の状況・判断の詳細：
[ここに入力]
```

**✅具体的な使用方法：**  
　「今しようとしている判断」と  
　「その根拠」を正直に書きます。

　「根拠が薄い」と感じながら  
　書いていたら、それ自体が  
　バイアスのサインかもしれません。

**✅読者への一言：**  
　投資の判断を下す前に、  
　「この考え、本当に自分のものか？」  
　と立ち止まる習慣が資産を守ります。

　AIが「もう一人の冷静な自分」に  
　なってくれます。

## 4. リスクを数値で把握する

「なんとなく怖い」では、  
投資は上達しません。

「何がどれだけリスクなのか」を  
数値と言葉で把握することで、  
初めて冷静に向き合えます。

リスクを「感覚」から「知識」に変えましょう。

---

### ◆投資リスク可視化プロンプト

**✅目的：**  
　今持っているポジション（保有銘柄）の  
　リスクを言語化・数値化し、  
　許容できる範囲かどうかを判断する。

**✅いつ使うか：**  
　新たに投資をする前、  
　または今の保有状況を  
　整理したいとき。

```
# ロール設定
あなたはリスク管理の専門家です。
個人投資家が感情ではなく

## 5. 「次こそ失敗しない」仕組みを設計する

投資で一番難しいのは、  
「わかっているのにできない」問題です。

「損切りが大事なのはわかっている」  
「感情で動いてはいけないとわかっている」

それでも、できない。

それは「意志の弱さ」ではありません。

**仕組みがないからです。**

---

### ◆投資失敗防止システム設計プロンプト

**✅目的：**  
　「わかっているのにできない」を解決する  
　具体的な仕組み・ルール・チェックリストを  
　自分専用に設計する。

**✅いつ使うか：**  
　「また同じ失敗をした…」と気づいたとき。  
　投資方針を一から見直したいとき。

```
# ロール設定
あなたは行動変容（人の行動を仕組みで変える）
の専門家であり、
個人投資家の行動改善を支援するコーチです。
「わかっているのにできない」を
仕組みで解決することを専門としています。

# 背景・目的・対象
・繰り返している失敗パターン：
　[例：損切りができない、

・投資スタイル：
　[例：個別株メイン、月3万投資]

　[例：「-10%で売る」と決めたが

1. 「買う前の儀式」

## まとめ

投資の失敗には  
「必然の型」があります。

その型を知り、自分の行動を点検し、  
感情ではなく仕組みで  
動けるようになること。

AIはその「気づきのパートナー」として、  
あなたの資産を守る  
最強の防波堤になってくれます。

「知っていれば防げた」を、  
今日から  
「知っているから防げる」に  
変えていきましょう。

ぜひ**スキ・フォロー**をいただけると  
とても励みになります。
