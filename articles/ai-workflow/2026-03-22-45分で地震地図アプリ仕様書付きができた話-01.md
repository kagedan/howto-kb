### なんで？

AIでほぼ全自動ソフト開発する(日英両対応の)フレームワーク [gr-sw-maker](https://github.com/GoodRelax/gr-sw-maker) を作ってて、誰かの役に立つWebアプリを**サクッと**作れるか試したかった。

---

### なにを？

オープンデータ（USGS地震API）を読み込んで世界地震地図を描く単一 `.html`ファイルアプリ

* ズーム可能な世界地図
* 時間範囲の指定（1時間〜30日・カスタム）
* 最小マグニチュードのフィルタ
* サーバーなし、ブラウザだけで動く

---

### どうした？

詳細 → [README-ja](https://github.com/GoodRelax/gr-sw-maker/blob/main/README-ja.md)

```
> npm init gr-sw-maker earthquake-map  
> cd earthquake-map  
> node setup.js  
> 1 : # Claude Code を使って英語で開発
```

**※ node setup.js で 2 を選択すれば日本語で開発可能**

#### 2. gr-sw-maker に**5行のメモ(要求仕様書)を書く**

| 日本語 | English |
| --- | --- |
| 地震地図を作りたい | Earthquake Map |
| 世界のどこでいつ地震が起きたか見たい | I want to see where and when earthquakes happened in the world. |
| ブラウザだけで動く。サーバーなし。 | Runs in the browser only. No server. |
| どこでもズームしたい | I want to zoom into any area. |
| 期間を指定したい | I want to specify a time span. |

#### 3. あとはAIのインタビューに答えて「全部お任せで」

#### 4 そのあと、**仕様変更を2回** （Updateボタンの追加、file://直開き対応）

#### 所要時間の詳細
