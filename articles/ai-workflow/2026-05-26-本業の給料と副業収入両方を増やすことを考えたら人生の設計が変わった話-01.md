副業を始める前、お金の話をこう考えていた。

「給料を上げるためには、会社で頑張るしかない。」

それしか選択肢がないと思っていた。

副業を始めてから、その考え方が変わった。

**「本業の給料」と「副業収入」、両方を育てることができる。**

その発想が生まれてから、お金への向き合い方が全然変わった。

今回はその話を書く。

副業を始める前の、お金への考え方

副業を始める前、こんな感覚があった。

給料は会社が決める。自分にはコントロールできない。

昇給を待つか、転職して給料を上げるか。それしか選択肢がないと思っていた。

「副業」という選択肢が、頭の中になかった。

でも実際には、第三の選択肢があった。

**本業を続けながら、副業で収入の柱をもう一本作る。**

これが、一番リスクが低くて、一番現実的な方法だった。

「本業か副業か」じゃなくて「本業も副業も」

副業を始めると「本業に集中できなくなるんじゃないか」と言われることがある。

私も最初はそう思っていた。

でも実際は逆だった。

副業を始めてから、本業への向き合い方が変わった。

**「会社しかない」という感覚が薄れたことで、本業でも余裕が生まれた。**

以前は「この会社を失ったら終わり」という感覚があって、それが見えないプレッシャーになっていた。

副業収入が少しずつ積み上がってから、そのプレッシャーが軽くなった。

余裕が生まれると、本業でも「自分がやりたいこと」を言えるようになった。

怒られることへの恐怖より、「どうすれば仕事がうまくいくか」を考えられるようになった。

**「副業のせいで本業が疎かになる」じゃなくて、「副業のおかげで本業も変わった」という実感がある。**

両方を育てるための、時間と頭の使い方

本業も副業も、どちらも中途半端になるんじゃないかという不安がある。

正直に言うと、最初はそうなりかけた。

副業に時間を使いすぎて、本業のパフォーマンスが落ちそうになった時期がある。

そこで決めたルールがある。

**「本業のパフォーマンスが落ちたら、副業のペースを縮小する」**

このルールを最初に決めておくだけで、両立がしやすくなった。

本業が繁忙期に入ったら、副業は縮小する。本業が落ち着いたら、副業のペースを戻す。

**「両方を同じ熱量でやり続ける」じゃなくて、「本業を軸にしながら副業を柔軟に調整する」という感覚だ。**

具体的な時間の使い方はこうだ。

```
平日朝：30分（副業の仕込み）
昼休み：15分（アイデア出し・メモ）
平日夜：副業はほぼやらない（本業の疲れを取る）
週末午前：2時間（記事を仕上げる）
週末午後〜日曜：完全に休む
```

この枠を守ることで、本業も副業も、どちらも続けられるようになった。

本業の給料を上げるために、副業が役に立った

これは意外な副産物だった。

副業でライティングを続けていたら、文章で伝える力がついた。

その力が、本業でも使えるようになった。

上司への報告書が変わった。メールの文章が変わった。会議での発言が変わった。

「最近、伝え方がうまくなったね」と言われるようになった。

**副業で鍛えたスキルが、本業の評価につながった。**

これは始める前には想像していなかった展開だった。

副業と本業は、切り離されたものじゃない。

どちらかで磨いたスキルが、もう一方でも活きる。

お金の柱を2本持つことの、本当の意味

本業の給料と副業収入、両方を持つことの意味は、「収入が増える」だけじゃない。

**「選択肢が増える」ということだ。**

本業の給料だけで生きていると、「この会社を続けるしかない」という感覚になりやすい。

理不尽なことがあっても、生活のために我慢する。

でも副業収入が月3万円でも安定してくると、感覚が変わる。

「ここを辞めても、副業を広げれば生活できるかもしれない」という選択肢が生まれる。

**その選択肢を「実際に使うかどうか」は関係ない。**

「使えるかもしれない」という選択肢があるだけで、今の仕事への向き合い方が変わる。

追い詰められた感覚が減る。余裕が生まれる。余裕があると、判断が変わる。

これが、お金の柱を2本持つ一番の意味だと思っている。

両方を育てていくための、長期的な考え方

本業の給料は、会社の評価で決まる部分が大きい。

すぐには変えられない。

でも副業収入は、自分の行動次第で変えられる。

今月頑張れば、来月の収入が変わる。

**「コントロールできる収入」を持つことが、人生の設計を変える。**

私が考えている長期的な方向性はこうだ。

```
今：本業を軸にしながら、副業で月3万円を安定させる
半年後：副業収入を月5万円に育てる
1年後：副業収入を月10万円に近づける
将来：本業と副業、どちらを選んでもいい状態を作る
```

「脱サラしたい」という気持ちが強いわけじゃない。

でも「選べる状態を作る」という目標が、副業を続けるモチベーションになっている。

**「会社に選ばれる存在」から「自分で選べる存在」へ。**

その方向に、少しずつ進んでいる。

最後に

本業か副業か、どちらかを選ぶ必要はない。

両方を育てることができる。

どちらかが崩れても、もう一方がある。

そういう状態を作ることが、今の時代の一番のリスクヘッジだと思っている。

副業を始めたのは、お金のためだった。

でも続けていくうちに、それ以上のものを得た。

選択肢が増えた。余裕が生まれた。本業も変わった。

**副業は「収入を増やすもの」じゃなくて「人生の設計を変えるもの」だった。**

始めてよかったと、今でも思っている。
