# Claudeで地方中小企業のDXを支援する

福岡県筑豊地域の中小企業向けに実際に使っているClaudeプロンプトを公開します。

---

## 1. 業務メール自動生成

```
あなたは日本の中小企業の経営者秘書です。
以下の条件でビジネスメールを作成してください。

送信者：[役職・社名]
宛先：[相手の役職・社名]
目的：[何を伝えたいか]
トーン：丁寧な敬語（ですます調）
長さ：200字以内の本文

条件：
- 冒頭に時候の挨拶を入れる
- 用件を明確に1段落で伝える
- 締めの挨拶を入れる
```

---

## 2. 会議議事録の構造化

```
以下の会議メモを整理してください。

【会議メモ】
[ここに生のメモを貼り付け]

出力形式：
## 決定事項
- 箇条書き

## アクションアイテム
| 担当者 | タスク | 期限 |

## 次回議題
- 箇条書き

日本語で出力してください。
```

---

## 3. 補助金申請書の下書き

```
以下の情報をもとに、[補助金名]の申請書の
「事業概要」欄の下書きを作成してください。

事業者情報：
- 業種：[業種]
- 従業員数：[人数]
- 所在地：福岡県[市町村名]
- 課題：[解決したい課題]
- 導入したいもの：[AIツール/DXツール名]
- 期待効果：[具体的な数字があれば]

字数：400〜600字
トーン：公的機関向けの丁寧な文体
```

---

## 4. LinkedIn投稿の生成（日本語）

```
あなたは日本のLinkedIn専門家です。
以下の情報をもとにLinkedIn投稿を作成してください。

投稿者：[役職] / [地域]在住
テーマ：[伝えたいこと]
ターゲット：福岡県・筑豊地域のビジネスパーソン

要件：
- 150〜200字
- 最初の1行で読者を引きつける
- 具体的な数字や事例を含める
- ハッシュタグ3〜4個（#筑豊 #福岡 を必ず含める）
- ですます調
```

---

## まとめ

これらのプロンプトはすべて無料のClaude（claude.ai）で使えます。

地方の中小企業こそ、AIで都市部との差を埋められます。
