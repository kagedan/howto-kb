## CoPawとは？

最近、OpenClawの人気の高まりにより、他社も類似プロジェクトを次々と立ち上げており、Copawもその一つです。  
[CoPaw](https://github.com/agentscope-ai/CoPaw) は **Co Personal Agent Workstation** の略で、あなた専用のAIアシスタントです。ローカルマシンやクラウドにデプロイでき、DingTalk・Feishu・QQ・Discord・iMessageなど複数のチャネルに対応しています。

## 主な特徴

* **あらゆるチャネルに対応** — DingTalk、Feishu、QQ、Discord、iMessageなどに1つのアシスタントで接続
* **あなたの管理下** — メモリとパーソナライズはローカルまたはクラウドで自己管理
* **マルチエージェント** — 複数の独立したエージェントを作成し、エージェント間で通信・協力が可能
* **スキル拡張** — 組み込みのcron機能、カスタムスキルの自動読み込み
* **ローカルLLM対応** — llama.cpp、MLX（Apple Silicon）、OllamaでAPIキー不要

## 3コマンドで起動

```
pip install copaw
copaw init --defaults
copaw app
```

ブラウザで `http://127.0.0.1:8088/` を開くとコンソールが利用できます。

## インストール方法

### ワンラインインストール（Python不要）

**macOS / Linux:**

```
curl -fsSL https://copaw.agentscope.io/install.sh | bash
```

**Windows (PowerShell):**

```
irm https://copaw.agentscope.io/install.ps1 | iex
```

### Docker

```
docker pull agentscope/copaw:latest
docker run -p 127.0.0.1:8088:8088 \
  -v copaw-data:/app/working \
  -v copaw-secrets:/app/working.secret \
  agentscope/copaw:latest
```

### デスクトップアプリ（Beta）

[GitHub Releases](https://github.com/agentscope-ai/CoPaw/releases) からダウンロード可能。Windows・macOS対応。

## ローカルモデル対応

| バックエンド | 最適な用途 | インストール |
| --- | --- | --- |
| llama.cpp | クロスプラットフォーム | `pip install 'copaw[llamacpp]'` |
| MLX | Apple Silicon (M1〜M4) | `pip install 'copaw[mlx]'` |
| Ollama | クロスプラットフォーム | `pip install 'copaw[ollama]'` |
