###### IBM Bob
- 概要
  - 2026年3月に正式リリースされたAIエージェント駆動のエンタープライズ向け開発支援ツール
&nbsp;
- 特徴
  - 学習にIBMの資産を使用
    - RPG(IBM独自の言語)やIBM i(IBM独自のOS)
  - 設計～展開(上流工程から下流工程)までの一連の作業を支援
  - 内部にClaudeを搭載
    - Anthropic社との戦略的パートナーシップを発表: https://www.ibm.com/jp-ja/new/announcements/ibm-project-bob
  - VS Code（Visual Studio Code オープンソース）ベースのフォーク
    - VS Code + AI機能 = 専用統合開発環境
&nbsp;
- 無料トライアル(2026年4月現在)
  - 利用期間1か月 + 40Bobコインの付与
  - https://bob.ibm.com/trial
