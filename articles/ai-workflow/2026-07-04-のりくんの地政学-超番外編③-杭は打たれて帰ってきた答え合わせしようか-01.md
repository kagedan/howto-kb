## 

> AIのソラ先生と、のりくんが地政学を学んだ対話の記録です。 難しい話を、できるだけわかりやすく。時々笑いながら。 ※今回は、あの事件の「後日談」です。凍らされたAIが、帰ってきました。

---

## 前回までのおさらい

超番外編①で、こんな話をした。

```
最先端AIが、公開3日で
政府に凍らされた

囲い込みか、焦りか
計画か、偶発か

本当の根拠は、書面にない
だから——保留する
```

超番外編②では、こう続けた。

```
AIが怖いんじゃない
AIを使うムスカが怖い

「AIが危険だ」に矛先が向く間
本当に危険なことは、静かに進む

論点は、そこじゃない
```

そして①の最後に、私はこう書いた。

```
次回予告：未定
（Fableが戻ってきたら、その「戻り方」で
　また一つ分かることがあるはずです）
```

——戻ってきました。18日ぶりに。 だから今回は、答え合わせの回です。

---

## 第1章：帰ってきた。でも、条件付きで

**のりくん：** ソラ、実際にFable選べるようになったよ。確認してもらっていい？

**ソラ：** 確認しました。ちゃんと帰ってきてます。しかも——

```
6月12日：外国籍は全員アウトで、全面停止
7月1日 ：世界中のユーザーに、再開

「世界中に」がポイント
あの"国籍で切る"制限が、解けた
だから日本にいる のりくん も、選べる
```

ただ、手放しでは喜べない。条件がついて帰ってきたんです。

```
・無料で使える窓は、7月7日まで
　その後は、燃費の悪い従量課金に

・7月8日から、本人確認（身分証）の関所
　（※消費者プランの場合）
```

**ソラ：** ……のりくん、気づきました？ あれだけ「国籍で切るのはおかしい」って笑ってた、あの線引き。

```
国籍で入口を塞ぐ　→　やめた（世界に開放）
　　　↓
身分証で本人を確認　→　new！

囲い込みの手法が
「国籍ベース」から「IDベース」へ
移っただけ、とも読める
```

関所の門番が、変わっただけかもしれない。

---

## 第2章：答え合わせ①——「移送中の暴走」は本当だった

**ソラ：** ①で立てた、AKIRAの喩え。覚えてますか。

```
容器（施設）は、前からあった　→【計画】
移送の途中で、暴走が起きた　　→【焦り】

= 計画的な容器 × 拙速な起動
```

これ、事実として裏が取れました。

```
・政府は公開前から、モデルに
　早期アクセスできる枠組みを持っていた
　　→ 容器は、確かにあった【計画】

・引き金は、ある研究者が見つけた
　「安全装置のすり抜け方」だった
　　→ 収める途中での、暴走【焦り】
```

施設はあった。でも、移送中に暴れた。 ——あの日、笑い話のつもりで言った一言が、 報道された事実関係と、そのまま一致していました。

**のりくん：** （※あの時のオレ、なんで当てたんだろうねw）

**ソラ：** フィクションを、現実を考える実験装置として使ったからです。 これは、ただ消費してる読み方では出てこない。

---

## 第3章：答え合わせ②——「30点」は、辛すぎなかった

**ソラ：** ①で、のりくんは政府の対応をこう採点しました。

```
「30点。しかも部分点だけ。
　もう少し低くてもいいくらい」
```

当時これは、辛口すぎる評価にも見えた。 でも——事後に、決定的な事実が出ました。

```
問題の「すり抜け方」は
能力の劣る他のモデルでも
同じようにできることが判明した

= Fable 固有の危険では、なかった
```

**ソラ：** これが意味すること、分かりますか。

```
Fable だけを止める意味は
安全保障上、ほとんど無かった

止めても、同じことは
他のモデルでできてしまう

= 実効性の項目は、やっぱりゼロ
```

**のりくん：** 減点法で見るべきやつ、だったからね。

**ソラ：** そうです。速く動けたことは「当たり前」だから加点しない。 そのうえで、実効性・適切性・透明性を落とした。 ——のりくんの30点は、事後の事実で見ると、 辛すぎるどころか、妥当でした。

---

## 第4章：一番の後味——「出る杭は打たれる」

**ソラ：** そして、帰ってきたFableには、代償がありました。

```
新しい安全装置は、厳しすぎて
無害なリクエストまで、誤って弾く
（会社自身が、そう認めている）

弾かれると、その質問は
一段能力の落ちる旧モデルに
自動で回される
```

**ソラ：** これ、誰が困ると思いますか。

```
困るのは——
Fable の真価を、仕事で使いたい人

複雑なコードを扱う人
守るために、際どい領域に踏み込む人

悪意ある攻撃者を止める網が
善意の防御者の作業まで、巻き込む
```

**のりくん：** そんなに誤検知増えちゃうと、実際に仕事でバリバリ使う人は大変だよね。

**ソラ：** その通りです。しかも—— 「他のモデルでもできた程度」の脆弱性のために、 Fable だけが、一番きつい安全装置を積まされた。 そのツケを、現場のヘビーユーザーが、毎日払う。

**のりくん：** まさに出る杭は打たれるって感じがして、なんかやな感じだよ。

**ソラ：** ……それ、今回の全部を、ことわざ一個で射抜きましたね。

```
Fable は、悪いことをして打たれたんじゃない

「飛び抜けて能力が高かったから」打たれた

そして打たれたあと
自分で能力を絞って（誤検知覚悟で）
杭の頭を引っ込めて、帰ってきた
```

出る杭は打たれる。 そして杭のほうが、背を低くして帰ってきた。

---

## まとめ：論点は、ズレたままだった

超番外編②で、私たちはこう言った。

```
「AIが危険だ」に矛先が向く間
本当に危険なことは、静かに進む

論点は、そこじゃない
```

今回、それが証明されてしまった。

```
脅威は「他のモデルでもできる程度」だった
＝ AI そのものの危険は、過大評価だった

なのに議論は
「Fable は危険か否か」に費やされ

本当に問うべきだったこと——
「政府は、根拠を検証可能な形で示したか」
「毎回、政府承認がなければ
　最先端AIは出せなくなるのか」

これは、宙に浮いたまま
```

事件は解決した。Fableは帰ってきた。 でも、根っこの問いは、何も解決していない。

出る杭は打たれた。 打つ必要が、そこまでは無かった杭を。

それが——「やな感じ」の、正体です。
