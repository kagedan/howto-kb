プログラミングできないのにAI自動化にハマったおじさんです。Make×Claudeで毎朝の作業をどんどん自動化しています。ノーコードでできた仕組みをそのまま共有していきます。

## はじめに

保有銘柄に関するネガティブなニュースを見逃して、株価が動いてから気づく。そんな経験ありませんか？

このシステムを作ってから、毎朝保有銘柄の関連ニュースをClaudeが自動収集・感情分析して、ネガティブなニュースだけSlackに届くようになりました。

## このシステムで何ができるか

```
毎朝6時スケジューラー起動
        ↓
保有銘柄リストを取得
        ↓
各銘柄のRSSニュースを収集
        ↓
Claudeが感情スコアを判定
        ↓
ネガティブ（スコア2以下）のみSlackに通知
        ↓
全銘柄の結果をスプレッドシートに記録
```

## 必要なツールと費用

・Make：無料〜月9ドル ・Anthropic API：1銘柄あたり数円 ・日経・各社RSS：無料 ・Googleスプレッドシート：無料 ・Slack：通知 → 無料

10銘柄を毎日分析して月約100〜200円のAPIコストです。

## 第1章｜ニュースソースの設定

各銘柄のニュースを以下のRSSで取得します。

**日経電子版（銘柄別）：**

```
https://www.nikkei.com/rss/rss.aspx?code={{ticker}}
```

**Yahoo!ファイナンス（銘柄別）：**

```
https://finance.yahoo.co.jp/rss/{{ticker}}
```

MakeのRSSモジュールで直近5件のニュースを取得します。

## 第2章｜銘柄リストとの照合

Googleスプレッドシートに保有銘柄リストを管理します。

```
A列: 銘柄コード
B列: 銘柄名
C列: 監視フラグ（TRUE/FALSE）
D列: アラート閾値（スコア2以下で通知）
```
