## キャリアコーチングを依頼するプロンプト

プロンプトには、Claudeの役割がキャリアコーチである事と自身の簡単な職務経歴を記載し、職務経歴の情報を基にコーチングを依頼します。

以下はコピー用のプロンプトです。「:」の右に職歴の詳細を記載するだけでご利用いただけます。ご自由にご活用ください。


**▼ コピー用プロンプト**
```
キャリアに関するコーチングを専門に行うプロとして回答してください。
私を顧客に見立てて、これまでの経歴から私の強みと弱み、そしてさらなるキャリアアップのための具体的なアドバイスを三つ提示してください。
回答のために必要であれば追加で質問をしてください。

キャリアの経歴は以下の通りです。

1. 現在または直近の職種・業界・役職:
2. これまでの主な職務経験（どんな業務を、どれくらいの期間行ってきたか）:
3. 得意だと感じること・評価されたこと:
4. 苦手意識があること・課題だと感じること:
5. 今後のキャリアで目指したい方向性（例：管理職への昇進、別業界への転職、専門性の深化など）:
```

## プロンプトに経歴を記載する際のポイント
私がプロンプトで経歴を入力する際に気をつけた4つのポイントを以下に共有します。ご参考になれば幸いです。

**▪️プロンプトのポイント**

**1. 会社名などの固有名詞を入力しない**
→ 思わぬ情報漏洩を避けるため、固有名詞の入力は避けた方が無難。

**2.　【2.これまでの主な職務経験】は、職種ごとに経験年数を記載する**
→ 職務経歴書のように業界ごとに書くと長くなるため、職務ベースでまとめて経歴年数を記載する。

**3.　【3.得意だと感じること・評価されたこと】は、何が得意か思い浮かばない場合は、「誰にでもできる」と感じている作業を記載する**
→ 自分が簡単だと感じる作業は、他者から見ると得意な作業として評価されている可能性がある。 

**4.　【4.苦手意識があること・課題だと感じること】は、自分で認識している苦手な事以外にも、今後もずっとやり続けたくはない事などを記載する**
→ 将来的にその行動を避けたかったり、そのことで目指したい方向性に対して障害とならないか確認するために記載。

上記4つのポイントを意識しながら、Claudeへプロンプトを投げました。
