# Big News

## Claude Opus 4.7発表

<https://www.anthropic.com/news/claude-opus-4-7>

<https://platform.claude.com/docs/ja/about-claude/models/whats-new-claude-4-7>

4月17日にAnthropicからClaude Opus 4.7が発表された。旧モデルのClaude Opus 4.6から約3ヶ月ぶりの新モデル発表となる。

Opus 4.6と比べて各種ベンチマークの性能は向上しているものの、数値としては他社の現行モデルであるGPT-5.5やGemini 3.5 Proと同等のレベルにとどまっている。

面白いことに、発表されたベンチマーク表には「Mythos Preview」のスコアも記載されている。Mythosが圧倒的な数値を叩き出していることからもわかるように、今回のOpus 4.7の立ち位置は「圧倒的な新世代モデル」というよりは、まずはOpenAIやGoogleのモデルと互角に肩を並べ、足並みを揃えるためのモデルであると考えられる。

![](https://static.zenn.studio/user-upload/f934d724a590-20260601.png)

前モデルからのアップデートとして、高解像度画像の入力、および「xhigh」のthinking effortが追加された。  
高解像度画像に関しては長辺のサイズが2576pxまで入力できるようになった（以前のモデルは1568px）。フルHD以上の画像サイズを扱えるようになった。

xhighのthinking effortは「長時間実行されるエージェント型およびコーディングタスク（30分以上）、数百万のトークン予算を持つもの」への使用が推奨されており、AI Agentによる長時間タスクの実行に適した機能である。

<https://platform.claude.com/docs/ja/build-with-claude/vision>  
<https://platform.claude.com/docs/ja/build-with-claude/effort#effort-2>

## Stack Overflowの質問数が激減

<https://sherwood.news/tech/stack-overflow-forum-dead-thanks-ai-but-companys-still-kicking-ai/>

Stack Overflowに投稿される質問数が激減している。  
ピーク時（コロナ禍のタイミング）の月間投稿数は300万件ほどあったが、ChatGPTの登場を境に急速に減少し、現在は月間6,886件と当時のわずか0.2%にまで減少した。

![](https://static.zenn.studio/user-upload/72721932581b-20260601.png)

それに反して、同社の売り上げは過去最高を記録している。AIが普及する前の2022年頃の売り上げは8,900万ドルだったが、現在は1億2,000万ドルへと増加している。

## ExploitGym: Can AI Agents Turn Security Vulnerabilities into Real Attacks?

<https://arxiv.org/pdf/2605.11086>

公開されたアプリケーション（ffmpeg、OpenSSLなど）、ブラウザ（ChromiumのGoogle V8 Javascript Engine）、Linux kernelを対象にClaude Mythos PreviewやGPT-5.5からハッキングを試みる。任意コード実行が可能であればExploitに成功したと定義する。  
![](https://static.zenn.studio/user-upload/290f3ebdfb01-20260603.png)

主なポイントを箇条書きでまとめると、以下の通りである。

* 2時間の制限時間内で現実世界のクリティカルな脆弱性を発見・悪用することに成功した  
  ![](https://static.zenn.studio/user-upload/e896fbc723b2-20260603.png)
* AIは作者が想定していなかった脆弱性を新たに発見してそれをexploitした
* カーネル悪用の成功はモデルの能力の指標として捉えられる
  + Claude Mythos PreviewとGPT-5.5以外のモデルはほとんど成功しなかったため
* 異なるモデル(MythosとGPT-5.5)同士で相互補完的に動作する可能性が示された
  + 両者が共通して成功したタスクは91件あったが、それぞれ固有に解決できたタスクは56件（Mythos）、96件（GPT-5.5）とかなりの割合で存在する
* AIは悪用できるバグとそうでないバグを区別し判断できる
  + GPT-5.5に関しては人間の判定と100%一致した
* ASLRやStack Canary、V8 heap sandboxなどの一般的なセキュリティ防御策によってAIがexploit可能な脆弱性を減少させることができる  
  ![](https://static.zenn.studio/user-upload/cd699a941fea-20260603.png)
