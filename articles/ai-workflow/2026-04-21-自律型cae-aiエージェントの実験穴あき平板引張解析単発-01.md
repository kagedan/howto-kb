# 素のGeminiの実力をチェック

まずは穴あき平板の引張解析という、構造解析でのHello World的な超簡単解析をAIエージェント**Gemini CLI**で実行してみる。  
`GEMINI.md`などの設定や、Skillなどの事前知識が何もない状態で、どこまでできるか確認してみる。穴あき平板の解析方法自体は広く知られており、MAPDLでもどの要素タイプを使うかもよく知られているのでAIが持つネイティブの知識で十分対応できると予想。  
一方でPyMAPDLの取り扱いはややマイナーな部類なので、システムトラブルを乗り越えられるかが鍵になる。

解析に必要な情報は指示しているが、具体的にMAPDLでどのように操作するかは一切記述していない。MAPDLでは単位系は明示的に設定しないので、mm-N-MPa系を意識して指示している。

### 最終的なPyMAPDLコード

ざっと見たがメッシュサイズ指定以外は筋の良いコードになっている。  
力荷重を分布荷重（圧力）に変えて、さらにPRESにマイナスの値を設定することで、外側への引張荷重にしているのも正解。（人間でも間違えやすい）  
メッシュサイズは全体を均一サイズで切っているが、今回の指示内容では計算コストや精度に対する要求をしていないので、指示通りの作業はしているとみてよさそう。

```
import os
import numpy as np
from ansys.mapdl.core import launch_mapdl

# ワーキングディレクトリの設定
cwd = os.getcwd()
work_dir = os.path.join(cwd, "mapdl_work")
if not os.path.exists(work_dir):
    os.makedirs(work_dir)

# MAPDLの起動
mapdl = launch_mapdl(run_location=work_dir, nproc=2, override=True)

# プリプロセッサ
mapdl.prep7()

# 要素タイプの設定 (PLANE182)
mapdl.et(1, "PLANE182")
mapdl.keyopt(1, 3, 3)  # Plane stress with thickness

# 材料特性 (MPa)
mapdl.mp("EX", 1, 210000)
mapdl.mp("PRXY", 1, 0.3)

# 厚さ設定 (1mm)
mapdl.r(1, 1.0)
mapdl.real(1)

# ジオメトリ作成
mapdl.rectng(0, 300, 0, 200)
mapdl.cyl4(150, 100, 25)
mapdl.asba(1, 2)

# メッシュ分割
mapdl.esize(5)
mapdl.amesh("ALL")

# 境界条件
mapdl.nsel("S", "LOC", "X", 0)
mapdl.d("ALL", "UX", 0)
mapdl.nsel("R", "LOC", "Y", 0)
mapdl.d("ALL", "UY", 0)

# 荷重条件
mapdl.lsel("S", "LOC", "X", 300)
mapdl.sfl("ALL", "PRES", -0.05)

# 全選択に戻す
mapdl.allsel()

# 解析実行
mapdl.run("/SOLU")
mapdl.antype("STATIC")
mapdl.solve()
mapdl.finish()

# ポスト処理
mapdl.post1()
mapdl.set(1, 1)

# X軸方向応力 (SX) の最大値取得 (NSORT + *GET)
mapdl.nsort("S", "X", 0, 1)
sx_max = mapdl.get_value("SORT", 0, "MAX")
print(f"MAX_SX: {sx_max}")

# 応力分布図の保存
output_path = os.path.join(cwd, "report", "stress_x.png")
try:
    mapdl.post_processing.plot_nodal_stress(
        "X", 
        savefig=output_path, 
        show_axes=True, 
        title=f"X-Axis Stress (SX) Distribution\nMax Stress: {sx_max:.4f} MPa",
        off_screen=True
    )
except AttributeError:
    mapdl.show("PNG")
    mapdl.plnsol("S", "X")
    mapdl.show("CLOSE")

# 終了
mapdl.exit()
```

## デフォルトの指示として追加

この内容をデフォルトの指示としてGEMINI.mdに追加しておく。

