# はじめに

こんにちは。すぎもんです![:smiley:](https://cdn.qiita.com/emoji/twemoji/unicode/1f603.png ":smiley:")（連投失礼します！）  
今後のために生成AIを利用した検証環境を準備した内容を残しておきます！

# 今回やること

生成AIを利用した検証をするための各種環境を構築します。  
具体的には、Claude と Node.js、Visual Studio Code の導入を行っていきます。

# 実施手順

以下の流れで 事前準備 を行っていきます。

1. Claude の導入
2. Node.js の導入
3. Visual Studio Code の導入

それでは早速やってみましょう！
