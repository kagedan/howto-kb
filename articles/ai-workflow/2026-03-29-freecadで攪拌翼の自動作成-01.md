## この記事について

本記事では、OpenFOAMによる流体解析で使用するCADモデルの作成方法を紹介します。オープンソースの3D CADソフトウェアであるFreeCAD の Python API を用いて、**攪拌翼**を 3D モデルとして生成するスクリプトを作成していきます。  
FreeCADによるモデリングはまだ手探りな部分もありますので、アドバイスをいただけると幸いです。

## PythonスクリプトによるCADモデル自動生成

FreeCADは、Pythonスクリプトで完全に制御可能な3D CADソフトウェアです。ユーザーインターフェース、3Dモデル、画面表示など、ほぼすべての機能をPythonから操作できます。

**1. 豊富なPython APIで自動化を実現**  
FreeCADは広範なPython APIを提供しており、GUI操作で行える作業のほぼすべてをスクリプトで自動実行できます。

* パラメータの変更: モデリング履歴を遡ってパラメータを変更し、形状を自動更新。
* 高度な形状処理: ジオメトリエンジン「OpenCASCADE」の機能を活用した複雑な形状計算。

**2. マクロ録画機能で簡単に自動化を開始**  
初心者でも簡単に自動化を始められるよう、マクロ録画機能が用意されています。

* 操作の記録: 「Macro > Macro recording」で録画を開始し、モデリング操作を行うと、その操作がPythonコードとして自動記録されます。
* 再利用と共有: 記録したコードを保存・実行することで、複雑な作業を瞬時に再現できます。作業効率化だけでなく、バグ報告時に問題を正確に伝える手段としても有効です。

今回はPythonスクリプトを直接コンソールに記述(コピー＆ペースト)してモデリングしていきます。

## 攪拌翼モデルの作成

## 1. 全体の流れ

今回のモデルは以下の作成の流れで行っています。

```
Step1: 接線方向オフセット矩形断面の生成
       ↓
Step2: ロフトで1枚の羽根ソリッド生成
       ↓
Step3: ポーラーパターン（n枚複製）
       ↓
Step4: ハブ円柱生成
       ↓
Step5: fuse + removeSplitter で一体化
```

## 2. 利用関数

| 関数 | 役割 |
| --- | --- |
| `make_rect_profile()` | 各高さの矩形断面ワイヤー生成 |
| `make_single_blade()` | ロフトで1枚の3Dブレード生成 |
| `polar_pattern()` | 円周方向に等角度で複製 |
| `make_hub()` | 中心軸（円柱）生成 |
| `fuse_all()` | 全パーツをブール結合 |

## 3. パラメータ一覧と設定

攪拌翼のパラメータ設定を以下に示しています。今回は出来るだけ設定を細かく変更できるように変数を多くしました。ここで寸法・形状を一箇所で管理しています。

```
BLADE_COUNT     = 4      # ポーラーパターンの複製数
BLADE_HEIGHT    = 0.015   # ロフトのZ方向距離 [mm]
TWIST_ANGLE     = 30.0   # 上面断面の回転角 [degrees]
                         # → 大きいほど強いねじれ

R_INNER         = 0.007    # 矩形断面の内側半径 [mm]
R_OUTER         = 0.045   # 矩形断面の外側半径 [mm]
                         # → R_OUTER - R_INNER = 羽根の径方向長さ

BLADE_THICKNESS = 0.002    # 接線方向の板厚 [mm] ← 全高さで均一
HUB_RADIUS      = 0.007    # ハブ円柱の半径 [mm]
                         # → R_INNER 以下にすること
```

## 4. Pythonコード

以下にPythonスクリプトを表示します。こちらのコードをコンソールにコピー＆ペーストすることで攪拌翼モデルが作成されます。

blade.py

```
import FreeCAD as App
import Part
import math

# ================================================
# パラメータ設定
# ================================================
BLADE_COUNT      = 4      # 羽根枚数
BLADE_HEIGHT     = 0.015   # 羽根高さ [mm]
TWIST_ANGLE      = 15.0   # ねじれ角 [degrees]

R_INNER          = 0.007    # 羽根内端半径（ハブ接続部）[mm]
R_OUTER          = 0.045   # 羽根外端半径 [mm]
BLADE_THICKNESS  = 0.002    # 板厚（均一）[mm]

HUB_RADIUS       = 0.007    # ハブ（中心軸）外径 [mm]  ← 調整可能
HUB_HEIGHT       = BLADE_HEIGHT  # ハブ高さ（羽根と同じ高さ）

PITCH_DEG = 360.0 / BLADE_COUNT

# ================================================
# Step 1: 矩形断面ワイヤーの生成
#   各高さで、径方向軸に対して接線方向に ±t/2 オフセットした
#   4点矩形 → 板厚が接線方向で均一
# ================================================
def make_rect_profile(z, rot_deg):
    """
    高さ z, 回転角 rot_deg [deg] で矩形断面ワイヤーを生成。
    厚さ方向 = 接線方向（radial 方向と直交）
    """
    a   = math.radians(rot_deg)
    t_half = BLADE_THICKNESS / 2.0

    # 径方向単位ベクトル: ( cos(a),  sin(a), 0)
    # 接線方向単位ベクトル: (-sin(a),  cos(a), 0) ← 厚さ方向
    tx = -math.sin(a)
    ty =  math.cos(a)

    def pt(r, sign):
        return App.Vector(
            r * math.cos(a) + sign * t_half * tx,
            r * math.sin(a) + sign * t_half * ty,
            z
        )

    # 4頂点（閉じた矩形）
    p1 = pt(R_INNER, +1)   # 内径・前縁
    p2 = pt(R_OUTER, +1)   # 外径・前縁
    p3 = pt(R_OUTER, -1)   # 外径・後縁
    p4 = pt(R_INNER, -1)   # 内径・後縁

    wire = Part.Wire([
        Part.makeLine(p1, p2),   # 前縁（外向き）
        Part.makeLine(p2, p3),   # 外縁
        Part.makeLine(p3, p4),   # 後縁（内向き）
        Part.makeLine(p4, p1),   # 内縁
    ])
    return wire

# ================================================
# Step 2: ロフトで1枚の羽根ソリッドを生成
# ================================================
def make_single_blade():
    w_bot = make_rect_profile(z=0,            rot_deg=0)
    w_top = make_rect_profile(z=BLADE_HEIGHT, rot_deg=TWIST_ANGLE)

    blade = Part.makeLoft(
        [w_bot, w_top],
        True,   # solid
        False,  # ruled
        False
    )
    return blade

# ================================================
# Step 3: ポーラーパターン
# ================================================
def polar_pattern(blade_one):
    blades = [blade_one]
    for i in range(1, BLADE_COUNT):
        b = blade_one.copy()
        b.rotate(App.Vector(0, 0, 0),
                 App.Vector(0, 0, 1),
                 PITCH_DEG * i)
        blades.append(b)
    return blades

# ================================================
# Step 4: ハブ・統合
# ================================================
def make_hub():
    return Part.makeCylinder(
        HUB_RADIUS, HUB_HEIGHT,
        App.Vector(0, 0, 0),
        App.Vector(0, 0, 1)
    )

def fuse_all(blades, hub):
    result = hub
    for b in blades:
        result = result.fuse(b)
    return result.removeSplitter()

# ================================================
# メイン処理
# ================================================
print("攪拌翼を生成中...")

assert HUB_RADIUS <= R_INNER, \
    f"HUB_RADIUS({HUB_RADIUS}) は R_INNER({R_INNER}) 以下にしてください"

doc = App.newDocument("TwistedBlade")

single    = make_single_blade()
blades    = polar_pattern(single)
hub       = make_hub()
all_shape = fuse_all(blades, hub)

feat_s = doc.addObject("Part::Feature", "SingleBlade")
feat_s.Shape = single

feat_a = doc.addObject("Part::Feature", "AllBlades")
feat_a.Shape = all_shape

doc.recompute()

try:
    import FreeCADGui as Gui
    Gui.activeDocument().getObject("SingleBlade").Visibility = False
    Gui.activeDocument().activeView().viewIsometric()
    Gui.SendMsgToActiveView("ViewFit")
except Exception:
    pass

print("=== 完了 ===")
print(f"  羽根枚数  : {BLADE_COUNT} 枚")
print(f"  羽根高さ  : {BLADE_HEIGHT} mm")
print(f"  ねじれ角  : {TWIST_ANGLE} °")
print(f"  内端半径  : {R_INNER} mm")
print(f"  外端半径  : {R_OUTER} mm")
print(f"  板厚      : {BLADE_THICKNESS} mm  ← 接線方向・全高さで均一")
print(f"  ハブ径    : {HUB_RADIUS * 2:.1f} mm")
```
