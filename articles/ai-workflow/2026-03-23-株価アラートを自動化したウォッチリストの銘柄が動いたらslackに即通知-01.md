プログラミングできないのにAI自動化にハマったおじさんです。Make×Claudeで毎朝の作業をどんどん自動化しています。ノーコードでできた仕組みをそのまま共有していきます。

## はじめに

ウォッチリストに入れている銘柄が急騰・急落しても、気づくのが遅れることありませんか？

このシステムを作ってから、設定した銘柄が急変したとき即座にSlackに通知が届くようになりました。

## このシステムで何ができるか

```
15分ごとにスケジューラーが起動
        ↓
ウォッチリストの銘柄の株価を取得
        ↓
前回取得時からの変化率を計算
        ↓
変化率が±3%以上の銘柄だけSlackに通知
```

## 必要なツールと費用

・Make：無料〜月9ドル ・Alpha Vantage API：株価取得 → 無料 ・Googleスプレッドシート：ウォッチリスト → 無料 ・Slack：通知 → 無料

APIコストはほぼ0円で運用できます。

## 第1章｜ウォッチリストの設定

Googleスプレッドシートにウォッチリストを作成します。

```
A列: 銘柄コード（例：7203.T）
B列: 銘柄名（例：トヨタ自動車）
C列: アラート閾値（例：3）→ ±3%で通知
D列: 前回株価（Makeが自動更新）
```

## 第2章｜株価取得の設定

```
URL: https://www.alphavantage.co/query
   ?function=GLOBAL_QUOTE
   &symbol={{ticker}}
   &apikey=YOUR_API_KEY
```

取得できるデータ： ・現在株価 ・前日終値 ・変化率（%）
