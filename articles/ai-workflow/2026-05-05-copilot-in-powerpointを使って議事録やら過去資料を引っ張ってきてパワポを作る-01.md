# はじめに

この記事は、営業・企画など、**ふだんコードを書かない非エンジニア職**向けです。

「CopilotでPowerPointを作れる」と聞いて試してみたけど、なんか微妙だった…という経験はありませんか？

検索して出てくる方法の多くは、Pythonが登場したり、Claudeと組み合わせたりとエンジニア向けのものが多いです。  
読み始めたのに途中で「あ、これ私には無理だ」と感じた方もいるのではないでしょうか。

この記事で紹介するのは、そういったツールを一切使わない方法で、  
**Copilot in PowerPointで完結します。**  
一切ツールの購入やインストール、構築などは不要です。

# こんな経験ありませんか？

「今回の提案先にはA商材とB商材が刺さりそう。先日のヒアリングで出た課題も入れて、提案先向けにまとめたい」

こういうとき、ゼロから資料を書き起こすのではなく、**既存の資料を組み合わせる**作業になりますよね。

* 商材資料Aから該当ページをコピー
* 商材資料Bから該当ページをコピー
* ヒアリングメモを見ながら課題スライドを手打ち
* 全体のデザインを統一して体裁を整える

地味に時間がかかる、でも避けられない作業です。

**これをほとんどCopilot in PowerPointに丸投げできます。**

# 必要なものは3つだけ

* M365 Copilotのライセンス
* テンプレートとなるPowerPointファイル（.pptx）
* 資料の内容をまとめたファイル（何でもOK、後述します）

# 実際にやってみる

架空の会社「YK technology」のパワポテンプレートを使って Copilotを用いた資料の作成を行ってみます。  
シチュエーションとしては株式会社日本精機製造（架空の会社です）にDX推進の課題をヒアリングした直後。次回打ち合わせで自社商材を提案するぞ、という状況です。

## ステップ① テンプレートを準備する

唯一の事前準備です。ただし、難しいことはしません。  
ほとんどの会社にはすでに社内のPowerPointテンプレートがあるはずなのでそれを使う、でOKです。

ない場合は普通のPowerPointファイルを用意して、デザイン・色・フォント・レイアウトをここで決めておきます。  
スライドの中身のテキストは「スライドタイトル」「本文」のような仮の文字列にしておきましょう。

> **ポイント：** テンプレートさえ作っておけば、次回以降はここをスキップできます。一度作ったら使い回しOKです。

## ステップ② 内容を準備する

作りたい資料の内容を用意します。形式は何でも構いません。

* テキストを書いたメモ帳やWordファイル
* 既存のPowerPointファイル

> **注意：** 2026/05/05の執筆時点では後述のPowerPoint内Copilotから、txtファイルなど一部形式が見えない状態です。もしメモを読み込ませたい場合は一度Wordに転記しておくのがオススメです。

「完璧に整理されていないといけない」ということはなく、箇条書き程度でもCopilotが読み取って資料にしてくれます。

## ステップ③ パワポの中のCopilotに頼む

ここが一番のポイントです。**難しいことは何もしません。**

パワポを開いてCopilotのチャット画面を出し、①のテンプレートファイルと②の内容ファイルをリンクで指定して、普通の言葉で頼むだけです。

[ホーム]→[Copilot]  
![](https://static.zenn.studio/user-upload/ba450ca9f5ba-20260505.png)

Claude Opus 4.7にするのを忘れずに！  
![](https://static.zenn.studio/user-upload/8ae85fb14966-20260505.png)

プロンプトはたとえばこんな感じです：

```
(ファイル名)を読んで資料を作成してください。
製品説明のスライドは(ファイル名)の内容をそのまま使ってください
```

ファイル名の部分は＋マークを押すと出てくる一覧から挿入可能です  
![](https://static.zenn.studio/user-upload/18b25a2cf0ec-20260505.png)

![](https://static.zenn.studio/user-upload/1b95e9abf2fd-20260505.png)

```
課題感ヒアリング 議事録.docx の中身を読み、次回の提案で使えるパワポを作ってください。
現状の課題感と打ち手の整理、提案できそうな商材のご紹介はマストで含めてください。商材の情報は以下から引っ張ってきて
DX推進コンサルティング.pptx サイバーセキュリティサービス.pptx マネージドITサービス.pptx
スライドのテンプレは全種類使う必要はありません。適宜必要なものを編集して使って。必要ないものがあれば削除してください
```

あとはCopilotがファイルを読み取り、テンプレートのデザインを崩さずに中身を組み立ててくれます。

詳しく説明すると

CopilotはPowerPointファイルの内部構造（XMLと呼ばれるテキストデータ）を直接読み取り、テンプレートのデザインを崩さずにテキストだけを書き換えています。Pythonなどのプログラムを使わずにこれができるのが、このやり方の特徴です。
