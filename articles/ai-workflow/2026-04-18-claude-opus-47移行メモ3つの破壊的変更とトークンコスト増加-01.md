# Claude Opus 4.7移行メモ：3つの破壊的変更とトークンコスト増加

Opus 4.7に移行する前に確認すべき変更点をまとめます。

## 破壊的変更3点

### 1. 拡張思考ペイロード

```
# 旧形式（400エラー）
thinking={"type": "enabled", "budget_tokens": 10000}

# 新形式
thinking={"type": "adaptive", "effort": "high"}
```

### 2. サンプリングパラメータ削除

非デフォルトの`temperature`/`top_p`/`top_k`は`400`エラー。

### 3. 思考テキストのデフォルト非表示

```
# 表示するにはオプトイン
thinking={"type": "adaptive", "effort": "high", "display": "summarized"}
```

## トークンコスト

表示価格は同じ（$5/$25 per MTok）。ただし更新されたトークナイザーにより、同じ入力が最大1.35倍のトークンを消費する場合があります。

## 移行推奨ワークロード

即時移行：コーディング、コードレビュー、エージェント、デバッグ

段階的移行：旧推論ペイロード使用、思考トレース表示、厳格なトークン予算
