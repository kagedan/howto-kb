## 添削結果は「ここまで」教えてくれる

### 模範訳が常に3パターン生成される理由

添削を依頼すると、Claudeは常に模範訳を3パターンで示してくれます。

```
① シンプル基本型：文法的に正確でシンプル
② 観光パンフレット型：印刷物に載るような流れる尚な文体
③ ガイドスピーチ型：実際に口から発するときの自然な論調
```

**この3型展開には深い意味があります。** 全国通訳案内士の試験では、「対話」「文章」「プレゼン」など場面によって求められる英語が全く違います。ガイドスピーチで「people」と言うとコナーになりますが、パンフレットで「residents」とあれば格が安定します。3パターンを比較すること自体がトレーニングになります。

### 100点満点・3軸評価の使い方

軸満点何を評価しているか文法の正確さ35点構造・時制・冠詞・語順観光英語の自然さ35点パンフレット・ガイドらしい表現か語彙の適切さ30点専門用語・固有名詞の正確な扱い

点数が低いとき、それが「文法の問題」なのか「語彙の問題」なのかによって、次にやるべき練習が全く変わります。文法が弱いなら基礎文法の復習、語彙が弱いなら単語平活用のトレーニング。この切り分けが、効率を最大化します。

## 英単語帳をそのまま活用する「一言コマンド」

```
「英単語帳の『使える』チェックがない単語を
3つ使った地理の問題を作って」
```

するとClaudeは自分で単語帳DBを参照し、「使える」になっていない単語を使った問題を生成してくれます。解いて正解できたらその単語の「使える」に自動でチェックが入る。**単語帳と問題帳が一体化されるとは、こういうことです。**

## 82語の英単語帳が爆速で完成した話

### 単語帳の「内容がスカスカ」問題

長年の英会話レッスンが経緯で自分の単語帳には82語が登録されていました。でも内容はスカスカでした。名前だけがあってテキストが空欄のものが30語、品詞はあるが語源がないものが50語……。「登録するまでが精一杯で、その後は漠えるまま」のあれです。

### 「全部埋めて」の一言

```
「英単語帳の空欄を全部埋めて」
```

Claudeは自分でNotionの全エントリーを調査し、各単語に以下の情報を追加していきました。

```
▪️ 活用形（過去形 / 名詞形 / 形容詞形）
■ ネイティブの使用頻度：1〜10のスコア＋使われるシーンの説明付き
■ 意味：日本語
■ 英語で意味：英英辞典スタイルの定義
■ 語源：ラテン語・古英語などの由来
■ collocation：実際によく使われる組み合わせ4例
```

### 実際の補完例：boasts

```
▪️ 過去形：boasted / 現在分詞：boasting / 名詞形：boast
■ 頻度：5〜6（観光ガイド・パンフレットでは頻出）
■ 意味：〜を誇る・〜が自慢である
■ 英語で意味：To have or own something impressive,
  often used when describing a place's notable features.
■ 語源：中英語「bosten」（自慢する）から
■ collocation：
  • Kyoto boasts numerous temples
  • boasts a rich history
  • boasts world-class facilities
  • boasts one of the largest ~
```

これが**全82語全て**に実行されました。

> 💡 **学習者が気づかないポイント** 「語源」を知ることは単なる雑知ではありません。語源を知ると**派生語が推測できる**ようになります。boastの語源が「自慢する」と知れば、boastful（自慢した）、boastfullyなども推測できる。語源は小さな投資で大きなリターンを生む語彙学習法です。

## 学習者が山来にするTips集

### Tip 1：採点後に「体感難易度」を必ず伝える

点数は客観的な指標ですが、「自分がどう感じたか」という主観的な情報も非常に有用です。

データと主観を組み合わせることで、**期待値の高い学習計画**が立てられます。

### Tip 2：模範訳は「ピックアップ」して自分のものにする

3パターンの模範訳を見たら、「この表現が喗めたい」と思ったものを必ず英単語帳に登録します。

```
「今の模範訳で使われているこれらの表現を単語帳に追加して」
```

これだけでClaudeが自動でNotionの英単語帳に登録してくれます。「boasts」「is home to」「welcomes」など観光英語のストックが山のように増えました。

### Tip 3：「one moreチャレンジ」を使う

同じ問題を模範訳を見た後にもう一度自分で訳す「one moreチャレンジ」がかなり効きます。

```
「先ほどの模範訳を参考に、同じ問題をもう一度自分で訳して添削して」
```

模範訳を見た後に反射的に訳せるかどうかをテストすることで、定着度が生まれます。

### Tip 4：化せない表現は「なぜ？」を問う

```
「with a population of と which have はなぜ意味が違うのか
観光ガイドの英語としてどちらが良いか具体的に教えて」
```

> 「which haveを使うと関係代名詞節になり『持っている都市』と指示語が必要になります。with a population ofは前置詞句なので文にスムーズに追加できます。観光英語では『人口』を記述するときはwith a population of が定番表現です」

「そういうことか」と理解すると、次からおのずと使えるようになります。

### Tip 5：定期的に「同カテゴリ5問」チャレンジをやる

```
「記録を見て、『地理・都市』カテゴリの未解答問題を参照しながら5問連続で出題して」
```

同じカテゴリを集中的に繰り返すことで、長期記憶に移行するスピードが上がります。

### Tip 6：「日本語→英語→日本語」の背訳トレーニング

英語を訳した後、模範訳を見ずに反射的に日本語に訳してみる。自分で訳せない表現が定着していないことの証拠です。
