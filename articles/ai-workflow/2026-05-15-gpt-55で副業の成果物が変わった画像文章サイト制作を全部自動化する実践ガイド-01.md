去年の秋、noteのサムネイルを作るのに毎回30分以上かかっていた。

日本語テキストを入れようとすると文字が崩れる。「副業」が「副棄」になって、「収入」が「収人」になる。何度プロンプトを書き直しても、文字化けは直らなかった。諦めてCanvaで手作業するのが習慣になっていた。

それがGPT-5.5とChatGPT Images 2.0に変えたら、1発目から日本語が正確に入った。しかも思っていたよりずっとクオリティが高い。ああ、ここまで来たか、と思った。

2026年4月下旬、OpenAIが公開したGPT-5.5は「これまでで最も賢く、直感的に使えるモデル」として発表された。全AIモデルの総合評価でClaudeやGeminiを抑えて首位に立った。でも大事なのはベンチマークの数字より「副業で何ができるようになったか」だ。

この記事では、GPT-5.5を副業の5場面でどう使うかを具体的なプロンプト付きで解説する。OpenAI公式のプロンプトガイド（2026年4月版）と、自分が実際に試した結果をもとにしている。

## 第1章｜旧ChatGPTと何が違うのか——「指示待ち」から「自律型」へ

![](https://assets.st-note.com/img/1778646204-tJWCc3Y5FKe17MoGLgImRD6V.png?width=1200)

### 1番大きな変化は「動き方」が変わったこと

旧モデルと比べて一番変わったのは、こちらが手順を細かく書かなくても動くようになったことだ。

旧モデルは「手順1・手順2・手順3」と全部書かないと迷子になりやすかった。GPT-5.5は「何を完成させたいか」だけ渡せば、自分で計画を立てて実行して確認しながら進んでくれる。整理されていないタスクでも、自分で考えながら動く。

OpenAIが特に得意だと言っている領域はこの6つだ。

副業で使うほとんどの作業はこのどれかに当てはまる。

### 今すぐ使えるプランとコスト

モデル使えるプラン月額目安GPT-5.5インスタント無料含む全ユーザー0円〜GPT-5.5Plus・Pro・Business3,000円〜GPT-5.5 ProPro・Business・Enterprise20,000円〜

副業で使うならPlusプランで十分だ。月3,000円で思考モード付きのGPT-5.5が使える。

## 第2章｜プロンプトの書き方を変えたら、出力が変わった

![](https://assets.st-note.com/img/1778646237-ryxU8HWfjCZpD4S6Bgo0vOAP.png?width=1200)

### OpenAI公式が「これで書いてくれ」と言っている

GPT-5.5のリリースと同時に、OpenAIは公式のプロンプトガイドを公開した。そこに書いてあった内容が面白かった。要約するとこうだ。

**「細かい手順を書けば書くほど、精度は下がる」**

過去のモデルでは「手順1→手順2→手順3」と分解して書く方がいい回答が出やすかった。GPT-5.5では逆で、ゴール・成功条件・制約の3点だけ書いて、解き方はモデルに任せる方が強い。

自分がこれを試したのは、ある日のnote改稿作業だった。いつも通り手順を6行書いてGPT-5.5に渡したら、途中で止まって「どの手順を優先しますか？」と聞いてきた。試しに3点だけに絞ったら、1回目で望んでいた出力が出た。手順が多すぎると、モデルが混乱するらしい。

### コピペ用：副業ライティングの汎用フレーム

```
ゴール：[何を作りたいか]
成功条件：[どんな状態になれば成功か・読者への効果]
制約：[使ってはいけない表現や守るべきルール]
参照：[添付ファイルまたはここに貼る内容]
```

このフレームに当てはめるだけで、ブログ記事・商品説明文・提案文・SNS投稿など、どんな文章生成でも使い回せる。

## 第3章｜副業で使える活用事例5選

![](https://assets.st-note.com/img/1778646249-WdhDM6x1m5IrN8C4aJbocXj3.png?width=1200)

### 事例1｜日本語入り広告バナーを1発生成

GPT-5.5の思考モードをオンにすると、ネットでリアルタイムに情報を調べながら画像を生成するようになる。この思考モードとChatGPT Images 2.0を組み合わせると、日本語テキストがほぼ正確に入った画像が1発目から出るようになった。

先週クライアントから「EC出品用の商品バナー5枚」を依頼されたが、プロンプトを1回送って10分後には5案が揃っていた。以前なら半日仕事だった。

```
ゴール：副業初心者向けSNS広告バナーを3枚生成したい
成功条件：
・日本語テキストが正確に入っていること
・それぞれ異なるレイアウト・配色
・サイズは1080×1080px（正方形）
制約：
・文字は「AI副業で月3万円」の1行のみ
・顔写真・ロゴ不要
・背景はグラデーション系で明るい印象
```

### 事例2｜ウェブサイトをフォルダに直接生成

CodexはOpenAIのデスクトップアプリだ。2026年5月時点でProプラン以上で使える。起動後にローカルのプロジェクトフォルダを指定して、そこで自然言語で指示を出す。

最初にこれを試したとき、フォルダにindex.htmlが突然出現したのには驚いた。Codex側の「プランモード」で事前に実装計画を確認してからOKを出す。その後でコードが一気に生成される。

```
ゴール：[サービス名]の紹介LP（日本語）を作りたい
成功条件：
・ファーストビューにキャッチコピーとCTAボタン
・料金プランのセクション
・FAQ3〜5個
制約：
・スマホ対応（レスポンシブ）
・外部画像は使わず、プレースホルダーで
プロジェクトフォルダ：[指定済み]
```

### 事例3｜リサーチ→スプレッドシートを一気通貫

「〇〇についてリサーチしてスプレッドシートにまとめて」という1文で、ネットから情報を集めて整理して表形式に出力してくれる。

実際に試した例：「2026年に公開された動画生成AIをリストアップして、ツール名・リリース機能・料金をスプレッドシートにまとめて」と送ったら、30秒後に30ツール分の比較表が出た。以前同じ作業を自分でやったとき、2時間かけて20件しか集まらなかった。

```
ゴール：[業界名]の競合サービスを調査してスプレッドシートにまとめたい
成功条件：
・サービス名・月額料金・主な機能・ターゲット層を列に持つ
・10〜15サービス以上
・最新情報（2026年）を優先
制約：
・廃止・サービス終了のものは除く
・公式サイト情報を優先する
```

### 事例4｜制約条件付きライティング

「NG表現あり・文字数制限あり・ターゲット読者指定」といった複数の条件を同時に守りながら書いてくれる。先日、メルカリで家電を20点出品したとき、全部の商品説明文をこの方法で出した。1点あたり5分以内で終わった。

```
ゴール：メルカリ出品用の商品説明文を書きたい
成功条件：
・300〜400文字
・購買意欲を高める書き方
・商品の状態・使用感が正確に伝わる
制約：
・「最安値」「即購入OK」などの誇大表現は使わない
・絵文字なし
・箇条書きで状態・付属品・発送について書く

商品情報：[ここに商品名・状態・詳細を書く]
```

### 事例5｜アプリ・ゲームのプロトタイプ生成

CodexのプランモードとChatGPT Images 2.0を組み合わせると「UIデザイン画像を作る→その画像をもとに動くアプリを生成」という流れが成立する。

1. Codexを起動してプロジェクトフォルダを指定
2. ChatGPT Images 2.0でゲーム・アプリのUIデザイン画像を生成
3. 生成した画像を参照してプランモードで「このデザインでゲームを作って」と指示
4. 実装計画を確認してOKを出す
5. 動くアプリがフォルダに生成される

最初に試したときは「シンプルなタスク管理アプリ」を指示したら、20分でブラウザで動くものが出てきた。社内ツール・管理画面のプロトタイプや、スタートアップのMVP開発補助として単価は高め（1件5〜10万円）で、Codexを使えるだけで差別化になる。

### 最初に取り組むべき組み合わせ

入口として始めやすいのは「事例4（ライティング）+ 事例1（バナー生成）」のセットだ。クライアント獲得はクラウドワークスやランサーズでEC出品代行を検索するか、Xで「出品代行します」と投稿してみる。動き始めて1〜2週間で最初の案件が入ることが多い。

## まとめ

GPT-5.5で変わったのは「できることの種類」より「できる速度と精度」だ。

バナー生成・スプレッドシート調査・ライティング・ウェブサイト生成——これらは以前のモデルでも一応できた。でも精度が低かったり、手順を細かく書く必要があったり、日本語テキストが崩れたりで、実用レベルに持っていくのが大変だった。

GPT-5.5はその「使えるけど惜しい」がなくなったモデルだ。
