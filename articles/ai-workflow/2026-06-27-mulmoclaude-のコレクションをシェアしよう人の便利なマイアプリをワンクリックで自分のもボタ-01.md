## 「これ、自分でも欲しい」をワンクリックで

[MulmoClaude](https://github.com/receptron/mulmoclaude) には **コレクション** という機能があります。「読書リスト」「観た映画の記録」「ワインのテイスティングメモ」みたいな “自分専用のデータアプリ” を、**コードを書かず会話だけ** で作れるものです。表・カレンダー・カンバンといった画面が自動でついてきます。

便利なんですが、こう思ったことはありませんか？

* 「ゼロから作るのちょっと面倒…誰かが作った良いやつをそのまま使いたい」
* 「逆に、自分が作った力作、他の人にも使ってほしい」

その **「使いたい」「使ってほしい」をつなぐ仕組み** ができました。人のコレクションを **ワンクリックで取り込み**、自分のコレクションを **ボタンひとつで共有** できます。

## 使ってみる①：人のコレクションを取り込む

いちばん簡単な始め方は、**人が作ったコレクションを取り込む** ことです。

1. **Collections（コレクション）** のページを開く
2. 上の **「Discover（発見）」** タブをクリック
3. 公開されているコレクションが並ぶので、気になったものの **「Import（取り込む）」** を押す

これだけ。取り込んだコレクションは自分のワークスペースに入って、**すぐ使えます**。画面やフィールドはそのまま、データは自分のものとして増やしていけます。

> 取り込むときは中身を自動でチェックしてから入るので、安心して試せます。

## 使ってみる②：自分のコレクションをシェアする

気に入ったコレクションができたら、**世界に共有** してみましょう。これも難しい操作はゼロ。**Git を知らなくても大丈夫** です。

1. 自分のコレクションのカードにある **共有アイコンのボタン（Share / 寄稿）** を押す
2. 「このコレクションをシェアしますか？」と確認が出るので **OK**
3. あとは **AI エージェントが自動でやってくれます**
   * 共有用のデータを組み立てて
   * GitHub に Pull Request（公開申請）を出す
4. 途中で 2 つだけ聞かれます
   * **GitHub のユーザー名**（あなたの名前で公開するため）
   * **自分のデータを “見本” として一緒に入れるか**（入れると、取り込んだ人がすぐ中身を見られます）

レビューを通れば、あなたのコレクションが **Discover に並んで、みんなが取り込めるように** なります。

ポイントは、**面倒な Git 操作をぜんぶエージェントが代わりにやってくれる** こと。あなたはボタンを押して、質問に 2 つ答えるだけです。

## どんなコレクションが向いてる？

たとえばこんなもの。想像してみてください。

* 📚 読書リスト（ステータスをカンバンで管理）
* 🎬 映画ログ（観た日・評価・ひとことメモ）
* 🍷 ワインのテイスティングノート
* ✈️ 旅行プラン（カレンダー表示）
* 📈 投資ウォッチリスト
* 🍳 レシピ集

“ちょっと管理したい” もの全部が候補です。あなたが工夫した分類やビューは、きっと他の誰かにも役立ちます。

## まずは試してみよう

* **使う人へ**：Collections → Discover → 気になるものを **Import**。1 分で始められます。
* **作る人へ**：コレクションをひとつ作って、**共有ボタン** を押してみてください。エージェントが共有まで連れて行ってくれます。

“自分専用” だったマイアプリを、**みんなで持ち寄れる** ようになりました。あなたの一品、ぜひシェアしてください 🤝
