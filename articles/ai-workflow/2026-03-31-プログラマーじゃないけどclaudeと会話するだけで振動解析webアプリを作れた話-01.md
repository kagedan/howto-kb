## はじめに

私は機械系のエンジニアです。プログラマーではありません。

業務では振動解析を日常的に行いますが、専用の計測器ソフトは高いし、起動が遅いし、ライセンスが1本しかないし……と何かと不便を感じていました。

「ブラウザでサクッとFFTできるツールがあればいいのに」と思い続けていたところ、**AIと会話するだけで作れるんじゃないか**と思い、挑戦してみました。

その結果、こんなツールが完成しました。

👉 **WaveAxis** : <https://waveaxis.vercel.app/>  
（PCでの閲覧推奨。最初の波形生成は少し時間がかかります）

## 「Claude Code」は使っていない

ここが一番伝えたいことです。

最近、AIを使った開発ツールとして「Claude Code」が話題になっていますが、**私が使ったのは普通のブラウザ版Claude（claude.ai）だけ**です。

ターミナルを開いて専用ツールを動かすような開発者向けの環境は一切使っていません。ブラウザのタブでClaudeと会話して、コードを1ファイルずつ受け取って、自分のPCに保存する。それだけです。

## Claudeとの開発の流れ

### 基本はこの繰り返し

やり取りの流れはシンプルです。

1. やりたいことをClaudeに伝える
2. コードを受け取る
3. 動かす
4. エラーが出たらエラーメッセージをそのまま貼り付ける
5. 「これどういう意味ですか？」と聞く
6. 修正コードをもらう
7. 3に戻る

これを何十回と繰り返しました。

### ファイルをダウンロードできるようになって感動した

以前はClaudeがコードを出力したら、アーティファクト（画面右側のコードブロック）から**手でコピペしてファイルを作る**必要がありました。

それがいつからかアップデートされて、**Claudeが直接ファイルを生成してダウンロードできるようになった**んです。

![](https://static.zenn.studio/user-upload/3d116bc6384a-20260331.png)

`main.py` とか `App.jsx` とか、ファイル名のついた状態でそのままダウンロードできる。地味に見えて、これが本当に快適でした。コピペのし忘れや貼り付け先の間違いがなくなって、開発のテンポが全然変わりました。

### GitHubもデプロイもClaudeに聞いた

GitHubは以前に一度使ったことがあるだけで、ほぼ覚えていない状態でした。「GitHubにアップロードするにはどうすればいいですか？」と聞いたら、こう教えてもらいました。

```
git add .
git commit -m "変更内容のメモ"
git push
```

「この3行を繰り返すだけです」と言われたとき、目からウロコでした。実際に今もこの3行しか打っていません。

VercelとRenderへのデプロイも、手順を一つひとつ教えてもらいながら進めました。エラーが出るたびにメッセージを貼り付けて、その都度対処法を教えてもらう。**「詰まったらそのまま貼る」これだけで全部解決できました。**

## できあがった機能を紹介

### 合成サイン波の生成

まず入力データとして、周波数・振幅・位相を自由に設定した合成サイン波を生成できます。

![](https://static.zenn.studio/user-upload/6e69c3bcc4cd-20260331.png)

### フィルタ処理（バターワースフィルタの例）

生成した波形にフィルタをかけられます。バターワース・チェビシェフ・楕円・ベッセル・FIRなど複数のフィルタに対応しています。

下の例は、50〜400Hzの9成分からなる合成波に対して、**バターワースローパスフィルタ（カットオフ200Hz）** をかけたものです。200Hz以上の成分がきれいに除去されているのがわかります。

![](https://static.zenn.studio/user-upload/5c6e7b6b3189-20260331.png)  
*フィルタ前*

![](https://static.zenn.studio/user-upload/7b8c969626ab-20260331.png)  
*フィルタ後*

### パワースペクトル

直接FFTで全サンプルを一括処理し、各周波数成分の振幅を正確に表示します。縦軸はPower・RMS・Peakの3種類で切り替えられます。

![](https://static.zenn.studio/user-upload/07d5067337a3-20260331.png)

### PSD（パワースペクトル密度）

Welch法による平均化でPSDを算出します。ランダム振動の解析などに使います。

![](https://static.zenn.studio/user-upload/3e6f4a6648ab-20260331.png)

### スペクトログラム

時間と周波数の両方を同時に見られるスペクトログラムも表示できます。周波数成分が時間とともにどう変化しているかを把握するのに便利です。

![](https://static.zenn.studio/user-upload/f6d13423c302-20260331.png)

## おわりに

「振動解析ツールって高くて、高性能すぎて使いにくい」という現場のリアルな課題から作り始めましたが、思った以上に本格的なものができました。

何より大きな収穫は、**プログラマーじゃなくてもAIと会話しながら本格的なWebアプリを作れる**ということを身をもって体験できたことです。

Claude Codeのような専用ツールがなくても、ブラウザ版のClaudeだけで、フロントエンドからバックエンド・デプロイまで全部できました。「詰まったらエラーをそのまま貼る」だけで、ほとんどの問題は解決できます。
