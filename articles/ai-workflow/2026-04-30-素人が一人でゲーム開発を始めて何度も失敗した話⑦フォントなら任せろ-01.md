今回は短め、ゲームに使うフォントの話

<font color = "cyan">「データ作成というボスは倒した！」
「あとは、完成度を上げていきますか」</font>と私は未だ勘違いしたままです、ご機嫌です

<font color = "cyan">「フォントも変えたらいい感じになるよね」
「WEBに使う際には別途使用条件があるフォントあるけどゲームだとどうなんだろ」</font>
と調べてみます
予想通り『ゲーム使用は別契約が必要』の記載が多いです

フリーフォント等ならOKなものもありますがイメージと合いません

仕事柄凝り性な上にフォントへの目が肥えてしまっています
フォント探しは難航しました

.

サブスク型のフォントサービスは『ゲーム使用は別契約が必要』が多く
<font color = "cyan">「これは個人開発には大げさすぎるなあ」</font>と思いました
切り替えて、買い切りのフォントを漁ると
こんな条件のものが多数
『ベクターデータとして再配布可能な状態でのゲーム使用不可』
<font color = "cyan">「つまり、ビットマップ化すればokか！」</font>

別の作業を挟む必要がありそう、
けど『完全にゲームでの使用禁止』というわけでないなら何とかなるでしょう
<font color = "cyan">「その作業も挑戦してみるか！それも経験か！」</font>と、イメージに合うフォントを購入！

.

<font color = "cyan">「けどそれってどうやるの？」
「まさか１文字ずつ変換するわけないし、そういうツールがあるのかな」</font>
ときくとAI<font color = "orange">「Unityでできます」</font>
<font color = "cyan">「え、そうなの」</font>
便利です、すごいですねUnity

教えてもらいながら、フォントアセット作成という作業を始めます
しかし、ここでまた<font color = "red">良くない病気が出ます</font>
<font color = "lime">「フォントはきれいならきれいなだけ良い」「適切な文字間は読み易さに必須」
「フォントは全体のイメージを左右する重要なもの」「妥協は許されない」</font>
グラフィック系の強い思想が溢れてきます

当然、すべて最高品質の設定を選択！
<font color = "red">作業開始！！！</font>

全っ然進みません、それはもう時間がかかります
進まない進度ゲージ、激しく回り続けるPCファン
<font color = "cyan">「このゲージ１本がいっぱいになったら終わるのかな」「時間かかるなあ」</font>
違います、<font color = "red">そのゲージは１０本近くあります</font>

私はその長さに驚きながらも、きれいなフォントを期待して
部屋が適度に暖かくなり<font color = "cyan">「春先の作業で良かったなあ」</font>と思いました

続きます
