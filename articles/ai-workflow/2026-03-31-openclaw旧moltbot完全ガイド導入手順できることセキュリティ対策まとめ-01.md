## OpenClawとは？

OpenClawは、**自分のPC上で動くオープンソースのAIエージェントプラットフォーム**です。

オーストリアの開発者Peter Steinbergerが2025年末に開発し、「Clawdbot」→「Moltbot」を経て、現在の名前に落ち着きました（AnthropicのAI「Claude」に似た名前だったため商標的な経緯で改名）。

ChatGPTやClaudeと何が違うのか？ 大きく3つのポイントがあります。

**① 実行能力がある** シェルコマンドの実行、ブラウザ操作、ファイル管理など、PCを「操作」できます。

**② 記憶が続く** セッションをまたいで会話内容やユーザー設定を記憶します。

**③ 自分から動く** 指示を待たずに、メール監視やスケジュール確認を自動実行します。

すべてのデータは自分のデバイスに保存され、クラウドにアップロードされません。プライバシーを守りながら強力な自動化機能を使えるのが最大の魅力です。

## どんなことができるの？

### 📅 スケジュール・メール管理

メール・カレンダー・ファイルにアクセスし、スケジュールを自律管理します。返信の下書き作成、会議の調整なども自動でこなします。

### 📝 コンテンツ作成の自動化

毎朝のデイリーノート自動生成、QiitaやnoteやZennへの記事下書き自動作成なども実績があります。

### 🔁 定期タスクの自動実行（cronジョブ）

たとえば「毎朝9時にタスクリストを確認して報告して」という指示を自動化できます。

### 💻 コーディング支援

コードの生成・レビュー・デバッグをチャット経由で依頼でき、ファイルの読み書きやGitHub操作も可能です。

### 🤖 自己改善

スキルと呼ばれる拡張機能を自分でインストールするだけでなく、AIが必要なスキルを自ら作成することもできます。

## 導入方法

### 必要なもの

### インストール手順

**ステップ1：インストール**

ターミナルで以下を実行します。

```
npm install -g openclaw@latest
openclaw onboard --install-daemon
```

スクリプトが自動でOSを判定し、必要なNode.jsのインストールからオンボーディングまで案内してくれます。

**ステップ2：管理画面を開く**

Gatewayが起動したら、ブラウザで以下にアクセスします。

```
http://127.0.0.1:18789/
```

**ステップ3：APIキーを設定する**

ClaudeまたはOpenAIのAPIキーを入力します。月間予算の上限設定も忘れずに（目安：Anthropic月5,000円、OpenAI月3,000円）。

**ステップ4：チャットアプリと連携する**

Discord・Telegram・WhatsAppなどのBotを作成し、発行されたトークンをOpenClawに設定します。

初めての場合はDiscordが設定しやすいです。

## カスタマイズ：エージェントの「個性」を作る

OpenClawのワークスペースには以下のファイルがあり、自由に編集できます。

```
~/.openclaw/workspace/
├── AGENTS.md   # エージェントの行動指針
├── SOUL.md     # エージェントのペルソナ設定
├── USER.md     # ユーザー情報
├── MEMORY.md   # 長期記憶
└── TOOLS.md    # ツール固有の設定
```

SOUL.md を編集すれば、エージェントの口調やキャラクターを自由に設定できます。たとえば「フレンドリーな秘書」「簡潔に答えるビジネスAI」など、自分好みの個性を与えられます。

## スキルの追加

機能を拡張するには「スキル」をインストールします。

```
# スキルを検索
openclaw skills search <キーワード>

# スキルをインストール
openclaw skills install @openclaw/github

# インストール済みスキル一覧
openclaw skills list
```

## セキュリティ：知っておくべきリスク

非常に強力なツールである一方、セキュリティリスクも存在します。導入前に必ず把握しておきましょう。

### ⚠️ 注意点①：広範な権限

メール・カレンダー・ファイルなど幅広い権限が必要なため、設定ミスや悪意あるスキルによる情報漏洩リスクがあります。

### ⚠️ 注意点②：MEMORY.mdの管理

過去の会話を記憶する MEMORY.md にはパスワード等の機密情報が含まれることがあります。定期的にリセットし、共有PCでの使用は避けましょう。

### ⚠️ 注意点③：スキルの信頼性

コミュニティ製スキルの中には悪意あるものが混入するケースも報告されています。インストールするスキルは慎重に選びましょう。

**安全に使うための基本原則：**

* 専用PC・VPSなど隔離された環境で運用する
* 最小権限の原則を守る
* スキルはコミュニティで評判の良いものだけ使う
