## はじめに：節約って、我慢することだと思っていた

節約というと「好きなものを買わない」「外食しない」「我慢する」というイメージでした。

でも、AIを使い始めてから気づいたのは、節約って「なんとなく使っているお金を見える化すること」だということ。

「なんとなく買っていたもの」「なんとなく続けていたサブスク」「なんとなく損していたこと」——それを整理するだけで、生活の質を下げずにお金が浮いてくる。

5歳の娘と3歳の息子を育てながらフルタイムで働く私が、AIと一緒に家計を見直したら、毎月1万円以上浮くようになりました。

この記事では、節約・ポイ活に使えるプロンプトを15個紹介します。

---

## 使う前に：AIはFPではありません

このカテゴリのプロンプトを使う際の大切なお断りです。AIが出す情報は参考情報です。具体的な金融商品・投資判断は専門家にご相談ください。

## 【前半】節約・ポイ活プロンプト①〜⑤

### プロンプト①：サブスクを棚卸しして削減する

```
サブスクを棚卸しして、削減できるものを見つけてください。

【現在のサブスク一覧】
[月額と一緒に書く。例：Netflix ¥1,490・Spotify ¥980・など]

【使用頻度（正直に）】
[例：Netflixは月2〜3回、Spotifyは毎日]

以下をお願いします。
①「すぐ解約していいもの」を見つけてください
②「頻度を下げて代替できるもの」を提案してください
③削減後の月額節約額を計算してください
④「これは残す価値がある」の判断基準
```

**使い方のポイント：** サブスクは「なんとなく続けている」ものが多い。正直に使用頻度を書くと、「これ実は要らなかった」が見つかります。

### プロンプト②：食費を無理なく削減する

```
食費を無理なく削減する方法を教えてください。

【現在の食費】
月約[○○]円（[○]人家族）

【削減が難しい理由】
[例：子どもの好みに合わせると食材が限られる、まとめ買いが難しい、外食が多い]

【目標】
月[○○]円以内に抑えたい

以下をお願いします。
①今すぐできる「食費削減の3ステップ」
②子どもがいても実践できる節約術5つ
③「食費を下げても満足度が下がらない」献立の考え方
④業務スーパー・コストコの上手な活用法
```

【現在持っているカード】
[例：楽天カード・Tカード・Pontaカード・イオンカード]

②使っていないカードの整理基準
③ポイントを無駄なく使い切る方法
④年会費のコスパ計算のやり方
```

```

### プロンプト⑤：通信費を見直す

```
通信費を見直して削減する方法を教えてください。

【現在の通信費】
・スマホ（私）：月約[○○]円・キャリア：[例：ドコモ]
・スマホ（夫）：月約[○○]円・キャリア：[例：au]
・自宅のネット：月約[○○]円

【状況】
・データ使用量：月約[○GB]
・家での使用が多いか・外出先が多いか：

以下をお願いします。
①削減できる可能性がある金額の目安
②格安SIMへの乗り換えのメリット・デメリット
③乗り換え時の手順と注意点
④家族全体で通信費を最適化する方法
```

**使い方のポイント：** 通信費は「見直すのが面倒」で放置されがちですが、年間で計算すると数万円の差が出ることも。まず現状を数字にすることから始めましょう。
