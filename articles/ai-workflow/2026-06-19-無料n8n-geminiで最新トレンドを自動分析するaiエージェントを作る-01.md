今回は **n8n・Gemini・Pollinations AI** を利用して、

✅ Googleニュースからトレンド収集  
✅ AIが話題性を分析

## 🤖 n8nとは？

![](https://static.zenn.studio/user-upload/deployed-images/7fb2b004114d1e6f60c8992d.png?sha=ecd0c442c11465b4db0670512b40a694d0d493ea)

n8n（エヌエイトエヌ）は、ノーコード・ローコードでワークフローを自動化できるオープンソースの自動化ツールです。

プログラミングをほとんど書かなくても、さまざまなサービスやAPIを連携させて処理を自動化できます。

例えば、

* フォーム送信時にSlackへ通知する
* RSSの更新を取得してSNSへ投稿する
* AIへ文章生成を依頼する
* データベースへ自動でデータを保存する

といった処理をGUI上で構築できます。

### 🧩 ワークフローの例

```
フォーム送信
      ↓
n8n
      ↓
OpenAI API
      ↓
生成結果を取得
      ↓
Slackへ通知
```

### 🚀 n8nの特徴

| 項目 | 内容 |
| --- | --- |
| オープンソース | 自分のサーバーへ構築可能 |
| ローコード | GUI中心でワークフロー作成可能 |
| API連携 | 外部サービスと簡単に接続可能 |
| AI連携 | OpenAIなどのAIサービスを利用可能 |
| 拡張性 | JavaScriptによるカスタマイズも可能 |

### 💡 なぜ人気なのか？

従来は自動化処理を作るためにプログラムを書く必要がありました。

n8nを利用すると、

```
トリガー
 ↓
データ取得
 ↓
データ加工
 ↓
通知
```

といった処理をノードとしてつなぐだけで構築できます。

そのため、業務効率化やAIエージェント開発の分野で広く利用されています。

## 🔧 Step8：JSONを整形する

## 🖼️ Step10：Pollinations AIで画像生成

Codeノードを2つ追加します。

![](https://static.zenn.studio/user-upload/deployed-images/695f57eef8ae7accd3e6de7a.jpeg?sha=c92b2c3eb60b5cd35b1b2bd5dc5293ed214a9bd1)

1つ目のcodeはstep9で生成したプロンプトを整形。

```
const text = $json.text;

const cleaned = text
  .replace(/^```json\s*/, '')
  .replace(/\s*```$/, '');

const data = JSON.parse(cleaned);

return [{
  json: data
}];
```

![](https://static.zenn.studio/user-upload/deployed-images/d2d6b1a275324487d7182a55.jpeg?sha=01da72e980044a52e08a64e36461a50d1f34cced)

}];
