もうちょっと丁寧に安全策を取るなら、カラーインデックス方式でn×nマス（端一列目のマスから0,1,2……です。自分が用意したドット絵と同じ縦横マス数）のドット絵を表示してもらってから、自分のドット絵とカラパレを渡して「置き換えてください」でもできなくもなかった。

  0: null,       // 白（背景・透明）
  1: "#23468C",  // 頭ワンポイント（紺）

  3: "#97E6BD",  // 目・明
  4: "#629359",  // 体・明

  6: "#3C6934",  // 体・暗

};
