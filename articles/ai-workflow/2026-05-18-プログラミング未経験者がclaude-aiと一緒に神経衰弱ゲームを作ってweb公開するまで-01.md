はじめに

プログラミングの経験がない私が、Claude AIのサポートを受けながら神経衰弱ゲームをゼロから作り、GitHub PagesとItch.ioで公開するまでの記録です。

「AIがあればプログラミング未経験でもゲームが作れるのか？」という疑問を持っている方の参考になれば嬉しいです。

完成したもの

🎮 **プレイはこちら**: <https://syg737usa.itch.io/memory-game>

* トランプ52枚を使った本格的な神経衰弱
* CPU対戦（難易度5段階）
* 12言語対応（日本語・英語・中国語など）
* ブラウザだけで動く（インストール不要）

使ったもの

* **Claude AI**（コーディングアシスタント）
* HTML / CSS / JavaScript（ゼロから学習）
* GitHub Pages（無料ホスティング）
* Itch.io（ゲーム公開プラットフォーム）

開発の流れ

1. まずシンプルなゲームを作る  
   最初はカードを並べてクリックできるだけの状態からスタート。Claudeに「神経衰弱ゲームを作りたい」と伝えるだけで、HTML/CSS/JavaScriptの基本構造を作ってもらいました。
2. バグとの戦い  
   一番苦労したのはカードの表示バグ。どのカードをめくっても「エース」しか表示されない問題が続きました。ClaudeとデバッグをしながらDOM操作の仕組みを少しずつ理解していきました。
3. デザインの改善  
   最初はシンプルすぎたので、カジノっぽい雰囲気にしたいとClaudeに相談。カード画像をプロ仕様のものに変更し、背景もダークグリーンのフェルト風にしました。
4. CPU AIの実装  
   難易度5段階のCPU対戦機能を追加。最高難度でもプレイヤーが勝てるよう、わざとミスをする確率（22%）を設定しています。
5. 多言語対応  
   12言語に対応するため、全テキストを言語オブジェクトで管理する仕組みをClaudeと一緒に設計しました。
6. Web公開  
   GitHubを使ったことがなかったのでゼロから学習。PowerShellでコマンドを実行しながらGitHub PagesとItch.ioに公開しました。

やってみて感じたこと

* プログラミング未経験でも**AIがあれば動くものが作れる**
* ただしAIに丸投げするだけでなく、**エラーの意味を理解しようとする姿勢**が大切
* 完成したときの達成感は本物

おわりに

このプロジェクトを通じて、HTML/CSS/JavaScriptの基礎、GitとGitHubの使い方、Web公開の流れを実践的に学べました。
