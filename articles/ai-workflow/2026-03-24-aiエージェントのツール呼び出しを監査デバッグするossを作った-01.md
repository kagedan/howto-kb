## 30秒で分かる agentwit

```
pip install agentwit

# MCPサーバーの前に挟むだけ
agentwit proxy --target http://localhost:3000 --port 8765
```

```
[agentwit] 14:32:01  tools/call  bash          HIGH ⚠  shell_exec
[agentwit] 14:32:03  tools/call  read_file     LOW  ✓
[agentwit] 14:32:05  tools/call  bash          CRITICAL 🚨 privilege_escalation
```

**AIエージェントが何のツールを呼んだか、リアルタイムで見える。**

接続先URLを1行変えるだけ。MCPサーバー側の改造不要。

---

## 何が嬉しいか

Claude / GPT / Gemini などのAIエージェントがMCPサーバーのツールを呼び出すとき、**何が起きているか見えない**。

```
AIエージェント
    ↓（ここが見えない）
MCPサーバー（bash / read_file / fetch ...）
```

agentwitはその間に透過プロキシとして入り、全通信を記録する。

```
AIエージェント
    ↓
agentwit  ← ログ記録・リスク判定・改ざん検知
    ↓
MCPサーバー（変更不要）
```

---

## 5分で動かす

```
# インストール
pip install agentwit

# プロキシ起動
agentwit proxy --target http://localhost:3000 --port 8765
# → エージェントの接続先を 8765 に変えるだけ

# 監査レポート生成
agentwit report --session ./witness_logs/SESSION_ID --format html

# ログの改ざん検証
agentwit verify --session ./witness_logs/SESSION_ID
# Chain integrity: VALID ✓
# Signature check: VALID ✓
```

## 主な機能

**ログ記録（v0.1.0〜）**

* SHA-256チェーンで改ざん検知
* ed25519署名で「誰が記録したか」を証明
* HTML / Markdown / JSON レポート自動生成

**リスク検知（v0.2.0〜）**

* `privilege_escalation` / `data_exfiltration` など11パターン
* プロンプトインジェクション検知（6パターン・CRITICAL/HIGH/MEDIUM）
* ツールスキーマの変更検知（追加・削除・改ざん）

**デバッグGUI（v0.3.0〜）**

* MCP Inspector — Tauri製デスクトップアプリ
* パラメータ編集・レスポンス表示・セッション比較

**監視・通知（v0.4.0〜）**

* 異常呼び出しパターンの検知（レート・連続呼び出し）
* Slack / Discord Webhook 通知
* Docker Compose で Grafana + Loki の監視スタックを一発起動

## LangChain との連携

```
from agentwit import AgentwitCallback

agent.run(
    "タスク内容",
    callbacks=[AgentwitCallback(output="./audit.json")]
)
# これだけで全ツール呼び出しが記録される
```

---

## なぜ作ったか

既存のMCPセキュリティツールは「ブロック型」がほとんどだ。

| ツール | ブロック | 改ざん検知ログ |
| --- | --- | --- |
| mcp-scan | ✅ | ❌ |
| Intercept | ✅ | ❌ |
| **agentwit** | **❌** | **✅** |

ブロックは事前対策。でも**インシデントが起きた後に「何があったか証明する」手段**が今はない。

agentwit はブロックしない。ただ全部記録する。
