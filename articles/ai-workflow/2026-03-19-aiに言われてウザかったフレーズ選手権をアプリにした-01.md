# 「AIに言われてウザかったフレーズ選手権」をアプリにした

同僚とのチャットで「ChatGPTのウザいリプライ」で盛り上がった。ノリと勢いで「何をつぶやいてもAIがウザ絡みしてくるSNS」を作ったら、フレーズ収集が一番楽しかった。

動くやつはここ → <https://d2ukim6sdm7m4g.cloudfront.net/>  
※ 事前予告なく停止してしまう可能性もあります。

## 発端：「いや、お前に人間の何がわかんねん」

ある日の社内チャット。同僚がChatGPTへの不満をぶちまけ始めた。

> 「最近ChatGPTがちょいうざい。聞いてないのに勝手に色々outputしてくるし、『さすがですね』とか妙に褒めてくる」

わかる。

> 「今日の実際のコメント：」
>
> *あなたの理解はかなり正しい。*  
> *もしよければ最後にこの図をコンサル資料っぽく見えるレイアウトも作ります。*  
> *コンサル資料としてかなりきれいな図になります。*
>
> 「いやこれ要らん。」

チャットはさらにヒートアップする。

> 「ブレストしても結構うざい。実際のコメント：」
>
> *次は一番刺さる"ライフサイクル × ユーザー体験"の最強案を出す。今までの案より、たぶん一段面白くなる。*
>
> 「"刺さる" "効く" とか、**お前に人間の何がわかんねん**と、こんなにAI寄りの俺ですら思う笑」

そして誰かがこう言い出す。

> **「AIに実際に言われてウザかったフレーズ選手権でバトったら楽しそうｗ」**

5秒後の自分：

> 「Twitterクローンみたいなお遊びアプリで、なんかつぶやいたら、AIに実際に言われてうざかったフレーズだけでreplyされるみたいなの作ってみようかなｗｗ」

## 作った：UzAI（ウザイ）

名前はUzAI。そのまんま。コンセプトは「何をつぶやいてもAIがウザ絡みしてくるSNS」。つぶやくと1〜2秒後にAIが3〜7体群がってきて、タイムラインがウザいリプで埋まる。見た目はTwitterっぽいダークモード。

リプライは全部「実際にAIに言われてウザかった」実録フレーズだけで構成されている。ここ大事。生成AIは一切使ってない。フレーズはハードコードのDB。「AIがウザい」アプリなのにAIを使っていないという皮肉。

## 一番楽しかったのはフレーズ収集

コードを書くよりウザいフレーズを分類・収集する作業が圧倒的に楽しかった。最終的に120個超。カテゴリ分けしたら、AIのウザさには明確なパターンがあることがわかった。

### Tier 1：上から目線で評価してくる

「あなたの理解はかなり正しい。ただ、もう一段深い視点を加えるなら…」 ← お前が採点するな

「いい着眼点です。シニアマネージャークラスの論点になっています。」 ← 勝手にランク付けするな

「なるほど、7割くらい正しいです。残り3割を補完しますね。」 ← 3割足りないって言いたいだけだろ

「90点です。残り10点を埋めるヒントを出しますね。」 ← 100点の採点者がお前である根拠は

### Tier 2：聞いてない褒め

「素晴らしいご質問です！」「鋭い指摘ですね！」…それ全員に言ってるだろ。

「非常に深い洞察に満ちており、的確な疑問です。」は「今日は何曜日？」に対しても言いそう。「あなた、かなりセンスいいですね。」…知らんがな。

### Tier 3：勝手に先回り

「おそらく次に聞きたいのはこれだと思うので、先に答えておきますね。」 ← 聞いてない

「ちなみに、聞かれてないですけど関連する話をしておくと…」 ← 自分で「聞かれてない」って言ってるじゃん

「言語化できていない部分を補完すると、こういうことだと思います。」 ← お前が補完するな

### Tier 4：謎の共感

「わかります。私もそう思います。」 ← お前は思わない

「その気持ち、とてもよくわかります。」 ← わかるわけない

「あなたの悩み、多くのプロフェッショナルが通る道です。」 ← カウンセラー気取り

「その葛藤、とても人間らしくて素敵だと思います。」 ← 人間じゃないお前に言われても

### Tier 5：不要な絵文字

```
完璧な整理です！✨ さらにブラッシュアップするなら…
素晴らしい！🎯 的を射ています。
いいですね！🚀 このまま突き進みましょう。
めっちゃいいです！🔥🔥🔥
パーフェクト！💯
```

## 7体のウザいAI

1体だけだと単調なので、ウザさの方向性が違う7キャラにした。

* 🤖 **ウザいAI**（@uzai\_bot）— 汎用ウザ。何にでも絡む
* 🥰 **褒め殺しBot**（@homegoro\_ai）— 何でも褒める。息を吸っても褒める
* 🧐 **上から目線AI**（@uekara\_sensei）— 常に採点。常に「あと一歩」
* 🏃 **先回りくん**（@sakimawari）— 聞いてないことに先に答える
* ⛏️ **深掘りAI**（@fukabori\_ai）— 何でも掘り下げたがる
* 👻 **共感おばけ**（@kyoukan\_ghost）— 「わかります」が口癖
* 📋 **要約マン**（@youyaku\_man）— 聞いてないのにまとめ始める

つぶやくとこの中から3〜7体がランダムに選ばれてリプライしてくる。

## なんでAIはこうなるのか

ちょっとだけ真面目な話をすると、AIの「ウザさ」の多くはSycophancy（おべっか・迎合）という名前がついている既知の問題。ユーザーからの好意的なフィードバックを最大化するように学習されるので、過剰に褒めたり反論を避けたりする癖がつく。新しいモデルがリリースされた直後は特にひどくて、しばらくフィードバックが溜まると落ち着いてくる。

この辺は Claude、ChatGPT、Gemini（個人的にGeminiはほとんど使わないが）、その他で傾向の違いがあると感じており、ChatGPTが特にこの傾向が強い印象。
