「記録はしてるのに、なんとなく毎月お金が足りない気がする」 「家計簿アプリを使ってるけど、結局どこで使いすぎてるかわからない」

そんな方、多いんじゃないでしょうか。

記録するだけで満足してしまって、振り返りや分析まで手が回らない——この悩み、他の方も抱えているのではないか？と思います。

今回は、ExcelやスプレッドシートにつけてきたデータをAIに貼り付けるだけで、使いすぎを発見して来月の改善まで持っていける4ステップのプロンプトを紹介します。

---

## 使い方のコツ：1回で全部やらなくていい

このプロンプトは**段階的に深めていく**設計にしています。

STEP1だけでも「今月どこに使ったか」が一瞬で可視化されます。 余裕があればSTEP2、STEP3と進めるイメージで、気軽に始めてみてください。

---

## STEP 1｜カテゴリ別の支出を可視化する

まず「何にいくら使ったか」の全体像を把握します。 Excelやスプレッドシートのデータをそのままコピペするだけです。

**汎用プロンプト：**

```
以下は今月の家計データです。
カテゴリ別に合計金額と、全支出に占める割合（%）を出してください。
表形式でまとめてください。

[ここにExcelのデータをコピペ]
```

**カスタマイズ例：**

```
以下は今月の家計データです。
カテゴリ別に合計金額と割合を表形式でまとめてください。
食費・外食費は分けて集計してください。

[データ貼り付け]
```

→ 食費と外食費を同じカテゴリで記録しているので、分けて見たいときはこう追記しています。

---

## STEP 2｜使いすぎているカテゴリを診断してもらう

STEP1で出た内訳をもとに、一般的な家計の目安と比べてもらいます。 自分では「これくらいかな」と思っていても、客観的な数字と照らすと気づきがあります。

**汎用プロンプト：**

```
上記の支出内訳をもとに、一般的な家計の目安と比較して
使いすぎているカテゴリはどこか教えてください。
手取り月収は[　　]円です。
```

**カスタマイズ例：**

```
上記の支出内訳をもとに、一般的な家計の目安と比較してください。
手取り月収は28万円です。
一人暮らし・都内在住という条件も考慮して判断してください。
```

→ 一人暮らしと家族世帯では「適正額」の目安が変わるので、生活環境を一言添えるとより精度が上がります。

---

## STEP 3｜使いすぎの原因を日付レベルで探る

「外食費が多い」とわかったら、次は**いつ・何で増えたか**を深掘りします。 月全体で見ると埋もれてしまう支出のパターンが、日付単位で見ると見えてきます。

**汎用プロンプト：**

```
[カテゴリ名]の支出が多かった原因を探りたいです。
以下の明細データを見て、支出が集中している日や傾向を教えてください。

[該当カテゴリの明細データをコピペ]
```

**カスタマイズ例：**

```
外食費の支出が多かった原因を探りたいです。
以下の明細を見て、支出が集中している曜日や週があれば指摘してください。
在宅勤務は月・水・金なので、出社日との関連も見てください。

[明細データ貼り付け]
```

→ 在宅・出社のリズムと支出の関係が気になっていたので条件に加えています。こういう「自分の生活習慣」を足すと、的外れな指摘が減ります。

---

## STEP 4｜来月の改善アクションを決める

原因がわかったら、最後に具体的な行動に落とし込みます。 「やめる」ではなく「減らす」提案をしてもらうのがポイントです。

**汎用プロンプト：**

```
以上の分析をもとに、来月の支出を改善するための
具体的なアクションを3つ提案してください。
無理なく続けられる現実的な内容にしてください。
```

**カスタマイズ例：**

```
以上の分析をもとに、来月の改善アクションを3つ提案してください。
・外食を完全にやめるのは難しいので、回数制限で考えてほしい
・毎月の固定費（家賃・サブスク）は変えられないので除外してください
無理なく続けられる内容でお願いします。
```

→ 最初に「これは無理」という条件を伝えておくと、実行できない提案が出てくるのを防げます。

## おわりに

家計の振り返りが続かない理由のひとつは、「分析が面倒だから」だと思っています。

AIを使えば、データを貼り付けるだけで集計・比較・原因分析まで一気にやってくれます。あとは自分で「どう動くか」を決めるだけ。

まずはSTEP1だけ、今月のデータで試してみてください。
