# 「ChatGPTにうまく答えてもらえない」を解決する5つのプロンプト技術【実例・コピペOK】

ChatGPT・Claudeを使い始めると、こんな悩みにぶつかりませんか？

- 「指示したのに的外れな回答が来た」
- 「長文が必要なのに短くまとめられた」
- 「もっと具体的な内容を出してほしいのに、なぜか抽象的」

実はこれ、プロンプトの「書き方の問題」でほぼ解決します。この記事では、**すぐ使える5つのプロンプト技術**を実例とともに紹介します。

---

## 技術1: 役割を最初に宣言する（Role Prompting）

**Before（悪い例）:**
```
マーケティングについて教えてください
```

**After（良い例）:**
```
あなたはB2B SaaS企業のマーケティングマネージャーです。
中小企業（従業員50名以下）向けにSaaSサービスを販売するための
リード獲得戦略を3つ、それぞれの実施コストと期待効果とともに教えてください。
```

**なぜ効くか:** AIに「誰として答えるか」を宣言すると、適切な視点・専門用語・粒度で答えてくれるようになります。

---

## 技術2: 出力形式を指定する（Format Specification）

「なんとなく長い文章が来る」問題は、出力形式の指定で解消します。

```
以下のトピックについて説明してください。

【トピック】: {説明してほしいこと}

出力形式:
- 1文サマリー（30字以内）
- 背景（3行）
- 具体的なメリット3つ（箇条書き）
- 注意点1つ
- 参考になるキーワード・書籍

専門用語は使わず、業界未経験者にもわかる言葉で書いてください。
```

---

## 技術3: 制約条件を先に書く（Constraints First）

**Before:**
```
新しいアプリのアイデアを考えてください
```

**After:**
```
以下の制約内で、新しいアプリのビジネスアイデアを5つ提案してください。

【制約】:
- 開発期間: 1人で1ヶ月以内
- 初期費用: 10万円以下
- マネタイズ: Stripeで決済可能
- ターゲット: フリーランスまたは中小企業
- NG: ハードウェア・在庫が必要なもの

各アイデアには「想定月収」「主な課題」「最短で収益化できる方法」を含めてください。
```

制約を先に書くことで、使い物にならないアイデアを最初から除外できます。

---

## 技術4: 段階的に深掘りする（Iterative Prompting）

一発で完璧な答えを求めず、**会話を続ける**のがコツです。

```
STEP 1:
「{テーマ}について、主要な論点を5つ挙げてください」

→ 回答を受け取る

STEP 2:
「その中で最も重要な論点を1つ選び、詳しく解説してください。
具体的な数字やデータがあれば含めてください」

→ 回答を受け取る

STEP 3:
「それを踏まえて、{自分の目的・立場}に応用するとしたら、
具体的に何をすべきですか？優先順に3つ教えてください」
```

「もっと具体的に」「別の角度から」「初心者向けに言い換えると」などの深掘りフレーズも有効です。

---

## 技術5: 「反論・欠点」を明示的に求める（Devil's Advocate）

AIはそのまま使うと「良い面ばかり」答える傾向があります。意思決定に使うなら、必ず反論も求めましょう。

```
以下のビジネスアイデアについて評価してください。

【アイデア】: {アイデアの概要}

以下の2つの視点でそれぞれ評価してください:

▼ 肯定的な観点（実現可能性・強み・市場性）
▼ 批判的な観点（課題・リスク・見落としている点）

最後に「このまま進めるべきか？改善してから進めるべきか？」を率直に判断してください。
```
