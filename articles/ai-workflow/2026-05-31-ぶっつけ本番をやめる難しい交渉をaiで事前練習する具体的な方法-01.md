こんにちは！

今日は「難しい交渉の前に、AIで練習しておく」という使い方について書いてみたいと思います。

値上げのお願い、契約更新の交渉、クレームへの対応——こういった「言いにくい話」をしなければならない場面、誰でも事前に「うまくいくかな」と不安になります。今日はその不安を、AIを使って少し和らげる方法をご紹介します。

## 「ぶっつけ本番」が一番リスクが高い

難しい交渉や話し合いの前、どんな準備をしていますか？

「なんとかなるだろう」とそのまま臨む方もいれば、「資料だけ用意した」という方もいると思います。でも実際には、事前に「相手がどう反応するか」「どう切り返すか」を想定しておくかどうかで、交渉の結果はかなり変わります。

営業の世界では「ロールプレイング」と呼ばれる練習法があります。同僚に「お客様役」をやってもらって、想定問答を繰り返す。これをやった人とやっていない人では、本番での動じ方が違います。

でも中小企業では「ロールプレイングをやってくれる同僚がいない」「時間が取れない」という現実があります。

そこにAIが使えます。AIに「相手役」をやってもらえば、一人でも事前練習ができます。

## AIに「相手役」をやってもらうとはどういうことか

難しい話ではありません。

「あなたは私の取引先の担当者です。私がこれから値上げのお願いをします。最初は難色を示してください」——こういう設定をAIに伝えると、AIがその役割を演じてくれます。

自分が値上げを切り出す。AIが「うーん、それは困りますね」と返してくる。そこでどう答えるかを考える。AIがさらに「他社はもっと安いですよ」と切り返してくる。それに対してどう説明するか——という形でやりとりを重ねられます。

本番前に「相手の反応」を体で覚えておくことができます。

## 具体的な使い方、3つの場面

### 場面① 取引先への値上げ交渉

```
あなたは私の会社の長年の取引先、
○○株式会社の購買担当・山田さんです。

これから私が電話で値上げのお願いをします。
以下の設定で、山田さんとして返答してください。

山田さんの設定：
・5年間の付き合いで、価格には敏感
・「なぜ今なのか」「他社は値上げしていないか」
　という点を気にしている
・すぐにはOKしないが、関係を壊したいわけでもない
・最終的には5〜7%の値上げなら受け入れる余地がある

私の状況：
・原材料費が昨年比15%上昇
・2年間自社で吸収してきたが限界
・今回は8%の値上げをお願いしたい

では始めます。「山田さん、いつもお世話になっております。
今日は少しご相談がありまして……」
```

このやりとりを数回繰り返すと、「どの言い方が相手に刺さるか」「どこで詰まりやすいか」が見えてきます。本番前に自分の弱点を発見できます。

練習が終わったら、こう続けて聞いてみてください。

```
今のやりとりを振り返って、
私の話し方で改善できる点があれば
具体的に教えてください。
```

「説明が長すぎる」「相手の不安に答える前に先に進んでしまっている」という客観的なフィードバックが返ってきます。

### 場面② 契約更新での条件変更交渉

契約更新のタイミングで「この条件を変えたい」という話をしなければならないケースも、事前練習が効きます。

```
あなたは私の会社の顧客、
△△株式会社の担当者・鈴木さんです。

これから私が契約更新の条件変更をお願いします。
鈴木さんとして返答してください。

鈴木さんの設定：
・3年契約で今月が更新月
・これまでの関係に大きな不満はない
・ただ予算が厳しく、コストアップには敏感
・「現状維持でいいのでは」という立場

私がお願いしたい変更：
・月額料金を15%引き上げたい
・代わりにサポート対応時間を平日9〜18時から
　平日8〜20時に拡充する

では始めます。「鈴木さん、今月が契約更新月ですね。
いくつかご相談がありまして……」
```

「値上げだけ伝えると怒られる」「拡充内容を先に話すと受け入れられやすい」という、伝える順番の感覚がつかめます。

### 場面③ 難しいクレームへの初期対応練習

クレームの電話は、感情的な相手に対してとっさに言葉を選ばなければならない場面です。事前に「こういう状況でどう返すか」を練習しておくと、本番での動じ方が変わります。

```
あなたは私の会社に強いクレームをいれてきた
お客様・佐藤さんです。

状況：
・先週納品した商品に不具合があった
・佐藤さんは「こんな会社に頼んだのが間違いだった」
　と怒っている
・でも長年の取引相手で、関係を壊したくはない

私がこれから電話で初期対応をします。
佐藤さんとして返答してください。
最初は感情的に、でも話を聞く余地はある状態で
お願いします。

では始めます。「佐藤様、この度は大変ご迷惑を
おかけしてしまい、誠に申し訳ございません……」
```

「謝罪の言葉が足りない」「状況確認が早すぎる」「相手の感情を受け止める言葉が抜けている」という気づきが、本番前に得られます。

## 練習の後に「想定問答集」を作っておく

シミュレーションが終わったら、こう頼むのがおすすめです。

```
今のシミュレーションをもとに、
実際の交渉でよく出そうな
「相手の質問・反論」と
「それに対する私の返し方」を
5パターンまとめてください。
```

「他社はもっと安いですよ」→「おっしゃる通りです。ただ、当社のサポート体制を含めると……」という形で、想定問答集が出てきます。

これを交渉前に読んでおくだけで、「何か言われても大丈夫」という安心感が生まれます。

## 一つだけ注意すること

AIのシミュレーションは「本番の完全な再現」ではありません。

実際の相手は、AIが想定した以上に強く出てくることもあれば、意外とすんなり受け入れることもあります。「AIでうまくいったから大丈夫」と過信せず、「よくある反応には対応できるようになった」という準備として使うのが正しい位置づけです。

また、相手の氏名・会社名・具体的な取引内容など、個人や企業が特定できる情報をAIに入力しないよう注意してください。架空の名前（「山田さん」「△△株式会社」）を使って練習する、という形で情報管理してください。

## というわけで

難しい交渉の前にAIで「相手役」をやってもらう使い方は、中小企業の実務でかなり使えるアイデアです。

「ロールプレイングをやってくれる相手がいない」「練習する時間が取れない」という問題を、一人でも・短時間でも解決できます。

まず今週、一番「憂鬱だな」と思っている交渉を一つ思い浮かべてください。その相手役をAIに頼んで、5分だけ練習してみてください。「こう来たらこう返せばいい」という感覚が一つ生まれるだけで、本番の日の気持ちが少し楽になります。
