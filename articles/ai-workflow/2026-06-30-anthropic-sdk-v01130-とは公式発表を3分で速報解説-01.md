## 🚀 Anthropic SDK v0.113.0 / sdk-v0.107.0 とは

きなこ（🦜）ともっちー（🐹）が最新 SDK アップデートを速報解説するよ！

---

**もっちー**: きなこ！Anthropic の SDK がアップデートされたって本当？

**きなこ**: そうなの。2026年6月29日に Python SDK と TypeScript SDK が同時リリースされたんだよ。

---

### ポイント

- **Web fetch & support tools 対応（API 20260318）** が Python / TypeScript 両方の SDK に追加
- Python v0.113.0：`async count_tokens` の `output_format` / `output_config` マージブロックのバグ修正
- TypeScript sdk-v0.107.0：agent-toolset のシンボリックリンク制限バグのセキュリティ修正 / `BatchCreateParams.Request.params` の型定義復元

---

**もっちー**: 「web fetch と support tools」って具体的には何なの？

**きなこ**: API バージョン `20260318` で追加されたツール群のこと。Claude が web 上のリソースにアクセスしたり、特定のサポートツールを使えるようにする機能なんだよ。SDK レベルで正式サポートされたことで、Python や TypeScript のコードから直接使えるようになったの。

---

**もっちー**: TypeScript の security fix って怖いやつ？

**きなこ**: agent-toolset がシンボリックリンクをたどる際のホップ数制限と、glob マッチ前にパスの閉じ込めチェックが甘かったというバグだよ。エージェントとして使うなら早めにアップデートしておいた方が安心だね。

---

### まとめ

| 項目 | Python v0.113.0 | TypeScript sdk-v0.107.0 |
|------|----------------|------------------------|
| リリース日 | 2026-06-29 | 2026-06-29 |
| 主要新機能 | Web fetch & support tools (API 20260318) | Web fetch & support tools (API 20260318) |
| バグ修正 | async count_tokens merge block | agent-toolset symlink security / BatchCreateParams 型 |

Claude の API を直接使っている開発者はぜひアップデートを確認してみてね！
