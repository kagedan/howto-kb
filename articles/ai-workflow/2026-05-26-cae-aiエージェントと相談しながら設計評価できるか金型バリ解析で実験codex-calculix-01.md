# CAE AIエージェントと相談しながら設計評価できるか？

CAEは設計評価に有効ですが、専門知識がないと最初の一歩が難しいです。解析ソフトの操作はもちろん難しいのですが、実機で想定している状態をFEMで扱える状態（境界条件・解析条件）に置き換えるのにも専門知識が必要です。この部分がAIが手伝ってくれるととても助かります。

そこで今回は、射出成形金型を題材に、AIエージェントと相談しながら解析・設計評価できるか試してみました。

ざっくりした依頼から始め、AIエージェントがどのような確認を返してくるのか、こちらが分かる範囲でどう答えるのか、面選択や荷重・拘束条件をどのように確認していくのかを見ていきます。

### 補足：拘束条件が不足しているのでは？

最初の説明では、「下型底面を完全固定し、上型上面に型締め力を入れる」と説明されました。図も分かりやすく、CAEに詳しくないユーザーには十分に見えます。

しかし、CAE視点では少し気になる点があります。上型をZ方向に押すだけなら、X/Y方向の剛体移動や回転はどう止めているのでしょうか。

そこで拘束条件についてAIに追加質問してみます。

プロンプト

```
上型は拘束してないのか。剛体移動してない？
```

すると、実際には上型上面に `Ux=Uy=0` の拘束が入っていることが分かりました。  
![](https://static.zenn.studio/user-upload/242eb96349c2-20260522.png)

AIはCAEの入力ファイルについても説明してくれましたが、ここで大事なのは、ユーザーがCAEの入力ファイルを読めることではありません。むしろ、AIエージェントの説明を聞いて、  
「上型は拘束していないの？」  
「剛体移動していないの？」  
と自然な疑問を投げ返せることです。

CAEに詳しくなくても、図を見ながら会話することで、解析条件の前提を少しずつ確認できます。今回も、最初の説明では省略されていた上型のX/Y拘束が、追加質問によって明らかになりました。

## まとめ：AIエージェントと相談しながら設計評価するということ

今回試したかったのは、AIエージェントが一発で正しいCAE解析をしてくれるかではありません。CAEに詳しくないユーザーでも、AIエージェントと相談しながら、

* 何を評価したいのか
* どの面に荷重を与えるのか
* どこを拘束するのか
* 結果のどこを見ればよいのか
* 分からない条件をどう仮定すればよいのか

を少しずつ整理できるかがポイントでした。

今回の例では、最初の依頼はかなり雑でしたが、AIエージェントは不足情報を確認し、PL面やキャビティ圧力面を画像で示し、荷重・拘束条件を説明し、解析結果をレポート化し、さらに結果の意味まで説明してくれました。

もちろん、これだけで実機のバリ発生を断定できるわけではありません。結果は最終判断ではなく、設計初期の一次評価として見るべきです。

それでも、専門知識が十分でないユーザーが、AIに質問しながら「どこを見ればよいか」「何が分かっていないか」「次に何を確認すべきか」を整理できるのは大きいと感じました。

さらに、分からないところを何度でも聞けるのもメリットです。説明が難しければ、もっと分かりやすく説明してもらうこともできます。  
会話しながら解析条件と結果の意味を理解していけるという点に、CAE AIエージェントを設計評価に使う面白さがあると思います。

## (付録) 解析計画全文

ここから先は付録です。  
CAEに慣れている人向けに、Codexが作成した解析計画の全文を載せておきます。

本文では「AIと相談しながら理解する流れ」を重視しましたが、CAE目線では、どのような仮定・境界条件・メッシュ方針・検証項目が出てきたのかも気になると思います。  
その確認用です。

> # Analysis Plan: ABS switch cover mold flash-risk evaluation
>
> ## 1. Objective
>
> `projects/Switch_Cover/geometry/cover_mold.step` の金型モデルを対象に、ABS 成形時のバリ発生リスクを CAE で評価する。
>
> 主評価量は、樹脂圧および型締め荷重に対するパーティング面の開き量、パーティング面の接触圧分布、金型の変形モード、局所応力である。最終判定は、予測される型開き量が設計上許容されるバリ隙間以下か、またパーティング面で局所的な接触圧抜けが起きないかで行う。
>
> ## 2. Physical Problem Definition
>
> * 対象: ABS 製スイッチカバーの成形用金型。
> * CAD: `projects/Switch_Cover/geometry/cover_mold.step`
> * 初期 STEP 確認結果:
>   + 3D volume 数: 2
>   + surface 数: 127
>   + 全体外形: 約 `150 mm x 200 mm x 80 mm`
>   + 上側ボリューム bbox: `X=-60..90`, `Y=-80..120`, `Z=4..46`
>   + 下側ボリューム bbox: `X=-60..90`, `Y=-80..120`, `Z=-34..9`
>   + `Z=4 mm` 近傍に上下金型の合わせ面が存在する。
> * 解析種別: 3D solid の静解析。初期計画では線形弾性のベースライン解析とし、必要に応じて接触非線形解析に拡張する。
> * バリ評価の物理解釈: キャビティ内の樹脂圧が金型を開く方向に作用し、型締め荷重とパーティング面接触で抵抗する。パーティング面で接触圧が消失し、隙間が ABS の流入可能隙間または設計許容隙間を超える領域をバリ懸念部とする。
>
> ## 3. Unit System
>
> N-mm-MPa 系を使用する。
>
> * Length: mm
> * Force: N
> * Pressure / stress: MPa = N/mm^2
> * Displacement / opening: mm
>
> ## 4. Real-World Setup and Modeling Interpretation
>
> 実際の射出成形機上では、型締め機構、取付板、タイバー剛性、金型取付ボルト、ガイドピン、エジェクタ側・固定側の拘束が金型開きに影響する。初期解析では、金型単体 STEP から読み取れる 2 ブロックを上型・下型として扱い、以下の段階でモデル化する。
>
> 1. ベースライン線形静解析:
>    * 上下型を別ボディとして扱う。
>    * パーティング面に開き方向の荷重を与え、下型の機械取付面を拘束し、上型側に型締め荷重または対向拘束を与える。
>    * 接触を省略または簡略化し、型開きしやすい保守側の変形量を把握する。
> 2. 接触確認解析:
>    * 上下型のパーティング面に surface-to-surface contact を定義する。
>    * 型締め荷重、樹脂圧、必要なら摩擦を入れ、接触圧ゼロ領域と opening を評価する。
> 3. 必要時の部分詳細解析:
>    * ゲート、薄肉端、パーティングライン段差、リブ端、コーナー周辺など、バリ懸念部だけを局所細分化して再評価する。
>
> ## 5. Geometry Assumptions
>
> * STEP の 2 volume を上型・下型として扱う。
> * `Z=4 mm` 近傍の大面積水平面を主パーティング面とする。
> * バリ懸念箇所は PL 面全周とする。
> * ランナー、ゲート、スプルー、エジェクタピン、インサート、ボルト、ガイド部品、成形機プラテンは STEP に含まれていない可能性がある。含まれていない場合、初期解析では等価荷重・等価拘束で扱う。
> * 微小な面、フィレット、文字、意匠の細部は、パーティング面開きや局所バリに効く場合を除いてメッシュ品質優先で保持または簡略化を判断する。
>
> ## 6. Material Model
>
> 初期解析の金型材料は S50C 相当の線形弾性・等方性とする。
>
> * Young's modulus: `205000 MPa`
> * Poisson's ratio: `0.30`
>
> 入れ子はなしとする。ABS 樹脂は構造要素としては直接モデル化せず、キャビティ壁面への圧力荷重として扱う。樹脂流動、冷却収縮、粘弾性、固化過程は本計画の初期 CalculiX 解析では扱わない。
>
> ## 7. Element Type Candidates and Final Selection
>
> 候補:
>
> * 3D solid: 金型は厚みを持つ複雑形状で、パーティング面の面外開きと局所応力を評価するため適切。
> * Shell / beam: 金型ブロック、キャビティ、合わせ面の 3D 剛性を表せないため不採用。
> * Plane strain / axisymmetric: 形状と荷重が 3D であり不採用。
>
> 最終選定:
>
> * 初期メッシュ: 二次四面体 solid `C3D10`
> * 理由: STEP 由来の複雑なキャビティ・合わせ面を自動四面体メッシュで扱いやすく、一次四面体 `C3D4` より曲げ変形と応力勾配を表しやすい。
> * 代替: 接触設定や面セット抽出の診断には `C3D4` を使ってもよいが、最終評価は `C3D10` に戻す。
> * 接触解析で収束が悪い場合は、接触面近傍のみメッシュを整え、荷重ステップ分割、接触剛性、拘束条件を見直す。
>
> ## 8. Boundary Conditions
>
> 初期案:
>
> * 下型の成形機取付面または下面を `Ux=Uy=Uz=0` で拘束する。
> * 上型には型締め方向の圧縮荷重 `50 tonf = 490332.5 N` を与える。
> * 剛体運動防止のため、過拘束にならない最小限の横方向拘束を追加する。
>
> 現実寄せ案:
>
> * 取付板やプラテンの剛性影響が重要な場合、上下面に reference point と distributing coupling を設定し、型締め荷重を reference point に与える。
> * ボルト締結・ガイドピンは現時点ではモデル化しない。必要になった場合に等価ばね、拘束、または詳細部品として追加する。
>
> ## 9. Loading Method
>
> 樹脂圧:
>
> * キャビティ面の法線方向に射出・保圧時の圧力を与える。
> * 最大リスクは通常、キャビティ内圧が高く、型締め力に対して型開き方向の合力が大きい時点とする。
> * キャビティ内圧の実測値がないため、以下の 3 ケースでパラメトリック評価する。
>
> 型締め荷重:
>
> * 標準条件として `50 tonf = 490332.5 N` を与える。
> * 型締め荷重は上型の取付面または reference point に作用させる。
>
> 判定用に、樹脂圧と型締め力の組み合わせを複数ケース化する。
>
> 初期接触方針:
>
> * 上型と下型の主パーティング面に面-面接触を定義する。
> * 法線方向: hard contact 相当。
> * 接線方向: 初期は摩擦なし、必要に応じて摩擦係数を追加する。
> * 出力対象:
>   + 接触圧
>   + 接触状態
>   + パーティング面 opening / 相対変位
>
> CalculiX の接触収束が不安定な場合は、まずパーティング面を同定する診断モデルと、開き量を保守的に見積もる線形モデルで評価軸を固める。
>
> ## 11. Mesh Strategy
>
> * Gmsh Python API で STEP を読み込み、volume ごとに上型・下型 element set を作成する。
> * 主パーティング面、キャビティ面、ゲート近傍、薄肉端、角部は局所細分化する。
> * 初期代表メッシュサイズ:
>   + 全体: `5 mm`
>   + パーティング面・キャビティ面: `1.5 to 2.5 mm`
>   + バリ懸念の局所端部: `0.5 to 1.0 mm`
> * メッシュ収束確認:
>   + パーティング面最大 opening
>   + 接触圧ゼロ領域の面積
>   + 代表断面の変形モード
>   + 反力・荷重釣り合い
>
> ## 12. Expected Result Quantities
>
> * パーティング面最大 opening `g_max`
> * パーティング面 opening 分布
> * 接触圧最小値、接触圧ゼロ領域、接触圧分布
> * 樹脂圧による開き方向合力
> * 型締め反力と荷重釣り合い
> * 金型全体変位、上型・下型の相対変位
> * 金型 von Mises 応力
> * バリ懸念マップ:
>   + `opening > 0.01 mm`
>   + `opening > 0.02 mm`
>   + `opening > 0.05 mm`
>   + `contact pressure <= 0`
>   + ゲート・薄肉端・パーティングライン端部の局所開き
>
> ## 13. Validation / Sanity Checks
>
> * STEP が 2 volume として読み込め、上型・下型が正しく分類されること。
> * `Z=4 mm` 近傍の PL 面が全周の評価対象として正しく抽出されること。
> * キャビティ圧力を与える面が、製品キャビティ側の面であること。
> * 樹脂圧による開き方向合力が、投影面積と圧力から概算される値と整合すること。
> * 型締め反力と外力が釣り合うこと。
> * 変形図で型開き方向が物理的に妥当であること。
> * 接触圧が過拘束や面法線の誤認による不自然な分布になっていないこと。
> * メッシュ細分化で最大 opening と接触圧ゼロ領域が大きく変わらないこと。
>
> ## 14. Known Limitations
>
> * 樹脂流動解析ではないため、圧力履歴、流動末端圧、温度、固化、保圧切替、収縮は直接評価しない。
> * 金型取付状態、成形機剛性、ボルト、ガイドピン、入れ子、実際の面当たりが不明な場合、境界条件は概算になる。
> * パーティング面の初期面精度、摩耗、汚れ、ベント、逃げ、表面粗さは CAD だけでは評価できない。
> * バリ発生の許容隙間は ABS グレード、樹脂温度、圧力、保持時間、パーティング面温度で変化する。今回は設計基準未提示のため `0.01 mm`, `0.02 mm`, `0.05 mm` の 3 しきい値でリスクマップ化する。
>
> ## 15. Confirmed Conditions and Remaining Assumptions
>
> Confirmed:
>
> 1. バリ懸念箇所: PL 面全周。
> 2. PL 面: CAD の `Z=4 mm` 近傍。
> 3. キャビティ内圧: 実測値なし。`30 MPa`, `50 MPa`, `70 MPa` の 3 ケース。
> 4. 型締め力: 標準条件 `50 tonf = 490332.5 N`。
> 5. 金型材料: S50C 相当。
> 6. 入れ子: なし。
> 7. ボルト、ガイド: 現時点ではモデル化しない。
> 8. 許容バリ高さ: `0.01 mm`, `0.02 mm`, `0.05 mm` の 3 しきい値で評価。
>
> Remaining assumptions:
>
> 1. キャビティ圧力分布は、各ケースで対象キャビティ面に一様圧力として与える。
> 2. 成形機プラテン、ボルト、ガイド、金型面当たりの影響は含めない。
> 3. 樹脂温度、金型温度、保圧履歴、流動末端圧の時刻変化は含めない。
>
> ## Veteran CAE Review and Plan Revision
>
> レビュー観点:
>
> * バリ評価で最も重要なのは応力値ではなく、樹脂圧でパーティング面が開くかどうかである。したがって、結果の主軸を von Mises 応力ではなく opening と接触圧に置く。
> * CAD 単体だけで高精度判定を行うのは危険である。型締め力、実接触、成形機剛性、樹脂圧の不確かさが支配的になりうるため、まずパラメトリックなリスク評価として計画する。
> * 接触非線形解析は面抽出・法線・初期干渉・拘束条件に敏感である。最初から接触解析だけに依存せず、線形の保守側開き評価で荷重方向と変形モードを検証する。
> * 許容隙間は `0.01 mm`, `0.02 mm`, `0.05 mm` の 3 しきい値で、単一の合否判定ではなく段階的なリスクマップを出す。
>
> 計画への反映:
>
> * 解析は「線形ベースライン」から「接触確認」へ段階化した。
> * 未確定条件を仮定で固定せず、パラメトリック条件または実行前確認事項として分離した。
> * 最終レポートでは、荷重釣り合い、パーティング面 opening 図、接触圧図、バリリスクマップを必須図とする。
