# 解決(?)策

Geminiに聞いてみた (AIのことをAIに聞くのか...) ところ、QUICに原因があることが判明。確かに、両方とも**ERR_QUIC_PROTOCOL_ERROR**って書いてありますね。
ということで、**一時的にQUICを無効化する**ことにしました。

## QUIC無効化の手順

### Chromeの場合

以下にアクセス。

```
chrome://flags/#enable-quic
```

**Experimental QUIC protocol**をDisableに変更してChromeを再起動。 (再起動といっても右下に出るボタンを押すと、すぐ閉じてすぐ起動します。タブの状態も同じままでした。)

![スクリーンショット 2026-06-18 202500.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/4450676/7305c0c9-3a1d-4d3d-8971-e21484173be0.png)

再度アクセスしたところ、Claudeにアクセス成功しました。
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/4450676/77b89dec-0e74-4b0d-bde6-f843dd2ec65a.png)

Claudeを使い終わったら、一応、**Default**に戻す。

### Edgeの場合

以下にアクセス。

```
edge://flags/#enable-quic
```

**Experimental QUIC protocol**をDisableに変更してEdgeを再起動。
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/4450676/3cd147d1-a69a-4bf1-8b3b-87281576117c.png)

再度アクセスしたところ、Claudeにアクセス成功しました。
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/4450676/72989d4e-0a8f-4d11-aa38-efac9d4b2279.png)

Claudeを使い終わったら、一応、**Default**に戻す。
