## 1. 使用するもの

* Windows 11
* Python 3.13.13
* Ansys Student 2026 R1 (Windows用)
* PyAnsys
* Gemini CLI
* Gemini 3 (CLI経由でAutoモードで利用）
* Git 2.53.0
* Visual Studio Code（必須ではない）

## 2. インストール

### 2.1. Ansys

世界を代表するCAEツールの1つ。インストーラをダウンロードしてインストールするだけなので簡単。Student版は教育・個人学習用で、モデル規模に制限があります。  
<https://www.ansys.com/ja-jp/academic/students/ansys-student>

### 2.2. Git

Gitは世界で最も人気のあるバージョン管理システムです。CAEユーザーにはあまり縁がないかもしれませんが、AIエージェントを使うときには便利なので入れておきます。  
以下のページからインストーラを入手してインストールします。  
<https://git-scm.com/install/windows>  
すでにGitをインストール済の場合は、適宜`git update-git-for-windows`で更新しておきます。

### 2.3. PyAnsys

AnsysツールをPython側から動かすためのOSS（オープンソースソフトウェア）です。  
<https://docs.pyansys.com/>  
各種Ansysツールによりモジュールが分かれており、ここではPyMAPDLとPyDPF CoreとPyDPF Postを利用します。これらを仮想環境にインストールします。作業ディレクトリを作成して以下のコマンドを実行。

```
uv venv .venv
uv pip install ansys-mapdl-core
uv pip install ansys-dpf-core
uv pip install ansys-dpf-post
```

また、グラフィックス関連の機能も必要なのでインストールしておきます。

```
uv pip install ansys-mapdl-core[graphics]
```

あとでGemini CLI側からこれらのリポジトリを参照して動かしたいため、作業フォルダにgit cloneしておきます。

```
git clone https://github.com/ansys/pymapdl.git
git clone https://github.com/ansys/pydpf-core.git
git clone https://github.com/ansys/pydpf-post.git
```

### 2.4. Gemini CLI

Gemini CLIはGoogleのAI「Gemini」をターミナル（コマンドライン）から直接操作できるオープンソースのAIエージェントツールです。以下のページを参照してインストールしておきます。  
<https://geminicli.com/docs/get-started/>

## 3. 動作確認

最初に個別のモジュールの動作確認を行い、最後にGemini CLI側から動作確認します。

### 3.1. PyMAPDLモジュールの動作確認

作業ディレクトリでPythonを起動し以下のコマンドを実行。

```
from ansys.mapdl.core import launch_mapdl
mapdl = launch_mapdl()
print(mapdl)
```

以下のような応答が返ってくればOKです。

```
Mapdl
-----
PyMAPDL Version:     0.72.1
Interface:           grpc
Product:             Ansys Mechanical Enterprise Academic Student
MAPDL Version:       26.1
```

このコマンドを実行することでAnsys MAPDLがBatchモードで起動していますので、確認が終わったら`mapdl.exit()`と入力して終了しておきます。

### 3.6. Gemini CLIからの実行確認

Gemini CLIを起動して以下のようなプロンプトを打ちます。ここではMAPDL用の作業ディレクトリとして`mapdl_work`フォルダを作成し、その中に`3d_plate_thermal.py`を置いてます。

```
mapdl_work/3d_plate_thermal.py を実行して結果を報告して。.venvを利用すること。
```

途中、先ほどのメッシュ図や温度分布図が表示されるので手動で閉じる必要があります。  
最終的に、下記のような出力がGemini CLIから返ってくれば動作確認完了です！
