# ChatGPTに断られた質問を、AIエージェントに自律課金させて答えさせた

## はじめに

「最新の情報へのアクセスはできません」——ChatGPTやClaude単体に聞くと返ってくるあの一言。

これを突破するために、**AIエージェントが自分でAPIを探して、USDCで課金して、リアルタイムデータを取得する**仕組みを90秒で作れるスターターキットを公開しました。

```bash
npx create-lemon-agent my-agent
```

![デモ](https://raw.githubusercontent.com/evidai/create-lemon-agent/main/demo.gif)

---

## 何が起きているのか

通常のAI（ChatGPT / Claude）はインターネットにアクセスできません。  
「SOLの今の価格は？」と聞いても、「最新情報へのアクセスはできません」と返ってきます。

このスターターキットを使うと、エージェントが次のことを**自律的に**やります：

1. 使えるAPIの一覧を取得（`list_services`）
2. 適切なAPIを選んで呼び出す（`call_service`）
3. **USDCで自動決済**してリアルタイムデータを取得
4. 回答を生成して返す
5. 課金をfreee / Money Forwardに自動仕訳（連携している場合）

人間が何かする必要はありません。

---

## セットアップ（90秒）

```bash
# 1. スキャフォールド
npx create-lemon-agent my-agent
cd my-agent
npm install

# 2. トークンを設定
cp .env.example .env
# .env に LEMON_CAKE_PAY_TOKEN を貼り付け
# → https://lemoncake.xyz/dashboard で発行（無料）

# 3. 実行
npm start "SOLの今の価格・過去7日のトレンド・関連ニュースTop3をまとめて"
```

## エージェントのコード（抜粋）

```typescript
for await (const message of query({
  prompt: userPrompt,
  options: {
    model: "claude-haiku-4-5-20251001",
    mcpServers: {
      "lemon-cake": {
        command: "npx",
        args: ["-y", "lemon-cake-mcp"],
        env: { LEMON_CAKE_PAY_TOKEN: PAY_TOKEN },
      },
    },
  },
})) {
  // メッセージを処理
}
```

MCP サーバーを1つ渡すだけで、ClaudeがAPIの発見・選択・決済・呼び出しを全部やってくれます。

| 会計連携 | freee / Money Forward（オプション） |

---

## なぜUSDCで払うのか

- **APIキーの管理が不要。** 各APIのアカウントを作る必要がありません
- **1回ごとの従量課金。** 月額固定ではなく使った分だけ
- **自動仕訳。** freeeやMoney Forwardと連携すると、APIコスト1件1件が経費として記録されます（インボイス対応）
