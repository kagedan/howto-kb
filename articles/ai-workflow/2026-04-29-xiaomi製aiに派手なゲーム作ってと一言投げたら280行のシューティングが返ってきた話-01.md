最近のAI界隈、もう本当についていくのが大変だ。

GPT、Claude、Gemini、DeepSeek、Qwen——ほぼ毎週のように新しいモデルが出てくる中で、つい先日、また「これは知っておかないとマズい」やつが出てきた。

**Xiaomi(シャオミ)の MiMo V2.5**。

スマホやEVを作っているあのXiaomiが、実は1年以上前からAIモデルを開発していて、その最新版が4月末に公開された。しかも **MITライセンスでフルオープンソース、商用利用OK**、コンテキストは **1Mトークン**、エージェント性能は **Claude Opus 4.6 に迫る** と公式は言っている。日本ではまだほとんど話題になっていない。

正直、半信半疑だった。  
だから、いつものやり方で試した。

「短いコードで派手なゲームを作ってください」

——投げたのは、これだけ。

## 出てきたのが、これだった

タイトル: **NEON BLASTER**

ネオン全開の縦スクロールシューティングである。マウスで自機が滑らかに追従し、撃ち続け、敵を倒すとパーティクルが爆ぜ、コンボ倍率が跳ね上がり、Wave 3 からは星型のボスが HP バー付きで弾を撃ちながら降ってくる。スペースキーで全敵消滅のボム。スマホタッチ対応。

仕様だけ並べると、こんな感じだ。

これ、**全部で約280行**である。HTMLファイル1個、12KB。

開いて中身を見て、ちょっと笑った。書き方が綺麗なのだ。`pPow`(パワー段階)に応じた弾の分岐、フレームカウンタによるWave管理、無敵時間の点滅処理、ボスの六芒星を三角関数で描く処理、すべて素直に読める。「派手なゲーム」という曖昧な一言から、ここまで仕様を補完してきたのは、正直、想像以上だった。

## 弱点も正直に

純粋な創造的執筆(小説・キャッチコピー・ライティング)は、まだ Claude Opus ほどではない、という声がある。エージェント・コーディング・長文脈タスクに振り切ってチューニングされているので、**用途がハマれば最強、外れると普通**、というクセは確かにある。

リリースから数日というタイミングなので、**実運用での真価**はこれからさらに固まっていく段階だ。

### おまけ【参考】サンプルコード

**ゲーム内容**   
NEON BLASTER　/　ネオン全開の縦スクロールシューティングゲーム

**操作説明**

1. マウス移動 自機の移動
2. スペースキー ボム（全敵消滅）
3. タッチ スマホ対応

**主な特徴**

* 3段階パワーアップ
* 撃ち方が単発 → 連射 → 3WAYに進化 3種の敵
* 小型(ピンク)、中型(黄色・回転)、ボス(マゼンタ・星型・射撃あり) コンボシステム
* 連続撃破でスコア倍率（最大x20）、色が段階的に変化 ボス戦
* Wave 3以降登場、HPバー付き、弾を撃ってくる パーティクル爆発 + 画面シェイク + スキャンライン + ビネット Web Audio API で効果音をリアルタイム生成 パワーアップドロップ
* 敵撃破時に確率で出現（P=武器強化、+=ライフ回復）
* 全コード約 280行 でこの表現量
