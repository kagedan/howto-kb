If you are building AI agents that need to improve through experience, study ART by OpenPipe.

Instead of relying only on prompt engineering, ART lets you train multi-step agents with reinforcement learning using GRPO. It supports tool use, multi-turn conversations, and iterative learning from rewards.

What you will learn:
🧠 Reinforcement Learning for LLMs
🔄 GRPO (Group Relative Policy Optimization)
🤖 Multi-step Agent Training
🛠️ Tool Calling & Function Execution
💬 Multi-turn Conversations
🏆 Reward Functions & RULER (LLM-as-a-Judge)
📉 LoRA Fine-tuning
⚡ vLLM-based Inference
📊 Trajectory Collection & Evaluation
🚀 GPU-efficient RL Training
📈 Agent Benchmarking
