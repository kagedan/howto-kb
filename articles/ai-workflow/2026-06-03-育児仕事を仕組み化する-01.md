「今日も仕事が終わらなかった…」

そう思いながら、  
子どものお迎えに  
走ったことはありませんか？

育児も仕事も、  
どちらも大切。

でも時間は有限で、  
どちらかを優先すると  
もう一方が崩れていく。

「私、どっちも中途半端だ」

そんな罪悪感を  
毎日抱えている人は、  
決して少なくありません。

でも、それは  
あなたの能力の問題でも、  
努力が足りないせいでも  
ありません。

**「両立できる仕組み」を  
持っていなかっただけです。**

---

この記事を読むと、  
こんな変化が起きます。

✅ **致命的ミスの回避**  
　→ 「今日も何もできなかった」  
　　という消耗の連鎖を断ち切れる

✅ **冷静な判断力**  
　→ 優先順位が整理され、  
　　「今やるべきこと」だけに  
　　集中できるようになる

✅ **機会損失の解消**  
　→ 育児も仕事も  
　　「どちらかを我慢する」  
　　から解放される

---

AIは「あなた専用のスケジュール設計士」であり、  
「家族全員が動きやすい仕組みを  
一緒に考えてくれる参謀」です。

今日からあなたの  
毎日の回し方は、  
根本から変わります。

## 基本のプロンプト

まず、例え話からはじめます。

「育児と仕事の両立を  
AIに手伝わせる」とは、  
どういうことか。

---

育児と仕事のピースを  
一つの机の上に  
広げているとします。

ピースの数が多すぎて、  
どこから手をつければいいか  
わからない状態。

でも、全体の完成図を  
先に見せてもらえたら、  
どこに何を置けばいいか  
すぐにわかります。

AIはその「完成図」を  
一緒に作ってくれる存在です。

あなたの生活リズムや  
家族の状況を伝えるほど、  
「あなたの家専用のパズルの解き方」を  
提案してくれます。

### ◆1週間スケジュール最適化プロンプト

**✅目的：**  
　育児・仕事・家事・休息を  
　無理なく1週間に収める  
　スケジュールを設計する。

**✅いつ使うか：**  
　週の始めに、  
　その週の動き方を整理したいとき。  
　または「毎週同じところで詰まる」  
　と感じているとき。

```
# ロール設定
あなたは育児と仕事を両立する人を
専門にサポートする
タイムマネジメントコーチです。
現実的で無理のない
スケジュール設計を行ってください。

# 背景・目的・対象
育児と仕事を同時に抱える
パパ・ママが対象です。
完璧を求めず、
「ちょうどいい両立」を
目指したプランにしてください。

# 指示内容
以下の【入力データ】をもとに、
月曜〜日曜の1週間スケジュールを
設計してください。
「育児時間」「仕事時間」
「家事時間」「自分の時間」の
4つのブロックに分けて整理すること。

# 制約条件
・「自分の時間」を
　必ず週に最低2コマ確保する
・詰め込みすぎない
　（余白を意図的に作る）
・パートナーや家族が
　動きやすい設計にする

# 出力形式
【1週間スケジュール案】
月曜：〜
火曜：〜
…（日曜まで）

【スケジュールのポイント】
・〜

【余白として空けたタイミングと理由】
〜

# 入力データ
・仕事の形態：
　（例：在宅／出社／混合）
・勤務時間帯：
　（例：9時〜17時）
・子どもの人数・年齢：
　（例：3歳と6歳の2人）
・保育園・学校の時間：
　（例：8時〜18時）
・パートナーの協力度：
　（例：平日は難しい・土日は動ける）
・最も困っていること：
　（例：夕方〜寝かしつけまでが
　　毎日バタバタする）
```

**✅具体的な使用方法：**  
　「入力データ」に  
　今週の状況をそのまま書くだけ。

　書けない項目は空欄でも  
　構いません。

　「今週特に困ること」を  
　一言添えると、  
　より精度の高い提案が届きます。

**✅読者への一言：**  
　スケジュールは「完璧に守るもの」  
　ではありません。

　「迷ったときに戻る地図」です。  
　まずここから作ってみてください。

### ◆家族の役割分担設計プロンプト

**✅目的：**  
　家族全員が動きやすい  
　役割分担を設計し、  
　「なんで私ばかり」を解消する。

**✅いつ使うか：**  
　パートナーとの家事・育児の  
　分担がうまくいっていないと  
　感じているとき。

```
# ロール設定
あなたは家族関係と
タスクマネジメントの専門家です。
家族全員が納得できる
役割分担を設計してください。

# 背景・目的・対象
共働きや育児中の家庭が対象です。
特定の人に負担が集中しない、
持続可能な分担を作ることを
最優先にしてください。

# 指示内容
以下の【入力データ】をもとに、
家族それぞれの得意・不得意と
時間的余裕を踏まえた
役割分担表を作成してください。
「固定タスク」と
「週次タスク」に分けて整理すること。

# 制約条件
・誰かの負担が突出しないよう
　バランスを意識する
・子どもが担える簡単なタスクも
　積極的に含める
・「やらなくていいタスク」も
　提案する

# 出力形式
【家族別・役割分担表】
（名前）：固定タスク〜 週次タスク〜

【子どもに任せられるタスク】
〜

【思い切ってやめていいタスク】
〜

# 入力データ
・家族構成：
　（例：夫・妻・子3歳・子6歳）
・各自の仕事状況：
　（例：夫は出社・妻は在宅）
・現在の分担状況：
　（例：家事育児はほぼ妻が担当）
・パートナーが得意なこと：
　（例：料理・買い物）
・最も「なんで私が」と感じる場面：
　（例：寝かしつけがいつも自分だけ）
```

**✅具体的な使用方法：**  
　「最も『なんで私が』と感じる場面」に  
　正直な気持ちを書いてください。

　AIはそこを優先的に  
　解消する分担を  
　提案してくれます。

**✅読者への一言：**  
　分担は「交渉」ではなく  
　「設計」です。

　感情的になる前に、  
　AIと一緒に整理してみると  
　話し合いがスムーズになります。

## 実践プロンプト集

## 1. 朝の「バタバタ」を設計で消す

毎朝、出発前に  
どこかで必ず詰まる。

「あれどこ？」  
「早くして！」  
「今日のご飯なに？」

このバタバタは  
「気合い」では解決しません。

**仕組みで消すものです。**

---

### ◆朝ルーティン自動化プロンプト

**✅目的：**  
　朝の準備を仕組み化し、  
　怒鳴らなくても  
　スムーズに動ける朝を作る。

**✅いつ使うか：**  
　朝の出発前が毎回バタバタして  
　消耗しているとき。

```
# ロール設定
あなたは育児家庭の
朝時間を専門にする
ライフオーガナイザーです。
「声を荒げなくても回る朝」を
設計してください。

# 背景・目的・対象
小さな子どもがいる共働き家庭が
対象です。
親の負担を最小化しながら、
子どもが自分で動けるよう
習慣化できる設計を優先してください。

# 指示内容
以下の【入力データ】をもとに、
起床から出発までの
「朝ルーティン」を設計してください。
「親がやること」と
「子どもに任せること」を
分けて整理すること。

# 制約条件
・怒鳴る場面が生まれない設計にする
・子どもが「自分でできた」と
　感じられるタスクを必ず入れる
・前日夜にやっておくことも
　セットで提案する

# 出力形式
【前日夜の準備（5分以内）】
・〜

【当日朝のルーティン】
（親）〜
（子ども）〜

【詰まりやすいポイントと対策】
〜

# 入力データ
・起床時刻：（例：6時30分）
・出発時刻：（例：8時）
・子どもの年齢：（例：4歳・7歳）
・毎朝詰まる場面：
　（例：着替えに時間がかかる・
　　朝食を食べない）
・パートナーの朝の状況：
　（例：6時に先に出社）
```

**✅具体的な使用方法：**  
　「毎朝詰まる場面」に  
　一番ストレスを感じる場面を  
　正直に書いてください。

　AIはそこを最優先で  
　解消する設計をしてくれます。

**✅読者への一言：**  
　朝の「怒鳴り声」は  
　あなたのせいではありません。

　仕組みがなかっただけです。

　設計を変えると、  
　朝の空気が変わります。

## 2. 「自分の時間」を死守する

育児と仕事に追われていると、  
「自分の時間」が  
最初に消えていきます。

でも、自分が消耗したままでは  
誰のことも大切にできません。

**余裕は「作るもの」ではなく、  
「守るもの」です。**

---

### ◆自分時間の確保プロンプト

**✅目的：**  
　忙しい毎日の中から  
　「自分だけの時間」を  
　意図的に生み出す仕組みを作る。

**✅いつ使うか：**  
　「最近、自分のことを  
　何もできていない」と感じたとき。

```
# ロール設定
あなたはセルフケアと
タイムデザインの専門家です。
育児・仕事に追われる人が
「自分の時間」を
無理なく確保できる方法を
提案してください。

# 背景・目的・対象
育児と仕事で手いっぱいの
パパ・ママが対象です。
「罪悪感なく休める仕組み」を
作ることを最優先にしてください。

# 指示内容
以下の【入力データ】をもとに、
1週間の中から
「自分の時間」を
最低3コマ確保するプランを
設計してください。
「誰かに何かを頼む方法」も
セットで提案すること。

# 制約条件
・1コマは15分以上確保する
・「子どもが寝た後」だけに
　頼らない設計にする
・「罪悪感を感じなくていい理由」を
　1文添える

# 出力形式
【確保できる自分時間3コマ】
コマ①：曜日・時間帯・内容〜
コマ②：〜
コマ③：〜

【そのために誰かに頼むこと】
〜

【罪悪感を感じなくていい理由】
〜

# 入力データ
・最後に「自分のための時間」を
　取れたのはいつ：
　（例：2週間以上前）
・やりたいけどできていないこと：
　（例：読書・ひとりカフェ・運動）
・頼れる人：
　（例：パートナー・実家・保育園の延長）
・自分の時間を取れない理由：
　（例：子どもが離れてくれない・
　　家事が終わらない）
```

**✅具体的な使用方法：**  
　「やりたいけどできていないこと」に  
　正直に書いてください。

　小さなことで構いません。

　「30分ひとりでコーヒーを飲みたい」  
　でも、立派な自分時間の目標です。

**✅読者への一言：**  
　自分を後回しにし続けると、  
　やがて「もう限界」が来ます。

　自分の時間を守ることは、  
　わがままではありません。

　家族を守るための  
　必要な習慣です。

## 3. パートナーとの対話を設計する

「なんでわかってくれないの」

そう思ったことは  
ありませんか？

実は、パートナーに  
伝わらないのは  
「気持ちの問題」では  
ないことが多いです。

**伝え方の設計が  
できていないだけです。**

---

### ◆パートナーへの伝え方設計プロンプト

**✅目的：**  
　感情的にならずに  
　パートナーへ現状を伝え、  
　協力を引き出す言葉を作る。

**✅いつ使うか：**  
　「また同じ話で  
　言い合いになりそう」という  
　場面の前に。

```
# ロール設定
あなたはカップルカウンセリングと
コミュニケーションの専門家です。
一方的な主張ではなく、
相手が「一緒に考えたい」と
感じるような伝え方を
設計してください。

# 背景・目的・対象
育児・家事の負担に
不満を感じている
パートナーの一方が対象です。
責め合わずに
現状を改善するための
対話を促す内容にしてください。

# 指示内容
以下の【入力データ】をもとに、
①現状を伝えるための「事実の言葉」
②自分の気持ちを伝える「感情の言葉」
③パートナーに動いてもらうための
　「お願いの言葉」
の3点を設計してください。

# 制約条件
・「あなたはいつも〜」という
　責める表現は使わない
・相手が防衛的にならない
　言い回しにする
・「一緒に解決したい」という
　姿勢が伝わる文にする

# 出力形式
①事実の言葉：〜
②感情の言葉：〜
③お願いの言葉：〜

【話し合いのベストなタイミングと場所】
〜

# 入力データ
・一番困っていること：
　（例：寝かしつけがいつも自分だけ）
・パートナーに
　わかってほしいこと：
　（例：毎日限界に近いということ）
・これまでの伝え方：
　（例：疲れたとため息をついていた）
・パートナーの性格：
　（例：論理的・感情的な話が苦手）
```

**✅具体的な使用方法：**  
　「パートナーの性格」を  
　できるだけ具体的に書くと、  
　その人に刺さる言い方が  
　届きます。

　「論理的な人」と  
　「感情に共感してほしい人」では、  
　最適な伝え方が  
　全く異なります。

**✅読者への一言：**  
　パートナーは「敵」ではなく  
　「同じチームの仲間」です。

　作戦会議の言葉を  
　AIと一緒に準備してから  
　話し合いに臨んでみてください。

## 4. 仕事の「集中時間」を死守する

育児中の在宅ワークで  
一番難しいのは、  
「集中すること」です。

子どもの声、  
急な呼び出し、  
家事の誘惑。

それでも、  
集中できる時間を  
作ることはできます。

**「集中できる環境」は  
整えるものです。**

---

### ◆集中時間設計プロンプト

**✅目的：**  
　育児の合間に  
　確実に集中できる時間を作り、  
　仕事のパフォーマンスを  
　最大化する。

**✅いつ使うか：**  
　「仕事が全然進まない」と  
　感じているとき。

```
# ロール設定
あなたは在宅ワークと
育児を両立する人の
生産性を高める専門家です。
「確実に集中できる時間」を
設計してください。

# 背景・目的・対象
子どもがいる環境で
仕事をしている親が対象です。
「子どもがいても集中できる仕組み」を
作ることを最優先にしてください。

# 指示内容
以下の【入力データ】をもとに、
1日の中から
「深い集中時間」を
最低2ブロック確保する
スケジュールを設計してください。
「子どもが邪魔しない仕組み」も
セットで提案すること。

# 制約条件
・1ブロックは最低45分確保する
・「子どもが機嫌よくいられる仕掛け」を
　必ず提案する
・緊急時の対応ルールも
　セットで作る

# 出力形式
【集中ブロック設計】
ブロック①：時間帯・場所・環境設定〜
ブロック②：〜

【子どもが機嫌よくいる仕掛け】
〜

【緊急時のルール】
〜
```
