# AIツール安く済んでるけど、これから先ちょっと不安（助けて😂）

## はじめに

私は現在、大学院生として日々の研究や開発に主に以下の4つのAIツールを使っています。

* ChatGPT （OpenAI）
* Googleの AIプラン（Google）
* Claude （Anthropic）
* GitHub Copilot（Microsoft

まず各AIツールについて簡単に整理しときます。

---

## 各サービス概要

### 1. ChatGPT（OpenAI）

Chatgptは言うまでもないですね。Chatgptのサブスクリプションを契約すると、チャット形式のChatgpt、コーディング特化のCodexが利用できます。CodexはVSコードでも使えますし、Codex cliというターミナルで実行できるものもあるので何かと便利です。あと、とにかく安いです。

料金体系は以下になります。詳しくは公式ドキュメントを参照してください。

| プラン | 月額 |
| --- | --- |
| Free | 無料 |
| Go | $8 |
| **Plus** | **$20** |
| Pro | $200 |

---

### 2. Gemini（Google）

Googleが提供するAIサービスですね。チャットAIとしても使えますが、Googleサービスとの連携が強いですね（ChatgptでもClaudeでもGoogle関連のツールとの連携はある程度できる）。あとはNotebookLMなんかも地味に便利ですね。画像生成用のNano Bananaはかなり優秀だと思います。あのNano Banana特有の画像は個人的にはまってます。

料金体系は以下になります。詳しくは公式ドキュメントを参照してください。

| プラン | 月額（日本） |
| --- | --- |
| Free | 無料 |
| Google AI Plus | 1200円 |
| **Google AI Pro** | **2,900円** |
| Google AI Ultra | 36400円 |

---

### 3. Claude（Anthropic）

去年あたりから勢いを増しているAIツールですね。Claudeのサブスクリプションを契約すると、Chatgptのようなチャット形式の`Claude.ai`、コーディングに特化した（今や何でもこなせるけど）`Claude Code`を利用できます。  
無料プランは、`Claude.ai`のみ。有料プランからは`Claude.ai`と`Claude Code`のどちらも利用できます。

料金体系は以下になります。詳しくは公式ドキュメントを参照してください。

| プラン | 月額 |
| --- | --- |
| Free | 無料 |
| **Pro** | **$17（年払い）/ $20（月払い）** |
| Max 5x | $100 |
| Max 20x | $200 |

---

### 4. GitHub Copilot（GitHub）

Github Copilotはコードの補完やチャット、さらにはエージェント機能も充実しています。  
GPT系、Gemini系、Claude系のモデルも使用できます。プレミアムリクエストというモデルを使用するためのポイントがあり、モデルによって消費量が異なる仕組みになっています。

料金体系は以下になります。詳しくは公式ドキュメントを参照してください。

| プラン | 月額 |
| --- | --- |
| Copilot Free | 無料（機能制限あり） |
| **Copilot Student** | **無料（学生認証が必要）** |
| Copilot Pro | $10 |
| Copilot Pro+ | $39 |

---

## 私の契約状況・学生キャンペーン

### ChatGPT Plus：12ヶ月50%オフキャンペーン 適用中

通常は月額$20ですが、GPT5がリリースされた2025年8月頃に、今なら12ヶ月50%オフプラン！みたいなレコメンドに誘われて契約しました。12ヶ月半額プランと調べても、どの層のユーザーにどれほど提供しているのかはいまいちわからなかったです。ですが、定期的に確認はしておいて損はないと思います。現状、1700円程度でChatgptからCodexまで使えるのでほんとにOpenAIに感謝です。

💰 通常$20 → **$10/月**（年間$120の節約）

---

### Google AI Pro：学生15ヶ月無料プラン 適用中

Google AI Proの学生向けトライアルというのに2025年の6月ぐらいに契約しました。私が契約したキャンペーンは学生が15ヶ月無料になるといった内容でしたね。学校発行している有効なアドレスがあれば登録できたと思います。2025年の10頃にも学生が12ヶ月無料プランになるキャンペーンを実施したりしていましたね。ただ現在は、米国の学生のみと公式サイトに記載されていました。

💰 通常約2,900円/月 → **0円**（15ヶ月間、約43,500円相当）

---

### Claude Pro：定価で契約中

現時点でAnthropicは学生割引を公式に設けていないです。上記の4サービスの中で唯一キャンペーン等を使わず、通常料金の月払い$20（約3,000円）で契約しています。

💰 **$20/月**（節約なし）

---

### GitHub Copilot：学生無料プラン 適用中

GitHub Copilotの学生プランは、学校発行のメールアドレスと学生証で本人確認を行うだけで契約できます。  
ただ、二段階認証とか支払い関連の情報を設定しておかないとおそらくエラーになります（ユーザーの情報が足りません！みたいなエラーだった気がする）

💰 通常$10/月 → **0円**

## コスト比較

| サービス | 正規コスト | 実質コスト | 節約方法 |

| ChatGPT Plus | $20（約3,000円） | $10（約1,500円） | 50%オフ（1年間） |
| Google AI Pro | 2,900円 | **0円** | 学生15ヶ月無料トライアル |
| Claude Pro | $20（約3,000円） | $20（約3,000円） | なし（定価） |
| GitHub Copilot | $10（約1,500円） | **0円** | GitHub Student プラン |
| **合計** | 約**10,400円/月** | 約**4,500円/月** | 約**5,900円/月**の節約 |

4サービスで本来なら月約1万円かかるところを、キャンペーンの活用で月約4,500円に抑えられている。

## これから先がちょっと不安

現状、かなり低額でたくさん使用させてもらっていますが、学生プランの有効期限が切れたり、割引プランが終わってしまうので、ツールを選別していく必要があります。

学生のうちに使えるキャンペーンを活用するのは間違いなくお得ですけど、それに慣れすぎてしまうのも考えものかもしれないですね。
