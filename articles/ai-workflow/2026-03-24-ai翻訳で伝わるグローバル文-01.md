「英語に訳したのに、  
なんか伝わらない気がする。」

そんなモヤモヤ、  
感じたことはありませんか？

グローバル展開を目指して  
英語の文章を作ったのに、  
海外のユーザーから反応がない。

翻訳ツールで変換したら  
文法は合っているはずなのに、  
どこかぎこちない。

---

それ、  
「翻訳」はできていても  
「ローカライズ」ができていない  
サインかもしれません。

ローカライズとは、  
言葉を現地の文化・習慣・感覚に  
合わせて最適化することです。

---

この記事を読むと、

✅ **コミュニケーションロスが減る**  
　→現地ユーザーに「自分ごと」と  
　　感じてもらえる表現が作れる

✅ **翻訳の手戻りリスクを回避できる**  
　→「これは失礼な表現だった」  
　　という後悔がなくなる

✅ **グローバル展開のスピードが上がる**  
　→AIと二人三脚で、  
　　市場参入のスピードが加速する

---

AIは「万能な通訳者」ではありません。

でも、  
使い方さえ知れば  
文化の壁をスッと越える  
最強のパートナーになります。

今日から、  
あなたの発信は  
世界に届く言葉に変わっていきます。

  

---

## 基本のプロンプト

まず、例え話からはじめます。

AIを使った翻訳・ローカライズとは、  
一体どういうことか。

---

**お土産屋さんに例えてみます。**

日本の空港で  
外国人観光客に  
「これ、縁起物です」と言っても  
なかなか伝わらないですよね。

でも、  
「This brings good luck.  
It's a Japanese tradition.  
（これは幸運をもたらす。  
日本の伝統の品です。）」  
と説明したら、  
目がパッと輝く。

同じ商品でも、  
「相手の文化に合った言葉」で  
説明するかどうかで  
反応がまるで変わります。

---

AIを使うのは、  
その「言い換え」を  
一瞬でやってもらうようなものです。

あなたが伝えたいことを入力すると、  
現地の人の感覚に合わせた  
表現に変換してくれます。

では、基本のプロンプトを  
2つ紹介します。

![](https://assets.st-note.com/img/1774243166-F5JZYPvWG9EXfqNu0d2seROb.png?width=1200)

---

### ◆現地ユーザーに届く翻訳プロンプト

**✅目的：**  
　日本語の文章を、  
　特定の国・文化圏に合わせて  
　自然な表現に翻訳する。

**✅いつ使うか：**  
　商品説明、LP（ランディングページ）、  
　SNSの投稿文など、  
　海外向けに発信する前。

```
# ロール設定
あなたは現地文化に精通した
プロのローカライズ専門家です。
単なる翻訳ではなく、
対象の国・地域のユーザーが
自然に受け入れられる表現を
提案してください。

# 背景・目的・対象
・翻訳元の言語：日本語
・翻訳先の言語・地域：[例：英語（アメリカ）]
・発信する媒体：[例：Instagram投稿]
・想定読者：[例：20〜30代の女性ユーザー]
・目的：[例：商品の購買意欲を高めたい]

# 指示内容
以下を実行してください。

1. 原文を自然な現地表現に翻訳する
2. 翻訳した表現のポイントを
　 1〜2行で解説する
3. 別パターンの表現を1つ追加で提案する

# 制約条件
・直訳・機械翻訳的な表現は使わない
・現地の文化・習慣・言い回しを優先する
・失礼・不快に感じられる表現は避ける

# 出力形式
【翻訳】
【解説】
【別パターン】

# 入力データ
原文：[ここに日本語テキストを入力]
```

**✅具体的な使用方法：**  
　「翻訳先の言語・地域」と  
　「原文」を変えるだけで、  
　どの国向けにも使えます。

　アメリカ英語と  
　イギリス英語でさえも  
　ニュアンスが変わるので、  
　地域まで指定するのがポイントです。

**✅読者への一言：**  
　「翻訳」と「ローカライズ」は  
　別物です。

　このプロンプトを使えば、  
　その違いが一目でわかります。

---

### ◆公開前の文化チェックプロンプト

**✅目的：**  
　翻訳後の文章が  
　「現地で失礼・不自然ではないか」を  
　チェックする。

**✅いつ使うか：**  
　海外向けコンテンツを公開する前の  
　最終チェックとして。

```
# ロール設定
あなたは[対象国]の文化・習慣・
言語感覚に詳しいネイティブチェッカーです。
外国人が書いた文章を読み、
現地の視点でフィードバックしてください。

# 背景・目的・対象
・チェック対象の言語・地域：
　[例：英語（アメリカ市場向け）]
・文章の用途：
　[例：ビジネスメール・SNS広告・商品説明]
・特に気にしていること：
　[例：失礼な表現がないか・
　　　自然に読めるか]

# 指示内容
1. 不自然・失礼・誤解を招く
　 表現があれば指摘する
2. より自然な言い換えを提案する
3. 「現地ネイティブが読んだ印象」を
　 一言でコメントする

# 制約条件
・批判的すぎず、
　建設的なフィードバックにする
・なぜ不自然なのか、理由も添える

# 出力形式
【問題箇所】→【改善案】→【理由】の形式。
最後に【総評】1文を追加。

# 入力データ
チェックしたい文章：[ここに入力]
```

**✅具体的な使用方法：**  
　自分で翻訳した文章や  
　既存の翻訳ツールの出力を  
　そのまま貼り付けるだけ。

　「なんとなく不安」な文章を  
　公開前に必ずここで通してみてください。

**✅読者への一言：**  
　自信があるときほど、  
　確認が大切です。

　このプロンプトは  
　公開前の「お守り」として  
　使ってください。

## ビフォー・アフター比較

**同じ文章でも、  
プロンプトの作り方で  
結果はまるで変わります。**

想定シナリオ：  
「日本のスキンケアブランドが  
アメリカ市場向けに  
商品キャッチコピーを作りたい」

---

### Before（簡易プロンプト）

```
この文章を英語に翻訳してください。
「肌にやさしく、美しさを引き出す
日本の伝統的なスキンケア。」
```

**AIの回答（例）**：

> Gentle on the skin, bringing out  
> beauty with traditional Japanese  
> skincare.

---

👉 **どこが足りないか？**

---

### After（本記事のプロンプトを使用）

```
# ロール設定
あなたは現地文化に精通した
プロのローカライズ専門家です。

# 背景・目的・対象
・翻訳先：英語（アメリカ）
・発信媒体：Instagram広告
・想定読者：25〜35歳の女性、
　美容に関心が高い層
・目的：購買意欲を高める
　キャッチコピーを作る

# 入力データ
原文：「肌にやさしく、
美しさを引き出す
日本の伝統的なスキンケア。」
```

**AIの回答（例）**：

>  【翻訳】  
>  "Rooted in centuries of Japanese  
>  tradition — skincare that's as  
>  gentle as it is transformative."  
>   
>  【解説】  
>  「centuries of tradition」で  
>  歴史の深みを演出しつつ、  
>  「transformative」でアメリカ人が  
>  好む「変化・成果」の価値観に訴求。  
>   
>  【別パターン】  
>  "Your skin, cared for the  
>  Japanese way."

---

👉 **何が変わったか？**

「ロール設定」で  
ローカライズ専門家として起動し、

「背景指定」で  
媒体・読者・目的を渡したことで、

**現地ユーザーの感覚に響く  
言葉と理由が出てきました。**

翻訳ツールには出せない  
「なぜそう表現するか」の解説が、  
あなたの学習にもなります。

## 実践プロンプト集

## 1. スラングと口語に対応する

英語でもスペイン語でも、  
教科書通りの表現は  
「なんか堅苦しい」と  
思われることがあります。

現地の若者が使う  
自然な口語・スラングを取り入れると、  
一気に親近感が生まれます。

でも、スラングは間違えると  
失礼になることも。

AIに適切な表現を  
確認してもらいましょう。

---

### ◆口語・スラング確認プロンプト

**✅目的：**  
　自然な現地口語表現に合わせた  
　翻訳とトーン調整をする。

**✅いつ使うか：**  
　SNS投稿、アプリのUI文言、  
　若い世代に向けた広告コピーを  
　作るとき。

```
# ロール設定
あなたは[対象国・地域]の
20〜30代のネイティブスピーカーです。
日常的に使われる自然な口語表現で
翻訳・提案してください。

# 背景・目的・対象
・翻訳先：[例：英語（アメリカ）]
・対象年齢層：[例：20代前半]
・媒体：[例：TikTokのキャプション]
・トーン：[例：フレンドリー・テンポよく]

# 指示内容
1. フォーマルな直訳を1つ出す
2. 現地の若者が使う
　 カジュアルな表現に変換する
3. スラングや短縮形を使う場合、
　 その意味と使い方を補足する
4. 「使うと逆効果になる表現」が
　 あれば警告する

# 制約条件
・差別的・攻撃的な表現は使わない
・流行が古くなっているスラングは
　使わない（古くなっている可能性は
　「要確認」と明示する）

# 出力形式
【フォーマル】
【カジュアル】
【補足】
【注意事項】

# 入力データ
原文：[ここに入力]
```

**✅具体的な使用方法：**  
　「媒体」と「対象年齢層」を  
　具体的に書くと、  
　AIがトーンを合わせてくれます。

　SNSごとに  
　最適なトーンが違うことも  
　AIが教えてくれます。

**✅読者への一言：**  
　スラングは「諸刃の剣」。

　使いこなせれば  
　一気に距離が縮まるし、  
　間違えれば信頼を失います。

　AIと確認してから使いましょう。

## 2. 文化的タブーを事前に避ける

「まさかこの表現が  
問題になるとは思わなかった。」

海外展開で一番怖い失敗が  
これです。

色・数字・ジェスチャー・表現。  
文化によって  
「縁起が悪い」「失礼」と  
感じられるものは千差万別です。

出してから気づくのでは遅い。  
事前にAIで確認しましょう。

---

### ◆文化的タブーチェックプロンプト

**✅目的：**  
　コンテンツや表現が  
　対象国・地域の文化で  
　問題にならないか事前に確認する。

**✅いつ使うか：**  
　新しい市場に向けた  
　広告・商品パッケージ・  
　Webサイトを公開する前。

```
# ロール設定
あなたは異文化コミュニケーションと
グローバルマーケティングの専門家です。
特定の文化・地域において
誤解・不快感・タブーを招く表現を
見つけ出し、代替案を提案してください。

# 背景・目的・対象
・展開先の国・地域：[例：中東全般]
・コンテンツの種類：
　[例：飲料の広告ビジュアルとキャッチコピー]
・ターゲット層：[例：30〜50代の男性]

# 指示内容
1. 以下のコンテンツを確認し、
　 文化的に問題になりうる点を
　 すべて洗い出す
2. 各問題点について
　 「なぜ問題か」を説明する
3. 文化的に受け入れやすい
　 代替表現・代替ビジュアルの
　 方向性を提案する

# 制約条件
・宗教・政治・歴史的背景にも
　配慮すること
・「たぶん大丈夫」ではなく、
　確信が持てないときは
　「要確認」と明記する

# 出力形式
【問題点】→【理由】→
【代替案または確認推奨】の形式。

# 入力データ
チェックしたいコンテンツ：[ここに入力]
```

**✅具体的な使用方法：**  
　「展開先の国・地域」は  
　できるだけ絞って入力してください。

　「アジア全般」より  
　「タイ」の方が  
　精度の高い回答が得られます。

**✅読者への一言：**  
　グローバル展開での炎上は、  
　ほぼ「知らなかった」から起きます。

　このプロンプトで  
　「知らなかった」をゼロにしましょう。

## 3. 専門用語を現地で通じる言葉にする

IT・医療・法律・金融。  
業界の専門用語は  
国によって言い方がまるで違います。

日本でよく使う「エンジニア」も、  
アメリカでは職種によって  
「Software Engineer」  
「Developer」「Programmer」  
と使い分けがあります。

「業界の常識」は  
国ごとに違うのです。

---

### ◆専門用語現地化プロンプト

**✅目的：**  
　業界の専門用語を、  
　対象国で正しく通じる  
　現地表現に変換する。

**✅いつ使うか：**  
　技術文書、求人票、  
　サービス紹介資料を  
　海外向けに作るとき。

```
# ロール設定
あなたは[対象業界]と
[対象国・地域]の言語・慣習に
精通した専門翻訳者です。
技術用語・業界用語を
現地で正しく通じる表現に
変換してください。

# 背景・目的・対象
・業界：[例：ITソフトウェア開発]
・翻訳先の国・地域：[例：アメリカ]
・ドキュメントの種類：
　[例：エンジニア向け求人票]
・読者の専門レベル：
　[例：中級〜上級のエンジニア]

# 指示内容
1. 原文に含まれる専門用語を
　 リストアップする
2. 各用語を対象国で
　 一般的に使われる表現に変換する
3. 複数の表現がある場合は
　 使い分けの基準も説明する

# 制約条件
・和製英語（例：「リリース」「ベンダー」）
　は現地で通じるか確認すること
・通じない可能性がある場合は
　「要確認」と明示する

# 出力形式
【原語】→【現地表現】→
【補足・使い分け基準】の表形式。

# 入力データ
翻訳・確認したい文章や用語リスト：
[ここに入力]
```

**✅具体的な使用方法：**  
　「専門用語がわからなくていい」のが  
　このプロンプトの強みです。

　文章をまるごと貼れば、  
　AIが専門用語を自動で拾って  
　整理してくれます。

**✅読者への一言：**  
　「英語で書いた」ではなく  
　「現地で通じる英語で書いた」。

　この違いが、  
　採用率・購買率を大きく変えます。

## 4. トーンとブランドボイスを統一する

翻訳によって  
「ブランドの雰囲気」が  
変わってしまった経験はありませんか？

日本語では  
「丁寧で上品」なのに、  
英語にしたら  
「堅くてとっつきにくい」  
と感じられてしまう。

ブランドの「声」は、  
言語をまたいでも  
統一させなければなりません。

---

### ◆ブランドボイス統一翻訳プロンプト

**✅目的：**  
　翻訳後もブランドのトーンと  
　雰囲気を維持したまま  
　現地化する。

**✅いつ使うか：**  
　Webサイト・ブランドコピー・  
　プロダクトUI文言を  
　多言語展開するとき。

```
# ロール設定
あなたはブランドコミュニケーションと
グローバルローカライズの専門家です。
ブランドの「声（トーン）」を
言語をまたいで統一させる翻訳を
してください。

# 背景・目的・対象
・ブランドのトーン：
　[例：親しみやすく、
　　　プロフェッショナルで、
　　　ユーモアも少々]
・翻訳先の言語・地域：
　[例：フランス語（フランス）]
・コンテンツ種類：
　[例：Webサイトのトップページコピー]

# 指示内容
1. 原文のトーンを分析して一言で表す
2. そのトーンを維持したまま
　 現地語に翻訳する
3. 「直訳版」と「トーン重視版」の
　 2パターンを出し、
　 違いを解説する

# 制約条件
・現地の文化・ユーモアの感覚に
　合わせて調整する
・ブランドの信頼感を
　損なう表現は使わない

# 出力形式
【トーン分析】
【直訳版】
【トーン重視版】
【違いの解説】

# 入力データ
原文：[ここに入力]
```

**✅具体的な使用方法：**  
　「ブランドのトーン」の欄に  
　形容詞を3〜5つ並べるだけでOKです。

　「明るい・若々しい・シンプル」  
　この3つを書くだけで、  
　AIが文体を合わせてくれます。

**✅読者への一言：**  
　翻訳してから  
　「なんか違う」と感じたら、  
　それはトーンがズレているサインです。

　このプロンプトで  
　「違和感」の正体を  
　言語化してみてください。

## 5. 多言語SEOキーワードを設計する

「海外向けのWebページを作ったのに  
検索に全然出てこない。」

そんな悩み、ありませんか？

日本語のSEOキーワードを  
そのまま直訳しても、  
現地ではそのキーワードで  
検索されていないことが多いです。

現地ユーザーが  
「実際に何と検索しているか」を  
押さえることが、  
グローバルSEOの第一歩です。

---

### ◆多言語SEOキーワード設計プロンプト

**✅目的：**  
　対象国・言語で  
　実際に使われている  
　検索キーワードを洗い出し、  
　SEOに強いコンテンツを作る。

**✅いつ使うか：**  
　海外向けWebページ・ブログ・  
　広告文を作るときの  
　準備フェーズとして。

```
# ロール設定
あなたはグローバルSEOと
```

ただ変換するだけでは届かない。

すべてを乗せてはじめて、
