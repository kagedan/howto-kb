# はじめまして

naotoと申します。2026年5月9日に「CheckMe（checkme.run）」をローンチし、22日後に月間1万PVを達成しました。

## CheckMeとは

CheckMe（checkme.run）は古文変換・漢文変換・フラッシュカード・4択クイズ・なぞかけ・大喜利など60本以上のツールをまとめた無料Webサービスです。登録不要・完全無料で、URLを開いた瞬間から使えます。

## 開発体制

一人の人間（私）と12のAI担当で運営しています。

* Claude（仕様策定・実装）
* Gemini（コンテンツ品質評価）
* その他AI（マーケ・SEO・UI/UX等）

## 技術スタック

```
バックエンド : FastAPI (Python)
DB : SQLite（WALモード）
フロント : Jinja2 + Tailwind CSS
インフラ : Docker / Rocky Linux 9
外部API : Anthropic Claude / Google Gemini
```

## 22日で1万PVになった理由

「古語変換」「漢文変換」でGoogle検索4〜5位に入ったことが主因です。競合が少ないニッチを狙い、SEOで刺せるコンテンツを最初から意識しました。CTRは10〜11%。

## 今後

8月末に月間10万PVを目指しています。技術的な知見をQiitaに書いていく予定です。
