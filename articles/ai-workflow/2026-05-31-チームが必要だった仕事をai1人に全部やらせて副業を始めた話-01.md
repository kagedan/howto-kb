副業を始めたいけど、スキルもない。時間もない。人脈もない。

ちょっと前まで、この3つが揃っていないと「自分でゼロから覚える」か「高いお金で外注する」かの2択しかなかった。でも去年あたりから、その前提が静かに崩れている。指示するだけで、実際に手を動かして作業を終わらせてくれるAIが出てきたからだ。

正直に言うと、私はAIツールを何年も触り続けてきて、もう驚かなくなっていた。「結局、提案してくるだけでしょ」と。チャットに質問を打つと、それっぽい答えが返ってくる。でも作業そのものは自分でやるしかない。そういうものだと思っていた。

恥ずかしい話を先にしておく。数年前、私は副業を始めようとして、ランディングページの制作を外注した。相場をよく知らないまま「お任せで」と頼んで、8万円を払った。上がってきたのは、テンプレートに文字を流し込んだだけのページ。修正をお願いするたびに追加料金。結局、公開する前に心が折れてやめた。お金も時間も、ただ消えた。

その経験があったから、最初は「AIに任せる」という言葉も信用していなかった。どうせまた中途半端なものが出てくるんだろう、と。

その認識が変わったのが「Manus（マナス)」というツールだった。これは答えを返すAIじゃない。勝手にブラウザを開いて、ボタンを押して、フォームを入力して、メールまで送る。例えるなら、月39ドルで24時間働くインターンを雇った感覚に近い。風邪も引かないし、文句も言わない。

この記事では、本来なら「チームを組む」か「数十万円払って外注する」しかなかった仕事を、副業初心者がAI1つでこなしていく道筋を見せる。難しいコードは一切出てこない。日本語で話しかけるだけだ。あの8万円の失敗をしていた頃の自分に、そのまま渡したい内容にした。

## 第1章｜今までのAIと、何が決定的に違うのか

![](https://assets.st-note.com/img/1780095973-nMuQ6ifyZTJkzBO54cqFHPKA.png?width=1200)

まず、ここを押さえないと話が進まない。ChatGPTやGemini、Claudeといった普段のAIは「会話して、答えをもらう」道具だ。「ブログの構成を考えて」と頼めば構成案が出てくる。でも、その構成で実際に記事を書いて、画像を入れて、公開するのは自分の仕事だ。

Manusはここが違う。やってと言えば、やる。タブを開く。情報を調べる。表を作る。フォームに打ち込む。メールを送る。カレンダーに予定まで入れる。全部、勝手に。

> あなたはAIを使って、自分自身を先に作り変えないといけない。他人やAIにそれをやられる前に。

動画の中で紹介者がこう言っていた。少し大げさに聞こえるかもしれない。でも、副業の世界で言い換えるとシンプルだ。今まで「発注する側」だった作業を、自分が「こなす側」に回れるということ。Web制作も、リサーチも、営業リスト作りも、これまでなら人を雇うかお金を払うしかなかった。それを自分の手元で動かせる。

具体的に、普通のAIとManusの違いを並べるとこうなる。

![](https://assets.st-note.com/img/1780097044-QJlWByt4RPopOKMc1EzAXmr9.png?width=1200)

左が「アドバイザー」だとしたら、右は「実行する部下」だ。副業を始めたばかりの人にとって、本当に足りないのは知識より「手」だったりする。その手が、ほぼタダ同然で手に入る。

## 第2章｜Web制作を「受ける側」になる

![](https://assets.st-note.com/img/1780095985-GauUY1DFczeKPfi23vZM0ohI.png?width=1200)

少し前まで、ホームページが1枚欲しいだけで3万円、しっかり作れば10万円が当たり前だった。制作会社とのやり取りに数週間。半年後に修正したくなったら、また費用が発生する。だから副業初心者にとってWeb制作は「お金を払う側」の話だった。

Manusを使うと、この立場が逆転する。コードを1行も書かずに、知り合いのお店のサイトを作って納品できるからだ。手順はこうだ。

下のプロンプトをそのまま使ってもいい。

```
東京都内の美容室のホームページを作ってほしい。
必要なページは、トップ・メニュー料金・スタッフ紹介・予約フォームの4つ。
デザインは清潔感のある白基調で、写真が映えるシンプルなものに。
参考にしたいサイトのURLを下に貼るので、雰囲気を寄せて。
始める前に、不明な点があれば質問してください。
```

最後の「不明な点があれば質問して」の一文が地味に効く。これを入れておくと、Manusが勝手に突っ走らず、足りない情報を聞き返してくれる。出来上がりの精度が上がる。

生成が終わったら、また話しかけて修正する。「トップの文字量を減らして、予約ボタンをもっと大きく」と言えば直る。仕上がったらドメインに繋ぐだけで公開できる。

実際にこの指示を出すと、どんな流れで進むのか。動画で紹介されていた挙動を整理すると、こうなる。

```
入力：「美容室のサイトを作って」と音声で依頼
　↓
Manusの動き：ブラウザを開く → コードを書く → ページを組む → 文章まで自動で入れる
　↓
出力：4ページ構成のサイトが完成 → 「ここ直して」と話すと即修正 → 公開
```

注目してほしいのは、Manusが「文章まで書く」点だ。デザインだけでなく、トップに載せるキャッチコピーや料金説明の文言まで埋めてくれる。文章が苦手で副業をためらっていた人ほど、ここが効く。

![](https://assets.st-note.com/img/1780097071-QszjWoyq1Tba8Yl7AeELpF3i.png?width=1200)

つまり、自分が3万円払っていた作業を、逆に3万円もらってこなせる立場になる。最初のお客さんは、行きつけの飲食店や知人の個人事業主で十分だ。「サイトないなら作りますよ」のひと言から始まる。

## 第3章｜SNS運用代行を、リサーチごと自動化する

![](https://assets.st-note.com/img/1780095998-jwH23rg0uFdBpkYyPDc6IRbC.png?width=1200)

SNS運用代行は、副業として根強い人気がある。月数万円で個人や小さなお店の投稿を任される仕事だ。問題は、ネタ出しが想像以上にしんどいこと。毎月10〜15時間、白い画面とにらめっこして消耗する。だから多くの人が途中で投げ出す。

Manusの「ワイドリサーチ」機能を使うと、ここが一気に楽になる。ネットを調べて「実際に伸びている投稿」のパターンを抜き出し、それを元に30日分の投稿カレンダーを作ってくれる。自分が詳しくないジャンルでも代行を請け負える、というのが大きい。

```
パーソナルジムのSNS運用を代行している。ターゲットは忙しい30〜40代の会社員。
InstagramとYouTubeで上位のフィットネス系発信者を20人リサーチして、
平均より明らかに伸びている投稿・動画を見つけてほしい。
そのパターンを参考に、Instagram用の30日投稿カレンダーを作って。
各投稿に「ネタ」「冒頭フック」「キャプション」「最適な投稿時間」を入れること。
テーマはダイエット・作り置き・自宅トレに絞って。
```

ポイントは「真似しろ」ではなく「伸びてるものを参考に」と指示している点だ。丸パクリではなく、勝った型を入力にして自分の投稿を組み立てる。これで30日分のネタ出しが、自分の作業時間15分ほどで片付く。

出てくるカレンダーは、ただの「ネタの羅列」ではない。1日ごとに、どのプラットフォームに、どんなフックで、どんなキャプションで投稿するかまで埋まっている。

> 注意：出てきたカレンダーをそのまま流すのはおすすめしない。最後に自分の言葉で1〜2割は手直しする。AIっぽさが消えて、代行クライアントの満足度がぐっと上がる。

この仕組みがあると、複数のクライアントを同時に持てる。1社につき作業15分なら、3社受けても1時間かからない。月3万円×3社で、副業として現実的な数字が見えてくる。

## 第4章｜営業リストと初回メールを、寝てる間に作る

![](https://assets.st-note.com/img/1780096008-xVYOybsU9QfCrgAXRij7L8uZ.png?width=1200)

副業で一番つらいのは、技術より「最初のお客さんを見つけること」だ。私も昔、リスト集めを外注したことがある。月10万円近く払って、上がってきたのは穴だらけのスプレッドシートだった。自分でやろうとすれば、リスト作りとメール作成だけで何十時間も溶ける。

Manusはこの工程を丸ごと引き受ける。条件に合う会社を探し、担当者の名前とメールを集め、表にまとめ、一人ひとり違う文面のメールを下書きし、送信までやる。これ自体が「リスト獲得・営業代行」という売れる副業になる。

```
【リスト作成】美容・健康系の中小企業を50社探してほしい。
各社について、代表者またはマーケティング担当者の名前・
会社のメールアドレス・ウェブサイトURLを調べて、
スプレッドシート形式でまとめてください。
```

```
【メール下書き】さっきのリストの各社に向けて、3〜4文の短い営業メールを作って。
相手のサイトから読み取れる「その会社ならではの特徴」を1つ必ず入れて、
パーソナライズすること。売り込みすぎず、丁寧で自然なトーンで。
```

![](https://assets.st-note.com/img/1780097092-3xkCWLMmKiE1GIo0ez4qTNPS.png?width=1200)

> 誠実に運用するために：いきなり50件送らないこと。まず10件送って反応を見る。文面を直してまた10件。これを繰り返す。一斉送信で雑にばらまくと、相手にも自分の評判にも傷がつく。「数」ではなく「仕組み」を作る意識でいたい。

ここで多くの人が不安に思うはずだ。「営業メールなんて送ったことがない自分にできるのか」と。でも、考えてみてほしい。これまでこの作業を「できなかった」のは、能力がなかったからではなく、リスト作りという膨大な下準備で力尽きていたからだ。その一番きつい部分をAIが肩代わりしてくれるなら、残るのは「送るボタンを押す勇気」だけになる。

そして勇気は、小さく始めれば自然とついてくる。10件のうち1件でも返信が来たら、世界の見え方が変わる。「自分にも仕事が取れた」という事実は、どんな教材よりも背中を押してくれる。

## 第5章｜副業初心者がやりがちな3つの失敗

![](https://assets.st-note.com/img/1780096018-ej04ZOYnwGSW5kMNhrHcAfF1.png?width=1200)

### 失敗① ツールに課金して満足する

### 失敗② できない仕事まで丸投げする

* AIは「調べる・作る・送る」は得意。でも「最終判断」と「責任」は持てない
* 納品物のチェック、相手との約束、品質の最終確認は人間の仕事
* 丸投げできる部分と、できない部分の線引きを最初に決めておく

### 失敗③ 1人で抱えて続かない

* 気合いだけで回そうとすると、本業が忙しくなった瞬間に止まる
* 「毎週月曜の朝にリスト30件」のように、AIに定例作業を仕込んで仕組みにする
* 続く人は根性ではなく、止まらない仕組みを持っている

この3つを避けるだけで、3ヶ月で消える9割の側から、静かに抜け出せる。

## おまけ｜AIを「毎朝の相棒」にする裏ワザ

あまり知られていないが、ManusはスマホのチャットアプリTelegramと繋げられる。これをやると、パソコンの前に座らなくても、通勤中にスマホから指示が出せる。副業の最大の敵は「机に向かう時間がない」ことなので、これは地味に大きい。

私がおすすめしたいのは、毎朝こう仕込んでおくことだ。

```
毎朝、私の副業の目標とカレンダーを見て、
今日できる作業を3つ提案してほしい。
本業が忙しい日でも15分で終わる粒度のものを優先して。
提案はTelegramに送って。
```

これだけで、AIが「今日はこれをやろう」と毎朝声をかけてくれる。専属の秘書が、自分の時間を守って、ネタを出して、背中を押してくれる感覚に近い。1人で抱えて続かない、という失敗③の最高の対策にもなる。

## まとめ｜午後1回分の時間で、できること

1年前なら、Web制作・SNS運用・営業リスト作り、この3つを揃えるには専属のチームか数十万円の外注費が必要だった。それが今は、月39ドルのAIエージェント1つと、午後1回分の時間で動き出せる。

怖いのは、「AIを使う人」と「使おうと思っているだけの人」の差が、毎月どんどん開いていることだ。知っているだけでは何も変わらない。動いた人だけが、その差の内側に入れる。

だから提案はシンプルだ。この記事で出てきた4つのこと（サイト制作・コンテンツ・リスト作り・失敗回避)のうち、どれか1つだけ、今日のうちに試してみてほしい。完璧じゃなくていい。手を動かした瞬間に、見える景色が変わる。

8万円を溶かして心が折れた頃の私に、いま一番伝えたいのはこれだ。「最初の一歩は、もう外注に頼まなくていい。お金じゃなくて、ほんの少しの行動でいい」。あの頃のあなたと同じ場所に立っているなら、この記事がその一歩の背中を押せたらうれしい。
