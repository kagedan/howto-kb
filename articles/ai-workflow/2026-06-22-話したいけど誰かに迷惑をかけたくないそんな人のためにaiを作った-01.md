夜中の2時に「誰かと話したい」と思うことがある。

でも、友人にLINEするには遅すぎる。家族を起こすわけにもいかない。SNSに書くほどの話でもない気がする。

そういう気持ちを、自分はずっと抱えていた。

それで作ったのが [Moshimo](https://moshimo-ai.web.app) だ。

---

## Moshimoとは

アドラーや歴史上の人物と話せるAIメンターアプリ。相談というより、「言葉にする場所」として設計している。

「もしも○○が隣にいたら」というコンセプトで、ニーチェ、アドラー、マルクス・アウレリウスなどと対話できる。

アドラーなら、こんな返しをする：

> 悩みを相談するのが苦手
>
> ——なぜ相談しない？
>
> 迷惑かけたくないから
>
> ——それは相手への気遣いじゃない。嫌われることへの恐怖だ。

刺さる人には刺さる。

## 技術スタック

| 層 | 技術 |
| --- | --- |
| フロントエンド | バニラJS |
| バックエンド | Python（Flask）on Cloud Run |
| DB | Supabase（PostgreSQL） |
| 認証・ホスティング | Firebase |
| AI | Claude API（Anthropic） |
| 課金 | Stripe |

MoshimoはBetashipというプラットフォーム上で動いている。認証・クレジット消費はプラットフォーム側が担い、ツール本体は会話体験の設計だけに集中できる構造にした。

---

## 一番こだわった部分：ペルソナ設計

「AIと話す」体験で一番つまらないのは、相手が「どのAIも同じ口調」になること。

Moshimoでは各メンターに固有のシステムプロンプトを設計した。アドラーは短く切り返す。マルクス・アウレリウスは静かに問い返す。ニーチェは容赦ない。

同じ悩みを持ち込んでも、誰と話すかで体験がまったく変わる。それが「もしも」の面白さだと思っている。
