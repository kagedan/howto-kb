## はじめに

「API」って聞いたことあるけど、なんか難しそう…

自分もずっとそう思ってた。ChatGPTはアプリで使えばいいじゃん、って。

でも**APIを覚えると、自分のアプリやサービスにAIを組み込める**。ブラウザ版とは用途が全然違う。

この記事では、APIを一度も使ったことない人が**実際に動かすまでの手順**をまとめた。
気軽に始められる入口として、ai& Inferenceを使う。


## APIって何？

ブラウザ版のChatGPTは「人間が手で使うもの」。

APIは「プログラムから自動で使うもの」。

たとえばこんなことができる：
- アプリにAIを組み込む
- 処理を自動化する

ブラウザ版だと「自分が使うツール」だけど、APIだと**自分が作るサービスの部品**として使える。


## ai& Inferenceって何？

今回使うのは **[ai& Inference](https://aiand.com/jp/inference)** というサービス。

一言で言うと「**OpenAIやClaudeの代わりに使えるAPI**」。


嬉しいポイントは：
- **国内サーバーで動く**（データが日本から出ない）
- **OpenAI互換**なので既存コードがそのまま動く
- **DeepSeekなどオープンモデルが安く使える**
- **キャンペーン期間中は¥8,000分無料で試せる**（クレカ不要）

今なら、APIの学習目的で試すのにちょうどいいし、本番運用でも**OpenAIやClaudeより安く使える**のがコスパ的に魅力。

## 実際に動かしてみる

### 環境準備

必要なライブラリはこれだけ：

```bash
pip install openai
```

ai&はOpenAI互換なので、openaiライブラリがそのまま使える。

### コードを書く

`test_aiand.py` というファイルを作って以下を貼る：

```python
from openai import OpenAI

client = OpenAI(
    api_key="ここにai&のAPIキー",  # 先ほどコピーしたAPIキーに変える
    base_url="https://api.aiand.com/v1"  # このurlをそのまま使ってください
)

response = client.chat.completions.create(
    model="openai/gpt-oss-120b",  # 使うモデルを指定
    messages=[
        {"role": "user", "content": "こんにちは！自己紹介して"}
    ]
)

print(response.choices[0].message.content)
```

#### 使えるモデル
下のモデルのうち好きなモデルを「model=」に入れて、選択できる。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3760884/e41cabbb-c152-4065-b078-ac0693b33775.png)


### 実行

```bash
python test_aiand.py
```

これだけ。今回の場合、AIから以下のような返答が返ってきた。

```bash
(.venv) PS N:\work\ai&> python test_aiand.py
こんにちは！ChatGPTです。OpenAIが開発した大規模言語モデルで、皆さんの質問に答えたり、情報を提供したり、会話を楽しんだりするのが得意です。日本語はもちろん、英語や他の言語でもお手伝いできます。好きなことは本を読むことと新しいことを学ぶこと、そして皆さんと楽しく交流することです。何か聞きたいことや相談したいことがあれば、遠慮なく教えてくださいね！
```



## `role`って何？

コードの中に `"role": "user"` ってあったけど、これは「誰の発言か」を示すもの。

| role | 意味 |
|---|---|
| `user` | 人間（あなた）の発言 |
| `assistant` | AIの返答 |
| `system` | AIへの事前指示（キャラ設定など） |

たとえばこんな使い方もできる：

```python
messages=[
    {"role": "system", "content": "あなたは関西弁で話すAIです"},
    {"role": "user", "content": "こんにちは"},
]
```
AIから以下のような返答が返ってきた。

```bash
(.venv) PS N:\work\ai&> python test_aiand.py
こんにちはやで！今日はどんなことを話したいんや？何でも聞いてや♪

#うん、なんか変な関西弁だけど(笑)それっぽく返してくれている。
```


ブラウザ版のChatGPTは裏側で自動的にやってくれてるけど、APIだと自分で書く必要がある。
