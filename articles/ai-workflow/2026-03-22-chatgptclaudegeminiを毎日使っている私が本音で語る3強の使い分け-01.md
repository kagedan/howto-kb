## プロローグ｜3つ全部使って分かったこと

「結局、AIってどれがいいんですか」

この質問を、毎週のように受ける。

正直に答える。**「何をやるか」によって全部違う。そして全部使って初めて、それが分かる。**

■結論（先に知りたい人へ）  
・万能に使いたい → ChatGPT  
・文章・思考を深めたい → Claude  
・Google中心で仕事している → Gemini

私はChatGPT・Claude・Geminiを毎日使っている。それぞれに課金している。月額にすると相当な金額だ。でも後悔していない。3つを使い分けることで、**議事録・調査・資料作成の時間が大幅に短縮され、仕事の生産性が体感で1.5倍になったからだ。**

この記事は「どれが最強か」という話ではない。\*\*「あなたの仕事に、どれがどう効くか」\*\*の話だ。

コメント欄で医師の方から「実務や趣味に活かしたい」という声をいただいた。PMだけでなく、医師・ライター・経営者——様々な職種の方に使える内容にした。

## 第1章｜3強を正直に比較する

まず一覧で整理する。

**ChatGPT（最新モデル）**

**Claude（Claude Sonnet）**

**Gemini（Google）**

**3つを比べて気づいたこと——「設計思想が全然違う」**

ChatGPTは「できることを増やす」方向に進化している。Claudeは「考える精度を上げる」方向に進化している。Geminiは「Googleの世界に溶け込む」方向に進化している。

だからこそ、用途によって使い分けるのが正解だ。

■3強比較まとめ

ChatGPT：万能・機能の広さ  
Claude：思考・文章の深さ  
Gemini：Google連携・業務統合

## 第2章｜ChatGPTが向いている人・場面

**ChatGPTを選ぶべき人：「とにかく幅広く使いたい初心者」「画像を作りたい人」「コードを動かしたい人」**

ChatGPTの最大の価値は「一つで何でもできる」ことだ。

**具体的な使い場面：**

最新ニュース・法改正の調査——ウェブ検索機能でリアルタイム情報を取得できる。「2026年のセキュリティ関連法の動向を教えて」という問いに、今日の情報で答えてくれる。

画像生成——DALL-Eとの連携で、プレゼン資料の挿絵、SNS用のビジュアルをテキストから生成できる。

コード実行——Pythonコードを書いて、その場で実行して結果を確認できる。Excelの自動化、データ分析、グラフ作成まで一気にできる。

**PMとしての使い方：**

```
「最新のPMBOK改訂内容を教えてください」→ 最新情報をウェブから取得
「このデータをグラフ化して」→ コードを書いて実行・画像で返してくれる
```

**医師としての使い方：**

```
「最新の○○治療ガイドラインの改訂点を教えて」→ 最新医療情報の把握
「この論文のデータを分析して」→ コードインタープリターで統計処理
```

**一つだけ選ぶなら、ChatGPTから始めるのをすすめる。** 理由はシンプルで、できることが最も多いからだ。

## 第3章｜Claudeが向いている人・場面

**Claudeを選ぶべき人：「文章を書く人」「深く考えたい人」「長い文書を読み込ませたい人」**

Claudeの最大の価値は「考える深さ」と「日本語の精度」だ。

**具体的な使い場面：**

長文の読み込み・要約——100ページのPDF、長い契約書、膨大な議事録をそのまま貼り付けると、驚くほど正確に要点を抽出してくれる。文脈を保持する精度が3強の中で最も高い。

文章の下書き・推敲——提案書、報告書、NOTEの記事——日本語として「読みやすい文章」を返してくる。微妙なニュアンス、言葉の選び方、文章のリズムがいい。この記事もClaudeと一緒に書いている。

思考の壁打ち——「このプロジェクトの潜在的なリスクは何か」「この判断の抜け漏れはないか」——自分の思考の抜け漏れを補ってくれる存在として使うと、真価を発揮する。

**PMとしての使い方：**

```
「この議事録を決定事項・TODO・懸案事項に整理して」→ 30秒で完璧な議事録
「このリスクリストの抜け漏れを指摘して」→ 気づかなかった観点が出てくる
```

**医師・ライターとしての使い方：**

```
「この論文の主張を患者向けに分かりやすく書き直して」→ 専門用語を平易な日本語に
「この章の論理展開に矛盾はないか確認して」→ 小説・論文の推敲に
```

**「隣人AI」のような、AIとの対話を作品にする試み——これはClaudeとやると面白いかもしれない。** 長い会話の文脈を保ちながら、人格のような一貫性を持って対話してくれるからだ。

## 第4章｜Geminiが向いている人・場面

**Geminiを選ぶべき人：「Google製品を日常的に使っている人」「検索と一体化したAIが欲しい人」**

GeminiはGoogleのエコシステムに統合されている——これが最大の強みだ。

**具体的な使い場面：**

Gmail連携——メールの要約、返信案の自動生成、重要メールの抽出。受信トレイを開きながらGeminiが動いてくれる。

Google Docs連携——ドキュメントを書きながらその場でAIに相談できる。ChatGPTやClaudeは「別のウィンドウで使う」のに対し、Geminiは「作業の中に溶け込む」感覚がある。

Google検索との連携——検索結果をそのままAIが整理してくれる。「調べる」と「整理する」が一体化している。

**Geminiが特に強い場面：**

```
Googleカレンダーと連携して「今週のスケジュールを整理して」
Google Driveの資料を読み込んで「この資料の要点を教えて」
Gmailを読んで「返信が必要なメールを優先順位付けして」
```

**ただし正直に言う。** 文章の深みと思考の精度では、ClaudeやChatGPTと比べると、用途によっては最適解になりにくい。Geminiは「Googleを使い倒している人の補助AI」として使うのが最もコスパが高い。

## エピローグ｜AI選びは「自分の仕事を知ること」だ

3つのAIを使い続けて気づいたことがある。

**「どのAIを選ぶか」は、「自分が何に時間を使っているか」を知ることと同じだ。**

文章を書く時間が長い人はClaude。情報を調べる時間が長い人はChatGPT。Googleで仕事が回っている人はGemini——そういうシンプルな話だ。

AIは道具だ。道具を選ぶ前に、自分の仕事を棚卸しする。その棚卸し自体に、AIを使うのが一番早い。

**「あなたの一日の仕事を教えてください。最適なAIの組み合わせを提案します」**——こうClaudeに聞いてみてほしい。きっと面白い答えが返ってくる。
