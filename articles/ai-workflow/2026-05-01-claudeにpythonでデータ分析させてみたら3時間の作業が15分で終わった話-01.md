こんにちは、りゅうきです。

先日、友人から「売上データを分析してほしい」と頼まれたんですが、正直データ分析って苦手で…😅 エクセルでグラフ作るだけでも時間かかるのに、どうしようかと思っていました。

そこで「Claudeにやってもらおう！」と思い立ち、Pythonでデータ分析をお願いしてみたところ、びっくりするほど簡単にできちゃいました✨

  

## やってみたこと

まず、友人からもらったCSVファイル（月別売上データ）をClaudeに見せて、こんな風にお願いしました：

```
このCSVファイルの売上データを分析して、以下をやってほしいです：
1. 月別売上の推移をグラフで可視化
2. 売上の増減率を計算
3. 今後3ヶ月の予測値を出す
```

Pythonコードで書いて、実行できるようにしてください。  
初心者でもわかるようにコメントも入れてくれると嬉しいです😊  
```

するとClaudeが、なんと完璧なPythonコードを書いてくれました！

[画像: Claudeとのやり取りのスクリーンショット]

コードの中身は、pandasでデータを読み込んで、matplotlibでグラフを作って、scikit-learnで予測するという本格的なもの。私が一人でやったら絶対に数時間かかってました💦

## 結果

できあがったものがこちら：

\*\*15分で完成した分析レポート\*\*  
- 売上推移の美しいグラフ  
- 前月比の増減率（自動計算）  
- 来月から3ヶ月間の予測売上  
- 季節性の分析まで！

[画像: 完成したグラフや分析結果の画面]

友人も「こんなに詳しく分析してもらえるなんて！」と大喜び😊 普通にデータ分析会社に頼んだら何万円もかかりそうな内容が、無料でできちゃいました。

特に驚いたのは、Claudeが勝手に季節性を見つけて「12月は例年売上が上がる傾向がありますね」なんて解説まで付けてくれたこと。私では絶対に気づけませんでした💡

## コツ・注意点

今回やってみて分かったコツをシェアしますね：

\*\*1. データの形を先に説明する\*\*  
「CSVファイルには日付、商品名、売上金額の列があります」みたいに、データ構造を最初に教えてあげると、より正確なコードを書いてくれます。

\*\*2. 欲しい結果を具体的に伝える\*\*  
「グラフを作って」だけじゃなく「月別の売上推移を折れ線グラフで、タイトルと軸ラベルも付けて」と詳しくお願いすると、期待通りのものができます🔥

\*\*3. エラーが出ても慌てない\*\*  
最初にライブラリのインストールでエラーが出ましたが、エラーメッセージをClaudeに見せたら「pip install pandas matplotlib scikit-learn」で解決してくれました。

## もっと詳しく知りたい方へ

今回使ったプロンプトの完全版や、他のデータ分析パターン（顧客分析、売上予測、A/Bテストなど）もまとめたプロンプト帳を作りました！

「データ分析なんて無理…」と思っている非エンジニアの方でも、プロンプトをコピペするだけで本格的な分析ができるようになります✨

データドリブンな意思決定って、今の時代すごく重要ですよね。でもそのためにプログラミングを一から学ぶのは大変すぎる…。
