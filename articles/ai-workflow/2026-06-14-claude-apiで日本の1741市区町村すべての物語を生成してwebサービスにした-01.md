半年かかった。  
「京都や東京の次、どこに行けばいい？」という問いに、ちゃんと答えられるものがないと気づいたのが始まりだった。  
なぜ作ったのか  
外国人の友人から、よく聞かれる。  
東京も京都も行ったあとで、「次はどこがいい？」と。彼らが求めているのはランキングじゃなくて、どこかに連れて行ってくれるような話だと思う。でも既存の旅行サイトには、過疎地のことはほとんど何も書いていない。  
書かれていない場所に、日本の本質がある気がした。  
もうひとつ、個人的な興味もあった。移住や多拠点生活を漠然と考えはじめていたとき、「どこに行けばいいか」より先に「その土地がどんな場所か」を知りたかった。数字やランキングではなく、物語として。  
だから作った。全部書いてやろうと思って。  
何を作ったか  
**[AURA Poetry](https://aurapoetry.com)** というWebサービスだ。  
日本全国1,741市区町村と176の有人離島を、文章で描いたプラットフォーム。観光情報は書かない。「ここが一番おすすめ」みたいな順位もつけない。ただ、その場所が何者かを書く。旅行ガイドというより、本の章として場所を読む感覚に近い。  
日本語は短い断言と体言止めで、英語はもう少しゆったりした観察者の視点で書いている。同じ場所でも、言語によって少し違う読み心地になるようにした。  
Claude APIで全自治体の文章をどう生成したか  
ここが一番聞かれそうなところなので、少し詳しく書く。  
対象は1,741市区町村＋176離島、合計1,900件近く。これを手で書くのは不可能なので、Claude APIを使ってバッチ生成した。  
プロンプト設計で一番こだわったのは「トーン」だ。  
観光紹介になってはいけない。「〇〇が有名です」「アクセスは〜」みたいな文章は書かせたくなかった。場所の空気感、歴史の重さ、そこに住む人間の気配——そういうものを散文として書かせたかった。  
プロンプトには文体の指示を細かく入れた。日本語版と英語版でそれぞれ別のトーン設定にしている。同じ自治体のデータを渡しても、言語によって全然違う文章が出てくる。それが面白かった。  
入力データはWikipediaの記事要約＋独自収集した構造化データ（文化財・温泉・山・湖・港・鉄道など）を組み合わせた。データが薄い過疎地ほど、LLMの「想像力」に頼る部分が増える。それが逆に良い文章になることもあった。  
生成した文章は全件レビューしていない。スポットチェックと品質フィルタリングを組み合わせた。1,900件を全部読むのは現実的ではないし、そもそもLLMが書いた文章の品質は思ったより安定していた。  
技術スタック  
シンプルにまとめると：

バックエンド: Django 5.2 on Heroku Basic dyno（512MB）  
DB: Postgres Essential-1  
地図: MapLibre GL JS + pmtiles  
CDN/SSL: Cloudflare Full (strict)  
静的配信: WhiteNoise  
インフラ費用: 月約¥2,650

Heroku Basic dynoは512MBしかない。重い処理はローカルで実行してJSONでエクスポートし、管理コマンドで本番にインポートするパターンで乗り切っている。  
地図まわりで一番苦労したのはデータ量だ。  
最初はGeoJSONで全国の市区町村ポリゴンを配信していた。16.5MB。当然重い。  
2025年にMapLibre GL JS + pmtilesに移行した。pmtilesはベクタータイルをひとつのファイルにまとめてRange Requestで配信できる形式で、必要なタイルだけを取得できる。結果、1リクエストあたり17KB程度まで落ちた。FCP（First Contentful Paint）も約4.4秒から3.2秒に改善した。  
地図タイルの生成にはtippecanoeを使った。文字列のfeature IDを数値に変換する必要があるなど、細かいハマりポイントがいくつかあった。  
作っていて見えてきた日本  
1,741の市区町村に向き合っていると、ガイドブックには絶対に載らないものが見えてくる。  
鉄道が一度も来たことのない村がある。  
奈良県十津川村。面積は奈良県の5分の1を占める日本最大の村なのに、鉄道が通ったことがない。バスで数時間かかる。それは「不便」ではなく、「時間の流れ方が違う」という話だった。  
人口が0人になりかけた離島で、若者が増えている。  
島根県の海士町（あまちょう）。隠岐諸島の小さな離島だが、島留学制度で毎年数十人の若者がやってくる。人口2,000人台の島に、外から人が来ている。  
2ヶ月間、毎夜踊り続ける祭りがある。  
岐阜県郡上おどり。7月下旬から9月上旬、夏の間じゅう踊る。最終日の徹夜踊りは夜通し明け方まで。観光のためではなく、町の人間が踊り続けるために続いている祭りだ。  
こういうものを見つけるたびに、日本ってまだこんなに知らないことだらけだったのかと思う。  
現在の状態

全1,741市区町村：物語・地図・文化財・温泉・美術館・山・湖・港・鉄道などのデータを統合済み  
176有人離島：個別ページ、物語生成完了  
イベント252件：郡上おどり・おわら風の盆・なまはげ柴灯まつりなど、観光過密地を避けて地方優先で選定  
英語版：全ページで日英切替可能

使っているのは今のところほぼ外国人だ。Xで英語で投稿したら、海外からのアクセスが増えた。それが一番嬉しかった。
