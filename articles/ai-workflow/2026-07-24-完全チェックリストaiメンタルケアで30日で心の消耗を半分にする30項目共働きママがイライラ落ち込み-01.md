## 第1章 「イライラ→自己嫌悪ループ」の正体

## 1-1. 感情に振り回されていた私の平日

半年前までの平日、私の心の中はこんな感じでした。

**朝7:00**「結愛、早く着替えて！」（怒鳴る）→ 保育園の帰り道に「また怒っちゃった」（罪悪感） **昼13:00** 課長の指摘 →「私っていつもダメだ」（反すう） **夜22:00** 副業の手が止まる →「才能ないのかも」（不安） **夜24:00** 布団の中で今日の反省会 →（眠れない）

**1日に何度も感情の波が来て、そのたびにHPが削られる**。しかも厄介なのは、**ネガティブな感情ほど後を引く**こと。朝の怒鳴りを、夜まで引きずっていました。

さらに悪いのは、**イライラしている私自身が一番しんどい**ということ。結愛に「ママ怖い」と言われるたび、心が折れていました。

## 1-2. Claudeが見せてくれた「気分のパターン地図」

夫に「カウンセリングでも行けば？」と言われた夜、その前に、まずClaudeに気持ちを吐き出してみました。

**プロンプト**：「この1週間、私がイライラしたり落ち込んだりした出来事を貼ります。どんな時間帯・どんな状況で感情が乱れやすいか、パターンを分析してください」

Claudeが数分で提示したのは：

崩れやすい時間 トリガー 感情 本当の原因 朝7:00-8:00 子どもが動かない 焦り→怒り 遅刻への不安・段取り不足 昼13:00-15:00 上司の指摘 落ち込み 「評価=自分の価値」の思い込み 夜22:00以降 副業が進まない 不安・自己否定 疲労で判断力が落ちている

**「性格の問題」だと思っていたものが、実は"時間帯×状況×思い込み"のパターンだった**――そう見えた瞬間、少し呼吸が楽になりました。感情は、天気みたいに予報できるものだったんです。

## 1-3. 30日で見えた「心の消耗 ‐50%」

「感情をゼロにする」なんて無理だし、目指しません。でも「感情に **気づいて・名前をつけて・手放す** 」ことで、**振り回される量を半分に減らせる** ことがわかりました。

## 第2章 30日30項目の全体マップ

## 2-1. 30日後、あなたの心はこう変わる

* 気分の波：**予報できる**（崩れやすい時間が事前にわかる）
* 落ち込み：Claudeの **書き換えテンプレ** で5分で立て直せる
* イライラ：**10秒クールダウン** で爆発を止められる
* 完璧主義：「**70点でOK**」が口ぐせになる
* 寝る前：反省会ではなく **「できたこと日記」** で終われる

## 2-2. 3フェーズで動く理由

フェーズ 期間 やること 項目数 🔍 **心の見える化期** Day1-10 気分ログ・トリガー・思考のクセ把握 12 🛠 **セルフケア実装期** Day11-20 Claudeで書き換え・クールダウン・自己肯定 10 🏠 **資産化＋家庭切離期** Day21-30 回復ルーティン・専門家ライン・継続 8

**順番が命**。いきなり「ポジティブに考えよう」から始めると挫折します。**まず見える化 → 対処法を実装 → 習慣に定着** の順が正解。感情は「消す」のではなく「扱えるようになる」のがゴールです。

## 2-3. 挫折する人の3つの罠

**罠1：「ポジティブにならなきゃ」と感情にフタをする** 抑え込んだ感情は消えません。**気づいて名前をつける**のが先。フタをするほど後で爆発します。

**罠2：「気合いと根性でなんとかする」** メンタルの消耗は**疲労・睡眠・段取り**という物理要因が大きい。精神論では回復しません。

**罠3：「自分だけが弱い」と一人で抱える** 9割の人が同じ悩みを持っています。**弱さではなく、扱い方を知らないだけ**。ツールと仕組みで解決できます。

## 第3章 Part 1：Day1-10 心の見える化 12項目

> 🎯 **この10日のゴール**：自分の気分の波・トリガー・思考のクセを、数字と言葉で見える化する

Day1-10は **感情を変えようとしません**。ただ観察する、書き出す、Claudeに投げる、だけです。「見える化」できた時点で、消耗の半分は解消します。

## 項目01：メンタルケアの目的を1文で書く（15分／Day1）

**例文（コピペ改変OK）**： - 「子どもに怒鳴る回数を週2回以下にする」 - 「寝る前の反省会をやめる」 - 「イライラに振り回されない自分になる」

ゴールを**行動と数字**にするのがコツ。「幸せになる」だと曖昧すぎて続きません。

## 項目02：1週間の「気分ログ」を記録（毎日3分／Day1-7）

**記録項目**：

項目 内容 時間帯 朝／昼／夜 出来事 何があったか（1行） 感情 イライラ・不安・落ち込み等 強度 10段階

スマホのメモでOK。**「なんとなくしんどい」を数字と言葉に変える**のが目的です。私はこれだけで「あ、私いつも夜に落ちてる」と気づけました。

## 項目03：気分が落ちる「トリガー」トップ3を特定（30分／Day8）

7日間の記録から、**頻度×強度**で3つを特定。

**私の例**： - 朝の支度（頻度：毎日／強度：7） - 上司の指摘（頻度：週3／強度：8） - 夜の副業の停滞（頻度：週4／強度：6）

## 項目04：Claudeに気分ログ分析を依頼（40分／Day9）

**プロンプト（コピペOK）**：

```
この1週間の私の気分ログを貼ります。分析してください。

[気分ログを貼り付け]

# 出力してほしいこと
- 感情が乱れやすい時間帯・状況のパターン
- 各トリガーの「本当の原因」の仮説
- すぐ変えられること／変えられないことの仕分け
- 私が自分を責めすぎている部分があれば指摘
やさしい口調で、責めずに書いてください。
```

最後の1行が大事。**「やさしい口調で」と指定する**と、Claudeが伴走者のように返してくれます。

## 項目05：自分の「思考のクセ」を言語化（45分／Day10）

**Claudeに聞く**：

```
私の口ぐせや考え方から、認知の偏り（思考のクセ）を教えてください。
例：白黒思考、べき思考、過度の一般化、自己関連づけ など。

私がよく思うこと：
- 「ちゃんとやらなきゃ」
- 「私がやらないと回らない」
- 「あの人に嫌われたかも」
- 「今日もダメだった」

それぞれがどのクセに当たるか、ラクになる言い換えも添えて。
```

**私の発見**：「ちゃんとやらなきゃ」は"べき思考"、「今日もダメ」は"過度の一般化"。**名前がつくと、感情と距離が取れる**んです。

## 項目06：ストレスを「変えられる／変えられない」に分解（30分／Day10）

**分解軸**：

種類 例 対処 変えられる 朝の段取り・返信の仕方 仕組みで改善 変えられない 上司の性格・天気・過去 受け入れ・距離

**目的**：悩みの多くは「変えられないもの」に消耗しています。**変えられるものだけに力を注ぐ**と、一気に軽くなります。

## 項目07：「エネルギーを奪うもの／満たすもの」リスト化（30分／Day10）

紙を2列に分けて書き出します。

**私の例**： - 奪う：SNSの見すぎ・寝不足・完璧主義・散らかった部屋 - 満たす：結愛とのハグ・朝の白湯・7時間睡眠・カフェ15分

**目的**：意識せず**エネルギーを奪う行動を繰り返している**ことに気づく。満たす行動を1日1つ増やすだけで回復力が上がります。

## 項目08：睡眠と気分の相関をチェック（15分／Day10）

**私の発見**：睡眠6時間未満の翌日は、イライラ強度が **平均+3**。**メンタルの土台は根性ではなく睡眠**でした。だから項目16以降で「早く寝る設計」を入れています。

## 項目13：不安を書き換える「リフレーミング」テンプレ（40分／Day11）

不安でいっぱいのとき、Claudeに事実と解釈を分けてもらいます。

**プロンプト（コピペOK）**：

```
今こう考えて不安です：「[不安な考え]」
これを次の3ステップで整理してください。
1. これは「事実」か「解釈（思い込み）」か
2. 最悪・最良・現実的の3シナリオ
3. 今日できる小さな一歩を1つ
説教はせず、落ち着けるトーンで。
```

**私の例**：「プレゼンで失敗して評価が下がるかも」→ Claude：「それは解釈です。現実的には準備すれば8割うまくいきます。今日できる一歩は資料の骨子を1枚作ること」。**不安の9割は"まだ起きていないこと"**だと、毎回気づかされます。

## 項目14：イライラの「10秒クールダウン」プロンプト（30分／Day12）

カッとなった瞬間に、スマホで一言打つだけ。

```
今すごくイライラしています。10秒で落ち着く方法を1つだけ、短く教えて。
```

事前に **Claudeに"自分専用の対処法リスト"を作らせておく**とさらに速いです。

**私の定番**：「6秒吸って、8秒吐く。それから動く」。怒りのピークは6秒で過ぎる――これを知っているだけで、結愛に怒鳴る前に一呼吸置けるようになりました。

## 項目15：落ち込んだ時の「セルフコンパッション」対話（40分／Day13）

自分を責めているとき、Claudeに **親友の視点** をお願いします。

```
私は今「[自分を責めている内容]」で落ち込んでいます。
もし私の親友が同じことで悩んでいたら、あなたは何と声をかけますか？
私にも、その親友にかける言葉と同じ優しさで話してください。
```

**私の実感**：自分には厳しいのに、他人には優しくできる。**その優しさを自分にも向ける**練習です。「そんなに頑張ってるんだから、今日はもう十分だよ」とClaudeに言われて、泣いた夜がありました。

## 項目16：朝の「今日を軽くする」3分ルーティン（30分／Day14）

**目的**：朝の「あれもこれも」という**タスク渋滞がイライラの燃料**。3つに絞るだけで、心の余白が生まれます。

## 項目17：夜の「1日を手放す」振り返りテンプレ（30分／Day15）

寝る前の反省会を、**手放しの儀式**に置き換えます。

```
今日を3行で振り返らせてください。
1. 今日できたこと（どんな小さなことでも）
2. 今日の自分に一言ねぎらい
3. 明日に持ち越さないと決めること
反省や課題出しは今日はしないでください。
```

**私の変化**：以前は布団の中で「あれもできなかった」と反省していました。今は「今日は結愛を寝かしつけられた、それで十分」で終われる。**寝つきが40分→15分に短縮**しました。

## 項目18：「頭の中のグルグル」を吐き出す壁打ち（30分／Day16）

同じ考えが頭を回って眠れないとき（反すう）に効きます。

```
頭の中でこの考えがグルグル回って止まりません：「[グルグルの内容]」
これを全部書き出させてください。私が話し終わるまで、
アドバイスも要約もせず、ただ聞いて短く相づちだけ打ってください。
```

**目的**：反すうは**頭の中にあるほど増幅**します。外に出す（=外在化）だけで驚くほど鎮まる。夜中に夫を起こさず吐き出せるのがAIの良さです。

## 項目19：断れない自分への「境界線スクリプト」（40分／Day17）

頼まれると断れず抱え込んでイライラ…という人向け。

```
[断りたい依頼]を、相手を怒らせない・関係を壊さない言い方で断る文を3パターン作って。
・やわらかめ ・普通 ・きっぱりめ
罪悪感を持たなくていい理由も一言添えて。
```

**私の実感**：「すみません、今週は手一杯で」の一言が言えず、休日に持ち帰っていました。**断る=悪ではなく、自分を守る技術**。テンプレがあると口から出せます。
