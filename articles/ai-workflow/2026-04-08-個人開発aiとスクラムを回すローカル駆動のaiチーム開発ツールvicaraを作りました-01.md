# 開発のきっかけ：AI将軍（AI Shogun）への感銘

vicaraの開発の原点として、以前話題になった[AI将軍（AI Shogun）](https://zenn.dev/shio_shoppaize/articles/5fee11d03a11a1)のような、AI同士が協調して開発を進めるマルチエージェントのプロジェクトから非常に大きな感銘を受けました。

「人間が抽象的な指示を出し、AIがAIをマネジメントしながらタスクをこなす」という未来のワークスタイルを、自分なりのローカルツールとして落とし込んでみたいと考えたのが開発のスタートです。

# vicara の主要機能

vicaraの最大の特徴は、役割を持ったAIエージェントたちがチームの一員としてアジャイル開発をサポートしてくれる点です。

## 1. POアシスタント

プロダクトオーナーの右腕となるエージェントです。  
曖昧なアイデアや要件を壁打ち感覚で相談すると、AIが「インセプションデッキ」の作成や「バックログの作成、タスク（チケット）の細分化や優先度の決定」を行ってくれます。人間はスプリントプランニングの意思決定のみに集中できるようになります。  
こちらは現在APIベースで動かしています。使いすぎ注意。

## 2. 開発エージェントによる自律的なタスク実行

POアシスタントが整理したチケットをもとに、今度は開発エージェントが実装や調査をサポートします。  
Claude Code CLIなどと連携し、エージェントが自律的に動いて技術的な課題解決やコードの提案を行ってくれる「マルチエージェント実行」の仕組みを取り入れています。  
（Claude Code CLIなのは、サブスクで実行できるからです！）

## 3. Git Worktree Review機能

vicaraならではの機能として「Git Worktree Review」を実装しています。  
現在の作業ブランチ（ワーキングツリー）を汚すことなく、AIが裏側でGit Worktreeを使って別ブランチを展開し、コードレビューや差分チェックを実行してくれます。

「自分の作業を中断せずに、別タスクのPRレビューをAIに任せる」といったシームレスな開発体験が可能です。もし並行開発でコンフリクトが発生した場合でも、ユーザーはワンボタンで「AIに修正実行を依頼する」か「破棄する」などを選択でき、柔軟に対応できます。

## 4. アバター設定機能

開発エージェントやPOアシスタントには、好きな画像を設定してアバター化することができます（デフォルト画像も用意しています）。  
無機質なツールではなく、色んなキャラクターと一緒にチームを組んで開発しているような、ワクワクする雰囲気を味わうことができます！

# 開発の裏話：API制限との果てなき戦い

AIエージェントを自律駆動させるツールを開発する上で、避けて通れないのが**LLMのAPI制限（Rate Limit）問題**です。

私自身、ClaudeのProプランを契約しているのですが、賢いモデル（Sonnetなど）にエージェントの自律ループを任せると、あっという間に上限に到達してしまいます。  
そのため、vicara自体の開発や検証においては、Antigravity、Codex、Geminiでの要件作成など、様々なAIコーディングツールを反復横跳びしながら進めています。「AIツールを作るために、複数のAIツールを駆使して制限を回避する」という荒業で、なんとかここまでこぎつけました。

# 現在のステータス（まだまだ未安定です🙇‍♂️）

このツールは現在もアクティブに開発中ですが、**正直なところまだまだ未安定な部分が多いです。**

* 作者がWindows環境しか持っていないため、Macでの動作確認ができていません…（一応動くようには作っています！）
* エッジケースでのバグが残っている可能性があります
* 今後のアップデートで破壊的な変更（仕様変更やデータ構造の変更など）が入る可能性が高いです

現状は「実業務のメインツールとしてガッツリ使う」というよりは、「AIと一緒に開発する未来の体験を、ローカル環境でいち早く試せるおもちゃ」として触っていただけると嬉しいです！

※ご利用には、Anthropic / Gemini のAPI Key、およびClaude Code CLIなどのセットアップが必要です。詳細はリポジトリのREADMEをご確認ください。

# 使用技術

ローカルでサクサク動くデスクトップアプリにするため、Tauriを採用しています。

* **フロントエンド:** React 19 (Vite) / TypeScript / Tailwind CSS v4 / Shadcn UI / React Context API
* **バックエンド:** Tauri v2 / Rust / sqlx (SQLiteでデータをローカル保存)
* **AI・LLM連携:** rig-core / Anthropic API / Gemini API
* **その他:** portable-pty (ターミナル統合) / Claude Code CLI / Git Worktree
