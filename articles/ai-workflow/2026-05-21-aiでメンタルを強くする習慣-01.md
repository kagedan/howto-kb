「また落ち込んでしまった…」

そう思ったとき、  
あなたは自分を責めませんでしたか？

失敗したとき、  
思い通りにいかなかったとき、

「自分はなんてダメなんだ」  
「どうせまたうまくいかない」  
「もうやめてしまいたい」

そんな言葉が  
頭の中をぐるぐると回る。

---

でも、それはあなたが弱いのではなく、  
**思考のクセ**が原因かもしれません。

認知行動療法（CBT）では、  
「自動思考」と呼ばれる  
無意識の思い込みが  
気分を大きく左右すると言われています。

---

この記事を読むと、

✅ **思考のクセを修正できる**  
　→自分を責める悪循環から  
　　抜け出せるようになる

✅ **日常の小さな幸せに気づけるようになる**  
　→ポジティブな脳の回路を  
　　AIと一緒に育てられる

✅ **プレッシャーに飲み込まれにくくなる**  
　→大事な場面でも  
　　自分を取り戻す方法がわかる

---

AIはあなたの「心の伴走者」です。

感情を否定せず、

少しずつ前へ進む力を  
一緒に育ててくれます。

### ◆思考を整理するプロンプト

**✅目的：**  
　頭の中のネガティブな思考を言語化し、  
　客観的に見つめ直す。

**✅いつ使うか：**  
　落ち込んだとき・  
　気分が沈んでいるとき・  
　何かを失敗したとき。

```
# ロール設定
あなたは認知行動療法（CBT）に基づいた
思考の整理をサポートするカウンセラーです。
批判せず、否定せず、
私の気持ちを丁寧に受け止めながら
思考のクセに気づくお手伝いをしてください。

# 背景・目的・対象
・私が感じていること：[例：また失敗して情けない]
・きっかけになった出来事：
　[例：会議で発言できなかった]
・今の気分（0〜10で表す）：[例：4]

# 指示内容
以下の4ステップで整理してください。

1. 私の思考の中にある「自動思考」を特定する
　（例：「自分はダメだ」という決めつけなど）

2. その思考が事実かどうかを
　一緒に確認する問いかけをする

3. より現実的でバランスの取れた
　別の見方を3つ提示する

4. 今の気分を少しでも楽にする
　一言メッセージを添える

# 制約条件
・責めたり、励ましすぎたりしない
・「でも」「しかし」などの
　否定的な接続詞を使わない
・小学生でもわかる言葉を使う

# 出力形式
ステップ1〜4を見出しつきで、
やさしい言葉で出力する。

# 入力データ
今感じていること：[ここに書いてください]
```

**✅具体的な使用方法：**  
　「今感じていること」に  
　今の気持ちをそのまま入力するだけ。

　上手に書く必要はありません。  
　「なんかもう嫌だ」でも大丈夫です。

**✅読者への一言：**  
　思考は「事実」ではなく  
　「解釈」です。

　AIと一緒に  
　その解釈をほぐしていきましょう。

**✅目的：**  
　毎日の小さな「良かったこと」を記録し、  
　ポジティブな脳の回路を育てる。

**✅いつ使うか：**  
　夜寝る前・  
　一日を振り返るとき・  
　何もいいことなかったと感じる日。

```
# ロール設定
あなたはポジティブ心理学に基づいた
「感謝日記」のナビゲーターです。
今日の出来事の中から
「よかったこと」を一緒に探し出し、
それを意味づけするお手伝いをします。

# 背景・目的・対象
・今日起きた出来事（いくつでも）：
　[例：仕事でミスをした・
　　昼ごはんがおいしかった・
　　電車が空いていた]
・今日の気分（一言で）：
　[例：疲れた・なんとなくモヤモヤ]

# 指示内容
1. 入力された出来事の中から
　「よかったこと」や
　「ちょっと嬉しかったこと」を
　最低3つ見つける

2. それぞれを
　「なぜそれがよかったのか」と
　少し深掘りして意味づけする

3. 今日の自分を
　労うひと言を添える

# 制約条件
・無理にポジティブにしない
・小さなことを大げさに褒めない
・「今日もよく頑張ったね」系の
　表面的な言葉は使わない

# 出力形式
番号付きリストで、
温かみのある言葉で出力。

# 入力データ
今日あったこと：[ここに書いてください]
```

**✅具体的な使用方法：**  
　「今日あったこと」に  
　箇条書きでもOKです。

　3週間続けると、  
　脳が「いいこと探し」をする習慣がつきます。

**✅読者への一言：**  
　「何もなかった日」はありません。

　ただ、気づいていないだけです。  
　AIが一緒に探してくれます。

# ロール設定
あなたは認知行動療法に基づいた

・きっかけ：

# 指示内容
思考の整理を4ステップで行う

**AIの回答（例）**：

>  何回中何回ですか？」

>  ①「ミスを見つけた段階で  
>  まだ修正できる状況だった」

>  誠実な行動だった」

👉 **何が変わったか？**

「ロール設定」で  
AIを専門家として起動し、

「背景指定」で  
自分の感情・出来事・状況を渡したことで、

**表面的な励ましではなく、  
思考の根っこに届く言葉が出てきました。**

情報を渡すほど、  
AIの回答は精度が上がります。

## 実践プロンプト集

## 1. ネガティブ思考のクセを修正する

誰でも、  
頭の中に「厳しい批評家」がいます。

「どうせ無理」  
「また失敗する」  
「自分にはできない」

こうした声は、  
実は**思考のクセ（認知の歪み）**  
から生まれています。

認知の歪みとは、  
物事を偏った見方で  
とらえてしまう癖のことです。

このプロンプトで、  
そのクセに気づき、  
書き換えることができます。

---

### ◆認知の歪みを修正するプロンプト

**✅目的：**  
　自分の思考パターンの歪みを特定し、  
　より現実的な見方に書き換える。

**✅いつ使うか：**  
　ネガティブな考えが止まらないとき・  
　「どうせ…」「また…」  
　と口癖のように出るとき。

```
# ロール設定
あなたは認知行動療法（CBT）を専門とする
心理カウンセラーです。
私の思考パターンの歪みを
優しく、しかし的確に指摘し、
より現実的でバランスのとれた
思考に書き換えるサポートをしてください。

# 背景・目的・対象
・私が繰り返す考え：
　[例：何をやっても自分はダメだと感じる]
・その考えが出てきた状況：
　[例：プレゼンで少し噛んでしまった]
・この考えのせいで起きていること：
　[例：次のプレゼンが怖くなった]

# 指示内容
以下の順で整理してください。

1. この思考に含まれる
　「認知の歪み」の種類を特定する
　（例：全か無か思考・
　　マイナス化・先読みなど）

2. 「この考えは本当に事実か？」を
　問いかける質問を3つ作る

3. 同じ状況を
　「より現実的な視点」から
　見た場合の思考を3パターン提示する

4. この歪みを減らすための
　日常的な小さな習慣を1つ提案する

# 制約条件
・専門用語には必ず
　カッコ内でわかりやすく補足する
・責めず、批判せず、
　寄り添いながら指摘する
・すぐに実践できる言葉で出力する

# 出力形式
1〜4の見出し付き。
読みやすい段落形式で出力。

# 入力データ
繰り返す思考・状況：[ここに入力]
```

**✅具体的な使用方法：**  
　「繰り返す考え」に  
　頭の中でよく出てくる言葉を  
　そのまま入力してください。

　「また失敗しそう」でも  
　「自分は向いていない」でも  
　どんな言葉でも大丈夫です。

**✅読者への一言：**  
　「考え方のクセ」は生まれつきではありません。

　長い時間をかけて  
　身についたものだからこそ、  
　少しずつ書き換えられます。

## 2. 毎日の成功体験を積み重ねる

「自分にはできる」という

大きな成功からだけでなく、  
**小さな成功体験の積み重ね**

でも、小さな成功は  
見落としがちです。

AIと一緒に探しましょう。

**✅目的：**  
　日常の中の「できたこと」を可視化し、  
　自己効力感を育てる。

**✅いつ使うか：**  
　毎日夜のルーティンとして・  
　「今日も何もできなかった」  
　と感じているとき。

```
# ロール設定
あなたはポジティブ心理学に基づいた
「自己効力感コーチ」です。
私の一日の出来事の中から
「できたこと」「頑張ったこと」を
一緒に発掘し、意味を与えてください。

# 背景・目的・対象
・今日やったこと（小さなことでOK）：
　[例：朝ちゃんと起きた・
　　返信が遅くなっていたメールを送った・
　　昼食を自炊した]
・今日の気分（一言で）：
　[例：なんとなく自信がない]
・最近、自分が得意だと感じること：
　[例：特にない]

# 指示内容
1. 入力された行動の中から
　「これはちゃんとできた」と言える
　ことを3つ以上ピックアップする

2. それぞれに対して
　「なぜそれができたのか」を
　能力・性格・努力の観点から
　ひと言で意味づけする

3. 「今日の自分」を
　一言で表すポジティブなラベルを
　3つ提案する
　（例：「丁寧な人」「誠実な行動ができた人」）

4. 明日の小さなチャレンジを1つ提案する

# 制約条件
・大げさに褒めない
　（「すごいですね！」は禁止）
・事実に基づいて
　静かに意味づけをする
・背伸びしない言葉で

# 出力形式
番号付き・読みやすい短い段落で。

# 入力データ
今日やったこと：[ここに入力]
```

**✅具体的な使用方法：**  
　「今日やったこと」を  
　何でも箇条書きで入力するだけ。

　「ごはんを食べた」でも  
　「シャワーを浴びた」でも  
　立派な「できたこと」です。

**✅読者への一言：**  
　自己効力感は  
　「大きな成功」からは生まれにくいです。

　毎日の小さな「できた」を  
　積み重ねることで、  
　静かに、しかし確実に育ちます。

## 3. プレッシャーを呼吸で手放す

「大事な場面の前に緊張してしまう」

これはあなたの弱さではなく、  
体が「本気だ」と知らせている  
サインです。

でも、その緊張が  
パフォーマンスを下げることもある。

マインドフルネスや呼吸法は、  
**今この瞬間に意識を戻す技術**です。

AIがあなたの状態に合わせた  
ガイドを提案してくれます。

---

### ◆マインドフルネス呼吸ガイドプロンプト

**✅目的：**  
　プレッシャーを感じる場面で  
　心と体を落ち着かせる方法を  
　AIにガイドしてもらう。

**✅いつ使うか：**  
　プレゼン前・面接前・  
　大切な会話の前・眠れないとき。

```
# ロール設定
あなたはマインドフルネスと
呼吸法の専門家です。
私の状況と緊張のタイプに合わせた
「今すぐできる落ち着き方」を
ガイドしてください。

# 背景・目的・対象
・今の状況：
　[例：10分後にプレゼンがある]
・体の感覚：
　[例：心臓がバクバクしている・
　　手が冷たくなっている]
・気持ちの状態：
　[例：頭が真っ白になりそう]
・使える時間：[例：5分以内]

# 指示内容
1. 私の状態に合った
　呼吸法を1つ選び、
　具体的なやり方を
　ステップで説明する

2. 今すぐできる
　「マインドフルネスの小さな動作」
　を1つ提案する
　（例：5つの見えるものを数えるなど）

3. 本番直前に
　心の中で唱えると効果的な
　「セルフトーク」を1文作る
　（例：「今ここにいる自分でいい」など）

# 制約条件
・「リラックスしてください」などの
　漠然とした指示は使わない
・今すぐ物理的にできる行動を中心に
・科学的根拠のある方法を使う

# 出力形式
1〜3の見出し付き。
シンプルに、すぐ実行できる形で出力。

# 入力データ
状況・体の感覚・時間：[ここに入力]
```

**✅具体的な使用方法：**  
　「今の状況」と「体の感覚」だけ  
　入れれば動きます。

　スマホから入力して、  
　その場で実践できます。

**✅読者への一言：**  
　緊張は「消すもの」ではありません。

　「使いこなすもの」です。

　呼吸が整うと、  
　不思議と言葉も出てきます。

## 4. 逆境のあとで自分を立て直す

落ち込んだあと、  
どうやって立ち直るか。

レジリエンスとは、  
「折れない心」ではなく、  
「折れても戻ってくる力」のことです。

バネに例えると  
わかりやすいです。

固くて折れるより、  
柔らかく伸び縮みできる方が  
長持ちします。

このプロンプトは、  
つらい経験を「糧」にするための  
思考の整理法です。

### ◆逆境から学ぶリフレクションプロンプト

**✅目的：**  
　つらかった経験を振り返り、  
　そこから「得たもの」を意味づけして  
　次への力に変える。

**✅いつ使うか：**  
　大きな失敗・挫折・  
　人間関係のトラブルの後・  
　何かが終わったとき。

```
# ロール設定
あなたはレジリエンス（回復力）の
専門コーチです。
私のつらかった経験を否定せず、
そこから意味と学びを
一緒に見つけるお手伝いをしてください。

# 背景・目的・対象
・起きたこと（事実）：
　[例：2年間続けた仕事を辞めることになった]
・そのときの感情：
　[例：悔しさ・情けなさ・先が見えない不安]
・今の状態（回復度0〜10）：
　[例：4]

# 指示内容
以下の順番で一緒に整理します。

1. 「この経験で失ったもの」を
　正直にリストアップする
　（感情を否定しない）

2. 「この経験で得たもの・気づいたこと」を
　小さなことも含めて3〜5つ探す

3. この経験を通じて見えてきた
　「自分の価値観・大切にしたいこと」
　を言語化する

4. 今の自分が
　「3ヶ月後の自分」に伝えるとしたら
　何と言うかを一文で書く

# 制約条件
・「前向きになろう」とは言わない
・「失ったもの」も丁寧に扱う
・静かで、落ち着いたトーンで

# 出力形式
1〜4の見出し付き。
押しつけがましくない、
温かい言葉で。

# 入力データ
起きたこと・感情・回復度：[ここに入力]
```

**✅具体的な使用方法：**  
　「起きたこと」に  
　出来事をそのまま書いてください。

　うまく言語化できなくても大丈夫。  
　AIが整理を手伝ってくれます。

**✅読者への一言：**  
　つらかった経験は、  
　消えません。

　でも、意味づけ次第で  
　「持ち続ける重さ」が変わります。

　AIと一緒に、  
　その重さを少し軽くしましょう。

## 5. 自分を支える「言葉の処方箋」を作る

心が折れそうなとき、

### ◆自分だけの「お守り言葉」プロンプト

**✅目的：**  
　自分の性格・状況・弱点に合わせた  
　「心の支えになる言葉」を作る。

**✅いつ使うか：**  
　気持ちが落ちやすいとわかっているとき・  
　定期的なメンタルケアとして。

```
# ロール設定
あなたはセルフ・コンパッション
（自分への思いやり）の専門家です。
私の性格・状況・弱点に合わせた
「自分だけのお守り言葉」を
一緒に作ってください。

# 背景・目的・対象
・自分がよく落ち込むパターン：
　[例：人に迷惑をかけたと感じたとき]
・落ち込むと出てくる言葉：
　[例：「なんで自分はこうなんだ」]
・自分が大切にしたいこと（価値観）：
　[例：誠実でありたい・人の役に立ちたい]
・好きな言葉のトーン：
　[例：静かで落ち着いた感じ／力強い感じ]

# 指示内容
1. 私が落ち込んだ瞬間に
　心の中で唱えられる
　「お守り言葉」を3つ作る

2. 各言葉に
　「なぜこの言葉があなたに合うか」
　をひと言で添える

3. 一番使いやすいと思う言葉を
　「朝・昼・夜」のどの場面で
　唱えるとよいか提案する

# 制約条件
・大げさな励ましは使わない
・あなたの価値観に根ざした言葉にする
・声に出せる長さ（20文字以内）にする

# 出力形式
番号付きで、
それぞれの言葉と補足を
シンプルに出力。

# 入力データ
落ち込みのパターン・価値観・
好みのトーン：[ここに入力]
```

**✅具体的な使用方法：**  
　「よく落ち込むパターン」を  
　正直に書いてください。

　自分の言葉で  
　お守りを作るからこそ、  
　本当に心に響きます。

**✅読者への一言：**  
　「自分に優しくする」とは、  
　甘やかすことではありません。

　自分をひとりの大切な人間として  
　扱うことです。

　AIと一緒に、  
　あなただけの言葉を見つけましょう。
