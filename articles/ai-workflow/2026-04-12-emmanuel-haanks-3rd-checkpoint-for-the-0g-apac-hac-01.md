3rd Checkpoint for the 0G APAC Hackathon:

Coal's MCP Server is live.

Any AI, Claude, ChatGPT, Gemini, Cursor, can now discover and buy from Coal merchants through one endpoint.

claude mcp add coal https://t.co/SQQhDNpvlo

9 tools. Zero setup. Connect and go:

→ discover_merchants - browse all products + paywalls indexed on 0G

→ search_products - filter by name, price, or tag

→ query_merchant_memory - AI Q&A via 0G Compute with Sealed Inference

→ create_checkout - create a USDC checkout on Base

→ verify_receipt - 3-step proof trail: Base TX → 0G Storage → 0G Chain

→ get_0g_health - all 5 components live on mainnet

What this means: a merchant installs coal-react, their catalog gets indexed on 0G. Now any AI with MCP support finds them automatically.

No integration on the agent side. No plugins. No marketplace. Just a protocol connection.
