会社員をしながら副業を始めようと思って  
早1年。「いつかやろう」が口癖だった私が  
ついに動き出しました。  
  
今日はClaude APIとMake・Bufferを使って  
Xの自動投稿を1日で完成させた手順を  
全部公開します。  
  
━━━━━━━━━━━━━━━  
■ 完成した自動化の仕組み  
━━━━━━━━━━━━━━━  
  
Claude API（投稿文を自動生成）  
　↓  
Make（自動化ツール）  
　↓  
Buffer（Xに自動投稿）  
  
この3つを繋げるだけで  
毎日7:30 / 12:00 / 20:30 / 22:30に  
自動で投稿されます。  
  
━━━━━━━━━━━━━━━  
■ 必要なもの・費用  
━━━━━━━━━━━━━━━  
  
・Claude APIアカウント（$5〜）  
・Makeアカウント（無料）  
・Bufferアカウント（無料）  
  
合計：約750円から始められます。  
  
━━━━━━━━━━━━━━━  
■ STEP 1　Claude APIキー取得  
━━━━━━━━━━━━━━━  
  
1. console.anthropic.com にアクセス  
2. Googleアカウントでサインアップ  
3. 左メニュー「API Keys」→「Create Key」  
4. 表示されたキーをメモアプリにコピー  
　（一度しか表示されないので注意）  
5. 「Plans & Billing」で$5チャージ  
  
━━━━━━━━━━━━━━━  
■ STEP 2　Makeでシナリオを作成  
━━━━━━━━━━━━━━━  
  
1. make.com でアカウント作成  
2. 「Create a new scenario」  
3. HTTPモジュールを追加して以下を設定：  
  
URL：https://api.anthropic.com/v1/messages  
Method：POST  
  
Headers：  
・x-api-key → 自分のAPIキー  
・anthropic-version → 2023-06-01  
・content-type → application/json  
  
Body（JSON）：  
{  
  "model": "claude-haiku-4-5",  
  "max\_tokens": 200,  
  "messages": [  
    {  
      "role": "user",  
      "content": "副業・AI・お金に関する  
      Xの投稿文を140文字以内で作成"  
    }  
  ]  
}  
  
━━━━━━━━━━━━━━━  
■ STEP 3　Bufferと連携  
━━━━━━━━━━━━━━━  
  
1. buffer.com でアカウント作成  
2. XアカウントをBufferに連携  
3. MakeにBufferモジュールを追加  
4. Text欄に {{4.data.content[].text}} を入力  
5. Saveして完了  
  
━━━━━━━━━━━━━━━  
■ STEP 4　スケジュール設定  
━━━━━━━━━━━━━━━  
  
Makeのスケジュールを  
「Daily」に設定して  
7:30 / 12:00 / 20:30 / 22:30  
の4回に指定。  
  
これで完全自動化の完成です。  
  
━━━━━━━━━━━━━━━
