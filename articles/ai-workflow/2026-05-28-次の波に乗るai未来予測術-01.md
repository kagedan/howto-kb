「また乗り遅れた…」

そう感じたことは、ありませんか？

あのとき、あのサービスに  
乗っていれば良かった。

あのトレンドを  
もう少し早く知っていたら、  
もっと違う動き方ができた。

変化が激しい今の時代、  
過去の成功体験や  
業界の常識だけでは戦えません。

でも「どうやって先を読めばいいの？」  
と思っている方がほとんどのはずです。

---

この記事を読むと、

✅ **致命的な乗り遅れを防げる**  
　→AIがトレンドの予兆を  
　　先読みしてくれるので  
　　気づかないうちに取り残される  
　　リスクを減らせます

✅ **複数シナリオで冷静に判断できる**  
　→楽観・悲観・中立の  
　　3パターンを手に入れることで  
　　不確実な状況でも  
　　根拠を持って動けます

✅ **競合より一歩早く動ける**  
　→業界の変化を先にシミュレーションし  
　　次の一手を準備できます

---

「未来を当てる」のが目的ではありません。

「変化に気づくのを早める」ことが  
今日から変わります。

AIは羅針盤です。

どちらに進むかを決めるのは、  
あくまであなた自身。

その判断の質を、  
確実に高めてくれるのが  
今日のプロンプトたちです。

## 基本のプロンプト

まず、例え話からはじめます。

AIで未来予測をするとは、  
どういうことか。

10歳の子どもでも  
わかるように説明しましょう。

---

**天気予報に例えてみます。**

昔の天気予報は、  
「空の雲の形」で  
翌日を予測するしかありませんでした。

でも今は、  
世界中のデータをコンピュータが  
高速で分析して  
「明日の降水確率80%」と出せます。

AIを使ったビジネスの未来予測も、  
それと同じです。

ニュース・業界動向・  
テクノロジーの変化・消費者の行動。

そういったデータをAIが読み解き、  
「この業界、こんな波が来るかも」  
と示してくれます。

当たるかどうかではなく、  
「傘を持って出るかどうか」の  
判断材料を得ることが大事です。

### ◆業界トレンド先読みプロンプト

**✅目的：**  
　自分の業界や関心分野の  
　トレンドの予兆を洗い出し、  
　次の一手を考える。

**✅いつ使うか：**  
　毎月1回の定期レビューや、  
　新規事業の方向性を考えるとき。

```
# ロール設定
あなたは業界トレンドと
未来予測の専門アナリストです。
膨大な情報を整理し、
次に来る変化の予兆を読み解いて
ビジネスパーソンに
実践的なインサイトを提供します。

# 背景・目的・対象
・分析対象の業界：[例：飲食業界]
・自分の立場・役割：
　[例：個人で飲食店を経営している]
・現在気になっている変化・出来事：
　[例：フードデリバリーの伸びと
　　人件費の高騰]

# 指示内容
以下の3点を整理してください。

1. 今この業界で起きている
　主な変化・トレンド（5点）
2. そこから読める
　「次の1〜3年で来そうな波」（3点）
3. それぞれの波に対して
　個人・中小規模でも取れる
　先手の行動（各1〜2点）

# 制約条件
・スタートアップや大企業だけでなく
　個人・中小が動ける視点で書く
・希望的観測ではなく
　現実的な視点を優先する
・専門用語には必ず一言補足を入れる

# 出力形式
見出しつきで
番号付きの箇条書き。
各項目に「なぜそう言えるか」を
1文で添える。

# 入力データ
業界：[ここに入力]
自分の立場：[ここに入力]
気になる変化：[ここに入力]
```

**✅具体的な使用方法：**  
　「業界」の部分を  
　自分の仕事に置き換えるだけ。

　「IT業界」「不動産業界」  
　「フリーランス全般」でも  
　問題なく使えます。

**✅読者への一言：**  
　「自分には関係ない話」と  
　思っていた変化が、  
　一番早く自分の足元まで  
　来ていることがあります。

　月1回、このプロンプトで  
　確認する習慣をつけるだけで  
　感覚が変わります。

### ◆3シナリオ分岐プロンプト

**✅目的：**  
　楽観・中立・悲観の3つの未来を  
　同時に描き、  
　どのシナリオでも動ける  
　準備をする。

**✅いつ使うか：**  
　重要な意思決定の前や、  
　先が読みにくいと感じているとき。

```
# ロール設定
あなたはシナリオプランニング
（未来を複数のパターンで
想定して備える思考法）の
専門コンサルタントです。
不確実な状況でも
根拠ある判断ができるよう、
複数の未来像を設計します。

# 背景・目的・対象
・検討しているテーマ・状況：
　[例：来年、副業としてオンライン
　コーチングを始めようか迷っている]
・現在わかっていること・懸念点：
　[例：AIコーチングツールの台頭、
　需要は増えているが競争も激しい]
・判断のタイムライン：
　[例：半年以内に方針を決めたい]

# 指示内容
以下の3シナリオを設計してください。

【シナリオA：楽観的な未来】
・どういう条件が揃えば実現するか
・その場合に取るべき行動

【シナリオB：中立的な未来】
・最も起こりやすい展開
・その場合に取るべき行動

【シナリオC：悲観的な未来】
・どういうリスクが顕在化するか
・その場合でもダメージを最小にする行動

# 制約条件
・どのシナリオも現実に起こりうる
　具体性を持たせる
・「最悪の場合」でも
　完全撤退ではなく
　次の一手が描けるようにする
・各シナリオに判断のトリガー
　（こうなったらこの方向へ）
　を1つずつ入れる

# 出力形式
A・B・Cの見出しつきで
構造的に整理。
最後に「今すぐできる共通の一手」を
1点追記する。

# 入力データ
テーマ・状況：[ここに入力]
懸念点：[ここに入力]
タイムライン：[ここに入力]
```

**✅具体的な使用方法：**  
　「検討しているテーマ」に  
　今自分が悩んでいる  
　ビジネスの選択を入れるだけ。

　転職・独立・新規事業・投資など  
　どんな意思決定にも使えます。

**✅読者への一言：**  
　「最悪の場合」を考えると  
　怖くなると思いきや、  
　逆に冷静になれます。

　最悪でも「こう動ける」と  
　わかると、不安が大幅に減ります。

## ビフォー・アフター比較

**同じ問いでも、  
プロンプトの作り方で  
AIの答えは別物になります。**

想定シナリオ：  
「EC（ネット通販）事業を  
展開している会社が、  
次の1〜2年の事業方針を  
考えたい」

---

### Before（簡易プロンプト）

```
EC業界の今後のトレンドを教えてください。
```

**AIの回答（例）**：

> AI活用の拡大、パーソナライズ化、  
> サブスクリプションモデルの普及が  
> 進むと考えられます。  
> また、物流コストの最適化も  
> 重要な課題です。

---

👉 **どこが足りないか？**

---

### After（本記事のプロンプトを使用）

```
# ロール設定
あなたはEC業界の未来予測
専門アナリストです。

# 背景・目的・対象
・業界：EC（ネット通販）
・自分の立場：
　従業員10名の中小ECサイト運営
・気になる変化：
　大手モールへの出店コスト増加と
　SNS経由の購買行動の変化
・判断タイムライン：
　次の期首（半年後）までに
　方針を決めたい
```

**AIの回答（例）**：

>  【今この業界で起きている変化】  
>   
>  ①SNSコマース（TikTokなど  
>  短動画からの直接購買）の急成長  
>  →大手モール依存からの脱却機会  
>   
>  【次に来る波】  
>   
>  「自社ECとSNSの直結」  
>  →小規模でもブランド独自の  
>  顧客接点を持てる時代が加速  
>   
>  【中小が今すぐ取れる先手】  
>   
>  ・月1本のショート動画を投稿して  
>  　自社商品の世界観を発信  
>  ・大手モールへの依存率を  
>  　今期中に70%以下に下げる目標設定

---

👉 **何が変わったか？**

「ロール設定」で  
AIを専門家として動かし、

「背景指定」で  
自分の規模・立場・懸念を  
渡したことで、

**「自分ごと」として  
使える情報が出てきました。**

AIはゼロから考えてくれません。

情報を渡すほど  
精度が上がります。

これがプロンプトの核心です。

## 実践プロンプト集

## 1. 競合の動きを先読みする

気づいたら競合が  
先に動いていた。

そんな経験はありませんか？

相手の動きを待って  
から考えるのではなく、  
AIに競合の「次の一手」を  
先に推測してもらいましょう。

---

### ◆競合先読みプロンプト

**✅目的：**  
　業界の競合他社が  
　今後取りそうな動きを予測し、  
　先回りして準備する。

**✅いつ使うか：**  
　競合の新サービス・新展開の  
　情報を見かけたとき。  
　または四半期ごとの定期レビュー。

```
# ロール設定
あなたは競合分析と
事業戦略の専門コンサルタントです。
業界の動向と競合他社の行動パターンから
次の一手を先読みし、
依頼者が先手を打てる情報を提供します。

# 背景・目的・対象
・自分のビジネス：
　[例：地域密着型の学習塾を運営]
・競合として意識している相手：
　[例：大手オンライン学習サービス]
・最近気になる競合の動き：
　[例：低価格プランの新設と
　AIを使った個別最適化の宣伝]

# 指示内容
1. この競合が今後
　1〜2年で打ちそうな施策（3点）
　とその根拠

2. それぞれの施策が
　自分のビジネスに与える影響

3. 今のうちに自分が
　取れる先手の行動（各1点）

# 制約条件
・競合の規模感や資本力を踏まえた
　現実的な予測にする
・「大手に勝つ」ではなく
　「差別化して共存する」視点を入れる
・中小・個人規模でも
　実行可能な行動のみ提示する

# 出力形式
1→2→3の順で
見出しつき箇条書き。
各行動に「優先度（高・中・低）」をつける。

# 入力データ
自分のビジネス：[ここに入力]
競合：[ここに入力]
最近気になる競合の動き：[ここに入力]
```

**✅具体的な使用方法：**  
　競合の「最近気になる動き」に  
　ニュースや目にした広告など  
　気になることを書くだけでOK。

　確実な情報でなくても  
　「こんな広告を出し始めた」  
　程度の観察で十分です。

**✅読者への一言：**  
　競合を意識しすぎると  
　消耗しますが、  
　無視しすぎると  
　気づかないうちに  
　置いてきぼりになります。

　このプロンプトで  
　「定点観測」する習慣を  
　作ってください。

## 2. 自分のビジネスに来るリスクを先に洗う

「まさかこんなことが…」

ビジネスの落とし穴は、  
いつも見えないところにあります。

順調なときほど、  
リスクを直視しにくいものです。

AIに「冷静な第三者」として  
自分のビジネスの弱点を  
洗い出してもらいましょう。

---

### ◆ビジネスリスク先洗いプロンプト

**✅目的：**  
　今後1〜3年で  
　自分のビジネスに影響しうる  
　リスク要因を網羅的に  
　洗い出す。

**✅いつ使うか：**  
　年初の事業計画時や、  
　ビジネスの転換点を感じたとき。

```
# ロール設定
あなたはリスクマネジメントと
事業継続計画の専門家です。
感情ではなく事実と論理で、
依頼者のビジネスにとって
潜在的なリスク要因を
洗い出してください。

# 背景・目的・対象
・ビジネスの概要：
　[例：都内でネイルサロンを
　1店舗経営。固定客が中心]
・現在の主な収益源：
　[例：常連客の月1〜2回の来店]
・現在感じている不安・懸念：
　[例：近隣への競合出店と
　物価上昇による値上げのしにくさ]

# 指示内容
以下の3カテゴリで
リスクを洗い出してください。

【外部環境リスク】
　市場・競合・経済・規制など

【内部リスク】
　依存度の高い顧客・
　スタッフ・コストなど

【見落とされがちなリスク】
　盲点になりやすい
　構造的な問題

各リスクに
「発生確率（高・中・低）」と
「影響度（大・中・小）」をつける。

# 制約条件
・脅かすためではなく
　「備えるため」の視点で書く
・各リスクに
　最低限できる対策を1点添える
・「あるある」の一般論ではなく
　入力したビジネスに
　固有の視点で書く

# 出力形式
カテゴリ別の表形式、または
箇条書きで見やすく整理。

# 入力データ
ビジネス概要：[ここに入力]
収益源：[ここに入力]
現在の懸念：[ここに入力]
```

**✅具体的な使用方法：**  
　「ビジネスの概要」に  
　今の仕事の状況を  
　ざっくり書くだけ。

　完璧でなくていいです。  
　書いた分だけ  
　精度の高い出力が得られます。

**✅読者への一言：**  
　リスクを直視するのは  
　怖いことではありません。

　知らないまま進む方が、  
　はるかに怖いです。

　「知っておくだけ」でいい。  
　それだけで準備の質が変わります。

## 3. 新規ビジネスチャンスを発掘する

「何かチャンスはないか」

そう思いながらも、  
日々の業務に追われて  
考える時間が取れない。

AIを「アイデアの壁打ち相手」として  
使いましょう。

あなたの経験と  
AIの情報分析力を  
掛け合わせると、  
思わぬ可能性が見えてきます。

---

### ◆ビジネスチャンス発掘プロンプト

**✅目的：**  
　自分のスキル・資産・環境を  
　整理し、次のビジネス機会を  
　具体的に見つけ出す。

**✅いつ使うか：**  
　新規事業・副業・  
　サービス追加を考えているとき。

```
# ロール設定
あなたはビジネスモデルの
設計と市場機会の発掘を
専門とするコンサルタントです。
クライアントが持つ資産・スキル・
人脈・環境を最大限に活かし、
次のビジネスチャンスを
具体的に提案します。

# 背景・目的・対象
・自分の強み・スキル：
　[例：10年以上の経理経験、
　Excelマクロが得意]
・現在使えるリソース：
　[例：週末5時間・PC1台・
　初期投資10万円以内]
・やりたい方向性（あれば）：
　[例：在宅でできること]
・やりたくないこと・苦手なこと：
　[例：営業・対面接客]

# 指示内容
1. 私の強みを活かした
　ビジネス機会を5つ提案する
2. 各機会の
　「始めやすさ」と
　「収益化しやすさ」を評価する
3. 最もおすすめの1つに絞り、
　最初の3ステップを教える

# 制約条件
・すぐに始められる現実的なものを優先
・趣味の延長線上や
　既存スキルの組み合わせで
　構わない
・「月◯万円」の数字は
　根拠ある範囲で示す

# 出力形式
提案は表または番号付き箇条書き。
最後のおすすめ1案は
3ステップで具体的に記載。

# 入力データ
強み・スキル：[ここに入力]
使えるリソース：[ここに入力]
方向性・苦手なこと：[ここに入力]
```

**✅具体的な使用方法：**  
　「強み・スキル」は  
　謙遜せずに書いてください。

　「当たり前にできること」が  
　他の人にとっての価値に  
　なることがよくあります。

**✅読者への一言：**  
　「自分には特別なスキルがない」  
　という人が一番多いですが、

　普通にできることの組み合わせが  
　一番再現性のある  
　ビジネスになります。

## 4. 業界の5年後をシミュレーションする

「5年後、この仕事はどうなっているか」

考えたいけど、  
考えると不安になる。

そんな気持ち、  
よくわかります。

でも、不安なまま知らないより、  
AIと一緒に見据えた方が  
絶対に楽になります。

---

### ◆業界5年後シミュレーションプロンプト

**✅目的：**  
　自分が関わる業界・仕事が  
　5年後にどう変わっているかを  
　具体的に描き、  
　今から準備できることを知る。

**✅いつ使うか：**  
　キャリアの見直し時や、  
　「この先どうなるか不安」と  
　感じたとき。

```
# ロール設定
あなたはビジネストレンドと
キャリア設計の専門アドバイザーです。
テクノロジーの進化・社会変化・
労働市場の動向をもとに、
依頼者の業界・職種の
5年後のリアルな姿を
描き出します。

# 背景・目的・対象
・現在の業界・職種：
　[例：中小企業で人事・採用を担当]
・現在のスキル・強み：
　[例：面接・制度設計・
　社員研修の企画運営]
・最近感じている変化・違和感：
　[例：採用ツールのAI化が進み、
　自分の仕事がどう変わるか不安]

# 指示内容
1. この職種・業界の
　5年後の変化（3〜5点）

2. AIや技術進化によって
　「なくなる可能性が高い業務」と
　「逆に価値が上がる業務」

3. 今のスキルを活かしながら
　5年後も必要とされるための
　今すぐできる準備（3点）

# 制約条件
・脅かすのではなく
　「機会として捉える」視点を入れる
・現在のスキルを
　ゼロベースで捨てる提案はしない
・「学び直せ」ではなく
　「今すぐできる小さな一歩」を示す

# 出力形式
1→2→3の順で見出しつき箇条書き。
最後に「5年後の自分像」を
1〜2文で描く。

# 入力データ
業界・職種：[ここに入力]
現在のスキル：[ここに入力]
感じている変化・不安：[ここに入力]
```

**✅具体的な使用方法：**  
　「感じている変化・不安」の欄に  
　今モヤモヤしていることを

　整理されていなくて大丈夫です。

## 5. 意思決定の「バイアス」を外す

人は感情で動く生き物です。

「なんとなく怖い」  
「なんとなく行けそう」

そういった直感は大切ですが、  
重要な意思決定の場面では  
自分のバイアス  
（思い込みや偏り）が  
足を引っ張ることがあります。

AIに「第三者の目」として  
チェックしてもらいましょう。

---

### ◆意思決定バイアスチェックプロンプト

**✅目的：**  
　重要な判断をする前に、  
　自分の思い込みや  
　感情的な偏りを  
　客観的にチェックしてもらう。

**✅いつ使うか：**  
　転職・独立・投資・事業転換など  
　「大きな決断」の前。

```
# ロール設定
あなたは行動経済学と
意思決定の専門家です。
人が陥りやすい思考の偏り
（認知バイアス）を把握した上で、
依頼者の判断に
どのようなバイアスが
混じっているかを
客観的に分析してください。

# 背景・目的・対象
・今検討している判断・決断：
　[例：会社を辞めて独立しようとしている]
・判断の理由（自分なりの）：
　[例：収入が上げられる、
　自由な時間が増える、
　今の職場の人間関係が辛い]
・迷っている理由・不安：
　[例：収入が安定するか不安、
　でも早く決めないと年齢的に厳しい]

# 指示内容
1. この判断に混ざっている可能性がある
　認知バイアス（思い込み）を
　3〜5点指摘する
　（名前と「なぜそう言えるか」を添えて）

2. 感情を一旦横に置いたとき、
　「事実として言えること」と
　「まだわからないこと」を整理する

3. この決断を
　「後悔なくするための
　確認事項」を3点提示する

# 制約条件
・「やめておけ」「進め」の
　結論は出さない
・あくまで「自分で考える素材」を
　提供する姿勢で
・バイアスの指摘は
　責めるのではなく
　「よくある傾向」として丁寧に伝える

# 出力形式
1→2→3の順で
見出しつき箇条書き。
最後に「決断の前にできる
小さな行動」を1つ追記する。

# 入力データ
検討中の判断：[ここに入力]
判断の理由：[ここに入力]
迷っている理由：[ここに入力]
```

**✅具体的な使用方法：**  
　「判断の理由」と  
　「迷っている理由」を  
　正直に書いてください。

　AIは批判しません。  
　材料を整理するだけです。

**✅読者への一言：**  
　「あのとき、なぜあんな決断をしたんだろう」

　そう後悔しないための道具として  
　このプロンプトを使ってください。

　決断するのはあなたです。  
　でも、その質を上げるのはAIです。
