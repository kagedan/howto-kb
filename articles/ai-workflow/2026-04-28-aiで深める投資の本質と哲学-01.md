「また買いどきを逃した…」

そう思いながら、  
相場のニュースを  
眺めていませんか？

---

SNSでは毎日、  
「これが次の10倍株だ」  
「今すぐ乗らないと損する」  
という声が飛び交います。

でも、  
その声に乗って動くたびに、  
なぜか裏目に出る。

「自分には向いていないのかも」  
と感じてきた方もいるかもしれません。

---

でも、原因は才能ではありません。

**「自分の投資軸」がなかっただけです。**

---

この記事を読むと、

✅ **致命的なミスを回避できる**  
　→感情や流行に流されず  
　　冷静な判断軸が手に入る

✅ **冷静な判断力が身につく**  
　→どんな相場でも  
　　「自分はなぜこれを買うのか」を  
　　説明できるようになる

✅ **機会損失がなくなる**  
　→情報に振り回されるのではなく  
　　本質から動けるようになる

---

AIは「投資の答え」を出してくれるわけではありません。

でも、  
「自分が何を大切にして投資するか」  
を言語化し、深めるパートナーになります。

今日から、  
考える力がレベルアップします。

## 基本のプロンプト

まず、例え話からはじめます。

AIを投資に使うとは、  
どういうことか。

---

**釣りの「名人」に例えてみます。**

釣り初心者は、  
「今日はどこに行けばいい？」  
と聞かれても答えられません。

でも名人は違います。

「今の季節」「水温」「天気」  
「魚の習性」を総合して  
「あの岩の裏あたりがいい」  
と言える。

---

投資も同じです。

初心者は  
「今どの株を買えばいい？」  
と聞きたくなります。

でも本当に大切なのは、  
「なぜその企業に価値があるのか」  
「自分はどんな基準で選ぶのか」  
という思考の軸です。

AIは、  
その軸を一緒に作ってくれる  
「考える練習相手」です。

### ◆投資哲学の言語化プロンプト

**✅目的：**  
　自分の投資スタイルと判断基準を  
　言葉にして整理する。

**✅いつ使うか：**  
　投資を始めるとき、  
　または「自分の軸がブレている」  
　と感じたとき。

```
# ロール設定
あなたは長期投資家の思考を整理する
コーチです。
投資の「正解」を教えるのではなく、
私が自分自身の投資哲学を
明確にできるよう
対話形式でサポートしてください。

# 背景・目的・対象
・投資歴：[例：3年、主にインデックス投資]
・今の悩み：
　[例：個別株も試したいが
　　何を基準に選べばいいかわからない]
・大切にしたい価値観：
　[例：長期・安定・社会貢献している企業]

# 指示内容
以下の3点を整理してください。

1. 私の投資スタイルを一言で表現する
2. 私が「買う理由」にすべき
　判断基準を5つ提示する
3. 私が「買ってはいけない」
　状況・銘柄の特徴を3つ挙げる

# 制約条件
・特定の銘柄は推奨しない
・私の価値観を尊重した提案にする
・難しい金融用語には補足を入れる

# 出力形式
見出しつきの箇条書きで
わかりやすく整理。

# 入力データ
投資に対する考え・状況：[ここに入力]
```

**✅具体的な使用方法：**  
　「今の悩み」と「大切にしたい価値観」に  
　今の自分の言葉を書くだけ。

　難しく考えなくてOKです。

　思ったままを書けば、  
　AIが整理してくれます。

**✅読者への一言：**  
　「投資哲学」というと難しそうですが、  
　要は「自分なりのルール」です。

　それを持つだけで、  
　判断が格段に楽になります。

### ◆企業価値の深掘りプロンプト

**✅目的：**  
　気になる企業が「本当に価値があるか」を  
　多角的に問い直す。

**✅いつ使うか：**  
　個別株を検討しているとき、  
　または「なんとなく気になる企業」が  
　あるとき。

```
# ロール設定
あなたはバリュー投資（割安株投資）と
企業分析の専門家です。
特定銘柄の推奨はせず、
企業の本質的な価値を
多角的に考えるための
視点を提供してください。

# 背景・目的・対象
・検討している企業・業種：
　[例：日本の食品メーカー全般]
・なぜ気になっているか：
　[例：人口減でも海外展開しているから]
・現在の私の理解レベル：
　[例：PERとPBRは知っている]

# 指示内容
以下の視点から分析の枠組みを教えてください。

1. この企業・業種が持つ
　「経済的な堀（競争優位性）」の
　チェックポイント3つ

2. 長期保有に値するかを判断するための
　「本質的な問い」を5つ

3. 見落としがちな
　「見えないリスク」の視点を3つ

# 制約条件
・投資の推奨・断言はしない
・「考えるための枠組み」を提供する
・初心者でもわかる言葉で説明する

# 出力形式
番号つき箇条書き＋
各項目に一言補足。

# 入力データ
企業名または業種・気になる理由：
[ここに入力]
```

**✅具体的な使用方法：**  
　「検討している企業・業種」に  
　気になる分野を書くだけ。

　個別銘柄でも業種全体でも使えます。

**✅読者への一言：**  
　「なぜこの会社は強いのか」を  
　言葉にできない投資は、  
　いわば「感覚買い」です。

　AIと一緒に言語化する習慣が、  
　長期で差をつけます。

## ビフォー・アフター比較

**同じ質問でも、  
プロンプトの精度で  
答えは大きく変わります。**

想定シナリオ：  
「相場が急落したとき、  
どう考えればいいかわからない」

---

### Before（簡易プロンプト）

```
相場が急落しました。
どうすればいいですか？
```

**AIの回答（例）**：

> 慌てて売るのは禁物です。  
> 長期投資を続けましょう。  
> 分散投資が大切です。

---

👉 **どこが足りないか？**

* 誰にでも当てはまる  
  　一般論しか返ってこない
* 自分の状況・資産・  
  　目的が反映されていない
* 読んでも「で、どうすれば？」  
  　という疑問が残る

---

### After（本記事のプロンプトを使用）

```
# ロール設定
あなたは長期投資家の
思考を整理するコーチです。

# 背景・目的・対象
・私の状況：
　30代、投資歴5年
　インデックス中心、
　個別株少し保有
・今起きていること：
　市場が1週間で15%急落
・私の感情：
　怖くて確認できていない
・本来の目標：
　20年後の資産形成

# 指示内容
①今の私が確認すべき
　「本質的な問い」を3つ
②急落時に「やってはいけないこと」を
　理由つきで3つ
③20年後の目標に照らして
　今取るべき行動の選択肢を2つ
```

**AIの回答（例）**：

>  【本質的な問い①】  
>  「この急落は、自分の投資目的を  
>  変えるほどの出来事か？」  
>   
>  【本質的な問い②】  
>  「20年後も、この資産は  
>  価値があると思えるか？」  
>   
>  【やってはいけないこと①】  
>  感情で一括売却する  
>  → 最も安い場面で確定損失になる  
>   
>  【取るべき行動の選択肢①】  
>  何もしない（目的と期間に変化がないなら）  
>   
>  【取るべき行動の選択肢②】  
>  余剰資金で少額追加購入を検討する  
>  （ただし生活資金は絶対に使わない）

---

👉 **何が変わったか？**

「ロール設定」で  
AIを思考コーチとして起動し、

「背景指定」で  
自分の年齢・目的・感情状態を渡したことで、

**自分の状況に合った  
具体的な思考整理が返ってきました。**

情報を渡すほど、  
AIの回答は自分ごとになります。

これがプロンプトの核心です。

## 実践プロンプト集

## 1. 自分のバイアスを点検する

投資判断を狂わせる最大の敵は、  
「自分自身の思い込み」です。

「この株は絶対上がる」  
「今さら売れない」  
「みんなが買っているから大丈夫」

そんな感覚、  
心当たりはありませんか？

AIに自分のバイアスを  
チェックさせてみましょう。

---

### ◆バイアス点検プロンプト

**✅目的：**  
　投資判断に潜む  
　認知バイアス（思い込みのクセ）を  
　洗い出し、冷静な視点を取り戻す。

**✅いつ使うか：**  
　「なんとなく売れない」  
　「なんとなく買いたい」  
　と感じているとき。

```
# ロール設定
あなたは行動経済学と
投資心理の専門家です。
私の投資判断に潜む
認知バイアスを客観的に指摘し、
冷静な思考を取り戻せるよう
サポートしてください。

# 背景・目的・対象
・今の状況・考えていること：
　[例：2年前に買った株が
　　30%下がっているが売れない]
・その判断の理由（正直に）：
　[例：いつか戻ると思っている・
　　損を確定させたくない]
・本来の投資目的：
　[例：10年後の資産形成]

# 指示内容
1. 私の判断に潜んでいる
　認知バイアスの名前と
　説明を2〜3つ挙げる

2. そのバイアスを乗り越えるための
　「自分への問い」を3つ提示する

3. 本来の目的に照らしたとき、
　合理的な選択肢を2つ示す
　（答えを断言しなくていい）

# 制約条件
・特定の売買判断は断言しない
・私を責めるのではなく、
　気づきを促す言葉で
・専門用語には補足を入れる

# 出力形式
番号つき見出しで
シンプルに整理。

# 入力データ
今の状況・判断・理由：[ここに入力]
```

**✅具体的な使用方法：**  
　「その判断の理由（正直に）」を  
　できるだけ正直に書くのがコツです。

　格好つけずに書くほど、  
　核心に近いフィードバックが返ります。

**✅読者への一言：**  
　バイアスは「頭の悪い人」がかかるものでは  
　ありません。

　むしろ、真剣に考えている人ほど  
　深くはまります。

　定期的な点検が、  
　長期投資を守ります。

## 2. リスクを多角的に洗い出す

「リスクを取れ」とよく言われますが、  
取るべきリスクと  
取ってはいけないリスクは別物です。

「気づいていなかっただけ」のリスクが  
一番怖い。

AIに死角を探してもらいましょう。

---

### ◆リスク多角分析プロンプト

**✅目的：**  
　特定の投資判断について、  
　見落としがちなリスクを  
　多角的に洗い出す。

**✅いつ使うか：**  
　「この投資、どこがまずいか  
　自分では気づけていない気がする」  
　というとき。

```
# ロール設定
あなたはリスク管理の専門家です。
投資の「うまくいくシナリオ」ではなく、
「うまくいかないシナリオ」を
洗い出すことに特化して
サポートしてください。

# 背景・目的・対象
・検討している投資・行動：
　[例：米国株インデックスの
　　積立額を月3万→10万に増やす]
・現在の状況：
　[例：30代、共働き、
　　子ども1人、貯金400万]
・期待していること：
　[例：15年後に2000万作りたい]

# 指示内容
以下の視点からリスクを洗い出してください。

1. 市場リスク（相場・経済環境）
2. 生活リスク（収入・支出の変化）
3. 行動リスク（感情・判断ミス）
4. 見落とされがちな「盲点リスク」

各リスクに対して、
簡単な対策アイデアも1つずつ添えてください。

# 制約条件
・投資の是非を断言しない
・生活を壊すリスクを特に重視する
・脅かすのではなく、
　準備を促す言葉で

# 出力形式
リスクカテゴリごとに
見出しつき箇条書き。

# 入力データ
投資内容・状況詳細：[ここに入力]
```

**✅具体的な使用方法：**  
　「検討している投資・行動」と  
　「現在の状況」を具体的に書くほど、  
　自分に合ったリスクが返ってきます。

　家族構成・収入・貯金額など、  
　少し詳しく書くのがポイントです。

**✅読者への一言：**  
　リスクを知ることは、  
　怖がることではありません。

　「それでも進むかどうか」を  
　自分で決められるようになることです。

## 3. 投資方針書を自分で作る

「なぜ今これを買ったのか」  
後から思い出せますか？

感情に流された売買を防ぐ最強の方法は、  
事前に「投資方針書」を作っておくことです。

AIと一緒に、  
自分専用のルールブックを作りましょう。

---

### ◆個人投資方針書プロンプト

**✅目的：**  
　自分だけの投資ルールを文書化し、  
　感情的な判断を防ぐ  
　「よりどころ」を作る。

**✅いつ使うか：**  
　投資を始めるとき、  
　または「自分のルールを整理したい」  
　と思ったとき。

　年に1回の見直しにも使えます。

```
# ロール設定
あなたは個人投資家の
ファイナンシャルコーチです。
私の状況と価値観をもとに、
「個人投資方針書」の草案を
作成してください。

# 背景・目的・対象
・年齢・家族構成：[例：35歳・既婚・子1人]
・投資目的：[例：老後資金・子の教育費]
・運用期間の目安：[例：20年]
・リスク許容度：
　[例：年30%の下落まで耐えられる]
・大切にしたい価値観：
　[例：社会に良い影響を与える企業に投資]

# 指示内容
以下の項目を含む
「個人投資方針書」を作成してください。

1. 投資の目的（一言で）
2. 投資対象のルール（買っていいもの）
3. 投資対象の除外ルール（買わないもの）
4. 買い増し・売却の判断基準
5. 相場急落時の行動ルール
6. 年に1回の見直しチェックリスト

# 制約条件
・難しい用語は使わず
　自分の言葉に近い表現にする
・「なぜそのルールか」の理由も
　簡単に添える
・現実的に守れるルールにする

# 出力形式
各項目を見出しつきで
1ページに収まる分量で。

# 入力データ
状況・価値観の詳細：[ここに入力]
```

**✅具体的な使用方法：**  
　「リスク許容度」は正直に書いてください。

　「どのくらいの下落まで  
　眠れなくならずにいられるか」で  
　考えると書きやすいです。

**✅読者への一言：**  
　投資方針書は、  
　相場が荒れたときの  
　「自分への手紙」になります。

　作っておくだけで、  
　判断のブレが格段に減ります。

## 4. 相場の本質を問い直す

「今の相場は割高か、割安か」

それを判断するには、  
「相場とは何か」を  
深く理解している必要があります。

AIに「本質的な問い」を  
立ててもらい、  
思考を深めましょう。

---

### ◆相場の本質思考プロンプト

**✅目的：**  
　市場の動きを表面的に見るのではなく、  
　背後にある構造・心理・本質を  
　深く考える力を養う。

**✅いつ使うか：**  
　「今の相場をどう見ればいいかわからない」  
　「ニュースを読んでも意味がわからない」  
　というとき。

```
# ロール設定
あなたは市場心理と
マクロ経済（国全体の経済の動き）を
研究する思考コーチです。
今の相場状況について、
私が「本質から考える」ための
問いと視点を提供してください。

# 背景・目的・対象
・今の相場状況（自分の理解で）：
　[例：米国株が高値圏にある気がする。
　　金利も高い]
・気になっているニュース・動き：
　[例：AI関連株だけが上がっている]
・自分の解釈（合っているか不安）：
　[例：バブルっぽい気がするが
　　違うかもしれない]

# 指示内容
1. 私の状況認識に対して
　「別の見方ができる視点」を3つ提示

2. 「今の相場の本質」を理解するための
　問いを5つ立ててください

3. 長期投資家として
　「今考えるべきこと・
　　考えなくていいこと」を整理

# 制約条件
・相場の予測・断言はしない
・「考えるための地図」を提供する
・初心者でも理解できる言葉で

# 出力形式
番号つき見出しで整理。
専門用語には（）で補足。

# 入力データ
今の相場観・気になること：[ここに入力]
```

**✅具体的な使用方法：**  
　「自分の解釈（合っているか不安）」の欄に、  
　たとえ間違っていても  
　正直に書いてください。

　AIはそれを材料に、  
　新しい視点を返してくれます。

**✅読者への一言：**  
　「相場の予測」は誰にも無理です。

　でも「相場の本質を理解する力」は、  
　誰でも鍛えられます。

　その差が、10年後に出ます。

## 5. 投資判断を振り返り、学びに変える

「なんで買ったんだろう…」

失敗した投資を  
ただ後悔して終わっていませんか？

失敗は、  
使い方次第で「最高の教科書」に変わります。

AIと一緒に振り返ることで、  
次の判断が確実に磨かれます。

### ◆投資振り返り学習プロンプト

**✅目的：**  
　過去の投資判断を客観的に振り返り、  
　再現性のある学びに変える。

**✅いつ使うか：**  
　売却後・損切り後・  
　「あのとき違う判断をすればよかった」  
　と思ったとき。

　月に1回の定期振り返りにも最適。

```
# ロール設定
あなたは投資の振り返りを専門とする
コーチです。
成功・失敗を問わず、
投資判断のプロセスを客観的に分析し、
次に活かせる学びを引き出してください。

# 背景・目的・対象
・振り返る投資の概要：
　[例：A社株を100万で購入し、
　　70万で売却した]
・買った理由：
　[例：SNSで評判が良かった・
　　なんとなく上がりそうだった]
・売った理由：
　[例：怖くなって我慢できなかった]
・当時の感情：
　[例：焦り・後悔・自信過剰]

# 指示内容
1. この判断プロセスの
　「良かった点」を1〜2つ挙げる
　（ゼロでもOK）

2. 「改善できた点」を3つ挙げ、
　どう変えればよかったかを具体的に示す

3. 同じ状況が来たとき、
　「次はどう判断すべきか」の
　マイルール案を2つ作る

4. この経験から得られる
　「投資の本質的な学び」を
　一言で言い換える

# 制約条件
・過去の判断を責めない
　「気づき」として捉える
・次の行動につながる言葉にする

# 出力形式
番号つき見出しで整理。
最後に「学びの一言」をまとめる。

# 入力データ
投資の詳細・感情・経緯：[ここに入力]
```

**✅具体的な使用方法：**  
　「当時の感情」を  
　正直に書くのが最大のコツです。

　恥ずかしくても大丈夫。  
　AIは責めません。

　むしろ、  
　感情を書くほど  
　核心をついた学びが返ってきます。

**✅読者への一言：**  
　投資の失敗は、  
　「お金を払って受けた授業料」です。

　振り返らなければ  
　ただの損失ですが、  
　振り返れば財産になります。

　AIをノート代わりに使ってください。

## まとめ

投資の世界では、  
情報は溢れているのに  
「自分の答え」はなかなか見つからない。

そのギャップを埋めてくれるのが、  
AIとの対話です。

相場の予測ではなく、  
「なぜ投資するか」「何を大切にするか」  
という自分軸を育てること。

それが、  
どんな相場でも揺らがない  
投資家への道です。

「考えた量だけ、判断は磨かれる」  
という言葉を信じて、  
今日から少しずつ始めてみてください。

参考になったら、  
ぜひ**スキ・フォロー**をいただけると  
とても励みになります。
