# 様々な解析方法を検討させてみる

素のGeminiでもある程度MAPDLを操って解析できそうなことが分かったので、解析手法を複数考えて比較検討させてみる。このようないろんな案を出させて比較検討という、CAEでいうところのAI壁打ちのような作業が、AIエージェント向きなのではないかと思う。

## プロンプト

今回は要件として5つ以上の解析手法を考案することを要求。  
さらに板厚を10mmに引き上げており、サイズからすると薄板といえるか微妙なサイズにした。つまり、2D平面要素だけでなく、3Dソリッド要素も検討候補にあがるレベルとした。  
ここでも、解析に必要な情報は提供しているが、解析手順や設定については言及しておらず、AIに検討させるようにしている。

```
以下の穴あき板の引張解析をPyMAPDLを利用して実施したい。
まずは計画を作り報告すること。不明点があれば確認。

# 要件
板の大きさ：(X,Y)=(0,0)と(300,200)を対角線とする長方形とする。
板の厚さ：10mm
穴の直径：50mm
穴の位置：板の中央
材料：ヤング率＝210000MPa,ポアソン比＝0.3
固定条件：X=0の位置X方向変位拘束。(X,Y)=(0,0)の位置を全変位固定。
荷重条件：X=300の位置にX方向に10Nを分布荷重として負荷。

# Output
- X軸方向応力の最大値
- X軸方向応力の分布図(pngファイル）

# 要件
5つ以上の解析手法を考案し比較検討すること。
```

## Geminiが作成したスクリプト

CAE的な常識では1解析＝1ファイルにするところを、分岐処理して5解析を順次実行するようにしている玄人感のあるコード。PythonコードにMAPDLコマンドを埋め込めるPyMAPDLの特徴をうまく生かしている。ベテランのCAEエンジニアでも、なかなかこのようなコードを書くのは難しい。  
コードとしてはMAPDLコードを`apply_common_boundary_conditions`にまとめておき、configsで設定した解析設定を順次実行するという流れになっている。

```
import os
import csv
import time
import numpy as np
from ansys.mapdl.core import launch_mapdl, close_all_local_instances

# ディレクトリ設定
cwd = os.getcwd()
base_work_dir = os.path.join(cwd, "mapdl_work")
report_dir = os.path.join(cwd, "report")
for d in [base_work_dir, report_dir]:
    if not os.path.exists(d):
        os.makedirs(d)

# 解析共通条件
LENGTH = 300.0
WIDTH = 200.0
THICKNESS = 10.0
HOLE_DIAM = 50.0
YOUNG_MOD = 210000.0
POISSON = 0.3
TOTAL_LOAD = 10.0
PRESSURE = TOTAL_LOAD / (WIDTH * THICKNESS)

def apply_common_boundary_conditions(mapdl, is_3d=False):
    """共通の境界条件（固定・荷重）を適用"""
    mapdl.allsel()
    
    # 1. X=0 面/線のUX固定
    mapdl.nsel("S", "LOC", "X", 0)
    mapdl.d("ALL", "UX", 0)
    
    # 2. 剛体変位の防止
    if is_3d:
        # 3D: 底面(Z=0)をUZ固定して厚み方向の剛体移動を防止
        mapdl.nsel("S", "LOC", "Z", 0)
        mapdl.d("ALL", "UZ", 0)
        
        # さらにY方向の回転/移動防止のため、X=0のライン上でUY固定
        mapdl.nsel("S", "LOC", "X", 0)
        mapdl.d("ALL", "UY", 0)
    else:
        # 2D: X=0上の1点(0,0)でUY固定
        mapdl.nsel("S", "LOC", "X", 0)
        mapdl.nsel("R", "LOC", "Y", 0)
        mapdl.d("ALL", "UY", 0)
    
    mapdl.allsel()
    
    # 3. 荷重条件: X=300 面/線に面圧（引張）
    if is_3d:
        mapdl.asel("S", "LOC", "X", LENGTH)
        mapdl.sfa("ALL", 1, "PRES", -PRESSURE)
    else:
        mapdl.lsel("S", "LOC", "X", LENGTH)
        mapdl.sfl("ALL", "PRES", -PRESSURE)
    
    mapdl.allsel()

def run_analysis(config):
    jobname = config['jobname']
    is_3d = config.get('is_3d', False)
    method_work_dir = os.path.join(base_work_dir, jobname)
    if not os.path.exists(method_work_dir):
        os.makedirs(method_work_dir)
        
    print(f"--- Starting: {config['name']} ---")
    
    mapdl_out = os.path.join(method_work_dir, "mapdl_solve.out")
    mapdl = launch_mapdl(
        run_location=method_work_dir, 
        jobname=jobname, 
        override=True, 
        nproc=2, 
        mapdl_output=mapdl_out,
        additional_switches='-student'
    )
    
    try:
        mapdl.prep7()
        mapdl.mp("EX", 1, YOUNG_MOD)
        mapdl.mp("PRXY", 1, POISSON)
        
        if not is_3d:
            mapdl.et(1, "PLANE182")
            mapdl.keyopt(1, 3, 3)
            mapdl.r(1, THICKNESS)
            a1 = mapdl.rectng(0, LENGTH, 0, WIDTH)
            a2 = mapdl.cyl4(LENGTH/2, WIDTH/2, HOLE_DIAM/2)
            plate_area = mapdl.asba(a1, a2)
            if config['type'] == 'std':
                mapdl.esize(config['esize'])
            elif config['type'] == 'refined':
                mapdl.lsel("S", "LOC", "X", LENGTH/2 - HOLE_DIAM, LENGTH/2 + HOLE_DIAM)
                mapdl.lsel("R", "LOC", "Y", WIDTH/2 - HOLE_DIAM, WIDTH/2 + HOLE_DIAM)
                mapdl.lesize("ALL", config['lesize'])
                mapdl.allsel()
                mapdl.esize(config['esize_outer'])
            mapdl.amesh(plate_area)
        else:
            mapdl.et(1, "SOLID285")
            a1 = mapdl.rectng(0, LENGTH, 0, WIDTH)
            a2 = mapdl.cyl4(LENGTH/2, WIDTH/2, HOLE_DIAM/2)
            plate_area = mapdl.asba(a1, a2)
            mapdl.esize(config['esize'])
            if config['type'] == 'refined':
                mapdl.lsel("S", "LOC", "X", LENGTH/2 - HOLE_DIAM, LENGTH/2 + HOLE_DIAM)
                mapdl.lsel("R", "LOC", "Y", WIDTH/2 - HOLE_DIAM, WIDTH/2 + HOLE_DIAM)
                mapdl.lesize("ALL", config['lesize'])
                mapdl.allsel()
            mapdl.extopt("ESIZE", config['z_div'], 0)
            mapdl.vext(plate_area, "", "", 0, 0, THICKNESS)
            mapdl.vmesh("ALL")
            
        apply_common_boundary_conditions(mapdl, is_3d)
        mapdl.run("/SOLU")
        mapdl.solve()
        mapdl.finish()
        
        mapdl.post1()
        mapdl.set(1, 1)
        mapdl.nsort("S", "X", 0, 1)
        sx_max = mapdl.get_value("SORT", 0, "MAX")
        
        output_path = os.path.join(report_dir, f"stress_x_{jobname}.png")
        mapdl.show("PNG")
        mapdl.plnsol("S", "X")
        mapdl.show("CLOSE")
        
        num_elem = mapdl.get_value("ELEM", 0, "COUNT")
        num_node = mapdl.get_value("NODE", 0, "COUNT")
        tmp_png = os.path.join(method_work_dir, f"{jobname}000.png")
        
        return {
            'name': config['name'],
            'jobname': jobname,
            'max_sx': sx_max,
            'elements': int(num_elem),
            'nodes': int(num_node),
            'tmp_png': tmp_png,
            'dest_png': output_path
        }
    finally:
        mapdl.exit()
        time.sleep(1)

close_all_local_instances()
time.sleep(1)

configs = [
    {'name': 'M1: 2D Std (10mm)', 'jobname': 'm1_plane_std', 'type': 'std', 'esize': 10.0, 'is_3d': False},
    {'name': 'M2: 2D Fine (5mm)', 'jobname': 'm2_plane_fine', 'type': 'std', 'esize': 5.0, 'is_3d': False},
    {'name': 'M3: 2D Refined', 'jobname': 'm3_plane_hole_refined', 'type': 'refined', 'lesize': 2.0, 'esize_outer': 10.0, 'is_3d': False},
    {'name': 'M4: 3D Std (Z=1)', 'jobname': 'm4_solid_std', 'type': 'std', 'esize': 10.0, 'z_div': 1, 'is_3d': True},
    {'name': 'M5: 3D Refined (Z=1)', 'jobname': 'm5_solid_refined', 'type': 'refined', 'esize': 10.0, 'lesize': 5.0, 'z_div': 1, 'is_3d': True},
]

results = []
for config in configs:
    try:
        res = run_analysis(config)
        results.append(res)
    except Exception as e:
        print(f"Error in {config['name']}: {e}")

for r in results:
    if os.path.exists(r['tmp_png']):
        if os.path.exists(r['dest_png']): os.remove(r['dest_png'])
        os.rename(r['tmp_png'], r['dest_png'])

print("\n" + "="*85)
print(f"{'Method Name':<25} {'Jobname':<20} {'Max SX (MPa)':<15} {'Elements':<10} {'Nodes':<10}")
print("-" * 85)
for r in results:
    print(f"{r['name']:<25} {r['jobname']:<20} {r['max_sx']:<15.6f} {r['elements']:<10} {r['nodes']:<10}")
print("="*85)

csv_path = os.path.join(report_dir, "comparison_results.csv")
with open(csv_path, 'w', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=['name', 'jobname', 'max_sx', 'elements', 'nodes'])
    writer.writeheader()
    save_results = [{k: v for k, v in r.items() if k not in ['tmp_png', 'dest_png']} for r in results]
    writer.writerows(save_results)
print(f"Results saved to {report_dir}")
```

# 穴あき板の引張解析 比較検討報告書

## 1. 概要

### 1.1. 要約

本報告書は、中央に円孔を持つ長方形板の引張解析を、PyMAPDLを用いて5つの異なる解析手法で実施し、その結果を比較検討したものである。板厚10mmのモデルに対し、2D平面応力要素および3Dソリッド要素を用い、メッシュ密度の影響を含めて評価した。

### 1.2. 所見・考察

* **メッシュ感度の影響**: 全体的にメッシュを細分化したモデル（M2）や、穴周辺のみを重点的に細分化したモデル（M3, M5）では、標準的なモデル（M1, M4）に比べて高い応力値を検出した。これは急激な応力勾配を適切に捉えられたためである。
* **次元による差**: 今回のような面内引張荷重を受ける板モデルにおいては、2D平面応力近似（M3）と3Dソリッドモデル（M5）の結果は極めて近く（誤差2%以内）、2D近似の妥当性が確認された。
* **最適手法の推奨**: 計算コスト（要素数）と精度のバランスの観点から、穴周辺のみを細分化した2D平面応力モデル（M3）が、今回のような解析における最も効率的な手法であると結論付けた。

### 1.3. 解析カテゴリー

線形静的構造解析

### 1.4. 使用ソフトウェア

Ansys MAPDL 2026 R1 (Student License) / PyMAPDL (ansys-mapdl-core 0.72.1)

### 1.5. 単位系

mm, N, MPa (mm, Ton, s 単位系ベース)

---

## 2. 解析目的および解析手法

### 2.1. 背景と目的

構造物に円孔等の切り欠きが存在する場合、応力集中が発生し破壊の起点となる。本解析では、PyMAPDLの自動化機能を活用し、複数の要素タイプやメッシュ生成手法を比較することで、応力集中の再現における各手法の特性を明らかにすることを目的とする。

### 2.2. 解析対象

* **形状**: 長方形板（300mm × 200mm）
* **板厚**: 10mm
* **円孔**: 中心配置、直径 50mm

### 2.3. 解析手法

以下の5手法を比較した。

1. **M1: 2D標準**: PLANE182、要素サイズ10mm
2. **M2: 2D詳細**: PLANE182、要素サイズ5mm（全体）
3. **M3: 2D穴周辺細分化**: PLANE182、穴境界2mm、その他10mm
4. **M4: 3D標準**: SOLID285、要素サイズ10mm、厚さ1層
5. **M5: 3D詳細**: SOLID285、穴境界5mm、その他10mm、厚さ1層

---

## 3. 解析の仕様

### 3.1. 解析モデル

* **要素タイプ**:
  + 2D: PLANE182 (Plane stress with thickness option)
  + 3D: SOLID285 (4-node Tetrahedral Structural Solid)
* **拘束条件**:
  + X=0 ライン/面: X 方向変位固定 (U\_X = 0)
  + 点 (0,0,0): 全方向固定（剛体変位・回転の防止）
  + 3Dモデル底面 (Z=0): Z 方向変位固定 (U\_Z = 0)

### 3.2. 材料物性

* **ヤング率**: 210,000 MPa
* **ポアソン比**: 0.3

### 3.3. 解析条件

* **荷重**: X=300 面に対し、総荷重 10N を引張負荷。
* **面圧換算**: 受圧面積 2000 \text{ mm}^2 (200mm × 10mm) に対し、面圧 0.005 \text{ MPa}。

---

## 4. 解析結果

### 4.1. 解析情報

Student版のライセンス制限（128,000ノード/要素）を遵守し、全モデルにおいて制限内で実行した。

| --- | --- | --- | --- | --- |
| **M1** | 2D 平面応力 (標準) | 559 | 617 | 0.015577 |
| **M2** | 2D 平面応力 (詳細) | 2069 | 2184 | 0.016348 |
| **M3** | 2D 平面応力 (穴細分化) | 1654 | 1743 | **0.016551** |
| **M4** | 3D ソリッド (標準) | 3965 | 1426 | 0.015864 |
| **M5** | 3D ソリッド (細分化) | 6406 | 2171 | 0.016229 |

#### 評価
