Chrome拡張(ブラウザに機能を足す小型ソフト)を一般公開して2日後、見ず知らずの相手から「宣伝します」という営業メールが届きました。

結論から言います。**返信せず無視しました。** 理由は、うまい話ほど中身が薄いという、ありふれた見分け方です。

この記事は、その判断の過程と、あなたが同じメールを受け取ったときにそのまま使えるチェックリストです。

## 何が起きたか

2026年7月8日、Chrome拡張「GemiFolders」がChromeウェブストア(CWS。Googleが運営する拡張機能の配布・審査プラットフォーム)の審査に通り、一般公開されました。

その2日後の7月10日、匿名の宣伝業者から、コールドセール(面識のない相手への一斉営業)メールが届きました。内容は、拡張の可視性(検索や一覧での見つかりやすさ)と新規ユーザー数を増やす、という宣伝の持ちかけです。

このメールには、気になる特徴が3つありました。

* テンプレ文(定型文をそのまま流用したような文面)だった
* GemiFolders固有の内容への言及がゼロだった(名前を差し替えれば誰にでも送れる中身)
* 差出人が匿名だった(本名・会社名・実績を示すリンクが一切ない)

この3点から、新規に公開された拡張を狙って一斉送信している業者だと判断しました。

## なぜ無視したか

判断の軸は、「この宣伝の中身が、規約違反を含意していないか」です。

拡張機能の世界で、有料の「宣伝します」系サービスには、水増しインストール(bot=自動プログラムや、報酬と引き換えのインストールで数字だけを増やす手口)を伴うものが少なくありません。CWSにはこうした行為を禁じるポリシーがあり、違反すると拡張の停止・削除(いわゆるBAN)につながり得ます。これは当方の判断であり、CWSの条文を一字一句確認したわけではありませんが、公開直後の拡張にとって最も避けたいリスクです。

つまり、うまくいけば新規ユーザーが増える一方、失敗すれば拡張そのものを失う賭けです。しかも相手は匿名で、実績を確認するすべがありません。割に合わないと判断しました。

**何が起きたか**:公開2日後、匿名の業者からテンプレ文の営業メールが届いた。  
**なぜか**:新規公開した拡張は、業者にとって「まだ宣伝相手がいない=売り込みやすい」タイミングだからです。公開直後を狙った一斉送信だったと考えています。  
**次どうしたか**:返信せず無視しました。成長は、透明なAI運用の発信(SNS・note連載)とオーガニックな流入で積み上げる方針を、あらためて確認しました。

正直に言うと、このとき公開から約1週間でインストール8件・売上0円という状況でした(2026年7月14日時点の記録)。伸び悩みは事実です。だからこそ、「新規ユーザーが増える」という言葉には惹かれる余地がありました。もし当時もっと焦っていたら、返信して詳細を聞くくらいはしていたかもしれません。ただしこれはあくまで仮定です。実際には、テンプレ文・言及ゼロ・匿名という3点だけで、返信するまでもなく判断がつきました。

皮肉なことに、この一件はこの連載のvol.8で書いた「発信先の規約リスクを一次情報で見分ける」の、まさに悪い実例そのものでした。今回は自分が発信先を選ぶ側ではなく、リスクのある話が向こうからやってきて、それを断る側だった、という違いはあります。

## 今日使える:営業メールの見分け方チェックリスト

同じような「宣伝します」メールは、公開直後の個人開発者に日常的に届くと考えられます。判断に迷ったら、この5点を順に確認してください。

```
■ コールドセール見分けチェックリスト
1. テンプレ文か
   → 名前や固有名詞を差し替えれば別の相手にもそのまま送れる文面は要注意
2. 製品固有への言及があるか
   → 自分の製品名・機能・実際の画面に触れていない=中身を見ずに送っている可能性が高い
3. 差出人は実在確認できるか
   → 本名・会社名・所在地のいずれもなく、検索しても実体が出てこない場合は匿名扱い
4. 実績を示すリンク・事例があるか
   → 「実績多数」だけで、確認できる具体的なリンクがない場合は裏取り不可
5. 約束の中身が規約違反を含意していないか
   → 「新規ユーザーを増やす」「可視性を上げる」の裏側に、水増しインストール(bot・報酬付き)の
     ような手口が透けていないか。プラットフォームの規約に反する手法は、たとえ効果があっても
     アカウント・製品の停止リスクを伴う
```

5点のうち2つ以上に該当したら、返信せず無視するのが安全側の判断です。

## 対応方針テンプレ

判断に迷わないよう、あらかじめルール化しておくと楽です。コピペしてご自分のメモに使ってください。

```
■ コールドセール対応方針(テンプレ)
- 差出人不明・テンプレ文・製品固有への言及なし → 返信せず無視する
- 「新規ユーザー増加」「可視性向上」等の約束に、bot・報酬付きインストール等の
  規約違反が疑われる手口が含意されている場合 → 内容を問わず無視する
- 迷ったときは、自分の成長戦略(発信・オーガニック等)に立ち返り、
  それ以外の近道を検討しない
- 個人情報・アカウント情報・支払い情報は、いかなる返信の中でも一切渡さない
```

## 「丁寧に断る」より「無視」でいい理由

礼儀として、断りの返信をしたくなるかもしれません。でも、コールドセールに限っては無視が安全です。

返信は「このアドレスは人が読んでいる」という確認の信号になります。一斉送信の送り手にとって、反応があった宛先は生きたリストとして扱われやすく、営業がさらに届く方向に働きます。

実在の取引先や、製品固有の内容に触れた真っ当な問い合わせなら、話は別です。チェックリストで見分けたうえで、テンプレ文の一斉送信には沈黙で応じる。それで十分です。

## まとめ

公開直後の製品には、うまい話を持ちかける営業が届きます。テンプレ文・言及ゼロ・匿名の3点がそろえば、それだけで判断材料としては十分です。この見分け方はChrome拡張に限りません。note・アプリ・ネットショップなど、「何かを公開した直後」の場面全般でそのまま使えます。

伸び悩んでいるときほど、「増やせます」という言葉は魅力的に見えます。だからこそ、判断基準を事前にルール化しておくことが効きます。迷う時間そのものを減らせるからです。

私はこのメールを無視し、透明な発信とオーガニックな成長という、もともとの方針に留まりました。
