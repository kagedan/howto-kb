# 無職がClaude（AI）だけで日本初の地域エコシステムを作った話

## はじめに

私は現在、**無職**です。

それを隠すつもりはありません。

福岡県飯塚市——かつて日本の炭鉱産業の中心地として栄え、閉山後の経済的変革を経た街——に住んでいます。

2025年、私はこの街から「Iizuka LinkedIn Network」を立ち上げました。

**使ったもの：** Claude（AI）、GitHub（無料枠）、LinkedIn（無料枠）  
**使った予算：** ¥0  
**チームの人数：** 1人

---

## 何を作ったか

筑豊地域（飯塚市・田川市・直方市ほか11市町村）を対象とした、日本初のオープンソース地域プロフェッショナルエコシステムです。

具体的には：

* **11都市のWikipedia型知識ベース**（産業・AI機会・DX機会・スタートアップ機会）
* **AI・DX学習リソース**（入門〜自動化まで）
* **求人データベース**（リモート・AI・DX・地元・インターン）
* **スタートアップ支援**（補助金・法務・無料ツール・メンター）
* **LinkedInグループ**（Iizuka Tagawa Nogata Professional Growth & Network Hub）
* **2026〜2030年イベントカレンダー**

規模：90以上のMarkdownファイル・47ディレクトリ・GitHub Pages公開済み

## なぜ作ったか

筑豊地域には優れた人材と文化遺産があります。

しかし、AI・DX・LinkedInといった現代の機会へのアクセスは、都市部と比べて圧倒的に不足していました。

「地域のプロフェッショナルが使える知識インフラが存在しない」

その課題を解決するために作りました。

---

## どうやって作ったか（全手順）

### ステップ1：設計（Claude使用）

まずClaudeに全体設計を依頼しました。

```
プロンプト例：
「筑豊地域（福岡県）のプロフェッショナル向けに、
Wikipedia + LinkedIn + スタートアップハブを
GitHubで作りたい。ディレクトリ構造を設計して。」
```

出力されたディレクトリ構造をベースに、47個のディレクトリを作成しました。

### ステップ2：コンテンツ生成（Claude使用）

各ページのMarkdownをClaudeに生成させました。

```
プロンプト例：
「飯塚市のプロフェッショナル向けページを作って。
以下を含めること：概要・人口・主要産業・
AI機会・DX機会・スタートアップ機会・政府支援」
```

11都市分を同様に生成。

### ステップ3：自動化（Python + GitHub API）

ファイルのGitHubへのアップロードはPythonスクリプトで自動化しました。

```
import urllib.request, json, base64

def upload(local_path, gh_path, token, repo):
    with open(local_path, "rb") as f:
        content = base64.b64encode(f.read()).decode()
    payload = {"message": f"add {gh_path}", "content": content}
    # GitHub Contents API でPUT
    ...
```

83ファイルを一括アップロード。手動作業はゼロ。

### ステップ4：SEO設計（Claude使用）

```
プロンプト例：
「このリポジトリのSEOキーワード戦略を設計して。
ターゲット：Iizuka LinkedIn / Chikuho AI / 
Fukuoka Professional Networking」
```

Tier1〜Tier3のキーワードアーキテクチャを設計。

### ステップ5：GitHub Pages公開

Settings → Pages → main → Save

2分後に公開完了。
