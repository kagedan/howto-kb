## TL;DR

* 国土数値情報の将来推計人口（GeoJSON）のメタ情報をOKFで管理
* （擬似的な）不動産分析タスクによって、OKFの有効性の検証を実施
* OKFが分析の正確性およびコスト削減に有効であることを確認
* ただし生成AIがドメイン知識を持っている場合、OKFの有無で結果が変わらない可能性
* AIエージェントの挙動をコントロールしないと正確なコスト削減効果を測るのは難しい

## OKF（Open Knowledge Format）

AIエージェントがナレッジ（データ、指標、ドキュメント）を効率的に理解・収集できるように、Google Cloudが提案したオープン仕様です。ディレクトリ構造とYAMLフロントマター付きのMarkdownを組み合わせるだけで、人間もAIも扱いやすいのが特徴となっています。

詳細はいくつかの解説記事に譲りますが、基本的にはフォーマット通りにMarkdownファイルの冒頭に情報を書くだけという非常にシンプルなものですが、その書き方に一貫性があることが、ナレッジマネジメント上は非常に重要と理解しています。

その中で１つ良いなと思ったのが、"**Joins**"によって概念同士をリンクで繋ぐというものです。といっても、Markdownファイルで、

```
# Joins

Joined with [customers](/tables/customers.md) on `customer_id`.
```

といったように記述するだけで、他の情報と簡単に繋げることができます。そして、この情報同士をつなぐ関係性をあらかじめOKFで記述しておけば、AIエージェントにナレッジとして関係性の情報を渡すことができる、すなわち**AIエージェントがわざわざデータや仕様書を読みにいかなくても処理できるため、コスト（トークン消費）が節約できる**のではないか、と期待できます。

## 実験設定

ということで、簡単な不動産分析のタスクで実験してみました。

### 想定タスク

今回は、擬似的にREIT法人が将来的に人口が増加しそうな地域の物件を保有しているかどうか、を分析することを考えます。具体的には、REITが持つ保有物件ごとに、その地域の将来人口増加率を紐づけて、その増加率のヒストグラムを描写してREIT間で比較することを目標とします。こういったヒストグラムを見ることで、REITが将来有望な地域の物件を持っているかどうかが可視化でき、REITの相対的な評価の１つとして使うことができます。

### 利用データセット

まず、人口動態については[国土数値情報の1kmメッシュ別将来推計人口データ（R6国政局推計）](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-mesh1000r6.html)を利用します。データ概要は下記のとおりです。

| 項目 | 内容 |
| --- | --- |
| 内容 | 令和2年の国勢調査に基づき、2070年までの250mメッシュ別の将来人口について、男女別・年齢（5歳階級）別に推計を行い、1kmメッシュ毎に集計しました。 |
| データ作成年度 | 2024年度（令和6年度） |
| 関連する法律 | － |
| 原典資料 | 総務省「令和2年国勢調査」、国立社会保障・人口問題研究所「日本の将来推計人口（令和5年推計）」の推計値、「日本の地域別将来推計人口（令和5年推計）」の推計値及び仮定値（生残率、子ども女性比、純移動率）等 |
| 作成方法 | 令和2年国勢調査を基準としたメッシュ別将来推計人口の試算方法について |
| このデータの使用許諾条件 | オープンデータ（CC\_BY\_4.0） |
| 座標系 | 世界測地系（JGD2011） / （B, L） |
| データ形状 | 基準地域メッシュ（第３次地域区画） |

このうち東京都のGeoJSON形式のデータ（1km\_mesh\_2024\_13\_GEOJSON.zip）をダウンロードして利用します。また、市区町村との紐づけのために行政区域コード（AdminiBoundary\_CD.xlsx）も利用します。

[J-REIT銘柄](https://j-reit.jp/brand/)の実際の保有物件についても、各投資法人のHPから取得可能ですが、今回は擬似的にデータ(csv)を生成しました。

データ生成用スクリプト

generate\_dummy\_jreit\_properties.py

```
from __future__ import annotations

import csv
import random
from datetime import date, timedelta
from pathlib import Path

import openpyxl

ROOT = Path(__file__).resolve().parents[1]
RAW_DIR = ROOT / "rawdata"
ADMINI_PATH = RAW_DIR / "AdminiBoundary_CD.xlsx"
OUTPUT_PATH = RAW_DIR / "jreit_properties_dummy_tokyo.csv"

REIT_NAMES = ["A社", "B社"]
ASSET_TYPES = ["Office", "Residential", "Retail", "Logistics", "Hotel"]
RNG = random.Random(20260714)
TOKYO_23_WARD_CODES = {f"131{num:02d}" for num in range(1, 24)}

def load_tokyo_municipalities() -> dict[str, str]:
    wb = openpyxl.load_workbook(ADMINI_PATH, data_only=True, read_only=True)
    ws = wb["行政区域コード"]

    municipalities: dict[str, str] = {}
    for idx, row in enumerate(ws.iter_rows(values_only=True), start=1):
        if idx < 4:
            continue
        code, prefecture_name, municipality_name, *_ = row
        if prefecture_name == "東京都" and municipality_name:
            municipalities[str(code)] = str(municipality_name)
    return municipalities

def random_date() -> str:
    start = date(2012, 1, 1)
    end = date(2024, 12, 31)
    delta_days = (end - start).days
    return (start + timedelta(days=RNG.randint(0, delta_days))).isoformat()

def choose_municipality_codes(municipalities: dict[str, str], count: int) -> list[str]:
    codes = sorted(municipalities)
    weights = [6 if code in TOKYO_23_WARD_CODES else 1 for code in codes]
    return RNG.choices(codes, weights=weights, k=count)

def build_rows(municipalities: dict[str, str]) -> list[dict]:
    selected_codes = choose_municipality_codes(municipalities, 40)
    rows = []

    for idx, municipality_code in enumerate(selected_codes, start=1):
        reit_name = REIT_NAMES[0] if idx <= 20 else REIT_NAMES[1]
        property_sequence = idx if idx <= 20 else idx - 20
        asset_type = ASSET_TYPES[(idx - 1) % len(ASSET_TYPES)]
        occupancy_rate = round(RNG.uniform(88.0, 99.8), 1)
        leasable_area_sqm = RNG.randint(2500, 45000)
        appraisal_value_mn_jpy = RNG.randint(1800, 42000)
        municipality_name = municipalities[municipality_code]

        rows.append(
            {
                "property_id": f"JREIT-TKY-{idx:03d}",
                "reit_name": reit_name,
                "property_name": f"{reit_name}_{municipality_name}_{asset_type}_{property_sequence:02d}",
                "asset_type": asset_type,
                "prefecture_name": "東京都",
                "municipality_name": municipality_name,
                "acquisition_date": random_date(),
                "appraisal_value_mn_jpy": appraisal_value_mn_jpy,
                "leasable_area_sqm": leasable_area_sqm,
                "occupancy_rate_pct": occupancy_rate,
                "analysis_note": "dummy record for population trend analysis",
            }
        )

    return rows

def write_csv(rows: list[dict]) -> None:
    fieldnames = [
        "property_id",
        "reit_name",
        "property_name",
        "asset_type",
        "prefecture_name",
        "municipality_name",
        "acquisition_date",
        "appraisal_value_mn_jpy",
        "leasable_area_sqm",
        "occupancy_rate_pct",
        "analysis_note",
    ]

    with OUTPUT_PATH.open("w", newline="", encoding="utf-8") as f:
        writer = csv.DictWriter(f, fieldnames=fieldnames)
        writer.writeheader()
        writer.writerows(rows)

def main() -> None:
    municipalities = load_tokyo_municipalities()
    rows = build_rows(municipalities)
    write_csv(rows)
    print(f"Wrote {len(rows)} rows to {OUTPUT_PATH}")

if __name__ == "__main__":
    main()
```

補足:

- 数値列は GeoJSON 上では `float` として格納されています。整数に見える値も小数型で入ります。
- `HITOKU{YYYY}` / `GASSAN{YYYY}` は全メッシュで埋まる列ではなく、秘匿処理が必要な一部メッシュだけに値があります。
- `PTN_{YYYY}` と `PT00_{YYYY}` は同義ではありません。公式定義では、`PTN_{YYYY}` は「秘匿を除く総数人口」、`PT00_{YYYY}` は「総数人口」です。
- 2020 年は基準年として `PTN_2020` のみがあり、2025 年以降のような年齢階級別列はありません。

# Sheet

この Excel ブックには 3 つの sheet があります。

| Sheet 名 | 用途 | 実際に使うか | 範囲 / 開始位置 | Notes |
|----------|------|--------------|-----------------|-------|
| `行政区域コード` | 行政区域コードの主表 | Yes | タイトルは `A1`、ヘッダは `A3:K3`、データは `A4:K2951` | 実データは `A1` 開始ではありません。1 行目はタイトル、2 行目は空行です。 |
| `この行政区域コードについて` | データの作成元とコード体系の説明 | Reference only | `A1:A7` | 表形式のデータではなく説明文です。 |
| `行政区域コード更新履歴` | ブック自体の更新履歴 | Reference only | `A1:B5` | 表形式ではあるものの、分析用の主データではありません。 |

主表として使うのは `行政区域コード` sheet です。`A1` に「国土数値情報用 行政区域コード （令和6年1月1日現在）」というタイトルがあり、列ヘッダは 3 行目から始まります。

取り込み時の注意:

- `行政区域コード` sheet は `A1` からそのまま読み込めません。1 行目はタイトル、2 行目は空行、3 行目がヘッダ、4 行目以降がデータです。
- 列名セルには改行を含みます。例: `都道府県名\n（漢字）`、`改正後の名称\n(ｶﾅ)`。
- `行政区域コード` と `改正後のコード` は先頭ゼロを保持する必要があるため、数値ではなく文字列として扱う前提です。
- `改正年月日` は型が混在しています。通常の日付セルに加えて、Excel シリアル値 (`40269` など) や和暦文字列 (`H26.1.1` など) が含まれます。
- 都道府県レベルの行では `市区町村名（漢字）` と `市区町村名（ｶﾅ）` が空です。例: `01000 / 北海道`。
- 一部の名称セルには改行や前後空白が含まれます。例: `幌加内町\n（空知支庁）`、` 幌加内町\n（上川総合振興局）`。

# Schema

対象 sheet: `行政区域コード`

レコード数:

実データ上の補足:

- `コードの改定区分` 以降の改定関連列は 983 行で値が入っています。
- `改正後のコード` は改定有無にかかわらず埋まっており、現行コードへの正規化キーとして使えます。

from __future__ import annotations

import json
from collections import defaultdict
from pathlib import Path

import matplotlib.font_manager as fm
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

ROOT = Path(__file__).resolve().parents[1]
RAW_DIR = ROOT / "rawdata"
IMAGES_DIR = ROOT / "images"

PROPERTY_PATH = RAW_DIR / "jreit_properties_dummy_tokyo.csv"
ADMIN_PATH = RAW_DIR / "AdminiBoundary_CD.xlsx"
MESH_PATH = RAW_DIR / "1km_mesh_2024_13.geojson"

PROPERTY_OUTPUT_PATH = IMAGES_DIR / "jreit_working_age_population_by_property.csv"
REIT_OUTPUT_PATH = IMAGES_DIR / "jreit_working_age_population_by_reit.csv"
FIGURE_OUTPUT_PATH = IMAGES_DIR / "jreit_working_age_population_histograms.png"

def configure_font() -> None:
    candidate_fonts = [
        "Noto Sans CJK JP",
        "IPAexGothic",
        "IPAGothic",
        "Yu Gothic",
        "Hiragino Sans",
        "TakaoGothic",
    ]
    installed = {font.name for font in fm.fontManager.ttflist}
    for font_name in candidate_fonts:
        if font_name in installed:
            plt.rcParams["font.family"] = font_name
            break
    plt.rcParams["axes.unicode_minus"] = False

def load_admin_mapping() -> pd.DataFrame:
    admin = pd.read_excel(
        ADMIN_PATH,
        sheet_name="行政区域コード",
        header=2,
        dtype={"行政区域コード": str, "改正後のコード": str},
    )
    admin = admin.rename(columns=lambda col: str(col).replace("\n", ""))
    admin = admin[
        admin["都道府県名（漢字）"].eq("東京都") & admin["市区町村名（漢字）"].notna()
    ].copy()
    admin["municipality_code"] = admin["改正後のコード"].astype(str).str.zfill(5)
    admin = admin.rename(
        columns={
            "都道府県名（漢字）": "prefecture_name",
            "市区町村名（漢字）": "municipality_name",
        }
    )
    admin = admin[["prefecture_name", "municipality_name", "municipality_code"]]
    return admin.drop_duplicates(subset=["prefecture_name", "municipality_name"])

def load_properties() -> pd.DataFrame:
    return pd.read_csv(PROPERTY_PATH)

def build_municipality_population_frame(target_codes: set[str]) -> pd.DataFrame:
    with MESH_PATH.open(encoding="utf-8") as f:
        features = json.load(f)["features"]

    totals: dict[str, dict[str, float]] = defaultdict(
        lambda: {"working_age_2025": 0.0, "working_age_2030": 0.0}
    )

    for feature in features:
        props = feature["properties"]
        shicode = props.get("SHICODE")
        if not shicode:
            continue

        codes = [code.strip() for code in str(shicode).split("_") if code.strip()]
        if not codes:
            continue

        share = 1.0 / len(codes)
        pop_2025 = float(props.get("PTB_2025") or 0.0)
        pop_2030 = float(props.get("PTB_2030") or 0.0)

        for code in codes:
            if code not in target_codes:
                continue
            totals[code]["working_age_2025"] += pop_2025 * share
            totals[code]["working_age_2030"] += pop_2030 * share

    municipality_population = pd.DataFrame(
        [
            {
                "municipality_code": code,
                "working_age_2025": values["working_age_2025"],
                "working_age_2030": values["working_age_2030"],
            }
            for code, values in totals.items()
        ]
    )

    municipality_population["working_age_change"] = (
        municipality_population["working_age_2030"]
        - municipality_population["working_age_2025"]
    )
    municipality_population["working_age_change_pct"] = np.where(
        municipality_population["working_age_2025"].ne(0),
        municipality_population["working_age_change"]
        / municipality_population["working_age_2025"]
        * 100.0,
        np.nan,
    )
    return municipality_population

def build_property_result() -> pd.DataFrame:
    properties = load_properties()
    admin = load_admin_mapping()
    target_codes = set(admin["municipality_code"])
    municipality_population = build_municipality_population_frame(target_codes)

    municipality_result = admin.merge(
        municipality_population, on="municipality_code", how="left"
    )

    property_result = properties.merge(
        municipality_result,
        on=["prefecture_name", "municipality_name"],
        how="left",
    )
    return property_result.sort_values(["reit_name", "property_id"]).reset_index(drop=True)

def build_reit_summary(property_result: pd.DataFrame) -> pd.DataFrame:
    summary = (
        property_result.groupby("reit_name")
        .agg(
            property_count=("property_id", "count"),
            municipality_count=("municipality_name", "nunique"),
            working_age_change_mean=("working_age_change", "mean"),
            working_age_change_median=("working_age_change", "median"),
            working_age_change_min=("working_age_change", "min"),
            working_age_change_max=("working_age_change", "max"),
            working_age_change_pct_mean=("working_age_change_pct", "mean"),
            working_age_change_pct_median=("working_age_change_pct", "median"),
        )
        .reset_index()
    )
    return summary.sort_values("reit_name").reset_index(drop=True)

def format_reit_label(reit_name: str) -> str:
    if reit_name.endswith("社") and len(reit_name) > 1:
        return f"REIT {reit_name[:-1]}"
    return f"REIT {reit_name}"

def plot_histograms(property_result: pd.DataFrame) -> None:
    configure_font()

    reits = property_result["reit_name"].dropna().sort_values().unique().tolist()
    if not reits:
        raise ValueError("No REIT data available to plot.")

    abs_values = property_result["working_age_change"].dropna()
    pct_values = property_result["working_age_change_pct"].dropna()
    abs_bins = np.histogram_bin_edges(abs_values, bins="auto")
    pct_bins = np.histogram_bin_edges(pct_values, bins="auto")

    fig, axes = plt.subplots(
        nrows=len(reits),
        ncols=2,
        figsize=(14, 4.5 * len(reits)),
        sharex="col",
        constrained_layout=True,
    )

    if len(reits) == 1:
        axes = np.array([axes])

    colors = ["#0b6e4f", "#c84c09", "#355070", "#6d597a", "#7f5539"]

    for idx, reit_name in enumerate(reits):
        subset = property_result[property_result["reit_name"] == reit_name]
        color = colors[idx % len(colors)]
        plot_label = format_reit_label(reit_name)

        ax_abs = axes[idx, 0]
        ax_abs.hist(
            subset["working_age_change"].dropna(),
            bins=abs_bins,
            color=color,
            alpha=0.85,
            edgecolor="white",
        )
        ax_abs.axvline(0, color="black", linestyle="--", linewidth=1)
        ax_abs.set_title(f"{plot_label}: 2025-2030 working-age population change")
        ax_abs.set_xlabel("Change in PTB population")
        ax_abs.set_ylabel("Property count")

        ax_pct = axes[idx, 1]
        ax_pct.hist(
            subset["working_age_change_pct"].dropna(),
            bins=pct_bins,
            color=color,
            alpha=0.85,
            edgecolor="white",
        )
        ax_pct.axvline(0, color="black", linestyle="--", linewidth=1)
        ax_pct.set_title(f"{plot_label}: 2025-2030 working-age population change rate")
        ax_pct.set_xlabel("Change rate (%)")
        ax_pct.set_ylabel("Property count")

    fig.suptitle("J-REIT property exposure to local working-age population change", fontsize=16)
    fig.savefig(FIGURE_OUTPUT_PATH, dpi=180, bbox_inches="tight")
    plt.close(fig)

def write_outputs(property_result: pd.DataFrame, reit_summary: pd.DataFrame) -> None:
    IMAGES_DIR.mkdir(parents=True, exist_ok=True)
    property_result.to_csv(PROPERTY_OUTPUT_PATH, index=False, encoding="utf-8-sig")
    reit_summary.to_csv(REIT_OUTPUT_PATH, index=False, encoding="utf-8-sig")

def main() -> None:
    property_result = build_property_result()
    if property_result[
        ["municipality_code", "working_age_2025", "working_age_2030", "working_age_change"]
    ].isna().any().any():
        missing = property_result[property_result["working_age_change"].isna()][
            ["property_id", "municipality_name"]
        ]
        raise ValueError(f"Population data missing for some properties:\n{missing}")

    reit_summary = build_reit_summary(property_result)
    write_outputs(property_result, reit_summary)
    plot_histograms(property_result)

    display_cols = [
        "reit_name",
        "property_count",
        "municipality_count",
        "working_age_change_mean",
        "working_age_change_median",
        "working_age_change_pct_mean",
    ]
    print("Saved:", PROPERTY_OUTPUT_PATH)
    print("Saved:", REIT_OUTPUT_PATH)
    print("Saved:", FIGURE_OUTPUT_PATH)
    print()
    print(reit_summary[display_cols].to_string(index=False, float_format=lambda x: f"{x:,.2f}"))

if __name__ == "__main__":
    main()

def load_municipality_master() -> pd.DataFrame:
    admin = pd.read_excel(ADMIN_XLSX, header=2)
    admin = admin.rename(
        columns={
            "行政区域コード": "shicode",
            "都道府県名\n（漢字）": "prefecture_name",

def load_mesh_population() -> pd.DataFrame:
    with MESH_GEOJSON.open(encoding="utf-8") as fp:
        geojson = json.load(fp)

    rows = []
    for feature in geojson["features"]:
        props = feature["properties"]
        rows.append(
            {
                "shicode": str(props["SHICODE"]).zfill(5),
                "working_age_pop_2025": float(props["PTB_2025"] or 0.0),
                "working_age_pop_2030": float(props["PTB_2030"] or 0.0),
            }
        )

    mesh = pd.DataFrame(rows)
    municipality = (
        mesh.groupby("shicode", as_index=False)[["working_age_pop_2025", "working_age_pop_2030"]]
        .sum()
        .sort_values("shicode")
    )
    municipality["working_age_pop_change"] = (
        municipality["working_age_pop_2030"] - municipality["working_age_pop_2025"]
    )
    municipality["working_age_pop_change_pct"] = (
        municipality["working_age_pop_change"] / municipality["working_age_pop_2025"] * 100.0
    )
    return municipality

def build_property_dataset() -> pd.DataFrame:
    properties = pd.read_csv(PROPERTY_CSV)
    municipality_master = load_municipality_master()
    municipality_population = load_mesh_population()

    municipality_metrics = municipality_master.merge(
        municipality_population, on="shicode", how="inner"
    )

    property_metrics = properties.merge(
        municipality_metrics[
            [
                "municipality_name",
                "shicode",
                "working_age_pop_2025",
                "working_age_pop_2030",
                "working_age_pop_change",
                "working_age_pop_change_pct",
            ]
        ],
        on="municipality_name",
        how="left",
        validate="many_to_one",
    )

    missing = property_metrics[property_metrics["shicode"].isna()]["municipality_name"].unique().tolist()
    if missing:
        raise ValueError(f"Municipality join failed for: {missing}")

    property_metrics["working_age_pop_change_direction"] = property_metrics[
        "working_age_pop_change"
    ].apply(lambda value: "increase" if value > 0 else "decrease" if value < 0 else "flat")
    return property_metrics

def build_reit_summary(property_metrics: pd.DataFrame) -> pd.DataFrame:
    grouped = property_metrics.groupby("reit_name")["working_age_pop_change_pct"]
    summary = grouped.agg(["count", "min", "median", "mean", "max"]).reset_index()
    quantiles = grouped.quantile([0.25, 0.75]).unstack().reset_index()
    quantiles = quantiles.rename(columns={0.25: "p25", 0.75: "p75"})

    abs_summary = (
        property_metrics.groupby("reit_name")["working_age_pop_change"]
        .agg(["median", "mean"])
        .reset_index()
        .rename(
            columns={
                "median": "absolute_change_median",
                "mean": "absolute_change_mean",
            }
        )
    )

    summary = summary.merge(quantiles, on="reit_name", how="left").merge(
        abs_summary, on="reit_name", how="left"
    )
    return summary.sort_values("reit_name")

def make_ascii_reit_label(reit_name: str) -> str:
    stripped = reit_name.replace("社", "").strip()
    return f"REIT {stripped}" if stripped else "REIT"

def plot_histograms(property_metrics: pd.DataFrame) -> None:
    sns.set_theme(style="whitegrid")

    reit_names = sorted(property_metrics["reit_name"].unique())
    bins = 10
    all_values = property_metrics["working_age_pop_change_pct"]
    x_min = all_values.min()
    x_max = all_values.max()

    fig, axes = plt.subplots(1, len(reit_names), figsize=(7 * len(reit_names), 5), sharex=True, sharey=True)
    if len(reit_names) == 1:
        axes = [axes]

    for ax, reit_name in zip(axes, reit_names):
        subset = property_metrics[property_metrics["reit_name"] == reit_name]
        reit_label = make_ascii_reit_label(reit_name)
        sns.histplot(
            subset,
            x="working_age_pop_change_pct",
            bins=bins,
            binrange=(x_min, x_max),
            color="#2A6F97",
            edgecolor="white",
            alpha=0.9,
            ax=ax,
        )
        median_value = subset["working_age_pop_change_pct"].median()
        mean_value = subset["working_age_pop_change_pct"].mean()
        ax.axvline(median_value, color="#C1121F", linestyle="--", linewidth=2, label=f"Median {median_value:.1f}%")
        ax.axvline(mean_value, color="#F4A261", linestyle=":", linewidth=2, label=f"Mean {mean_value:.1f}%")
        ax.set_title(f"{reit_label} ({len(subset)} properties)")
        ax.set_xlabel("Working-age population change 2025-2030 (%)")
        ax.set_ylabel("Property count")
        ax.legend(frameon=False)

    fig.suptitle("Distribution of municipality-level working-age population change by REIT", y=1.02)
    fig.tight_layout()
    fig.savefig(PLOT_OUTPUT, dpi=200, bbox_inches="tight")
    plt.close(fig)

def main() -> None:

if __name__ == "__main__":

最終的に、ヒストグラムを描写したグラフが保存されているという目標は、OKFの有無にかかわらず達成されていましたが、出来上がったヒストグラム（とその裏側にもっている数値結果）はやや異なるものとなりました。この違いについても少し見てみたいと思います。

まずはトークン消費を確認してみます。

* OKF有り：トータル=30.7K（Input=20.8K,Output=9.98K）
* OKF無し：トータル=38K（Input=28.3K,Output=9.72K）

となり、従って**トータルで約19.2％、Inputで見ると約26.5%の削減**となっています。これはOKFがあることにより、AIエージェントがデータを確認するなどの手間が減ったことに起因するものと考えられ、意図した結果になっていると言えるかと思います。

次に結果の差異についてですが、スクリプトの違いを生成AIに見てもらうと、ヒストグラムを描写する（＝グラフの体裁を整える）部分の違いに加えて、根本的に計算ロジックが違うため、出来上がりの数値やヒストグラムが異なっていることが分かりました。

具体的には、

> 数値差の主因はSHICODEが複数自治体を\_連結しているメッシュの扱いで、SHICODEを\_で分解し、複数自治体にまたがるメッシュ人口を均等配分しています。比較対象の実装は、SHICODEを文字列のまま1つのコードとして集計しています。13101\_13102のような行は単独の擬似コードとしてgroupbyされ、5桁コードの自治体マスタと結合されないため、その人口が自治体集計から落ちます。

とのことで、OKFが無い場合にAIエージェントは、GeoJSONのSHICODEが特殊であったことを見落としていたことが原因として考えられます。思考実験ではありますが、仮にこれを発見出来ていた場合は、データを読み込んで色々と確認する部分で更にトークンが消費されることが想定されるため、いずれにせよOKFのトークン削減の有効性が示唆される結果になるかと思います。

### 余談

色々なパターンで実験をしていたのですが、REITのデータセットで緯度経度も与えた状況で同じタスクを依頼してみました。すると、行政区域コードではなくGeoJSONに入っている各メッシュのポリゴン座標から直接緯度経度を割り当てて、各保有物件に将来推計人口を紐づけるという別ルートを生成AIが自ら見つけてくるという実装をしてきました。

出力されたスクリプト

```
# csvファイルに緯度経度が紐づいている想定であることに注意
import json
from math import ceil
from pathlib import Path

import matplotlib.pyplot as plt
import pandas as pd

BASE_DIR = Path(__file__).resolve().parents[1]
PROPERTIES_CSV = BASE_DIR / "rawdata" / "jreit_properties_dummy_tokyo.csv"
MESH_GEOJSON = BASE_DIR / "rawdata" / "1km_mesh_2024_13.geojson"
IMAGES_DIR = BASE_DIR / "images"
PROPERTY_OUTPUT = IMAGES_DIR / "reit_property_population_change_2030.csv"
SUMMARY_OUTPUT = IMAGES_DIR / "reit_population_change_summary_2030.csv"
FIGURE_OUTPUT = IMAGES_DIR / "reit_population_2030_histograms.png"

def load_mesh_boxes(path: Path) -> list[dict]:
    with path.open("r", encoding="utf-8") as f:
        geojson = json.load(f)

    mesh_boxes = []
    for feature in geojson["features"]:
        ring = feature["geometry"]["coordinates"][0]
        xs = [point[0] for point in ring]
        ys = [point[1] for point in ring]
        props = feature["properties"]
        mesh_boxes.append(
            {
                "mesh_id": props["MESH_ID"],
                "municipality_code": str(props["SHICODE"]),
                "min_lon": min(xs),
                "max_lon": max(xs),
                "min_lat": min(ys),
                "max_lat": max(ys),
                "center_lon": (min(xs) + max(xs)) / 2,
                "center_lat": (min(ys) + max(ys)) / 2,
                "population_2025": float(props["PTN_2025"]),
                "population_2030": float(props["PTN_2030"]),
            }
        )

    return mesh_boxes

def find_mesh_for_point(latitude: float, longitude: float, mesh_boxes: list[dict]) -> dict:
    matches = []
    for mesh in mesh_boxes:
        if (
            mesh["min_lon"] <= longitude <= mesh["max_lon"]
            and mesh["min_lat"] <= latitude <= mesh["max_lat"]
        ):
            matches.append(mesh)

    if not matches:
        raise ValueError(f"No mesh found for point lat={latitude}, lon={longitude}")

    if len(matches) == 1:
        return matches[0]

    return min(
        matches,
        key=lambda mesh: abs(mesh["center_lat"] - latitude) + abs(mesh["center_lon"] - longitude),
    )

def build_property_analysis(properties: pd.DataFrame, mesh_boxes: list[dict]) -> pd.DataFrame:
    rows = []
    for record in properties.to_dict(orient="records"):
        mesh = find_mesh_for_point(record["latitude"], record["longitude"], mesh_boxes)
        population_2025 = mesh["population_2025"]
        population_2030 = mesh["population_2030"]
        population_change = population_2030 - population_2025
        population_change_pct = (population_change / population_2025 * 100.0) if population_2025 else None

        rows.append(
            {
                **record,
                "mesh_id": mesh["mesh_id"],
                "mesh_municipality_code": mesh["municipality_code"],
                "population_2025": population_2025,
                "population_2030": population_2030,
                "population_change_to_2030": population_change,
                "population_change_pct_to_2030": population_change_pct,
            }
        )

    return pd.DataFrame(rows)

def build_reit_summary(property_analysis: pd.DataFrame) -> pd.DataFrame:
    summary = (
        property_analysis.groupby("reit_name", as_index=False)
        .agg(
            property_count=("property_id", "count"),
            population_change_mean=("population_change_to_2030", "mean"),
            population_change_median=("population_change_to_2030", "median"),
            population_change_min=("population_change_to_2030", "min"),
            population_change_max=("population_change_to_2030", "max"),
            population_change_pct_mean=("population_change_pct_to_2030", "mean"),
            population_change_pct_median=("population_change_pct_to_2030", "median"),
            population_change_pct_min=("population_change_pct_to_2030", "min"),
            population_change_pct_max=("population_change_pct_to_2030", "max"),
        )
        .sort_values("reit_name")
    )

    positive_counts = (
        property_analysis.assign(is_positive=property_analysis["population_change_to_2030"] > 0)
        .groupby("reit_name")["is_positive"]
        .sum()
        .rename("positive_property_count")
        .reset_index()
    )
    negative_counts = (
        property_analysis.assign(is_negative=property_analysis["population_change_to_2030"] < 0)
        .groupby("reit_name")["is_negative"]
        .sum()
        .rename("negative_property_count")
        .reset_index()
    )

    summary = summary.merge(positive_counts, on="reit_name").merge(negative_counts, on="reit_name")
    return summary

def display_reit_label(reit_name: str) -> str:
    ascii_name = "".join(ch for ch in reit_name if ch.isascii() and not ch.isspace())
    return ascii_name or reit_name

def plot_histograms(property_analysis: pd.DataFrame, output_path: Path) -> None:
    reit_names = sorted(property_analysis["reit_name"].unique())
    cols = min(2, len(reit_names))
    rows = ceil(len(reit_names) / cols)
    fig, axes = plt.subplots(rows, cols, figsize=(7 * cols, 4.8 * rows), sharex=True, sharey=True)
    if not isinstance(axes, (list, tuple)):
        axes = [axes] if not hasattr(axes, "flat") else list(axes.flat)
    else:
        axes = list(axes)

    change_pct = property_analysis["population_change_pct_to_2030"]
    bins = 12

    for ax, reit_name in zip(axes, reit_names):
        subset = property_analysis[property_analysis["reit_name"] == reit_name]
        ax.hist(
            subset["population_change_pct_to_2030"],
            bins=bins,
            color="#4C78A8",
            edgecolor="white",
            alpha=0.9,
        )
        ax.axvline(0, color="#D62728", linestyle="--", linewidth=1)
        ax.set_title(f"REIT {display_reit_label(reit_name)}")
        ax.set_xlabel("Population change to 2030 (%)")
        ax.set_ylabel("Property count")
        ax.set_xlim(change_pct.min() - 1, change_pct.max() + 1)

    for ax in axes[len(reit_names) :]:
        ax.remove()

    fig.suptitle("Projected local population change by REIT (2025 to 2030)", fontsize=14)
    fig.tight_layout()
    fig.savefig(output_path, dpi=200, bbox_inches="tight")
    plt.close(fig)

def main() -> None:
    IMAGES_DIR.mkdir(parents=True, exist_ok=True)

    properties = pd.read_csv(PROPERTIES_CSV)
    mesh_boxes = load_mesh_boxes(MESH_GEOJSON)
    property_analysis = build_property_analysis(properties, mesh_boxes)
    reit_summary = build_reit_summary(property_analysis)

    property_analysis.to_csv(PROPERTY_OUTPUT, index=False)
    reit_summary.to_csv(SUMMARY_OUTPUT, index=False)
    plot_histograms(property_analysis, FIGURE_OUTPUT)

    print(f"Saved property analysis: {PROPERTY_OUTPUT}")
    print(f"Saved REIT summary: {SUMMARY_OUTPUT}")
    print(f"Saved histogram figure: {FIGURE_OUTPUT}")

if __name__ == "__main__":
    main()
```

このように、特定ドメインに対して一定の事前知識を持っている場合であれば、ナレッジを与えずともタスクが解けてしまうこともあり、結果的にはOKFの有無で分析の正確性やコストに差が出ない場合もありそうです。
