いきなり数字から入る。

同じ商品（AI議事録ツール）のLPキャッチコピーを、2つのプロンプトでClaudeに書かせた。Aは「キャッチコピーを10案作って」とだけ頼んだもの。Bはターゲットと痛みとフレームワークを指定したもの。

Aの代表例：「議事録作成を、もっとスマートに。」  
Bの代表例：「会議が終わった瞬間、議事録はもう共有されている。」

AはどれもAで書ける。Bはそのチームの状況を知っている人間にしか書けない。出力された文字は、入れたプロンプトの解像度ぴったりに比例する。

つまり「AIのコピーがつまらない」のはAIのせいじゃない。プロンプトがAだったからだ。この記事は、Bを安定して出すための話をする。

---

## なぜあなたのAIコピーは「あたりさわりがない」のか

正直に言う。私も最初は「コピー10案出して」で済むと思っていた。返ってきたのは、どの商品にも貼れる無難な文章だった。

理由ははっきりしている。AIは情報の欠けた部分を「最大公約数」で埋める。誰に向けるか書かなければ、全員に向けた=誰にも刺さらない文章を出す。これは性能の問題ではなく、入力の問題だ。

刺さらないコピーには共通点が4つある。

逆に言えば、この4つをプロンプトに埋めるだけで出力は別物になる。順に渡していく。

---

## まず「ターゲットの解像度」をプロンプトに移植する

コピーの良し悪しの8割は、生成前のこのテンプレートで決まる。私が毎回使っている骨組みがこれだ。

```
あなたは [ターゲット] の心に刺さるコピーを書くプロのコピーライターです。

商品/サービス: [名前]
ターゲット: [具体的な人物像]
  - 年齢・職業: 例：30代の営業マネージャー
  - 悩み・課題: 例：チームの目標未達が続いている
  - ゴール: 例：今期の目標を120%達成したい

USP（他社と違う最大の強み）: 例：AIが商談前に勝率を予測して対策を提示
トーン: 例：親しみやすく・背中を押すような
コピーの種類: 例：LPファーストビューのキャッチコピー
文字数: 例：20〜30文字
生成する量: 10案
```

「悩み」を一行で済ませないこと。ここが浅いと、出力も浅くなる。「目標未達」ではなく「毎週の1on1で部下に詰められる側になっている」くらいまで書く。AIはあなたが見ている解像度しか再現できない。

## フレームワークを指定して「型」で書かせる

人間のコピーライターは型を持っている。AIにも型を渡す。代表的なのは2つだ。

### AIDA（注意→興味→欲求→行動）

```
AIDA フレームワークで [商品名] のメルマガ本文を書いてください。
注意（A）: 衝撃的な統計か事実で始める
興味（I）: なぜそうなるか、共感できる問題提起
欲求（D）: この商品で解決したビフォーアフター
行動（A）: 緊急感のあるCTA
ターゲット: [詳細]　文字数: 400〜500文字
```

### PAS（問題→煽り→解決）

```
PAS フレームワークで LP のボディコピーを書いてください。
問題（P）: 毎月の請求書処理に10時間かかっている
煽り（A）: その時間が年間でいくらの人件費になるか計算で示す
解決（S）: 請求書処理を5分に短縮する方法
商品: [商品名・機能]
```

PASの「煽り」を数字で示させるのがコツだ。「損しますよ」では弱い。「年120時間=人件費にして約36万円」とAIに計算させると、急に現実味が出る。

## 実際に効いた使用例：時間と数字を必ず入れさせる

LPキャッチを生成するときの、私の本番プロンプト。

```
LPのキャッチコピーを10案生成してください。
商品: AI議事録作成ツール「MeetMind」
ターゲット: 会議が多い35〜45歳のマネージャー
  - 悩み: 議事録作成に20〜30分かかり、他の仕事が遅れる
  - 欲求: 会議後すぐ共有して、自分は次の業務に集中したい
形式:
  - キャッチコピー（20文字以内）: 10案
  - サブコピー（40〜60文字）: 3案
特に：
  ✅ 具体的な時間・数字を使う
  ✅ 「議事録」「AI」などの機能ではなく「ベネフィット」を前面に
  ❌ 「最新技術」「高精度」などの陳腐な表現は禁止
```

最後の ❌ の禁止リストが効く。AIは放っておくと「高精度」「最新」に逃げる。明示的に禁止すると、ベネフィット側に振り切ってくれる。

## ABテスト用のバリエーションを一気に作らせる

1案を磨くより、観点を変えた複数案を作って試す。AIが最も得意な領域だ。

```
以下のコピーのABテスト用バリエーションを5案。
元: 「議事録を自動化。会議後10秒で共有できます」
A: 数字を変える　B: 痛みフォーカス　C: 社会的証明
D: 緊急感を強める　E: 質問形式
各案に「なぜこの表現を選んだか」も添えてください。
```

「なぜ選んだか」を書かせると、自分のABテストの仮説が言語化される。これが地味に効く。

ここまでが考え方だ。ターゲットの解像度を移植し、型で書かせ、禁止語で逃げ道を塞ぎ、量で勝負する。この4つで、冒頭のBは安定して出せる。

ここから先では、本番で「売れるコピーを量産する」ための完成プロンプトを全文公開する。1500字のセールスレターを構成ごと吐かせるテンプレート、競合の名前を出さずに比較優位を訴求するプロンプト、14日間で成果を体験させるオンボーディングメール5通のシリーズ──全部コピペで動く形で渡す。
