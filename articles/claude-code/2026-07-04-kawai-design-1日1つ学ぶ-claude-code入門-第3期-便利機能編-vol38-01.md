＼ 1日1つ学ぶ ／
Claude Code入門
第3期 便利機能編【 Vol.38 】

`--safe-mode` は
不調時の「最低限起動」です。

Claude Codeは便利ですが、
設定を増やすほど原因不明の
不調が起きることがあります。

・Hooks
・MCP
・Skills
・プラグイン
・カスタム設定

どれが原因か分からない時、
全部入りで確認すると迷います。

そこで使えるのが
`claude --safe-mode` です。

イメージは、家電の
セーフモード起動。

余計な設定を外して、
まず本体が動くか確認します。

使いどころはこの3つです。

✅ 起動がおかしい
設定由来か切り分ける

✅ MCPが怪しい
外部連携なしで見る

✅ Hooksが怪しい
自動処理を外して見る

Vol.38はここだけ覚えればOKです。

Claude Codeが不調な時は、
追加設定を疑う前にsafe-modeで切り分ける。
