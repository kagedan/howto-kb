Claude Codeに指示したのに全然守ってくれない人、書き方が甘いだけ。
「〇〇してください」だと普通に無視される。
これ書き方を変えるだけでほぼ100%守らせることができます。
貼るだけでClaude Codeを自動強化するプロンプトも置いてるので保存推奨↓

Claude Codeが指示を無視する原因はほぼこの4つ。

①表現が弱い
→「〜しないでください」は優先度が低く扱われる。
「厳禁」「絶対にするな」まで書くと守ってくれます。

②解釈がブレてる
→「簡潔に書け」だけだと基準が曖昧。
✅「1文40字以内」❌「80字以上の文」
のようにOK例/NG例を並べると精度が跳ね上がります。

③理由がない
→AIは理由があると優先度を上げる。
「なぜダメか」を1行添えるだけで遵守率が変わります。

④全部同じ重みで書いてる
→全ルール同じ扱いだと全部薄まる。
「最重要」「必須」「推奨」に分けると大事なものほど守られます。

この4つを全部自動で直すプロンプトは↓

Claude Codeに貼るだけで使えます↓

-------
以下の手順でCLAUDE.mdを強化しろ。

①現在のCLAUDE.mdを読み込め
②以下の基準で「弱いルール」を全て検出しろ
- 「〜してください」「〜しないでください」など曖昧な表現
- OK例/NG例がセットになっていないルール
- 理由が書かれていないルール
- 優先度（最重要/必須/推奨）が明記されていないルール

③検出したルールを以下の形式に書き換えろ
- 「〜してください」→「厳禁：〜するな。違反時は即座に修正しろ」
- 全ルールにOK例とNG例を追加
- 全ルールに「理由：〜」を追加
- 各ルールに優先度を付与（最重要/必須/推奨の3段階）

④書き換え前と書き換え後を並べて出力しろ
-------
