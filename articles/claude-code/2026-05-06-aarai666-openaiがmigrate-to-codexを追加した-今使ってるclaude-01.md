OpenAIが「Migrate to Codex」を追加した

今使ってる、Claude CodeやCursorで使っていた設定を、Codexへ移行できる機能

instructions、settings、MCP、hooks、skills、subagents、直近30日の会話まで対象になってる https://t.co/3w97LIrE1W

操作手順：

1. Codex AppのSettingsを開く
2. Generalページで「Import other agent setup」を探す
3. 「Import」または「Import again」をクリック
4. Codexが検出した内容を確認し、取り込みたいプロジェクトを選択
5. インポート完了後、「View imported files」から結果を確認できる https://t.co/ak4MogUv3a

例えば、今までCursorやClaude Codeで育ててきた、

・プロジェクトごとの指示
・コードの書き方ルール
・使っているMCP連携
・hooksやskills
・subagents
・直近30日間の会話履歴

などを、Codex側にまとめて移せる。

たとえば、Cursorで
「このプロジェクトは日本語で説明して」
「このファイルは勝手に触らないで」
「Reactはこのルールで書いて」
みたいな設定をしていた場合、それをCodexでまた一から作り直す必要が減る。

つまり、単なるファイル移行ではなく、
「自分専用に育てたAI開発環境」をCodexへ持っていける機能。

これが重要なのは、AIコーディングツールが単なるチャットではなく、
開発ルール、外部ツール連携、作業履歴まで含めた“個人専用の開発OS”になり始めているから
