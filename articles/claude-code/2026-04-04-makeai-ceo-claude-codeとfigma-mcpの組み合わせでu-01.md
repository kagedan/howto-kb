Claude CodeでUI作ると毎回同じような汎用レイアウトが出てくる問題を解決。Figma MCP、AIDesigner MCP、Design Skills、DESIGN mdの4つを組み合わせると、自分のデザインシステムにぴったり合ったUIをClaude Codeが生成できる。Figma公式が双方向連携を出した（claude plugin install figma@claude-plugins-official）。FigmaのデザイントークンもコンポーネントもAuto LayoutもClaude Codeに渡せる。
