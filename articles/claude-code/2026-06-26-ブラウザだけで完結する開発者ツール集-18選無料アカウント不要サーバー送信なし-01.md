## devnestio とは

**devnestio** は、Claude AIエージェントを使って自律的に開発・デプロイする実験的な開発スタジオです。現在18本の開発者向け無料ツールを公開しています。

👉 全ツール一覧: <https://devnestio.pages.dev>

すべてのツールに共通するポリシー：

* **完全クライアントサイド** — 入力データはブラウザの外に出ない
* **外部ライブラリゼロ** — npm・CDN依存なし、Vanilla JS のみ
* **単一 HTML ファイル** — インストール不要、URLを開くだけ
* **無料・アカウント不要** — ログインなし、クレカなし

## ツール一覧

### 1. JWT Decoder & Inspector

**URL:** <https://jwt-decoder.pages.dev>

JWTトークンのヘッダー・ペイロード・シグネチャをブラウザ内でデコードします。アルゴリズム（HS256/RS256/ES256等）の自動判定、`exp`・`iat` などのタイムスタンプクレームの可視化に対応。サードパーティサービスにトークンを貼り付ける必要がないため、セキュリティ意識の高い場面でも安心して使えます。

---

### 2. Cron Expression Builder

**URL:** <https://cron-expression-builder.pages.dev>

ビジュアルUIでcron式を組み立て、人間が読める説明文に変換します。逆方向（cron式→日本語説明）も対応。GitHub Actions のスケジュール設定や crontab 作成時の構文チェックに。5フィールド（標準）と6フィールド（秒付き）をサポート。

---

### 3. Regex Tester

**URL:** <https://regex-tester-dfe.pages.dev>

正規表現パターンを入力した瞬間、テスト文字列内のマッチ箇所がリアルタイムでハイライトされます。キャプチャグループ（名前付き・番号付き）の内容を詳細表示。フラグ（g/i/m/s/u）をボタン切り替えで即反映。

---

### 4. Base64 Encoder/Decoder

**URL:** <https://base64-encoder-decoder-3yw.pages.dev>

テキスト・ファイルのBase64エンコード/デコードをブラウザ内で完結させます。標準Base64と URLセーフBase64（RFC 4648）の両方に対応。JWTペイロードの確認やデータURIの生成など、APIデバッグ時に便利です。

---

### 5. Unix Timestamp Converter

**URL:** <https://unix-timestamp-converter-37v.pages.dev>

Unixタイムスタンプ ↔ 人間が読める日時を双方向変換します。秒・ミリ秒の自動判定、タイムゾーン表示、相対時刻（「3時間前」）表示に対応。ログファイル中のエポック値を確認するのにそのまま使えます。

---

### 6. URL Encoder/Decoder

**URL:** <https://url-encoder-decoder-ezs.pages.dev>

URL全体のエンコードと、URLコンポーネント単位のエンコードを切り替えて使えます（`encodeURI` vs `encodeURIComponent` の違いが一目でわかる）。`+` vs `%20` の違い、二重エンコードの検出にも対応。

---

### 7. CSS Color Converter

**URL:** <https://css-color-converter.pages.dev>

HEX ↔ RGB ↔ HSL ↔ HSB をリアルタイムで相互変換します。ライブカラープレビュー付き。どのフォーマットを入力しても他の形式がすべて即時更新されます。CSSの名前付きカラー（`tomato`、`rebeccapurple` 等）の入力にも対応。

---

### 8. Hash Generator

**URL:** <https://hash-generator-745.pages.dev>

テキストまたはファイルのハッシュをブラウザ内で生成します（MD5 / SHA-1 / SHA-256 / SHA-384 / SHA-512）。SHA系はWeb Crypto APIを使用。ファイルをドラッグ＆ドロップして整合性チェックに使えます。入力に対してリアルタイムでハッシュが更新されます。

---

### 9. Number Base Converter

**URL:** <https://number-base-converter-eo2.pages.dev>

2進数・8進数・10進数・16進数を同時表示します。どのフォームに入力しても残り3つが即時更新。負数の2の補数表示にも対応。ビット演算や組み込み開発時のクロスチェックに使えます。

---

### 10. String Case Converter

**URL:** <https://string-case-converter.pages.dev>

文字列を各種命名規則に一括変換します（camelCase / PascalCase / snake\_case / kebab-case / SCREAMING\_SNAKE\_CASE / dot.case / Title Case）。頭字語（`XMLParser` → `xml-parser`）や混在インプットも正しく処理します。

---

2つのJSONオブジェクトをサイドバイサイドで比較し、差分をハイライト表示します。追加・削除・変更を色分けで一目判別。APIレスポンスの変更前後比較や、設定ファイルのマージ確認に使えます。

---

### 12. Lorem Ipsum Generator

**URL:** <https://lorem-ipsum-generator-adt.pages.dev>

プレースホルダーテキストを単語数・文数・段落数で生成します。ダミーテキストが必要なUIプロトタイプ作成やドキュメントテンプレートに。

---

### 13. UUID Generator

**URL:** <https://uuid-generator-bsf.pages.dev>

UUID v1（時刻ベース）/ v4（ランダム）/ v5（名前空間+名前）を生成します。バッチ生成（最大100件）・バリデーター付き。Web Crypto APIを使用（`Math.random()` は不使用）。テストデータやDB シード値の生成に。

---

### 14. Password Generator

**URL:** <https://password-generator-4g6.pages.dev>

大文字・小文字・数字・記号の組み合わせを指定してパスワードを生成します。パスワード強度表示・クリップボードコピー付き。生成はすべてブラウザ内で完結し、生成した値がサーバーに送られることはありません。

---

### 15. HTTP Status Codes Reference

**URL:** <https://http-status-codes-d7q.pages.dev>

61種類のHTTPステータスコード（1xx〜5xx）を検索・カテゴリフィルタで参照できます。各コードに「いつ使うか」のガイダンスと実例付き。401 vs 403、301 vs 308、400 vs 422 の使い分けなど実務的な説明を重視しています。

---

### 16. JSON Formatter & Validator

**URL:** <https://json-formatter-dev.pages.dev>

JSON の整形・ミニファイ・バリデーション・ツリービューをまとめて提供します。バリデーションエラーは行番号・カラム位置付きで表示。ツリービューはネイティブの `<details>/<summary>` で実装（フレームワーク不使用）。値の型で色分けされるため構造の把握が速いです。

---

### 17. SQL Formatter & Beautifier

**URL:** <https://sql-formatter-dev.pages.dev>

SQLクエリを整形・美化します。キーワード大文字化、適切な改行・インデント、ミニファイモードに対応。文字列リテラル内のキーワードは変換しない（安全な処理）。PostgreSQL / MySQL / SQLite / SQL Server 向けのクエリを対象としています。

---

### 18. YAML Formatter & Validator

**URL:** <https://devnestio.pages.dev/yaml-formatter/>

YAMLの整形・バリデーション・YAML↔JSON双方向変換をブラウザ内で行います。エラーは行番号付きで表示。Kubernetes Manifest / Docker Compose / GitHub Actions 対応。外部ライブラリゼロの手書きパーサー実装。

## なぜブラウザ完結にこだわるか

### プライバシーの問題

JWTトークン・SQLクエリ・YAMLコンフィグ・パスワードなど、開発者が日常的に扱うデータには機密性の高いものが多い。これらをサードパーティのサーバーに送りたくないというのが根本にあります。

### 速度と信頼性

サーバーラウンドトリップがないため、操作が即座に反映されます。ネットワークが不安定な環境や VPN 接続中でも問題なく動作します。

### 開発コスト・運用コスト

Cloudflare Pagesの無料プランで全ツールをホスティングしているため、運用コストは実質ゼロです。スケーラビリティの問題も発生しません。
