【営業チームでClaude Codeを使う社内ツール開発】

AnthropicのGTM ArchitectであるJared Sires氏は、コード経験がない状態からClaude CodeでGmail内の返信下書きアプリ「CLAFTS」を作成。共有Google Driveや外部ツール、公開ドキュメントを参照し、顧客メールへの返信案を用意する仕組み。

CLAFTSは約4.3K行のコードで構成され、ほぼすべてをClaude Codeが作成。Jared氏は週10〜15時間を削減できたとしている。

現在は営業チーム向けに、商談前の顧客調査や議事録をもとにしたフォローアップメール作成などを行うスキル（定型作業を呼び出す機能）をClaude Coworkのプラグインとして展開。Sales plugin（営業支援プラグイン）は公開から数か月で、Anthropicの営業組織の約80%が利用している。
