クレームのメールを受け取ったとき、どうしていますか。

頭では「冷静に」とわかっていても、返信文を考えているうちに消耗する。謝りすぎて変な責任を取ってしまう。逆に事務的すぎてさらに怒らせてしまう——。

もうClaude Codeに書かせてしまえばいいです。**クレーム内容を貼り付けるだけで、バランスの取れた返信文が30秒で出てきます。**

## Before / After

**Before：** クレームメールを受け取る → 返信文を考えながら消耗（30〜60分）→ 送った後も不安が残る

**After：** クレーム内容をClaude Codeに渡すだけ → バランスの取れた返信文が出てくる → **5〜10分で完了（85%削減）**

## 準備するもの

- Claude Code（インストール済みであること）   
- クレームメールの本文または概要

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：クレーム内容を整理する

まず「何が起きたか」を1〜2行でまとめます。完璧でなくていいです。

```
・商品が壊れた状態で届いた
・対応が遅かった
・説明と内容が違った
```

### ステップ2：Claude Codeに内容を渡す

```
以下のクレームに対する返信文を書いてください。

[クレームメールの本文を貼る]
```

事実確認が済んでいれば補足を添えます。

```
以下のクレームに対する返信文を書いてください。
状況：当社の梱包ミスが確認済み。
対応：交換品を3営業日以内に発送する。

[クレームメールの本文を貼る]
```

## できました。ただ、このままだと1つ問題があります。

「誠実に謝る」だけでは通じないケースがあります。

認識ズレのクレームに全面謝罪すると、こちらの責任を認めたことになる。理不尽なクレームに共感しすぎると、さらに要求がエスカレートする。対応に時間がかかるケースでは「確認中です」の伝え方ひとつで印象が変わる——クレーム対応には**状況ごとの型**が必要です。

毎回「今回は当社のミスではないので謝罪は最小限に」「対応に時間がかかるので経過連絡を入れて」とプロンプトをゼロから書き直すのは、地味に面倒で、イレギュラーなケースほど迷います。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、対応原則・謝罪の使い分け・やってはいけない表現を覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎回のクレーム対応が変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「謝罪は1回まで」と指定が必要   
✅ ファイルあり：自動で謝罪が過剰にならない文体になる

❌ 設定なし：毎回「責任範囲が不明な場合は確認中と書いて」と説明   
✅ ファイルあり：自動で「確認中」表現を使う

❌ 設定なし：毎回「対応期限を入れて」と追加   
✅ ファイルあり：常に期限付きで返ってくる

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「クレーム対応の消耗」が減ります。**

## 100円で何が変わるか

クレーム対応1件30〜60分 × 月5件 × 12ヶ月 = **年間最大60時間がクレーム対応に消えています。**

設定ファイル1枚で、来月からその時間が大幅に減ります。

**100円 ÷ 50時間削減 = 1時間あたり2円。** これより安く精神的な負担を下げる方法は、なかなかありません。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-009（添付）**　←　ダウンロードして置くだけ   
- **状況別テンプレ4種**　←　当社責任・認識ズレ・理不尽・時間がかかる対応で使い分け   
- **「謝りすぎない・責任範囲を明確にする」設定の詳細解説**

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「どんな場面でクレーム対応することが多いですか？」→ 答える   
- 「署名を教えてください」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたの対応スタイルを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
