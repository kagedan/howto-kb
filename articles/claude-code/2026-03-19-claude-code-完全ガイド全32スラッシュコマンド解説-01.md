## 会話・セッション管理（10コマンド）

### `/help`
ヘルプとコマンド一覧を表示

### `/clear`
会話履歴をクリアしてリセット

### `/compact`
会話を圧縮してコンテキストを節約

### `/context`
現在のコンテキスト（トークン）使用状況を表示

### `/cost`
セッションのトークン消費とコストを表示

### `/exit`
Claude Codeを終了（`/quit`のエイリアスあり）

### `/export`
会話履歴をファイルやクリップボードに出力

### `/rename`
セッション名
