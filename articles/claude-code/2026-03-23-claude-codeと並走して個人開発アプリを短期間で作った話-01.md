 はじめに
バイブコーディングと初めて聞いたとき、ノリノリのウキウキウェーイでコーディングすることなのかと思っていた私です。
どうやら Gemini 曰く、自然言語（普段の言葉）で AI に「こういう機能が欲しい」と伝えるだけでアプリや Web サービスの開発・修正を AI が自動で行う新しい手法のこと、なのだそうです。
これまで AI といえば主に ChatGPT ぐらいしか使っていなかったのですが、昨年下旬あたりからようやく重い腰を上げて ClaudeCode を本格的に使い始めまして。
そして迎えた年末年始。長期休暇。ノリと雰囲気で何か作ってくれ...
