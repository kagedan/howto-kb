Trading bots are easy to sketch. The hard part is keeping backtests and live execution honest.

AlgoTrader is a Claude Code skill for builders working on quantitative trading systems for Indian equity markets with Zerodha/Kite workflows.

It helps you move from idea to a safer implementation by combining a bot-generation wizard, NSE universe fetching, backtest/live checks, and a documented set of trading-system gotchas.

Key features:

• Interactive bot wizard – scans existing code and asks about trade style, universe, capital, and risk tolerance
• Zerodha/Kite workflow guidance – covers tick-size rounding, position reconciliation, stop-loss lifecycle, and credentials setup
• Live universe fetcher – creates JSON files for Nifty 50, Nifty 100, Midcap 150, and Smallcap 250 constituents
• Backtest-live parity checks – documents data caching, VWAP reset, candle completion, and slippage issues that break live results
• Risk and analytics patterns – includes Kelly sizing, portfolio heat, regime detection, structured logs, and post-trade reporting

It’s open-source (MIT license).
