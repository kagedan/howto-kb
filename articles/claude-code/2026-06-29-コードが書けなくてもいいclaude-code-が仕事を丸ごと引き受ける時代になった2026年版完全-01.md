正直に言う。

最初に「Claude Code」という名前を聞いたとき、「エンジニア向けのツールだな」と思ってスルーしていた。コード、という単語が入っているから。

でも今は違う考えを持っている。

Claude Code は、コードを書くためのツールじゃない。**仕事を自律的に進めてくれるAIエージェントだ。** コードはその手段に過ぎない。

この記事では、エンジニアにも非エンジニアにも使える Claude Code の実践的な使い方を、インストールから具体的なプロンプトまで全部まとめる。有料記事として書く以上、「読んで終わり」ではなく「今日から動ける」内容にした。

## 第1章　Claude Code とは何か――普通のClaude との決定的な違い

### チャットAIとエージェントAIの違い

普通の Claude（claude.ai）は「チャットで質問に答えてくれるAI」だ。あなたが聞いて、Claudeが答える。主導権は常に人間側にある。

Claude Code は根本的に違う。指示を出したら、自分でファイルを開き、読んで、書き換えて、確認して、また修正する。テストを走らせて、エラーが出たら原因を調べて、また直す。人間が「次は何をすればいい？」と聞かれるのを待つのではなく、**タスクを最後まで完走しようとする。**

この違いは小さいようで大きい。チャットAIは「アドバイスをくれる人」だ。エージェントAIは「実際に手を動かしてくれる人」だ。

### 「コード」という名前に騙されない

Claude Code という名前のせいで「エンジニア専用」だと思われがちだが、実際に使われているユースケースは幅広い。

議事録の自動整形、CSVデータの集計、フォルダの一括整理、メール文面の一括生成、社内ドキュメントの検索と要約。これらはすべてプログラミングの知識がなくても使える。

コードが書けない人にとって Claude Code は「コーディングツール」ではなく「日本語で指示するだけで複雑な作業を自動でこなしてくれるアシスタント」だ。

### 2026年現在の位置づけ

2025年2月にリサーチプレビューとして公開されてから、Claude Code はほぼ毎日のペースでアップデートされ続けている。2026年6月現在の最新バージョンは v2.1.161。

2026年4月のデスクトップアプリのリデザイン以降、ターミナルの知識ゼロでも使えるようになった。CLI・VS Code拡張・デスクトップアプリの3形式が揃い、「エンジニアだけのツール」という時代は実質終わっている。

## 第2章　始める前に知っておくべきこと――料金・プラン・制限

### 料金の現実

Claude Code は無料プランでは使えない。**Proプラン（月額$20、日本円で約3,000円）以上が必要だ。**

ただし、Proに入っていれば追加料金なしで使い始められる。

問題は使用量だ。Claude Code は1タスクで大量のトークンを消費する。プロジェクト全体を読み込んで、複数のファイルを書き換えて、テストを走らせる。その分だけトークンが使われる。Proプランだと1日数タスクで上限に達することがある。

本格的に使いたい人向けのプラン比較はこうなる。

プラン月額向いている人Pro$20週数回・試しに使いたい人Max 5x$100週数十時間・業務で使う人Max 20x$200毎日フル活用・開発者Team$30/人〜複数人で使う組織

APIで従量課金という選択肢もある。使った分だけ払う方式で、使用量が少ない月はProより安くなる可能性がある。ただし Claude Code は突発的にトークンを大量消費することがあるため、コスト管理が難しい。初心者にはProプランの定額制が安心だ。

### 無料で試す方法はあるか

Claude Code そのものは無料では使えないが、claude.ai のFreeプランでも「Cowork」機能を使って一部のエージェント的な作業はできる。完全な代替にはならないが、「どんな感じか雰囲気を掴みたい」人はまずそこから触ってみるのも手だ。
