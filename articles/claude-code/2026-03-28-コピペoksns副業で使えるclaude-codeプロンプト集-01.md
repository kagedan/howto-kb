「Claude Codeを使ってみたいけど、何て指示すればいいか分からない」  
「毎回プロンプトを考えるのが面倒…」

そんな悩みを持っている人は多いはずです。

結論から言うと、SNS副業は“プロンプトを固定するだけ”で一気に楽になります。  
そして一度テンプレを作ってしまえば、あとはコピペで回せます。

この記事では、SNS副業でそのまま使える「Claude Codeプロンプト集」を、目的別にまとめて解説します。

---

**なぜプロンプトが重要なのか？**

SNS副業で効率が悪い人の特徴は、

・毎回違う指示を出している  
・出力が安定しない  
・修正に時間がかかる

という点です。

つまり問題は、

「スキル」ではなく「プロンプト設計」

です。

---

**結論｜“型を固定する”だけで効率は倍になる**

Claude Codeは非常に優秀ですが、

・曖昧な指示  
・毎回違う指示

では、出力がブレます。

そこで重要なのが、

「使い回せるプロンプト」

です。
