2ヶ月間、Search Consoleを見なかった。

正確には、開いてはいた。数字が並んでいた。でも何も読めなかった。「表示回数」「クリック数」「平均順位」——数字の意味がわからないまま、毎回画面を閉じていた。

2026年5月28日、初めてGSCを「読んだ」日がある。そこで気づいたのは、「AI活用系の記事だけが動いていた」という事実だった。法人カード・補助金・転職エージェントで記事数は多いのに、クリックが来ていたのはAI記事だけだった。

---

## 数字がゼロだった最初の6週間

公開から2ヶ月。181記事を書いた。

GSCはWordPressと連携してすぐに設定したが、毎日開いていたわけではない。数字がゼロのまま3週間が過ぎた。開くのが怖かったというより、数字が変わっても意味がわからなかった。

Googleのインデックス登録は、記事公開から2〜8週間かかる。私の記事が本格的にインデックスされ始めたのは4月下旬〜5月初旬。実質的にGSCに載り始めたのは公開後6週間経ってからだった。

最初のクリックが来たのは2026年5月21日。「クリック数1」と表示された日の感想は「増えてた」だった。ほぼ無感動だった。数字の意味よりも「誰かが来た」という事実の方が先に来た。

---

## 5月28日、初めて「読む」目的でGSCを開いた

5月28日、過去3週間のデータを分析する目的でGSCを開いた。

| 指標 | 数値 |  
| 総クリック数 | 5件 |  
| 総表示回数 | 426回 |  
| 初クリック日 | 2026-05-21 |  
| 最多クリック | Claude決算書分析（2件） |

表示426回に対してクリック5件。クリック率は約1.2%。これが高いのか低いのかもわからなかった。Google検索の平均CTRは「位置1で約28%、位置10で約2%」というデータがある。私の記事は位置5〜15の帯域にある。その帯域でのCTR1.2%は、普通かやや低めという位置づけだ。

GSCのデータをどう記事改善につなげるかは、[SEO対策にAIを活用する方法](https://keiei-ai.com/seo%e5%af%be%e7%ad%96%e3%81%abai%e3%82%92%e6%b4%bb%e7%94%a8%e3%81%99%e3%82%8b%e6%96%b9%e6%b3%95%e3%80%90%e4%b8%ad%e5%b0%8f%e4%bc%81%e6%a5%ad%e3%81%8cclaude%e3%83%bbchatgpt%e3%81%a7%e3%82%ad%e3%83%bc/)で書いている。

---

## 「AI記事だけが動いていた」という発見

記事別のクリック元を見ると、すべてAI活用系だった。

| 記事テーマ | 位置 | クリック |  
| Claude決算書分析 | 7位 | 2件 |  
| AIで営業資料作成 | 5位 | 1件 |  
| ClaudeでPowerPoint | 7位 | 1件 |  
| Microsoft Copilot中小企業活用 | 15位 | 1件 |

法人カード系は54〜63位。補助金系は100位圏外が多い。転職エージェント系も同様。

181記事のうちAI活用系は10〜15本程度、全体の8%くらいだ。ところがクリックの100%がそこから来ていた。

なぜAI系が強いのか。その日考えた理由は3つある。

**競合が薄い。** 法人カード・補助金・転職エージェントは大手比較サイトや専門メディアが上位を占める。AI活用×中小企業経営者向けの記事は、まだ専門サイトが少ない。

**E-E-A-Tが立ちやすい。** 私は実際にClaude Codeで業務システムを作り、AIで決算書を分析し、プロンプトを毎日使っている。法人カードの記事を書いても「現役ユーザー」ではないが、AI活用記事なら「今日も使っている人」として書ける。Googleが評価するE-E-A-T（経験・専門性・権威性・信頼性）の「経験」が、AI系記事でだけ本物になる。

**検索意図とコンテンツが合っている。** 「Claude 決算書分析 やり方」で検索している人は、AIで決算書を分析した体験談を読みたい。私の記事はその体験談だ。一般論ではなく「自分がやってみた記録」が検索意図と一致している。

---

以下では、GSCの4指標を私がどう読み解いたか、データを見た直後に実際に何をしたか、週次でGSCを見続ける仕組みの設計を、省略なしで書く。「表示はあるのにクリックが来ない」記事のタイトルをどう修正すればいいかの判断基準も含む。
