## 今日やったこと

Claude Codeに運勢占いツールを作ってもらいました。

指示はこれだけです。

```
ターミナルで動く今日の運勢占いツールをPythonで作ってください。生年月日を入力したら、今日の運勢（総合・恋愛・仕事・金運）をスコアと一言コメントで表示してください
```

## 実際に試した結果

生年月日を入力すると今日の運勢が出ます。

```
══════════════════════════════════════════════════
  ✨  今日の運勢占い  ✨
══════════════════════════════════════════════════

  生年月日を入力してください
  例）1990/05/15  /  1990-05-15  /  19900515

  生年月日 > 20101115

  占い対象 : 2010年11月15日生まれ（15歳）
  占い日   : 2026年03月25日（Wednesday）

──────────────────────────────────────────────────

  【総合運】 ★★★☆☆  (3/5)
  　特別な出来事はないが、着実に前進できる日

  【恋愛運】 ★★☆☆☆  (2/5)
  　すれ違いに注意。気持ちを丁寧に伝えて

  【仕事運】 ★★★☆☆  (3/5)
  　コツコツ積み上げることが大切な日

  【金運】 ★★★☆☆  (3/5)
  　大きな出費はないが、節約意識を持って

──────────────────────────────────────────────────

  🍀 ラッキーアイテム : 木製のもの
  🎨 ラッキーカラー   : 銀
  🔢 ラッキーナンバー : 5

──────────────────────────────────────────────────

  💬 今日もあなたらしく、笑顔で過ごしましょう！

══════════════════════════════════════════════════

  別の生年月日で試しますか？ (y/n) >
```

何も指示していなくてもつけてくれた機能

```
機能:                                                                                                                 
  - 生年月日を入力（1990/05/15・1990-05-15・19900515 形式に対応）                                                     
  - 今日の日付 × 生年月日から再現性のあるハッシュでスコアを決定（毎回同じ結果が出る）                                   
  - 4カテゴリを★1〜5 + 一言コメントで表示                                                                             
    - 総合運 / 恋愛運 / 仕事運 / 金運                                                                                   
  - ラッキーアイテム・カラー・ナンバーも表示                                                                            
  - 繰り返し別の生年月日で占える
```

## 仕組みが気になった

同じ生年月日を入れると毎回同じ結果が出ます。でも日付が変わると結果も変わる。

コードを読んでみると、生年月日と今日の日付を組み合わせて計算しているようでした。完全にランダムではなく、「その人のその日」に固定された値が出る仕組みです。

相性診断ツール（Day.13）と同じ考え方だと気づきました。名前や生年月日をハッシュ化して数値にする。15日間毎日触っていると、こういうパターンが見えてくるようになりました。
