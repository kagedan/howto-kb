if you're an academic working with Codex/Claude Code, use a monorepo in which your code, brainstorming, and paper are comingled in a text-first format (i.e. LaTeX/markdown, not Word)

this will give your agentic coding tool the context to be a lot smarter on both your code and your paper

something like this:

README.md
./docs/brainstorm
./docs/plan
./docs/paper
./src

literally just copy the text of this tweet, tell it about your project, and ask it to suggest a structure
