## AIにkintone操作させてる？

直近だと Claude Code にやってもらってます！  
ただ原則として私のローカルPCから動かしてます  
アプリ側にJS配置したりはしてないの

ローカルPC上でMCPやpy使ってレコードやアプリをいじってます  
だから他の人にはできなくて、私自身の業務改善にしかなってない現状

でも今はそれでいいかなって思ってます  
AIを使うとこんなことができるよと社内にデモしてる段階  
イノベーターやアーリーアダプターがやりたいと言ってくれるのを待ってる

思えば2019年頃までkintoneもこんな感じでした懐かしい

## で、具体的に何やらせてるかというと！

ズバリ見た目！  
見た目がきれいになるのが一番わかりやすい！

そしてその実現のために役立ってくれたのがリッチエディター！  
正直申し上げて2013年導入以来初めてこんなに活躍してますｗ

Claude Code に html として保存してもらうとこんな感じ！  
例えば営業日報のサマリー！
