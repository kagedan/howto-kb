前回の中級では、**Claude Code**を使って、**記事管理、CSV整理、チェックリスト生成、小さな自動化**にどうつなげるかを整理しました。  
  
今回のテーマは「**上級**」です。  
  
ここからは、**Claude Code**を単発の便利ツールとして使う段階から、**自分の作業基盤**として整える段階に入ります。  
  
具体的には、次のような内容です。  
  
**カスタムコマンド  
Hooks  
MCP  
権限管理  
設定スコープ  
複数ファイル運用  
サブエージェント**  
  
Claude Code公式ドキュメントでも、カスタムコマンドは繰り返し使うワークフローをまとめる機能、HooksはClaude Codeの動作前後にコマンドなどを実行する機能として説明されています。 ￼  
  
ただし、最初に大事なことを言います。  
  
**上級だからといって、全部を一気に使う必要はありません。**  
  
むしろ、全部を雑に入れると危険です。  
便利になるどころか、何が動いているのかわからなくなります。  
  
Claude Code上級で大事なのは、機能を増やすことではありません。  
  
**自分の作業を、再現性のある流れにすること**です。

### 1. 上級で目指すのは「毎回頼む」から「仕組みにする」こと
