寝ている間に仕事を片付けてくれるGitHubリポジトリ10選：

1. OpenHands
AIエージェントが代わりにコードを書いてくれる自律型の開発支援ツール。自分でキーボードを叩かなくても、指示するだけで動かせる。
https://t.co/2V4QkE6iE0

2. Hermes Agent
Nous Researchが2026年2月にリリースした、自己改善するパーソナルAI。使うほど賢くなり、Telegram・Discord・Slackなど複数のチャネルから操作できる。
https://t.co/Xj5kY3HUz4

3. CrewAI
複数のAIに役割を与えてチームとして動かすフレームワーク。単体のAIでは難しいタスクを、役割分担させて実行できる。
https://t.co/AklNDTXjUf

4. Aider
ターミナルで動くAIペアプログラマー。指示を出すと差分を自動でGitにコミットしてくれる。コードが書けない人でも会話するだけで変更が入る。
https://t.co/6oFgziLHR0

5. n8n
400以上のサービスとつながるフェアコードのワークフロー自動化ツール。サブスク不要でサーバーに置いて動かし続けられる。
https://t.co/pvJGzLdiyo

6. LangGraph
本番環境で動くAIエージェントの土台となるフレームワーク。複雑な処理フローを状態管理しながら安定して動かせる。
https://t.co/eLWEIkFvQt

7. Cloudflare Agentic Inbox
AIエージェント内蔵のセルフホスト型メールクライアント。Cloudflare上で動き、受信メールへの対応を自動化できる。
https://t.co/nd8rFqQGm9

8. Browser Use
AIがWebブラウザを直接操作して、フォーム入力・データ抽出・予約などを代行するツール。人がブラウザでやっている作業をそのまま任せられる。
https://t.co/bDao4MLSqr

9. awesome-mcp-servers
AIエージェントが使えるツール（GitHub・Slack・Notion・Stripe・Postgresなど）をまとめたカタログ。使いたい連携先をここで探せる。
https://t.co/1Yf8Kh9AC2

10. claude-task-master
Claude Codeで複数のAIエージェントを束ねて動かすオーケストレーションツール。1つのプロンプトからチームを編成して並列で作業させられる。
https://t.co/y4LQqkqmyE
