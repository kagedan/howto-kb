Claude Codeで複数プロジェクトを並行開発していると、Claudeからの承認待ちや定型的な指示入力が頻繁に発生します。左手デバイス「XPPen ACK05」のボタンに定型文入力を割り当てて、ワンボタンで応答できるようにしました。

EnterやDeleteといった単純なキー操作はXPPenドライバ側で直接割り当てられます。本記事では、ドライバだけでは実現できない「定型文の入力」にフォーカスします。

## 全体の構成

```
ACK05のボタン
  → キーボードショートカットを発火
    → Automatorクイックアクションが実行
      → AppleScriptでテキスト入力を送信
```

ACK05のボタンにはキーボードショートカットを割り当てられるので、そのショートカットをトリガーにしてMac側でAutomatorの処理を実行する2段構成です。
