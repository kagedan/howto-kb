ChatGPTが出始め時から触っていて、今はClaude Coworkメインで使っているが、進化が早すぎる。当初は検索の延長線上だったが、今は検索はもちろん、資料作り、財務分析、事業アイディアの壁打ち、スケジューリング、メール自動返信など、少なくとも5倍くらいは生産性が上がっている感じ。エンジニアの人はClaude Code Fable5を使い倒して、もっと高い生産性を生み出しているのだろう。

自分は2018年に起業して、数十名の方が弊社で働いてくれたが、正直仕事が遅い人や雑な人もいた。それは致し方ないとして、割とあるのが仕事を依頼すると不機嫌になる人だ。残酷なことを言うと、このタイプは本当にもう終わりだと思う。確実に昇給も昇進もないだろう。なぜならAIの方がいいからだ。

一方でAIにできないことはまだ結構ある。まず仮説を立てること。それもAIでできるよ、という人も多いと思うが、私は違うと思う。本当にモノになる仮説、平たくいうとお金をもらえるアイディアというのは実際に手を動かした人しか出せないし、そういう人じゃないと結局入金までやり切れない。AIで綺麗な資料を作ること自体に意味はないですからね。

次はフィジカルな部分。我々はトナリゾートでホテル開発&運営をしているが、例えばホテルの建築自体は人じゃないとできない、清掃もリネン交換も緊急駆けつけも人しかできない。その部分をしっかり組織化し、みんなが楽しく、やりがいを持って働けるような場を作れる会社は、むしろ今後も伸びると思う。

経営者である自分は、何のために働くのか、何のために生きるのか。みたいな部分をビジョンとして打ち出していかないとダメだよな、と強く思う。
