i've been using codex for personal projects on plan mode, and deepseek for execution with opencode. Good cost-benefit relation.

At work, where budget is unlimited, claude code is my main tool. Codex xhigh is just for a "second opinion".

If budget is not a real concern: claude ftw! No brainer. Results are better, execution is faster, multi-agentic and looping feels really easier to manage on claude. Also, community support, claude design, and claude code's TUI.

However, claude may sound a bit more invasive in comparison to codex, so you need to kinda add more harness to your workflow so it does not harm your systems. Codex, on the other hand, is more "respectful", less invasive, creates less files and does not often over-engineer.
