# Claude Codeの話題が、急に増えました。

Fable 5が出た。3日で制限に当たるほど使われている。

一気にメジャーなAIに躍り出てきました。

私はClaudeを使い始めて、丸2年になります。

今日はClaude.aiの始め方を書きます。

環境構築は要りません。ブラウザさえあれば動きます。

チャットとプロジェクト、コピペで使えるテンプレートつきです。

# **やってはいけない3つのミス**

![](https://pbs.twimg.com/media/HLArbBLbYAAy4Mn.jpg)

① **毎回新規チャットで最初から説明する**

Claude.aiは新規チャットを開くと、基本的にゼロから始まります。
仕事の内容、文章のトーン、避けたい表現を毎回入れる運用は、すぐに疲れます。
同じ説明を3回以上しているなら、チャットだけで頑張る段階は終わりです。

私も最初はこれをやっていました。
「前にも言ったのに」と感じた瞬間が、運用を変える合図でした。

② **無料プランの制限を知らずに制限で戸惑う**

無料プランには使用量の上限があります。
何度も長文を投げると、途中で制限に当たる日があります。
壊れたわけではありません。時間を置けば使える状態に戻ります。

③ **プロジェクトとチャットを混同する**

チャットはその場の会話です。

プロジェクトは、Claudeに前提条件を持たせて繰り返し使う場所です。

この2つを混ぜると、「便利なはずなのに毎回説明している」という状態になります。
一回だけ聞くならチャットで足ります。
同じテーマを何度も扱うなら、最初からプロジェクトに寄せた方が楽です。

# **アカウント作成（3分でできます）**

## **Step 1｜ claude.aiを開く**

ブラウザで claude.ai を開きます。
表示された画面から登録に進みます。

## **Step 2｜ Googleアカウントで登録**

登録方法は、Google、メール、Microsoftの3つです。
一番早いのはGoogleアカウントでの登録です。

メールアドレスとパスワードを入力する手間が減ります。

## **Step 3｜ プランを選ぶ**

最初は無料で使ってみてください。

使用量が多い時間帯は上限に達することがあります。日常的な使い方なら気にならない範囲です。

Claude.aiの利用条件は18歳以上です。
登録時に年齢条件と利用規約を確認してから進めてください。
ここで有料プランを選ばなくても始められます。

最初に確認したいのは、Claudeの返答が自分の作業に合うかどうかです。
数日使って、毎日制限に当たるなら、その時点でプランを見直してください。

![](https://pbs.twimg.com/media/HLAsOoTacAA24Q4.jpg)

*Free → Pro → Max(5x) → Max(20x)。まずFreeで始めて、制限を感じたタイミングでProを検討します。*

## **画面が英語のままの場合｜日本語に変更する**

登録直後は英語表示になることがあります。
日本語に変更するのは1分もかかりません。

1. 画面左下に表示されている自分の名前をクリックします
1. 「Language」をクリックします
1. 「日本語（日本）」を選択します
これだけです。英語のまま使い続ける必要はありません。

![](https://pbs.twimg.com/media/HLAsr3rbIAA7Zu0.jpg)

# **チャット**

チャットは、Claudeとその場で会話する機能です。

メールの返信、文章の直し、調べものの整理、アイデア出しなど、一回で完結する相談に向いています。
ChatGPTを使ったことがあるなら、画面の感覚は近いです。

まずは簡単な文章から始めていきましょう。

**最初のメッセージ**

```markdown
日本語で話してください。私は[職業や用途]をしています。文章は[希望のスタイル]でお願いします。
```

チャットでできることは、例えばこのあたりです。

- メールの返信文を作る
- 長い文章を短く要約する
- 会議のメモを整理する
- 投稿文や説明文のたたき台を作る
難しいプロンプトは不要です。

まずは普通の文章で頼めば返ってきます。

うまく返ってこないときは、命令を複雑にするより、前提を1つ足します。

「誰向けか」「何文字くらいか」「どんな雰囲気か」
