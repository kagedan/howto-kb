We've just added several new features to Claude Managed Agents.

You can now configure effort levels per agent, seed sessions with events, add up to 500 skills per session, use webhooks for environments + memory stores, and stream events for sub-agents. https://t.co/MofmNFv7k4

Effort controls how much each Managed Agent thinks. 
 
Lowering effort gives faster responses and lower cost. https://t.co/sM3nVk1KDD

Before, creating a session and sending it events took separate API calls.

You can now seed a new session with up to 50 user_message and define_outcome events in the create call. https://t.co/cGt8m6izG2

Skills give your Managed Agent task-specific instructions. 

You can now add up to 500 skills across all Managed Agents in a session. https://t.co/0Uttoy64ce

Try out these new features using our cookbook:
https://t.co/F4lTpWUGd8
