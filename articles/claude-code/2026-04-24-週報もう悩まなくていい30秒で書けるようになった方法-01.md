週報って、なぜあんなに時間がかかるんでしょう。

実際にやったことは覚えている。でも「それをどう書くか」に詰まる。1週間の出来事を整理して、適切な言葉にして、上司に伝わる形にして——気づいたら30分経っている。

しかも毎週。

これ、Claude Codeで終わりにできます。**やることをメモしておくだけで、週報が自動で出来上がります。**

## Before / After

**Before：** 1週間を思い出しながら30分かけて書く → 言葉選びに詰まる → 毎週憂鬱

**After：** 走り書きメモをClaude Codeに渡すだけ → 30秒で完成 → **時間を98%削減**

## 準備するもの

- Claude Code（インストール済みであること）   
- 今週やったことのメモ（どんな形でもOK）

Claude Codeのインストールがまだの方は、記事001「Claude Codeのインストール方法」を先にご覧ください。

## 手順（4ステップ）

### ステップ1：「今週やったことメモ」をその日に書く

週報が難しい理由の9割は「1週間分を一気に思い出そうとするから」です。

解決策はシンプル——**その日の終わりに1〜3行だけメモする。**

```
月: 〇〇社への提案書作成、田中さんとミーティング
火: 提案書修正、△△の見積もり確認
水: 先方への提案、承認もらえた
木: 新規案件の初回ヒアリング
金: 月次レポート作成、来週の準備
```

メモ帳でも、スマホのメモでも。形は何でも構いません。

### ステップ2：週報フォルダを用意する

週報に使うメモをまとめておくフォルダを1つ作るだけです。

```
📁 週報フォルダ
  └── メモ.txt
```

### ステップ3：メモを貼って送るだけ

Claude Codeを起動して、こう送ります。

```
今週の週報を作ってください。

月: 〇〇社への提案書作成、田中さんとミーティング
火: 提案書修正、△△の見積もり確認
水: 先方への提案、承認もらえた
木: 新規案件の初回ヒアリング
金: 月次レポート作成、来週の準備
```

### ステップ4：出てきた週報を確認する

数秒で以下の形式で出力されます。

```
## 週報（4月7日〜4月11日）

### 今週の主な成果
1. 〇〇社への提案が承認されました
2. 新規案件のヒアリングを実施、課題を整理しました

火曜日：提案書修正完了、△△案件の見積もり確認
（以下続く）

### 来週の予定
- 〇〇社との契約手続き
- △△プロジェクトのキックオフ準備
```

あとは確認して送るだけ。

## 毎週30分の憂鬱が、消えます。

「メモを渡せば週報ができる」のはわかった。でも毎回「うちのフォーマットで」「KPI数値を入れて」と説明するのは手間です。来週も、再来週も、毎回同じ説明。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、自社のフォーマット・ルールを覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎週の週報が変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「ですます調で」と説明が必要   
✅ ファイルあり：自動でですます調

❌ 設定なし：毎回「成果3項目で」と伝える   
✅ ファイルあり：毎回3項目で出る

❌ 設定なし：毎回「来週の予定も入れて」と追加   
✅ ファイルあり：必ず入る

❌ 設定なし：毎回「うちのフォーマットで」と説明   
✅ ファイルあり：毎回そのフォーマット

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「週報の憂鬱」が消えます。**

## 100円で何が変わるか

毎週30分 × 52週 = **年間26時間が週報に消えています。**

設定ファイル1枚で、来週からその26時間が戻ってきます。

**100円 ÷ 26時間 = 1時間あたり8円。** これより安く時間を買う方法は、なかなかありません。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-003（添付）**　←　ダウンロードして置くだけ   
- **フォーマットを自社に合わせる3ステップ**　←　KPI数値・プロジェクト別など対応   
- **30秒メモ法**　←　1日30秒のメモで1週間分のデータが溜まる

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「どんな業務や形式で使いますか？」→ 答える   
- 「フォーマットに決まりはありますか？」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたの週報スタイルを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
