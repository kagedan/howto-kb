Claude Codeを使っていると、こんなコマンドに出くわす。

```
npx skills add python-patterns
```

あるいはこんなメッセージ。

> このプラグインをインストールします。スキル10個とコマンドが一緒に入ります。

「スキル」と「プラグイン」。名前は聞いたことあるけど、何が違うのか。多くの人は、ぼんやりしたまま使っている。

でも違いを一度押さえると、Claude Codeを「自分向けに改造する」感覚が劇的に変わる。ここで整理しよう。

## スキル——「1つの技」を追加する

スキルは、**特定分野に特化した専門知識パッケージ**だ。

`python-patterns` なら、Pythonの設計パターン（ファクトリー・シングルトン・オブザーバーなど）だけ。`stripe-best-practices` なら、決済サービスStripeの使い方だけ。

スキルは**点**のようなもの。「Pythonパターンはどう書く？」と聞きたいとき、ピンポイントで答える。他の分野は関係ない。

インストールすると `/python-patterns` みたいなコマンドで呼び出せる。あるいは文脈に応じてClaude自身が「このタスク、python-patternsが効きそうだ」と自動起動することもある。

## プラグイン——「セットで整える」

プラグインは何か。**スキル＋コマンド＋設定＋エージェントをまとめて、ある分野まるごと整えたもの**だ。

営業支援プラグイン `sales` なら、一度にこれらが入ってくる。

* スキル4〜5個（営業メール・提案資料・顧客分析など）
* コマンド3〜4個（新規開拓・フォローアップ・商談メモなど）
* ダッシュボードとテンプレート
* 営業チーム向けの設定ルール

スキルは**単品アプリ**。プラグインは**Officeのようなアプリスイート**だ。

## 使い分け——ピンポイントか、ドメインごとか

どちらを選ぶか。判断は単純だ。

| 状況 | 選択 | 理由 |
| --- | --- | --- |
| Rustのテストコード、今すぐ書きたい | スキル (`rust-testing`) | 1つの技だけ必要 |
| Rustプロジェクト全体を立ち上げたい | プラグイン（あれば） | ビルド・テスト・レビュー・パフォーマンスまでセット必要 |
| SEOキーワード調査だけ | スキル (`claude-seo:seo`) | その1つで充分 |
| Webサイト全体のSEO診断～改善まで | プラグイン | SEOスキル複数＋サイト監査＋コンテンツ戦略がセット |

ピンポイントで1つの技を足したい——**スキル**。ある分野の仕事をレベルアップさせたい——**プラグイン**。

## 実装側の工夫——ダッシュボード付き

プラグインの強さは、スキル単体を並べるだけでなく、**それらを組み合わせるための仕組みも入ってくる**ところだ。

営業プラグインなら、見込み客リスト、商談パイプライン、提案文面テンプレート、フォローアップのスケジュール——これらが1つの「営業ダッシュボード」になって動く。スキル5個を自分で組み立てる苦労がない。

つまり、プラグインは**スキルの詰め合わせであると同時に、その使い方の設計図も込みで配ってくれる**わけだ。

## マーケットプレース——どこから入手する

スキルもプラグインも、公開されているものは `npx skills` で検索・インストール可能。

```
npx skills search marketing
npx skills add marketing:cold-email
```

プラグインも同じ仕組みで入手できる。ただし、すべてがマーケットプレースにあるわけではない。社内用や限定公開のものは、別ルート（プライベートリポなど）から手動で入れることもある。

## よく聞く用語ミニ辞典

* **コマンド**：`/python-patterns` みたいなスラッシュコマンド。スキル・プラグイン内で定義され、Claude Code内から呼び出すショートカット。
* **エージェント**：より複雑なタスク用。複数ステップを組み合わせて実行する仕組み。スキルより粒度が大きい。
* **ダッシュボード**：プラグインに含まれることもある、統合的なUIやテンプレート集。
* **マーケットプレース**：スキル・プラグインを配る公式窓口。AppStoreのようなもの。

## まとめ

|  | スキル | プラグイン |
| --- | --- | --- |
| 正体 | 単一分野の専門知識パッケージ | スキル＋コマンド＋設定の詰め合わせ |
| 粒度 | 1つの技（点） | 分野全体（面） |
| こんな時に | Pythonパターンだけ、メール作成だけ | 営業業務全体、Rustプロジェクト一式 |
| 組み立て | 必要なスキルを自分で選ぶ | 前提となる組み合わせが決まってる |

ひとことで言えば——**スキルで「技」を足して、プラグインで「仕事」を整える**。どちらも「Claude Code化」の武器だが、使う局面が違うだけだ。
