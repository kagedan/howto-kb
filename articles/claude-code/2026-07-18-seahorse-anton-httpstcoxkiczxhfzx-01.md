## 2. The tech: Pharma.AI — an end-to-end generative stack

Insilico runs **two engines in parallel**: a licensable software platform *and* an in-house pipeline that eats its own dog food.

- **Biology42 (PandaOmics + TargetPro):** multi-omics target discovery; scores targets on confidence, druggability, "mechanism clarity." Launched **TargetBench 1.0**, the industry's first open target-discovery benchmark. New **PandaClaw** agent adds autonomous multi-omics reasoning.
- **Chemistry42:** generative + physics-based small-molecule design (de novo, scaffold-hopping, ADMET, kinome off-target prediction). New **Nach01** — a multimodal chemistry foundation model (billions of molecule/text tokens) — now on AWS Marketplace and Microsoft Discovery. **Chemistry42+** exposes the whole workflow via MCP into Claude Code/Cursor — genuine "agentic CADD."
- **Medicine42 (inClinico):** predicts clinical trial outcome probability — used to de-risk indication/trial design before dosing patient one.
- **Science42 (DORA):** open-sourced, Azure-backed multi-agent research assistant for literature synthesis and manuscript drafting.
- **Science MMAI Gym:** foundation-model training framework; first external deployment with **Liquid AI** produced a 2.6B retrosynthesis model and a 24B general model, on-prem/Azure deployable.
