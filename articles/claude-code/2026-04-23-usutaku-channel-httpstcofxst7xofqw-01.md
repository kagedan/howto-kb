# 初心者向けの投稿一覧

## **コードが書けない税理士が、Claude Codeを使えるようになるまでにやった4つのこと→初級向け**

[https://x.com/kandmybike/status/2033148559894438337?s=20](https://x.com/kandmybike/status/2033148559894438337?s=20)

実際にClaude Codeを使い倒している税理士の方の投稿で、初心者向けではあるがシンプルで良い記事。特に、AIに作業場所を決めさせるのは大事。初心者の場合、特にCLIでClaude Codeを使っている人は「今作ってもらったファイルどこやねん！！」とひたすら怒ることになる。その他、MCP連携の話やSkillsの話は中級者には当たり前だが必ず知っておきたい基礎知識。

## Claude Codeの操作終了時にずんだもんで音を鳴らす→初心者向け

[https://x.com/0xfene/status/2030504252561617126?s=20](https://x.com/0xfene/status/2030504252561617126?s=20)

中々ユニークなアプローチの記事。Claude Codeを使っていて、許可が求められているのにもかかわらず気づかないといったことはよくある。そんな時にフリー音声素材のずんだもんで音を鳴らすというTips。くだらなく見えるが、やってみると体験として面白い。他にも色々Tipsが載っていて、音声入力AIを使う、そのためにマイクを買う、など。

## Claude Codeの進捗状況を教えてくれるキャラを召喚する→初心者向け

[https://x.com/dingyi/status/2046966571135021190?s=20](https://x.com/dingyi/status/2046966571135021190?s=20)

終了時にずんだもんで音を鳴らすのと近い思想で、Claude Codeの作業状況を教えてくれるミニキャラ。とても可愛い。音を出せない場面は意外と多いので、そういう時に良いかも。めっちゃ便利かと言われるとそうでもないが、可愛い。

![](https://pbs.twimg.com/media/HGlHAoZa8AAITRz.jpg)

## AskUserQuestionsToolを使って仕様を固める→初心者・中級者向け

[https://x.com/trq212/status/2005315275026260309?s=20](https://x.com/trq212/status/2005315275026260309?s=20)

Claude Code に全部任せると変な方向に行くことも。そこで、最初にAskUserQuestionToolを使うことで、まずClaudeが色々質問してくれるようになる。基本は開発で使うけれど、スクショはX投稿を要約して欲しい時の質問。何を知りたいのかや、どれくらい出力して欲しいのかを事前にすり合わせる。

![](https://pbs.twimg.com/media/HGlHK_qbIAAGKDQ.jpg)
