こんにちは、K-SOCKETのほんだです。

非エンジニアです、GA4とWeb広告の波に日々揺られながらレポート作成と提案などを担当しています。

一応、html/cssはここをいじったらここがちょっと動くんじゃね？ぐらいには読めます。

![](https://assets.st-note.com/img/1772761626-CeBhyUFZpjGPTaRr3tNs7g8M.png?width=1200)

大体こんな格好しています  
（しばらくこの画像を使っていく所存）

ターミナルって知ってますか。  
Macだと「ターミナル」、Windowsだと「コマンドプロンプト」とか「PowerShell」とか呼ばれているやつです。真っ黒な画面に文字だけが並ぶ、あれです。

![](https://assets.st-note.com/img/1772761777-7bDFy2ceqwsUHXKpmdziJfL9.png)

MATRIXを思い出したら大体同い年

**SF映画で「椅子の人」がカタカタやってるやつです。**  
ハッカーが緑の文字を高速スクロールさせながら「侵入完了」とかやってるあの画面。私の中でターミナルはずっとそういうイメージでした。自分が使うものだとは思っていなかった。

### 最初に開いたとき、普通にテンションが上がった

上がるじゃん。黒い画面がカラフルな文字列で埋め尽くされていたら。

文字しかない。でもかっこいい。  
私が知っているコマンドは「ls」と「cd」だけです。lsはファイルの一覧を表示するやつ、cdはフォルダを移動するやつ。それだけ。でも打つと反応する。画面が動く。なんか俺、やってる感じがする。

**Claude Codeとの出会い——「これやっていい？」地獄**

そこにClaude Codeが登場します。ターミナル上でClaudeに話しかけると、Claudeが代わりにコマンドを打ってくれる、というツールです。  
使い始めてすぐ気づいたのですが、Claude Codeは作業を進めるたびに確認を求めてきます。

> 「このファイルを作成してもいいですか？」  
> 「はい」  
> 「このコマンドを実行してもいいですか？」  
> 「はい」  
> 「この変更を適用してもいいですか？」  
> 「はい」

画面にはどんどんコードが生成されていきます。Claudeが勝手に動いて、ファイルが作られて、何かが実行されていく。**「これやっていい！？」という承認を連打しているうちに、なんかすごいことが起きている気がしてくる。**  
これが「ちょっとだけやばい」と思った瞬間です。俺、なんかやってる。何をやってるかはわからないけど、やってる。

### そして「何をやってるかわからなくなった」

ただ正直に言います。承認を続けていたら、途中から**何をやっているのか完全にわからなくなりました。**

Claudeは親切なので、実行するたびに説明をしてくれます。でもその説明自体がわからない。「依存関係をインストールしています」「環境変数を設定しています」「ポートを開いています」……。

ひとつひとつの言葉は日本語なのに、意味がわからない。気づいたらlocalhost:3000が立ち上がっていて、ブラウザで何かが表示されていて、「あ、動いてる……？」という状態になっていました。  
自分で運転していないのに、気づいたら目的地に着いていた感じ。

![](https://assets.st-note.com/img/1772762069-M0UvEV6mctGyOPNFqhRTsdSI.png?width=1200)

前回記事でお馴染み？の成果物画面です

※今回も感想をこの組み上がったであろうnoteクリエイターさんに、原稿をええ感じにしてもらって書いています。

**lsとcdしかわからなくても、なんとかなるのかもしれない**

そのうちきっと覚える。RPGの新しい技のように…

今のところの結論はこれです。  
ターミナルはかっこいい。Claude Codeは勝手に進む。何が起きているかはよくわからない。でも、なんか動く。

非エンジニアがこの領域に踏み込むのは、地図なしで知らない街を歩くような体験です。でも案内人（Claude）がいると、とりあえず迷子にはならない。どこに来たかはわからないけど。

ls、cd、そして「はい」。今の私のターミナルスキルはだいたいこの3つです。
