## Claude Codeのセットアップ

### ① Node.jsの更新

Claude Code CLIはNode.js v18以上が必要です。既存のバージョンがv16だったため、公式サイトからv24（LTS）のインストーラーをダウンロードして更新しました。

```
node --version
# v24.14.1
```

### ② Claude Code CLIのインストール

PowerShellのスクリプト実行ポリシーを変更してからインストールしました。

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
npm install -g @anthropic-ai/claude-code
```

### ③ VS Code拡張機能のインストール

VS Codeの拡張機能から「Claude Code for VS Code」（Anthropic製）をインストールしました。

### ④ Claude Codeの起動・ログイン

Claudeアカウント（Proプラン）でログインすることで利用できます。

## プロジェクトの初期化

`aws-saa-quiz-app` フォルダをVS Codeで開き、Claude Codeのターミナルで `/init` を実行しました。

Claude Codeが `prompts/base_prompt.md` の内容を自動で読み取り、以下の情報を含む `CLAUDE.md` を生成してくれました。

* プロジェクト概要
* 技術スタック（Python / Streamlit / Gemini API）
* ファイル構成
* 起動コマンド
* 状態管理・JSONスキーマ

**自然言語で指示するだけでプロジェクトを理解してくれるのが印象的でした。**

## 実装の流れ

基本設計で決めた順番で進めました。

```
① questions.json の作成
② gemini_client.py の作成
③ app.py の実装
④ 動作確認
⑤ 5問に拡張
```

### ① questions.json の作成

Claude Codeに以下のように指示しました。

```
data/questions.jsonを作成してください。
IAMカテゴリのAWS SAA問題を1問作成してください。
```

IAMグループに関する問題が自動生成されました。データ構造も設計通りに作られています。

```
{
  "id": 1,
  "question": "AWS IAMにおいて、複数のユーザーに同じ権限をまとめて付与したい場合、最も適切な方法はどれですか？",
  "choices": [
    "各ユーザーに個別にIAMポリシーをアタッチする",
    "IAMグループを作成し、グループにポリシーをアタッチしてユーザーを追加する",
    "AWSルートアカウントを共有して使用する",
    "IAMロールを各ユーザーに直接割り当てる"
  ],
  "answerIndex": 1,
  "category": "IAM"
}
```

### ② gemini\_client.py の作成

```
services/gemini_client.pyを作成してください。
google-genai SDKを使用し、APIキーは.envから読み込み、
初心者向けの解説を生成して返してください。
```

シングルトンパターンでクライアントを管理し、エラー時は適切なメッセージを返す実装が生成されました。

### ③ app.py の実装

```
app.pyを作成してください。
StreamlitでAWS SAA問題集アプリを作成してください。
st.session_stateで状態管理し、
回答後にGemini APIで解説を生成・表示してください。
```

`@st.cache_data` でJSONをキャッシュ、`st.session_state` で問題番号・回答済みフラグ・解説を管理する実装が生成されました。

## Gemini APIのセットアップ

### APIキーの取得

1. <https://aistudio.google.com> にアクセス
2. 「APIキー」→「APIキーを作成」でキーを取得
3. `.env` ファイルに保存

```
GEMINI_API_KEY=your_api_key_here
```

### ハマったポイント

最初に `gemini-2.0-flash` を使ったところ `429 RESOURCE_EXHAUSTED` エラーが発生しました。新規APIキーの無料枠クォータが0になっていたため、`gemini-1.5-flash` に変更することで解決しました。

```
# 変更前
model="gemini-2.0-flash"

# 変更後
model="gemini-1.5-flash"
```

## Claude Codeを使った感想

**良かった点**

* 自然言語で指示するだけでファイルが生成される
* エラーが出ても自動で原因を特定して修正してくれる
* `.env` を `.gitignore` に含めるなどセキュリティ面も配慮してくれる
* コミット・プッシュまで一貫して対応してくれる

**注意点**

* 生成されたコードは必ず自分で確認する
* 一度に大量の指示を出すより、小さく分けて指示した方がうまくいく
