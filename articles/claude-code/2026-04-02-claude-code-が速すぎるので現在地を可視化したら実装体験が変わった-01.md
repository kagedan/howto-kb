AI が速すぎて、ついていけない
Claude Code に「この機能を実装して」と頼むと、一瞬でソースコードを生成してくれます。
便利になったと思う反面、自分の理解が追いついていない...そんな状態ではレビューもデバッグもできません。
仕様駆動開発などのアプローチもありますが、クリーンアーキテクチャのような複数レイヤーのシステムでは、テキストベースでは全体像を直感的に把握しづらいと感じていました。

 シーケンス図で「現在地」を示してみた
そこで思いついたのが、 シーケンス図にレイヤーごとの実装状態を色で表現する ことです。
各レイヤーを box でグループ化し、実装済みを緑、未実...
