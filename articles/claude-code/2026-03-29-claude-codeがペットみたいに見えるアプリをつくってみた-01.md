## Claude Petでできること

デスクトップの好きな位置に設定した画像（キャラクター）が常駐します。  
Claudeが考えている間と、入力を待っている間で、表示されるアニメーションを設定できます。  
キャラクターはデスクトップ上に配置できます。

**idle（入力待ち）の状態**

Claudeがユーザーの入力を待っているときのアニメーションを設定できます。

**thinking（処理中）の状態**

Claudeがコードを書いたり、ファイルを読んだり、検索したりしているときのアニメーションを設定できます。

**Escで中断したとき**

Claudeの応答を途中でEscキーで止めた場合も、idleに戻ります。

![](https://static.zenn.studio/user-upload/0b2214cb888b-20260329.png)

## アニメーションは好きなものに変えられる

設定画面からidle用とthinking用のアニメーションファイルを選べる。

対応している形式は以下の通り。

* **Lottieアニメーション（JSON）** — ベクターのなめらかなアニメーション
* **GIF** — アニメーションGIF
* **SVG / PNG / JPG** — 静止画

LottieやGIFを設定すると、状態が変わるたびにアニメーションがループ再生される。好きなキャラクターの素材を用意すれば、自分だけのデスクトップペット風になります。
