【Claude Code公式ドキュメント解説】
CLAUDE.mdについて。

Claude Codeodeに「いつもこうしてほしい」というルールを書いておけるのがCLAUDE.mdです。

毎回のチャットで自動的に読み込まれるので、ファイルの保存場所やコードの書き方のルールなど、一度書けば毎回守ってくれます。

似た機能にSkillsとRulesがあり、
・CLAUDE.mdは毎回必ず読まれる
・Rulesは特定のファイルを開いた時だけ読まれる
・Skillsは必要な時だけ呼び出す
という違いがあります。

CLAUDE.mdは200行以下が目安(日本語なら100〜150行を意識すると安心)。

膨らんできたら、参照用の知識はSkillsに移すか、.claude/rules/に分割するのが公式の推奨です
この辺り、あやふやな解説をよく見かけますね😅
