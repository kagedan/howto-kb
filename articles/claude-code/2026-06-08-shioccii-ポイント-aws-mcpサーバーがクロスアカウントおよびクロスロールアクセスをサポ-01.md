▶️ポイント
・AWS MCPサーバーがクロスアカウントおよびクロスロールアクセスをサポートし、AIコーディングエージェントが複数のAWSアカウントで作業できるようになった。
・開発者は、各コマンドでプロファイルを指定することで、アカウントやロールをシームレスに切り替えられるようになり、作業効率が向上する。
・これにより、DevOpsエンジニアは異なるアカウント間でログを照会したり、アプリ開発者は異なるアカウントで設定を変更したりできる。
