If you bounce between Claude Code, Codex, Grok and OpenCode all day, this is for you.

AOP turns your agent CLIs into one factory. You talk to one chat. Type /task create and a full spec is written for you. Assign it to a worker and a workflow takes over: one CLI implements, another reviews, and it loops until the tests actually pass. Each step picks its own model, so cheap models do the cheap work.

Don't want workflows? Just chat. Start with Grok, type % and hand the next message to Codex for a review, type $ and borrow a browser from another CLI to verify your endpoint live. Same conversation, different vendor, one keystroke.

Bring any CLI. Add your own with a custom provider.

And the part we care about most: everything runs on your machine. Your code, your keys, your data never leave it. No servers, no accounts, no telemetry. AOP is just an orchestrator for the CLIs you already trust.
