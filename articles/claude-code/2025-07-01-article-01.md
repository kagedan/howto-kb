Claude Code の hooks でユーザーに確認を求める時に、AivisSpeechの音声で声をかけてもらう方法

1️⃣AivisSpeechで音声を作って書き出しする
2️⃣Claude Codeを開いて /hooks と打つ
3️⃣Notificationを選択
4️⃣+ Add new hook...を選択
5️⃣afplay {書き出した音声ファイルのパス} を設定
