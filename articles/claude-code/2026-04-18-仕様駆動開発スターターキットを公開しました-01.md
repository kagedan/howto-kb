# 仕様駆動開発の基本思想

このスターターキットが前提とする役割分担は以下の通りです。

- **人間の役割**: 仕様の決定、レビュー、テストによる品質担保
- **AIの役割**: 仕様に基づくコードの実装、ドキュメントの生成・更新

**人間がコードを直接書くのではなく、仕様（ドキュメント）を通じてAIに意図を正確に伝え、その出力をテストで検証する**のが基本の開発サイクルです。
