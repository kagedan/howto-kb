「ハーネスエンジニアリング」って、何から始めればいいの？
Claude Codeを使い込んでいくと、ある時点でハーネスエンジニアリングという概念に出会います。CLAUDE.md、hooks、permissions、memory、subagents——Claudeの動作を制御する設定群を総称して「ハーネス」と呼び、それを体系的に設計・運用することをハーネスエンジニアリングと言います。

 なぜハーネスエンジニアリングが必要なのか
