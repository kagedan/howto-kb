今日は久しぶりに出勤で、電車から書いてます。もう世の中は暖かいですね。AnthropicのClaudeに関する記事を見つけましたので紹介します。  
なんでも、AIが直接PCを操作できるようになるらしいんです。最近、バイブコーディングの相棒をClaude Codeに乗り換えようかと検討している私としては、これはかなり気になるニュース。今回はその詳細と、私たちのPC作業がどう変わるのかについて書いてみます。

### まるで透明人間？Claudeの「Computer Use」機能とは

　AIの進化って、本当に日進月歩ですよね。テキストを生成したり、画像を作ったりするのはもう当たり前になりましたが、今回Anthropicが発表したClaudeの新機能は、次元が違います。  
　なんと、AIが私たちの代わりにPCの画面を見て、マウスカーソルを動かし、クリックし、キーボードでテキストを入力するんです。これ、けっこう驚きでした。まるで、透明人間が自分のパソコンを操作しているような感覚ですよね。  
　現在は開発者向けAPIを通じたパブリックベータ版として提供されている「Computer Use」という機能なんですが、これまでの「AIに指示を出して、返ってきたテキストを人間がコピペする」という枠組みを完全に壊すものです。AI自身がツールを使いこなす時代に突入したんだなと、強く実感しました。

### どんなことができるようになるの？

　で、具体的に何ができるのかというと、これがまた夢が広がるんです。  
　例えば、ソフトウェアのテスト。人間が何度もクリックして動作確認するような作業を、Claudeが代わりにやってくれます。他にも、複数のウェブサイトを巡回して情報を集め、それをスプレッドシートにまとめるようなデータ入力作業も自動化できるそうです。  
　私は某通信関係の会社で働いているんですが、社内には「システムとシステムの間を人間が手作業でコピペして繋ぐ」みたいな、泥臭い業務がまだまだ残っています。そういう単純だけど時間が溶けていく作業をAIが丸投げでやってくれたら、どれだけ楽になるだろうか……と、つい自分の職場に重ねて妄想してしまいました。  
　金融関係のデータ集めなんかにも使えそうですよね。毎日決まった時間に特定のサイトから為替や株価の情報を引っ張ってきて、自分用のレポートを作らせる、なんてことも簡単にできそうです。

### 完璧ではない？現在の課題と注意点

　ただ、夢のような機能とはいえ、今のところはまだ完璧ではありません。  
　記事によると、スクロールがうまくできなかったり、クリックする場所を間違えたりすることもあるみたいです。人間なら「あ、ちょっとずれた」と一瞬で気づいて修正できるようなミスでも、AIは気づかずにそのまま次の作業に進んでしまうことがあるんだとか。  
　正直、最初はピンと来ませんでした。でも、よくよく考えてみると、これって結構怖いことですよね。もしAIが間違えて重要なファイルを削除してしまったり、関係ない人にメールを誤送信してしまったりしたら……と考えると、ゾッとします。  
　だからこそ、今はまだ完全に手放しで任せるのではなく、人間が横で見ていて「ストップ！」と言える環境で使うのが正解なんだと思います。あくまで私たちの作業を助けてくれる「優秀だけどちょっとおっちょこちょいな助手」くらいの距離感が、ちょうどいいのかもしれませんね。

### バイブコーディングの未来とClaude Codeへの期待

　それで、なぜ私がこの「Computer Use」のニュースにここまで食いついているかというと、最近「Claude Code」への乗り換えを真剣に検討しているからなんです。  
　ここ最近、私は「Antigravity」というツールを使って、AIにコードを書かせる「バイブコーディング」にすっかりハマっています。  
　とはいえ、最近Anthropicがリリースした「Claude Code」の話を聞くたびに、心が揺れています。ターミナルから直接プロジェクトを操作して、自律的にコードを編集したりコマンドを実行したりしてくれるエージェント型のツールらしいんですが、今回の「Computer Use」機能の進化を見ると、AnthropicのAIがどれだけ「自律的にタスクをこなす」ことに長けてきているかがよく分かります。  
　PCの画面を直接操作できるAIを作れる企業が作ったコーディングエージェントなら、もっと直感的に、それこそ私のフワッとした指示でも意図を汲み取って、勝手に良い感じのシステムを作ってくれるんじゃないか。そんな期待が膨らんでしまって、乗り換えたくてウズウズしているところです。

### 50代ガジェット好きの、新しい技術との付き合い方

　年齢を重ねると新しい技術についていくのが億劫になる、なんてよく言われますが、私にとってはまったくの逆ですね。52才になった今でも、こういう新しいテクノロジーのニュースを見るとワクワクします。  
　とりあえず、近いうちにClaude Codeを導入して、自分のバイブコーディング環境がどう進化するのか、じっくり遊んでみようと思います。その時の様子は、またこのnoteで報告しますね。
