この記事では 2026 年 4 月時点の全機能をカテゴリ別に整理し、「意外と使われていない便利機能トップ 10」も紹介します。

:::note info
本記事は Claude Code v2.1.x 系（2026 年 4 月時点）の情報です。バージョンによって利用できない機能があります。
:::
