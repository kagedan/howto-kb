これ、毎日開発してる人間からしたら
致命的すぎる落とし穴なんだよな。

せっかく蓄積した成功パターンも
エラーの解決策も、全部消えてなくなる。

対策は3つ。

・外部ストレージへの自動保存
・CLAUDE.mdでのコンテキスト管理

・フックを使った強制的な防御

これだけでAIのミス率は激減する。
