Claude Code の入門書はもう十分あります。本書が扱うのは、その先です。

「自分が手を動かす開発」から「AIチームに任せる開発」へ。サブエージェント、動的ワークフロー、/loop、/goal、スケジュール実行、メモリ、Skills、リモート制御――Claude Code を"自律的に働かせる"ための機能を、実際のコマンドと図解でひとつずつ解説します。

こんな人にオススメ
✅ Claude Code は使えるが「指示待ち」の使い方から抜け出せていない方
✅ 複数のAIエージェントを並列で働かせたい方
✅ CI監視や定期レポートをAIに任せて自動化したい方
✅ 寝ている間・外出中もエージェントに開発を進めさせたい方
✅ 自律実行の暴走・コスト・セキュリティのガードレールを知りたい方

各章は独立した「問い」に答える形で構成しています。図解（mermaid）とコード例を多用し、読んだその日から手を動かせる粒度でまとめました。
