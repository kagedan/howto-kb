「Claude Codeって、結局みんな何に使ってるんだろう？」

ひとりで触っていると、ふとそう思うことはありませんか。

私もずっと、そうでした。

はじめまして、「もの」と申します。

元中学校の理科教員、今はフリーランスでデザインとAI活用支援をしています。

非エンジニアの私がAIと一緒に進む試行錯誤を、同じ立場で悩む方の参考になればと書いています。

先日、Claude Codeを使っている人たちが集まる勉強会（ランチ会）に行ってきました。

そこで聞いた「みんな何作ってるの？」のリアルを、５つにまとめて持ち帰ります。

## ひとりで触っていると、現在地が見えなくなる

Claude Codeをひとりで触っていると、「これってあってる？」「みんなどこまでやってるんだろう？」と、わからなくなる瞬間があります。

公式ドキュメントを読んでも、SNSの発信を見ても、「自分の現在地」がぼんやりしてしまう。

だから、同じツールを使っている人と顔を合わせて話せる場は、本当にありがたいです。

美味しいランチを食べながら、それぞれの活用事例を順番に共有しました。

## みんな、Claude Codeで何を作ってる？

**① 写真を撮ったら、メルカリに自動で出品される仕組み**

物を売るとき、写真を撮って、説明文を書いて、価格を決めて、カテゴリを選んで…と手数が多い。

これを「写真を撮るところから先は、ぜんぶ自動」にしたい、という発想です。

これは便利✨

**② Claude Codeで作ったホームページを、どこに置くか問題**

Claude Codeでホームページを作った方も。

でも「で、これ、どうやって公開するの？」が次の壁です。

Cloudflare（クラウドフレア）に置いてみたら？ という話も出ていました。Cloudflareは、**作ったページをネット上に置いておける場所（サーバー）** のひとつです。

**③ 公式LINEから予約を受け付ける仕組み**

お客さんがLINEから予約を入れられるようにしたい。

電話やフォームではなく、ふだん使っているLINEで完結させたい。

これも「お客さんも自分も楽になる」発想です。これもすごい✨

**④ 同じプロンプトでホームページを作ったら、どこまで違う？**

「もしかしたら、知り合いからホームページ作成の依頼が来るかも」という方もいて、みんなで同じプロンプトを使ってホームページを作ってみました。

今回はClaude Codeではなく、ブラウザで使えるClaudeのチャット版（[claude.ai](http://claude.ai/)）で実験。

同じプロンプトなのに、出来上がるまでの過程も違うし、完成したホームページもちょっとずつ違うことにびっくり！

みんなで見せ合いっこしました💡

**⑤ せどり用のツールを作りたい**

仕入れの判断や、価格チェック、出品作業など、手でやるとしんどい部分を仕組みにしたい。

全部いっぺんに作らず、「リサーチの部分」「出品の部分」と少しずつ業務を分解して自動化するといいかも、とアイデアが出ました。

## ネット越しに知るのと、目の前で見るのは別物だった

正直に言うと、こういう活用法は、ネットの記事や発信で「こんなのできます」「便利ですよ」と紹介されているのを、私もどこかで見たことがあります。

それでも、目の前の人が実際に作って動かしているのを見ると、頭への入り方が全然違いました。

「あ、本当に作っている人がいるんだ」と、距離がぐっと縮まる感じです。

同じClaudeでも、人によって出てくる答えや反応がまちまちなのも、見ていて興味深かったです。

やはり人と会って話すのは楽しい、と改めて思った1日でした✨

## おわりに

5つの実例のうち、あなたが「これ気になるな」と思ったのはどれでしたか？

よかったらコメントで番号を教えてもらえるとうれしいです。
