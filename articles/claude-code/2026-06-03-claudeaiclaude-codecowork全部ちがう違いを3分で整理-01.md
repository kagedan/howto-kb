## まず知るべき3つの製品

Claude 関連の製品は全部で10種類以上あるが、最初に知るべき製品は3つだ。

[**Claude.ai**](http://Claude.ai)　チャット・文書作成・画像分析など、AIとの対話全般。ブラウザとアプリで使える。無料プランあり。

**Claude Code**　ソフトウェア開発専用のAIエージェント。プロジェクトフォルダの中に入ってファイル編集・コマンド実行まで行う。エンジニア向け。

**Claude Cowork**　ファイル操作・業務自動化に特化した機能。コードを書かなくても使える。エンジニア以外の知識労働者（財務・法務・マーケターなど）向け。

## 4軸の早見表

[**Claude.ai**](http://Claude.ai)

**Claude Code**

* インターフェース：ターミナル（コマンドを入力する画面）・IDE（VS Codeなどのコード編集ツール）・デスクトップアプリ・ブラウザ（4形態）
* コマンドライン：必要（CLI＝コマンドラインが中心）
* 主な用途：コードの編集・テスト・デバッグ・Git操作（バージョン管理）
* 向いている人：ソフトウェアエンジニア・開発者
* 無料プラン：なし（Pro以上）

**Claude Cowork**

* インターフェース：Claudeデスクトップアプリ内の「Cowork」タブ
* コマンドライン：不要
* 主な用途：ローカルファイルの読み書き・スケジュールタスク・外部ツール連携
* 向いている人：コードは書かないが自動化したい知識労働者
* 無料プラン：なし（Pro以上）

**ポイント：** [Claude.ai](http://Claude.ai) だけが無料で始められる唯一の入口だ。Claude Code と Cowork は両方とも月額$20以上のProプランが必要になる。

## 最大の落とし穴：「Claudeデスクトップ」と「Claude Codeデスクトップ」は別アプリ

ここが一番多い混乱なので、はっきり書いておく。

どういう混乱が起きるかというと、こうだ。

1. 「Claude Code を使いたい」
2. 「デスクトップアプリをインストールした」（Claudeデスクトップアプリをインストール）
3. 「あれ、Claude Code の画面が出てこない……」（Claude Codeデスクトップアプリは別のアプリ）

「Claude Code」という言葉はClaudeデスクトップアプリの中にも登場するため、混同しやすい。ただしClaudeデスクトップアプリに存在する「Chat・Cowork・Code」の3タブのうち、「Code」タブはあくまで基本的なコードのやり取りを指すものであり、専用の開発GUIである「Claude Codeデスクトップアプリ」とは別物だ。専用のGUI開発環境が欲しいなら、Claude Codeデスクトップアプリを別途インストールする必要がある。この2つは名前が似ているだけで、まったく別のアプリだ。

---

## 「Coworkはアプリではなくタブ」も押さえておく

同じような混乱として「Coworkをダウンロードしたい」というケースもある。

**Coworkは独立したアプリとして配布されていない。**

Claudeデスクトップアプリを開いて、「Cowork」タブに切り替えることで使える機能だ。単独でインストールするものではないため、Coworkの公式ダウンロードページは存在しない。

## 「どれを使えばいい？」判断フロー

自分がどれを使えばいいか迷ったら、この順番で考えてみてほしい。

```
Q1. AIと会話したい・文書を作りたい・画像を分析したい？
　→ YES → Claude.ai（まずここから）

Q2. コードを書く・ソフトウェア開発をしている？
　→ YES → Claude Code

Q3. コードは書かないが、ローカルファイルの操作や業務を自動化したい？
　→ YES → Claude Cowork（Claudeデスクトップアプリが必要）

Q4. 全部使いたい、または用途が複数ある？
　→ まず Claude.ai を試して、必要を感じたら ProプランでCodeまたはCoworkに拡張
```

「どれを選ぶか」より「まず [Claude.ai](http://Claude.ai) で何ができるか試す」が実用的な最初のステップだ。Pro以上に進むのは、無料プランを使い込んで「もっとやりたいこと」が明確になってからで十分。
