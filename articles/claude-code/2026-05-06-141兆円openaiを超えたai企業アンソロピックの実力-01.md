ChatGPTを擁するOpenAIが長らく君臨してきた「最大のAI企業」の座が、Claude（クロード）を開発するアンソロピックに奪われようとしている。

報じられた資金調達規模は500億ドル（約7.9兆円）、評価額は9000億ドル（約141兆円）。人類の企業史上、これほどの規模でプライベート企業が資金を集めた前例はない。今回はこの数字の裏に何があるのか、できる限り丁寧に解説したい。

  

### 「OpenAI超え」とは何を意味するのか

OpenAIの評価額は2026年3月時点で**8520億ドル（約133兆円）**。アンソロピックが今回のラウンドを9000億ドルで成立させれば、単純にこれを上回る計算になる。

さらに興味深いのは、株式のセカンダリーマーケット（既存株主が売り出す二次流通市場）では、アンソロピックの評価額がすでに1兆ドル（約157兆円）に到達しているという事実だ。市場の期待値では、すでにOpenAIを追い抜いている。

この高評価を支えているのは、期待ではなく実績であり、アンソロピックのARR（年間換算売上）の推移を追うと、その異常さがよくわかる。

![](https://assets.st-note.com/img/1778026219-QulTpDXmsSIf15qxey0j2cnB.png?width=1200)

わずか5ヶ月で売上が5倍近くに膨らんでいる。ある投資家は200社以上のIPOデータを調査した上で、「この規模でこれほどの成長率を見たことがない」と発言している。

### 成長の火付け役はClaude Code

この急成長の核心にあるのが、2025年後半から爆発的に普及したClaude Codeだ。Claude Codeはエンジニア向けのAIコーディングアシスタントで、単なる補完ツールを超え、プロジェクト全体の設計・実装をAIと共同で進められるレベルに達している。

* 2025年11月：Claude CodeのARRが10億ドルに到達（ChatGPTより速く、エンタープライズソフトウェア史上最速）
* 2026年2月：Claude CodeのARRが25億ドル超
* 2026年1月以降：週間アクティブユーザーが2倍に

エンジニアリングの生産性を根本から変えるツールとして評価されており、エンタープライズへの浸透が加速している。フォーチュン10企業（米国最大手10社）のうち8社がすでに顧客であり、年間100万ドル以上を支出する法人アカウントは1000社超に上る。

### 5年で690倍という異次元の成長

アンソロピックは2021年、OpenAI共同創業者のダリオ・アモデイら元OpenAI幹部が設立した会社だ。当時の評価額は5.5億ドルだった。それが今や9000億ドルの調達を目指している。わずか5年足らずで、評価額は約690倍になった計算だ。

今回のラウンドは、IPO（株式公開）前の最後のプライベート調達になる可能性が高いと見られている。IPOは2026年10月を目標にゴールドマン・サックスとJPモルガンが主幹事として準備を進めているという報道もある。

### CodexはClaude codeに追いつけるのか？

エンタープライズAI市場におけるシェアを見ると、アンソロピックのOpenAIに対する相対シェアは、2025年初頭の10%から2026年2月時点で65%超にまで逆転している。この数字は、企業が「使うAI」の選択を現時点では変えてしまった事を示している。OpenAIもCodexというコード補完AIを開発していたが、Claude Codeの追い上げで2026年に入ってからは「複数エージェントを並列で走らせて、長時間タスクを管理するデスクトップ常駐型AI」に変貌した。さらにOpenAI自身も “Codex for (almost) everything” って言い方をしていて、もう「コーディング専用」から離れ始めてる。
