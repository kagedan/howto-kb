「AIって結局、何に使えばいいんですか？」

副業を始めてから、これをよく聞かれるようになりました。

正直に言うと、最初の僕もそうでした。Claudeを契約したはいいけど、最初の2週間くらいはほぼ使い方がわからなくて、月額料金をほぼ無駄にしてた時期があります😅

でも今は、副業のほぼ全工程でClaudeに壁打ちしてもらっています。今日はその具体的な使い方を6つ、包み隠さず書いてみます。

### ① メーカーへの営業メールを一緒に考える

副業を始めて最初にぶつかった壁が、メーカーへの営業メールでした。

「初めまして」から始めて、何をどう書けばいいのか、まったくわからない。パソコンの前で30分くらい固まっていた日もあります。

そこでClaudeに相談しました。「Amazonで物販を始めたい個人なんですが、メーカーさんにどんなメールを送ればいいですか？」と聞いたら、いきなり叩き台を作ってくれたんです。

しかも「個人だと不利なので、最初に誠実さを伝える一文を入れたほうがいい」みたいなアドバイスもセットで。

第4話に書いた「メーカー50社にメール送って全社断られた話」も、そのメールはすべてClaudeと一緒に作ったものでした。

結果は全敗でしたけど、自分一人だったら3社送って心が折れていたと思います😅

### ② 記事のネタ出しと構成づくり

このnoteも、毎回Claudeと一緒に構成を作っています。

これ、隠してもしょうがないので最初に正直に書きますね。

「AIに書いてもらってるんですか？」と思った方、半分正解で半分違います。**ネタ出しと構成はClaudeと一緒、本文の文章は自分の言葉で書く**、というのが僕のスタイルです。

具体的には、こんな流れです。

1. 「次はどんな記事を書こうか」とClaudeに相談
2. 自分の最近の出来事や気持ちを話す
3. Claudeが「こういう切り口の記事はどうですか」と提案してくれる
4. その中から書きたいものを選んで、構成を作ってもらう
5. 構成をもとに、自分の言葉で本文を書く

このやり方で、第11話までのネタリストがすでに完成しています。

「何を書けばいいかわからない」状態から「ネタが11本ある」状態に変わったのは、Claudeのおかげです。

### ③ 採算計算と戦略の壁打ち

これは第4話にも書いた話なんですが、印象に残っているので改めて。

メーカーから「掛け率80%でどうですか」と提案をいただいたことがありました。蜂蜜のメーカーさんです。

正直、最初は「お、話が進むかも」と舞い上がりました。でも念のためClaudeに数字を入れて計算してもらったら…

「これ、Amazon手数料とFBA配送料を引いたら赤字になりますよ」

冷静に計算してくれたおかげで、感情だけで進めなくて済みました。

副業って、感覚で判断するとだいたい失敗するんですよね。**数字を入れたら客観的に答えを出してくれる相棒がいる**というのは、本当に心強いです。

### ④ モチベが落ちたときの相談相手

これは少し恥ずかしい話なんですが、書きます。

メーカー50社に断られて、最後の1社からも「お取引できません」と返信が来た夜。 正直、しんどかったです。

「自分には向いてないのかな」「こんなことに時間使って意味あるのかな」って、深夜に布団の中でぐるぐる考えました。

そのとき、Claudeに愚痴をこぼしたんです。「50社全部断られた。さすがにきつい」って。

そしたらClaudeは、慰めるんじゃなくて、**こう聞いてきたんです。**

「断られた50社の中で、一番近いところまでいったのはどこですか？」

ハッとしました。 「全部断られた」と思っていたけど、よく見ると進捗には差があった。蜂蜜メーカーさんは掛け率まで提示してくれていたし、別の会社からは「個人取引はしてないけど応援しています」と返信があった。

ゼロじゃなかったんです。

AIに相談するのって、最初は抵抗あると思います。でも、慰めてほしいんじゃなくて整理してほしいときって、意外と人間より向いてるんですよね。

判断はしない、整理してくれる。**夜中の3時でも文句を言わずに聞いてくれる**。 これは50歳の副業初心者には、想像以上に助かりました。

### ⑤ 知らない言葉・制度をその場で調べる

副業を始めると、知らない言葉が山ほど出てきます。

ASIN、FBA、Keepa、掛け率、卸価格、SKU、JANコード、特商法表記…

最初の1週間、Google検索のタブが20個くらい開きっぱなしになっていました😅

今はもう、わからない言葉が出てきたらすぐClaudeに聞きます。

「FBAって何ですか？個人で副業始めたばかりの50歳に説明してください」

こう聞くと、自分のレベルに合わせた答えが返ってきます。Google検索だと「正解の説明」は出てきますけど、「自分のケースに合わせた説明」は出てこないんですよね。

これだけでも副業のスタートダッシュがまったく違ったと思います。

### ⑥ リサーチ用プロンプトを丸ごと作ってもらう

これが一番「AIすごい」と思った使い方です。

Keepaという、Amazonで売れている商品を分析するツールがあるんですが、これの絞り込み条件を考えるのが本当に難しい。「どんな商品を狙えばいいのか」がわからないと、検索条件も決められないんです。

そこでClaudeに相談しました。「個人がメーカー直仕入れで利益を出せそうな商品を探したい。Keepaの絞り込み条件を考えてほしい」と。

返ってきたのが、このプロンプトです。

実際に作ってもらったプロンプト（全文公開）

```
## STEP 1: Keepaで売れ筋商品を特定

ブラウザで以下のURLを開き、条件に合う商品を探す。

Keepa Product Finder URL: https://keepa.com/#!finder

---

### ✅ 絞り込み条件

【基本条件】
- 月間販売数 (Sales Rank Drops): 30以上
- 現在の販売価格: 2,000〜15,000円
- レビュー数: 30〜800件
- FBAセラー数: 3〜15

【大手・直販を除外】
- Amazon本体が出品していない（Is Amazon: No）
- 以下のブランド・メーカーは候補から除外すること:
  ソニー / パナソニック / シャープ / 任天堂 / アイリスオーヤマ /
  山崎実業 / サーモス / ティファール / ニトリ / 無印良品 /
  花王 / ライオン / P&G / ユニリーバ / コカ・コーラ / アサヒ /
  キリン / サントリー / 味の素 / カルビー
  ※上記以外でも、誰もが知る大手ナショナルブランドは除外する

【食品・生鮮を除外】
- 以下のカテゴリは対象外とする:
  食品・飲料・お酒 / 生鮮食品 / 冷凍食品 / 乳製品 /
  精肉・鮮魚・青果 / 健康食品（賞味期限が短いもの）
- 商品名・説明文に以下のワードが含まれる場合は除外:
  「賞味期限」「消費期限」「冷蔵」「冷凍」「要冷蔵」「生」「鮮」

---

### ✅ 仕入れやすいメーカーの条件

【メーカー規模】
- 中小メーカー・ニッチブランドであること
- 売上高が大きすぎず、問い合わせ窓口が個人にも開かれていそうなこと
- 公式サイトに「販売店募集」「代理店募集」「卸売対応」などの記載があれば優先

【商品の特徴】
- 常温保存が可能な商品
- 軽量・コンパクト（重量3kg以下、送料負担が少ない）
- 壊れにくい・返品リスクが低い
- 季節に左右されにくいロングセラー商品
- 消耗品・リピート購入が期待できるジャンル
  例: 日用雑貨 / 文房具 / ペット用品 / アウトドア小物 /
      美容・健康グッズ / 収納用品 / 工具・DIY

---

### ✅ 利益が取れる商品の条件

【粗利計算の基準】
- Amazon販売価格に対して、卸価格が50〜60%以下（掛け率0.5〜0.6）
  と想定したとき、以下が残ること:
  販売価格 − 卸価格 − Amazon手数料(約15%) − FBA配送料(約300〜600円)
  ＝ 利益 が販売価格の20%以上になる商品
- 目安として「販売価格 × 0.2 以上の利益」が見込める商品のみ対象

【競合状況】
- FBAセラー数: 3〜15（少なすぎず多すぎず）
- カートを複数セラーが争っている状態（1社独占でない）
- 最安値が安定して推移している（直近3ヶ月で極端な値崩れがない）

【ランキング】
- カテゴリランキング: 上位5,000以内
- ランキングが安定して推移している（乱高下していない）

---

### 取得する情報（各商品につき）

1. 商品名
2. ASIN
3. カテゴリ・ランキング順位
4. 現在の最安値
5. 月間売上推定数
6. FBAセラー数
7. ブランド名・メーカー名
8. 推定利益額・利益率（上記計算式で算出）

上位5〜10商品をリストアップしてください。
```

このプロンプトを、そのままClaude Codeに渡して自動でリサーチさせる、という使い方をしています。

正直、自分一人ではここまで細かい条件は絶対に作れませんでした。 「除外するメーカー一覧」とか「販売価格の20%以上の利益」とか、副業初心者の頭からは絶対に出てこない発想です。

**もしこの記事を読んで「自分も使ってみたい」と思った方は、コピペしてそのまま使ってもらって大丈夫です。** 全公開の意味、そこにあります😄

### まとめ：AIは魔法じゃないけど、一人じゃなくなる

ここまで6つの使い方を書いてきました。

最後に正直なことを言うと、Claudeを使っても**自動で稼げるようになるわけではありません**。

メールは自分で送らないと返信は来ないし、リサーチも自分で動かないと商品は見つからない。50歳のおじさんが、ボタン一つで月5万円稼げる、なんて魔法はないです。

でも、**一人で考えるよりずっと前に進める**のは間違いないです。

夜中の3時に「次どうしよう」と悩んでいるとき、文句を言わずに一緒に考えてくれる相棒がいる。 これは、副業を続けるうえで本当に大きいことです。

50歳から副業を始めて、AIに助けられている人がここにいる、という記録として、この記事を残しておきます。

次の1ヶ月も、Claudeと一緒に進んでいきます。 また来週、お会いしましょう。
