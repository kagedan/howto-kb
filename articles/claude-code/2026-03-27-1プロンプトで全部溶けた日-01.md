ガソリン満タンで出発したのに、最初の交差点で燃料計がゼロになったら、あんたどう思う？
Claude Code Max 20xプランで、まさにそれが起きてる。

 レート制限バグ、また来たで
3月23日あたりから、Max 20xプランのユーザーが悲鳴を上げてる。1プロンプト送っただけで、使用量が21%から一気に100%に跳ね上がる。5時間の利用枠が、たった1-2回のやりとりで蒸発するんや。
GitHub Issueには報告が殺到してる。#38335では「以前は数十回やりとりできてたのに、今は1-2回で終わる」、#38345では「$100/月払ってるのにプロンプト数回で5時間ロックアウト」...
