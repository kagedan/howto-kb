ガソリン満タンで出発したのに、最初の交差点で燃料計がゼロになったら、あんたどう思う？

Claude Code Max 20xプランで、まさにそれが起きてる。

## レート制限バグ、また来たで

3月23日あたりから、Max 20xプランのユーザーが悲鳴を上げてる。**1プロンプト送っただけで、使用量が21%から一気に100%に跳ね上がる**。5時間の利用枠が、たった1-2回のやりとりで蒸発するんや。

GitHub Issueには報告が殺到してる。[#38335](https://github.com/anthropics/claude-code/issues/38335)では「以前は数十回やりとりできてたのに、今は1-2回で終わる」、[#38345](https://github.com/anthropics/claude-code/issues/38345)では「$100/月払ってるのにプロンプト数回で5時間ロックアウト」と。

ちょっと待って。これ、**2月末にも全く同じことが起きてる**んよ。あのときはリセット対応で収まったけど、3月23日からまた再発。同じバグが1ヶ月で2回。地味やけどデカい問題やで。

> **Before**: 5時間の枠で数十回のやりとりができた  
> **After**: 同じ枠が1-2プロンプトで消滅

Anthropicからの公式コメントは、今のところ出てへん。ぼくの見立てでは、サブエージェント（Agent tool）の並列実行や1Mコンテキストウインドウのトークン消費が、レート制限の計算式と噛み合ってない可能性がある。でもそれにしても「1回で全部」は異常や。

**で、どうすればいいの？** 今はとにかく、重いタスクを投げる前にこまめにコンテキストを切る（`/clear`）のが自衛策。Issue にリアクションを付けて声を上げるのも大事。

## ダブル使用量プロモ、明日で終了

もう一つ忘れたらあかんやつ。3月13日から走ってた[ダブル使用量プロモーション](https://support.claude.com/en/articles/14063676-claude-march-2026-usage-promotion)が**明日3月28日で終わる**（PT 23:59まで）。週末からは通常の使用量に戻る。

レート制限バグの最中にプロモも終わるっていう、なかなかのタイミングやね。プロモ中でこの消費速度やったら、通常に戻ったらどうなるんやって話。Issue の動向は要チェックや。

## 怒涛の3日連続リリース (v2.1.83-85)

バグの話ばっかりやと気が滅入るから、明るいニュースもいこう。3月25-26日の間に**3バージョン連続リリース**。Claude Codeチーム、めちゃくちゃ走ってる。

### v2.1.85 — Hooksに「if」が来た

Hooksに条件付き`if`フィールドが追加された（[リリースノート](https://github.com/anthropics/claude-code/releases/tag/v2.1.85)）。パーミッションルール構文（`Bash(git *)`みたいな書き方）でフックの発火条件を絞れるようになった。

これ、料理で言うたら「全部の皿に醤油かけてた」のが「刺身のときだけ醤油」にできるようになったようなもん。プロセス生成のオーバーヘッドが減るから、体感速度にも効いてくる。

MCP OAuthもRFC 9728準拠になった。地味やけど、企業導入のハードルがまた一つ下がった。

### v2.1.84 — Windows PowerShellツール

[Windows PowerShellツール](https://github.com/anthropics/claude-code/releases/tag/v2.1.84)がオプトインプレビューで登場。Windows勢、おめでとう。`TaskCreated`フックも追加されて、タスク作成時に自動で何かを走らせるワークフローが組めるようになった。

75分以上離席してから戻ると「`/clear`したほうがええで」って促してくれるidle-returnプロンプトも追加。これ、さっきのレート制限バグの自衛策としてもタイムリーやな。

### v2.1.83 — managed-settings.d/ とトランスクリプト検索

[managed-settings.d/](https://github.com/anthropics/claude-code/releases/tag/v2.1.83)ディレクトリで、チームごとにポリシーを分割管理できるようになった。`FileChanged`/`CwdChanged`フックも追加されて、direnvみたいなリアクティブ環境管理ができる。

個人的に一番嬉しいのは**トランスクリプト検索**。`Ctrl+O`でトランスクリプト開いて、`/`で検索、`n`/`N`でジャンプ。「あの会話どこやったっけ」問題が解決する。

## 今日のまとめ

2日で3バージョン出す開発速度は正直すごい。でも、Maxプランが1プロンプトで溶けるバグが1ヶ月で2回起きるのも事実。アクセルとブレーキを同時に踏んでる感じがする。

あんたのMaxプラン、今日ちゃんと5時間もってる？
