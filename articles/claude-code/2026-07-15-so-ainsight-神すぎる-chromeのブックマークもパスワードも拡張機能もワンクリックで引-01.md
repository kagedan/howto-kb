神すぎる。

Chromeのブックマークもパスワードも拡張機能もワンクリックで引き継いで、そのブラウザが自分でタブを開き、フォームを埋め、調べ物を終わらせる。

BrowserOS、もう完全にAIブラウザの決定版になった件。

2026年7月時点の全機能やばすぎる↓
・AIエージェント内蔵（53種類以上の操作を、話し言葉の指示で実行）
・Cowork（Webで調べた内容を、そのままパソコンのフォルダに保存）
・スケジュール実行（毎朝の情報収集を自動操縦）
・Claude CodeやGemini CLIから、このブラウザを操作できる
・Gmail、Slack、Notionなど40以上のアプリ連携

GitHubスター1.2万・完全無料の全機能を解説する↓

まず、BrowserOSとは何か。

ひと言で説明すると↓
「AIエージェントが最初から内蔵された、中身はChromeと同じブラウザ」

【規模感】
・GitHubスター 1.2万
・完全無料（ライセンスはAGPL v3、ソースコード全公開）
・macOS / Windows / Linux 対応
・つなげるAIは11種類以上（Claude、GPT、Geminiから、パソコン内で動くAIまで）

作ったのは、シリコンバレーの登竜門Y Combinator出身のエンジニア2人。YouTubeの推薦AIを作っていた人と、Instagramのリールの裏側を作っていた人。

もはや単なるブラウザじゃなく「AIが常駐する作業机」レベル。

【機能①】AIエージェント：ブラウザ操作を話し言葉で丸投げ。

「航空券を調べて予約まで進めて」のような、手を動かす作業に特化。

【機能】
・53種類以上の操作ツール（ページを開く、クリック、入力、データ抽出）
・ログイン済みの状態をそのまま使うので、認証をやり直さなくていい
・処理は目の前のタブで進むので、様子を見ながら途中で止められる
・フォーム入力やリサーチなど、手順が決まった仕事を丸ごと任せられる

つまり「自分のブラウザの中で動く、操作の代行者」。
拡張機能でもクラウド上のロボットでもなく、ブラウザ自身が動く。

【機能②】Cowork：調べる係と書く係が、同じ指示の中にいる。

リサーチしてレポートに落とす作業に特化。

【機能】
・Webの情報収集と、パソコン内のファイル読み書きを1つの指示で完結
・調べた結果をそのまま自分のフォルダに保存できる
・アプリ間をコピペで往復する手間が消える

つまり「調べる→まとめる→保存する」が一筆書きになる。
ブラウザとパソコンの間の壁が無くなる感覚。

【機能③】スケジュール実行：エージェントを自動操縦に。

毎日くり返す定型作業に特化。

【機能】
・毎日、1時間ごと、数分ごとの間隔で指示を自動実行
・BrowserOSを開いたままにしておけば、情報収集や価格チェックが夜のうちに終わっている
・サインインすれば、チャット履歴や設定を別のパソコンとも同期できる（内容はクラウドに上がる。APIキーだけは同期されず手元に残る）

つまり「気づいたら終わっている」タイプの自動化。
立ち上げっぱなしにしておくだけで、エージェントが勝手にページを開いて、勝手に調べて、保存まで無人で回る。

【機能④】MCPサーバー内蔵：外のAIから、このブラウザを操作できる。

MCPは、AIに道具を持たせるための共通の差込口。

【機能】
・ブラウザ自体が「AIの道具箱」として公開される
・Claude CodeやGemini CLIから「このページを開いて調べて」と指示できる
・MCP対応のAIツールなら何でもつながる
・Gmail、Slack、GitHub、Linear、Notion、Figma、Salesforceなど40以上のアプリ連携もこの仕組み経由

つまり「AIに手渡せるブラウザ」。
コードを書くAIに、目と手が生える。

【最大の差別化】自分の手元で完結する。

ChatGPT AtlasやPerplexity Cometといった大手のAIブラウザの、プライバシー最優先版という立ち位置。

【仕組み】
自分のAPIキーを入れる→エージェントはブラウザの中で動く→閲覧履歴を広告や検索のために集める会社の持ち物ではない

【特徴】
・手持ちのClaude／GPT／Geminiのキーを差して使える
・OllamaやLM Studioを使えば、ページの要約や質問はパソコンの中のAIだけで完結（API料金ゼロ。サインインしなければ、そのデータはパソコンの中に留まる）
・ただし公式いわく、ローカルAIは自動操作にはまだ力不足。エージェントを走らせるならクラウドのAI推奨
・ソースコードが全公開なので、中で何が起きてい
