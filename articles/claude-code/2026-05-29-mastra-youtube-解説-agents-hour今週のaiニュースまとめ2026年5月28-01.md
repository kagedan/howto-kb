## 概要

メモリアルデー明けの週、通常の月曜日配信を水曜日にずらして放送した Agents Hour。Shane Thomas と Obby Ayer（ともにMastraの共同創業者）が、サウスダコタ州のブラックヒルズ旅行から戻りつつ、週次のAIニュースをテンポよく紹介しました。

主なテーマは「モデル単体ではなくハーネス・プラットフォームが勝負どころになった」という認識の共有と、Anthropic・Google・OpenAIそれぞれの動向、サプライチェーン攻撃の連続発生、そして各社の資金調達ニュースです。

## 詳細

### Andre Karpathy、Anthropicへ

Andre Karpathy が Anthropic への入社を表明したツイートは2,710万回表示を記録しました。ホストたちは「才能だけでなく、著名人としてのブランド効果も大きい」と見ています。OpenAI共同創業者・テスラオートパイロット指揮・「バイブコーディング」命名者という経歴を持つ彼の加入は、Anthropicに「最高の人材がいる」というシグナルを業界に送る効果があります。

今回の件に合わせて、AIで生成したオフィスのパロディ動画も紹介されました。新入社員として Karpathy を紹介する場面が、カメラ目線や間の取り方も含めてオフィスのユーモアを忠実に再現しており、完成度が高いと好評でした。

### モデル単体はもはやプロダクトではない

Greg Brockman のツイートを起点に、ホストたちはフレームワーク・ハーネス・プラットフォームの重要性について議論しました。

Shane は「過去6ヶ月、11月以来の大きな飛躍は感じられていない」と述べ、モデル性能の向上が鈍化しつつあるとすれば、ハーネスで差がつく時代になると指摘しています。視聴者の Carlos のコメント「同じモデルでもハーネスが違えばパフォーマンスが異なる」もこれを裏付けます。

一方で、モデル開発が仮に止まっても、まだ膨大な未実装の可能性が残っている。「フロンティアが落ち着くほど基盤が安定し、何ができるかを自信を持って見極められるようになる」とも語っています。

各モデルラボがマネージドエージェント・コーディングエージェントを一斉に出し始めた動きも、この「モデルの周辺」競争の表れといえます。

### OpenAI：YCスタートアップへ$200万のトークンオファー

Tyler（YC）によると、Sam Altman が Y Combinator の現バッチ全スタートアップに対し、エクイティ（約2%）と引き換えに$200万分のOpenAIトークンをオファーしました。Sam Altman は「スタートアップがトークンをフル活用したとき、内部の働き方とプロダクト双方でどうなるかを見るのが楽しみ」とコメント。

Shane と Obby はYC卒業生ですが、当時のクレジットは約3万ドル。$200万はその67倍の規模です。エンドユーザーがトークンを大量消費するビジネスモデルであれば、ライフタイム利用可能という条件付きで非常に有利な取引だとホストたちは見ています。一方で「2%のエクイティは安くはない」とも述べており、条件次第というのが正直なところのようです。

### o.md：エージェント向けオープンプロトコル

WorkOS の Michael Grenwich（数週間前に番組ゲスト）が発表した o.md は、エージェントがウェブ上のサービスに登録・認証するためのオープンプロトコルです。Cloudflare・Firecrawl・Resend が初期プロバイダーとして参画、Mastra も承認待ちでまもなくプロバイダーになる予定とのことです。

WorkOS が提供する Auth Kit（OKit）との自然な統合が強みで、OKitを使っていれば追加実装がほぼ不要。MCP Night という注目度の高いイベントでの発表も相まって、良いスタートを切ったとホストたちは評価しています。

### 中国モデルの台頭

DeepSeek は一時的な値引きキャンペーンとしていた75%オフを永続化しました。既に激安だった DeepSeek V4 Pro がさらに安くなり、10億トークン$9という水準に。オープンソースのKVキャッシュ計算ツールで解析したところ、コスト競争力の源泉はキャッシング効率の高さにあることが確認できます。

Qwen 3.7 Max（5月21日）は複数のベンチマークで Opus 4.6 を上回る結果を示し、競争力のある中国モデルの存在感が高まっています。Alibaba からは5月20日以降毎週 Mastra にコンタクトが来ているとのこと。

MiniMax のスパースアテンション技術は、処理コストを指数的でなく線形にスケールさせる可能性を持ちます。「データを中国に送りたくない場合は Fireworks など認可プロバイダー経由で利用できる」というのがホストたちの現実的な提案です。

### Anthropic × SpaceX：月12.5億ドルのコンピュート契約

Anthropicは現在、SpaceXのコロッサス（Colossus）クラスターに月12.5億ドルを支払っているとされています。Tom Brown（Anthropic）は「Colossus 2のGB200キャパシティを6月中にスケールアップ予定」と発表しました。

将来的に競合になりうる相手にインフラを依存する戦略について、ホストたちは「SpaceXはモデル競争で勝たなくてもインフラレイヤーで勝てる」と整理しています。軌道上データセンターが実現した場合、大量の質量を軌道に打ち上げられるのは現状 SpaceX のみ。Starship の同日再打ち上げを目指す設計が実現すれば、その優位性はさらに強固になります。

「データセンターを国内に置きたくない声もあるが、ならば宇宙に置けばいい」という逆説的な解決策として興味深い見方も出ました。

### コーディングエージェントベンチマーク

DeepSWE は「実際の開発者の日常作業」を模したベンチマークです。Ansible や Chef などのツールを使って実際のゴールを達成するタスクを評価します。GPT-5.5 と GPT-5.4 が上位、その下に Opus 4.7 が続きます。Terminal Bench が「ランダムで非現実的なコンピュータサイエンス問題」に見えることと対比し、より実践的な評価指標として注目されています。

Cursor Bench 3.1 ではコストを評価軸に加えており、Opus 4.7 Max（65%、高コスト）vs Composer 2.5（〜64%、低コスト）という比較が示されました。「本番でOpusを常用すると資金が尽きる」という現実的なコスト意識が業界全体で高まっているとホストたちは指摘しています。

### Claude / Anthropic のアップデート

* **セルフホステッドサンドボックス＋MCPトンネル**（リサーチプレビュー）：自社のセキュリティコントロールを適用しながら、エージェントを社内ペリメーター内で実行可能
* **/workflows in Claude Code**：変更ログには記載されたが実際にはリリースされていない状態。ワークフローファイルでコントロールフローをコードに任せ、LLMは判断のみを担う設計（Mastraのワークフローエンジンと近い概念）
* **トークン使用量の詳細内訳**（次バージョン予定）：スキル・エージェント・MCP・プラグインごとの利用状況を確認可能に
* **セキュリティガイダンスプラグイン**：コーディング中にリアルタイムで脆弱性を特定・修正支援

なお Anthropicの告知が最近「Claude Devs」アカウントから行われることが増えたという観察について、Obby は「本体アカウントが炎上を避けているのかも」と推測しています。

### Google の週

*Gemini 3.5 Flash* は旧世代のGemini 3 Flashに比べて価格が3倍に上昇し、Theo は「Gemini 3.1 Proより2倍のコスト、GPT-55 mediumより高い」と指摘しています。

*Gemini Omni* は動画をあらゆる入力から生成できる新モデルで、YouTube での利用も開始。しかし Sora Dance 2.0 と並べると物理法則が成立しない場面が目立ち、「PlayStation 1のゲームのよう」と評されました。

*Anti-gravity CLI* はGo言語で実装されたコーディングエージェント向けCLIで、Gemini CLI を置き換える可能性があります。内部の複数チームの競合バトル（ベイクオフ）に勝ったバージョンとも見られています。

最も注目を集めたのは *Google検索の25年ぶり最大のアップデート*。Gemini 3.5 Flash を全ユーザーのデフォルトモデルに採用し、検索体験をAIチャット形式に刷新しました。Shane 自身が「近くのレストランを探した際、検索結果ではなくGeminiとの会話になり、地図リンクも含めてほしいものが全部得られた」と実体験を共有しています。

### サプライチェーン攻撃の連続

1週間の間に複数の攻撃が重なりました：

* npm での mini-shade関連パッケージへの悪意あるコード埋め込み（5月18日）
* npm・PyPI・crates.io に34パッケージ・384バージョンに及ぶ暗号通貨スティーラー（5月24日）
* GitHubの5,561リポジトリへの6時間で5,718件の悪意あるコミット（「メガロドンキャンペーン」、5月21日）
* GitHubが内部リポジトリへの不正アクセスを調査中と発表（5月19日）

Cloudflare はAnthropicの Mythos を使い自社50リポジトリへのオフェンシブAI評価を実施。「自分たちが常に攻撃されていると仮定することで、セキュリティ体制が根本から変わる」という知見を公開しました。Perplexity の Bumblebee は開発者マシン向けの読み取り専用スキャナーで、新しいアンチウイルスのような位置づけです。

Nvidia は検証済みエージェントスキル（Verified Agent Skills）を発表。スキルカードで出所・リスク・改ざん有無を確認できる仕組みで、agent-skills.io のオープン仕様に準拠しています。

### MCP 2026 RC：ステートレス化

7月28日リリース候補となったMCP 2026では、プロトコルがステートレス化されます。ハンドシェイクなし・セッションIDなし・任意のリクエストが任意のサーバーインスタンスに届く設計へと変わります。タスクのファーストクラス対応と、適切な非推奨ポリシーの整備も含まれます。

### 音声・音楽・動画

11 Labs の Music V2 は、V1 が競合（Suno等）に見劣りしていたことを受けたアップデート。来週の番組でSunoとの比較テストを実施予定とのことです。同社の Speech Engine は既存のチャットエージェントを1プロンプトで音声エージェントに変換できます。

Runway の ALF 2.0 は動画内の1フレームを編集し、その変更を動画全体に伝播させる機能を実装。「Runwayは革新的な機能を最初に出すが、後から他社がより豊富な機能セットで追いついてくる」という見方が共有されました。

### 資金調達

* **EXA**：エージェント向けウェブデータ整理サービスがSeries Cで2億5,000万ドル（バリュエーション22億ドル、A16Z主導）
* **Open Router**：APIゲートウェイがSeries Bで1,300万ドル調達。過去6ヶ月でウィークリーボリュームが5兆→25兆トークンへ5倍増

Open Router の5倍成長は「AIが実験から本番移行する加速」の象徴だとホストたちは見ています。

### その他ラピッドファイアニュース

* **browse.sh（Browserbase）**：エージェント向け最大級のオープンソーススキルカタログ
* **Cerebras + Kimi K2.6**：1兆パラメータモデルを毎秒1,000トークンで実行。Frontier最速を主張（Cerebrasは5月14日にIPO済み）
* **DataDog Lapdog**：Codex・Claude Code等での推論とツールコールをローカルでリアルタイムトレース
* **OpenAI の数学的発見**：1946年以来未解決のポール・エルデシュ「平面単位距離問題」を自律的に解決。「正方グリッドが最適」という80年来の仮説を覆す新構造族を発見
* **Polar**：既存エージェントハーネスをそのままRL（強化学習）のトレーニング環境として利用できるインフラ
* **Devon 自動トリアージ（Cognition）**：長期メモリを持ち、バグ・アラート・インシデントを自律的に調査してPRを返す「AIファーストレスポンダー」
