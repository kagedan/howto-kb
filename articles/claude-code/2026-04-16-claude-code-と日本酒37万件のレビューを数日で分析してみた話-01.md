<p name="ffc2be85-fe17-4d77-9244-a99dba5405ad" id="ffc2be85-fe17-4d77-9244-a99dba5405ad">先週の土曜日、日本酒のYouTube動画を観ていたら、「SAKETIMEというレビューサイトのランキング上位は、甘旨ジューシー系の酒ばかり」という話が出てきた。言われてみれば確かにそうかもしれない。でも、「日本酒は辛口に限る」と昔は言われていたはず。本当に辛口の時代は終わったのだろうか。</p><p name="94d37ba8-c02c-4d9c-8713-f8a959a18e27" id="94d37ba8-c02c-4d9c-8713-f8a959a18e27">漠然とした違和感のまま、翌日曜日に Claude Code を開いて「SAKETIMEをスクレイピングして、上位が本当に甘口なのか調べたい」と相談してみた。そして水曜日には、5,386銘柄・約37万件のレビューを集め、国立国会図書館の出版物データとGoogle Trendsまでつなぎ合わせた、140年分のトレンド分析記事をNoteに公開していた。
