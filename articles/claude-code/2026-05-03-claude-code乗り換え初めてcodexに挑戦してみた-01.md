今日は、「Codex」についての投稿です。

Codexとは、OpenAI（ChatGPTを出してる会社）が提供している、ツール作成ソフトのことです。

「〇〇を作って」  
と指示を出すだけで、作りたいものが何でも作れる夢のソフトです。

Claude Codeでも同じようなことができますが、これからはCodexを使っていこうと思っています。

理由はシンプルで、**「使用制限までかなり余裕があるから」**です。  
私はChatGPTのPlusプラン（月額3000円）に入っています。

今回紹介する「英語学習ツール」を作った後でもこの残量。

![](https://assets.st-note.com/img/1777766677-Wl0JEF1792GjNRaSuzTvgAhZ.png)

最新モデルの「GPT-5.5」を使っての数値です。

Claude Codeは最新モデル「Opus」を使うとすぐに制限がかかるので慎重にならざるを得ません。

心理的安全性が全く違います。

ちなみに今回作った「英語学習ツール」ですが、プロンプトはこんな感じです。

> 英語学習を進めるうえで使えるツールをweb上で公開したい。初心者から上級者までレベルに応じて学習方法を変えてほしい。最初に診断テストがあるといいね。英語をどんな場面で使いたいかを確認して、それに応じた問題を解くシステムがいい。Duolingoのように毎日勉強するとカウンターが表示されてレベルアップしている感じがユーザーに分かるといい。スピーキング・ライティング・リーディング、全てを網羅してほしい。

思いつくままに打った雑なプロンプトですが、この指示でこんな感じのツールが完成しました。

クリックはしたくないけど、画面は見たいという方はこちらをご覧ください。

もっとやり取りを続ければ、クオリティはさらに上がります。
