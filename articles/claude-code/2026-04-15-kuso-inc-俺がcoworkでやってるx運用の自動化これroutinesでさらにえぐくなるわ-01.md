PCを閉じてもClaude Codeが動き続ける
「Routines」機能が登場！！！これかなりデカい。
ㅤ
何ができるかというと、、、Claude Codeのセッションを「テンプレート化」して、3種類のトリガーで自動実行できるようになった。
https://t.co/3242w2hJOG
① Schedule — cronみたいに定期実行（毎時/毎日/毎週）
② GitHub Events — PRマージ、Push、Issue作成をトリガーにエージェントが起動
③ API — HTTP POSTで外部から起動（Zapier、アラート等なんでも）
④ しかもAnthropicのクラウドインフラで動くので、ノートPC閉じても止まらない。
ㅤ
自分が一番刺さったのはGitHubトリガーです。
PRマージ → ドキュメント自動生成とか、
Issue作成 → バグ調査自動開始とか。
