💡 たった10分でAIがコードを書いてくれる！

まずは、みんなが一番身近なAIを使って、簡単なコードを生成してみよう！

ここでは、OpenAIのChatGPT（最新のGPT-5系モデル）か、AnthropicのClaude（最新のClaude Sonnet 4.xモデル）のWeb UIを使うねっ。どちらもブラウザでサクッと使えるから安心だよ！

📌 ステップ1：AIに「何をしたいか」伝える！

これは簡単！普段チャットするように、AIに『どんなコードを書いてほしいか』を自然な言葉で伝えるだけだよっ。

例えば、こんなプロンプトをコピペしてみてね。

```

Pythonで、今日の日付と時刻を「YYYY年MM月DD日 HH時MM分SS秒」の形式で表示するスクリプトを書いて。

このスクリプトは実行するだけでいいよ。

```

GPT-5系やClaude Sonnet 4.xなら、すぐにこんな感じのPythonコードを生成してくれるはずだよっ。

```python

import datetime

現在の日付と時刻を取得

now = datetime.datetime.now()

指定された形式で表示

print(now.strftime("%Y年%m月%d日 %H時%M分%S秒"))

```

ね、とっても簡単でしょ？

📌 ステップ2：AIが書いたコードを実行してみよっ！

AIが生成してくれたコードを、あなたのパソコンで動かしてみるよっ。ターミナル（WindowsならコマンドプロンプトやPowerShell、Macならターミナル）を開いてみてね。

1. まず、AIが生成したPythonコードをコピーして、テキストエディタ（Windowsならメモ帳、Macならテキストエディット、またはVS Codeなど）に貼り付けてね。

    その後、『ファイル』メニューから『名前を付けて保存』を選んで、デスクトップなど分かりやすい場所に`get\_date.py`という名前で保存するの。

2. 保存したファイルがあるディレクトリにターミナルで移動するよっ。

    例えば、デスクトップに保存したなら、Windowsなら`cd C:\Users\あなたのユーザー名\Desktop`、Macなら`cd ~/Desktop`って入力してEnterキーを押してね。

    その後、  
`python get\_date.py`って入力して実行するだけ！

ほら、ちゃんと今日の日付と時刻が表示されたでしょ？😳
