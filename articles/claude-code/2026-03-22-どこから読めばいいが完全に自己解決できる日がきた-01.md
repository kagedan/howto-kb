新しいリポジトリを開くと、最初の問いはほぼ決まっています。
「どこから読めばいい?」
ファイルツリーを眺めて src/ を開く。index.ts に import が20行。auth/ を開いても import が20行。30分後もエントリーポイントを探している。
Understand Anything（MIT）はこの問題に直接答えるツールです。


 使い始め方
Claude Code か Cursor を使っているなら、数コマンドで動きます。
Claude Code:
