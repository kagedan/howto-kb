## ブラウザを開かずにAIに画像を作らせる方法

AIに画像を作らせる作業は、自動化されてるようで、実はけっこう手作業です。

プロンプトの貼り付け、生成、ダウンロード、保存先への移動。全部手でやってます。

「APIを叩けばいいじゃん」と思うかもしれません。

確かにAPIを使えば自動化はできます。  
でも画像生成はガチャです。一発でイメージ通りに出ることは、ほぼありません。

何度も生成し直すうちに、APIの利用料金がじわじわ積み上がっていく。  
気づいたら「あれ、今月いくら使ったっけ？」となります。

「自動化したいけど、コストが怖い」  
という方は結構多いのではないでしょうか？

## そもそもAI業界がカオスすぎる

少し前まで**「ClaudeがAGIに到達したのでは」**と海外で騒がれていました。

そう思っていたら、今度は**ChatGPT 5.5がImage 2というぶっ飛んだ画像生成**を引き連れて追い上げてきた。

文章の整理はClaude、画像はChatGPT、コーディングは同等みたいに道具を使い分けるのが普通になっています。

でもブラウザを行ったり来たりするのは、正直しんどい。

## API以外の道を探した

そこでAPIを使わずに自動化できないかを考えました。

ヒントになったのはClaudeの性質です。

実はClaude単体では、画像も動画も音楽も作れません。  
でもClaudeには別の強みがあります。

それは**「他のツールを呼び出すのが、異常にうまい」**こと。

SkillやCLAUDE.mdといったレシピやルールを作って動かさせるのもClaudeが作り出したものです。

画像はClaude本体に作らせるのではなく、別の道具に作らせて、Claudeに取りまとめてもらえばいい。  
そう考えるようになりました。

（ちなみにGeminiはスキルの呼び出しが下手です、スキル作ればどれでも良いってものでもなさそう）

## ChatGPTにも「Codex」がある

ChatGPTにもターミナルから動かせるCodexというツールがあります。

Claudeでいう「Claude Code」みたいな存在です。

このCodexの中に、画像を作る機能（imagegen）が入っています。

つまり**Claude Codeに「Codexのimagegenで画像作って」**とお願いすれば、勝手に画像を生成して保存してくれる。

実際に私がClaudeに送っている指示は、たったこれだけです。

> Codexのimagegenで「猫がコーヒーを飲んでる写真」作って

これでClaudeが裏でCodexを呼び出し、Codexがimagegenで画像を生成し、出来上がったファイルを所定の場所に置いてくれます。

出力した画像を見て「Image2だ！」と感じました。  
日本語もきちんと崩れず出力しているからです。

日本語の性能もバッチリ！

ターミナルのコマンドは一切打ちません。  
チャットに日本語で頼むだけです。

ちょっとした落とし穴もあります。

画像はCodexが管理する内部フォルダに勝手に保存されるので、最初は「どこに保存された？」となりがちです。  
  
これも「使いたい場所にコピーして」とClaudeにお願いすれば、勝手に持ってきてくれます。

## 一度仕組みにすれば、あとはラク

毎回このコマンドを打つのは面倒なので、Claudeの「スキル」に登録しました。

これで「画像作って」と一言伝えるだけで、勝手にClaudeがCodexに画像を作らせて、所定の場所に置いてくれます。

LPの素材、パワポのスライド画像、ブログの挿絵。

ぜんぶこれで作れるようになりました。

ちなみにClaude Code デスクトップでも使えます。

## まとめ

Claudeは画像、音楽、動画を作れない。

でもClaudeを中心に、他のAIツールを呼び出して連携させると、できることが一気に広がります。

AIは1つで全部やろうとすると、どれも中途半端になります。

得意な道具を、得意な場所で使う。

それを束ねるハブとしてClaudeを置く。

これが一番ラクで、一番早い使い方だと、強く感じています。
