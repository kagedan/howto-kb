Anthropicが「Claude Codeで会社を作るプレイブック」を無料公開していました🤖

30分で読める、Anthropic自身が書いた公式ドキュメント。スタートアップの立ち上げから拡大まで、AIツールをどう使うかが具体的に書かれています。

スタートアップを4フェーズに分けて、ツールの使い分けを整理しているのが特徴です。

・Idea（検証）→ Claude Chatで仮説を対話形式で詰める
・MVP（試作）→ Claude Codeでコードを生成、エンジニアなしで作れる
・Launch（リリース）→ Claude Coworkで業務フローをまるごと自動化
・Scale（拡大）→ 3つを組み合わせて「AIが回す組織」を設計

料理にたとえると、Claude Chatが「シェフと相談する人」、Claude Codeが「実際に調理する人」、Claude Coworkが「厨房全体を仕切るマネージャー」という感じです。

プレイブックの中で印象的だったのがこのフレーズ。

「創業者の役割は、自分でやる人から、やらせる仕組みを作る人へ移行した。」

コードを書けなくても、採用費がなくても、設計さえできれば事業が動く時代になっているというのが、このドキュメントの核心だと思います。
