毎日のように更新されている
Claude Desktopを開くと、こうなっていることが多い。
v2.○○○ を更新しました
最初は「おっ、新機能かな」と思って更新していた。でも気づいたら毎日のようにこの画面が出るようになって、最近は「またか」という気持ちで更新ボタンを押していた。
月曜日：「v2.○○を更新しました」→ 更新
火曜日：「v2.○○を更新しました」→ 更新
水曜日：「v2.○○を更新しました」→ 更新
木曜日：「v2.○○を更新しました」→ 更新
金曜日：「v2.○○を更新しました」→ 更新
土曜日：休日くらい休んで
日曜日：「v2.○○を更新しました」→ 更新
