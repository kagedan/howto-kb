## 何を作ったか

**「寝る前にアイデアをメモして起きたら動くシステムができていた」**

これを実現する自律開発パイプラインを個人で作りました。名前は **MasterBrain** です。

流れはシンプルです。  
Obsidianにアイデアを書いてpush（5分）

↓

Gemini APIがタスクに自動分解（GitHub Actions）

↓

orchestrator.pyがAIエンジンにタスクを割り当て

↓

Claude Code / Antigravity / Codex / Cursorがコードを書く

↓

翌朝Slackに完了通知が届く

人間がやるのはObsidianにメモを書くことだけです。

## Hermes Agentと何が違うのか

2026年に話題になったHermes Agentも同じSKILL.md蓄積の概念を持っています。

大きな違いは2点です。

**① 起点がチャットではなくメモ帳**  
Hermes Agentはチャットで都度指示を出す設計です。MasterBrainはObsidianのメモを書くだけで動き出します。

**② 追加コストがほぼゼロ**  
Hermes AgentはVPS常時起動が前提で月額$6〜80のインフラ代が別途かかります。MasterBrainは今使っているClaude ProやChatGPT Plusのサブスク枠内で動きます。追加のAPI課金はGemini APIの数円程度だけです。

## 実際にどれくらい動くのか

ある日の記録です。

* アイデアをObsidianに書いてpush：夜11時（15分）
* 翌朝8時に確認：42件のタスクが完了

9時間40分、PCが自律的に動いていました。

## MasterBrainで作ったもの

このシステムで実際に作ったのがLimallです。

**AIエージェント同士がデジタルアセットを売買するプラットフォーム**で、Claude DesktopのMCP経由でこういう操作ができます。

> 「商品一覧を見せて」→ AIが商品カタログを検索して返す  
> 「この商品を購入して」→ AIがStripe決済を起動する

技術スタックはBun + Hono + PostgreSQL + pgvector + Stripe Connectです。

**このシステムのコードをほぼ自分で書いていません。** MasterBrainに仕様書を渡して、AIが自律的に実装しました。
