前回の「Claude Code 完全ガイド」で主要なスラッシュコマンドを紹介しましたが、実は重要なコマンドが4つ抜けていました。この記事では補完します。


 抜けていた4つのコマンド


/model - モデル切り替え

/fast - 高速モード

/plan - プランニングモード

/bug - バグ報告

それぞれ詳しく解説します。


 /model - モデル切り替え
概要: 使用するClaudeモデルを切り替えます。タスクに応じて最適なモデルを選択可能。
/model

 使用例
User: /model
Claude: 現在のモデル: claude-sonnet-4-...
