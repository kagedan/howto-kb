Claude Codeで施工計画書ジェネレータ（RAG）を試作！
コーディングが先ほど終わって無料版を提供できる段階まで来た。
工種選んで、設計図書をアップロードするだけで、雛形が作れる仕様。
不足してる情報はウィザード形式で答えるだけで補完される。
勿論、工程表もバーチャートで出る。
Excel出力はカスタマイズで自社フォーマットも使える。
