OpenAIがCodexの大型アップデートを発表。AnthropicのClaude Coworkのガチ競合！w

Codexが非エンジニアの業務領域まで本格的に踏み込んできた。プラグイン、アノテーション、URLで共有できるSites。完全にCoworkと同じ土俵です。

さらに注目したいのが、MCPやSKILL.md的なagent skillsの設計を取り込んできた点。エージェントの拡張アーキテクチャが、かなりClaude側に寄ってきています。仕組みそのもので差別化する時代は、もう終わりつつある。

ここから先は、モデル性能・仕事をやり切る力・配布力・業務ツール連携の総合戦。その中でも本丸は配布力だと見ています。  Anthropicはこの未来をClaude Code / Coworkで先に描いていた。一方でOpenAIは、ChatGPTの圧倒的な配布力を武器に、非エンジニア層を一気に取りにきた。

Claude / Coworkにとって本当に怖いのは、Codexが性能で完全に上回ることではなく、「そこそこ同じことができる」状態のまま、ChatGPT経由で業務AIの標準入口になってしまうこと。

ただ、配布力＝勝ち、と単純化はできないとも思っています。標準入口が効くのはコンシューマやSMBの世界で、Anthropicが狙うエンタープライズのエージェント席は、ガバナンス・出力品質・ロックイン回避で選ばれる比重が高い。しかもMCPもskillsもオープン標準なので、これは諸刃で、Claude側も他社のレールに乗っていける。

勝負はモデルのベンチ数字よりも「誰が業務の入口を押さえるか」に移ってきている。次の一年はそこを見ています。
