# AIってなんぞ

昔から「AI」って単語自体はありましたね。  
僕のイメージ流暢に会話できるロボット的な。

AIの正式名称は「Artificial Intelligence（アーティフィシャル・インテリジェンス）」で、日本語では「人工知能」と訳されます。  
ロボットまではいかないですが、先述の通りChatGPTを始め広く一般生活に浸透して\*\*「隣にいる相棒」\*\*のような存在になりました。

# 気になるAI（ワード）

使っているAIや、SNS・社内で流れるワードを中心にピックアップしました。  
僕はもっぱらコーディングのサポートと個人サービスの相談役で使用しています。

1. Gemini (Google)  
   文章作成、計画、リサーチ、学習に特化した\*\*「あなただけのAIアシスタント」\*\*です。  
   Googleの強力な検索・情報網を背景に、高速な情報整理を得意としています。
2. ChatGPT (OpenAI)  
   最新のGPT-5.4モデルが登場し、対話の質がさらに向上しています。  
   また、動画生成AIのSoraにより、テキストから高品質な動画を生み出すことも可能になりました。  
   ※Soraはこの記事投稿の2日前くらいにサービス停止になりました
3. Cursor (AIエディタ)  
   社内ですごい話題で、Installまでしてまだ使えてないですが・・・  
   GPT-5.4やGemini 3 Proなどの最新モデルを選択して利用でき、\*\*「アイデアをコードに変える」\*\*エージェント機能が非常に強力です。
4. NotebookLM (Google)  
   特定の資料（ソース）を読み込ませて、その内容に特化した回答を引き出す\*\*「AIリサーチ・執筆アシスタント」\*\*です。 提供された情報を元に要約や分析、記事の構成案作成などを行うため、情報の正確性を保ちながら深い洞察を得るのに適しています。
5. Claude Code (Anthropic)  
   ターミナルで対話しながら開発を進める、エンジニア向けの\*\*「コマンドラインAIエージェント」\*\*です。 コードの書き換え、テストの実行、Gitの操作などをCLI（コマンドライン）上で完結させることができ、開発スピードを劇的に向上させます。

# AI関連の用語集

| 用語 | 例え：体や身近なもの | 解説 |
| --- | --- | --- |
| **LLM（大規模言語モデル）** | **脳（推論エンジン）** | 言葉を理解し、次に何をすべきか、どう答えるべきかを判断する「思考」の核となる部分です。 |
| **ツール / インターフェース** | **体・手足** | CursorやClaude Codeなどがこれに当たります。脳（LLM）の命令を受けて、実際にコードを書いたりファイルを操作したりする「実行」の役割です |
| **モデル** | **脳の種類・専門家** | GPT-5.4、Claude 3.7、Gemini 3 Proなどの実体です。「数学が得意な脳」「長文を読むのが得意な脳」のように、それぞれ異なる得意分野を持っています |
| **エージェント** | **自律して動くスタッフ** | 「これをやっておいて」と目的（ミッション）を伝えるだけで、自分で考えて、テストやデモまで完結させてくれる頼もしい分身のような存在です。 |
| **ルーター（CCRなど）** | **司令塔・マネージャー** | 現場の状況に合わせて、「この仕事は安くて速い脳（モデル）に」「この難しい仕事は賢い脳に」と、体と脳のつなぎ先を交通整理する役割です |
| **インデックス化** | **記憶・自分専用の図書館** | 膨大なソースコードや資料を整理し、必要な時にAIがすぐ参照できるようにした知識のベースです。 |
| **コンテキスト** | **作業机の広さ** | AIが一度に「覚えていられる」情報の量のこと。机が広い（コンテキスト窓が大きい）ほど、大量の資料を一度に広げて検討できます |
| トークン | 電気やガスの使用量（メーター） | AIがどれだけ「思考」したり「資料」を読んだりしたかを示す数値です。このメーターが回った分だけ、従量課金としてコストが発生します |
