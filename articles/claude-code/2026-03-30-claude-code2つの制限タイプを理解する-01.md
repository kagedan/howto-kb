# はじめに
Claude Codeにはいくつかの制限タイプが存在するのですが、Claude Codeを使い始めて間もない頃はちゃんと理解できていなかったため、まとめてみました。
※今回は、多くの方が個人で契約しているであろうproプランにおいて、知っておきたい「レート制限」と「コンテキストの制限」の2つを取り上げます。

まずは以下の公式ドキュメントを読むことをおすすめします。

https://support.claude.com/ja/articles/11647753-%E4%BD%BF%E7%94%A8%E9%87%8F%E3%81%A8%E9%95%B7%E3%81%95%E3%81%AE%E5%88%B6%E9%99%90%E3%81%AF%E3%81%A9%E3%81%AE%E3%82%88%E3%81%86%E3%81%AB%E6%A9%9F%E8%83%BD%E3%81%97%E3%81%BE%E3%81%99%E3%81%8B
