# そもそもトークンとは何か

まず「トークン」からいきます。初めて聞く人も多いと思いますが、難しくないので大丈夫です。

トークンは、AIが文章を読んだり書いたりした量を数えるための単位です。文字数のAI版、と思ってもらえればだいたい合っています。

AIは、自分とやり取りした文章の量を、このトークンという単位で数えているんですね。

ざっくりは文字数に近いんですが、正確には少し違います。

英語だとだいたい4文字で1トークン、日本語はもっと効率が悪くて、1〜2文字で1トークンくらい使います。

日本語は英語より重い、とだけ覚えておけば十分です。

トークンを使うのは、自分が打った文字だけではありません。

AIに渡した画像も、読み込ませたファイルの中身も、AIが返してきた文章やコードも、全部トークンとして積み上がっていきます。

やり取りを続ければ続けるほど、使うトークンはどんどん増えていきます。

このトークンが、次に話す「使用上限」と「コンテキスト」の両方に直接関わってきます。だからここだけ先に押さえておいてください。

# Claude Codeには5時間と1週間の上限がある

Claude Codeをサブスクで使うと、使えるトークンの量に上限があります。

しかも枠が2つあって、5時間ごとにリセットされる枠と、1週間でリセットされる枠の両方で見られています。

5時間の枠は、短い時間にガッと使ったときに当たります。

1週間の枠は、毎日コツコツ使っていると、こっちが先に当たることもあります。どちらか片方でも上限
