どうやったら弱気相場で、日々残高が減る中でも冷静に判断できるだろうか？

今回は、強気相場の時に、弱気相場での判断軸を作っておいて、いざ弱気相場に入っても、冷静に判断基準を使えるように**「AIのスキルを作成」**してみました。

  

### ▶︎投資パラメータを個別サイトで毎朝確認するのは大変だ

調整相場の時には、主に次の５つの指標確認していますが、一覧に表示させることはできず、個別サイトを巡回する必要があります。

1. VIX指数（恐怖指数）  
2. CNN Fear & Greed Index（恐怖と欲望指数）  
3. AAIIセンチメント調査　※週更新のため省略  
4. 米国10年債利回り  
5. プット・コールレシオ（Put/Call Ratio）

加えて、前日からの変化量が分かりにくい指標もあり、「前日から比べて結局今日はどう変わったのか？」を把握するのが大変でした。そこで、Claude Codeを使い自動化してみます。

### ▶︎Claude Code のSkills機能を使い、自動化

Skillsとは、AIに命令する手順書のようなもの。毎朝AIに日本語で「VIXと、Fear &Greed Indexと、、、を取得して、前日との比較をした上で、私の投資方針に基づいて判定まで行って」と指示する代わりに、

/invest（スキル名）と打つと、指標の現在値と、3段階の判定が返ってきます。

![](https://assets.st-note.com/img/1774656136-VFSCeAGDpirov4EtR3cjn5af.png?width=1200)

investスキルの実行結果（例）

🟢 見るだけ・アクション不要  
🟡 注目・継続観察  
🔴 要判断・アクション検討

そして最後にはAIが気を利かせて「狼狽売り判定」が出ます。「このニュースはNISA長期積立の根拠を揺るがすか？」と判断してくれます。

ちなみに、今朝は🔴警戒でした。狼狽売りをする確率が高いという意味の警戒ですね。

### ▶︎投資方針も予めSkillsに組み込んでおき、感情を挟まない

私の投資方針は、よくあるコアサテライト戦略です。

コアである市場平均を取りに行く（β）は、まだまだ若く10年以上待てますので、株式比率を高めて相場が戻るのを待ちます。老後資金確保のために、インフレに勝てれば良い。

サテライト枠のアップサイドを取りに行く（α）は、弱気相場に入る前には基本的には現金比率を最大限まで高めて避難しています。

これらの投資方針を具体的な数値に落とし込んで、Skillsに登録しておけば、人の感情を挟まずに判断しやすくなります。かなり便利。

### ▶︎終わりに

以上、調整相場で確認したいパラメータ５つを、Claude CodeのSkillsで毎朝確認できるよう自動化した話でした。数字の示す意味が横に表示されるので、見慣れない内はかなり分かりやすいと思います。

例えば、VIXが27と聞いても位置感がわからない。でも「警戒水準・20〜30の間」と色付きで表示されると、今どこにいるかがわかる感じです。

「感情で動かないために、仕組みを作る。」が、AIによってかなり簡単になりましたね。
