## 概要

Claude Code でいろいろ作業していると、新しい知識をたくさん目にしますが実際に身についている実感がありませんでした。  
このスキルでは **「一般化された解説 + 4 択クイズ」** に変換してMarkdown ノートに保存し、ローカルのクイズサイトで何度でも復習できるようにしました。

▼リポジトリ  
<https://github.com/koukikki/mybook>

## セットアップ

### CLI を PATH に通す

```
ln -sf "$HOME/.claude/skills/mybook/bin/mybook" "$HOME/Library/pnpm/mybook"
```

確認:

`~/Library/pnpm` 以外でも `~/.local/bin` や `/usr/local/bin` など、  
PATH の通った任意の場所に symlink すれば動きます。

### （オプション）打ち忘れ防止リマインダー

`mybook --add` を打ち忘れる人向けに、Claude Code のセッション終了時に「mybook --add でクイズ化できます」と1度だけ表示させる設定です。

`~/.claude/settings.json` の `hooks.SessionEnd` に以下を追加します。

```
{
  "hooks": {
    "SessionEnd": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "echo '{\"systemMessage\":\"📚 mybook --add で今のセッションをクイズ化できます (mybook --open で復習)\"}'"
          }
        ]
      }
    ]
  }
}
```

セッションを `/exit` やウィンドウ閉じで終わらせたタイミングで一度だけ表示されます。

## できること

この3つのコマンドを使用します。

| やりたいこと | 入り口 | 実行場所 |
| --- | --- | --- |
| 今のセッションを記録（解説 + 4 択問題を生成） | `mybook --add` | Claude Code セッション内 |
| クイズサイトを開く | `mybook --open` / `mybook` | スキル経由 / ターミナル |
| ローカルサーバーを停止 | `mybook --stop` | スキル経由 / ターミナル |

`--add` は会話履歴の抽出が必要なので **Claude Code セッション専用**、  
`--open` / `--stop` はターミナルから直接叩いても動きます。

### 1. セッションの終わり際にノートを作成

Claude が今までの会話履歴を読んで、以下を行います。

1. **学習トピックを抽出**（必要に応じて複数ファイルに分割）
2. **プロジェクト固有名詞を取り除いて「一般化された解説」を生成**
3. 解説に基づいた **4 択問題（3〜6 問）** を作成
4. `~/.claude/mybook/notes/` に Markdown を保存
5. `index.json`（全ノートのメタ情報）を再生成

### 2. クイズサイトで復習する

ローカルで `python3 -m http.server 8765` が立ち上がり、  
ブラウザで `http://localhost:8765/site/` が開きます。  
![](https://static.zenn.studio/user-upload/deployed-images/da219ed564ac11c61565348b.png?sha=254e45992628af528eb75d6672c9c9cf0206dd18)

サイト側は素の HTML / CSS / JS。  
`index.json` を読み込んでノート一覧 → 各ノートの 4 択を出題します。
