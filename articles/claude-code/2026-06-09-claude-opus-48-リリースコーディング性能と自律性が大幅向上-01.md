# Claude Opus 4.8 リリース：コーディング性能と自律性が大幅向上

Anthropicは本日、最新モデル **Claude Opus 4.8** の提供を開始しました。Opus 4.7 をベースに、判断力・誠実さ・長時間の自律作業能力が強化された、現時点で最も強力なコーディングモデルです。

---

## 主な改善点

### 📈 SWE-bench Pro スコアが大幅向上

Opus 4.8 は SWE-bench Pro において **69.2%** を達成し、前モデル Opus 4.7 の 64.3% から大きくスコアを伸ばしました。

### 🔍 より誠実で正確なコードレビュー

Opus 4.8 は自分自身の成果物に対してより正直に評価を行います。

* 不確かな点があれば、きちんとその旨を伝える
* 早まった「完了宣言」をせず、自分のバグを自ら検出する
* Opus 4.7 と比べて、**自身が書いたコードの欠陥を見逃す可能性が約4分の1**に低下

### 🤖 長時間・自律的な作業能力の向上

Opus 4.8 は経験豊富なエンジニアのように、頻繁な確認なしで作業を進めます。

* 長時間セッションにわたってタスクのトラックを維持
* リポジトリ全体を通じた作業の一貫した遂行
* フィーチャー開発やバグ調査をまるごと任せることが可能

---

## Claude Code での使い方

### `/goal` との組み合わせ

長期間にわたる作業（Long-horizon work）において特に力を発揮します。Claude Code では **`/goal`** コマンドと組み合わせることで効果を最大化できます。Claude Managed Agents を利用している場合は **Outcomes** と組み合わせて使用してください。

### エフォートレベルの設定

Opus 4.8 はデフォルトで **高エフォート（high effort）** モードで動作します。

| 用途 | 推奨設定 |
| --- | --- |
| 通常のコーディングタスク | デフォルト（4.7 と同等のトークン消費で高性能） |
| 難易度の高いタスク・長時間の非同期作業 | `xhigh` |

また、増加したトークン使用量に対応するため、Claude Code のレート制限も引き上げられています。

---

## Fast モード：2.5倍の速度、3分の1のコスト

**Fast モード** が Opus 4.8 に対応しました。

* 同一モデルで約 **2.5倍の処理速度**
* 以前と比べて **3倍安価**
* Claude Code では **`/fast`** コマンドで有効化可能
* API 経由での利用はアカウントマネージャーへの連絡またはウェイトリストへの参加が必要

---

Anthropic では、新モデルをリリースする前に専門チームがモデルを実際に使い込み、限界まで試し、改善点をフィードバックするプロセスを踏んでいます。このような徹底した検証が、最終的なモデルの品質向上につながっています。
