「Claude Codeって名前は聞いたことあるけど、何をするものかわからない」

正直、最初の自分もそうでした。

「Code」という単語で「自分には関係ない」と思って、3ヶ月スルーしていた。

損をしていました。

文系・プログラミング未経験の自分でも、副業の作業効率が変わったからです。

  

---

## 🔥 第1章：3行で説明する

結論から言います。

**Claude Codeとは「パソコンの中で実際に作業してくれるAI」です。**

普通のClaudeは「答えをチャットで返す」。

Claude Codeは「実際にファイルを作ったり・読んだり・保存したりする」。

この違いだけ理解すれば、今日から使えます。

---

📰 Anthropic公式ブログ（2025年） 「Introducing Claude Code」

【要約】 Claude Codeはターミナル（黒い画面）上で動作するAIエージェント。通常のチャット型AIと異なり、ファイルの読み込み・作成・編集・保存などの実際のコンピューター操作を自律的に行う能力を持つ。ユーザーは日本語で「○○をやって」と指示するだけでよく、必要な複数のステップをAIが自分で考えて実行する。コードを書く能力は不要で、指示を出す能力があれば使える。

📍 出典：Anthropic公式ブログ。信頼性は高い。 URL: https://www.anthropic.com/claude-code

---

「ターミナル（黒い画面）」という言葉で怖くなった人、大丈夫です。

黒い画面に日本語で入力するだけです。

英語もプログラミングの知識も不要です。

---

## 💀 第2章：普通のClaudeとどう違うか

具体例で説明します。

**普通のClaudeでやること**

「この3社分のデータをまとめたレポートを書いて」と頼む。

→ Claudeが文章を生成して、チャット画面に表示する。

→ 自分でコピーして、ファイルを作って、保存する。

**Claude Codeでやること**

「この3社分のデータをまとめたレポートを書いて、reportsフォルダに保存して」と頼む。

→ Claude Codeがデータを読み込んで、レポートを書いて、フォルダを作って、保存まで全部やる。

→ 自分は「できました」という報告を受け取るだけ。

この差が、副業の作業時間を変えます。

---

## 🎯 第3章：文系大学生でも使えた理由

Claude Codeを使い始めて最初に驚いたのは、「エラーが出ても自分で直す」という動作でした。

普通のAIは「エラーが出ました。こう直してください」と返してくる。

Claude Codeは「エラーが出ました。直しました。続けます」と自分で対処する。

自分がやることは「何をしてほしいか」を日本語で伝えることだけ。

**「コードを書く能力」は不要です。「指示を出す能力」があれば使えます。**

---

## ✅ まとめ

* Claude Codeは「パソコンの中で実際に作業してくれるAI」
* 普通のClaudeとの違い：答えを返すだけ vs 実際にファイルを作って保存する
* プログラミング・英語の知識は不要。日本語で指示を出すだけでいい
* エラーが出ても自分で修正して続けるため、途中で詰まらない
* 「コードを書く能力」ではなく「指示を出す能力」が必要なツール

## 🤖 Claude Codeを始める前に知っておくべきこと

Claude Codeを使うには、まずインストールが必要です。

ターミナルに以下の1行を入力するだけで完了します。

npm install -g @anthropic-ai/claude-code

「npm」「install」の意味がわからなくても大丈夫です。

コピペして実行するだけで、5分以内にインストールが終わります。

インストール後は「claude」と入力してEnterを押すだけで起動します。

起動したら、あとは日本語で話しかけるだけです。
