## claudeくんと企画を詰める

一度走り出していたコーディングですが、一時ストップをかけclaudeくんと企画を延々と詰めていました。  
２、３日かけて、ああでもないこうでもない、これはどうだ、をAIと喧喧諤諤ずっと討論してるのシュールですね。

今は企画系の詰めはWEBのclaudeくんを使い、コーディングはclaude codeをCLIで叩く運用をしています。

企画部分はスマホからも、思いついたことなどさっと聞いたり話したりできるのがありがたいですね。  
本当に市場調査がすばらしく楽になり、自分で一次資料を見に行かなくなってしまいそうなのが怖いほど。

今後は、 **情報は簡単に集まるので、いかに手を動かして行動できるか**が、生き残りの分かれ目になりそうだなぁという感覚を覚えます。

あとWEB版のclaudeくん、時々怠慢を起こしたアウトプットを出してくるので逐一指摘している。

## Apple Developer Programnの登録

いざ鎌倉！とコーディング再開したんですが、早々にxcodeで詰みました。  
認証周りの確認のため、Sign in with Apple Capability を Xcode の Signing & Capabilitiesタブで有効化して実機確認〜〜〜をしようと思ったら、

**無いな・・・・・・Sign in with Apple Capability・・・・・・・**

設定画面を開いても全然ありません。検索してもno match。  
Geminiさんらにも詳細を伺うと、どうやら「Apple Developer Program（有料） への登録が必要です。」と言われる。罠かよ。

まだコーディング走り出したばっかりなのに・・・・・・

というわけで、本人確認書類地獄と言われているApple Developer Programの登録をしてきました。  
アプリ経由で最初本人確認をやろうとしたら、「提出する本人確認書類が該当する地域で認められていないか〜云々」と、お前の国は対象じゃ無いから本人確認できないよ！みたいな謎の返しをされたので**ブチ切れ**。

WEBのほうから先にクレジットでお会計→あとからメールで来る本人確認書類アップしてねフォームでアップしました。2日以内には結果がもらえるとのこと。  
WEBからの場合は人の目で確認なのかしら。

## claude codeとの二人三脚

claude codeを使っても、xcodeのiOSアプリ開発って、人がやることめちゃくちゃ多いんですよね、  
主に設定画面のチェックボックスつけたりなんだり・・・・・・

毎回claude codeが実装した内容の検証をするため、あそこの設定をいじり、ここの設定をいじり、実機検証して、を繰り返し。  
claude codeくんのための検証係してます。なんだか想像していたバイブコーディングと違うな・・・・・・

先日までは全然そのままいけたのに、ここに来て突如プログラム登録しないと検証できない壁にぶちあたり、ここから登録されるまでは別のことでもして時間を潰そうと思います。  
完成もまだまだ先だし〜と言ってApple Developer Programの登録先延ばししてたらいきなり転けて悲しいです。
