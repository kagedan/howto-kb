「AIに仕事を奪われる」とか言うけど、じゃあ実際のところAIと人間の知能差ってどのくらいなの？

自分はIQテストを個人開発している。全30問、数値推理・空間認識・論理推理・言語理解の4分野で測定するやつ。

で、思いついた。**Claude Code（Opus 4）と、人間のエンジニア5人に同じテストを受けさせて直接比較してみよう。**

https://picoli.site/q-iq-0006a

## 対戦メンバー

### AI側

- **Claude Code（Claude Opus 4）**
- 各問題のスクリーンショットを画像で渡して1問ずつ回答させた
- ヒントなし、制限時間なし（人間は1問60秒だが、AIには制限なし）
- つまりAI側はハンデなし、むしろ有利な条件

### 人間側（5人）

全員ソフトウェアエンジニア。自分の周りに声をかけて集めた。

- **Aさん**: バックエンド歴8年。Go/Rust。32歳
- **Bさん**: フロントエンド歴5年。React/TypeScript。28歳
- **Cさん**: MLエンジニア歴3年。Python。2
