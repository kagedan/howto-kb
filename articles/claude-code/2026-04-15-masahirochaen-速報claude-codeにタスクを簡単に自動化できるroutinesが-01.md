【⚡️速報】Claude Codeにタスクを簡単に自動化できるRoutinesが登場。

Anthropicのクラウド基盤で動くので、別途MacMiniなどで常用型のPCが不要。

以前から愛用してましたが、大幅アップデートして嬉しい。

トリガーは3種類：
・スケジュール（毎時/夜間/週次）
・APIエンドポイント（POSTで即起動）
・GitHubイベント（PR・push等のwebhook）

1ルーティンに複数トリガーの同時設定も可能。
利用上限はPro:5回/日、Max:15回/日、Team/Enterprise:25回/日

以前は3つしか自動化設定できなかったが、増えている模様。
