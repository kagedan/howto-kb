Claude Codeが「考えている」あいだ、ついつい別のタブに切り替えて、気づいたら20分後に「あれ、自分は何を頼んだんだっけ？」となることがよくありました。コンテキストは飛び、集中も切れる。ターミナルから離れたくなる衝動と戦う代わりに、その待ち時間をその場で埋めることにしました。

それが claudenews です。エージェントが動いているあいだ、ステータスラインに Hacker News・GitHub Trending・各言語の開発ニュースが静かに流れます。ちらっと一つ眺めるだけで、Claudeが返ってきたときにはまだ同じ場所にいられる、という狙いです。

仕組み

* エージェントの動作を hook で検知し、バックグラウンドのプロセスが記事を取得・キャッシュします。だからステータスラインの描画をブロックしません。
* タイトルはハイパーリンク対応ターミナル（iTerm2 / WezTerm / kitty など）ではクリック可能。tmux やリンクを開けないターミナルでは短縮URLにフォールバックします。HN記事はスコアとコメント数も表示。
* 要約は取得され次第フェードインします。

個人的にいちばん推したい点：自動翻訳

ここが日本語ユーザーにとって本題かもしれません。タイトルも要約も OSの言語に自動翻訳 されます。設定は不要で、日本語・韓国語・中国語のユーザーは同じ記事をそれぞれの母国語で読めます。英語のニュースを「読むのが面倒だから後で」と後回しにしていた人ほど、待ち時間との相性が良いはずです。

インストール（サインアップ不要）

/plugin marketplace add bhpark1013/claudenews  
/plugin install claudenews@claudenews  
/claudenews:setup

リポジトリ: <https://github.com/bhpark1013/claudenews>
