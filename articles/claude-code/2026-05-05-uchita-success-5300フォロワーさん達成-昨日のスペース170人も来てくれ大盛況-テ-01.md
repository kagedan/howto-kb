話した内容のハイライト、下にまとめます👇

【1】メールの半自動化で1日1時間削減

・Claude CoworkでNotion / Gmail / Salesforceを連携
・顧客情報＋商談履歴＋過去のやりとりを全部統合→AIが返信ドラフトを作成。

・全商談を録音→Notionにリアルタイム文字起こし→最新情報を常にAIへ流す。
・リアルタイム文字起こしのおかげで、お客様の質問も文字を見ながら考えれる。

このデータが【1】でも活用されるので重要です。

【3】AIで資料化の話

・ChatGPT / Gemini / Genspark / Claude Design など複数ツールを横並びで比較検証
・自分で 「パワポ出力でレイアウトが崩れない」プロンプトの型 を開発 → 投稿が好評
※投稿あり、ブクマ2,000獲得

ポイントは実務で崩れず使える形に整える"型"を持っておくこと。

全社員Claude CodeかCoworkを活用してる。
