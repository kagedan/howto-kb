Claude Codeに、毎回同じ指摘をしていませんか。CLAUDE.mdに書き足すほど効かなくなる構造をほどいて、「指摘を記録して、資産に変えて、効いているか測る」運用に組み替える本です。実在のリポジトリ・実測のトークンデータ・失敗の記録を現物のまま出します。各章の終わりには、あなたの環境で同じ仕組みを作る最小手順を置きました。対象は、すでにClaude Codeを日常的に使っている方です。
