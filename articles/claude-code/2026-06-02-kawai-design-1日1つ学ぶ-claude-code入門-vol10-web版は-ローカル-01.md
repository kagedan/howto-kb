＼ 1日1つ学ぶ ／
Claude Code入門【 Vol.10 】

Web版は
ローカルPCの外にある
「クラウド作業室」です。

自分のPCで全部動かすのではなく、
ブラウザからリポジトリをつなぎ、
クラウド上で作業を進められます。

イメージは外部の作業部屋。

自分の机を占領せず、
長めの作業を頼み、
終わったら結果を確認します。

Web版が向くのはこの3つです。

✅ 長時間タスク
完了まで待つ作業

✅ ローカル外のrepo
手元にないリポジトリ

✅ 並行作業
複数タスクを同時に進める

さらにWebで始めた作業を、
あとでターミナルへ
持ってくる使い方もあります。

Claude Codeは
場所に縛られない作業担当です。

CLI、Desktop、Webを
作業内容で使い分けると強いです。

Vol.10はここだけ覚えればOKです。
