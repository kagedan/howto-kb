2026年7月14日、Anthropic が「Claude for Teachers」を発表した。米国の認証済みK-12教育者に対し、プレミアム機能のClaudeを無料で開放するというものである。翌15日には公式アカウントからの告知が1000万回以上表示され、日本語圏でも「教師にAIが無料で配られた」という文脈で流通した。

だが、この理解では発表の半分も捉えていない。無料であることは、この施策の結論ではなく前提にすぎない。競合であるOpenAIは、すでに2025年11月19日に「ChatGPT for Teachers」を発表し、米国のK-12教育者に対して2027年6月まで無料提供を開始している。無料化は8か月前に先行されており、そこに新規性はない。にもかかわらずAnthropicが同じ土俵に乗り、しかも「サインアップは2027年6月30日まで、そこから1年間無料」という、競合の期限をほぼなぞる設計にしたのはなぜか。

発表内容を分解すると、実装されたものは四層に整理できる。

第一に、Learning Commons との接続。全米50州すべての学力基準（academic standards）にマッピングされたデータ層へ、Claudeが直接アクセスする。単に「基準の一覧」を持っているのではない。各基準の下位に位置する細分化された学習コンポーネント（learning components）と、生徒が通常それを習得していく順序（progressions）まで含まれている。加えてOpenSciEd、Illustrative Mathematics の IM v.360 といった、実証に裏打ちされたカリキュラム資源が組み込まれる。

第二に、教育スキルのライブラリ。Learning Commons と共同開発され、学習科学（learning science）に基づいて設計されている。教師が「最も重要だ」と回答したタスクを起点に設計され、厳密性・教育学的整合性・教室での実用性の観点から評価を経て、ブルックリンの Prospect Schools をはじめとする現場教員のフィードバックで改良されている。

第三に、K-12エコシステムへの9つのコネクタ。ASSISTments（自動採点される基準準拠の数学問題生成）、Brisk Teaching（対話型アクティビティと基準準拠授業の生成）、Canva Education（教材の教室向けデザイン化）、Coteach（K-12カリキュラムに根ざした数学図版）、Diffit（教材の個別最適化）、Eedi（英語・スペイン語での診断的設問生成）、MagicSchool、Snorkl（クラス・課題・生徒進捗のインサイト）、TeachFX（実際の教室での発話に基づく指導フィードバック）。これらはすべて Model Context Protocol（MCP）で実装されている。

第四に、法務・信頼の層。18歳以上ポリシーに沿った「教育者のみ」の設計、K-12プライバシー専用の教師向け利用規約、モデル学習への不使用、FERPA準拠のK-12データ処理補遺（DPA）。そして全米教職員組合（AFT）との連携により、同組合が策定中の「ゴールドスタンダード」への準拠を表明している。AFT会長ランディ・ワインガーテンは、Anthropicがこの原則にコミットしていることの重要性に言及し、学びの中心にある人間的な関係のための時間を教師に取り戻すツールであってほしいと述べている。

さらに、Claude Code と Cowork が含まれる。これが最も見落とされている点である。

## 「無料」は価格政策ではなく、配布経路の設計である

無料提供という言葉が思考を止めさせる。ここで問うべきは、無料で配られているものが何か、である。

答えは、モデルではない。モデルへのアクセスは、米国の教師にとってすでに複数の選択肢から無料で手に入る状態にあった。ChatGPT for Teachers があり、Google は Workspace for Education に Gemini を織り込み、Microsoft は Copilot を教育スイート全体に埋め込んできた。モデルの無料アクセスはコモディティである。

無料で配られている実体は、標準準拠データ層への接続権である。全米50州の学力基準と、その下位の学習コンポーネント、習得順序。これは公開情報の集積ではあるが、機械可読な形で、粒度を揃えて、AIエージェントが参照可能な構造で整備されたものは、これまで市場に存在しなかった。教育における「地図データ」に相当する。
