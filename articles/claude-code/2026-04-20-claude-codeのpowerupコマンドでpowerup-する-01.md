はじめに
クラシルでSREをしています、moikeiです。
Claude Codeは多機能でアップデートも多いですが、もはやよく使う機能のみで「なんとなく使っている」状態の方も多いと思います。私もそうでした。
v2.1.90で、Claude Codeの主要機能を体験できる公式チュートリアル /powerup コマンドが追加されました。
機能を「読んで理解する」より「動かして理解する」方が定着しやすいため、基礎からざっと体系的に把握できて便利でしたので紹介します。
