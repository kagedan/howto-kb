はじめに — 20年ぶりにコードに触れたおじさんが、Claude Codeとアプリを設計した話
20年ぶりにコードに触れました。
IT企業に勤めてはいるものの、プログラミングからは長く離れていた46歳のおじさんです。最近Claude Codeを使い始めて、また開発を試しています。
きっかけは、日報の集約作業でした。毎日書いている日報を週報にまとめ、さらに月報にまとめる——この作業を手動でやるたびに「もう少し楽にならないか」と思っていました。5日分の日報を読み返して週報にまとめるだけで30分以上かかります。月報となると、4〜5週分の週報をさらに抽象化するので、もっと時間がかかります。
