I will never build a workflow the same way again.

I've been playing around with n8n with Claude Code through MCP.

I'm mid-build on a content research automation right now. All I did was connect n8n to Claude via MCP and API and started talking to it.

Natural language. No node hunting. No documentation tabs.

Claude walks through every node configuration in detail and just builds it.

No clicking through menus. No forgetting which credential goes where. No staring at a broken workflow trying to figure out what went wrong.

Just describe what you want. Watch it appear.

If you're building on n8n and you're not using Claude Code yet, you're doing it the hard way.
