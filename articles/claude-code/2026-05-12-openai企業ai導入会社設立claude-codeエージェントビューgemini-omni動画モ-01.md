## 🏢 OpenAIが「AI導入専門会社」を設立！コンサル業界を直撃か

OpenAIが「OpenAI Deployment Company」を立ち上げました。一言で言うと、**企業へのAI導入を直接支援する会社**です。

これ、地味にインパクト大きいんですよね。今まではAIコンサルとかITコンサルが間に入って、企業にAIを使わせようとしてたわけじゃないですか。でも、実際のところ「AIは導入したけど、全然使いこなせてない」っていうギャップがめちゃくちゃあるらしくて。そのギャップをOpenAI自身が埋めに来たって感じ。Anthropicも同様にマッキンゼーなど大手コンサルを巻き込んで同じような動きをしているらしく、AIモデル開発会社が直接企業支援に乗り出す流れが本格化しています。既存のAI導入コンサル会社にとってはかなり脅威になりそうですね。

## 🛡️ OpenAIがサイバーセキュリティAI「Daybreak」を発表

OpenAIから「Daybreak（夜明け）」という、サイバー防御に特化した最先端AIが発表されました。

名前の通り、**問題が起きる前に脆弱性を発見する「予防型」のAI**というコンセプトみたいです。CodexのデスクトップアプリにあるオートレビューGit機能の裏側で動いているんじゃないかという見方もあります。AIの普及とともに、サイバー攻撃の手口も高度化しているので、防御側もAIで対抗する時代になってきましたね。今日のニュースを見ていると、サイバーセキュリティ関連のトピックが一気に増えた印象で、業界全体でこのテーマへの注目が高まっているのを感じます。

## 🚀 Claude Codeに「エージェントビュー」が登場！並列作業がより見やすく

Anthropicが、Claude Codeに「エージェントビュー」という新機能を正式リリースしました。

これ、ちょっと嬉しい機能ですよね！以前からtmux（ターミナル複数起動ツール）を使ってClaudeを並列で動かす「チーム作業」みたいなやり方が一部で流行っていたんですが、それを**Claude Code公式として実装してきた**という感じ。1つのターミナル内で何をやっているかがリアルタイムで見えるようになったり、キーボード操作でビューを切り替えられたりするらしいです。コードを書かせながら「今どのタスクをやってるの？」が一目でわかるのは、管理のしやすさが格段に上がりそう。Claude Codeをメインで使っている方はぜひ試してみる価値ありです！

## 🤖 Gemini Omni：Googleのリアル系動画生成モデルが激ヤバ

Googleから「Gemini Omni」という新しい動画生成モデルのデモ映像が流出（？）しています。Google I/Oで正式発表される可能性が高いと言われているやつです。

このデモ映像がとにかくすごい。黒板にチョークで板書する人物の動画なんですが、**チョークの粉がリアルに削れて落ちる様子まで再現されてる**んですよ。じっくり見ないと「これAI生成なの？」って分からないレベルです。動きの自然さ、質感のリアルさ、全部が一段上に来た感じ。最初はUltraプランなど上位ユーザー向けの提供になりそうですが、正式リリースされたら話題になること間違いなしですね。Google I/Oが楽しみ！

## ⚡ CursorがTeams連携＆Bugbotの思考深度カスタマイズに対応

コーディングAIのCursorが、開発者にうれしいアップデートを2つ同時に発表しました。

1つ目は**Microsoft Teamsとの連携**。Teams上で「@Cursor」とメンションするだけでタスクをAIエージェントに委任したり、Teams内の情報をCursorに引き込んだりできるようになりました。チームでの開発作業がさらにスムーズになりそうです。

2つ目は**Bugbotの思考深度カスタマイズ**。PRレビュー時にBugbotがどれだけ深く考えるかを調整できるようになりました。重要なレビューはじっくり、スピードが必要なときはサクッと、と使い分けができる感じですね。Cursor × Teamsどちらも使っているチームには刺さる機能です。

## 🌐 Thinking Machines：リアルタイム音声×マルチモーダルの新星

「Thinking Machines」という、ちょっと気になる新しいAIが登場しました。
