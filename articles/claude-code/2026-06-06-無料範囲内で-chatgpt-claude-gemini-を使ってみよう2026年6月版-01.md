## はじめに

この記事では、ChatGPT、Claude、Geminiを **無料の範囲内で短時間に試します**

https://chatgpt.com/

https://claude.ai/

https://gemini.google.com/app

比較する共通タスクは、初心者が試しやすい次の3つだけに絞ります

| 共通タスク | 何を見るか |
|---|---|
| 要約 | 長い文章を短く、わかりやすく整理できるか |
| 画像説明 | 画像内の文字・数値・グラフ・注意点を読み取れるか |
| CSV分析 | 表データから傾向や次のアクションを出せるか |
| 画像生成 |  日本語文字入りの画像生成、画像の修正、データ分析 |

※タスクごとに新しいチャットを作成しましょう

そのあと、各サービスならではの機能を試します

| サービス | この記事で特に試す機能 |
|---|---|
| ChatGPT | Canva との連携 |
| Claude | ArtifactsでHTMLレポートや簡単な画面を作る |
| Gemini / NotebookLM | DeepResearch、NotebookLMの動画解説 |

APIについては、初心者向けの記事なので最後に「存在だけ」触れます

ブラウザやアプリで使う ChatGPT Plus などの月額課金と、開発者向けAPIの従量課金は別物です

## ざっくり比較

| 使いたいこと | まず試す候補 |
|---|---|
| 文章作成、要約、画像生成、ファイル分析をまとめて試したい | ChatGPT |
| 長めの文章の整理、自然な日本語、HTMLや簡単な成果物作成を試したい | Claude |
| Googleサービス連携、NotebookLM、Canvas、動画解説を試したい | Gemini / NotebookLM |
| どれが自分に合うか知りたい | 同じ素材を3つに入れて比べる |

ChatGPTの無料版では、Web検索、データ分析、画像・ファイルアップロード、GPTsの利用、画像生成などが利用できます

https://help.openai.com/ja-jp/articles/9275245-chatgpt-free-tier-faq

Claudeの無料プランでは、チャット、コード生成、データ可視化、Web検索、ファイル作成とコード実行、Memory、SlackやGoogle Workspace連携、MCPコネクタ、拡張思考などが利用できます

https://claude.com/ja/pricing

Geminiの無料プランでは、Geminiアプリ、画像生成と編集、Deep Research、Gemini Live、Canvas、Gems、NotebookLM、Google Flowへのアクセスなどが利用できます

https://gemini.google/jp/subscriptions/?hl=ja

ただし、無料版は回数や処理量の制限が有料版より厳しいです

## 無料で試すときの注意

無料プランでは、利用回数、アップロード、画像生成、動画生成、ディープリサーチなどに制限があります
制限はサービス、地域、アカウント、混雑状況、プラン変更によって変わります

初心者は、まず次のルールで試すと失敗しにくいです

| ルール | 理由 |
|---|---|
| 文章は1,000〜2,000字くらいから始める | 長すぎると制限に当たりやすい |
| 画像は1枚だけアップロードする | 読み取り結果を比較しやすい |
| CSVは数十行から始める | 集計ミスや読み込み制限を確認しやすい |
| 個人情報・社外秘は入れない | 無料プランでは特に安全側に倒す |
| AIの回答は必ず確認する | 数値、固有名詞、日付は間違うことがある |

また、業務利用する場合は必ず社内の規定に従いましょう

### 入力テキスト

<details><summary>以下のテキストを要約元とします（折りたたみを開いてください）</summary>

```text
株式会社サンプルウェアでは、営業・開発・カスタマーサポートの3部門を対象に、生成AIの社内活用を始めることになりました。最初の目的は、業務をすべて自動化することではなく、日々の小さな作業を短時間で改善できるかを確認することです。たとえば、営業部では商談メモの要約、提案メールの下書き、競合情報の整理に使う予定です。開発部ではエラーメッセージの読み解き、テストケースの洗い出し、READMEの改善に使います。カスタマーサポート部では問い合わせ内容の分類、返信文のたたき台作成、FAQの更新案づくりを試します。

一方で、注意点もあります。顧客名、契約金額、個人情報、未公開の設計情報などは、社内ルールで許可された環境以外には入力しません。また、AIの回答は必ず人が確認し、事実確認が必要な内容については公式資料や社内ドキュメントに戻って確認します。特に、法律・医療・財務・セキュリティに関わる判断をAIだけで行うことは禁止します。

試験導入は4週間行います。1週目は全員が同じプロンプトで要約、文章作成、表作成を試します。2週目は部門ごとに実務に近い使い方を試します。3週目は便利だったプロンプトと失敗例を共有します。4週目は、効果が大きい用途、継続利用に向く用途、使わないほうがよい用途を整理します。評価指標は、作業時間の短縮、出力品質、確認にかかった時間、情報漏えいリスク、利用者の満足度です。

今回のゴールは「AIに仕事を丸投げすること」ではありません。AIを、下書き、整理、発想、確認の補助として使う感覚を身につけることです。初心者でも安心して試せるように、最初は短い文章、小さなCSV、公開しても問題ない画像だけを使います。最終的には、各部門から1つずつ、すぐに業務に取り入れられる安全な活用例を選び、社内ガイドラインに反映する予定です。
```

</details>

## ChatGPTで試す: Canva連携でデザインを作る / 探す

ChatGPT と Canva を連携させて、デザインを作ってみましょう

※Claude も Canva と連携可能です

### 入力素材

以下のテキストをファイルとして保存し、アップロードして使います

```text
イベント名: はじめての生成AI体験会
対象: 生成AIを初めて使う社会人・学生
時間: 30分
内容:
- ChatGPT、Claude、Geminiに同じ素材を入れて比較する
- 要約、画像説明、画像生成、CSV分析を試す
- 無料プランでできることと、制限に当たったときの考え方を知る
雰囲気: 明るい、安心感がある、専門用語が少ない
作りたいもの: Qiita記事のアイキャッチ、SNS告知画像、5枚以内のミニスライド
```

### 試す手順

1. ChatGPTを開く
2. 入力欄で `Canva` と入力する、またはアプリ/コネクタ一覧からCanvaを探す
3. 初回はCanvaアカウントへの接続許可を確認する
4. 次のプロンプトを送る

```text
Canva、以下の内容でQiita記事のアイキャッチ画像を作ってください。
初心者向けなので、専門的すぎない雰囲気にしてください。

サブタイトル: 無料で30分、ChatGPT・Claude・Geminiを比べよう
対象: 生成AIを初めて使う社会人・学生
雰囲気: 明るい、安心感がある、シンプル、読みやすい
入れてほしい要素: ノートPC、スマホ、3つのAIを比較するイメージ
```

5. ChatGPT上でプレビューできる場合は確認する
6. 気に入った案をCanvaで開いて、文字、配色、画像を微調整する

### 追加で試すプロンプト1: スライドに変換する

```text

スライド構成:

2. 今日試す4つの共通実験
```

### 追加で試すプロンプト2: 既存デザインを探して要約する

Canva内に過去の資料やテンプレートがある場合は、検索・要約も試せます。


既存デザインの検索は、アクセス許可やCanva側のプラン・ワークスペース設定に左右されます。何も出てこない場合は、新規作成だけ試せば十分です

## Geminiで試す: Deep Researchで比較レポートを作る

GeminiのDeep Researchは、普通のチャットよりも「調査」に寄せた機能です

Googleのヘルプでは、GeminiアプリのDeep Researchはほとんどのテーマについてリアルタイムで詳しく調査でき、既定ではGoogle検索をソースとして使うと説明されています

必要に応じて、Gmail、Googleドライブ、アップロードファイル、NotebookLMノートブックなどもソースにできます

https://support.google.com/gemini/answer/15719111?co=GENIE.Platform%3DDesktop&hl=ja

Deep Researchを「最新情報を集めて、記事の下調べレポートを作る機能」と、考えるとわかりやすいです

1. Geminiを開く

6. レポートが完成したら、次のプロンプトでQiita向けに整える

```text
このレポートをQiita記事に反映しやすいように、

```

### Deep Researchに貼るプロンプト

```text
生成AI初心者向けに、ChatGPT / Claude / Gemini を無料で試すQiita記事を書くための調査をしてください。

2. それぞれの特徴的な機能を1〜2個ずつ挙げる
   - ChatGPT: 日本語文字入り画像生成、画像修正、データ分析
   - Claude: Artifacts、Claude Desktopでできること
   - Gemini: Deep Research、NotebookLMの動画解説、Canvas

出力してほしい形式:
```

| 見るところ | 確認方法 |
|---|---|
| リサーチプラン | 調べる範囲が広すぎないか、APIに寄りすぎていないか |
| 公式情報の扱い | 料金や無料枠の説明に公式ページが付いているか |

Deep Researchは便利ですが、完成したレポートをそのまま記事に貼るのではなく、必ず公式ページを開いて料金、上限、提供地域を確認します。生成AIサービスは仕様変更が多いため、記事の先頭に「最終確認日」を入れておくと読者に親切です。

## Gemini / NotebookLMで試す: 資料から動画解説を作る

NotebookLMは、資料をアップロードして、要約、質問応答、音声概要、動画解説などを試せます

https://notebooklm.google.com/

## APIについて

各種 AI は API から呼び出すこともできます

APIは、次のような段階になってから調べればOKです

- 自分のアプリや社内ツールからAIを呼び出したい
- チャット画面ではなく、プログラムから自動処理したい
- 大量の文章やデータを決まった手順で処理したい

注意点として、ChatGPT Plusなどの月額プランと、OpenAI API、Claude API、Gemini APIの料金は基本的に別です

アプリの有料プランに入っていても、API利用料は別にかかる場合があります
