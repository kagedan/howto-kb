Agent serving shouldn’t force you into a framework rewrite.

Motus is an open-source agent-serving project for builders who want to serve and deploy agents built with Motus, OpenAI Agents SDK, Anthropic SDK, Google ADK, or plain Python.

It helps you move from local agent code to a served endpoint faster by combining a CLI, local serving, cloud deployment, and a Python runtime for parallel, resilient workflows.

Key features:

• Coding-agent workflow – install the plugin/CLI and use Motus directly from tools like Claude Code, Codex, or Cursor
• Local + cloud serving – run an agent locally with `motus serve`, then deploy with `motus deploy`
• Framework-flexible support – works with Motus agents, major agent SDKs, and plain Python agents
• Python task graphs – `@agent_task` turns normal functions into dependency-aware workflows with parallel execution
• Runtime primitives included – retries, timeouts, caching, tracing, observability, tools, memory, and guardrails are documented in the stack

It’s open-source (Apache 2.0 license).
