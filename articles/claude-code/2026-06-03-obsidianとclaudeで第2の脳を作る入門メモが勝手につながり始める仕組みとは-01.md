## 「第2の脳」とは何か

「第2の脳（Second Brain）」は、Tiago Forteが提唱した個人ナレッジマネジメント（PKM）の概念です。脳の外に「思考を補助するシステム」を構築し、記憶・発想・アウトプットを効率化します。

従来のメモシステムとの違いは\*\*「つながり」\*\*にあります。単なるメモはキャビネットに整理された書類と同じ——必要なときに探さなければなりません。第2の脳は、関連する情報が自動的にリンクし、新しい洞察が自然と浮かび上がる生きたシステムです。

ObsidianはこのPKMに最も適したツールです。そしてClaudeを組み合わせることで、メモの価値が何倍にもなります。

## Obsidianの基本：ノートをつなぐ「双方向リンク」

Obsidianの核心は\*\*双方向リンク（`[[ノート名]]`）\*\*です。

例えば「マーケティング戦略」というノートに `[[コンテンツSEO]]` `[[ターゲット設定]]` とリンクを貼ると、それぞれのノートからも「マーケティング戦略」に戻るリンクが自動で生成されます。これを繰り返すことで、バラバラだったメモがネットワーク状につながります。

\*\*最初の3ステップ\*\*:

1. Obsidianをダウンロード（無料）してVaultを作成

2. 読んだ本・記事・会議のメモをどんどん書く

3. 関連するノートを `[[]]` でリンクしていく

これだけで、1ヶ月後には「思考のマップ」が出来上がっています。

## ClaudeをObsidianと組み合わせる：3つの使い方

### 使い方1：メモの「要約・整理」を自動化

長い会議メモや読書メモをClaudeに渡し、整理させます。

\*\*プロンプト例\*\*:

```

以下のメモを「①重要ポイント3つ」「②アクションアイテム」「③関連する既知の概念」の形式に整理してください。

[メモをペースト]

```

整理したアウトプットをObsidianに保存すれば、後から使えるノートになります。

### 使い方2：「つながり」の発見を助ける

複数のノートをClaudeに渡し、共通テーマや意外なつながりを発見させます。

\*\*プロンプト例\*\*:

```

以下の3つのメモを読んで、共通するテーマや意外なつながりを3つ指摘してください。

[ノートA]

[ノートB]

[ノートC]

```

人間が気づかない「ノート間のパターン」をClaudeが発見してくれます。これが第2の脳のハイライトです。

### 使い方3：メモからアウトプットを生成

蓄積したノートをソースに、記事・提案書・発表資料を生成します。

\*\*プロンプト例\*\*:

```

以下のメモ群を素材に、[読者]向けの[形式]を作成してください。

主張は一つに絞り、具体例を2つ以上使ってください。

[関連メモをペースト]

```
