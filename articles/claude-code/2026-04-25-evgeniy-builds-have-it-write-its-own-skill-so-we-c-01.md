"Have it write its own skill so we can repeat it" is the unlock. Built ~30 skills for my Claude Code workflow on Jobbi using the same loop. The trade you're paying for is the right one: hundreds of lines today, a one-shot tomorrow. Where it bites is when you forget step two. The first session writes the elaborate solution and the second session re-writes it from scratch because no one captured the recipe.
