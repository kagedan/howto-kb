## 概要

著者（スナガク）がClaude Codeを使いこなすために日常的に意識している5つのポイントをまとめた記事。

## 5つのポイント

1. **公式ドキュメントをちゃんと読む** — 知らない機能は活用できない。Agent Teams、Hooks、Skillsなどの最新機能を公式ドキュメントから学ぶことが重要。

2. **効率化の視点を常に持つ** — 同じ指示パターンを繰り返しているなら、CLAUDE.mdやSkillsで自動化できないか検討する。

3. **AI処理をブラックボックスにしない** — 「なぜこの結果になったのか」と思考過程を理解することで、指示精度の向上と再現性が生まれる。

4. **新しい仕組みは小さく試す** — 最小限の構成から段階的に検証し、問題の切り分けを容易にする。

5. **フレームワークを自作してみる** — 既存ツールの中身を理解することで設計思想が腑に落ち、他者の工夫からも深い学びが得られる。
