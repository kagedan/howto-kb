AIにコードを書いてもらうだけでなく、実際にファイルの読み書きやコマンド実行まで任せられる「Claude Code」。  
名前は聞いたことがあっても、何から手をつければいいか分からず足踏みしている人は多いはずです。  
この記事では、Claude Codeのインストールから初回起動、最初に試すべき一歩までを順番に解説します。  
つまずきやすいポイントも合わせて紹介するので、この記事どおりに進めれば今日中に動かせます。

## Claude Codeとは何か

![](https://assets.st-note.com/img/1784399912-ukDCGOqFmgIf9hXNT8zrHAvx.jpg?width=1200)

Claude Codeは、Anthropic社が提供するAIコーディングエージェントです。  
「ターミナル」と呼ばれる、文字だけでコンピュータを操作する画面上で動きます。

普段使うAIチャットとの一番の違いは、会話だけで終わらない点です。  
Claude Codeはチャットで回答するだけでなく、実際にファイルを開いて中身を読み、コードを書き換え、  
必要であればコマンドまで実行してくれます。

例えば「このプロジェクトの説明ファイルを作って」と頼めば、フォルダの中身を自分で確認し、  
内容に合った説明文を作成してファイルとして保存してくれます。  
「バグを直して」と頼めば、該当のファイルを探し出し、原因を特定したうえで修正案を提示し、  
了承すればその場でファイルを書き換えてくれます。  
指示を出す相手が「答えてくれる人」から「実際に作業してくれる人」に変わるイメージを持つと分かりやすいでしょう。

## 始める前に用意するもの

![](https://assets.st-note.com/img/1784399922-3WqgEdn8Duzb5AmMIafxyioV.jpg?width=1200)

インストール作業に入る前に、以下の3つを準備しておきましょう。

* ターミナル: 文字でコマンドを入力する画面。Windowsなら「PowerShell」、Macなら「ターミナル」というアプリが標準で入っている
* Node.js（エヌオードジェイエス）: JavaScriptというプログラミング言語を動かす土台となるソフト。npm版のインストールに必要
* Anthropicアカウント: Claude Codeを使うために必須。Claude Pro以上の有料プランへの加入が前提となる

Node.jsは公式サイト（[nodejs.org](http://nodejs.org)）からダウンロードしてインストールできます。  
インストール後にターミナルで `node -v` と入力し、バージョン番号が表示されれば準備完了です。  
何も表示されない、またはエラーになる場合はインストールがうまくいっていないので、  
一度パソコンを再起動してから再度確認してみてください。

なお2026年に入ってからは、Node.js不要でインストールできる「ネイティブインストーラー」が公式に推奨されるようになりました。  
Mac・Linuxを使っている場合はこちらの方法がより簡単なので、次の章で両方の手順を紹介します。  
どちらの方法を選んでも、最終的にできることは同じです。まずは自分の環境に合う方を選べば十分です。

## インストール手順

![](https://assets.st-note.com/img/1784399930-cyn9RKe4lGkJv5VL2uQMHzbD.jpg?width=1200)

### 方法1: Claude Code Desktop（GUIアプリ・ターミナル操作なし）

「ターミナルという黒い画面を触るのはハードルが高い」と感じる場合は、Claude Codeを内蔵したデスクトップアプリ「Claude Desktop」から始めるのが最も簡単です。

1. 公式サイト（[claude.com/download）からWindows・Mac・Linux（ベータ）向けのインストーラーをダウンロードする](http://claude.com/download%EF%BC%89%E3%81%8B%E3%82%89Windows%E3%83%BBMac%E3%83%BBLinux%EF%BC%88%E3%83%99%E3%83%BC%E3%82%BF%EF%BC%89%E5%90%91%E3%81%91%E3%81%AE%E3%82%A4%E3%83%B3%E3%82%B9%E3%83%88%E3%83%BC%E3%83%A9%E3%83%BC%E3%82%92%E3%83%80%E3%82%A6%E3%83%B3%E3%83%AD%E3%83%BC%E3%83%89%E3%81%99%E3%82%8B)
2. ダウンロードしたファイルを開いてインストールする
3. Windowsならスタートメニュー、Macならアプリケーションフォルダから「Claude」を起動する
4. Anthropicアカウントでログインする

このアプリにはClaude Codeの機能がそのまま組み込まれており、コマンドを一切打たずにインストール〜利用開始まで完結します。ただし利用にはClaude Pro以上の有料プランへの加入が前提になる点は、後述のターミナル版と同じです。「まずは画面を見ながら操作したい」という方は、この方法から試すのがおすすめです。

一方で、後述のターミナル版（CLI）は、複数のプロジェクトを行き来したり自動化を組んだりする際に柔軟性が高く、慣れてきたらこちらに移行する人も多くいます。まずはどちらか始めやすい方で構いません。

### 方法2: ネイティブインストーラーを使う（Mac・Linux推奨）

ターミナルを開き、以下のコマンドを1行入力してEnterキーを押すだけです。

```
curl -fsSL https://claude.ai/install.sh | bash
```

`curl`（カール）は「インターネット上のファイルを取得する」ためのコマンドです。  
このコマンド1つで、Claude Code本体のダウンロードから設定までが自動的に完了します。  
Node.jsのインストールも不要なので、ターミナルを使う中では最も手間が少ない方法です。  
実行後に「Successfully installed」のような完了メッセージが表示されれば成功です。

### 方法3: npm（Node.jsのパッケージ管理ツール）でインストールする

Node.jsをすでに入れている場合や、Windows環境の場合はこちらを使います。  
npm（エヌピーエム）とは、Node.jsに付属している「ソフトを配布・管理する仕組み」のことです。

※npm経由のインストールは2026年1月よりAnthropic公式で非推奨とされています。Windows環境などで他に選択肢がない場合の方法として利用し、可能であれば方法2のネイティブインストーラーや方法4のHomebrewを優先してください。

1. ターミナル（PowerShell）を開く
2. 以下のコマンドを入力する

```
npm install -g @anthropic-ai/claude-code
```

`-g` は「パソコン全体で使えるように登録する」という意味のオプションです。  
これを付け忘れると、そのフォルダの中でしか使えない状態になってしまうので注意しましょう。  
インストールが終わったら、ターミナルに `claude` と入力してEnterキーを押します。

### 方法4: Homebrew（macOSのみ）

Macで「Homebrew」というソフト管理ツールをすでに使っている場合は、次のコマンドでも導入できます。

```
brew install --cask claude-code
```

いずれの方法でも、最終的にできることは同じです。  
ターミナルに抵抗がなければ方法2〜4、まずは触ってみたいという場合は方法1のデスクトップアプリを選べば十分です。

## 初回起動とログイン

![](https://assets.st-note.com/img/1784399952-MsOfWz0eEno7RvZNyDPlJarY.jpg?width=1200)

ターミナル版（方法2〜4）は、インストールが終わったらターミナルで `claude` と入力して起動します。  
Desktop版（方法1）の場合はアプリを開くだけで、次のログイン画面がそのまま表示されます。  
いずれも初回起動時はAnthropicアカウントへのログインを求められる点は共通です。

1. ターミナル版は `claude` と入力してEnter／Desktop版はアプリを起動
2. 表示された案内に従いブラウザが自動で開く（Desktop版は不要な場合もある）
3. Anthropicアカウントでログイン（アカウントがなければこの画面から新規作成する）
4. ログイン許可を行うと元の画面に戻り、利用開始の表示が出る

ここまで進めば、あとはターミナルにやってほしいことを日本語や英語の文章で入力するだけです。  
特別なプログラミング言語のコマンドを覚える必要はありません。  
普段友人にお願いごとをするときのような、自然な文章で構いません。

## 料金プランの考え方

![](https://assets.st-note.com/img/1784399961-uZvUm2pc0hxqt7ORkIEWDsCL.jpg?width=1200)

Claude Codeは無料のClaude.aiアカウントだけでは利用できず、有料プランへの加入が前提になります。  
執筆時点で確認できた情報では、個人利用ならまず「Claude Pro」（月額20ドル程度）から始めるのが  
一般的な入り口とされています。より多く使いたい場合は上位の「Max」プラン、チームでの利用には  
「Team」「Enterprise」プランも用意されています。

どのプランを選べばよいか迷う場合は、まず最も下のプランで契約し、  
使ってみて上限に届くようであれば上位プランへ切り替えるという進め方でも十分です。  
一度契約したら変更できないというものではなく、後から見直しがしやすい仕組みになっています。

料金や利用上限は改定される可能性が高い情報のため、契約前に必ずAnthropic公式サイトの  
最新の料金ページで確認してください。この記事では金額を断定的な情報として保証するものではなく、  
あくまで執筆時点の目安としてお伝えしています。

## 最初の一歩：まず何を頼めばいいか

![](https://assets.st-note.com/img/1784399970-NCvJ6kgphZaKVMzjwylUtb43.jpg?width=1200)

インストールが終わったら、いきなり難しい注文をする必要はありません。  
最初は次のような、結果が合っているかどうか自分で判断しやすい小さな作業から試すのがおすすめです。

* 既存のプロジェクトフォルダで `claude` を起動し「このフォルダの中身を説明して」と頼む
* 「このプロジェクトのREADME（説明書きファイル）を作って」と頼む
* 手元の簡単なプログラムを見せて「このエラーを直して」と頼む
* 「このコードに日本語のコメントを付けて」と頼む

例えば、手元に簡単な計算プログラムがあり、実行するとエラーが出るとします。  
そのフォルダで `claude` を起動し「実行するとエラーが出る、原因を調べて直してほしい」と伝えるだけで、  
ファイルの中身を読み込み、原因を特定し、修正した上で結果を報告してくれます。  
最初の数回は「本当にファイルを読んで作業しているのか」を確認する意味でも、  
身近な題材で試してみると仕組みへの理解が深まります。

## つまずきやすいポイント

セットアップの際によく発生する困りごとと、その対処法をまとめておきます。

つまずきポイント　原因　対処法  
`claude` コマンドが見つからないと表示される　インストール先にパス（コマンドの置き場所の登録）が通っていない　ターミナルを一度閉じて再度開く。それでも直らなければパソコンを再起動する  
インストール時に権限エラーが出る　パソコンの管理者権限が必要な場所に書き込もうとしている　Windowsは管理者権限でターミナルを開き直す。Macは `sudo` を付けて実行する  
ログイン画面がブラウザで開かない　標準ブラウザの設定や通信環境の問題　表示されたURLを手動でコピーしブラウザに貼り付けて開く  
コマンドを打っても反応がない　有料プランへの加入が完了していない　Anthropicアカウントの契約状況を確認する

これらはClaude Codeに限らず、ターミナルを使った作業全般でよく起こる一般的なトラブルです。  
「エラーが出た＝失敗した」ではなく「よくある通過点」だと考え、焦らず一つずつ確認すれば、  
たいていはすぐに解決します。

## よくある質問

Q. プログラミングの知識がなくても使えますか？  
A. コードを1行も書いたことがなくても操作自体は可能です。ただし、Claude Codeが行った変更の  
意味を理解するには、多少のプログラミングの基礎知識があった方が安心して使えます。

Q. WindowsとMacで使い方に違いはありますか？  
A. 起動後の操作方法はほぼ同じです。違いが出るのはインストール方法の部分のみで、  
Windowsはnpm経由、Mac・Linuxはネイティブインストーラーが推奨されています。

Q. 無料で試すことはできますか？  
A. 執筆時点では、Claude Codeの利用には有料プランへの加入が前提となっています。  
無料お試し枠の有無や条件は変更される可能性があるため、公式サイトで最新情報を確認してください。

Q. インストールに失敗したらどうすればいいですか？  
A. 多くの場合はパスの設定漏れかNode.jsのバージョン不一致が原因です。  
一度ターミナルを再起動し、それでも解決しない場合はパソコン自体を再起動してから  
同じコマンドを再実行してみてください。

## 次のステップ

最初の一歩を踏み出したら、少しずつできることを広げていきましょう。  
Claude Codeには、決まった作業を自動化する「スキル」や、役割ごとにAIを分担させる「サブエージェント」  
といった仕組みも用意されており、慣れてくるとより複雑な作業も任せられるようになります。  
これらの詳しい使い方については、別の記事で改めて詳しく解説する予定です。

まずは今日紹介した手順どおりにインストールし、身近な小さな作業を1つ任せてみることから始めてみてください。  
一度動かしてしまえば、次に何を頼めばいいかは自然と見えてくるはずです。

## まとめ

Claude Codeは、ターミナル上で動きファイルの読み書きまで行うAIコーディングエージェントです。  
ネイティブインストーラーかnpmでインストールし、Anthropicアカウントでログインすれば準備は完了します。  
最初は既存プロジェクトの説明作成や簡単な修正など、身近な作業から試すのがおすすめです。  
つまずいたときは焦らず、この記事の対処法を参考に一つずつ確認しながら進めてみてください。
