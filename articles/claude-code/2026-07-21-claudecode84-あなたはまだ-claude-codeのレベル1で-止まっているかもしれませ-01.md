あなたはまだ、
Claude codeの「レベル1」で
止まっているかもしれません。

実はClaude codeには、
その先に6つのレベルがあります。

しかも、すべてのレベルで
使える共通の魔法の一言があります。

それが、                                                                                                                                                                                                                                                                            

「まず私に質問して」です。

レベル1｜1文で終わらせない

プロンプトの最後に、

「まず私に質問して」

と追加するだけ。

Claudeが先にあなたへ質問し、
目的や条件を整理してくれます。

その質問に答えるだけで、
出力の解像度とパーソナライズ度が
一気に上がります。

レベル2｜打たずに話す

タイピングは1分間に約60語。

でも、話すなら
約150語まで伝えられます。

音声入力ツールを使えば、

短時間で、
より長く、具体的で、
情報量の多いプロンプトを作れます。

プロンプトの質を上げたいなら、
文章力より先に
音声入力を使ったほうが早いです。

レベル3｜チャット画面を離れる

Claudeアプリを開き、
「Cowork」タブを使います。

乱雑な資料、
古い企画書、
数字だらけのファイルを
まとめて渡して、

「この資料をもとに、
〇〇を管理する
スプレッドシートを作って。
まず必要なことを私に聞いて」

と指示。

数分後には、
実際に使えるExcelファイルが
完成します。

レベル4｜外部アプリと接続する

左サイドバーから、

カスタマイズ
↓
コネクター

を開きます。

Gmail、
Googleカレンダー、
Granolaなどを接続。

そのうえで、

「私のメール、
カレンダー、
ミーティングノートを確認して、
明日の準備をして」

と依頼します。

ここからClaudeは、
ただ答えるAIではなく、
あなたの情報を横断して動く
アシスタントになります。

レベル5｜一度教えて、何度も使う

毎回同じ説明をする必要はありません。

スキル作成機能を使って、

「私の好みに合わせて
スプレッドシートを作る方法を
Claudeに覚えさせたい。
まず私に質問して」

と入力。

質問に答えたら、
スキルとして保存します。

すると次回からは、
保存したスキルを呼び出すだけで、
あなた専用のやり方を
何度でも再現できます。

レベル6｜モデルを使い分ける

すべての作業を、
同じモデルに任せるのは非効率です。

軽い作業は高速モデル。

重い分析や複雑な作業は、
高性能モデルと
高い推論設定を使う。

同じプロンプトでも、
モデル選びを間違えるだけで
時間も出力品質も大きく変わります。

Claudeを使いこなす人は、
プロンプトだけでなく、
モデルまで使い分けています。

レベル7｜英語でソフトウェアを作る

「Code」を開き、
必要に応じて権限設定を変更。

そして、

「私との30分の通話を
予約できるページを作って。
まず私に質問して」

と伝えます。

質問に答えると、
Claudeがコードを書き、
ページを構築し、
実際に動く状態まで進めます。

プログラミング言語を
完璧に覚えていなくても、

自然な言葉で
ソフトウェアを作れる時代です。

そして、
この7段階すべてに共通するのが、

「まず私に質問して」

という一言。

Claudeに
いきなり答えさせるのではなく、

先に目的や条件を
聞き出してもらう。

これだけで、
出力の質は大きく変わります。
