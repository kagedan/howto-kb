Claude codeに初チャレンジして、無職さんのBrain「人生管理OSの作り方完全マニュアル」を完走しました。

購入者特典で質問できる環境はあるんですけど、気が短くて返答を待てなかったわたしは、お金に糸目をつけずClaude code（とGoogle）に聞きまくりました！

そしたらまあ！ トークン消費量のすごいことすごいこと！  
2日間で75ドルくらい使ってました！ もちろん手動で課金しました。勝手に引き落とされることはないのでご安心ください。

いや、正直めちゃくちゃ楽しいんです。目の前でClaud codeがどんどん作業を進めてくれるところが見られるので（そこをリッチな設定にしてたのでお金がかかってたんですけど）。

しかしさすがに優しい先輩方が「設定見直したほうがいいよ」と止めてくれたので、Claud codeに直接プロセスの最適化や省略化を頼んだ後、モデルや処理速度の変更を行いました。

詳しくはこちらの記事を参考にしました↓

とりあえず普段使いのモデルは一番軽量のHaiku、処理速度（effortというもの）はmediumにしてみました。設定前は、モデルは2段階も上のOpus、処理速度はhighになっていました。

これで月額料金20ドル（今だと3600円くらい）に収まるはず。たぶんね。クレジットを使い切ったら「クレジットないよ」ってClaud codeのほうでエラーが出るので大丈夫です。

いやースリリングな体験でした。面白かったです。Claude codeでやりたいことがたくさんある人が、最新モデル（6/22まで無料期間らしい）で何ができるのかを試しまくってる気持ちが分かる気がする。

しかし同時に、Discordやnoteなど安価なツールのありがたみを噛み締めました。どんなに書き込んでも実質タダ。いや、厳密にはタダではないんですけど、月額500～1000円くらいでこれを楽しめる仕組みに感謝。

そして新しいものを人に教わりながら知っていくことも大切である一方で**「やり方を知っているけど根気が必要なアナログ作業」**への取り組みもきちんとしていきたいなと思いました。

こうして文章を書くことも、パソコンを使ってはいるけど、「自分の頭で文章を構成する」ことに焦点を当てれば、やっていることとしてはアナログに近いと思います。非公開の日記ももっと書いていきたい。

最近SNSで「ドメイン知識（を持つべき）」という話題を見かけるんですけど、それに近いものを自分のために備えたいと感じましたね。別に知識じゃなくてもよくて、簡単なことで、行動を起こせればいいです。料理を楽しむ力とか、掃除で自分をもてなす力とか。

それで、社会で勝てなくても別にいいんです。自分の人生に豊かさをもたらして、暮らしの中で小さな勝利を収めることができれば。デジタルに挑戦するのと同じくらい、アナログも、もっと面白くしたいな！
