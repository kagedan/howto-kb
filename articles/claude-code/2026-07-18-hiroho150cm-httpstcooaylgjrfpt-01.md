# **タイトル『マスターと事業計画書』**

## 第1章　白紙のフォーマット

三沢美咲、32歳。会社を辞めて、自家焙煎のコーヒー店を開くと決めた日から、ちょうど1ヶ月がたちました。

物件の候補も見つけました。貯金も200万円あります。

足りない開業資金は、日本政策金融公庫で借りるつもりでした。ところが公庫のサイトからダウンロードした「創業計画書」のフォーマットを前に、美咲の手は止まります。

「創業の動機」「取扱商品・サービス」「必要な資金と調達方法」「事業の見通し」。

どの欄も、何をどう書けばいいのか分かりません。気づけば3日間、白紙のままでした。

煮詰まった美咲は、商店街のはずれにある古い喫茶店「木曜社」に逃げ込みます。カウンターの奥には、白髪のマスター・榊さんがひとり。

「浮かない顔だね」

事情を話すと、榊さんはコーヒーを差し出しながら、こう言いました。

「わたしは昔、公庫で融資の審査をしていてね。計画書なら、何千枚も読んだよ」

「えっ」

「まず一つ、いいことを教えよう。事業計画書はね、"作文"じゃない。**読む相手がいる手紙**なんだ。君のは誰に宛てた手紙だい?」

「……公庫の、担当者さん?」

「そう。相手は毎日何枚も計画書を読むプロだ。彼らが知りたいことは、実はたった2つしかない。**『この事業は続くのか』『貸したお金は返ってくるのか』**。全部の欄は、この2つの問いに答えるためにある」

👉**教え① 事業計画書は相手への手紙。読み手の問いに答える文書である。**

## 第2章　誰の、どんな困りごとか

翌週、美咲は書きかけの計画書を持って木曜社を訪れます。

「創業の動機」の欄には、こう書いてありました。「コーヒーが好きで、自分の店を持つのが夢だったからです」。

榊さんは一読して、静かに突き返しました。

「好きなのは分かった。でもね、審査員はこう思う。**『好き』はお客の来る理由にならない**、と」

「……厳しいですね」

「事業の核はね、3つの穴埋めで書けるんだ。**『誰の』『どんな困りごとを』『どう解決するか』**。君の店は、誰の何を解決する?」

美咲は考え込みます。候補物件は、駅から少し離れた住宅街でした。

近くに保育園があり、送り迎えの親たちが行き交います。でも、ゆっくり座れる店は一軒もありません。

「……子どもを送ったあとの30分、ほっとひと息つける場所がない人。その人たちに、静かな席と、おいしい一杯を」

「いいね。それが**コンセプト**だ。一行で言えるものは強い。逆に一行で言えない事業は、たいてい計画も曖昧なんだよ」

**👉教え② 「誰の・どんな困りごとを・どう解決するか」を一行で書く。それがコンセプト。**

## 第3章　足で書く市場調査

「次はこの欄だね。『市場と競合』」

「それ、ネットで市場規模を調べて書きました。コーヒー市場は約3兆円で……」

「ストップ」榊さんは笑いました。「3兆円の市場は、君の店には1円も関係ない。審査員が知りたいのは日本のことじゃなく、**あの駅の、あの通りのこと**だ」

榊さんの宿題は、シンプルでした。

平日の朝、候補物件の前に立って、通行人を1時間数えること。近隣のカフェを全部まわって、席数と客層と価格をメモしてくること。

一週間後、美咲のノートは数字で埋まっていました。

朝8時台の通行量は約400人、うち保育園帰りとみられる大人が約60人。半径500メートルにカフェは2軒、どちらもチェーン店で、ベビーカーで入りにくい構造。

「これだよ」榊さんは頷きました。「**市場は検索するものじゃなく、足で数えるもの**。そして競合欄に書くべきは悪口じゃない。『相手ができていなくて、自分にはできること』。君なら?」
