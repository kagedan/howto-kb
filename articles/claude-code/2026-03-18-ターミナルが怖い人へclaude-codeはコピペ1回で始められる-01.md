**「Claude Codeってすごいらしい」。**

そんな話を聞いて調べたら、最初に出てきたのが「ターミナル」「CLI」という見慣れない言葉でした。

黒い画面に白い文字。  
プログラマーの世界だ、と思って閉じた経験はありませんか。

実は、あの黒い画面にコマンドを1行貼り付けるだけで、AIエンジニアがあなたのPCに住み着きます。ファイルを読んで、書き換えて、エラーまで直してくれる。話すだけじゃなく「手を動かすAI」です。

この記事では、ターミナルを一度も開いたことがない方でも迷わずClaude Codeを始められるよう、やさしく案内します。

## そもそもClaude Codeって何？

![](https://assets.st-note.com/img/1773793164-qaDZyOonWGus3Bwix0bXjHAU.png?width=1200)

ChatGPTやClaude（Web版）は、ブラウザの中で会話するAIです。  
Claude Codeは、あなたのPC上で動くAIアシスタントです。

違いは「手を動かせる」こと。

たとえば、フォルダの中にある100個のファイル名を一括で変えたいとき。Web版のAIに聞くと「**こうやればできますよ**」と方法を教えてくれます。Claude Codeは「**やっておきました**」と実際に変えてくれます。

プログラミングの相談はもちろん、ファイル整理、レポートの下書き、データの集計まで。日本語で頼めば、あとは勝手にやってくれます。

## インストール手順

![](https://assets.st-note.com/img/1773793178-Lp9EzVctMlQqkjrPh0sdYTag.png?width=1200)

ここが最大の山場ですが、やることは「**コマンドを1行コピペする**」だけです。

---

### Macの場合

「ターミナル」を開きます。  
キーボードで `⌘ + スペース` を押して「ターミナル」と入力すれば見つかります。

開いたら、以下をまるごとコピーして貼り付けてください。

```
curl -fsSL https://claude.ai/install.sh | bash
```

Enterキーを押したら、あとは待つだけです。

---

### Windowsの場合

「PowerShell」を開きます。  
スタートメニューで「PowerShell」と検索してください。

```
irm https://claude.ai/install.ps1 | iex
```

これだけです。  
「え、これだけ？」と思った方、はい、これだけです。

## はじめての起動

![](https://assets.st-note.com/img/1773793190-pOwT847BLaQMCqZGJFEl23Xn.png?width=1200)

インストールが終わったら、同じ画面にこう打ちます。

```
claude
```

あとは日本語で話しかけるだけです。

「このフォルダの中身を教えて」  
「READMEファイルを作って」

こんな風に、普段どおりの言葉で頼めば動いてくれます。

## 会話中に使える便利機能

Claude Codeと会話している最中に `/`（スラッシュ）を打つと、特別なコマンドが使えます。よく使うものだけ紹介します。

会話を終えるときは `/exit` と打つか、`Ctrl + C` を2回押します。

## 困ったときの対処法

![](https://assets.st-note.com/img/1773793208-tuamoRN62nvkQG4Iis9EhbpH.png?width=1200)

### 「command not found」と表示される

インストールしたのに「**コマンドが見つかりません**」と言われるパターンです。

PCがClaude Codeの場所をまだ認識していません。

**やること**: ターミナルを一度閉じて、もう一度開いてください。  
これだけで直ることがほとんどです。

---

### ログインがうまくいかない

ブラウザが開かない、またはエラーが出る場合です。

**やること**: 会話中に `/logout` と打ってから `/login` をもう一度試してください。  
セキュリティソフトがブラウザの起動をブロックしていることもあります。

---

### 動作が止まった・フリーズした

**やること**: `Ctrl + C` を押してください。  
実行中の処理が止まります。そのあと、もう一度話しかければ続きから対応してくれます。

---

### それでも解決しない場合

Claude Codeに直接聞くのが一番早い方法です。  
「インストールできないんだけど、どうすればいい？」と日本語で相談してみてください。自分自身のことなので、的確に答えてくれます。

公式ドキュメントも参考になります。

## よくある質問

**Q: プログラミングができなくても使えますか？**  
使えます。「このファイルの中身を教えて」「エラーを直して」と日本語で話しかけるだけです。Python、JavaScript、TypeScriptなど主要な言語をすべてサポートしています。

**Q: 勝手にファイルを消されませんか？**  
Claude Codeは、ファイルの削除や大きな変更をする前に必ず確認を求めます。画面に「y/n」と表示されたら、内容を読んで「y」（はい）を押してください。安心して使えます。

**Q: ChatGPTと何が違いますか？**  
ChatGPTはブラウザの中で会話するだけです。Claude CodeはあなたのPC上で実際にファイルを操作できます。「教えてくれる」と「やってくれる」の違いです。

**Q: どんな作業を頼めますか？**  
コードの修正、ファイルの整理、データの変換、レポートの下書き、エラーの修正など。「こういうことできる？」と聞けば、できるかどうかも教えてくれます。

## まとめ：今日やること

Claude Codeを始めるのに必要なステップは3つだけです。

1. ターミナル（またはPowerShell）を開く
2. インストールコマンドを貼り付けてEnter
3. `claude` と打って話しかける

ターミナルは怖くありません。  
「**コマンドを貼る場所**」だと思えば、それだけです。

一度入れてしまえば、あとは日本語で頼むだけ。ファイル整理、コード修正、レポートの下書き。あなたの「面倒」を引き受けてくれるAIエンジニアが、PCの中で待っています。
