## はじめに

前提条件として、これは 2026年3月時点での話であり、来月には状況が変わっている可能性が充分にある。企業で導入する場合、手続きが何かと面倒かとは思うが、定期的に見直し、すぐに切り替えられるようにすべきだ。そんなの無理という企業は、競争力を保つためのリスクが検討できていないと自覚すべきだ。
