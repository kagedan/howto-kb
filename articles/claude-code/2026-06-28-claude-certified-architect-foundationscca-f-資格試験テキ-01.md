# Claude Certified Architect – Foundations（CCA-F） 資格試験テキスト

<<教材の目的と対象者>>
本教材は、Anthropic が提供する Claude 関連の入門レベル資格「Claude Certified Architect – Foundations（CCA-F）」の合格を目指す方のための、出題範囲を網羅した日本語学習テキストです。
Claude を業務や個人開発で触ったことがあるものの、エージェントアーキテクチャ・MCP・Claude Code・プロンプト設計・コンテキスト管理といった試験出題範囲を「資格試験で問われる粒度」で言語化できていない初中級者を対象としています。
<<学習内容と目標>>
- ドメイン1：エージェントアーキテクチャとオーケストレーション（27%）を体系的に理解し、シングル/マルチエージェント・ハブ&スポーク・タスク分解の設計判断ができる
- ドメイン2：ツール設計とMCP統合（18%）を習得し、Tool Use と MCP サーバー/クライアントの設計ベストプラクティスを説明・実装できる
- ドメイン3：Claude Code 設定とワークフロー（20%）を理解し、CLAUDE.md 階層・スキル・スラッシュコマンド・フック・CI/CD 統合を構築できる
- ドメイン4：プロンプトエンジニアリングと構造化出力（20%）を習得し、Few-shot・JSONスキーマ・バリデーション・反復洗練・マルチパスレビューで高品質な構造化出力を設計できる
- ドメイン5：コンテキスト管理と信頼性（15%）を理解し、コンテキスト劣化防止・エスカレーション設計・ヒューマン・イン・ザ・ループを設計できる
- 60問のシナリオベース模擬試験で合格ライン（720/1000点）を安定して超えられる
