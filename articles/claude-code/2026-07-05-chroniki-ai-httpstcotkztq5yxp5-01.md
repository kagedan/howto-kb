■ コードゼロで「$50,000相当」のWebサイトが数時間で動く

最近チュートリアル動画を見ていて、思わず何度か見返してしまいました。

Claude Fable 5を使って、映画のプロモーションサイトみたいなリッチなWebサイトを1セッションで仕上げてしまう内容で、「コードを一行も書かなくてよかった」という仕上がりが、本当にシネマ級なんです。

動画内では「$50,000相当のクオリティ」という表現も出てきていて、最初は大げさだなと思ったんですが、実際に完成サイトを見るとスクロールのたびに商品が分解・組み立てされるアニメが動いて、これが数時間＋APIコスト数千円レベルで再現できると聞いてびっくりしました。

このチュートリアルを通して「あ、Webサイトの作り方の常識が変わってきてるかも」と感じたので、手順と仕組みを整理してみます🐾

■ 一言でいうと、Claude Fable 5の何がすごいのか

まず結論だけ3行でまとめると、こんな感じです。

・Claude Fable 5は「企画→素材生成→コーディング→デバッグ」を1セッションで自律的にやってくれるモデル

・MCPという仕組みで外部の画像・動画生成AIとシームレスにつながる

・コードを書かなくても、ブランドの世界観を言語化さえすれば、リッチなWebサイトが完成する

以前のAIアシスタントは「聞いたら答えてくれる」という感じでしたが、Fable 5はもっと「一緒に作業してくれるディレクター」に近いイメージです。

指示を出したら、必要な素材を外部ツールに発注して、コードを書いて、表示を確認して、エラーが出たら自分でデバッグして、また確認して——という一連の流れを人間がいちいち指示しなくても回してくれます。

■ そもそもClaude Fable 5って何？

Anthropicが2026年6月にリリースした最新のAIモデルです。

Anthropicの公式ページによると、Fable 5は複数日にまたがる長時間タスクを自律で実行できる設計になっていて、コードを書いてビジュアルを確認してデバッグまで自分でやってくれるとのこと。

以前のClaude（たとえばClaude 3.5 Sonnetなど）も優秀でしたが、Fable 5の大きな違いは「1回のセッションで完結する作業の規模と自律性が明らかに上がった」という点です。

僕がチュートリアルを見ていて感じたのは、途中でつまずいた箇所をAI自身が「これは違う」と判断してやり直している場面があったことで、人間が「エラーが出たのでどうすればいいですか？」と聞かなくても、勝手にリカバリーしてくれるんですね。

これはコーディングに限らず「長くて複雑な制作フロー全体を任せられる」という体験につながっていて、ここが一番の変化かなと感じています。

■ MCPって何？（まずここを押さえると全部わかる）

MCP（Model Context Protocol）は、ClaudeのようなAIモデルと、外部のツールやサービスをつなぐための「共通のコネクタ規格」です。

ちょっとわかりにくいので、USB-Cで例えてみます。

USB-Cというコネクタ規格があれば、パソコンに「マウスでも」「外付けSSDでも」「モニターでも」つなげますよね。どのメーカーかは関係なく、規格さえ合えばつながる。

MCPはそのAI版で、Claudeという本体に「動画生成AIでも」「画像生成AIでも」「コード実行環境でも」、MCPに対応してさえいれば何でもつなげられる設計なんです。

今回のチュートリアルでは、ClaudeにHiggsfield（ハイグスフィールド）という動画生成AIをMCP経由でつないでいました。

Higgsfieldとは、テキストの指示から高品質な動画クリップを生成できるAIサービスです。

これをMCPでつなぐと何がうれしいかというと、Claude Codeが「このシーンの映像素材が必要だ」と判断した瞬間に、自分でHiggsfieldに発注して、生成された素材を受け取って、そのままWebサイトの実装に組み込む、というフローが全部Claudeの中で完結するんです。

人間は「ツールを切り替えてダウンロードしてアップロードして……」という面倒な手順をいっさい踏まなくていい。これがMCPの地味にすごいところです。
