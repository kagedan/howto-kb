【速報】Claude Code に「Artifacts」が来た。

コーディングセッション全体が、リンク一発でチームに共有できるライブページになる。
「作業が終わってから説明する」という慣習が、根本から変わるかもしれない。

2026年6月18日、Anthropic が Claude Code に「Artifacts」機能をベータ公開した。

何が変わるかを整理すると：

・Claude Code のセッション全体（コード・会話・接続ツール）を素材に、インタラクティブな HTML ページを自動生成
・そのページはプライベートURLで即座に共有可能
・URLは変わらず、Claudeが変更を加えるたびにページがリアルタイム更新される
・閲覧者は Claude Code のライセンス不要、ブラウザさえあれば見られる

Before：作業が終わるまで誰も中身を見られない
After ：作業中からチームが同じ画面をリアルタイムで確認できる

ユースケースを具体的に見るとイメージが掴みやすい。

Anthropicが挙げているのはこのあたり：
① PR（プルリクエスト）の内容をチームにウォークスルー共有
② インシデント発生時のタイムライン自動生成
③ アーキテクチャ概要の可視化ページ
④ リリースチェックリストのライブ管理

❌ ありがちな誤解：「結局、スクリーンショットやドキュメントを別途作る手間が減るだけでは？」
⭕ 実態：セッションの全コンテキスト（コードベース・プラグイン・会話履歴）から自動生成されるので、人間が説明資料を書く工程が丸ごとなくなる

→ 「ドキュメント会議」そのものが不要になる可能性がある。

現時点で把握しておくべき制限事項も整理しておく。

【対象プラン】
Team・Enterprise のみ（ベータ）
Free・Pro は現時点では非対応

【セキュリティ設計】
・デフォルトは「自分のみ閲覧可能」
・共有は同一組織の認証済みメンバーに限定
・Enterprise は管理者がロール別アクセス制御・保存期間設定・監査ログが使える
・ページは最大 16MiB、外部ネットワーク呼び出し不可（完全な閉域設計）

→ 「社外に情報が漏れるのでは？」という懸念には、設計段階で対処済み。
エンタープライズ利用でも安心して使える構造になっている。

Claude Code を業務で使っている方、チームへの展開を検討するなら今が動きどき。

Anthropic の売上ランレートは 2025 年末の約 90 億ドルから、2026 年 5 月には 470 億ドル超に急拡大している。
その牽引役が Claude Code のエンタープライズ採用。
今回の Artifacts はその流れを加速させるための、チーム協業レイヤーの強化だ。

→ Claude Code は「個人の開発ツール」から「チームの知的作業インフラ」へシフトしつつある。
経営判断のスピードに直結する機能になってきた。
