### やってみてわかったこと

**ツールが止まると自分も止まる**

Claude Codeが使えない3日間、作業が完全に止まった。

Anthropicは「ユーザーが予想よりもはるかに速く使用制限に達している」と認めていた。だいたい他のLLMも、新しいモデルが発表される前は、変な挙動する気がします。

ツールへの依存度が上がるほど、ツールの安定性がリスクになる。PMとして当然考えるべきことだと体験した。

**待った分、再開後に一気に進んだ**

制限が解消した04/09、バグ修正と新機能を一気に実装できた。3日間で頭の中で整理できていたのかもしれない。止まっていた時間が無駄だったとは思っていない。

AIが止まった間も、人間がやれることはある。

**LRカードの招き猫が金色のドラえもんになった**

最高レアリティのカードは招き猫にした。生成したら、金色のドラえもんみたいになった。まあ、強そうだからいいか。
