## ① 🤖 【Microsoft Build 2026】「プロンプト不要の24時間AIワーカー」Scout登場 — 7つの新モデルと「エージェントファースト」なWindowsの全貌

Microsoft Build 2026で、MicrosoftはAIの競争を「モデルの賢さ」ではなく「**仕事環境そのものを握る**」戦争として再定義しました。

最大の目玉は「**Scout**」。OpenClawを搭載した初の「Autopilot」エージェントで、ユーザーがプロンプトを打たなくても**先回りで**会議の準備・To Doの洗い出し・情報収集を行います。Microsoft Frontierプログラム経由で提供開始。あわせて\*\*7つの新社内AIモデル（MAI）\*\*も発表されました。

エージェント対応のインフラも一気に整備されました：

* **Aion 1.0 Instruct / Plan**：推論・計画・ツール使用・デバイス上エージェントワークフロー向けのローカルWindowsモデル
* **Web IQ / Work IQ / Foundry IQ / Fabric IQ**：エージェントにWebデータ・社内コンテキスト・モデル構築情報・データコンテキストを提供する4種の「IQ」
* **GitHub Copilotアプリ**：OpenAI・Anthropic・Googleのモデルにアクセスできる「エージェントネイティブなデスクトップ環境」。Claude CodeやCodexへの直接対抗です
* **Agent 365 / MXC**：Microsoft 365をエージェントの「仕事用メモリ」に。Defender・Entra・Purviewがガードレールを担う

Satya Nadella CEOはキーノートで「AIスタックはコンピューティング・モデル・コンテキスト・ツール・ランタイム・セキュリティで構成される」と宣言。エージェントにはチャットボックス以上のものが必要——チップ・社内データ・権限設定・制御ツール・監査証跡——この全体を一社で提供しようとしているのがMicrosoftです。

## ② 💼 【OpenAI反撃】CodexがOfficeワーカー向け汎用ツールに進化 — 週間利用者500万人・非開発者が急増中

OpenAIはMicrosoft Buildに合わせてCodexの大型アップデートを発表。**6つの職種別プラグイン・62種のアプリ・110のスキル**を追加し、コーディングツールからホワイトカラー全般向けの「汎用ワークプレースツール」へと進化させました。

**Codexの週間利用者数は現在500万人超**。非開発者の利用が全体の約20%を占め、開発者より**3倍以上速いペースで増加**しています。

職種別プラグインの活用例：

* **営業チーム**：Salesforce・HubSpot・Outreach・Clay・Slackと接続し、アカウント優先順位付け・バイヤーリサーチ・フォローアップ草案・CRM更新・リスク案件レビューを自律実行
* **アナリスト**：Snowflake・Databricks・Hex・Tableauと連携してデータ分析を完結
* **クリエイティブチーム**：Figma・Canva・Shutterstock・Picsartと接続

さらに「**Codex Sites**」も発表。プロンプト一つでデプロイ可能なウェブサイトやアプリを生成し、共有URLで即座にチームに配布できます。発表動画は300万回以上再生されています。

## ③ 🖥️ 【オープンソース旋風】Hermes Desktopが全OS対応で登場 — トランプがAI大統領令に署名・Anthropicがプロジェクトグラスウィング拡大

**Hermes Desktopが簡単インストールで全OS対応** Nous Researchの自己改善型AIエージェント「Hermes」がmacOS・Windows・Linux向けデスクトップアプリとして正式リリース。MIT ライセンスで無料公開されています。Telegram・Discord・Slack・WhatsApp・Signal・メール・コマンドラインをまたいで**プラットフォーム横断の永続メモリ**を持ちます。スキル自動生成・タスクスケジュール・Webブラウジング・独立したサブエージェントの起動・Docker/SSH/Modalなどのサンドボックス内実行が可能。発表動画は200万回以上再生されました。

あわせてCognitionの「**Devin Desktop**」も登場。Devin・Claude Code・Codex・カスタムエージェントをAgent Client Protocolで一元管理できる無料デスクトップアプリです。Perplexityも「**Perplexity Computer**」を発表——デバイス上のローカルモデルとクラウドのフロンティアモデルをタスクに応じて自動振り分けし、プライベートデータはデバイス上に留めます。

**トランプ大統領がAI安全保障大統領令に署名** フロンティアAIモデルのリリース前に、対象企業が政府への自発的な事前共有を行う仕組みを創設。分類されたサイバーセキュリティベンチマークの整備、AIによる脆弱性検出への連邦支援も含まれます。強制的なモデルライセンス制度は設けられませんでした。

**Anthropicがプロジェクトグラスウィングを拡大** 15カ国以上の約150組織に拡大。重要インフラチームがソフトウェアの脆弱性を発見・修正できるよう支援しています。マーティン・スコセッシ監督がBlack Forest LabsのアドバイザーにFLUXによるストーリーボードセッション後に就任したことも話題です。
