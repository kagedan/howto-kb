## はじめに

このシリーズは毎日Claude Codeで何かを作ってnoteに投稿しています。

ただ、Claude Codeには使用制限があります。制限が来た時でも投稿が途切れないよう、前もって書いておいた記事です。

## やったこと

Day.42で作ったシンプルなカレンダーアプリを、Googleカレンダー・Apple Calendarにある機能を参考に大幅改良しました。

## 追加した機能

**予定の詳細設定：**

* 繰り返し予定（毎日・毎週・毎月・毎年・カスタム）
* 場所・URLリンク
* 優先度（高・中・低）
* 所要時間見積もり
* 体力/集中度（重い・普通・軽い）
* 締切設定

**通知・制約：**

* ブラウザ通知（5分・10分・30分・1時間前）
* 重複予定の自動検知
* 移動時間バッファ（前後）
* タイムゾーン設定

**ログ・分析：**

* 実績時間の記録（予定vs実際）
* 完了/未完了の管理
* 集中度ログ
* カテゴリ別時間集計グラフ

**表示・操作：**

* 日・週・月ビューの切り替え
* タイムライン表示
* ドラッグ&ドロップで予定を移動
* 検索・フィルタ

**その他：**

## できなかった機能

以下はGitHub Pagesの制限上、実装できませんでした。

* 参加者招待・出欠管理（サーバーが必要）
* 添付ファイル（ストレージが必要）
* 外部カレンダーとの同期（APIが必要）
* 場所連動通知（位置情報APIが必要）
* 共有権限の設定（データベースが必要）

## Claude Codeの使用制限について

Claude Codeには1日あたりの使用制限があります。制限に達すると、その日はそれ以上コードを生成できなくなります。

毎日投稿を続けるためには、制限が来た日でも出せる記事を前もって用意しておく必要があります。これがその記事です。

制限が来た日は新しいものを作れませんが、「作り続けている記録」は途切れません。

このシリーズはプログラミング未経験の高校生がClaude Codeを毎日使い続けた記録です。
