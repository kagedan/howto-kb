# Claude Codeのソースコード50万行が流出、Anthropic 1週間で2度目の事故

**Claudeユーザーは知っておくべき** — 2026年3月最終週、Anthropicで前代未聞の事態が起きています。次世代モデル「Mythos」の情報漏洩、Claude Codeの全ソースコード流出、そして米政府との裁判勝訴。わずか1週間に起きた3つの出来事を整理します。

## TL;DR

* 3/27：設定ミスで次世代モデル「Claude Mythos」の存在が判明
* 3/31：Claude Codeのソースコード約50万行（1,900ファイル）が流出
* 3/26：トランプ政権によるClaude使用禁止に対し、裁判所がAnthropicの主張を認める

> 「Anthropic、1週間で2度の情報流出。AI企業のセキュリティ体制に疑問の声」

## 時系列で見る3つの出来事

### 3月26日：米国政府との裁判で仮差止命令を獲得

トランプ政権下で、国防総省（DOD）が連邦政府機関でのClaude使用を禁止する動きを見せていました。これに対しAnthropicは「憲法修正第1条（言論の自由）への報復だ」として提訴。

サンフランシスコの連邦判事はAnthropicの主張を認め、**仮差止命令**を出しました。これにより、政府機関でのClaude使用禁止は一時的に差し止められています。

### 3月27日：次世代モデル「Claude Mythos」の情報が流出

Anthropicの設定ミスにより、開発中の新モデル「**Claude Mythos**」に関するドラフトブログ記事が誤って公開されました。約3,000ファイルが一時的にアクセス可能な状態になっていたと報じられています。

Anthropicは事態を認め、「Mythosはこれまでで最も高性能なモデル」とコメント。高度な推論機能を備えていることが明らかになりました。

### 3月31日：Claude Codeのソースコード50万行が流出

Mythosの情報漏洩からわずか数日後、今度は**Claude Codeのソースコード**が流出。約50万行、1,900ファイルに及ぶコードが外部に公開される事態となりました。

これはAnthropicにとって1週間で2度目のセキュリティ事故であり、業界内で「何が起きているのか」と懸念の声が上がっています。

## それぞれの詳細

### Claude Mythosとは何か

流出した情報から判明していること：

* **「これまでで最も高性能なモデル」** とAnthropicが表現
* **高度な推論機能**を搭載
* Claude 5との関連は不明（別ラインの可能性も）

正式発表はまだありませんが、2026年春〜夏のリリースが予想されています。

### ソースコード流出の影響

50万行のソースコードが公開されたことで：

* Claude Codeの**内部アーキテクチャ**が明らかに
* 競合他社による**技術分析**が可能な状態に
* セキュリティ上の**脆弱性**が発見されるリスク

ただし、モデルの学習データや重み（weights）は含まれていないため、Claude自体の複製は不可能です。

### 政府との対立の背景

Anthropicと米国政府の対立は、AI規制をめぐる大きな議論の一部です。

* トランプ政権は特定のAI企業に対し、政府調達からの排除を進めていた
* Anthropicは「技術的な理由ではなく、政治的な報復だ」と主張
* 裁判所の判断は、AI企業の権利を認める先例となる可能性
