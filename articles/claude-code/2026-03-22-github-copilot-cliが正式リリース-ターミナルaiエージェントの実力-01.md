はじめに
2026年2月25日、GitHub Copilot CLIが全有料プランで正式リリース（GA）された。2025年9月のパブリックプレビュー開始から約5ヶ月、数百件の改善を経ての一般提供だ。
ターミナルで動くAIコーディングエージェントといえば、すでにClaude Codeが先行して存在感を示している。そこにGitHubが本格参入してきた形になる。Copilot CLIは単なるコード補完ツールではなく、計画・実行・レビューまでこなすエージェント型ツールに仕上がっている。ターミナル派の開発者にとって選択肢が増えたのは間違いない。
