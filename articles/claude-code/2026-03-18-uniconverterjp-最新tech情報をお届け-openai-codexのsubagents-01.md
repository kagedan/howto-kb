＼最新Tech情報をお届け／
✨ OpenAI CodexのSubagents、全開発者に開放

AIコーディングで大きなプロジェクトを進めると、会話のコンテキストが膨れ上がって処理が重くなる問題がありました。
Subagentsはこれを解消するために、メインエージェントが複数の専門サブエージェントを自動生成して並列処理します。
各エージェントへの個別指示も可能で、TOMLファイルでカスタムエージェントも定義できます。
CodexアプリとCodex CLIどちらからでも使えます。
