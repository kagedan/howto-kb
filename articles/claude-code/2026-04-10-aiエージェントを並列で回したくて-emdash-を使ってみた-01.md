はじめに
AIコーディングエージェントを日常的に使っていると、ある壁にぶつかります。「1つのエージェントが作業している間、自分も同じリポジトリで別タスクを進めたい」「複数のエージェントを同時に走らせたいけど、ブランチやファイルが衝突してしまう」——こうした並列開発の悩みです。
Emdash は、まさにこの課題に向き合ったツールです。Claude Code や Codex といったCLIエージェントを、それぞれ独立した Git worktree で隔離し、並列に起動・管理できるデスクトップアプリケーションになっています。
