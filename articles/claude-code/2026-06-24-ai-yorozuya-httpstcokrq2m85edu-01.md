# 知識ゼロから始める、AIエージェントの作り方と育て方のすべて

※動画で見たい方はこちら

請求書が20枚。

取引先と金額と日付を、1枚ずつExcelに手で打ち込んでいく。

気づいたら2時間。

あの地味な作業、ほんとにしんどいですよね。

でも、その作業。

もう「やっといて」と頼むだけで、一瞬で終わります。

僕が今回話したいのは、Claude Codeというツールです。

名前は聞いたことあるけど、難しそうで触れてない。

コードなんて分からないし、自分には無理そう。

そう思って手を出せていない人にこそ、読んでほしいです。

コードは一行も書きません。

日本語で話しかけるだけ。

知識ゼロから始められます。

この記事を読み終わるころには、あなたの一番面倒な定型作業をひとつ、AIに渡せるようになっています。

# 請求書20枚のExcel転記が、頼むだけで一瞬で終わる時代になった

まず、AIエージェントって何？という話から。

ChatGPTみたいに「質問したら答えてくれるAI」。

あれとは、ちょっと違うんですよね。

AIエージェントは、答えるだけじゃない。

あなたの代わりにパソコンを操作して、作業そのものをやってくれます。

たとえば、さっきの請求書。

フォルダに20枚入っているとします。

「このフォルダの請求書を、取引先・金額・日付に分けてExcelにまとめて」

そうお願いするだけ。

ほら、一瞬でまとまる。

1枚ずつ手で打っていた2時間が、確認するだけの5分になります。

これ、めちゃくちゃ時短ですよね。

もうひとつ。

動画編集のあとに残る、ぐちゃぐちゃのバックアップフォルダ。

画像も音声もごちゃ混ぜで、何がなんだか分からない。

「このフォルダの中身を、データの種類ごとに仕分けして」

これだけで、きれいに整理してくれます。

もっとすごいのもあります。

世界中の貿易ルート、フライトの遅延、抗議活動や紛争が起きている場所。

それを全部リアルタイムで映すアプリ。

下にスクロールすれば、世界中のニュースも全部見られる。

こんなものまで、Claude Codeで作れてしまうんです。

僕の知り合いは、自分用のダイエットアプリを作っていました。

毎日の食事を写真で撮って投げると、AIがカロリーと栄養を分析してくれる。

「もうお昼なのにカロリーオーバーですよ」

そんなことまで教えてくれるそうです。

これ、2時間くらいでできたらしいんですよ。

何時間もかかっていた作業が、5分で終わる。

もう、それが当たり前になりつつあります。

# なぜ数あるAIエージェントの中で、Claude Codeから始めるのか

AIエージェントには、いくつか種類があります。

一番有名なのが、AnthropicのClaude Code。そしてGoogleのアンチグラビティ。OpenAIのCodex。

番外編だとManusやGenspark。

たくさんあるんですよね。

じゃあなぜClaude Codeから始めるのか。

正直に言うと、性能が断トツでNo.1、とは言いません。

今この時点では一番使えると僕は思っています。

でもCodexが追いついてきたり、期待度が高い。

先週はこれが強かったけど、今週はこっち。

そういう世界なんですよね。

それでもClaude Codeをすすめる理由はひとつ。

これらのソフトは、ほとんど同じ仕組みで動いています。

そして、その仕組みの基準を最初に作ったのがClaude Codeなんです。

つまり、後から出た他社のソフトは、Claude Codeが作った型をもとにしている。

だからClaude Codeの使い方や考え方を理解すれば、他のツールの理屈も一気に分かるようになります。

最終的にどれを使ってもいい。

でも最初の1個としては、ここが一番おすすめなんです。

料金にも触れておきます。

Claude Codeでエージェントを使うなら、プロプランから。

月20ドル、だいたい3000円くらいです。

東大クラスの頭脳を、何人雇っても月3000円。

そう考えると、安いですよね。

ただ、今すぐ契約しないでください。

これは課金をすすめる記事じゃないので。

この先の操作を読んで、「これなら自分でも使えそう」「3000円の価値はある」。

そう感じた人だけ、契約すればいいと思います。
