「Claude Codeをやった方がいい」という情報、Xでよく見かけますよね。

私もずっと気になっていました。いざ調べてみると「何をどうすればいいのか全然わからない」状態で、長い間足踏みしていました…。

ターミナルさえもわからなかった元体育会系の私が、プログラミング知識ゼロから自力でClaude Codeをセットアップした方法を、お伝えします。

## 「ターミナル」って何？

セットアップを始める前に、意味がわからなくて詰まったのが「ターミナル」という言葉。「1分でできる！」という動画でも、ターミナルが何なのか教えてくれませんでした。

正直に言います。**私、今でもよくわかっていません。**

Claude AIに聞いて説明を読みましたが、それでもふんわりした理解しかありません。でも、それで全然大丈夫でした。

大事なのは「ターミナルを完璧に理解すること」ではなく、「ターミナルを開いて、指示通りに進められること」です。

小学生でもわかるようにターミナルについてClaudeがイラストを作ってくれたので、理解してから進みたい人はこれを見てください。

## ターミナルの開き方（Macの場合）

※今回はMacでの設定方法のみご紹介します。Windowsは操作名が異なるため、別途調べることをおすすめします。

**キーボードの一番上の段を見てください。**

虫眼鏡のマークと「F4」と書かれたキーがあるはずです。そこを押すと、アプリ一覧が表示されます。

![](https://assets.st-note.com/img/1774398746-3Xh69b2enyBmHQGONzqA5luV.jpg?width=1200)

これです。

検索バーに「ターミナル」と入力すると、黒い画面のアプリが出てきます。

この黒い画面、なんか「コードが書けそう」な雰囲気ありますよね。これがターミナルです。

## セットアップの全体的な流れ

### ① Claude公式サイトを開く

Googleで「Claude Code」と検索すると、[公式ページ](https://code.claude.com/docs/ja/overview)が出てきます。そこにインストール手順が書かれているので、上から順番に進めていきます。

### ②Claudeサブスクに入る

Claude Codeを使うにはサブスクに入る必要があります（または[**Anthropic Console**](https://console.anthropic.com/) アカウントが必要らしい）。

### ③ターミナルを開く

F4のコマンドキーを押し、ターミナルを開きます。

### ④ターミナルにコードを貼り付ける

![](https://assets.st-note.com/img/1774399217-qSGUMdeLnfX4PikQ1mpvFgVC.png?width=1200)

ここからはステップに沿って、コピーして、ターミナルに貼り付けてEnterを押すだけ。

## エラーが出ても焦らないで

**普通にエラーが出ます。**

私も最初、エラーが出るたびに「もうダメだ」と思いました。でも大丈夫。エラーが出たときの対処法があります。

**スクリーンショットを撮って、Claude AIに聞く!!!!!!**

これだけです。

![](https://assets.st-note.com/img/1774399351-QAEmFc6btG1uWiaryCBDMLRe.png?width=1200)

するとClaudeが「これはこういう状態だから、次はこれを貼ってみて」と教えてくれます。

**それでもうまくいかなければ、また聞けばいい。**

「あなたの言う通りやったけど、うまくいきませんでした。どうすればいいですか？」

これを繰り返すだけです。私もこのやり取りを何度も繰り返して、気づいたらセットアップができていました。

## まとめ

Claude Codeのセットアップで大事なことは、たった2つです。

* **ターミナルを開けるようになる**
* **エラーが出たらClaudeに聞きながら進める**

完璧に理解しなくていいです。私もわかっていません。それでも使えるようになりました。

「難しそう」と思って足踏みしていた方、今日から始めてみてください。意外と、なんとかなります。
