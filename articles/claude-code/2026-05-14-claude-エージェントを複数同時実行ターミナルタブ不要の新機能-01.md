# Claude エージェントを複数同時実行！ターミナルタブ不要の新機能

## 概要

Claude に、複数のエージェントセッションを同時に管理・実行できる新機能が追加されました。1つのエージェントから多数のエージェントへとスケールアップする、最もスマートな方法です。

## 何が変わるのか

### 複数セッションの同時ディスパッチ

このコマンドを実行するだけで、複数のセッションを一度に起動・管理できます。各セッションは**ターミナルタブを占有することなく**バックグラウンドで動き続けます。

もうターミナルのタブを何枚も開いて行ったり来たりする必要はありません。

### 一目でわかるセッション管理

以下の情報をダッシュボード形式で把握できます：

* ▶️ **実行中**のエージェント
* ⏳ **あなたの入力待ち**のエージェント
* ✅ **完了済み**のエージェント

### シームレスな割り込み・再開

* **インラインで返信**して、待機中のセッションをすぐにアンブロック
* **任意のセッションに自由に出入り**しても、作業の流れを見失わない

## まとめ

| 従来 | 新機能 |
| --- | --- |
| ターミナルタブを複数開いて管理 | タブ不要でバックグラウンド実行 |
| 各エージェントを個別に監視 | 一画面でステータスを一覧確認 |
| コンテキストが途切れがち | セッションの出入りが自由 |

エージェントを使った並列作業のワークフローが、大幅に効率化されます。
