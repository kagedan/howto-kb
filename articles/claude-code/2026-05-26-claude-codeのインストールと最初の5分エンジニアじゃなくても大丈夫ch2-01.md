# Claude Codeのインストールと最初の5分【エンジニアじゃなくても大丈夫】

「インストールって難しそう……」

そう思って後回しにしている人へ。Claude Codeのセットアップは、思っているより全然難しくない。やることは3ステップだけで、所要時間は10分もかからない。

この記事ではMacを前提に説明する。Windowsでも基本的な流れは同じだ。

## 事前に必要なもの

* **Anthropicのアカウント**（Claude.aiで使っているアカウントがあればOK）
* **APIキー**（後で説明する）
* **Node.js**（インストールされていなければ入れる）

Node.jsとは、JavaScriptというプログラミング言語をパソコン上で動かすための土台みたいなもの。Claude Codeを動かすために必要だが、自分でプログラムを書く必要はない。

## Step 1: Node.jsを入れる（まだの人だけ）

ターミナルを開く。Macの場合、スポットライト検索（Cmd+Space）で「ターミナル」と検索すると出てくる。

黒い画面が開いたら、以下を貼り付けてEnterを押す：

`v18.0.0` のような文字が出れば、すでに入っているのでこのStepはスキップでOK。

「command not found」と出た場合は、[nodejs.org](https://nodejs.org) にアクセスして「LTS版」をダウンロード・インストールする。インストール後にターミナルを再起動して、もう一度 `node --version` を実行してバージョンが表示されれば完了。

## Step 2: Claude Codeをインストールする

ターミナルで以下を実行する：

```
npm install -g @anthropic-ai/claude-code
```

`-g` は「グローバル（どこからでも使えるように）インストール」という意味。

インストールが完了したら：

バージョン番号が表示されればインストール成功。

## Step 3: APIキーを設定する

Claude Codeを使うにはAnthropicのAPIキーが必要だ。

APIキーとは、「このユーザーがClaude Codeを使う権限を持っています」という認証のための文字列のこと。

**取得方法：**

1. [console.anthropic.com](https://console.anthropic.com) にアクセス
2. アカウント作成・ログイン
3. 「API Keys」メニューから「Create Key」
4. 表示された `sk-ant-...` から始まる文字列をコピーしておく

⚠️ **このAPIキーは絶対に他人に見せてはいけない**。パスワードと同じ扱いをする。GitHubなどにアップロードするコードの中に書いてはいけない。

**APIキーの設定：**

ターミナルで以下を実行する（`your-key-here` の部分を自分のキーに置き換える）：

```
export ANTHROPIC_API_KEY=your-key-here
```

毎回ターミナルを開くたびに設定するのが面倒な場合は、`.zshrc`（Macの設定ファイル）に追記すると永続化できる。このあたりはClaude Codeに「APIキーを永続化したい」と聞けばやり方を教えてくれる。

**料金について：**  
Claude Codeは使った分だけ課金される従量制。月$5〜20くらいが一般的な使用量の目安。最初は$5分のクレジットから始めてみるのがおすすめ。

## うまくいかないときのチェックリスト

**「command not found: claude」と出る**  
→ Node.jsのインストールが完了していないか、ターミナルを再起動していない。ターミナルを閉じて開き直してから再試行。

**「Invalid API key」と出る**  
→ APIキーのコピーミスが多い。スペースや改行が混入していないか確認。

**何も反応しない**  
→ Enterキーを押したか確認。「Ctrl+C」で一度キャンセルしてから再試行。

## まとめ

セットアップの流れはこれだけ：

1. Node.jsを入れる
2. `npm install -g @anthropic-ai/claude-code`
3. APIキーを取得して設定
4. `claude` で起動
5. 日本語で話しかける

「思ったより簡単だった」と感じてもらえたなら、それが正しい感想だ。
