【オリエンから企画書の自動生成をできる限りやってみた】

claude codeでの企画書自動作成についてまとめてみました。オリエンを入れるとそのまま分析・課題設定・コアアイデア・戦略・戦術・スケジュール・シミュレーション・予算まで、アウトプットしてくれるskillを配布しています。

現時点でのクオリティは50-60点程度になりますが、ここから手を入れていけば十二分のクオリティを目指せるかなと。

あと１ヶ月くらいかけて精度を少しずつ高めていきますが、自分で使いやすいようにアレンジするのが良いですね。

※画像は架空の企画書です。
