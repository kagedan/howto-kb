## 1. Claude を作った Anthropic 社とは

**Anthropic** は 2021 年に米サンフランシスコで設立された AI 企業です。創業者の **Dario Amodei** と **Daniela Amodei** 兄妹は、いずれも OpenAI の元研究者で、「AI を**安全に**(safely)発展させる」を会社のミッションに掲げています。

主要投資家には Google、Amazon、Salesforce などが名を連ねており、2026 年時点で評価額は **$ 数百億ドル規模**に達しています。

---

## 2. Claude のラインナップ — Haiku / Sonnet / Opus

Claude には用途別に 3 つのモデルがあります。俳句 → ソネット → オーパス と、文学形式の長さに合わせて命名されています。

| モデル | リリース | 入力 / 出力 (1M tokens) | SWE-bench | 主な用途 |
| --- | --- | --- | --- | --- |
| **Claude Haiku 4.5** | 2025-10 | $1 / $5 | — | チャットボット、要約、簡単な分類 |
| **Claude Sonnet 4.6** | 2025-12 | $3 / $15 | 79.6% | コーディング、分析、本番運用の主力 |
| **Claude Opus 4.7**(最新) | **2026-04-16** | $5 / $25 | 87.6% | 複雑推論、研究、最高品質を要する場面 |

実務での選び方:

* **Haiku** — 大量バッチ処理、レスポンス速度が重要な対話型 UI
* **Sonnet**(推奨) — 8 割の用途はこれで十分。コーディング能力が特に強い
* **Opus** — Sonnet で品質が足りない場面のみ(全体の 5% 程度)

## 3. ChatGPT / Gemini との違い

2026 年時点で、世界の 3 大 AI モデルは以下の状況です:

| 軸 | Claude | ChatGPT | Gemini |
| --- | --- | --- | --- |
| 日本でのユーザー数 | 拡大中 | 最多 | 多 |
| コーディング能力 | 業界トップクラス | 高い | 改善中 |
| 長文処理 | 200K (Opus) / 1M (Sonnet) | 256K | 1M |
| 日本語の自然さ | 敬語が秀逸 | 高い | 高い |
| 安全性・倫理 | Constitutional AI で業界トップ | 標準 | 標準 |
| マルチモーダル | 画像入力対応 | 画像/音声/動画 | 画像/音声/動画 |
| API 価格 | やや高め | 中位 | 安価 |

### 開発者視点で Claude を選ぶ理由

1. **コーディング能力が圧倒的** — SWE-bench(ソフトウェアエンジニアリング能力評価)で 2025 年以降、Claude Sonnet が首位を独走しています
2. **長い指示を忠実に守る** — システムプロンプト数千文字のエージェント設計で破綻が少なく、本番運用で安定
3. **「無理なことは無理」と素直に言う** — 嘘をつかない、過剰に丁寧、安全性重視
4. **Claude Code / Cursor との相性が抜群** — 主要開発ツールが Claude を第一候補に推奨

ChatGPT が「大衆向け」、Gemini が「Google 連携・コスト重視」、\*\*Claude が「開発者・企業向け」\*\*というのが 2026 年時点の業界構図です。

## 4. Claude が特に得意なこと

### 圧倒的に得意

* **コードの理解・生成・リファクタリング**(SWE-bench Verified 1 位)
* **長文ドキュメントの分析・要約**(200K〜1M トークン)
* **エージェント設計**(複数ツール協調の指示遵守率が高い)
* **日本語の敬語・ビジネス文体**

### 標準的(他社と同等)

* 画像入力の理解
* 数式・科学計算
* クリエイティブライティング

### 苦手なこと

* 画像生成(用途は OpenAI / Midjourney 等)
* 音声合成 / 認識(用途は ElevenLabs / Whisper 等)
* 動画生成
* リアルタイム情報(Web 検索ツールで補完)

## 6. 日本でよく使われる開発ツール

### 1. Claude Code(Anthropic 公式)

ターミナルベースの AI コーディングアシスタント。**ファイル編集・テスト実行・コミットメッセージ生成**まで自動化できます。

```
npm install -g @anthropic-ai/claude-code
claude
```

### 2. Cursor IDE

VS Code のフォーク版エディタで、AI 機能が深く統合。**Claude Sonnet 4.6 を Cursor 上で使うのが 2026 年の主流**です。月額 $20(Pro)で利用枠付き。

### 3. Cline(VS Code 拡張)

VS Code 拡張機能。Cursor と違い**無料**で、API キーを自分で持ち込み(BYOK)。Claude を直接コントロールしたい開発者向け。

### 4. Continue.dev(VS Code / JetBrains)

オープンソースの AI コーディング拡張。複数モデル(Claude / GPT / Gemini / オープンソース)を切り替えて使えます。

## 9. コスト面で困ったら

Claude Code を本番運用していると、月額 ¥10 万 〜 ¥30 万に達することは珍しくありません。Cursor の月 $20 枠もすぐに使い切ってしまいます。

そんなときの選択肢:

1. **モデルを Sonnet → Haiku に切り替える** — 8 割の用途は Haiku で十分。価格は 1/3 以下
2. **Cursor の Pro+ にアップグレード** — 月 $40 になるが、利用枠が大幅増
3. **chotto.ai 経由に切り替える** — base\_url を `https://api.chotto.ai/v1` に変更するだけで、**最大 ~82%(一般)/ ~95%(RAG)のコスト削減**

新規登録時 **¥500 分の無料クレジット**(Claude Sonnet 4 直接利用換算で約 ¥2,500 相当)が即時付与されるので、クレジットカード登録なしでテストできます。

## 10. よくある質問

### Q: Claude と ChatGPT、どちらから始めるべきですか?

A: 既に ChatGPT を使っているなら、Claude は\*\*「コードと長文分析」を強化する補完\*\*として試すのがおすすめです。両方使い分けるのが現代の開発者のスタンダードです。

### Q: 日本語の品質は ChatGPT より上ですか?

A: **敬語・ビジネス文体**の自然さでは Claude が一歩リードしているという声が多いです。両者ともプロダクション品質に達しているので、好みの問題に近いです。

### Q: Claude を企業で導入する場合、データは安全ですか?

A: Anthropic のエンタープライズプランでは、**学習に使われない**契約条項があり、HIPAA 等のコンプライアンスにも対応しています。

### Q: chotto.ai と claude.ai は同じものですか?

A: いいえ、まったく別物です。**claude.ai** は Anthropic 公式の Web チャット。**chotto.ai** は株式会社ＴＩＥが運営する Claude API ゲートウェイで、Anthropic SDK 互換のまま日本市場向けに最適化されています。

### Q: Claude Code は無料ですか?

A: ツール自体は npm で無料インストールできますが、**API 利用料金**が別途発生します(従量課金)。
