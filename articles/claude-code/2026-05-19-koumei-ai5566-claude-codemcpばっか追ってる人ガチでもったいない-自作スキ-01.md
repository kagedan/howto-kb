Claude Code、MCPばっか追ってる人ガチでもったいない。

自作スキル50超を運用しながら検証した、
繋いでよかった個人的TOP5を5位から発表！

1位は多くの人が「えっ、そこ？」ってなる。

土台無いと、何作っても流れて消えるんよね😎
👇

5位 x-search-mcp

Xのリアルタイム投稿検索。
裏ではGrok 4.1 Fastが動いてる。

検索演算子（min_faves:50）でバズ投稿だけ拾えるから、ノイズ排除して発信ネタが一気に集まる。

「Xで何が伸びてるか」を毎朝3分で把握できる

4位 Exa MCP

Claudeに繋ぐWeb検索で個人的に1番ハマってる。
普通のWebSearchより文脈理解が深い気がする。

競合分析・ファクトチェック・市場リサーチ、
全部これに集約してる。

「ググる」より精度が高い場面が多いんよね

3位 Googleカレンダー連携

秘書業務には欠かせない。

予定の追加・変更・確認、全部Claudeに話しかけるだけ。
スケジュール調整もすぐ終わる。

「カレンダー開いて入力」のひと手間が消える

2位 Firecrawl

Webスクレイピングを丸投げできる。
LPの構造解析、競合記事の中身取得、
画像・テキスト全部いっぺんに取れる。

「このサイトをClaudeに読ませたい」が一瞬で終わる。
リサーチ時間が半分以下になった

1位 Obsidian

これだけ言うと「は？」って言われそうだけど、
Obsidianこそ全ての土台になる。

僕のVaultにはこれが全部入ってる：

・自作スキル50超
・ナレッジ（personal/content/product）
・日次リサーチ結果
・過去の発信・記事ドラフト全部
・プロジェクト別フォルダ

加えてWeb Clipperがエグい。
気になった記事をブラウザから1クリックでVaultに保存。
Claudeにそのまま読ませて要約・分析できる。

MCPは「外と繋ぐ」ツール。
Obsidianは「中に貯める」基盤＋「外から取り込む」入口。

土台がないと、何作っても流れて消えちゃうんよね

TOP5まとめ
5位 x-search-mcp
4位 Exa MCP
3位 Googleカレンダー連携
2位 Firecrawl
1位 Obsidian

連携ツール語る人は多いけど、
「貯める仕組み」を語る人は少ない。

ここに気づくと運用が変わる
