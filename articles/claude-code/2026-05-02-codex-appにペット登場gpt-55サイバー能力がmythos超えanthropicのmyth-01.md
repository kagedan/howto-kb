## 🐾 Codex Appにペットが飼える！？画面上でちょこちょこ動く相棒が登場

OpenAIのCodexデスクトップアプリに、なんとペット機能が追加されました！かわいいキャラクターが画面上に常駐して、タスクの進捗を通知してくれたり、アニメーションでリアクションしてくれたりする機能です。

設定方法はシンプルで、Codexアプリの「外観」設定の一番下にある「ペット」から起動するだけ。デフォルトのキャラクターを選ぶか、自分でカスタムペットを作ることもできるんです！ Xユーザーのぬこのこさんが実際に作り方をツリーで詳しく解説してくれているので、挑戦してみたい方はぜひ参考にしてみてください。ずっと画面にいるので「ちょっと邪魔かも…」と感じる人もいるかもしれないですけど、作業のモチベーションアップには良さそうですよね！

## 🛠️ Codex Appにサイドチャット機能が登場！メイン会話を汚さず質問できる

Codexデスクトップアプリに「サイドチャット」機能が追加されました！使い方はシンプルで、チャット画面の3点リーダーから「サイドチャットを開く」を押すだけ。するとメインチャットのコンテキストを引き継いだ別画面が開きます。

これが地味に便利なんです！実装作業を進めながら「このコードってどういう意味？」とか「この設計でいいっけ？」みたいなちょっとした質問を、メインの会話に混ぜずに確認できる。論文やドキュメントを読みながら、サクッと質問するのにも使えますよ。ただし、サイドチャット側からコードの書き換えや実行はできないので、そういった作業はメインチャットでやる必要があります。使い分けのコツを掴めば、作業効率がぐっと上がりそうですね！
