先に、誤解されたくないので最初に言っておきます。

この記事には「ボタン1つで月100万」みたいな話は1行も出てきません。

書いてあるのは、Claude Skills（再利用可能な指示書）と MCP（外部接続の標準規格）を組み合わせて、高単価アフィリエイトの作業を90%以上消したうえで、月100万に必要な数字を「逆算、仕組み化、自動運転」する手順だけです。コピペできるプロンプトを約40個、すべて初心者前提で並べました。Claudeに触ったことがない人でも、最初から最後まで順番に進めれば、ひと通りの仕組みが立ち上がります。

なぜここまで具体的に書くのかというと理由は単純で、「AI副業」「自動化」と言いながら、中身が抽象論で止まっている発信が多すぎるからです。「Claudeがすごい」「AIで自動化できる」と言うだけで、何を、どの順番で、どのプロンプトで動かすのか、誰も書いていません。だから始められない。始めても続かない。

正直に言いますね。 高単価アフィリで月100万は、誰でも数ヶ月で行ける魔法ではありません。ただし、構造はめちゃくちゃシンプルです。

「単価2万から10万円の案件 × 月10から50件の成約 × その仕組みを自動運転」、これだけです。

逆に言えば、この3つを愚直に組み立てれば、再現できる人は再現できます。Claudeはその3つの組み立てを、ほぼ全部肩代わりしてくれる相棒です。

この記事で扱うのは、案件選定、ペルソナ設計、記事量産、SNS導線、セールス文AB、プロンプトの中身、MCPで手足を増やす、Claude Code × cronで自動運転、落とし穴回避、古い習慣を捨てる、の10章。約40個のプロンプトと運用ノウハウを全部置いていきます。

最後まで読み終えたら、あなたの手元には「月100万を狙うための作業の地図」と「コピペで動く実装手順」が両方そろっている状態になります。コピペして、自分の案件名・ジャンル名に書き換えて、その場で投げる。これで動きます。

それではいきましょう。

**■第1章：案件選定 ─ 月100万の8割はここで決まる**

最初の章は「何を売るか」です。

正直、ここで間違えると、後ろの章をどれだけ頑張っても月100万には届きません。1件1,000円の物販案件をClaudeで自動化しても、月100万には1,000件必要です。1件3万円の高単価案件なら、月33件で同じ売上に届きます。やる作業はほぼ同じ。違うのは「最初に何を選んだか」だけです。
