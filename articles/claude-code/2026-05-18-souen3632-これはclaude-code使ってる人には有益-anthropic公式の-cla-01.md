"claude-code-setup" というプラグインが出ています。

簡単に言うと、
Claude Codeをもっと使いやすくするための「設定アドバイザー」です。

プロジェクトを見て、

・MCPサーバー
・Skills
・Hooks
・Subagents
・自動化できる作業

などを提案してくれます。

ただし、入れたら全部自動で完成する魔法ツールではありません。
あくまで「こう設定するとよさそう」と教えてくれる支援プラグインです。

Claude Codeを素のまま使っている人ほど、試す価値あり。

"/plugin install claude-code-setup@claude-plugins-official"
