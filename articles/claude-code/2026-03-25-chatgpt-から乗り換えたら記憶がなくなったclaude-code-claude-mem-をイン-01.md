# ⬜️ はじめに

ChatGPT から Claude に乗り換えて、こんな不満を感じたことはありませんか？

> 「ChatGPT は前回の続きから話せたのに、Claude は毎回ゼロからのスタートになる…」
> 「先週アップロードした PDF、また最初から読み込ませないといけないの？」

Claude は返答の自然さや文章の質が高く、ChatGPT から乗り換えたユーザーも多いと思います。しかし、**会話をまたいだ記憶がない**という点に不便を感じている方も多いのではないでしょうか。

この記事では、Claude にセッションをまたいだ記憶を持たせる方法として **Claude Code ＋ claude-mem** の導入手順を解説します。あわせて、用途によっては **Claude.ai のプロジェクト機能** の方が適している場合もあるため、最初に両者を比較します。コマンドプロンプトに触ったことがない初心者の方でも、手順通りに進めれば設定できます。
