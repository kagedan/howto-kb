## 🤖 Googleから拡散モデル「DiffusionGemma」登場！最大4倍の速度が話題

Googleが新しいオープンソースモデル「DiffusionGemma」をリリースしたんです！

通常のAIモデルって、文章を上から順番にズラーっと出力していくイメージですよね。でもDiffusionGemmaは全然違って、まずバラバラに全体像が浮かび上がり、最終的に整ったアウトプットが完成する「拡散モデル」方式を採用。この仕組みのおかげで、標準的なアクセラレーターで**最大4倍の速度**を実現しているんです！

パラメーター数は26B（A4Bアーキテクチャ）なので、メモリが20GB前後あればローカル環境でも動かせる可能性があるとか。すでにHugging Faceに公開されているので、環境がある方はぜひチェックを。LM StudioやOllamaでの対応はまだのようですが、今後の追加に期待大ですね。オープンソースでこのパフォーマンスが手に入るのは、なかなかインパクトある話かも！

## ⚡ Cursorのコードレビューエージェントが大幅パワーアップ！

AIコーディングツールのCursorが、コードレビューエージェントを強化しました。その数値がなかなか印象的で、**処理速度が3倍以上に向上**、**コストは22%ダウン**、そして**バグの発見率が10%アップ**と、三拍子揃ったアップデートです。

/reviewや/agent reviewコマンドでコードの品質チェックができる機能で、コンポーザー（Composer）でのコスト消費が少ない分、レートリミットをあまり使わずにサクッと走らせられるのが便利なんですよね。開発フェーズを区切って「フェーズ1が完了したらレビュー実行」みたいな使い方をしている方も多いとか。コストが下がってバグ発見率が上がるなら、ルーティン的に組み込みたい機能ですね！
