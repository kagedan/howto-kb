## STEP1：AIの種類と「できることの違い」を知る

![](https://pbs.twimg.com/media/HLq3Sl0aYAAhtUn.jpg)

最初にやるのは、ツールを買うことじゃありません。 **AIにはざっくり“種類”があって、それぞれ役割が違う**——これを知るだけです。

大きく２つに分けると、こんな感じです。

- **会話型AI**
- **コーディングAIエージェント**
**会話型AIは、**ChatGPT /Claude.ai/ Gemini など、文章・アイデア・相談ごとを回答してくれるツールです。

誰でもチャットで使えるのが特徴で、

相談事まではできるけど実際にパソコンの内部のファイルを触ったり実際の作業を最初から最後までやってもらうなどはできません。

一緒に頭脳としてプロジェクトを考えてくれるまでをやってくれるイメージです。

**コーディングAIエージェントとは、**Claude Code / Codex など、黒い画面（ターミナル）または専用アプリで、AIがファイル作成やプログラム実行まで自律でやってくれるAIです。

**コーディングAIエージェントはもともとエンジニア向けですが、Xの記事投稿作成から投稿までや毎日勝手にやってくれたり、**

**AI内で会議をさせてタスクをあぶり出して勝手にプロジェクトを進めてくれたりとあなたの業務の実働を肩代わりしてくれるイメージです。**
