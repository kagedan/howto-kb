# 「LP1本を、コーヒーが冷める前に。」

月20ドルで"AIに仕事を任せる側"になる、たった1つの方法。

ChatGPTに課金してるのに、作業はあいかわらず手作業のまま。 そんなあなたのまま、半年後どうなってると思います？

## 「もう、自分で作る時代じゃない」と気づいた瞬間に、人生が変わった話

正直に言わせてください。

このX記事を書いてるいま、僕の手元にはたった30分前に作られたばかりのLPがあります。デザインも、画像も、HTMLも、デプロイまで全部終わってる。手は1ミリも動かしてない。指示を1行入れて、コーヒーを淹れに行って、戻ってきたら完成してたんです。

「なに言ってんの？」って思いますよね。半年前の僕も思ってました。LP1本作るのに丸1日。慣れてない頃は3日かかってた。Midjourneyで画像作って、Photoshopで切り抜いて、Figmaで組んで、Cursorで書いて、ブラウザで確認して、また直して。気力が削られて、納品の頃にはクライアントへの愛着もなくなってる。

それが、いまは30分。何が変わったかというと、OpenAIが2026年に出した「Codex」というツールを、ちゃんと使い倒し始めただけ。これだけで、僕の働き方が完全にひっくり返りました。

このX記事は、その入口です。続きは本編X記事にあって、僕がこの半年で発見した「非エンジニアがCodexを武器にして、AIに仕事を任せる側に回るための全手順」を、20,000文字越えで全部公開してます。

## 「まだ手で作ってるんですか？」と聞かれる側に、半年で回った人たちがいる

先に、ちょっと数字の話をさせてください。

2026年4月時点の公式ベンチマーク「Terminal-Bench 2.0」で、Codexは77.3%。Claude Codeの65.4%を11.9ポイント上回ってます。これがどういう意味かというと、「同じ作業を任せたとき、Codexのほうが2割近く高い確率で一発で正解する」ってこと。しかも、3〜4倍トークン効率がいいので、同じ月20ドルで何倍も多くの仕事を回せる。

これだけでも十分エグいんですけど、本当にやばいのはここから。

X上で報告されてる事例を、いくつか拾います。

- iOSヘルスケアアプリを5分で生成
- 業務効率化Webアプリを30分で生成
- フィンテックMVPを通常2ヶ月→3週間に短縮
- 大規模APIマイグレーションを3日→1日で完了
- エンジニアが2日間で80個近くのプルリクエストを作成
「いや、これはエンジニアの話でしょ？」って思いました？それが違うんですよ。むしろ恩恵を一番受けてるのは「コードが書けないけど、自分の仕事を効率化したい人」のほう。理由は、本編の中で全部お伝えします。

## 「丸1日かかるLP」を毎週納品してる、あなたへ

ちょっと胸に手を当てて、思い出してみてください。

最近、こんなふうに思ったことありませんか？

> 「ChatGPT Plus、Claude Pro、Cursor、Midjourney、Figma…月の課金が1万円超えてるのに、実際の作業時間は減ってない」「AIツールのアップデートが速すぎて、何から触ればいいかわからない」「Codexってコーダー向けでしょ？自分には関係ないし」「LP1本に1日かかるから、月に5本受けるのが限界」

これ、半年前の僕の口癖だったんです。たぶんあなたも、似たような感じで止まってる。違いますか？

ここで知ってほしいのが、この"止まってる感"の正体です。

正体は、AIツールの「個別最適」で止まってること。ChatGPTで文章書いて、Midjourneyで画像作って、Figmaでデザインして、Cursorでコード書く。ツール単位で見ると優秀なんだけど、ツール間の橋渡しは全部あなたの手作業だった。だから時間がかかる。

そして、Codexがやったのは、この「橋渡しを全部AIに任せる」というシンプルな思想転換。あなたが「LPを作って」と言うだけで、Codexが内部でMidjourney的な画像を生成して、Figma的なデザインを組んで、Cursor的にコードを書いて、Vercelにデプロイする。5ツール往復が、1コマンドに圧縮されるわけです。

これが理解できると、あなたの月1万円の課金額が、ようやく価値に変わり始めます。
