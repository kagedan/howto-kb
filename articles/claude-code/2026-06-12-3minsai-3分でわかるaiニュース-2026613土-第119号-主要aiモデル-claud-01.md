━━ 主要AIモデル ━━

■ OpenAI
Codexのレートリミットのリセット権を保存して好きなタイミングで使用できる新機能が、有料プラン向けに提供開始されました。

━━ 開発者向け ━━
■ OpenAI Developers
・OpenAI製品の疑問に答え、開発ガイドやプロンプトを自動生成するドキュメント用エージェントが公開されました。

━━ ツール ━━
■ Claude Code
・バージョン2.1.176が公開され、検索機能の改善や利用可能なモデルの制限を強制するセキュリティ設定が追加されました。

AIエージェントが業務アプリの構築からデプロイまでを完結させ、少人数での大規模な組織運営を可能にする事例が公開されました。

・MCPにおいてHTTPループバック転送や認証機能が改善され、外部クライアントとの互換性が高まりました。
