中国AIの無料公開は、善意ではない。市場を取るための価格兵器である。誰でも使えるオープンウェイトは競争を民主化するように見えるが、実際には性能、資本、計算基盤を持つ上位企業へ利用者を集中させる。JPモルガンが示した「勝者総取り」より不気味なのは、無料で配った企業ほど最後に強く課金できるという逆説だ。

# 無料公開が強者をさらに強くする構造

中国の大規模言語モデル市場では、アリババ、DeepSeek、Zhipu AI、MiniMax、テンセントなどが、モデルの性能だけでなく、公開方法、API価格、クラウド連携、開発者基盤を競っている。

米国の主要AI企業が最先端モデルを閉じたサービスとして提供してきたのに対し、中国企業はモデルの重みを公開し、企業や開発者が自分の環境で動かせる選択肢を広げてきた。低価格で利用でき、改変もしやすく、特定企業のAPIへ完全には依存しなくて済む。この開放性が世界中の開発者を引き寄せた。

一見すると、モデルを公開した企業は不利になる。競合が技術を参考にでき、クラウド企業が同じモデルを再販売でき、利用者は公式APIへ料金を払わず自前で動かせるからだ。「無料で配った後、どこで稼ぐのか」という疑問が生まれるのは当然である。

だがJPモルガンの見方は逆だ。オープンウェイトは上位企業の収益機会を奪うのではなく、性能差を世界へ拡散する。優れたモデルが広く使われれば、対応ツール、学習資料、追加開発、企業導入が増える。その結果、公式API、クラウド、運用支援、業務プラットフォームへ需要が戻ってくる。

無料公開で勝てるのは、公開後も改善を続けられる企業だ。モデルを配った瞬間が完成ではない。推論速度を上げ、障害を防ぎ、キャッシュを効かせ、利用量へ耐え、次のモデルを出す必要がある。資金と技術が足りない企業には、その継続戦ができない。

オープン市場は弱者へ機会を与える。しかし同時に、性能の低いモデルを世界規模で比較される場所でもある。公開しただけで利用者が増える時代は終わり、公開した結果、弱さが鮮明になる時代へ入った。

# オープン配布は無料商売では終わらない

オープンウェイトと無料サービスは同じではない。モデルの重みを利用可能にすることと、推論に必要な計算資源を無料で提供することは別の話だ。

大規模モデルを自社環境で動かすには、GPU、電力、保守、人材が要る。モデルをダウンロードできても、それを高速かつ安定して運用する費用は消えない。企業利用では、アクセス管理、監査、障害対応、データ保護、利用量の監視も必要になる。

そこで多くの利用者は、結局モデル提供企業やクラウド企業のAPIを使う。自前運用の自由を持ちながら、通常時は便利な公式サービスへ料金を払う。オープンであることが安心材料となり、導入を促し、有料利用へつながる。

この構造は、無料の基本ソフトと有料のクラウド運用を組み合わせて成長したソフトウェア企業に近い。コードやモデルを囲い込むのではなく、使いやすい運用環境、速度、信頼性、支援体制で稼ぐ。

アリババならQwenを公開しながら、Alibaba Cloudの利用へ接続できる。Zhipu AIはGLMを広めながら、API、企業向け導入、クラウド経由の販売を伸ばせる。モデル単体を売るのではなく、モデルを中心とする経済圏から収益を取る。

ただし、この商売は誰にでも成立するわけではない。性能が平凡なら、利用者は別の公開モデルへ移る。公式APIが高ければ、外部の推論事業者へ逃げる。企業向け機能が弱ければ、試用されても本番環境には残れない。

無料公開は顧客獲得の入口にすぎない。入口を広げた後、有料サービスへ利用者を運ぶ道を持つ企業だけが勝つ。

# 公式APIの安さが収益を生む逆説とは

JPモルガンの分析で興味深いのは、DeepSeek V4 Proの公式APIと、外部事業者による提供コストの差だ。レポートの試算では、キャッシュヒット率93.5％を前提にすると、公式APIの月間ワークロード費用は約24〜41ドルに収まる一方、一部の外部提供では85〜196ドルへ膨らむとされる。

この数字は、すべての利用者にそのまま当てはまる料金表ではない。入力と出力の量、会話履歴の長さ、キャッシュの使い方、処理速度、外部事業者の構成によって費用は変わる。それでも、公式提供者が持つ構造的な強みは見える。

モデル開発企業は、モデルの内部構造、推論処理、キャッシュの仕組みを最も深く理解している。自社設備や長期契約した計算資源を使い、モデルに合わせて運用を最適化できる。外部事業者は公開されたモデルを動かせても、同じ効率を再現できるとは限らない
