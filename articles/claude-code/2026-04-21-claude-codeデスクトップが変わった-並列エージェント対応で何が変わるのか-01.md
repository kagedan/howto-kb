## 背景

AnthropicがClaude Codeのデスクトップアプリを2026年4月14日に大幅刷新した。

これまでのClaude Codeは、1セッション＝1タスクが基本だった。
複数の作業を並行したい場合、ターミナルを複数開いてセッションを立ち上げ直すしかなく、状態を跨いだ連携は難しかった。

今回の更新で、デスクトップ上で複数セッションを同時進行させ、並列エージェントとして協調動作させる仕組みが加わった。
同時期に発表された「Routines」（ワークフローの自動実行機能）と合わせて、Claude Codeは「手元で動かすツール」から「エージェント型ワークフロー基盤」へと性格を変えつつある。
