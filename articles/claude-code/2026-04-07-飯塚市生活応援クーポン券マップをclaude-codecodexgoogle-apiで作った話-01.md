### PDFの一覧表、本当に使いにくい

飯塚市では2026年度、「生活応援クーポン券」が配布されました。でも配布されるのはPDFの一覧表。店舗名がびっしり並んでいるだけで、「この近くに使えるお店あるかな？」と思っても全然わかりません。
