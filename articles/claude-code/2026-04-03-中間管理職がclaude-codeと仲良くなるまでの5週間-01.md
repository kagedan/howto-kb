## 自己紹介

40代。エンジニアではない。コードはしばらく書いていない。

そんな中間管理職が、Claude Codeを使って80要件ある案件管理ダッシュボードを作ろうとした。

ただし「作った」ではなく、**毎回スクラッチから同じ80要件を渡して、最もよく動くClaudeの構成を検証し続けている**、という記録。
