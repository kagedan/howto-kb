はじめに
Claude Codeは単なるチャットAIではありません。ターミナルに常駐し、自律的にコードを書き換える「エージェント」です。その中でも、特定のプロジェクトや独自のルールをAIに覚え込ませる機能が**「Skills」**です。


Claude Codeの「Skills」とは？
一言でいえば、**「特定のタスク専用の実行マニュアル」**をAIに持たせる機能です。


通常、AIにコーディング規約やデプロイ手順を守らせるには、毎回長いプロンプトを打つ必要があります。しかし、Skillsとして登録しておけば、AIが必要な時だけそのマニュアルを自動で読み込み、実行してくれます。
