## そもそもマネーフォワードって何？

マネーフォワード ME は、**銀行口座やクレジットカードなどを連携すると、お金の出入りを自動で記録してくれる家計簿アプリ**です。

家計簿が続かない最大の理由は「手入力が面倒」なことだと思います。  
マネーフォワードは口座やカードを一度連携しておけば、使ったお金が自動で取り込まれてカテゴリ分けまでしてくれるので、入力の手間がほぼゼロになります。

そして今回の主役であるClaude Codeと組み合わせると、\*\*「貯まったデータを、自分好みの形で自動集計する」\*\*ところまでできるようになります。マネーフォワードが「データを集める係」、Claude Codeが「データを整理して集計する係」というイメージです。

## マネーフォワードに登録する

まずは無料のアカウントを作ります。公式サイト、またはApp Store / Google Playからアプリをインストールして、メールアドレスやGoogleアカウントで登録しましょう。数分で完了します。

スマホアプリ（iPhone / Android）やパソコンのブラウザどちらでも使えますが、最初はアプリのほうが手軽です。

## 基本的な使い方：口座やカードを連携する

登録できたら、普段使っている口座やカードを連携していきます。

1. アプリの「口座」メニューを開く。
2. 連携したい銀行・クレジットカードなどを選ぶ。
3. それぞれのログイン情報を入力して連携する。

連携が終わると、過去の利用履歴が自動で取り込まれ、「食費」「日用品」「交通費」などのカテゴリに自動で振り分けられます。カテゴリが間違っているときは手動で直せる仕様で、数件直しておくと、次回以降の自動振り分け精度が上がっていきます。

これだけで、**何も入力しなくても家計簿が出来上がっていく**状態になります。

実際に私も１か月やってみたところ、最初の２週間くらいはいくつか手直しをする項目がありましたが、今はほとんど理想通りに自動振り分けをしてくれています。  
支出を管理するを目的としては、以下のやり方が一番しっくりきていて、  
家計簿の自動化（頑張らない家計簿作り）も完了しています。  
・クレカ１枚（楽天カード）を連携し、基本そのカードですべて決済する。  
・現金を使った時だけは手入力またはレシートスキャンで付ける。

## 無料版でどこまでできる？

マネーフォワードには無料版と有料版（プレミアム）がありますが、  
**今回のやり方は無料版のままでOK**です。

無料版にはいくつか制限があります。代表的なのは、**連携できる口座・カードが4件まで**という点と、**過去データの閲覧が直近1年まで**という点です。よく使う口座やカードが4件以内なら、無料版でまったく問題ありません。

> プレミアム版にすると連携件数の制限がなくなり、CSVでのデータ書き出しなどもできるようになります。ただ、今回紹介する方法はCSV書き出しを使わないので、まずは無料版で十分です。  
>   
> また、無料版だとマネーフォワード上では過去データを直近1年分しか閲覧できませんが、Claude Codeでスプレッドシートに転記する手法を活用するため、**過去すべてのデータを確認することが可能**になります。  
> ※クレカ明細の連携ができれば、マネーフォワードではなく他の家計簿でも  
> 　対応可能です。

## ここからが本題：Claude Codeでどう集計するのか（全体像）

ではいよいよ、Claude Codeでの自動集計です。今回は「どんな流れで動くのか」という全体像を押さえてください。具体的な作り方は次回お見せします。

私のやり方は、CSVの書き出し（有料機能）を使わず、**マネーフォワードに表示されている家計簿データをそのままClaude Codeに取得させ、所定のスプレッドシートにまとめてもらう**という方法です。大きな流れはこの3ステップです。一度覚えさせれば、**毎日自動で以下のステップ1～3をClaude Codeが実行**してくれます。

**ステップ1：マネーフォワードでデータを最新にする** Web版でクレカ明細の連携を「更新」して、カードの明細を取り込みます。  
→この手順は割愛しても問題ありませんが、自動更新だと数日間更新されな  
　い日もあるため、Claude Codeに更新してもらうのが確実です。

**ステップ2：データを取得する**  
マネーフォワードに表示された家計簿の明細データを、Claude Codeに読み取ってもらいます。

**ステップ3：Claude Codeがスプレッドシートに整理して貼り付ける**  
取得したデータを、Claude Codeが**あらかじめ用意しておいた集計用スプレッドシートに貼り付け**ます。

ここがこの仕組みの一番の工夫ポイントになりますが、**毎回データを取得した際、二重に記録されない**ようにしています。具体的には、Claude Codeが\*\*「すでにスプレッドシートに貼ってある分」と「今回取得した分」を照合して、まだ記録されていない新しいデータだけを追加する\*\*仕様にしました。

これがあるおかげで、「これ重複して金額おかしくなってない・・？」など自分で管理する必要がなく、ダブりなくきれいにデータが積み上がっていきます。

データがスプレッドシートにそろえば、あとは月ごと・カテゴリごとの集計はお手のもの。「今月の食費はいくら？」「先月と比べて増えた費目は？」といった集計も、Claude Codeにお願いするだけで出してくれます。  
→私は一度スプレッドシートに関数を入れて、毎月の食費や日用品費は  
　Claude Codeに聞かずに集計していますが、もちろん集計も頼めます。

## まとめ

今回やったのは、家計簿自動化の「土台づくり」でした。

1. マネーフォワード（無料版）に登録する
2. 口座・カードを連携して、データが自動で貯まる状態にする
3. マネーフォワードの明細データをClaude Codeに渡す
4. Claude Codeが、重複を避けながらスプレッドシートにデータを積み上げ、集計してくれる

有料プランに課金しなくても、無料版＋Claude Codeでここまでできます。
