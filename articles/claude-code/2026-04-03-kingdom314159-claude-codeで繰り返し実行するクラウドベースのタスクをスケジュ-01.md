Claude Codeで、繰り返し実行するクラウドベースのタスクをスケジュールできるようになりました！！

リポジトリ（1つでも複数でもOK）・スケジュール・プロンプトを設定するだけで、Claudeがその内容をクラウド上で自動実行してくれます。

つまり、ローカル環境でClaude Codeを常に起動しておく必要はありません。

私たちは、この機能がさまざまな用途で役立つと感じています。例えば：

•オープンなPRを整理・解消する
•承認済みのIssueをもとに機能を実装する
•夜間にCIの失敗を分析する
•新しくマージされたPRに合わせてドキュメントを更新する

といった場面で活用できます。
