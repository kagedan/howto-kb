移動中に「こんなアプリあったらいいな〜」「いつか開発したいな〜」と思って放置した経験ありませんか？

最近の私はclade codeで移動中にアプリ作りまくりなので、その方法を共有します！

## 必要なもの

* **Claude Pro 以上**（Freeは不可）
* GitHub アカウント

## やり方（3ステップ）

1. スマホで GitHub リポジトリを作る
2. Claude アプリで Claude Code → GitHub を連携
3. 「〇〇を作って」と投げる

あとは Claude がクラウド上でコードを書き、テストして、**PRまで作ってくれます**。  
帰宅後にPCでレビュー、マージ。以上！

## 作ったもの

* 🥟 **餃子の味記録アプリ** — 店・皮の厚さ・餡の好みをメモして、自分だけの餃子データベースに！
* 👕 **服管理アプリ** — 手持ちの服を登録して、「今日何着る」問題を減らすやつ
* 📔 **英語日記アプリ** — 書いた英文をその場で添削。続けられる仕組みが欲しかった

## いちばんの変化

ローカル環境の準備がゼロなので、**思いついた瞬間に着手できる**。  
「あとでやる」が「いま投げとく」になりました。3つできたのは、たぶんそれだけの理由です！

皆さんもぜひclaude codeでどんどんアプリ作りましょう！！
