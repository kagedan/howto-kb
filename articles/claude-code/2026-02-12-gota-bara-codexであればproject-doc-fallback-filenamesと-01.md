Codexであれば、project_doc_fallback_filenamesという設定がconfig. tomlにありまして下記記載しておけばAGENTS .mdがない時にCLAUDE .md読んでくれるようになります！

project_doc_fallback_filenames = ["CLAUDE .md"]
