2026年、寝てる間に実務を片付けてくれるGitHubリポジトリ10選：

2. Hermes Agent
使うほどあなたに合わせて賢くなるパーソナルAI。Nous Researchが公開、スター約19万。

3. CrewAI
複数のAIが役割分担して、調査から作成までを丸ごとこなすチーム型ツール。スター約5.3万。
https://t.co/AklNDTXjUf

4. Aider
ターミナルで動くAIのペアプログラマー。直した内容をそのまま履歴に残してくれる。スター約4.6万。
https://t.co/6oFgziLHR0

5. n8n
ノーコードで業務を自動化できるワークフローツール。400以上のサービスと連携でき、自分のサーバーで動かせばサブスク料金もかからない。
https://t.co/pvJGzLdiyo

6. LangGraph
本番のAIエージェントを動かす土台。複雑な処理の順番や分岐を設計できる。スター約3.5万。
https://t.co/eLWEIkFvQt

7. Cloudflare Agentic Inbox
受信メールを読んで返信の下書きまで作るAI内蔵のメールソフト。自分の環境で動かせる。
https://t.co/nd8rFqQGm9

8. Browser Use
AIがブラウザを操作して、フォーム入力・情報収集・予約までこなす。スター約9.9万。
https://t.co/bDao4MLSqr

9. awesome-mcp-servers
AIに繋げる外部ツールのカタログ集。GitHub・Slack・Notion・Stripeなど何でも接続できる。
https://t.co/1Yf8Kh9AC2

10. claude-task-master
Claude Code上で複数のAIに作業を割り振る司令塔。1つの指示からチームを動かして機能を仕上げる。
