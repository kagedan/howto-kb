## Cursor

## Claude Code

最近オープンソースで作ってみたものがあるのですが、これです。  
（また、そのうち別でnoteに書こうと思ってます。）

こっちの場合は、壮大なアイデアはあるけど具体は決まっていない。AIと壁打ちしながら具体に落とし込みたい。特にコードに興味があるわけではない。みたいな、とんでもなく抽象的な思いつきを形にしたいときはClaude Codeを使っています。

リモートコントロールをActiveにしておけば、iPhoneからも出来ますし、コードがみたいわけじゃない時に使う感じにしています。

Claude CodeはTerminalから対話するので、ちょっと玄人向けな気もします。（Claudeだけに…………….）  
って思っているんですけど、𝕏とかYoutubeを見ていると、非ITの方にとても人気ですよね。どうやってClaude Codeを扱っているんでしょう？CoWorkとかそっちを使っているのかな？（ぜひ、コメントなどで教えてください。）

あと、Claude Codeの時だけ、Amicalという音声入力を利用しています。理由は単純で、無料で使えるからです。

ぼやっとしたことを入力するには、キーボードだと直感的じゃ無いので、音声入力でダラダラ喋ってAIに引き出してもらうのは相性が良いなと思ってます。
