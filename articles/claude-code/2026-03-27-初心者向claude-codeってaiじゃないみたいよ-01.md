# Claude Code is 何？

Claude Codeは「LLM（脳）」ではなく、その脳を使って作業をする「エンジニアの体（ツール）」です。  
平たくいうと、自分の分身体（は言い過ぎですが）ある程度の命令を与えたらファイルの読み書き、コード生成（開発）、デプロイ（リリース）などを勝手にやってくれるみたいです。

それではLLMの部分はどこかというとClaude Codeの開発元の「Anthropic社」が提供している「Claude 3.7 Sonnet」などがLLMです。  
ちなみに、このLLMも頭がいいらしいと噂です。

# Claude Codeはお高いの？

Claude Codeを利用するには、大きく分けて\*\*「既存のチャットプラン（サブスクリプション）」で使う方法と、「API（従量課金）」\*\*で使う方法の2パターンがあります。  
※ 2026/03/28時点

1. サブスクリプションプラン（個人・チーム）

| プラン | 料金 | 主な特徴 |
| --- | --- | --- |
| **Pro** | $20/月（年払いの場合は $17/月） | 個人向け。Sonnet 4.6 および Opus 4.6 にアクセス可能。 |
| **Max 5x** | $100/月 | より大きなコードベースでの日常的な使用に適したプラン。 |
| **Max 20x** | $200/月 | パワーユーザー向け。最も多くのモデルアクセス権が含まれます。 |
| **Team** | $20/席/月（5〜150名のチーム） | セルフサービスでの席数管理が可能。標準APIレートでの追加利用にも対応。 |
| **Enterprise** | 要問い合わせ | 高度なセキュリティ・データ管理。Amazon BedrockやGoogle Cloud Vertex AIのインスタンスでも実行可能。 |
| ※ ちなみにFreeプランもありますが後述します |  |  |

2. Claude API（従量課金）  
   「使った分だけ支払いたい」場合や、開発者数に制限なく導入したい場合に適したプランです。  
   料金形態: 標準のClaude API価格に基づいた従量課金となります

* メリット: 月額固定費やユーザーごとの席数料金、プラットフォーム利用料はかかりません
* エンタープライズ連携: 企業ユーザーであれば、既存のAmazon BedrockやGoogle Cloud Vertex AIのインスタンス上で動作させることも可能です

💡 補足：高速モード（Fast mode）  
Opus 4.6モデルを2.5倍速で実行できる「高速モード」が用意されています

* 料金: 100万トークンあたり $30（入力）/ $150（出力）の設定です
* 対象: 従量課金プランのユーザー、またはサブスクリプションプランの「追加利用枠」として利用可能です

# 有料と無料で何が違うの？

上記に有料プランだけ記載しましたが、実はFreeプランもあります。  
ただ有料と無料ではできることが異なるので、こちらで記載します。  
で、ざっくり有料と無料の違いですが・・・

## 有料版

WindowsやらMacやらにClaude Code（CLI）をインストールして、PC内にあるファイルや開発ツールなどにアクセスさせたりできます。  
SNSでClaude Codeに課金してるー！って人は、これで作業の効率化をしています。

## 無料版

上記のようにPC内にあるファイルなどにはアクセスできません。  
ブラウザから対話する形になるので、使用感的にはGeminiやChatGPTに近いです。
