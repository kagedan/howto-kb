■ 有料コースに課金してた人、ちょっと待って

「Claudeをもっと使いこなしたくて、有料の学習コースを探してる」という声、Claude界隈でよく見かけます。でも実は、Anthropic自身が公式の学習コースを13本、完全無料で公開してたんです。クレジットカード不要、メールアドレスだけで全コースにアクセスできます。

Claude 101のような入門コースから、Claude Codeの実践ワークフロー、MCPを使った外部ツール連携まで体系的に揃っていて、しかも修了するとAnthropicの公式修了証が発行されます。LinkedInのプロフィールにも追加できるやつです。

他社の有料コースが数万円で販売しているレベルの内容が、全部ゼロ円。これを知らずに課金してた人がいたら、ちょっともったいないなと思って整理しました。

■ そもそも13コース、何が揃ってるの？

Anthropicが公開している学習プラットフォームは2つあります。**anthropic.com/learn**（公式ラーニングページ）と、**anthropic.skilljar.com**（Skilljar製のコース配信プラットフォーム）です。この2つを合わせて、現在13コースが公開されています。

大きく4つのトラックに分かれているので、順番に見ていきましょう。

**入門トラック（コード不要）**

・Claude 101 — Claudeの基本操作とプロンプトの書き方（所要時間：約1時間）

・AI Fluency — AIと「いつ・何を」任せるかの思考法（所要時間：約3時間）

・Introduction to Claude Cowork — ファイルやドキュメント業務への活用

**Claude Codeトラック**

・Claude Code 101 — インストールから実践ワークフローまでの基礎

・Claude Code in Action — サブエージェント・Hooks・カスタムコマンドの実践

**開発者トラック**

・Building with the Claude API — API設計からデプロイまで（8時間超）

・Introduction to MCP — 外部ツール連携の標準規格を学ぶ入門編

・MCP Advanced — MCPの発展的な実装技術

**マルチエージェント/サブエージェント関連**

・Introduction to Subagents — タスクを複数のエージェントに委任する設計

以上のように、超入門から本格的な開発まで、段階的に並んでいます。全部で13コース、どれも無料です。

![](https://pbs.twimg.com/media/HMSk-jCaIAAZ7EK.jpg)

■ 「自分には関係ない」と思ってる人へ

「Claude Codeって、エンジニアのためのツールでしょ？」という印象を持っている人が多いんですが、13コース中、プログラミング知識がまったくなくても受けられるコースが6つあります。

特に経営者・個人事業主・ビジネス職の方におすすめしたいルートがこれです。

① **Claude 101**（約1時間） — まず触ってみる。プロンプトの書き方・Claudeとのやりとりの基本を掴む

② **AI Fluency**（約3時間） — 「このタスクをAIに任せていいか？」の判断軸を作る。AIリテラシーの土台になるコース

③ **Introduction to Claude Cowork** — ファイル処理・ドキュメント作業など、日々の業務に直結する活用法

この3コースだけで、合計4〜5時間。Claude Proの月額課金も不要で、カード登録すら要りません。メールアドレスだけで今日から始められます。

僕が特に「これいいな」と思ったのはAI Fluencyです。ただの操作説明じゃなくて、「AIに何を・いつ・どこまで任せるか」という思考法を教えてくれるコースなので、エンジニアでなくても仕事の進め方が変わると思います。
