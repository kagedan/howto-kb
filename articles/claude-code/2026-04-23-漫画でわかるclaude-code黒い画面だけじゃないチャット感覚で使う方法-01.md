## 「Claude」と「Claude Code」の違い

ざっくりですが、違いはこれ↓

> **・Claude → AIと「会話する」  
> ・Claude Code → AIに「作業してもらう」**

![](https://assets.st-note.com/img/1776872616-QhBVZITDwGW2EOM7FdCRcjz9.png?width=1200)

**▶Claudeは、**  
・文章を書いてもらう  
・相談にのってもらう  
・アイデア出し

など、「会話ベース」で使います。  
ChatGPTやGeminiと同じですね。

**▶Claude Codeは、  
一歩進んで、実際に作業してくれるAIエージェント**  
・ファイルを作る  
・作業を自動化する  
・ブラウザ操作や処理を実行する　などなど

つまり「自分でやってた作業」を丸ごとやってくれるんです！  
時間がかかっていた作業を自動で任せられます。

## でも…料金かかるんじゃ？

私がClaude Codeを使っていない一番の理由はこれです。

**無料プランではデスクトップ版Claude Codeは使えません**

一応、ターミナルのClaude Code自体は無料で使えますが、  
必ずAPIという部分で料金がかかります。

なので、どの形で使うにしろ、月額プランに加入して使うのが一般的です。

![](https://assets.st-note.com/img/1776872143-YfbkrUn62iQWlSaNcJtegKAR.png?width=1200)

※Proプランでは使えなくなる可能性があります
