## CLAUDE.md とは？？
CLAUDE.md とは、Claude に対する**説明書・ルールブック**を書くファイルです。
Claude との会話(セッション)を開始時に自動的に読み込まれ、CLAUDE.md に書かれた内容は Claude が理解してくれる特別なファイルです。

みなさんもこういった経験はありませんか？
「コーディングをする場合はコメントルールを反映させたい」
「語尾ににゃーをつけ続けたい」
このような、一つの会話だけでなく、ずっと続けてほしい内容を Claude Code へ提示するファイルが**CLAUDE.md**ファイルです。

### CLAUDE.mdの種類
CLAUDE.md はいくつか置く場所によって意味合いが異なってきます。

|ファイルの場所|効果範囲|
|-----|-----|
|~/.claude/CLAUDE.md|全プロジェクト共通のルール|
|/"プロジェクト直下"/CLAUDE.md|そのプロジェクト専用のルール|
|/"プロジェクト"/"サブフォルダ"/CLAUDE.md|そのサブフォルダ内だけのルール|
|CLAUDE.
