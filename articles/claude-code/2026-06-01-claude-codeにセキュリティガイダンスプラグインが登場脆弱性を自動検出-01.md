# Claude Code にセキュリティガイダンスプラグインが登場

Claude Code に、コードを書きながらリアルタイムで脆弱性を検出・修正を支援する **セキュリティガイダンスプラグイン** が新たに追加されました。

## インストール方法

全ての Claude Code ユーザーが利用可能です。プラグインマーケットプレイスからインストールできます。

上記コマンドからプラグインマーケットプレイスを開き、セキュリティガイダンスプラグインを検索してインストールしてください。

## 3段階のセキュリティチェック

このプラグインは **hooks** を通じて動作し、以下の3つのタイミングでコードをレビューします。

### 1. ファイル編集時

* 危険なライブラリの誤用など、よく知られたリスクパターンを検出
* コードを書いている最中にリアルタイムで警告

### 2. モデルターン終了後

* 差分（diff）全体をレビュー
* 単体では気づきにくい、より高度な問題を検出

### 3. コミット時

* 変更箇所の周辺コードも含めて読み込み
* 脆弱性の妥当性をコンテキストを踏まえて検証

## まとめ

セキュリティチェックをコーディングのワークフローに自然に組み込めるこのプラグインは、開発速度を落とさずにコードの安全性を高める強力なツールです。Claude Code を利用している方はぜひ試してみてください。
