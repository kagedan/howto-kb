Claude Codeを使っていて、こんな経験はないだろうか。

作業を頼んでいたら突然英語が返ってきて、なんとなく「yes」と打ったらファイルが消えた——とまではいかなくても、「これ、何を聞かれてるんだろう」とモヤっとしたことが一度くらいはあるはず。

Claude Codeはデフォルトで英語を混ぜてくる。日本語で話しかけても、処理の途中や確認のタイミングでは英語が出てくる。

でも安心してほしい。よく出てくるフレーズは15個くらいに絞られる。これを知っておくだけで、Claude Codeとのやり取りがかなり楽になる。

## よく出る英語フレーズ15選

### 「進めていいですか？」系

**Would you like me to proceed?**  
→ 「このまま進めていいですか？」  
答え方：「はい」「進めて」「OK」でOK。迷ったら「どんな変更が入る？」と聞き返せばちゃんと教えてくれる。

**Should I overwrite the existing file?**  
→ 「既存のファイルを上書きしていいですか？」  
答え方：問題なければ「はい」。大事なファイルなら「バックアップしてから上書きして」と追加するといい。

**Let me verify before proceeding.**  
→ 「進む前に確認します」  
答え方：何かを確認しに行っているので、そのまま待てばOK。

### 「危険な操作です」系

**This is a destructive operation. Are you sure?**  
→ 「元に戻せない操作です。本当にいいですか？」  
これが出たら一旦立ち止まる。ファイルの削除やデータの書き換えなど、取り消しが効かない操作のときに表示される。「何を削除しようとしてる？」と確認してから判断するのが正解。

**I don't have access to [tool]. Would you like to...**  
→ 「[ツール]にアクセスできません。代わりに〜しますか？」  
権限がないか、そのMCPが繋がっていないとき。「代わりの方法を提案してくれてる」ので、提案内容を読んでYes/Noを判断する。

### 「バックグラウンドで動かします」系

**I'll run this in the background.**  
→ 「バックグラウンドで実行します」  
重い処理を裏で走らせる。終わったら通知が来るので、その間は別の作業をしていてOK。

**Running in background with ID: xxxxxxx**  
→ 「バックグラウンドでID:〇〇として実行中」  
長い処理が裏で動いてる状態。終わると「完了しました」と報告が来る。

### 「どうしますか？」系

**Which approach would you prefer?**  
→ 「どちらのやり方がいいですか？」  
複数の実装案や方法を提示してくれている。番号か名前で選ぶだけでOK。「1番で」「Aのやり方で」と答えればいい。

**Do you want me to also update [file]?**  
→ 「[ファイル]も更新しましょうか？」  
関連するファイルも一緒に直す提案。「全部直して」か「そこは触らなくていい」で答えればOK。

**I found X issues. Should I fix all of them?**  
→ 「X個の問題を発見。全部直しますか？」  
「全部直して」か「重要なものだけ先に直して」で答える。多すぎると感じたら「上位3つだけ直して」でもOK。

### 「確信がない」系

**I'm not sure about this. Let me check...**  
→ 「確信がないので確認します」  
Claudeが調べに行くサイン。そのまま待てばいい。

**Could not find [file/tool]. Did you mean...?**  
→ 「[ファイル/ツール]が見つかりません。〜のことですか？」  
候補を提示してくれているので、「そう、それ」か「違う、こっち」で答える。

### 「完了しました」系

**I've completed the task. Here's a summary:**  
→ 「タスクが完了しました。概要は：」  
作業結果のまとめ。内容を確認してOKなら次のタスクを投げればいい。

**Would you like me to commit these changes?**  
→ 「この変更をコミットしますか？」  
git commitしていいか確認。よければ「コミットして」。「まだ待って」も通じる。

### 「ブロックされました」系

**The sandbox is preventing this operation.**  
→ 「サンドボックスがこの操作をブロックしています」  
権限の制限で操作できないとき。「許可して実行して」と伝えるか、Claudeの判断に任せればいい。

## まとめ：英語が出てきても慌てない

Claude Codeの英語は、ほぼ「進めていいですか」か「どっちがいいですか」のどちらかだ。

迷ったら「今何をしようとしてるの？日本語で説明して」と聞けばいい。Claudeはちゃんと日本語で説明し直してくれる。

英語が読めなくても使えるのが、Claude Codeのいいところでもある。
