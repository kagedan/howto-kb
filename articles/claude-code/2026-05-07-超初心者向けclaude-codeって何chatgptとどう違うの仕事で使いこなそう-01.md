「Claude Code、気になってるけどまだ触れてない…」  
「みんな進んでる気がして、今から始めて大丈夫かな…」

こんなふうに思っている方に向けて、Claude Codeの使い方を"最初から丁寧に"解説していきます。

ChatGPTとの違いを比較しながら、できるだけ難しい言葉は使わずに、  
「とりあえず触ってみよう」と思える内容にしていくので、気楽に読んでもらえたら嬉しいです。

## ■ Claude CodeとChatGPTの一番大きな違い

まず最初に、ここだけ押さえておけばOKです。

![](https://assets.st-note.com/img/1777676830-Fxn3dip1A5JoNP4wl7jeTyIO.png?width=1200)

Claude Codeは「実際に手を動かしてくれるAI」です。

「それってChatGPTと何が違うの？」と思いますよね。似ているようで、役割が少し違います。

### ① ChatGPTは「相談に乗ってくれる人」

ChatGPTは、質問すると答えを返してくれるAIです。

例えば、「自己紹介ページのHTMLを書いて」とお願いすると、コードを画面に表示してくれます。ただ、その後は自分で  
・コピーして  
・別のツールに貼って  
・保存して  
・表示する  
という作業が必要になります。

いわば、「やり方を教えてくれる人」という感じです。

### ② Claude Codeは「横で一緒に作業してくれる人」

一方、Claude Codeは少し違います。

「自己紹介ページを作って」と頼むと、  
・ファイルを作る  
・保存する  
・必要な処理をする

ここまでを、パソコンの中で実際にやってくれます。

イメージとしては、「隣で一緒に手を動かしてくれる人」に近いです。

### ③ 使えば使うほどラクになる

これが地味に大きいポイントです。

Claude Codeは、作業した内容がフォルダに残っていきます。

例えば、

・過去に作ったLP  
・自分のプロフィール  
・好きなデザインの雰囲気  
・記事の書き方

こういった情報が積み重なっていきます。

なので次からは、「前と同じ感じで」みたいなざっくりした指示でも通じるようになります。

毎回説明しなくていいのは、かなり楽です。

## ■ 非エンジニアでも使える理由

ここ、気になる方多いと思います。

「便利そうだけど、自分には難しそう…」

自分も最初そう思っていました。

でも実際に触ってみると、思っていたよりずっとシンプルでした。

### ① 日本語でそのままお願いできる

特別な書き方は必要ありません。

例えば、

「自己紹介ページを作って」  
「ファイル名を日付順に並ぶようにして」  
「この文章を短くまとめて」

こんな感じでOKです。

普段人にお願いするのと、ほとんど同じ感覚です。

### ② 専門知識がなくても始められる

「HTML？Python？よく分からない…」

この状態でも問題ありません。

わからなくても、「ホームページ作って」と頼めば形になります。

最初は意味が分からなくても大丈夫です。  
使っているうちに、少しずつ理解が追いついてきます。

### ③ できることが一気に増える

ここが一番大きなメリットかもしれません。

今まで「外注するほどじゃないけど、自分でやると時間かかるな…」と思っていたことが、自分でできるようになります。

例えば、

・簡単なLPを作る  
・スプレッドシートの整理を自動化する  
・PDFの内容をまとめる  
・日報を書く

こういった作業を任せられるようになります。

## ■ もう少しイメージを具体的に

実際にどんなことができるのか、3つだけ紹介します。

![](https://assets.st-note.com/img/1777677017-vil0Z42dTWkVxz9KhbnmNqFa.png?width=1200)

### ① LPが短時間で形になる

サービス内容を伝えるだけで、簡単なページなら短時間で形になります。

私の場合、自己紹介ページが30分でだいたい形になりました。

もちろんプロレベルではないですが、「まずは出す」という意味では十分使えます。

### ② 書類作成がかなり楽になる

請求書や見積書など、毎回同じ形式のもの。

一度仕組みを作ってしまえば、ほぼ手作業なしで更新できるようになります。

### ③ 文章をまとめて処理できる

例えば、

・過去の記事を全部リライト  
・SNS用に短くする

こういった作業も、まとめてお願いできます。

1つずつコピペする必要がないのは、かなり楽です。

## ■ 「自分には無理かも」と思った方へ

ここまで読んで、「なんとなくすごいのは分かったけど、自分にできるかな…」と思った方もいると思います。

大丈夫です。

まずは手を動かして、「できた」という感覚をつかむことを優先していきます。
