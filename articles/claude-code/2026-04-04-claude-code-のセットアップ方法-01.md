### 実行環境 (Ubuntu 20.04.6 LTS)

```
Ubuntu 20.04.6 LTS (GNU/Linux 5.15.0-1081-oracle aarch64)
```

今回はオラクルのクラウドを利用しました。

## Claude Code のインストール

ネイティブインストーラが現在の推奨方法で、Node.js不要で完了します。

```
curl -fsSL https://claude.ai/install.sh | bash
```

実行結果：

```
curl -fsSL https://claude.ai/install.sh | bash
Setting up Claude Code...

✔ Claude Code successfully installed!         
  Version: 2.1.92
                                                                           
  Location: ~/.local/bin/claude

  Next: Run claude --help to get started

⚠ Setup notes:
  • Native installation exists but ~/.local/bin is not in your PATH. Run:

  echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc

✅ Installation complete!
```

`⚠ Setup notes:`に従い、PATHを通します。

```
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc
```

以下で、インストールできているか確認します

## 認証

UbuntuサーバやリモートSSHセッションではブラウザが自動起動しないため、`c`を押してログインURLをコピーし、手元のブラウザで開く必要があります。

```
claude  # 起動すると認証URLが表示される
# → URLをコピーしてスマホやPCのブラウザで開く
# → Anthropicアカウントでログイン
# → サーバ側ターミナルに戻りENTER
```

## tmux のセットアップ

スマホからSSHが切れてもセッションを維持するために **tmux** が必要です

```
# tmuxセッション作成
tmux new -s claude

# この中でClaude Codeを起動
claude

# スマホのSSHが切れても → tmuxセッションは生き続ける
# 再接続後
tmux attach -t claude
```
