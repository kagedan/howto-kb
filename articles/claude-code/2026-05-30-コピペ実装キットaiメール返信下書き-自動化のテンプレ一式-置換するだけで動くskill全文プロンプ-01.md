* 「メールチェックして」で動く **skill 全文**

> コードやプロンプト中の <...> は**あなた専用の値に置き換える場所**です。

## 0. 前提（これだけ用意してください）

* **AIエージェント**：私は Hermes を使っています（Claude Code等でも考え方は同じ）。ここでは Hermes 前提でコマンドを書きます。
* **VPS**：月数百円〜のレンタルサーバー1台。ここに Hermes を常駐させ、PCを閉じても動く状態にします。
* **Gmailアカウント**：1つでも2つでもOK（このキットは2アカウント対応で書いてあります。1つなら account2 の行を消すだけ）。

ディレクトリは Hermes のプロファイル配下を <PROFILE> と表記します（例：~/.hermes/profiles/<あなたのプロファイル名>）。

## 1. Gmailと“安全に”つなぐ ＆ 認証の橋渡し（つまずきNo.1）

### 1-1. 権限は「読む＋下書き」まで

OAuthの**スコープ**で、Gmailの「読み取り」と「下書き作成」までに絞ります。**送信スコープは付けない**のが安全設計の肝。複数アカウントは各々連携します。

### 1-2. VPSはブラウザが無い → 認証はこの“橋渡し”で

VPSには画面が無いので、OAuthの「ブラウザでログイン」が普通にはできません。私がハマった所です。解決策は **「手元PCのブラウザで認証 → 取れたトークンをVPSへ転送」**。

手元PC（Mac例）でローカルサーバ方式の認証を実行：

```
# 手元PCで（uv等の一時環境でOK。ライブラリはgoogle-auth-oauthlib）
# 1) VPSからclient_secretを取得
scp <VPSホスト>:<PROFILE>/google_client_secret.json ./client_secret.json

# 2) ローカルサーバ方式で認証（ブラウザが自動で開く→対象アカウントでログイン→許可）
#    redirect は http://localhost:<空きポート> を使う（Desktop型クライアントなら任意ポートOK）
#    ※ ブラウザ指定が必要な環境では open -a "Google Chrome" "<認証URL>" で開く

# 3) 取得した google_token.json をVPSへ戻す
scp ./google_token.json <VPSホスト>:<PROFILE>/google_token.json
```

> ポイント：認証後ブラウザは http://localhost:... で「接続できません」になりますが**正常**。ローカルで立てたサーバが裏でコードを受け取ってトークン化しています（コピペ不要）。

### 1-3. トークンが数日で切れる人へ

Google Cloudの**OAuth同意画面を「本番公開」**にしておくと、リフレッシュトークンが7日で失効しなくなります（“テスト”状態だと数日で切れる）。

### 1-4. アカウント用ラッパー

各アカウントは、環境変数で切り替えるラッパーを1枚ずつ用意します（中身はGmail API呼び出しスクリプトを叩くだけ）：

```
# <PROFILE>/gmail-account1.sh
#!/usr/bin/env bash
exec env HERMES_HOME=<PROFILE> <python> <PROFILE>/skills/.../google_api.py "$@"

# <PROFILE>/gmail-account2.sh  （2つ目は HERMES_HOME を別ディレクトリに）
#!/usr/bin/env bash
exec env HERMES_HOME=<PROFILE>/acct2 <python> <PROFILE>/skills/.../google_api.py "$@"
```

以降、bash <PROFILE>/gmail-account1.sh gmail search ... のように使います。

## 2. 署名ファイル（地味だけど効く“仕上げ”）

下書きの末尾に貼る署名を、**1ファイルに正本として置く**。AIには「ここから一字一句コピペ」とだけ指示します（毎回生成させない＝崩れない）。

<PROFILE>/skills/.../references/signatures.md を作り、こんな形で：

```
## Account 1
（ここに、あなたが実際に使っている署名ブロックをそのまま貼る）
<あなたの署名1>

## Account 2
<あなたの署名2>
```

> 記号（区切り線・全角スペースの位置）まで**1文字も変えない**のがコツ。AIには「U+25A1 などの記号もそのまま」と釘を刺します。
