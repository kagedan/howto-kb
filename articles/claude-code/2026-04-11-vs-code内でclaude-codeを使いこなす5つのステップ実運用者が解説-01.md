Claude Code の VS Code 拡張機能、「とりあえず入れてみたけど使いこなせてない」状態になりがちです。チャットパネルが開いて質問に答えてくれるのはわかった、でもそれ以上の使い方がよくわからない——そういうフェーズで止まっている方に向けて、実際に差が出る5つのポイントを整理しました。

:::note info
対象読者: Claude Code VS Code 拡張機能をインストール済みで、基本的な使い方は把握している方。VS Code 1.98.0 以上が必要です。
:::

## ステップ1: パーミッションモードを使い分ける

ほとんどの方がデフォルトのまま使っていますが、**パーミッションモードの選択が作業効率に直結します**。

プロンプト入力欄の下部にあるモード表示をクリックすると、3つのモードを切り替えられます。

| モード | 動作 | 向いている場面 |
|--------|------|---------------|
| `default` | アクションごとに確認を求める | コードレビューや慎重に進めたい変更 |
| `plan` | 変更前
