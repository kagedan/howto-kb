【Goose Adsスキル 要約】

Claude（Claude Code / Cowork / Codex）で使える「/goose-ads」スキルが登場。
競合の実績広告を分析しながら、自分のブランドに最適化された広告クリエイティブを自動生成してくれます。

■ 核心
「すでに成果を出している広告」をAIが参考にしつつ、ブランドの声・ロゴ・アセットを忠実に守って生成。
デザインが苦手でも、プロ並みのMeta広告クリエイティブが短時間で作れます。

■ 主要機能
・競合広告インテリジェンスを自動活用
・ブランドキット（メッセージ・トーン・資産）を自動解析
・最適テンプレートを選んで複数バリエーション生成
・Claude内で完結（外部ツール不要）

■ 利用方法（超簡単）
1. インストール（1回だけ）
   npx gooseworks install --all

2. Claude内で実行
   /goose-ads create ads for my brand [あなたのサイトURL]

3. （任意）生成後に https://t.co/mC5MTcLuHt でテンプレート調整も可能

例：https://t.co/wuCfbQ01wBで試すと、数分で6つのオン・ブランド広告が完成しました。

■ 背景
Gooseworksが公開している108個のオープンソース成長スキル（Ads・競合分析・コンテンツ・SEOなど）のひとつ。
世界の急成長スタートアップがすでに活用中です。
