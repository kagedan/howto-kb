Claude Code 的 session 管理有比较高的技巧天花板，这个观察很准。

rewind、主动 compact 并附带 handoff 消息、拆分 subagent、开新 session，这几个操作看起来都是基础功能，但什么时候用哪个，怎么组合，确实需要思考。

不管好不好，这是 agentic workflow 的普遍问题。context 是有限资源，管好它本身就是一项技能，跟管理内存或数据库连接池的思路类似。
