【速報｜Claude Code が大幅アップデート】

ついに来ました！
Claude Desktop 経由で、Claude Code がローカルファイルに直接アクセス可能に！

これ、地味に見えて
開発体験が一段階変わるアップデートです😳

どう使うのか？解説していきます👇

これまで
「Claude Code＝ターミナル前提」
でしたが──

その前提、崩れました。

【今回のアップデートで何が変わった？】

Claude Desktop から
そのままローカルの Claude Code を操作可能になっています。

つまり👇
・黒い画面を常に開く必要なし
・GUI感覚でコード生成／修正を指示
・ローカルファイルを直接読ませて実装
・「ターミナル怖い問題」が実質解消

【使い方（拍子抜けするほど簡単）】

① Claude Desktop をダウンロード
② サイドバーの「Code」トグルを ON
③ Claude Code に触らせたいローカルフォルダを選択
④ いつも通りプロンプトを送るだけ

これで
「考える → 指示する → 実装される」 が
一気に手元で完結します。

Claude Code を
「難しそう」で止めていた人ほど、
今回のアップデートは本気で触る価値あります。
