会議前にアジェンダを作るのが地味に面倒です。

「何を話すか」は頭にある。でも「それをどう整理して参加者に送る形にするか」に時間がかかる。特に急に決まった会議や、毎週繰り返す定例会議は、毎回同じような作業をしている。

Claude Codeで、これを終わりにできます。**「今日話したいこと」を箇条書きで書くだけで、整形されたアジェンダが1分で出てきます。**

## Before / After

**Before：** 会議内容を頭で整理 → 清書 → フォーマット調整 → 送信（10分）

**After：** 話したいことの箇条書きをClaude Codeに渡すだけ → 整形済みアジェンダが出てくる → **1分で完了（90%削減）**

## 準備するもの

- Claude Code（インストール済みであること）   
- 話したい内容のメモ（箇条書き・走り書きでOK）

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：話したいことをざっくり書く

完璧でなくていいです。メモ程度で十分。

```
・先週の進捗確認
・〇〇の件どうするか決める
・来月のスケジュール調整
・田中さんから連絡事項
```

### ステップ2：Claude Codeに渡す

```
以下の内容で会議アジェンダを作ってください。
会議名：週次定例
日時：〇月〇日 〇時〜〇時（30分）
参加者：〇〇、〇〇、〇〇

話したい内容：
・先週の進捗確認
・〇〇の件どうするか決める
・来月のスケジュール調整
・田中さんから連絡事項
```

![](https://assets.st-note.com/img/1776991902-ML2A7chv1PR3YSfbjCUOa9em.png?width=1200)

### ステップ3：出てきたアジェンダを確認する

Claudeが時間配分を考えながら整形してくれます。

```
## 週次定例 アジェンダ

日時：〇月〇日（〇）〇時〜〇時
場所：〇〇（Zoom）
参加者：〇〇、〇〇、〇〇

1. 先週の進捗確認（10分）
2. 〇〇の方針決定（10分）
   ・現状の整理
   ・選択肢の確認
   ・結論を出す
3. 来月スケジュール調整（5分）
4. 連絡事項 / 田中（5分）

決定が必要な事項：〇〇の方針
```

![](https://assets.st-note.com/img/1776991910-89jutQLe2cKHXhvfBlpaOV6N.png?width=1200)

### ステップ4：時間配分を調整して送付

```
2番の議題をもう少し詳しく展開してください。
全体を20分に収めてください。
決定事項の欄をもう少し目立たせてください。
```

## できました。ただ、このままだと1つ問題があります。

毎週同じメンバー・同じ構成の定例会議に、毎回「参加者：〇〇、〇〇、〇〇」「会議名：週次定例」と説明し直すのは手間です。

しかも「アジェンダが終わったら議事録フォーマットに変換して」も毎回指示が必要。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、会議名・参加者・固定アジェンダ・出力ルールを覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎回のアジェンダ作成が変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「参加者は〇〇と〇〇で」と説明が必要   
✅ ファイルあり：自動でメンバー構成を覚えたアジェンダが出てくる

❌ 設定なし：毎回「30分に収めて」と追加   
✅ ファイルあり：常に時間配分が自動調整される

❌ 設定なし：会議後に「議事録に変換して」と改めて指示   
✅ ファイルあり：アジェンダ→議事録が1コマンドで完成

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「アジェンダ作成の手間」が消えます。**

## 100円で何が変わるか

会議アジェンダ作成10分 × 週3回 × 50週 = **年間25時間がアジェンダ作成に消えています。**

設定ファイル1枚で、来月からその時間が大幅に減ります。

**100円 ÷ 20時間削減 = 1時間あたり5円。** 定例会議が多い方ほど効果が大きい設定です。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-010（添付）**　←　ダウンロードして置くだけ   
- **定例会議「毎回1行で完成」設定**　←　懸案事項だけ入れればフル形式で出てくる   
- **アジェンダ→議事録 自動変換の使い方**　←　会議前後の資料が一貫したフォーマットになる

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「定例会議の名前と参加者を教えてください」→ 答える   
- 「毎回必ず入れるアジェンダ項目はありますか？」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたの会議設定を覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
