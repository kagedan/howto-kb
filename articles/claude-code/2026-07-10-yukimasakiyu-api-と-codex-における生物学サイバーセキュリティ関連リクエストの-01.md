2 つ目の条件に該当する場合、GPT-5.6 Sol および Terra を利用している間は、サイバーセキュリティや生物学に関するリクエストだけでなく、アカウントのすべてのリクエストに対して追加チェックが一時的に実行されることがあります。

GPT-5.6 が ChatGPT で利用可能になると、これらのチェックは ChatGPT と Codex 全体でアカウント レベルで適用される場合があります。つまり、同じ OpenAI アカウントを使用している場合、Codex での最近のアクティビティが ChatGPT のリクエストに影響したり、その逆が起きたりすることがあります。

追加チェックは停止しますか？
アカウント レベルの追加チェックは一時的なもので、最近の使用状況によって変わります。最近のアクティビティが該当する安全性基準を満たさなくなり、一時的な期間が終了すると、リクエストは通常の動作に戻ります。

新たに潜在的リスクのあるアクティビティがあると、その期間が延びることがあるため、すべてのアカウントに共通する一定の期間はありません。

より高速なモデルを使用できますか？
より高速なモデルで再試行 が表示されている場合、これを選択すると、同じリクエストが GPT-5.6 Luna で再試行されます。GPT-5.6 Luna は、この追加チェックのエクスペリエンスを使用しない、より高速ですが高度な処理能力は低めの GPT-5.6 モデルです。より高速なモデルは応答が速い場合がありますが、複雑なタスクへの対応力は低い可能性があります。別のモデルを使用しても、OpenAI の利用ポリシーやその他の保護措置を回避することにはなりません。

モデルの変更は任意です。Codex（将来的には ChatGPT も）は、このコントロールによって通知なしにモデルを切り替えることはありません。
