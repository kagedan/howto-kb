【悲報】Claude Code でサブエージェントを並列で回しすぎると壊れる

複数エージェントの返答が一気にコンテキストに流れ込んで 、Auto compact 用のコンテキストが残らなくなったみたいです...

CLAUDE. md に 「サブエージェントの結果を受け取ったら、必要に応じて /compact を実行すること」 的な指示を入れておくと良さそうです。
