海外の大学生はバイトせずにこれでメシ食ってる。 

AIに子供向けショート動画作らせて、あとは放置でチャリンチャリン🥹

① YouTubeで「子供向けアニメ」「幼児向け教育」「3Dカートゥーン」などのジャンルを検索する。 https://t.co/t5hoDbFbK7

①YouTubeで「子供向けアニメ」「幼児向け教育」「3Dカートゥーン」などのジャンルを検索します。

動画内で紹介されている 「HappyTales Kids」 のように、数百万人規模の登録者がいて、ミリオン再生（100万回以上）を連発しているチャンネルを見つけます。

そしたら、ターゲットにしたチャンネルのトップ画面（動画一覧が並んでいる画面）、または動画内の特徴的なキャラクターが映っているシーンをスクショ

② Higgsfield MCPのページへアクセス

連携用のURLをコピーしときます。

③ Claudeの設定を開く

Claudeにログインし、「Settings（設定）」 ＞ 「Connectors（コネクター）」 を開きます。

「Add custom connector」をクリックし、以下のように入力して「Add」で保存します。

Name（名前）：Higgsfield
Remote MCP server URL：先ほどコピーしたURLを貼り付け

⑤動画を勝手に作ってくれる

Claudeがチャンネルの強み（キャラクター主導、不変の価値観など）を分析し、週ごとのロードマップを提案したのち、Higgsfield AIを通じて実際の3Dアニメーション動画を自動で生成・出力します。
