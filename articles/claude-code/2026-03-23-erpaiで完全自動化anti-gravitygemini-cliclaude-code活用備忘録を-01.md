はじめまして。現在は、社内のERPシステム（会計、人事、在庫など）に対するRPA（業務自動化）の実装と、それを効率化するためのAI技術の学習・実務適用を担当しています。

これまでは手作業で行っていたデータ処理を、AIと連携した自動化スクリプトを組んで効率化することを目指しています。

普段使っている技術・環境

主たる環境：Anti-gravity, Gemini CLI, CLAUDECODE

言語：Python、C#、Excel VBA（AIコード生成の対象として）

環境：Windows 11

最近は、AI（Claude CodeやGemini CLIなど）を活用した開発（プロンプトによるコード生成、環境構築のトラブルシューティング）を重点的に学習しています。  
![](https://static.zenn.studio/user-upload/d93e6a787e1a-20260323.png)  
Zennを始めた理由  
AIを用いた開発フロー（AI駆動開発）において、どのような指示（プロンプト）を出せば実務に耐えうるERP RPAスクリプトが作れるのか、日々の試行錯誤のプロセスをアウトプットするためです。

学習の定着を図りつつ、同じように業務自動化やAIツール活用に取り組む方の参考になればと思いZennを始めました。

これから発信していくこと

Anti-gravityとAIを組み合わせたRPA構築ノウハウ

Gemini CLI / Claude Code を用いたコード生成の検証
