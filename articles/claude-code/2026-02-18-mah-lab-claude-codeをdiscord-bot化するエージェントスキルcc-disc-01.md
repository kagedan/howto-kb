Claude Codeをdiscord bot化するエージェントスキル「cc-discord-bot」をつくったら、Claude Codeが完全にパーソナルエージェントになってくれた。

discordでチャットできるし、スケジュール起動もできるし、エージェントから話しかけてくれたりもするよ。

ファイルも作って送ってくれる。便利。
