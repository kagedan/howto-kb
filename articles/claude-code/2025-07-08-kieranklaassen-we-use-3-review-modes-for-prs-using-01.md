We use 3 review modes for PRs using Claude Code:

1. Regular review (catches basics)
2. Best practices (thinks deeply about conventions)
3. Critical review (goes nuclear on edge cases)

Different tools for different situations. Sometimes you need a gentle nudge, sometimes a sledgehammer. cc @nityeshaga
