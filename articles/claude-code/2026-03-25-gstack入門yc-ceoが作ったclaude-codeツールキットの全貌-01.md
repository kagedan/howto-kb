# GStack入門：YC CEOが作ったClaude Codeツールキットの全貌

> 本記事は2026年3月時点の情報をもとに書いています。GStackは活発に開発が続いているOSSなので、最新情報はGitHubリポジトリを合わせてご確認ください。

---

## はじめに — Garry TanがClaude Code用に作ったもの

YCombinator — スタートアップ界隈では知らない人はほとんどいない、あのYCのCEOです。そのGarry Tanさんが、2026年3月14日に自分自身のClaude Codeワークフローのために作ったツールキットがあって。それが **GStack** なんです。

GitHubに「自分用に作ったワークフローツールキット」として公開されています。オープンソースで、誰でも使える。

正直に言うと、最初に見たとき「なんでYCのCEOがこんなもの作ってるんだろう」と思ったんですよね。でも調べていくうちに、「プロダクトを磨き続けてきた人がAIコーディングツールをどう使っているか」の答えのひとつが、このGStackに込められているんだと気づいた。
