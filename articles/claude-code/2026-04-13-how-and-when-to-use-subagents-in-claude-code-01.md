Claude Codeのサブエージェント機能の使い方とベストプラクティスを解説する公式ブログ記事。サブエージェントは独立したコンテキストウィンドウを持つ隔離されたClaudeインスタンスで、リサーチ集約タスク・独立した並列タスク・客観的レビュー・検証・パイプラインワークフローに適している。自然言語での指示やCLAUDE.mdでのルール定義で利用でき、順序依存の作業や同一ファイルの同時編集には不向き。会話的な使用から始めてパターンに基づいて自動化を進めるアプローチを推奨している。
