## Mythosとは何か──3行で

Claude Mythosは、Anthropicが2026年4月7日に発表した同社史上最高性能のAIモデルだ。  
  
ただし**一般公開はされていない**。

現在アクセスできるのは、「Project Glasswing（プロジェクト・グラスウィング）」という防衛検証プログラムに参加する審査済み企業のみ。  
  
参加企業は約50社で、AWS・Apple・Microsoft・Google・Nvidia・JPMorganChase・Ciscoなど世界的なテック・金融・セキュリティ企業が名を連ねる。

つまり今この瞬間、あなたの隣の人間がMythosを使って何かを攻撃することは、**構造的に不可能**だ。

---

## 嘘① 「MythosはハッカーのAIだ」

→ **嘘。設計思想からして、これは防衛のためのAIだ。**

Mythosはサイバー攻撃ツールとして設計されたわけではない。  
Anthropicがセキュリティ評価を実施した結果、想定外に高い脆弱性発見能力が判明したため、一般公開を差し控えた。  
これは責任あるAI開発の模範例であり、隠蔽でも暴走でもない。

Project Glasswingという名称は、羽が透明で見えにくいグラスウィング蝶にちなんだものだ。「脆弱性はずっとそこにあった。  
  
ただ見えていなかっただけ」という哲学が込められている。

---

## 嘘② 「Mythosさえあれば誰でも高度な攻撃ができる」

→ **大嘘。使いこなすにはインフラと専門知識が不可欠だ。**

ソフトウエアセキュリティー企業セムグレップの創業者CEOは「その能力が現実の運用環境でどう活用されるか、現時点で分かっている事実に照らすと反応は過剰だ」と断言している。

Ciscoのグリエコ上級副社長は「F1カーを持っていても自転車しか乗ったことがなければ、最初から性能を最大限引き出すことはできない」と例えた。  
  
Mythosを最大限活用するには大規模な計算インフラと「ハーネス（実行環境）」が不可欠で、それを揃えられる主体は現時点では限られている。

さらに別の研究では、Mythosで報告された主要な成果の多くは、より安価なモデルを**並列**で動かすことで再現できるとされており、最新モデルへのアクセス以上に「規模と調整能力」が重要だと示唆している。

---

## 嘘③ 「Mythosが登場したから急に危険になった」

→ **嘘。脅威は以前から存在していた。Mythosは可視化しただけだ。**

早期アクセスを得た脆弱性調査の専門家は「この数カ月、いや場合によっては何年も前から、AIを使って対処しきれないほど多くのバグを見つけることができている」と指摘している。  
問題は脆弱性の「発見」ではなく、発見後に「検証し、優先順位を付け、システムを壊さずに修正する」ことだという。

FBIの元高官で現在セキュリティプラットフォームに籍を置くカイザー氏は「敵対勢力はAIがなくても、すでに非常に高い能力を持っている。  
ランサムウェア攻撃は1時間足らずで実行されている」と述べ、現時点で多くの脅威は依然としてAI頼みではないことを強調した。

---

## 嘘④ 「Mythosは守れないが攻めには使える」

→ **半嘘。守備に使えば最強の盾になる。これが証明されつつある。**

英国政府のAIセキュリティ機関（AISI）の独立テストにより、**Mythosは十分に堅牢な防御を持つ組織に対して、自律的な攻撃を信頼性高く実行できないことが確認されている**。

逆説的だが、Mythosへの最善の回答はMythosだ。  
守備に使えば守れる。これが今、グラスウィング参加企業が実証しつつある事実だ。

## Project Glasswingの実績──今、何が起きているのか

これが本稿で最も重要な部分かもしれない。  
メディアが騒ぐ「危険性」より、静かに進む「守備の実績」を見てほしい。

**✅ 修正済みの主な発見例**

・OpenBSDに潜んでいた**27年間見逃された脆弱性**（2026年3月25日パッチ適用済） 　  
→ 世界で最もセキュアとされるOSの一つに、1999年から存在していた重大なバグ

・Firefoxの複数の高深刻度脆弱性   
→ Mozillaと連携し修正済み

・FFmpegに16年間存在していたバグ（2003年のコミット由来）  
→ 発見・記録済み

・Linuxカーネルに関わるマルチステップの攻撃チェーン   
→ 文書化・対応中

**⚠️ しかし、本当の課題が浮き彫りになった**

Anthropicが公開したデータによると、Mythosが発見した脆弱性のうち**完全にパッチが当たったものは1%未満**だ。

「世界で最も強力な脆弱性発見エンジンが動き出したのに、世界のIT業界がその出力を吸収できていない」

これがGlasswingが暴いた、より深刻な構造問題だ。  
  
攻撃者は「機械の速度」で動く。  
守備側は「カレンダーの速度」でしか動けない。  
  
Mythosはこのギャップを、もはや無視できないものにした。

## 本当のこと──「参入障壁は下がった」

これは認める必要がある。  
Mythosは「以前のモデルより弱い指示文でも、より多くの脆弱性を発見できる能力」を持ち、「参入障壁が下がった」という側面は事実だ。

ただし「下がった」と「なくなった」は全く異なる。  
今でも高度な計算インフラと専門的な実行環境は必要だ。

金融大手ステート・ストリートの担当者も「確かに参入障壁は存在するが、それはかなり早く解決されるだろう」と述べており、現時点での壁は存在しつつも、**その壁は時間と共に低くなる**という認識は業界で共有されている。

## 現在のAI勢力図──Anthropicはどこにいるか

コーディング・セキュリティの主要ベンチマーク「SWE-bench Pro」では、MythosはGPT-5.4を約20ポイント、Gemini 3.1 Proを約23ポイント上回っている。  
大学院レベルの科学的推論「GPQA Diamond」でもMythosが94.5%でトップだ。

**現在の正直な勢力図：**

コーディング・セキュリティ領域 → Mythosが頭一つ抜けている   
汎用マルチモーダル（音声・動画処理）→ Gemini 3.1 Proが優位   
エージェント型ワークフロー → GPT-5.5（4月23日公開）が強い   
コスト効率 → Gemini Flash 2.5が最安クラス

どのAIが「最強」かは用途次第だ。  
ただしセキュリティと複雑な推論においては、現時点でAnthropicに対抗できる存在はない。

## Mythosの技術はいつ一般人に届くか──現実的な予測

ここが多くの人が知りたい部分だろう。

現在のロードマップをまとめると：

**フェーズ1（現在進行中）**   
Project Glasswing参加の約50社限定。  
価格は入力100万トークンあたり約3,600円・出力100万トークンあたり約18,000円で、一般モデルの5倍。

**フェーズ2（2026年Q3〜Q4、最速）**   
厳格な審査付きの企業向けAPI限定公開。  
誰でも申し込めるわけではなく、用途の制限が伴う見込み。  
「一般公開」とは程遠い。

**フェーズ3（2027年以降）**   
コンシューマー向けの一般提供。  
ただしMythosというブランドのまま出てくるとは限らない。

**現実的な解釈：**   
Mythosの能力は、おそらく今後登場するClaude Opus次世代版（「Claude 5」と呼ばれる可能性もある）に段階的に統合されて降りてくる。  
Mythosというモデルそのものを一般ユーザーが直接触れるのは、早くて2027年、もしくは永遠に来ない可能性もある。

ただし朗報がある。  
Anthropicは2026年2月にすでに「Claude Code Security」という企業向けセキュリティ製品を出しており、これが「Mythosに乗れない企業のための入口」になっている。  
今すぐ防衛強化したい企業にとっての現実的な選択肢だ。

なお、2026年7月初旬にはGlasswing開始から90日間の公式レポートが出る予定で、これがAnthropicの次の展開を読む最大の手がかりになる。

## 結論：Mythosで攻められるのが怖ければ、Mythosで守れ

恐怖を煽るメディアが見落としているのは、**AnthropicがMythosの発表と同時に何をしたか**という事実だ。

彼らはそれを秘匿しなかった。売らなかった。独占しなかった。

世界のサイバーセキュリティ最前線の企業に、100億円分の利用枠と共に無償提供した。

英国政府機関の独立検証が示した通り、**十分に堅牢な防御を持つ組織に対して、Mythosは自律的な攻撃を実行できない**。  
つまりMythosの最大の皮肉は**「Mythosで守れば、Mythosから守れる」**という構造だ。

Anthropicの創業者ダリオ・アモデイの哲学——

> 「もし強力なAIが登場するなら、それは安全性を最優先にする組織から生まれるべきだ」

は、今回の対応に一貫して反映されている。

恐怖を売るメディアより、責任を取る企業の行動を見ろ。  
  
それだけのことだ。
