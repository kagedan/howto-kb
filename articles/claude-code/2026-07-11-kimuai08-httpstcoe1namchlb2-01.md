## 第1章 生みの親が「8ヶ月コードを書いていない」の衝撃

まず、ボリス・チェルニーという人について、簡単に。

彼はAnthropicで Claude Code を作り、今もそのトップを務めている人です。以前はMetaで5年間プリンシパルエンジニアをやっていて、『Programming TypeScript』という技術書の著者でもあります。つまり、ゴリゴリに手を動かしてきた、本物のエンジニアなんですね。

![](https://pbs.twimg.com/media/HM6_9qIbsAAavKn.png)

そのボリスが、2026年6月のFortuneのカンファレンスで、こう言いました。

「もう8ヶ月くらい、手でコードを1行も書いていない。Claude Codeのコードは、100%Claude Codeが書いている」

自分が作ったツールのコードを、自分のツールに書かせている。ちょっと出来すぎた話に聞こえますよね。でも本人が公の場で言っているんです。2025年の後半以降、彼のコードは100%AIが書いている、と。

じゃあ彼は何をしているのか。遊んでいるわけじゃありません。

毎日、大量のAIエージェントを管理しているんです。彼の言葉を借りると、「今朝は数百を管理していた。日によっては数千、数万のこともある」。
