## そもそもClaude Codeって何？

簡単に言うと、**ターミナル上で動くAIペアプログラマー**です。

ChatGPTやClaudeのWeb版を使ったことがある方は多いと思います。あれのチャット画面で「このコード書いて」と頼む感じ、ありますよね。

Claude Codeは、それを**ターミナルの中で直接やる**んです。

何が違うかというと：

* **ファイルを直接読み書きできる**（コピペ不要）
* **プロジェクト全体を理解してくれる**（1ファイルだけじゃない）
* **コマンド実行もやってくれる**（テスト実行、git操作など）
* **コードの文脈を深く理解する**（100万トークンの超長文脈）

つまり、\*\*「AIに口頭で指示したら、勝手にコードを書いて、テストして、コミットまでしてくれる」\*\*みたいな世界です。

正直、最初に使ったとき鳥肌が立ちました。

## インストール方法（5分で完了）

ここからは実際の手順です。思ったより簡単なので、読みながらやってみてください。

### ステップ1：Node.jsを入れる

Claude Codeを動かすには、Node.js（バージョン18以上）が必要です。

まだ入っていない方は、ターミナルを開いて以下を実行：

```
# Macの場合（Homebrewを使う）
brew install node

# バージョン確認
node -v
```

Windowsの方は[Node.js公式サイト](https://nodejs.org/)からインストーラーをダウンロードしてください。

「v18.0.0」以上が表示されればOKです。

### ステップ2：Claude Codeをインストール

```
npm install -g @anthropic-ai/claude-code
```

これだけです。本当にこれだけ。

### ステップ3：認証する

```
claude
```

初回起動時に認証画面が出ます。Anthropicのアカウントでログインするか、APIキーを設定します。

**料金について：**

### ステップ4：使ってみる

プロジェクトのフォルダに移動して、`claude`と打つだけ：

```
cd my-project
claude
```

するとインタラクティブなチャットが始まります。

```
╭──────────────────────────────────────╮
│ ようこそ Claude Code へ！            │
│                                      │
│ 何でも聞いてください。               │
╰──────────────────────────────────────╯

> このプロジェクトの構成を説明して
```

これで準備完了。たった5分です。

## 実際に何ができるの？具体例で見てみよう

「インストールしたけど、何を頼めばいいの？」

はい、ここが一番大事なところ。具体的な使い方を紹介します。

### 1. バグ修正

```
> このエラーを直して: TypeError: Cannot read properties of undefined
```

エラーメッセージを貼り付けるだけで、Claude Codeが：

1. 関連するファイルを自動で探して
2. 原因を特定して
3. 修正コードを提案して
4. 許可すればそのまま修正してくれる

### 2. 新機能の追加

```
> ログイン機能を追加して。メールアドレスとパスワードで認証する形で
```

Claude Codeは**プロジェクト全体の構成を見て**、適切なファイルに適切なコードを追加してくれます。既存のコードスタイルに合わせてくれるのが地味にありがたい。

### 3. コードの説明

```
> src/utils/auth.ts の処理を初心者にもわかるように説明して
```

「この関数何やってるんだっけ…」という場面、ありますよね。Claude Codeは対象のファイルを読んで、わかりやすく解説してくれます。

### 4. リファクタリング

```
> この関数を小さな関数に分割して、テストしやすくして
```

「動いてるけど読みにくいコード」をきれいに整理。テストまで書いてくれることも。

### 5. Git操作

```
> 今の変更をコミットして。コミットメッセージはよしなに
```

差分を見て、適切なコミットメッセージを自動生成。プルリクエストの作成までやってくれます。

## 使いこなすコツ（これ知ってると全然違う）

半年ほど使い込んだ中で見つけた「これは便利！」というTipsを共有します。

### コツ1：CLAUDE.mdを書く

プロジェクトのルートに`CLAUDE.md`というファイルを置くと、Claude Codeが**毎回それを読んでくれます**。

```
# CLAUDE.md

## 禁止事項
- 既存の設定ファイル（package.json, tsconfig.json等）を無断で変更しない
- テストなしでコードを本番環境にコミットしない
- 承認なしにパッケージを追加しない
```

CLAUDE.mdは「やること」を書くより、**「やってはいけないこと」だけを書く**のがおすすめです。AIに自由度を持たせつつ、絶対に守ってほしいルールだけを明示する。シンプルで管理もしやすい。

### コツ2：具体的に指示する

**悪い例：**

```
> いい感じにして
```

**良い例：**

```
> src/components/Header.tsx のナビゲーションにドロップダウンメニューを追加して。
> メニュー項目は「ホーム」「プロフィール」「設定」「ログアウト」の4つ。
> Tailwind CSSでスタイリングして。
```

AIは具体的な指示ほど精度が上がります。\*\*「何を」「どこに」「どうやって」\*\*を明確にしましょう。

### コツ3：段階的に進める

一気に「アプリ全部作って」と頼むより、こうした方がうまくいきます：

1. 「まずプロジェクトの構成を提案して」
2. 「データベースのスキーマを設計して」
3. 「APIエンドポイントを実装して」
4. 「フロントエンドの画面を作って」

**小さなステップに分けて、確認しながら進める**のが成功の秘訣です。

### コツ4：/で始まるコマンドを覚える

Claude Codeには便利なスラッシュコマンドがあります：

* `/help` - ヘルプを表示
* `/clear` - 会話をリセット
* `/compact` - 会話を要約して圧縮
* `/cost` - 現在のセッションのコスト確認

特に`/compact`は長い会話でメモリが膨らんだときに重宝します。

### コツ5：怖がらずに「ダメ」と言う

Claude Codeが提案した変更が気に入らなかったら、遠慮なく：

```
> それじゃなくて、もっとシンプルなやり方でやって
> その変更は戻して
```

AIは何度やり直しても怒りません（ここ重要）。

## よくある質問

### Q. 料金はいくらかかる？

**Anthropic Maxプラン**なら月額$100（Opus 4モデル対応）または$20（Sonnet対応）で使い放題。APIキー利用の場合はトークン従量制です。

個人開発者なら、まずは$20のプランで試してみるのがおすすめ。

### Q. 日本語で使える？

はい！日本語で指示を出せば、日本語で返してくれます。コメントやコミットメッセージも日本語OK。CLAUDE.mdに「日本語で返答」と書いておけば完璧です。

### Q. オフラインで使える？

いいえ、Claude Codeはクラウド上のClaude AIと通信するため、インターネット接続が必要です。

### Q. セキュリティは大丈夫？

コードはAnthropicのAPIに送信されますが、Anthropicの利用規約ではAPIから送信されたデータをモデルの学習に使用しないと明記されています。ただし、機密性の高いプロジェクトでは社内ポリシーを確認してください。

### Q. VSCodeでも使える？

はい！VS Code拡張として**Claude Code for VS Code**が提供されています。ターミナルが苦手な方はこちらもおすすめです。
