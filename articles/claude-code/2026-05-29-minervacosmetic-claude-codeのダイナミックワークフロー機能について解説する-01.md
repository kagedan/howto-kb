Claude Codeの「ダイナミックワークフロー」機能について解説する。

「create a workflow」と入力するか、effortメニューで「ultracode」をオンにすると、数百のエージェントが並列で動き、互いの成果物をチェックし合う。

作業単位が「1ファイル」から「コードベース全体」に変わる。マイグレーション、監査、リライト、フレームワーク移行——スプリント計画が必要だった作業が一晩で終わる。

特筆すべきポイントがある。エージェントたちは同じ問題に独立してアタックし、その後、別のエージェントが答えを壊そうと試みる。収束するまで繰り返す。これはシニアエンジニアチームの動き方と同じだ。ただし、このチームは深夜3時でも稼働し続けて疲れない。

ワークフローが中断されても、続きから再開できる。つまり何日間も走り続けるタスクを投げられる。セッションではなく、日数単位だ。

トークン消費は確かに速い。Anthropic自身もそう言っている。ただし計算は単純だ——本来チームで3ヶ月かかるコードベース移行が1週間で終わるなら、500ドルのトークン代はソフトウェア開発史上最高のトレードになる。

一人の人間が作れるものの上限がまた上がった。
