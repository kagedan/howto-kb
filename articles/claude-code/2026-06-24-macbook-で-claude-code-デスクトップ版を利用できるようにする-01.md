# 💬 概要

本記事では、プログラミング未経験の **Mac ユーザー**でもスムーズに開発を始められるように、**Claude Code デスクトップ版**のセットアップ手順をまとめています。MacBook はもちろん、iMac や Mac mini でも同じ手順で進められます。所要時間の目安は **15〜30 分程度**です。

---

# 🏁 ゴール

> **Mac で働く非エンジニアが、Claude Code を使って業務効率化ツールを開発できるように、Claude Code デスクトップ版をセットアップした状態にする。**

---

# 💡 前提知識

## ❓ Claude Code とは

**Claude Code は、あなたの PC 上で実際に作業をしてくれる AI アシスタント**です。ファイルを読み書きしたり、データを整理したり、簡単なツールを作って動かしたりできます。

普段ニュースなどで耳にする「AI に相談する」イメージよりも一歩進んでいて、**「相談相手」ではなく「手を動かしてくれる作業担当」** だと考えるとイメージしやすいです。

非エンジニアの方でも、たとえば次のような業務効率化に使えます。

* Excel の定型集計やレポート作成を自動化する
* 大量のファイルやデータをまとめて整理・名前変更する
* 日々の繰り返し作業を「ツール化」して、ボタンひとつで終わるようにする
* 簡単な入力フォームや変換ツールを作る

## 🆚 ふだんの Claude（claude.ai のチャット）との違い

「Claude」と「Claude Code」は名前が似ていますが、できることが違います。

|  | ふだんの Claude（チャット） | Claude Code |
| --- | --- | --- |
| 主な使い方 | 文章で相談・質問・文章作成 | PC 上で実際に作業（ファイル操作・ツール作成など） |
| あなたの PC のファイル | 触れない | 直接読み書きできる |
| 成果物の使い方 | 自分でコピー＆ペーストして使う | Claude が PC 上に直接作ってくれる |
| たとえると | 物知りな相談相手 | 手を動かしてくれるアシスタント |

## 📌 3つの利用方法と、今回の選択

実は Claude Code を利用する方法は 3 つあります。

| 利用方法 | どんな画面か | 向いている人 |
| --- | --- | --- |
| **CLI（コマンドライン）** | 黒い画面に文字でコマンドを打ち込む「ターミナル」を使う | エンジニア／ターミナルに慣れた人 |
| **VS Code 拡張機能** | プログラマー向けエディタ「VS Code」に組み込んで使う | 普段から VS Code を使う人 |
| **デスクトップ版（専用アプリ）** | ボタン中心の分かりやすい画面。**ターミナル不要** | 初めての人・非エンジニア |

**今回は非エンジニアの初心者の方が対象なので、ターミナルを使わずに直感的に操作できる「デスクトップ版」** を使います。

# 💻 セットアップ手順

## STEP 0. 有料プランとアカウントを確認する

まず、サインインに使う **Anthropic アカウント**と、そのアカウントに**有料プランが紐づいているか**を確認します。

* 会社からアカウントが支給される場合は、**社内の案内に従ってアカウント情報を受け取って**ください。
* 個人で用意する場合は、有料プラン（Pro など）への加入が必要です。

!

⚠️ **Anthropic アカウント + 有料プランは必須です**

Claude Code（後述の「Code」タブ）を使うには、**Anthropic の有料プラン（Pro / Max / Team / Enterprise）** が必要です。無料プランでは利用できません。

どのアカウント・プランでサインインすればよいかは、**社内の案内に従ってください**。会社からアカウントが支給される場合や、プランの契約が必要な場合があります。不明なときは、セットアップを始める前に社内の担当者へご確認ください。

## STEP 1. Claude デスクトップアプリをインストールする

Claude Code は「**Claude**」という名前のデスクトップアプリに含まれています。まずはこのアプリをインストールします。

1. 下記の公式ページを開きます。  
   👉 <https://code.claude.com/docs/ja/desktop-quickstart>
2. ページ内の「**インストール**」セクションにある **macOS 版のダウンロードボタン**をクリックします。
3. ダウンロードした **`.dmg` ファイルをダブルクリックして開き**、表示された **Claude のアイコンを「アプリケーション（Applications）」フォルダにドラッグ&ドロップ**します。

## STEP 2. サインインする

1. **Launchpad**（または「アプリケーション」フォルダ）から **「Claude」を起動**します。
2. **STEP 0 で確認した Anthropic アカウントでサインイン**します。
   * ブラウザが開いてログインを求められる場合があります。画面の案内に従って完了してください。

!

🔐 **初回起動で確認画面が出たとき**

インターネットからダウンロードしたアプリのため、初回起動時に「**“Claude”はインターネットからダウンロードされたAppです。開いてもよろしいですか？**」と表示されることがあります。その場合は「**開く**」をクリックしてください。

もし「**開けません（開発元が未確認）**」のような表示でブロックされた場合は、「アプリケーション」フォルダの Claude を **右クリック →「開く」** を選ぶか、Apple メニュー →「**システム設定**」→「**プライバシーとセキュリティ**」を開き、下のほうにある「**このまま開く**」を押してください。

## STEP 3. 「Code」タブを開く

Claude アプリには「**Chat**」「**Cowork**」「**Code**」の 3 つのタブがあります。今回使うのは「**Code**」タブです。

1. 画面**上部中央の「Code」タブ**をクリックします。
2. 正しく開けば、プロンプト（指示）を入力する画面が表示されます。

!

🔍 **このとき表示されるかもしれないメッセージ**

* **「アップグレードしてください」と表示された場合** → 有料プランが必要です。STEP 0 に戻り、プランとアカウントを確認してください。
* **サインインを促された場合** → サインインを完了し、アプリを再起動してください。
* **「403」エラーが出た場合** → いったんサインアウト → サインインし直し、アプリを再起動してください。

## STEP 4. 開発言語（Python / Node.js）をインストールする

業務効率化ツールの多くは、**Python（パイソン）** や **Node.js（ノード）** という「プログラムを動かすための土台（言語の実行環境）」の上で動きます。これらをインストールしておくと、Claude Code が作ったツールを実際に動かせるようになります。

うれしいことに、**このインストール作業自体も Claude Code に手伝ってもらえます**。

1. 「**Code**」タブを開いた状態にします。
2. 画面下のほうで、実行環境として「**Local**（自分の PC 上で動かす）」を選びます。
3. 「**Select folder**」をクリックして、**作業用のフォルダ**を選びます。
   * まだ無ければ、デスクトップなどに「`claude-practice`」のような分かりやすい名前のフォルダを 1 つ作って、それを選んでください。

4. プロンプト（指示入力欄）に、次のように入力して送信します。

```
Python と Node.js をこの PC にインストールして、コマンドで実行できる状態にしてください。
インストールが終わったら、python3 --version と node --version でバージョンを確認してください。
```

5. Claude Code が**実行しようとする操作（コマンド）を提案**してきます。**「Accept（承認）」や「許可」を押して進めます**。
6. インストールが完了し、`python3 --version` と `node --version` でバージョン番号（例：`Python 3.x.x` / `v20.x.x` など）が表示されれば成功です。

!

💡 **うまく確認できない／自動インストールが失敗するとき**

まずは **Claude アプリ（または Mac 本体）を一度再起動**してから、もう一度「Code」タブで `python3 --version` と `node --version` を確認してください。

それでもうまくいかない場合は、公式サイトから手動でもインストールできます。

ダウンロードしたインストーラーを開き、画面の指示に従って進めればOKです。

## STEP 5. 動作確認（仕上げ）

最後に、ひととおり動くかを確認します。「Code」タブで、作業用フォルダを選んだ状態で次のように入力してみましょう。

```
このフォルダで Next.js (フロントエンド)と FastAPI (バックエンド)のアプリケーションを実装し、疎通できることを確認できるサンプルアプリケーションを実装してください。

実装したら、localhost:3000 でアプリを閲覧できるようにしてください。
```

提案された操作を承認し、**フォルダの中にファイルが作られ、ブラウザで <http://localhost:3000> が確認できたら、セットアップは完了です！**

セットアップが完了したら、最後にアプリケーションを停止するプロンプトを入力してください。

# 🆘 うまくいかないときは

| 症状 | 対処法 |
| --- | --- |
| 「Code」タブでアップグレードを求められる | 有料プランが必要です。社内の担当者に確認し、有料プランのアカウントでサインインしてください。 |
| サインインできない／403 エラーが出る | サインアウト → サインインし直し → アプリを再起動してください。 |
| 初回起動でアプリがブロックされる | 「アプリケーション」フォルダの Claude を右クリック →「開く」、または「システム設定」→「プライバシーとセキュリティ」→「このまま開く」。 |
| `git` が見つからないと言われる | 表示される「コマンドラインデベロッパツール」のインストールを実行してください。完了後にアプリを再起動します。 |
| インストール中にパスワードを求められる | Mac のログインパスワードを入力してください（Homebrew の導入などで必要になります）。 |
| `python3` や `node` のバージョンが表示されない | Claude アプリ、または Mac 本体を再起動してから、もう一度確認してください。 |
| ダウンロードしたファイルがどれか分からない | 通常は「ダウンロード」フォルダに保存されています。**Finder**（顔のアイコン）から探してください。 |
| インストールがブロックされる／管理者権限を求められる | 会社支給 Mac の場合は権限制限の可能性があります。社内の担当者または情報システム担当へご相談ください。 |
