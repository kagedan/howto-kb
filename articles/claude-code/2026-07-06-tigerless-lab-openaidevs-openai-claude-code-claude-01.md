Record & Replay focuses on active learning inside Codex: you intentionally demonstrate a workflow, hit record, and teach Codex how to turn that task into a reusable skill.

Auto Harness focuses on passive learning inside Claude Code: it observes real work traces as they happen in the CC environment, capturing commands, files, checks, preferences, failures, and edge cases, then turns them into reusable execution patterns.

One is a Codex-native way to teach by demonstration.
The other is a CC-native way to learn from the work already happening.
