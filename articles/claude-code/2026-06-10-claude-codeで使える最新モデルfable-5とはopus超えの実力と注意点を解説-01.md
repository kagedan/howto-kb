## 第1章:Fable 5とは何か

Claude Fable 5は、Anthropicが2026年6月9日に発表したClaude 5ファミリーの第1弾モデルである。位置づけが少し特殊で、従来の最上位だったOpusの上に新設された「Mythos級」という能力帯に属する。

Mythosは2026年4月に発表されたフロンティアモデルで、性能が高すぎるあまり、サイバー攻撃への悪用リスクなどから一般提供されず、防御的なセキュリティ用途に限定したプレビュー提供にとどまっていた。今回の発表は、その能力を2つのモデルに分けて出すという二正面戦略である。

1つがClaude Fable 5。高リスク領域に安全対策を施した一般向けモデルだ。もう1つがClaude Mythos 5。制限を外した完全版だが、「Project Glasswing」と呼ばれる枠組みを通じて、米政府のサイバー防御機関など承認された組織にのみ提供される。重要なのは、この2つが同一の基盤モデルだという点である。違いはセーフガードの有無だけで、通常の利用ではほぼ同等の性能を体験できる。

提供範囲も広い。APIのモデル名はclaude-fable-5で、Claude APIのほか、Amazon Bedrock、Google CloudのVertex AI、Microsoft Foundryからも同日に利用可能になった。さらにGitHub Copilotでも、Pro+以上のプランでモデル選択肢に追加されている。Anthropic自身はFable 5を「当社が提供する、最もインテリジェントな一般利用可能モデル」と位置づけており、Claude 5世代の顔として展開していく構えだ。
