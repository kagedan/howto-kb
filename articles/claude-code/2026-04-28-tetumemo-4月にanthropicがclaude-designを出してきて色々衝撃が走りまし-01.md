Claude Designをおさらいすると、、、

テキストを貼るだけで
アニメーション動画・スライド・Webサイト・アプリ・デザインシステムまで
全部コードで生成してくれるツール。

① アニメーション動画を4〜5分で生成
② スライドデッキをアニメーション付きで作成
③ ランディングページを80%の精度で再現
④ モバイルアプリのUIプロトタイプを生成
⑤ デザインシステムまで丸ごと構築

5つの実例を見てみると、使い方がシンプルすぎて笑う
ㅤ
◆ アニメーション動画
→ ブログ記事のテキストを貼って「動画にして」と指示するだけ
→ 長さ・トーン・用途をAIが逆質問してくれる
→ 4〜5分で本当に動くアニメーション動画が完成
→ ただし今はMP4エクスポート非対応、画面録画で保存が必要
ㅤ
◆ スライドデッキ
→ 普通に作ると「退屈で静的」なスライドになりがち
→ プロのコツは「先に動画として生成させてからスライドに変換」
→ これだけでアニメーション付きの豪華なスライドになる
ㅤ
◆ ランディングページ
→ 既存サイトの名前を言うだけで80%の精度で再現
→ インタラクティブな3D地球儀や、単語に合わせたパーティクルエフェクトも生成可能
ㅤ
◆ モバイルアプリ
→ 「忙しいプロ向けの筋トレアプリ」と言うだけで美しいUIが生成
→ AIが自分でスクリーンショットを撮って表示崩れを自己修正する機能が内蔵
ㅤ
◆ デザインシステム
→ FigmaのUIキット画像を読み込ませると、タイポ・カラー・アイコン・モーションまで一式生成
→ ただしトークン消費が多く、まだバグも多い段階
