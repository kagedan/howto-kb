プラグイン向けの新機能2つと、体感に効く安定性改善、そしてバグ修正が24件。地味ながら着実に磨かれたリリースです。

**今回の注目ポイント**

1. **PreCompact フックでコンテキスト圧縮を制御可能に** --- 長いセッションの自動圧縮をフックで止められるようになりました
2. **プラグインにバックグラウンドモニター機能が追加** --- セッション裏で監視処理を自動起動するプラグインが作れます
3. **ストリーム・ネットワーク周りの安定性向上** --- 無限ハングや無言スピナーがようやく解消されました

## PreCompact フックでコンテキスト圧縮を制御できるようになった

対象: 長いセッションを日常的に使うユーザー、フックを活用しているパワーユーザー

### 何が変わった?

Claude Codeにはセッションが長くなるとコンテキストを自動圧縮する「コンパクション」という仕組みがあります。コンテキストウィンドウを効率よく使うには必要な処理ですが、「今まさにこのコンテキスト全体が必要なのに...」というタイミングで圧縮が走ってしまうことがありました。

今回 `PreCompact` というフックイベントが追加され、コンパクション発動の直前に介入できるようになりました。

### どう使う?

`settings.json` のフック設定に `PreCompact` イベントを追加します。

```
{
  "hooks": {
    "PreCompact": [
      {
        "type": "command",
        "command": "/path/to/check-compaction.sh"
      }
    ]
  }
}
```

フックの戻り値でコンパクションの実行を制御します。

* 終了コード `0` --- そのまま圧縮を実行
* 終了コード `2` または JSON `{"decision":"block"}` を返す --- 圧縮をブロック

たとえば「特定のブランチで作業中はコンパクションを止めたい」「デバッグセッションの途中で圧縮されたくない」といった制御ができます。長時間のデバッグ中に重要なコンテキストが消えて「あれ、さっき何の話してたっけ」となっていた方にはありがたいはずです。

## プラグインにバックグラウンドモニター機能が追加

対象: Claude Code プラグイン開発者、プラグインエコシステムに注目している方

### 何が変わった?

プラグインの manifest に `monitors` キーをトップレベルで定義すると、セッション開始時や skill 呼び出し時にバックグラウンドで自動起動する監視処理を登録できるようになりました。

### 何がうれしい?

これまでプラグインは、ユーザーが明示的にスラッシュコマンドを叩くかツールを呼ぶかしないと動けませんでした。「裏で常に何かを監視して、必要なときにだけ通知する」というパターンが取れなかったわけです。

今回の `monitors` でそれが可能になります。

```
{
  "name": "my-build-watcher",
  "monitors": [
    {
      "name": "build-watcher",
      "command": "node watch-builds.js"
    }
  ]
}
```

ビルド結果の監視、テストの自動実行チェック、外部サービスのヘルスチェックなど、裏方に徹するタイプのプラグインが作りやすくなります。プラグインの可能性がまた一つ広がった感じですね。

## ストリーム・ネットワーク周りの安定性が改善

対象: Claude Code ユーザー全般。特にネットワーク環境が不安定な方。

正直これが一番「日々の体験」に効くかもしれません。

### API ストリームのハング対策

これまで、APIからのストリーミングレスポンスが途中で止まると Claude Code はひたすら待ち続けるだけでした。ネットワークが不安定な環境だと、数分間ぼーっとスピナーを眺める羽目になることも。

今回、**5分間データが来なければストリームを打ち切り、ノンストリーミング方式でリトライ** する仕組みが入りました。完全に止まってしまうケースが大幅に減るはずです。

### ネットワークエラー表示の改善

接続エラー時に無言のスピナーが回り続けていた問題も修正されました。エラーが起きたらすぐにリトライメッセージが表示されます。「動いてるのか止まってるのかわからない」というストレスがなくなります。

どちらも「なんで今まで放置されてたんだ」系の改善ですが、入ってくれて素直にうれしいです。

## まとめ

v2.1.105 は PreCompact フックとプラグインモニターという2つの拡張ポイントが加わり、カスタマイズの幅がまた広がりました。ユーザー目線では、ストリームのハング対策とネットワークエラー表示の改善が地味に大きいです。24件のバグ修正も含めて、全体的に安定性を底上げするリリースになっています。
