Figma just acquired the team behind Bud, an AI agent platform that could browse the web, write code, and automate tasks. Figma is shutting down Bud's standalone product by July 18.

This is not a design tool acquisition. This is Figma buying agent infrastructure.

Figma already integrated Claude Code and Codex. They released Figma Make for building web apps. Now they own an agent platform.

The pattern: every major software company is acquiring agent capability. Genesys bought Pinkfish for workflow orchestration. Duck Creek bought Send for agentic underwriting. Backbase bought Kasisto for agentic banking.

The agent infrastructure acquisition wave has started. The companies already building are the targets.
