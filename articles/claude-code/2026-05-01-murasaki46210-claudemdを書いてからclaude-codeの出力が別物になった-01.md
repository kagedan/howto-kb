CLAUDE.mdを書いてから、Claude Codeの出力が別物になった。

「毎回同じ前置き説明をしなくていい」状態になった。
非エンジニアの自分が使ってるテンプレートを公開する。

書き方はこの4つだけ↓

① 自分のことを書く
「非エンジニアです。コードの説明は初心者向けに」

② よく頼む作業を書く
「文章整理・Excel自動化・メール下書きが多い」

③ やってほしくないことを書く
「承認なしにファイルを削除しないで。変更前は必ず確認して」

④ 返し方の好みを書く
「日本語で。手順は省略しないで。コードは先に何をするか説明して」

このファイルを1回作って置いておくだけ。
Claude Codeが毎回「自分向け」に動いてくれるようになった。

10分かからないから、まず書いてみて。
