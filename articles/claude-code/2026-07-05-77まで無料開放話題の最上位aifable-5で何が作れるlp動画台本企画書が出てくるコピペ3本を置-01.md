この記事は解説で終わりません。**貼るだけでLP・動画台本・商品企画書が丸ごと出てくるコピペを3本**置いておきます。締切は日本時間で**7月8日の15:59**。今日貼るのが一番得です。

## 30秒だけ状況整理（何が起きてる？）

![](https://assets.st-note.com/img/1783197916-fHYbK6C3BFz89qlOG5EDI72U.png?width=1200)

Fable 5は、Anthropicの新世代「Claude 5」ファミリー最初のモデル。いつものClaude（Opus）の**さらに上位クラス**です。6月9日に登場→約19日間の提供停止を経て、7月1日に復活しました。

その復活記念が今回の開放です。

・対象：**Pro／Max／Team／一部Enterpriseプラン**（無料プランは対象外）

・量：**週の利用上限の50%まで追加課金なし**

・期限：**7/7（太平洋時間）まで。7/8以降は別途クレジット購入制**

ひと言でいうと「**いつもの料金のまま、期間限定で最上位の頭脳に格上げできる**」週です。

## 【本題】コピペで作れるもの3つ

![](https://assets.st-note.com/img/1783197960-se0iQJwd2VW7CgnzD1TUBxX3.png?width=1200)

Fable 5の強みは「**時間のかかる一式仕事を、途中で崩れずに丸ごとやり切る**」こと。つまり、部品ではなく**完成品を一発で**頼むのが正解です。

そのまま貼れる形で3本置きます。【 】の中だけ自分のものに書き換えてください。

## ①LPを丸ごと作らせる

```
あなたは、売れるLP（販売ページ）を数多く作ってきた制作ディレクターです。
時間をかけて深く考え、次の商品のLPを「一式」で作ってください。

【商品】（例：個人事業主向けのAI活用テンプレート集）
【買う人】（例：AIに興味はあるが何から始めるか分からない30〜40代）

出してほしいもの（全部）：
1. LPの構成案（セクション順と、それぞれの役割）
2. 各セクションの見出しコピーと本文（そのまま使える文章で）
3. 買う人が「断る理由」を3つ挙げ、それを消す一文をLP内に配置
4. 配色・写真・ボタン文言のデザイン指示
5. 最後に、このLPの一番弱い箇所と直し方

途中で手を抜かず、1〜5を順番に、完成形で出してください。
```

## ②60秒ショート動画を丸ごと作らせる

```
あなたは、ショート動画で商品を伸ばしてきた放送作家です。
時間をかけて深く考え、次のテーマの60秒動画を「一式」で作ってください。

【テーマ】（例：Fable 5が7/7まで無料という話題を、初心者向けに解説）
【見る人】（例：AIニュースは知ってるが自分ごとにできていない会社員）

出してほしいもの（全部）：
1. 冒頭2秒のフック（3案。ギャップ型・質問型・断言型で）
2. 60秒の台本（秒数割りつき）
3. 各カットの絵コンテ指示（何を映すか・テロップ文言）
4. ナレーション原稿（話し言葉で）
5. 概要欄の文章とハッシュタグ

1〜5を順番に、そのまま撮影に入れる完成度で出してください。
```

## ③商品企画書を丸ごと作らせる（穴の指摘つき）

```
あなたは、量販店に商品を通してきた商品企画のプロです。
時間をかけて深く考え、次のアイデアを「1枚企画書」に仕上げてください。

【アイデア】（例：忙しい人向けの朝10分AI習慣サービス）

出してほしいもの（全部）：
1. 誰の・どの瞬間の不便を消すか（1行）
2. 今の代用品と、その不満
3. この企画の解決（before→after）
4. 価格案と、その根拠
5. この企画の「穴」を3つ、遠慮なく指摘。それぞれの塞ぎ方も

甘い評価はいりません。通る企画にするために、5の指摘を一番丁寧に。
```

**3本に共通するコツ**は「時間をかけて深く」「一式で」「完成形で」と頼むこと。Fable 5は考える体力が別格なので、部品の外注ではなく**丸ごとの納品**を頼むほど差が出ます。

## 正直な注意（誇大なしで）

・50%枠は**週上限の内数**です。軽い質問に使うと枠が溶けます。上の3本のような「重い一式仕事」だけに使うのが得です

・返事は遅めです（常に深く考えるモードのため）。急ぎの軽作業はいつものモデルで

・7/8以降の継続利用は有料クレジット制（API価格は入力$10／出力$50・100万トークンあたり）。まず無料期間中に、自分の仕事で効くか試すのが正解です
