### 「楽天ROOMは、毎日投稿した方がいい」

そう聞いても、毎日商品を探して、紹介文を考えて、投稿するのは簡単ではありません。

少なくとも私は、これを手作業だけで続けられる自信がありませんでした。  
というか、絶対にやりたくないとまで思っていました。

そこで、普段から本業の業務改善アプリ制作に使っていたAIコーディングツール「Claude Code」（Anthropic社が提供するAIコーディングツール。指示を伝えると、プログラムの作成や実行を代わりにやってくれる）に、楽天ROOMへの投稿作業を任せられないか試してみました。

その結果、Claude Codeと対話しながら作成した指示書「SKILL.md」を起点に、楽天市場での商品検索から楽天ROOMへの投稿、投稿履歴の保存までを自動で進める仕組みが動くようになりました。

2026年7月10日時点で、運用開始から2日、投稿数は5件です。

実績としては、まだ小さな数字です。収益を語れる段階でもありません。

それでも、パソコンとブラウザが使える状態であれば、毎回「投稿して」と指示しなくても、設定したスケジュールに沿って投稿処理が始まるところまで進められたことは、私にとって大きな一歩でした。

なお、この記事で紹介する楽天ROOMのアカウントは「Fumi Lab（アカウントID：room\_498c6c0843）」です。冒頭でお伝えした通り、このアカウントの投稿にはアフィリエイトリンクが含まれています。

## 自動化の知識があったわけではありません

最初から、必要な仕組みや作り方が分かっていたわけではありません。  
私がClaude Codeに伝えたのは、最初の段階では「楽天ROOMへの投稿を自動化したい」という希望だけでした。  
そこからClaude Codeに聞かれた質問へ、ひとつずつ答えていきました。

こうした内容を対話しながら整理した結果、投稿ルールをまとめたSKILL.mdが形になっていきました。  
つまり、自分で完璧な指示書を書いてから始めたのではありません。  
「こういうことをしたい」と伝え、AIからの質問に答えながら、必要な指示を一緒に作っていきました。

## 最初から順調だったわけでもありません

一度仕組みを作れば、すぐに安定して動くと思っていました。  
しかし実際には、ログイン状態を確認するつもりが、別の管理画面「my Rakuten」へ進んでしまい、本来の商品検索画面に戻れなくなったことがあります。その回の投稿は完了しませんでした。  
最初に投稿した商品も、電源タップ、トースター、アウトドア収納ボックスと、ジャンルがばらばらでした。  
失敗するたびに、私はその内容をSKILL.mdへ追加しました。

失敗を「その場限りの修正」で終わらせず、次回からClaude Codeが同じ失敗をしないためのルールへ変えていったのです。  
この作業を繰り返すうちに、SKILL.mdは単なる指示書ではなく、私自身の運用ノウハウを蓄積するファイルになっていきました。

## この記事でお伝えすること

ここから先では、私が実際に作った仕組みと、運用開始後に修正してきた内容を、体験ベースで詳しく紹介します。

購入者特典として、私が実際に使用しているSKILL.mdをベースにしたテンプレートも添付しています。  
テンプレートは、ご自身の環境に合わせて設定項目を変更できる形にしています。ただし、利用にはClaude Codeを使える環境、ブラウザ拡張機能のセットアップ、楽天ROOMへの事前ログインが必要です。利用環境や画面表示によっては、手順の調整や動作確認も必要になります。  
この記事は、「この方法なら必ず稼げる」とお伝えするものではありません。開始2日・投稿5件の段階なので、同じ成果が得られる保証もありません。  
それでも、AIを質問相手として使うだけでなく、決めた手順を実際に動かすところまで任せてみたい方には、ひとつの具体例として参考にしていただけると思います。
