Claude Codeで毎日の投稿スケジュール自動化してみた。やってることはシンプル。

まずnoteで書いた記事のURLと「この投稿、FFログとコラボ要素入れて欲しい」みたいな指示を入力→Claude Codeが自動で既存のテンプレートを読み込んで→文言を組み替えて→X用にフォーマット整えて→スプレッドシートに出力。

ここまで1分。

正直「コードなんて書いてない」レベル。やってることは「prompt書いてClaude Codeに『こういうフォーマットで出力して』って指示する」だけ。

最初つまずいたのが「どうやってスプレッドシートに直接書き込ませるんだよ」ってとこ。APIがどうたら、認証がどうたらみたいなのがあるんだけど、俺はそこ完全にスキップした。

代わりに「出力内容をコピペしやすいテキストブロックにまとめる」→「手作業でシートに貼る」の2秒作業にした。完璧さよりスピード優先。

これで何が変わったかって、毎日「今日何書こう」「どうフォーマットしよう」の意思決定がゼロになった。朝起きてFF14のトレンド見て「あ、これネタにできるな」って思ったら即座に投稿できる。

副業って「技術」じゃなくて「決定疲労をどう減らすか」だと気付かされた。
