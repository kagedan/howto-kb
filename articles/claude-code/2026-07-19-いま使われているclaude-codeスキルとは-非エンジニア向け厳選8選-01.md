Claude Codeを使っていると、こんな指示が出てくる。

```
/skill-creator を使ってカスタムスキルを作ります
deep-research で競合調査を自動化しました
code-review でセキュリティをチェックしています
```

教科書シリーズでも、スキルが何度か登場した。でも「スキルって結局なに？」「今、実際に人気のものは？」「自分にも使えるの？」という疑問は、残ったままかもしれない。

ここで一度、整理する。スキルはClaude Codeの拡張機能だ。使い方を知れば、毎日の仕事が一気に変わる。

## スキルって何？——AIに「専門技能」を与える仕組み

スキルをひとことで言うと、**Claudeに特定の分野での「専門知識と実行能力」を追加する機能**だ。

スマートフォンを想像してほしい。買ったばかりのスマホは、基本機能しかない。でもアプリをインストールすると、カメラ、銀行、GPS、メモ帳など、いろんなことができるようになる。スキルはこれと同じ。Claudeは標準で「考えて答える」ができるが、スキルを入れると「ビジネス分析ができるClaude」「デザインができるClaude」「セキュリティを監査できるClaude」に変身する。

本来なら、こういう複雑な作業は人間がやる。でも専門スキルがClaudeに「搭載」されていれば、指示を出すだけで自動実行してくれる。

## 非エンジニア向け、いま定番の8つのスキル

実際のところ、スキルは数百種類ある。でも毎日の仕事で活躍するのは、意外と限られている。営業・企画・マーケティング・事務に携わる人が、今、本当に重宝しているものを8つ厳選した。

### 1. pdf / docx / pptx / xlsx——書類を秒で作る

企画書、提案書、請求書、プレゼン資料。「テンプレートから手作りするのに2時間」という仕事は珍しくない。このスキルを使うと、**内容を伝えるだけで、プロ品質の書類が自動生成される**。

営業が新規顧客に送る提案資料も、経理が月次で出す請求書も、全部Claudeが作ってくれる。テンプレートいじりの時間は二度と戻ってこない。

### 2. deep-research——競合調査・市場分析が5分で終わる

新規事業を始めるなら、競合や市場を調べるのは必須。でもネットから情報を集めてまとめるのに丸一日かかることもある。このスキルは、**複数のウェブサイトから自動で情報を拾って、構造化された分析レポートに変換する**。

「同業他社5社の料金・機能・顧客層を比較」「この業界の成長率は？」「顧客のペインは何か」——こういう問い一つで、5分後には答えがレポートになっている。

### 3. cold-email / email-sequence——営業メール文面が何パターンも生成される

新規営業の一番の面倒は「メール文を書く」ことだ。ターゲット企業の情報を入力すると、このスキルは**営業メールの文面を複数パターン自動生成**してくれる。Aさんには提案A、Bさんには提案Bというふうに文面を自動で変えることもできる。

パターンを選ぶだけで終わる。返信率も上がる傾向にある。

### 4. claude-seo / seo-audit——ウェブサイトの「検索流入」を診断・改善

自社サイトを作ったけど、Google検索に引っかからない。そんな悩みは多い。このスキルは**ウェブサイト全体をスキャンして「どこを直すと検索に強くなるか」を自動診断**する。

タイトルタグが抜けてる、メタ説明文が短すぎる、見出し構造が噛み合ってない——こういう「検索エンジンが嫌う書き方」を全部見つけて、修正案まで提示する。

### 5. copywriting / ad-creative——広告文・LPの文面を複数案同時生成

「このバナー広告、いい文面が思いつかない」「ランディングページのキャッチコピーを3案欲しい」——こういうリクエストに、このスキルは**複数の文面パターンを一度に生成**する。

プロのコピーライターが書いたような文が、人力ゼロで出てくる。A/Bテスト用の素材も、その場で10案作ることもできる。

### 6. code-review / security-review——ソースコード・API設計の品質をチェック

開発チームがコードを書いたあと、本番に上げる前に。このスキルは**コードの問題点・セキュリティ脆弱性・設計ミスを自動検査**する。バグが本番に行く前に、ここで止められる。

エンジニア以外が使うことは少ないが、チーム内に「こいつに見てもらえば安心」という検証段階が用意できるのは、心強い。

### 7. thinking-first-principles / thinking-jobs-to-be-done——ビジネス課題を根本から分析する

新規事業、経営戦略、顧客分析。こういう「つかみどころのない問題」に直面したとき、このスキルは**課題を根本原因から分析**する。顧客のほんとうのニーズ、競争の本質、隠れた機会を透視する。

戦略立案の時間が10分の1になることもある。

### 8. skill-creator / mcp-builder——自分専用のスキルを自動化する

最後は、やや上級向けだ。毎日やってる単純作業（請求書集計、メール返信パターン、顧客データクリーニング）を「自分専用スキル」として自動化できる。一度作れば、以降は1クリックで実行。

繰り返す作業を、手動から自動に変えるための基盤だ。

## よく聞く用語ミニ辞典

**スキルマーケットプレイス**：Claude Code内で「使えるスキル一覧」が並んでいる場所。アプリストアと同じ。

**npx skills add**：スキルをインストールするコマンド（ターミナルで打つ）。でも非エンジニアなら、マーケットプレイスからGUIでポチるのが簡単。

**カスタムスキル**：自分や会社専用に作った、世に出ていないスキル。skill-creatorで自動化する。

## 入れ方——ほぼ「ポチるだけ」

「難しそう」と思うかもしれないが、実際は簡単だ。

Claude CodeのUIに「スキルマーケットプレイス」というボタンがある。そこをクリックして、欲しいスキルを見つけて「インストール」を押すだけ。アプリをスマホに入れるのと同じ手順だ。

コマンドを打ちたい人は`npx skills add 〜`でもできるが、非エンジニアなら最初はマーケットプレイスのGUIが直感的。

## まとめ

| スキル名 | 何ができるか | こんな人に |
| --- | --- | --- |
| pdf / docx / pptx / xlsx | 仕事の書類を自動生成 | 営業・企画・事務 |
| deep-research | 競合調査・市場分析を自動化 | 事業企画・営業企画 |
| cold-email / email-sequence | 営業メール文面を複数案生成 | 営業・BDR |
| claude-seo / seo-audit | ウェブサイトのSEO診断・改善 | マーケ・Webサイト管理者 |
| copywriting / ad-creative | 広告文・LP文面を複数案生成 | マーケティング・企画 |
| code-review / security-review | コード品質・セキュリティを自動検査 | 開発チーム・CTO |
| thinking-first-principles | 戦略課題を根本から分析 | 経営層・事業企画 |
| skill-creator / mcp-builder | 単純作業を自動化 | チーム・個人の効率化担当 |

ひとことで言えば——**スキルはClaudeの「専門技能パック」だ。入れるだけで、毎日のルーチン仕事が一気に自動化される**。アプリを足すのと同じ感覚で、試してみる価値は十分ある。
