「世界で一番使われてるAI、GPTでもClaudeでもないって知ってました？」

7月に入ってから、世界の開発者が実際に使った量のランキング（OpenRouter）で、Tencentの「HY3」というモデルが1位になりました。1週間で6.13兆トークンです。

そしてこのHY3、いま7/21までの期間限定で無料開放中です。残り3日。設定はコピペで終わります。

この記事では、そのHY3を7/21までタダで使う手順を、コピペ3本で配ります。

## いま何が起きているか（30秒で現在地）

事実だけ並べます。

・HY3は7/6公開のオープンモデル。総パラメータ295B、実際に動くのは21B（MoE方式）、記憶できる長さは262K  
・OpenRouter経由でモデル名「tencent/hy3:free」を指定すると、7/21まで無料  
・無料期間が終わると、100万トークンあたり入力$0.14／出力$0.58の有料ルートに戻る

有料でも安い部類ですが、今週はゼロ円です。

![](https://assets.st-note.com/img/1784147362-0GrQ3YKU9vIX25PMk1aJWV8N.png?width=1200)

世界で一番使われてるのに、今週は0円。

先に注意も書いておきます。無料ルートには回数制限があります。1分20回・1日50回まで（OpenRouterに$10以上の入金歴があれば1日1000回に拡大）。お試しには十分な枠です。

もう1つ。「7/21まで」の締め切り時刻は時差で前後する可能性があります。ギリギリを攻めず、今週前半に試すのが安全です。

## コピペ1：まずブラウザで30秒さわる

コードは一切いりません。

1. [openrouter.ai](http://openrouter.ai) で無料アカウントを作る
2. チャット画面でモデルに「Hy3 (free)」を選ぶ
3. 下のプロンプトを貼る

コピペ↓

```
あなたの実力を測ります。次の3つを順にやってください。

①下の【私の仕事】を読み、内容を確認する質問を3つだけする
②私が答えたら、その仕事を30分短くする具体案を1つ出す
③その案の実行手順を5ステップで書く

【私の仕事】
（ここに、あなたが毎日やっている作業を2〜3行で書く）
```

これを貼ると、HY3が「自分の仕事に効くか」を5分で判定できます。

![](https://assets.st-note.com/img/1784147382-F5OSyJap0ItqGhrcZNWXdw96.png?width=1200)

コード不要。ブラウザだけで30秒で試せる。

## コピペ2：Pythonなら10行で呼べる

すでにChatGPTのAPIを使っている人は、接続先（base\_url）を1行変えるだけです。書き換えるのはたった2箇所。

コピペ↓

```
from openai import OpenAI

# OpenRouterはOpenAI互換。接続先を1行変えるだけで乗り換えられる
client = OpenAI(
    base_url="https://openrouter.ai/api/v1",  # ←この1行がポイント
    api_key="ここにOpenRouterのAPIキー",
)

res = client.chat.completions.create(
    model="tencent/hy3:free",  # ←7/21まで無料のルート
    messages=[{"role": "user", "content": "自己紹介と、得意な仕事を3つ教えて"}],
)
print(res.choices[0].message.content)
```

コードの他の部分はOpenAIのときと同じ。今あるプログラムがそのまま流用できます。

![](https://assets.st-note.com/img/1784147403-3AgKti9rZwEsmUTlJ4YbyNvh.png?width=1200)

変えるのは base\_url の1行だけ。

## コピペ3：Codex CLIを丸ごと無料AIで動かす

コーディングの相棒「Codex CLI」も、設定ファイルにこれを貼るだけでHY3で動きます。

`~/.codex/config.toml` に追記↓

```
model = "tencent/hy3:free"
model_provider = "openrouter"

[model_providers.openrouter]
name = "OpenRouter"
base_url = "https://openrouter.ai/api/v1"
env_key = "OPENROUTER_API_KEY"
```

ターミナルでAPIキーを設定（.zshrc等に追記）↓

```
export OPENROUTER_API_KEY="あなたのOpenRouterキー"
```

これで、月額課金なしのコーディング相棒が1週間使い放題になります。

ただし無料ルートの回数制限（1日50回）があるので、ヘビーに使う日は詰まります。「夜に1タスク任せてみる」くらいの試し方がちょうどいいです。

![](https://assets.st-note.com/img/1784147424-h942ws6kZ3Pya1e0ntUiEQWG.png?width=1200)

無料枠の条件はこれだけ。ギリギリ狙いは禁物。

## 本質：これは「乗り換え」ではなく「評価の練習」です

私は事業部長時代、新しい仕入先が来たら必ず同じテストを出していました。定番の案件を1つ渡して、今の取引先と見比べる。それだけで、その会社の実力は大体わかります。

AIも同じです。

無料開放は「乗り換えのチャンス」というより、自分の仕事でAIを評価する練習台です。コピペ1のプロンプトに自分の仕事を貼って、いつも使っているAIと答えを見比べてください。

「どっちが上か」を自分の目で判定できた人から、AI選びで損をしなくなります。締め切りがあるから、今日やる。それだけです。
