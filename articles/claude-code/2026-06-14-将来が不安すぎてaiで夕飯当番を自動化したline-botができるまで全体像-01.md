## はじめに

将来が不安で仕方ない、どこにでもいる有象無象のサラリーマンです。

「AIに触っとかないとマジでやばい」という焦りに耐えきれず、とりあえず手を動かすことにした。題材は、大層なものではなく、家の日常タスク。今回はその第一弾として作った「夕飯当番LINE Bot」の話を、これから連載で残していく。

この記事はその入口（ハブ）。**全体像だけ**をまとめて、詳しい手順や失敗談は後続記事に送る地図として書く。

## 何を作ったか

一言でいうと、**毎週の夕飯当番をLINEで回す仕組み**。

我が家は2人暮らしで、月曜日の夕食当番は私。その「今週は何を作るか」を決めて、当日レシピを用意するまでを、LINEのグループトーク上で自動化した。

## ビフォー / アフター

ここが一番伝えたいところ。

### ビフォー

* 日曜に口頭で「今週なに食べたい？」と聞いていた
* だいたい忘れる。忘れたまま月曜になって、慌ててLINEで聞く
* いざ作る段になると、毎回YouTubeなどでメニューと作り方を調べ直す
* 「決める」「思い出す」「調べる」が毎週バラバラに発生して、地味に脳のメモリを食っていた

### アフター

* **日曜の昼、LINEグループに今週のメニュー候補が自動で届く**
* 家族が番号かメニュー名を送るだけで選択完了。最後に選ばれたものが確定
* **月曜の昼、選んだメニューのレシピ（材料・手順）が自動で流れてくる**
* 未選択なら月曜の朝にリマインドが飛ぶ

「決める・思い出す・調べる」が、**LINEを眺めて選ぶだけ**に圧縮された。何より「聞き忘れ」と「毎回ゼロから調べる」がまるごと消えたのが大きい。

## 全体の構成（ざっくり）

最終的にこういう構成に落ち着いた。

```
GAS（定期トリガー / JST）
   ↓ 決まった時刻にエンドポイントを叩く
Render（Flask サーバー）
   ↓ 選択状態を読み書き
Googleスプレッドシート（保存先）
   ↓
LINEグループ（Messaging API）
```

* **LINE Messaging API**：家族とのやり取りの窓口
* **Render（Flask / Python）**：受信・返信・レシピ配信を担うサーバー
* **Googleスプレッドシート**：「今週選ばれているメニュー」の保存先
* **GAS（Google Apps Script）**：日曜・月曜の決まった時刻に通知を起動するトリガー
* レシピは1メニュー1ファイルの `.txt`（Markdown）で管理。ファイルを足すだけで選択肢が増える

そして、これを**設計から実装までAIに伴走してもらった**。要件定義や構成の相談はClaudeのチャットで、実際のコード生成はClaude Codeで進めた。プログラミング経験が少しある程度の自分でも、最後まで完走できた。

ちなみにこの構成、**最初からこうだったわけではない**。最初は GitHub Actions（スケジュール）と Supabase（保存先）で組んでいた。それが両方ともあちこちで壊れて、つど別のサービスに逃がした結果、いまの GAS ＋ スプレッドシート構成に着地した。その「壊れて乗り換えた記録」が、このシリーズの本体でもある。

## 結果と所感

完成して一番うれしかったのは、「毎週地味に発生していた小さな面倒」が、意識から消えたこと。自動化の効果って、浮いた時間そのものより、**そのタスクを覚えておかなくてよくなる**ことなのかもしれない。

技術的にすごいものを作ったわけではない。むしろ、無料サービスのクセに何度も足をすくわれて、回り道ばかりだった。それでも「不安で立ち止まる」より「不安だけど一個作ってみる」の方が、精神衛生上ずっとよかった。動くものが手元に残るので。

## このシリーズの地図

ここから先は、テーマごとに記事を分けて書いていく。

* **②：何から始めればいいか** — 要件定義の進め方、Claude / Claude Code の使い方、環境構築まで
* **③：失敗シリーズ（全3本）** — 作る過程でぶつかった失敗を1つずつ
  + 返信が一部だけ届かない（reply tokenの罠）
  + 定期通知が10時間遅れた（GitHub Actionsのcron遅延）
  + DBが勝手に止まってサーバーが起動しない（Supabase一時停止）
* **④：振り返り** — 失敗しない構成を最初からAIに提案させるには／指示と要件定義の工夫／AIの限界

## おわりに

不安は消えないが、消えないなりに、手だけは動かしていく。次は「何から始めればいいか」を、要件定義の話から書く。

同じように将来が不安な人へ。すごいものじゃなくていいので、一緒に何か一個、動かしてみませんか。
