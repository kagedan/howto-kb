はじめに
Playwrightを用いて、複数ブラウザでのレスポンシブ対応の確認を自動化したいと考えました。
複数ブラウザ・複数サイズでのレイアウト確認を手動で行うのは手間がかかります。今回はそういった課題をClaude Code + Playwright MCPを利用して解決していきます。
最終成果物は複数ブラウザで撮影をするスクリプトになります。

用語や、環境構築方法についてはこちらの記事をご参照ください。
