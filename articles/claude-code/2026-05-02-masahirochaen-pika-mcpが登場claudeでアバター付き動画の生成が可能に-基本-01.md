Pika MCPが登場。Claudeでアバター付き動画の生成が可能に。

基本Claude Codeでの動画生成はRemotionだったが、アバター付きならPikaが良さそう。

・Pika MCPで動画/画像/音声を生成
・解説動画、Podcast、UGC広告に対応
・顔/声/性格付きAIアバターも作成
・URLやGitHubから動画化も可能

↓詳細 https://t.co/1uNGFGdipE

🎬 Claudeが“動画を作るAI”に近づいた
https://t.co/nq8DxE8Fy6

Pika MCPは、Claudeに動画生成モデルが直接入る機能ではなく、ClaudeからPikaの制作機能を呼び出せるようにする連携。

できることはかなり広い。

・解説動画
・Podcast風の対談動画
・UGC広告
・商品紹介動画
・フォトリアル画像
・音声/音楽/ナレーション
・AIアバター的なキャラクター制作

特に面白いのは、Pika Agentに顔・名前・性格・記憶・声・見た目を持たせられる点。

URL、GitHubリポジトリ、短い企画メモを渡すだけで、Claudeが内容を理解し、Pika MCP経由で動画や広告素材に変換できる。

つまり、AIが文章で答えるだけの時代から、顔を持ち、話し、出演し、コンテンツまで作る時代に進んでいる。

これは単なる動画生成AIではなく、Claudeを“制作チーム化”するMCP。
