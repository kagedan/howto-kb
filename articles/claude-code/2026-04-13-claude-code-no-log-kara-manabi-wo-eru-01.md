Claude Codeの実行ログを分析することで動作理解やトラブルシュートに活かす方法を解説するスライド。ログは~/.claude/projects/にJSONL形式で保存され、デフォルト30日間保持される。サブエージェントによるコンテキスト汚染防止、Bashコマンドの洗練された使い方、曖昧な指示の自動具体化など、ログから読み取れる興味深い動作パターンを紹介。意図通りに動かない場合のログ診断が有効であると結論し、AIから人間が学ぶ姿勢の重要性を将棋のアナロジーで説明している。
