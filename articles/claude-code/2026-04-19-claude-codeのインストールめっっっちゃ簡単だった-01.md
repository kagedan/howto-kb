## インストール手順

### Step 1: 公式クイックスタートを開く

まずは公式のクイックスタートページを開きます。

![公式クイックスタートページ](画像/スクリーンショット 2026-04-16 202603.png)

### Step 2: PowerShellでインストールコマンドを実行

Windows PowerShellを開いて、以下のコマンドをコピペして実行するだけです！

```
irm https://claude.ai/install.ps1 | iex
```

[![スクリーンショット 2026-04-16 203128.png](https://qiita-user-contents.imgix.net/https%3A%2F%2Fqiita-image-store.s3.ap-northeast-1.amazonaws.com%2F0%2F2774344%2F367f8bef-b18f-4cb4-9207-dd9d13bb28c6.png?ixlib=rb-4.0.0&auto=format&gif-q=60&q=75&s=da65943b8b087e125f17e4a1535f3c35)](https://qiita-user-contents.imgix.net/https%3A%2F%2Fqiita-image-store.s3.ap-northeast-1.amazonaws.com%2F0%2F2774344%2F367f8bef-b18f-4cb4-9207-dd9d13bb28c6.png?ixlib=rb-4.0.0&auto=format&gif-q=60&q=75&s=da65943b8b087e125f17e4a1535f3c35)

1分ほど待つとインストール完了！

インストール完了後に「システム環境変数にPATHを追加してください」という注意書きが表示されます。これを無視すると次のステップでエラーになるので、必ず対応しましょう！（後述）

### Step 3: システム環境変数にPATHを追加する

注意書きを無視してそのまま `claude` を実行するとこうなります。

注意書きの通り、システム環境変数にPATHを追加します。

### Step 5: アカウント作成＆認証

初回起動時はアカウント作成が必要です。

[![スクリーンショット 2026-04-16 204419.png](https://qiita-user-contents.imgix.net/https%3A%2F%2Fqiita-image-store.s3.ap-northeast-1.amazonaws.com%2F0%2F2774344%2F5095228a-8012-47fe-b9cf-14c06dea55ed.png?ixlib=rb-4.0.0&auto=format&gif-q=60&q=75&s=20da5587f18c4a7ac64232da0d139d58)](https://qiita-user-contents.imgix.net/https%3A%2F%2Fqiita-image-store.s3.ap-northeast-1.amazonaws.com%2F0%2F2774344%2F5095228a-8012-47fe-b9cf-14c06dea55ed.png?ixlib=rb-4.0.0&auto=format&gif-q=60&q=75&s=20da5587f18c4a7ac64232da0d139d58)

* ブラウザが開いてアカウント作成画面に
* 登録自体はすぐ終わりますが、認証トークンの入力が必要です

**トークン入力でハマりポイント！**  
認証後に表示されるトークンをPowerShellの画面に貼り付けるのですが、右クリックで貼り付けてもうまくいかない場合があります。PowerShellウィンドウ上で右クリックするか、タイトルバーを右クリックして「編集 → 貼り付け」を試してみてください。
