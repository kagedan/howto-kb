## Claude Codeが気になっているそこのあなたへ

「Claude Code、めっちゃすごい！」ってのはよく聞くんだけど、
でもインストールがめんどくさかったりするんじゃない？と思っているそこのあなた！！

その気持ち、めっちゃわかるぞ！！

実は僕も「やってみたいな〜」と思ってから早半年が経過していた。
ずっと気になりつつも、なんとなく腰が重くて手を出せずにいたんだよね。

でもある日、友人に「いいからやってみろよ」と言われ、
**いざインストールしてみたらめちゃくちゃ簡単だった。**
しかも **10分くらいで終わった。**

「クッソ簡単にインストールできるやんけ！！」
「もっと早くやればよかった！！！」

そんな気持ちになったので、同じように迷っている人に向けてインストール手順を共有します！
