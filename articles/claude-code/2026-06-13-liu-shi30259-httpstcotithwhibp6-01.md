# 「Claudeに課金してるのに、結局ChatGPTと同じ使い方してる…」

![](https://pbs.twimg.com/media/HKrTQbdacAAebml.jpg)

**Chat・Cowork・Claude Codeという「3つ方法」を知らないだけ**なんですClaudeってすごいけどなんかよく分からないってことあると思います

![](https://pbs.twimg.com/media/HKrUqM-aQAA9bKx.jpg)

毎回「占い系の台本を作って」「仮想通貨ショート動画の台本を」と

ゼロから指示し直している

Claude Codeって名前は聞いたことあるけど、　　　　　　　　　　　　　　　　　エンジニアじゃないから関係ないと思っていた

同じAIを使ってるはずなのに　　　　　　　　　　　　　　　　　　　　　　　他のクリエイターの動画の方が明らかにクオリティが高い

「正しい使い方」が分からないまま、もう3ヶ月が経ってしまった

この記事を読み終わる頃には

**「自分にはClaudeが合っているか」が明確になります**

Claudeには3つの使い方があります

- Chat→webサイト
- Cowork→アプリ
- Claudecode→Vscodeなどのプログラミングツール
と使える場所が違うってことです

![](https://pbs.twimg.com/media/HKrVUbObYAAeXjA.jpg)

一番性能がいいのがClaude Code

ただし慣れるのに一番時間がかかります

だからいきなりClaude Codeに飛びつく必要はありません

じゃあ、どうやって作っていくのか解説していきます

![](https://pbs.twimg.com/media/HKrUqM-aQAA9bKx.jpg)

**Claudeのチャットバージョン**

ChatはChatGPTの延長で使い始められます

ただし「チャットで質問するだけ」では　　　　　　　　　　　　　　　　　　　　Claudeの能力の20%しか引き出せていません

**ジャンルごとの前提を一度だけ教える**

毎回「占い系の台本を作って　神秘的な雰囲気で　冒頭3秒でバズワードを使って…」

とコピペしていませんか？

それでは時間の無駄になってしまいます

**Projects機能**を使えば

ジャンルごとの　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　前提情報を一度だけ設定すれば毎回コピペしなくても使い回しができます

![](https://pbs.twimg.com/media/HKrTQbdacAAebml.jpg)

具体例として

こんな使い方があります

- **占い系プロジェクト**：「視聴者は30〜50代女性・神秘的・共感系のトーン・冒頭は必ず煽り系バズワードから入る。400文字以内で起承転結」
- **仮想通貨系プロジェクト**：「視聴者は20〜40代男性・データ重視・興奮系トーン・専門用語は使いすぎない・常識を覆す切り口で始める」
設定してしまえば

翌日からの台本作成を　　　　　　　　　　　　　　　　　　　　　　　　　　テンプレート化できるのでそのまま使いまわせるんですよね

![](https://pbs.twimg.com/media/HKrVUbObYAAeXjA.jpg)

**自分専用のClaudeを作成**

**Memory機能**で

Claudeに「あなたの作風」を覚えさせることができます

「台本は400文字以内で」

「起承転結を必ず意識して」

「冒頭3秒でバズワードを入れて」

毎回打っていたこの手の指示が

一度教えたら不要になります

さらに**Styles機能**で応答のトーンをプリセット保存できます

- 占い系：神秘的で共感を呼ぶトーン
- 仮想通貨系：興奮感があってデータ重視のトーン
- 健康系：親しみやすくて科学的根拠を含むトーン
切り替えるだけで文体が変わります

![](https://pbs.twimg.com/media/HKrVj2-aMAAMLQl.jpg)

**競合の台本を一瞬で分析**

**Deep Research機能**は

東大卒の外資系コンサルタントを1人雇ったようなものです

```
「TikTokで今バズっている占い系AIショート動画の構成パターンをまとめて」
```

と指示するだけで

数十の事例を横断した分析レポートが出てきます

今まで2時間かけていた競合リサーチが数分で終わります

競合の台本を文字起
