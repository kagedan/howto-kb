はじめに
Claude Code に長めのタスクを任せた後、こんな経験はありませんか？

「どうせ時間かかるだろう」と別の作業をしていたら、とっくに終わっていた
ブラウザを見ていたら「確認が必要です」という入力待ちになっていて、数分ロスした
複数の Worktree を並列で動かしているとき、どれが終わったのか把握できない

Claude Code の Hooks 機能を使えば、これらのシチュエーションで macOS の通知センターに通知を飛ばす ことができます。
※ 最初は cmux を使っていましたが、VSCodeで開いた Claude Code の通知はできないのでこの機能を作...
