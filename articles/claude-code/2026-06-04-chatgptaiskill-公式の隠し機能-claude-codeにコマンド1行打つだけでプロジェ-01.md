【公式の隠し機能！？】
Claude Codeにコマンド1行打つだけで、プロジェクトを自動でスキャンして最適な設定を全部提案してくれる公式プラグインがあります。
それは↓ https://t.co/zYPygHIQYW

/plugin install claude-code-setup@claude-plugins-official

コマンドはこれだけです。インストールして
「このプロジェクトの設定を提案して」と伝えるだけ。

自動で提案してくれるのは5つです。
・MCP（外部ツールとの接続方法）
・Skills（再利用できる専門スキル）
・Hooks（自動で動くトリガー）
・Subagents（役割分担できるAIチーム）
・Slash Commands（よく使う作業の1行化）

ファイルは一切変更しません。分析して提案するだけの安全な設計です。
