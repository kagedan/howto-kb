### 1. はじめに：Claude Codeとは何か？

プログラミングの世界に、あなたの思考を形にする強力な「魔法の杖」が届きました。それが **Claude Code** です。

Claude Codeは、Anthropic社が開発した\*\*「エディタに依存しないターミナルベースのAIアシスタント」\*\*です。「ターミナル（黒い画面）」と聞くと難しそうに感じるかもしれませんが、実は「AIと直接チャットしながら、その場でコードを書いてもらう」ための最も効率的な舞台なのです。

このツールを使えば、あなたの開発体験は劇的に変わります。

まるで熟練のエンジニアが隣に座って代筆してくれるような、ワクワクする体験が待っています。まずは、あなたのPCがこの魔法を受け入れる準備ができているか確認しましょう。

### 4. ステップ・バイ・ステップ：各OSごとのインストール手順

ここでは、最も強力な「コマンドツール（CLI）」のインストール手順を解説します。

macOS / Linux / WSL の場合

「ターミナル」アプリを開き、以下のコマンドをコピー＆ペーストしてEnterキーを押してください。

```
curl -fsSL https://claude.ai/install | sh
```

Windows の場合

🛠 Windowsでつまずきやすい「PATH」の設定方法

インストール後に claude と入力して「コマンドが見つかりません」と出る場合は、PCにプログラムの場所を教えてあげる（PATHを通す）必要があります。

1. **[Windowsキー]** を押し、\*\*「環境変数」\*\*と入力して「システム環境変数の編集」を開きます。
2. 画面下の **[環境変数]** ボタンをクリック。
3. 上の「ユーザー環境変数」枠内の **[Path]** を選択して **[編集]** をクリック。
4. **[新規]** を押し、以下を貼り付けます（[ユーザー名]部分は自分の名前に書き換えてください）。 C:\Users\[あなたのユーザー名]\.local\bin
5. すべての画面で **[OK]** を押し、ターミナルを一度閉じてから開き直してください。

インストールが完了したら、次はClaudeに「あなたの身元」を教える認証作業です。

### 5. 認証とログイン：AIとの対話を始めるための鍵

Claude Codeを動かすには、以下のいずれかの有料プランが必要です。

認証のフロー

1. ターミナルで claude と入力して実行します。
2. 「ログイン方法を選んでください」と出るので、矢印キーで **1. Claude account with subscription** を選んでEnter。
3. ブラウザが自動で開きます。ログイン後、\*\*「Authorize（承認）」\*\*ボタンを押してください。
4. ターミナルに戻り、ウェルカムメッセージが表示されれば成功です！

認証が無事に済んだら、次は快適に使うための「魔法の設定」をいくつか行います。

### 6. 成功のための初期設定：これだけはやっておこう

Claude Codeをより便利に、安全にするための設定を行いましょう。設定ファイル（~/.claude/settings.json）を編集することで適用されます。

📋 コピペでOK！推奨設定テンプレート

以下の内容をコピーして、設定ファイルに書き込んでみましょう。

```
{
  "commands": {
    "stopHook": "powershell.exe -c \"(New-Object Media.SoundPlayer 'C:\\Windows\\Media\\chimes.wav').PlaySync()\""
  },
  "permissions": {
    "allow": ["web_search", "fetch_url"],
    "deny": [".env", "*.pem", "config/secrets.yml"]
  }
}
```

(※Windowsの場合の例。macOSなら stopHook に afplay /System/Library/Sounds/Glass.aiff などを使います)

この設定の「ベネフィット」

* **完了通知 (Stop Hook)：** AIが長い作業を終えたら、音で「終わったよ！」と知らせてくれます。画面をずっと眺めている必要はありません。
* **パーミッション設定：** Web検索など安全な操作をするたびに「許可しますか？」と聞かれるのを防ぎ、作業の手を止めません。
* **🔒 セキュリティの鉄則：** deny 項目に .env（パスワード情報）などを指定することで、AIが誤って機密情報を読み取るのを防ぎます。

準備はすべて整いました。それでは、実際にWebサイトを作ってみるという初仕事に挑戦しましょう。

### 7. 実践！最初のタスク：Webサイトを作成してみる

それでは、AIと一緒に「おしゃれな自己紹介サイト」を作ってみましょう。

1. **作業フォルダの作成：** デスクトップに「mysite」という空のフォルダを作ります。
2. **フォルダで起動：** フォルダを右クリックして\*\*「ターミナルで開く」\*\*（Windows/Mac共通）を選びます。これでフォルダ移動（cdコマンド）の手間が省けます。
3. **Claudeを召喚：** 画面に claude と入力してEnter。
4. **魔法の言葉（指示）：** 「モダンで清潔感のある自己紹介サイトを作って。名前は[自分の名前]で。」
5. **対話と実行：** AIが構成案を出し、「index.htmlを作ってもいいですか？」と聞いてきます。矢印キーで **Yes, allow all edits this session** を選ぶと、その後の作業をスムーズに任せられます。
6. **ブラウザで確認：** フォルダ内にできた index.html をダブルクリックして開きましょう。

もし途中で動かなくなっても大丈夫。頼もしい助っ人がいます。

### 8. 困った時の救世主：claude doctor の活用法

何かトラブルが起きたら、慌てずにこのコマンドを打ってください。

```
claude doctor
```

これはAIによる「健康診断」ツールです。ネットワーク、認証、環境設定などを自動でチェックし、「何が原因で、どう直せばいいか」を具体的に教えてくれます。

💡 知っておくと便利なヒント

* **AIが物忘れを始めたら：** 会話が長くなり、AIが混乱してきたら /compact と入力してください。重要な記憶を残したまま、頭の中を整理してくれます。
* **設定が反映されない：** JSONファイル（設定ファイル）の末尾にカンマ（,）が余計に付いていたり、不足していたりしないか確認しましょう。
* **ゼロからやり直したい：** /clear と打てば、今の会話をリセットしてまっさらな状態から始められます。
