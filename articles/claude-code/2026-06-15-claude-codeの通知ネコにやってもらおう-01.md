## Claude Codeから目が離せない問題

Claude Codeに作業を頼んで、別のタブを見ていると  
気づいたら終わってた、ということが続いていました。

逆に、ずっと見張っていても何も起きないことも多い。

そこで最初に「終わったら教えてくれる」[プラグイン](https://github.com/qvtec/claude-result-speak)を作りましたが、可愛げがない。

なので **猫** に来てもらいました🐱  
![cat1](https://static.zenn.studio/user-upload/69bde395ae1e-20260615.png)

## 作ったもの

Claude Codeが何かあると、画面の右下から猫が出てきてお知らせしてくれるプラグインです。

![demo](https://static.zenn.studio/user-upload/65f24ce39b52-20260615.gif)  
![cat2](https://static.zenn.studio/user-upload/7967f89e3126-20260615.png)

完了・権限確認・入力待ちを教えてくれます。  
複数セッション開いていても気づけます。

## インストール

```
claude plugin marketplace add https://github.com/qvtec/claude-result-speak-cat.git
claude plugin install claude-result-speak-cat@claude-result-speak-cat
```

## にゃ語モード

`~/.claude/settings.json` に1行追加するとねこ語で通知してくれます。

```
{
  "env": {
    "CLAUDE_RESULT_SPEAK_CAT_LANGUAGE": "cat"
  }
}
```

かわいいのでおすすめです。

## まとめ

WSL2・Windows・macOS対応、外部パッケージ不要です。  
よかったら使ってみてください🐱
