「来月末で辞めます」

受け取ってから2週間。業務は山積みで、引き継ぎ資料を作る時間なんてどこにもない。でも作らないと次の担当者が困る。毎回こうなる。

引き継ぎ資料を作るとき、いつもこうなります。

- フォーマットがない → ゼロから設計   
- 「何を書けばいいか」がわからない → 手が止まる   
- 時間がない → 箇条書きメモだけ渡して終わり

引き継いだ側は何もわからない。また同じことが繰り返されます。

これ、Claude Codeで変えられます。**頭の中にある業務の手順を話しかけるだけで、引き継ぎ資料の骨格が5分で出来上がります。**

## Before / After

**Before：** 引き継ぎ資料をゼロから書く → 3〜5時間かかる → 退職直前に徹夜

**After：** 業務の流れをClaude Codeに話す → **骨格が5分で完成（工数90%削減）**

## 準備するもの

- Claude Code（インストール済みであること）   
- 引き継ぎたい業務の概要（思いつく範囲でOK）

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：引き継ぎたい業務を思い出す

完璧に整理しなくていいです。自分がいつもやっていることを箇条書きにするだけです。

```
・月末に売上データをExcelに貼り付けて営業部長に送る
・毎週月曜に在庫確認して発注メールを送る
・新入社員のシステム登録を行う
・システムが止まったときの緊急対応
```

「定常的にやること」と「たまに起きる突発対応」を分けて書くと後が楽です。

### ステップ2：Claude Codeに業務内容を話す

難しく考えなくていいです。いつも自分がやっている作業を、そのまま話しかけるだけです。

```
引き継ぎ手順書を作ってください。

業務名：月次売上レポートの送付
頻度：毎月末
使用ツール：Salesforce、Excel、メール
手順：Salesforceの「月次レポート出力」から始まって、
     Excelに貼り付けて、営業部長にメールで送ります。
所要時間：約30分
```

【IMAGE:02\_talk-to-claude.png】

### ステップ3：出てきた手順書を確認する

Claudeが「手順」「確認方法」「よくあるミス」の欄まで整理して出力します。

「ここをもっと詳しく」「この手順が抜けています」と追加指示すれば、その場で補完できます。

![](https://assets.st-note.com/img/1776991324-NeiP63IxC879qnmfU4uTH5Xo.png?width=1200)

### ステップ4：確認して保存する

出てきた手順書をコピーして、ドキュメントやテキストファイルに貼るだけです。

業務の数だけ繰り返せば、引き継ぎ資料セットの完成です。

![](https://assets.st-note.com/img/1776991331-2NIBJsfwOTigYa58U3Lhtx71.png?width=1200)

## できました。ただ、このままだと1つ問題があります。

定常業務の手順書は作れます。でも実際の引き継ぎには、こういう業務もあります。

- 月に1回しかない特殊な処理   
- システムが止まったときの対応   
- 上長への報告が必要な突発事態

毎回「定常業務です」「突発対応です」「緊急度は高いです」と説明するのは手間です。漏れたら引き継いだ側が困ります。

**これでは半分しか楽になっていません。**

## その解決策を、300円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、定常業務・突発対応それぞれのフォーマットを覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎回の引き継ぎ作業が変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「定常業務の手順書を作ってください」と説明が必要   
✅ ファイルあり：自動で定常業務用フォーマットで出力される

❌ 設定なし：突発対応のフォーマットを毎回ゼロから考える   
✅ ファイルあり：緊急対応も自動でフォーマット通りに出力される

❌ 設定なし：「確認方法」「よくあるミス」を自分で考えて書く   
✅ ファイルあり：質問しながら自動で埋めてくれる

一度設定すれば、ずっとそのまま使えます。   
**300円の投資で、この先ずっと続く「引き継ぎの憂鬱」が消えます。**

## 300円で何が変わるか

引き継ぎ資料作成1回3時間 × 年2回 = **年間6時間が引き継ぎ作業に消えています。**

設定ファイル1枚で、来月からその6時間が戻ってきます。

**300円 ÷ 6時間 = 1時間あたり50円。** これより安く時間を買う方法は、なかなかありません。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-006（添付）**　←　ダウンロードして置くだけ   
- **定常業務テンプレ**　←　毎月・毎週・毎日の繰り返し業務を漏れなく文書化   
- **突発対応テンプレ**　←　緊急時・例外ケースも自動でフォーマット提示   
- **3日間完成スケジュール**　←　退職直前でも間に合う進め方

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「定常業務と突発対応、どちらから始めますか？」→ 答える   
- 「引き継ぎ先への説明は口頭ですか、書面のみですか？」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたの引き継ぎスタイルを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
