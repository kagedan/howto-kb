やっぱりClaude Codeはすごい。

1600件以上あるイベント情報の中から、日々の行動履歴、趣味嗜好、過去の投稿内容、事業内容まで踏まえて、参加すべきイベントを選定。

さらに時間・場所・移動効率まで考慮して、最適なスケジュールを自動で組んでくれる。

しかもイベントの申し込みまで実行。

APIが用意されていないサイトでも、Claude_in_Chromeでブラウザを操作して完了させる。

もはや秘書というより、自分専属のエージェント。
