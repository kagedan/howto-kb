補足で説明すると、Claude Codeという機能をつかうためには、どのプラットフォームで使うかによって何パターンか種類があります。  
※わかりづらいかもしれないので説明すると、スマホで言うところのアプリからアクセスするか、Webブラウザからアクセスするかのような違いです。

Claude Codeの公式ドキュメントには以下のように種類が書かれています。  
1のRemote Controlが黒い画面のターミナルを操作するもの。2のウェブ版、３のデスクトップ版を使う方も多いです。

現在、**Claude Cowork**というものもありこれは2のウェブ版、３のデスクトップ版での利用です。  
Claude CoworkはClaude Codeのエージェントアーキテクチャをデスクトップアプリに持ち込んだもので、ターミナル不要で使えます。

**プラットフォームと統合**

1.Remote Control

2.ウェブ上の Claude Code

3.Claude Code デスクトップ版

3.Chrome 拡張機能（ベータ版）

4.VS Code で Claude Code を使用する

5.JetBrains IDEs

6.コードレビュー & CI/CD

7.Slack での Claude Code

## セットアップの仕方

1.パソコン上でターミナルを開き、以下を打ちます。  
  
<https://code.claude.com/docs/ja/setup#native-install-recommended>  
**macOS, Linux, WSL:**

```
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**

```
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**

```
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**Windows requires** [**Git for Windows**](https://git-scm.com/downloads/win)**.** Install it first if you don’t have it.

Native installations automatically update in the background to keep you on the latest version.
