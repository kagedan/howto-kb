はじめに

「AIで副業を始めたい」と思っても、  
何から手をつければいいか分からない。

そういう状態、ありませんか。

ChatGPTやClaudeを触ってみたはいいものの、そこから先に進めない。調べれば調べるほど情報が増えて、結局何もできないまま時間だけが過ぎていく。

私もそうでした。

この記事では、AI副業を全く知らない状態から始めた私が、最初に実際にやった5つのことを、やった順番のまま書きます。

先に全体の地図を見せます。

```
STEP1｜Xアカウントを発信用に整える
STEP2｜noteアカウントを開設する
STEP3｜AIツールに課金するか決める
STEP4｜Claudeで記事を書いてみる
STEP5｜ASPに登録する
```

流れとしては  
「**場所を作る → 道具を揃える → 実際に書く → 収益の入り口を用意する**」という順番です。

「完璧な準備をしてから始めよう」は罠です。動きながら覚える前提で、読んでください。

### STEP1｜見る専だったXアカウントを副業用に作り直す

### 最初にやったのは、Xアカウントの整理でした。

それまで見るだけだったアカウントを、副業発信用に作り直しました。プロフィールに「何をしている人か」を書いて、投稿できる状態にする。それだけです。

**なぜこれが最初か。**

Xは「人に見つけてもらう場所」です。noteもアフィリエイトも、読んでくれる人がいて初めて成立します。発信の土台がないまま記事を書いても、誰にも届きません。

ゼロから新規アカウントを作るより、既存アカウントの作り直しで十分でした。

**やること：** プロフィールに「何の発信をする人か」を1行書く。 例：「AI副業に挑戦中の49歳サラリーマン。学んだことを毎日記録中」

**所要時間：** 30分

STEP2｜noteアカウントを開設する

次に、noteアカウントを作りました。

最初の記事は「AI副業を始めます」という宣言だけ。完成度は低くていいです。「公開した」という事実を作ることが目的でした。

**なぜnoteか。**

Xの140字では伝えきれない体験や学びをまとめる場所が必要です。Xで興味を持ってもらった人が「もっと読みたい」と思ったとき、飛んでくる先がnoteです。

Xで知ってもらう → noteで信頼してもらう。この流れの「信頼を貯める場所」がnoteの役割です。

**やること：** アカウントを作って、宣言記事を1本だけ公開する。タイトルは「AI副業を始めます」で十分です。

**所要時間：** 1時間

### STEP3｜AIツールに課金するか決める

場所ができたところで、道具を決めました。

私はClaudeの有料プラン（Claude Code）に課金しました。

「コードが書けないのにClaude Code？」と思った方、正しい疑問です。Claude Codeはプログラミング用ツールとして知られていますが、実際は文章の構成検討や記事の改善にも使えます。私はコードを一行も書けないまま、文章作業の道具として使っています。

**ただし、最初から課金は必須ではありません。**

無料のClaude（claude.ai）から始めても、この後のSTEP4は問題なく実践できます。

**課金判断の基準：** 「このツールがあれば、1時間かかる作業が30分になるか」。なるなら課金する価値があります。分からないうちは無料版で十分です。

**注意点：** 課金ツールを増やしすぎると管理コストも月額負担も増えます。最初は1つに絞るのがおすすめです。

**所要時間：** 判断のみ（迷ったら無料版でスタート）

### STEP4｜Claudeで記事を書いてみる

道具が決まったら、実際に書きます。

最初は「記事を書いて」と雑に投げていました。すると、すぐに問題が起きました。

**AIに丸投げすると、自分が言っていない事実が混入します。**

私の場合、Claudeが私の経歴にない体験を「あたかも実話」として書いてきたことがありました。この失敗の詳細は[前回の記事](https://note.com/ai_fuku_salary49/n/n9face0cb53f9)に書いたので、ここでは対策だけ共有します。

**コピペで使える指示の型：**

```
この記事の構成案を作ってください。
各セクションの要点だけ箇条書きで出してください。
本文はまだ書かないでください。
私が伝えていない体験談・数字は入れないでください。
```

構成と要点だけAIに出させて、肉付けは自分の言葉でやる。これだけで捏造リスクが大幅に下がり、文章に自分らしさが残ります。

**所要時間：** 記事1本あたり1〜2時間（慣れると短縮できます）

### STEP5｜ASPに登録する

記事が書ける状態になったら、収益の入り口を用意します。

ASPとは、アフィリエイト広告の仲介サービスです。私はA8.netともしもアフィリエイトに登録しました。どちらも無料です。

**なぜこのタイミングか。**

登録してはじめて「どんなサービスを紹介できるか」が見えます。紹介できるサービスが頭の中にあるかどうかで、これから書く記事の設計が変わります。

逆に、記事を書く前に登録しても使い道がありません。「書ける状態になってから登録」がちょうどいい順番でした。

収益はすぐには出ません。ただ、登録しておかないと「記事は書けたのに収益化の仕組みがない」という状態になります。

**やること：** A8.netともしもアフィリエイトに無料登録する。

**所要時間：** 各30分・合計1時間


全部やっても、作業時間は合計3〜4時間。週末1日で終わる量です。

この5ステップが向いている人・向いていない人

**向いている人**

**向いていない人**

おわりに：始めるのが遅いほど、記録の価値が減る

私はまだ収益がゼロの段階です。

それでも始めてよかったと思うのは、「**初心者の視点は今しか書けない**」と気づいたからです。

慣れてしまえば、初心者がどこでつまずくか分からなくなります。つまり、始めたばかりの今の記録こそ、同じ場所にいる人への一番価値のある情報になります。

これは始めるのが遅くなるほど失われていく価値です。

まずSTEP1だけ、今日やってみてください。30分で終わります。

次回は、AIの学び方の選択肢（独学・書籍・講座）を私が実際に比較した記録を書く予定です。
