前回の「Claude Code 完全ガイド」で主要なスラッシュコマンドを紹介しましたが、実は**重要なコマンドが4つ抜けていました**。この記事では補完します。

## 抜けていた4つのコマンド

1. `/model` - モデル切り替え
2. `/fast` - 高速モード
3. `/plan` - プランニングモード
4. `/bug` - バグ報告

それぞれ詳しく解説します。

---

## `/model` - モデル切り替え

**概要:** 使用するClaudeモデルを切り替えます。タスクに応じて最適なモデルを選択可能。

```
/model
```

### 使用例

```
User: /model
Claude: 現在のモデル: claude-sonnet-4-20250514

利用可能なモデル:
1. claude-sonnet-4-20250514 (現在)
2. claude-opus-4-20250514
3. claude-haiku

切り替えるモデルを選択:
```
