Claude CodeをただのAIチャットだと思ってる人、かなり損してます。

本当にヤバいのは、
・スマホ操作
・調査
・テスト
・修正
全てAIに自動で回させる使い方。

もはや道具じゃなくて小さな開発チーム。

その使い方を20個だけ、超わかりやすく要約します👇

① /teleport → スマホの作業をPCに引き継ぐ
② /remote-control → 外からPCを操作する
③ /loop → テストと修正を自動で繰り返す
④ /schedule → 毎日の作業を自動化する
⑤ Chrome拡張 → 画面を見ながらAIに直させる
⑥ Hooks → AIの暴走を防ぐルールを付ける
⑦ SessionStart → 最初に必要な情報を自動で読む
⑧ PreToolUse → 実行前に確認や記録を入れる
⑨ PostToolUse → 保存後に整形やテストを動かす
⑩ Stop → 終了前に追加チェックする

11個目は👇

⑪ Git worktree → 作業を分けて同時進行する
⑫ /batch → 大量作業を小分けで進める
⑬ /btw → 作業を止めず横から質問する
⑭ /voice → 音声で背景や仕様を伝える
⑮ Codeタブ → 外出先から作業を始める
⑯ Chrome確認 → エラー原因をブラウザで確認する
⑰ テストループ → 落ちた原因と修正を回収する
⑱ スケジュール実行 → 毎朝の確認をAIに任せる
⑲ 調査タスク → 寝る前に原因調査を投げる
⑳ 小さく試す → いきなり全部自動化しない

Claude Codeは👇

Claude Codeは、ただコードを書かせるCLIではありません。

スマホ、PC、ブラウザ、自動ループをつなぐと、自分専用の開発チームみたいに動きます。

最初は
/teleport
/remote-control
/loop
Chrome拡張
Hooks
この5つだけ覚えれば十分です。

ここを理解するとAIの使い方がかなり変わります。
