## Claude Codeを副業に使うと時給は本当に上がるのか

結論から言います。Claude Codeを副業に使うと、時給は上がります。ただし「向く案件に投入した場合に限り、1.62倍」というのが、3ヶ月間使い倒した私の結論です。

月10万円とか時給5,000円とか、SNSで流れてくる数字は盛ってます。盛り方を解体しないまま「Claude Codeで副業革命」と煽る記事が多すぎて、2年前の私に見せたら絶対に踊らされたタイプの情報ばかりです。だから今日は、40歳メーカー営業係長の私が、Claude Code月額20ドル（約3,100円）を3ヶ月払いながら副業に投入した実数字を全部出します。

私は2018年から副業を20種やって、そのうち13種を月1万円以下で撤退してきました。せどりは半年で利益率5%に絶望して撤退、Webライターはクラウドワークスの文字単価0.5円地獄で3ヶ月目にギブ、Kindle出版は2冊目の印税が月714円で沈没、Uber Eatsは真夏の7月に週末稼働で腰を痛めて終了。今はnoteの有料記事とAIライティング代行で月8万前後を維持しています。その私が、Claude Codeに2026年1月17日から手を出して、3月31日時点で出した時給データが本記事の中身です。

先に断っておきます。この記事は「Claude Codeを買わせる」側の記事ではありません。むしろ「向いてない人が買うと3,100円×6ヶ月で1万8,600円をドブに捨てる」という警告記事です。そこを踏まえて読んでください。

## 結論、時給は1.6倍になる。ただし「案件の選び方」が9割

結論を先に出します。Claude Codeを副業に使うと、私の実測で時給は1,240円→2,010円に上がりました。1.62倍です。

ただしこれは「Claude Codeが得意な案件に限定した場合の時給」であって、向かない案件に投入すると逆に時給は下がります。ここを理解せずに「Claude Codeを買えば時給2倍」と読むと、また情報商材の罠です。3年前に私が29,800円のAI副業塾に入って、3ヶ月で退会した記憶がそのまま重なります。

### 3ヶ月使って出た時給の平均値

私が2026年1月17日に契約してから3月31日までの73日間で、Claude Codeを投入した副業の時給を1件ずつ計測しました。

* AIライティング代行（記事構成+本文の下書き）: 時給2,340円
* Chatbot構築支援（千葉県内の中小企業向け・要件定義+プロトタイプ）: 時給3,800円
* noteの有料記事執筆（タカシの副業帳）: 時給1,860円
* X運用代行（AIツール紹介アカウント）: 時給940円

平均すると時給2,010円。Claude Code導入前の同案件群の時給が1,240円だったので、1.62倍になった計算です。測定はすべて、着手から納品完了までの時間をNotionのタイマーで計測し、5分単位で切り上げて記録しています。

ただし、X運用代行だけは時給940円で、導入前の1,100円から下がっています。これがこの記事の肝です。

### 向かない副業に投入すると逆に時給は下がる

Claude Codeは「コードを書く」「構造化された長文を吐く」道具です。140字のツイートを作らせても、人間の推敲時間のほうが長くなるだけ。私はこれに気づくのに3週間かかりました。……いや、これは私の見立てが甘かった。

「AIを使えば何でも時給が上がる」は嘘です。道具には向き不向きがある──当たり前のことですが、副業ブームの煽りを浴び続けると、この当たり前を忘れます。2年前の私なら確実に「Claude Codeで副業全部効率化！」と浮かれて、X運用代行にも投入して時給を下げていたはずです。

**よくある撤退パターン**: 導入1ヶ月目で「全部の副業で試す」と宣言して、4〜5案件に同時展開するパターン。これは私が2020年のChatGPT登場時にもやった失敗で、結局どの案件でも中途半端な改善しか出ず、2ヶ月目で全部元のやり方に戻りました。新ツールは1案件ずつ差し込むのが鉄則です。

## Claude Codeを副業に突っ込んだ3ヶ月の実数字

先に数字を出します。73日間でClaude Codeに払った費用は9,300円、投下した副業時間は112時間、この期間の副業収益は225,400円でした。時給換算で2,012.5円。

### 月額料金と作業時間の内訳

内訳を全部出します。

* 月額料金: 20ドル × 3ヶ月 = 約9,300円（為替レート155円換算）
* 投下時間: 1月36時間・2月39時間・3月37時間 = 計112時間
* 副業収益: 1月66,400円・2月78,200円・3月80,800円 = 計225,400円
* 時給換算: 225,400円 ÷ 112時間 = 2,012.5円

Claude Codeを入れる前、2025年10〜12月の同じ副業群の時給は1,240円でした。月の収益は5〜6万円台で伸び悩んでいた。その理由ははっきりしていて、AIライティング代行で1記事書くのに4時間半かかっていたからです。構成の試行錯誤が長すぎた。特に12月の3連休に1記事に7時間かけて時給780円まで落ちた週があり、あの週の妻の冷たい目が今も記憶に残っています。

### 副業別の時給ビフォーアフター

案件ごとの時給推移を出します。

* AIライティング代行: 1,380円 → 2,340円（1記事の所要時間 4.5h → 2.6h）
* Chatbot構築支援: 2,200円 → 3,800円（要件定義書をClaude Codeで骨組み生成）
* noteの有料記事: 1,100円 → 1,860円（構成段階の試行錯誤が短縮）
* X運用代行: 1,100円 → 940円（後述する失敗案件）

伸びた3つには共通点があります。「下書きの試行錯誤に時間を食われていた工程」がボトルネックだった案件です。Claude Codeは、0から1を作る工程を圧縮する道具として非常に優秀でした。ただし、完成品をそのまま納品できるわけではない。ここは誤解されがちな点です。

**タカシの過去の失敗と重なる点**: 2022年に月額4,980円のライティングツールを契約したとき、私は「全案件で均等に時短効果が出る」と勘違いして、3ヶ月で1万4,940円を捨てました。ツールの効果は案件特性に依存する──この教訓を3年越しに思い出したのが、今回のX運用代行の時給下落でした。

## 時給が上がる人・下がる人の分かれ目

結論を先に。Claude Codeで時給が上がる人は「納品物の60%以上をテキスト生成物が占める副業をしている人」です。逆にテキスト生成が副業全体の30%以下なら、月3,100円の元は取れません。

### 時給が上がる人の3条件

私が3ヶ月で確認した、時給が上がる人の共通点は3つです。

* 条件1: 納品物の6割以上がテキスト・コード・構造化ドキュメント
* 条件2: 案件単価が5,000円以上（安すぎる案件はAI導入してもリカバーできない）
* 条件3: クライアントから「骨格は任せる」と言われるポジションにいる

条件3が特に重要です。細かい修正指示がびっしり飛んでくる案件だと、Claude Codeが出した下書きを直す工数のほうが、自分で1から書く工数より膨らむことがあります。これはnoteの有料記事で最初にハマった罠で、1記事目はAI補正に6時間かかって、導入前より時給が落ちました。妻に「また怪しいの始めた？」と言われた夜の話です。

### 時給が下がる人の典型的な使い方

逆に、時給が下がる人のパターンも3つあります。

* 「何でもAIに投げれば楽になる」と思って短文SNS運用に突っ込む
* プロンプトを保存・テンプレ化せず、毎回ゼロから書き直す
* 出力を鵜呑みにして納品 → 修正地獄で結局時間が倍

X運用代行で時給が下がったのは、140字のツイートにClaude Codeを使ったからです。AIに「面白いツイート5本」と投げて、出てきた5本の添削に40分。自分で書いたら20分。これは完全に私の判断ミスでした。2週目で気づいて、X運用はClaude Codeから切り離しました。

自分ツッコミですが、「AI副業」と「AIを副業に使う」は別の話です。混同すると、3,100円を毎月払いながら時給が下がる珍現象が起きます。

**よくある撤退パターン**: プロンプトを保存せず、毎回「いい感じに書いて」と投げるパターン。これだと3ヶ月経っても出力品質が安定せず、結局「AIは使えない」と結論づけて解約する人が多い。Notionに案件別のプロンプトテンプレを20個以上貯めて初めて、Claude Codeは時給を押し上げる道具に変わります。ここを飛ばした人は、高確率で3ヶ月目に解約しています。
