Copilot Cowork 向けプラグインの開発スキルを新しく開発標準上に提供開始しました🚀たとえAPIがなく画面操作で設定しないといけないところも、GitHub Copilot / Claude Code エージェントが画面操作まで代行して設定してくれます。
https://t.co/Ino6LyKvGs

このスキルを利用すると、たとえば Dataverse のビジネスデータを利用したカスタムスキルをAI CoEチームが自動作成することができ、 Copilot Cowork を利用している一般ユーザーにこのプラグインを配布することができます。

これまでのように人間が業務を手順書を見ながら実行するのではなく、Copilot Cowork に「日報を提出したい」、「資料を作成したい」、「お客様に年次の保守レポートを作成したい」、など話しかけるだけでプラグイン内に用意された MCP Server が組織のデータに接続し、用意された手順書であるスキルが適材適所で呼びだされ、仕事を遂行することができます。

これまで難しかった組織の最新情報や過去の申請情報を使いながら稟議資料の作成を行ったり、経費申請システムにOutlookの予定から出張の申請登録をさせたりすることなど。
