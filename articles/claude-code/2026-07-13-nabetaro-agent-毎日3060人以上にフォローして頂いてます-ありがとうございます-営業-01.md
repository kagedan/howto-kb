毎日30~60人以上にフォローして頂いてます！

ありがとうございます！

・営業しなくても上場企業からお願いしてくるくらいClaude Codeを極めた自動化ノウハウと、

・AIエージェントでの稼ぎ方を発信しています。

AIエージェントで稼ぐための条件は3つあります！！！

①：AIエージェントを使いこなすこと

→ どれだけ自動化できるか、自動でどれだけクオリティが高いものを作れるか、など。

②：参入するビジネスで勝ち上がる実力があること

→ Xなら、Xでフォロワーを増やすスキル、コンテンツを売るスキル、バズらせるスキルなど。

③：お金儲けやビジネス全般の実力があること

→ 逆算して計算通りに楽々稼ぐスキルなど。

たとえば、

・稼ぎ方、戦略、施策、スケジュールを全部描いて、計算通りに1億位上稼ぐ
・自分はあまり稼働せずに、AIや人の時間を使って楽々稼ぐ

とか。

①②だけでも普通に月300万円はいける。

③あれば、楽に稼げるし億いける。

①②③全部できる人がXにいないので、僕が発信して行きます。

フォローどうぞ。

困ってます。。。

何に困ってるかと言うと、

AIエージェントで稼げるアイデアやチャンスが多すぎる・・・！

・あれしたら稼げるだろうな
・こうしたら稼げるだろうな

ありすぎて手が回らない。

AIエージェントですでに320個以上自動化してるけど、それでもまだまだやりたいことが多すぎる。

海外の事例だけ見ててもいろんな稼ぎ方あるし、それを横展開すれば無限に稼ぎ方あります。

稼げるアイデアが多すぎるので、こんな時の僕の対策は3つ。

・A：人に教えて、実行してもらって、利益を折半する
・B：人を雇ったりAIの自動化でぶん回す
・C：諦めて、大きく儲かる少数のことにだけ絞る

自分でAIエージェントで自動化するにしても、それ以上にアイデアが多くて流石に追いつかない。

今回は「C：諦めて、大きく儲かる少数のことにだけ絞る」ですかね。

時価総額5億~15億の会社は作れそう。

作れるのは作れるけど、あとはどれだけスピード上げるかってところ。人事戦略だけ描けばいけるな。あとはVCも入れるか。

時価総額1億の会社を何個も作らせてもいいけど、1つにコミットした方が流石に儲かりそう。
