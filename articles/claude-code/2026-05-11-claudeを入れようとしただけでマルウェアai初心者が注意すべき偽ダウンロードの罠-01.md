最近、ClaudeやChatGPTなどのAIツールを装った偽サイトや不正なインストール手順を使って、マルウェアに感染させようとする事例が報じられています。

たとえば、Claudeをまねた偽サイトから不正なインストーラーを配布したり、Google広告を悪用してMac向けの不正なコマンドを実行させようとする手口が確認されています。

ここで大事なのは、**Claudeそのものが危険という話ではない**ということです。

狙われているのは「AIを使いたい人」です。 便利そうだからと検索して、入口を間違えてしまう人が増えているのと同じタイミングで、攻撃する側もそこを狙うようになっています。

この記事では、AI初心者がやりがちな危険な導入パターンと、安全に始めるために確認したいポイントを整理します！

**「入れ方」が危ない、という話**

AIを仕事で活用するとき、多くの人は「使い方」を気にします。 プロンプトの書き方、どの業務に使えるか、ChatGPTとClaudeはどう違うか。

でも、**その前に意外と見落とされているのが「入れ方」**です。

AIツールを使い始めるとき、こんな検索をすることはないでしょうか。

* Claude ダウンロード
* Claude Code インストール
* ChatGPT アプリ 入れ方
* Mac AIツール 使い方

このとき、検索結果の一番上に広告が表示されていると、つい公式サイトだと思ってクリックしてしまうことがあります。

**検索結果の上にあるからといって、公式とは限りません。**

広告枠は誰でも出稿できるため、見た目だけ公式に似せた偽サイトが混ざっていることがあります。

**「共有チャットだから安心」とも限らない**

今回の事例で特に注意したいのが、Claude.aiの共有チャット機能も悪用されているという点です。

共有チャットは便利です。 誰かが作った設定手順を見たり、プロンプトを参考にしたりできます。 見た目は正規のサービスのページです。

ただし、**その中に書かれているコマンドやリンクが安全とは限りません。**

特に以下のものが書かれていたら、実行前に立ち止まることをおすすめします。

初心者ほど、「詳しそうな人が書いているから大丈夫」と感じてしまいがちです。

**AIを安全に使う人は、"それっぽい手順"をそのままコピーしません。**

**Macだから安全、は言い切れない**

「Macはウイルスに強い」というイメージを持っている人も多いと思います。

たしかに、Windowsと比べて狙われ方の仕組みは違います。 ただ、**Macだから絶対に安全というわけではありません。**

今回報じられているように、Macユーザーを狙ったマルウェア誘導も確認されています。

特に危険なのが、ターミナルでコマンドを実行させる手口です。 見た目には「1行コピーするだけ」の作業に見えますが、ターミナルのコマンドは状況によっては次のような操作ができます。

* 外部からファイルをダウンロードして実行する
* 設定ファイルを書き換える
* 外部プログラムを呼び出す

つまり、**"コピーして貼るだけ"の操作が、実はかなりリスクの高い行動になることがある**ということです。

**会社PCでAIツールを入れる前に確認したいこと**

個人のPCであれば、自分の判断で入れることもあるかもしれません。 ただ、会社PCの場合はもう一段階の確認が必要です。

以下の点を入れる前に確認してみてください。

✅ そのAIツールは会社で使ってよいか   
✅ 公式サイトのURLを直接確認して入れているか（検索広告からではなく）  
✅ ターミナルのコマンドを実行する手順の場合、内容を理解しているか   
✅ ブラウザ拡張機能なら、要求している権限が強すぎないか  
✅ 顧客情報や社内資料を入力してよいルールになっているか

AIツールは仕事をラクにしてくれる存在です。 だからこそ、**「とりあえず入れてみる」ではなく、「安全な入口から入れる」**ことが大事です。

**まずこれだけやっておくと安心**

難しい対策は要りません。 次の3つを意識するだけでも、リスクをかなり下げられます。

**① 公式サイトのURLを直接確認して入れる**

ClaudeならAnthropicの公式サイト、ChatGPTならOpenAIの公式サイトを、ブックマークに登録しておくのがおすすめです。 検索結果の広告や、見た目が似ているサイトには注意が必要です。

**② 意味がわからないコマンドをターミナルに貼らない**

手順が書かれていたとしても、何をしているコマンドか理解できない場合は実行しないほうが安全です。 「動いたからOK」では判断できないのがコマンドの難しいところです。

**③ 会社PCでは勝手に入れない**

便利そうだからといって、管理者の確認なしにアプリや拡張機能を入れると、後で問題になる可能性があります。 まず「使ってよいか」を確認するのが安全です。

**まとめ：AI活用は「使い方」と「入れ方」がセット**

AIツールは、仕事を大きくラクにしてくれる存在です。 ただし、人気があるからこそ、悪用される入口も増えています。

今回の事例から学べることは一言で言うと、**AIを使い始める前の「入口の確認」が大事**ということです。

* 検索結果の一番上をそのまま信じない
* 公式サイトかどうかURLを確認する
* 共有チャットやSNSの手順を鵜呑みにしない
* 意味がわからないコマンドはターミナルに貼らない
* 会社PCでは導入ルールを先に確認する

「どう使うか」だけでなく、「どう安全に始めるか」をセットで考えると、AIをより安心して活用できるようになります。
