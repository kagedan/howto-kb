1日に何通、「似たようなメール」を書いていますか。

問い合わせへの返信、依頼への承諾・断り、確認メール、お礼メール……。内容はほぼ同じなのに、毎回ゼロから書いている。

Claude Codeで、これを変えられます。**メールの本文を貼り付けるだけで、返信の下書きが30秒で出てきます。**

## Before / After

**Before：** 受信メールを読む → 返信内容を考える → 書く → 確認 → 送信（1通5〜10分）

**After：** メール本文をClaude Codeに貼るだけ → 返信下書きが出てくる → **1通1〜2分（80%削減）**

## 準備するもの

- Claude Code（インストール済みであること）   
- 返信したいメール本文

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：返信したいメールをコピーする

メールアプリを開いて、返信したいメールの本文をCtrl+Cでコピーします。

![](https://assets.st-note.com/img/1776991593-sRfwDjWH3M2oSc09mZUIgerX.png?width=1200)

### ステップ2：Claude Codeに貼って依頼する

```
以下のメールに返信してください。

[メール本文をここに貼る]
```

一言添えるとさらに精度が上がります。

```
以下のメールに返信してください。
内容：会議の依頼を受諾する。参加可能な日程は〇月〇日〜〇日の午後。

[メール本文をここに貼る]
```

![](https://assets.st-note.com/img/1776991601-OHqcgudDJVy0Er6fUFlvxLNP.png?width=1200)

### ステップ3：出てきた下書きを確認する

Claudeが返信文を出してくれます。固有名詞や数字だけ確認して、問題なければそのまま送信。

![](https://assets.st-note.com/img/1776991612-wDOKVYMalbiN6HF9pPgz5BdS.png?width=1200)

### ステップ4：一言添えてトーンを調整する

```
もう少し丁寧なトーンで書き直してください。
3行以内に短くしてください。
もっとフランクな表現にしてください。
```

これだけで瞬時に修正されます。

![](https://assets.st-note.com/img/1776991621-ZQomjvrGMThgNAxJnEHFq1t9.png?width=1200)

## できました。ただ、このままだと1つ問題があります。

下書きを出してもらうだけなら今すぐできます。でも毎回「〇〇さんへの敬語で」「署名は〇〇で」と説明するのは、結局また時間がかかります。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、自分の署名・文体ルール・使わない表現を覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎回のメール返信が変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「〇〇さんへの敬語で」と説明が必要   
✅ ファイルあり：自動で適切な敬語レベルで返信される

❌ 設定なし：毎回「署名を入れて」と追加   
✅ ファイルあり：毎回自動で署名が入る

❌ 設定なし：毎回「短めに書いて」と指定   
✅ ファイルあり：常に簡潔な文体で返ってくる

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「メール返信の憂鬱」が消えます。**

## 100円で何が変わるか

メール返信1通5分 × 1日10通 × 240日 = **年間200時間がメール返信に消えています。**

設定ファイル1枚で、来月からその時間が4分の1になります。

**100円 ÷ 150時間削減 = 1時間あたり約1円。** これより安く時間を買う方法は、なかなかありません。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-008（添付）**　←　ダウンロードして置くだけ   
- **シーン別プロンプト8種**　←　依頼受諾・断り・催促・お礼・確認・謝罪・社内連絡・営業で使い分け   
- **「自分らしい文体」を覚えさせる方法**　←　過去のメールを渡すだけで文体学習完了

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「どんな場面でメールを書くことが多いですか？」→ 答える   
- 「普段使っている署名を教えてください」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたの文体・署名・ルールを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
