今更だけど、CodexデスクトップアプリのSSHとか他のデバイスを操作とかの体験良すぎ…
各パソコンの境界が溶けて、今どこで作業しているか分からなくなるレベル

Mac miniとか、眠っていた昔のパソコンにUbuntu入れて、Codex経由で操作するのがめっちゃ楽しい

・機械学習担当のデスクトップ(Ubuntu)
・環境ぶっ壊れてもいい前提のめっちゃ古い実験用のAlienwareのラップトップ（これもUbuntu）
・定期的なプロモーションの投稿とか仮想通貨の自動取引Botなどの軽い処理を担当するcron担当用のmac mini

などをMacBook、GPD microPC、iPhone経由で操作してる

特に、低頻度だが何度も使いたいセッションとかがすぐに見つけられて、パソコン苦手なわいにとってはtmuxよりも体験良いと感じる

複数セッションの並列実装を、それぞれの進捗をチラ見しながら開発とかは難しそうだけど、終わったら通知来るから左側にあるセッションをぽちぽち切り替えながら進めてる感じ
あと、tmuxのほうが便利だと思ったのは、PCを跨ぐセッションの参照

Mac mini上のCodexのセッションについて、MacBook上のCodexに参照して、と言っても、無理っすと言われてしまった

tmuxだったら開いていさえすれば、PC跨いでもこのPaneを参照してって言えば、読んでくれていた気がする

やり方あるのか分からんけど、セッションID渡したら参照してくれるようになると、便利なのになぁ、と思う
