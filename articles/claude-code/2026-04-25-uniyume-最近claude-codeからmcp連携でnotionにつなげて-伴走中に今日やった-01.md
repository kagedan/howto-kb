最近、Claude CodeからMCP連携でNotionにつなげて、
「伴走中に今日やった作業まとめといて〜」って言うだけで
作業ログを整理してもらえるようにしたんだけど、
これがかなり助かってる。図にまとめます👇

その場でやったことって、
終わった直後は覚えてるつもりでも、
あとから見返すと意外と忘れてる。

でもNotionにまとめておけば、

・今日やったこと
・変更したポイント
・詰まったところ
・次回やること
・生徒さんへの共有メモ

まで残せる。

ただ記録するだけじゃなくて、最後に

「今日の作業、もっといい手順なかった？」
「次回どんな展開できる？」

って聞けるのもめちゃくちゃ良い✨

その場でやったことって、どうしてももっとこうすればよかった！がでやすいので。

ページを複製できる形にしておけば、
生徒さん側もそのまま自分用のメモにできるし、
復習もしやすい。（といいなと思っている）

一緒にやった作業が
その場限りで終わらずに、
あとから見返せる教材みたいに積み上がっていく感じ✨

これ、伴走やコンサルとの相性めちゃくちゃ良い。
Claude Code × MCP × Notionはおすすめです✨
