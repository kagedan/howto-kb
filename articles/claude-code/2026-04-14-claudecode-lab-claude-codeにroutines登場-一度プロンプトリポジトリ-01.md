Claude Codeに「Routines」登場🚀

一度プロンプト・リポジトリ・コネクタを設定するだけで、  
・スケジュール実行  
・API呼び出し  
・GitHubなどのイベント応答  

で自動で動くようになりました🔥

なんと、Claude側環境上で実行されるからPC開けっぱなし不要。

例：
- GitHub PRで特定ファイル変更を検知→Slackにサマリー投稿
- 毎朝2時にLinearからトップバグ取得→自動修正→Draft PR作成

CLIの/scheduleも統合済み。  
全有料プランで即利用可（Claude Code Web有効時）

Docs: https://t.co/ddmS0ZbKod

これでAIが本格的に「働く」時代到来🔥
