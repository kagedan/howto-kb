Claude Codeユーザー必見。
Claude公式のCLAUDE. mdを改善するプラグインが良い。

できること2つ：

① CLAUDE. mdの品質監査
→ 6項目（コマンド網羅性・アーキテクチャ明確さ・簡潔さ等）でA〜Fスコアリング
→ 実際のフォルダ構成とCLAUDE. mdを照合して「ここ古くなってるよ」をdiffで提案

② セッション終了時の学び記録（/revise-claude-md）
→ そのセッションで発見したコマンド・gotchas・パターンを自動抽出してCLAUDE. mdに追記提案

試しに自分のCLAUDE. mdを監査したら Grade C（52/100）...

「フォルダ構成が実態と乖離」「新規プロジェクトのルールと矛盾」等、自分では気づかなかった問題点をちゃんと指摘してくれた。

CLAUDE. md、書きっぱなしで放置しがちなので定期的にこれ回すと良さそう。
