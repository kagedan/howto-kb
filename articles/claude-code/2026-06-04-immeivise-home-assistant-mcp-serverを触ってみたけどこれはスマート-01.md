Home Assistant MCP Serverを触ってみたけど、これはスマートホーム界隈に革命起こしそう。Model Context Protocolを使ってClaudeが直接Home Assistantのエンティティを操作できるのが衝撃的だった。これまでのYAMLゴリゴリの自動化設定から、自然言語で「寝る前のルーティン実行して」って言うだけでライトを消して温度調整してくれる世界になる。実際に試したら応答速度も悪くないし、デバイス認識の精度もかなり高い。HA NOVAプロジェクトも並行して進んでるみたいだけど、こっちの方向性の方が一般ユーザーには刺さりそう。ただMCPの仕様がまだ流動的なのが少し心配で、本格運用するには様子見が賢明かもしれない。
