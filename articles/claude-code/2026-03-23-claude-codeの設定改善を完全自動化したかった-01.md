本記事の内容は執筆時点の Claude Code 2.1.81 をもとにしています。

Claude Codeを使っていると、こうすればもっと便利になりそうという気づきが積み重なります。ただ、そのたびに作業を止めて設定ファイルを開くのは気が乗りません。
「気づいたら設定が整っている」という状態が欲しくて、試行錯誤しました。

 /insights だけでは物足りなかった
Claude Codeには/insightsというビルトインコマンドがあります。過去のセッション履歴を分析して使い方のパターンや改善ポイントをレポートしてくれる機能です。有用なのですが、使っていて2点が気になりました...
