# Claude Code の Auto モードを使いこなす

## Auto モードとは？

Claude Code を最大限に活用するための最重要テクニックとして、**Auto モード**が注目されています。

Auto モードを有効にすると、**パーミッション（権限確認）のプロンプトが表示されなくなります**。これにより、Claude の作業フローが中断されることなくスムーズに進みます。

## Multi-Clauding（マルチクローディング）の実現

Auto モードの真価は、**Multi-Clauding**と呼ばれる並列作業スタイルにあります。

1. まず1つのセッションを開始する
2. そのセッションが実行中の間に、**別のセッションで別のタスクに取り組む**
3. 複数のセッションを並行して動かし、作業効率を大幅に向上させる

## Auto モードの有効化方法

キーボードショートカット **`Shift + Tab`** を押すだけで Auto モードに切り替えられます。

## 最新アップデート

* ✅ **Pro プランでも利用可能**に（以前は上位プランのみ）
* ✅ **Sonnet 4.6** のサポートを追加（Opus 4.7 と並んで選択可能）

## まとめ

Claude Code をヘビーに使うなら、Auto モードはぜひ活用したい機能です。`Shift + Tab` でオンにして、複数セッションを並行させる Multi-Clauding スタイルを試してみましょう。
