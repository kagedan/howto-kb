② Claude Code /simplify・/batch

1つずつ振り返る🧵

① Claude Coworkに「スケジュールタスク」が追加された。

朝のブリーフィング自動生成、週次スプレッドシート更新、金曜のチームプレゼン準備。

全部Claudeが自動でやってくれる。

繰り返し作業をAIに投げて、人間は判断だけに集中する。

そういう働き方が現実になってきた。

https://t.co/iRUwOvRw3E

② Claude Codeに「/simplify」と「/batch」が追加されたらしい。

/simplifyはコードを整理・簡素化。
/batchは複数のPRを並列処理。

開発者が「毎日使っている」と言うくらい実用的。

エンジニアの仕事が「コードを書く」から「Claudeに指示する」に変わっていく流れが加速している。

https://t.co/HDtLWRfOiG

③ Claudeのメモリ機能が無料プランで使えるようになった。

インポートも簡単に。エクスポートもいつでも可能。

AIが「前回の会話を覚えていない」という最大の不満が解消される。

しかも無料で。

有料プランの差別化要素を一つ手放してでも「まず使ってもらう」を優先した。

Anthropicの判断として正しいと思う

https://t.co/cHqYIHZo6C

④ Chrome拡張に「Quick Mode」が追加されて、処理速度が以前の3倍になったらしい。

「ちょっと遅いな」というストレスが消える。

AIツールの普及において速度は思っている以上に重要なファクター。

速いから使う、ではなく、遅いから使わない、がなくなる。

ユーザー体験の改善が一番の機能追加だったりする。

https://t.co/lJrsMGRu78

⑤ Claude Codeに「Remote Control」が追加された。

ターミナルでタスクを開始して、散歩中や会議中にスマホから続きを操作できる。

マシン上でClaudeが動き続けて、自分はどこからでも制御できる。

「AIが常時稼働している状態」を前提にした設計。

リモートワークの次は「AIワーク」という概念が来るかもしれない。

https://t.co/NGMMrFuFAn

⑥ Coworkのプラグインとコネクタがアップデートされてエンタープライズ向けが本格化。

・企業がClaudeをカスタマイズできる
・チーム間のコラボレーションを向上
・外部サービスとの連携を拡張

OpenAIのGPTsに対するAnthropicの答えがこれだとすると、かなり面白い競争になってきた。

企業がAIを「使う」から「自分たちに合わせてカスタマイズする」時代に入ってきた。

https://t.co/I4QI3uz6RI

⑦ Claude Codeに「Auto-Memory」が追加された。

セッションをまたいでClaudeが学んだことを自動記憶して、次回以降に活かす。

プロジェクトのコンテキスト、デバッグのパターン、コーディングの癖。

全部Claudeが勝手に覚えてくれる。

「新しいセッションのたびに説明し直す」という開発者の最大のストレスが消える。

https://t.co/5GSYWakMF1
