「Claude Codeがすごい」という投稿、最近やたら見かけませんか。無料でここまでできる、副業にも使える、みたいな雰囲気。  
AI副業を検証している以上、触らずにスルーはできないなと思って、やってみました。 これはその顛末です。

## ターミナルと格闘した1時間

作業自体はシンプルなはずでした。  
コマンドを打って、設定を入れて、エラーが出たら直す。

![](https://assets.st-note.com/img/1774937803-7J2VUQITgPebGAK1f4dxkMmp.jpg?width=1200)

↑こういう画面です。  
ただ、実際はなかなかしんどかったです。コピペしてもエラー、権限エラーで止まる、英語のメッセージが読めない。  
「あれ、詰んだかも」と思いながら、画面をスクショしてChatGPTに見せて、一個ずつ進める感じになりました。

それでも粘って、エラーを潰して、  
「さあ、使うぞ！」ってなったタイミングで。

**「Claude Codeを使うにはProまたはMaxプランが必要です」**

止まりました。

## 「無料でできる」の意味が違った

「無料でここまでできる」って、環境構築までの話だったようです。実際に使う段階では課金が必要な構造でした。

設定はできる。でも使うなら有料。

このズレ、けっこう大きいと感じました。  
正直、ChatGPTに八つ当たりしました（ChatGPTは悪くない）。

## それでも収穫はあった

ただ、今回やってみて分かったこともあります。環境構築の手順、エラーへの対処、AIと一緒に進める感覚。これは無料でも体験できました。

ツールそのものより、「作るプロセスを体験すること」の方が副業的には本質かもしれないな、と感じています。

## これから触る人へ

Claude Codeに興味がある方は、この線引きだけ先に知っておくと消耗が減ると思います。

環境構築まで → 無料でできます

実際に使う → 有料プランが必要です（ProまたはMax）

「誰でも簡単に無料で使える」という印象ではなかった、というのが正直なところです。同じ落とし穴にハマった人、いますか・・・？😢
