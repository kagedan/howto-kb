. @SentientAGI  has released an update to EvoSkill to version 1.3.0, and this time the changes are not only about small improvements. While the first releases were focused mainly on Claude Code and OpenCode, now the platform has become much more versatile. EvoSkill has received support for Codex, OpenHands and Goose, so the same mechanism for automatic skill development can be used with different agents.

Another important novelty is the integration of Fireworks AI. Now it can be used both to run open models during skill evolution and to automatically evaluate the results. Previously, users were limited to the Claude API and OpenRouter, but now they have more choices without changing their own workflow.

EvoSkill is gradually transforming from a research tool into a universal platform where you can experiment with different agents and models without rebuilding the entire pipeline after each update.
