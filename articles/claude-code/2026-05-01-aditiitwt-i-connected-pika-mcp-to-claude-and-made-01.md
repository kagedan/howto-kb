How I did it:

1. Give birth to your Pika self
→ Open Claude Code / Cowork
→ Go to Customize
→ Click + Personal Plugins
→ Select Create Plugin → Add Marketplace
→ Paste: Pika-Plugins
→ Click Sync

2. Connect Pika to Claude
→ Open the Pika plugin
→ Go to Connectors
→ Click Connect
→ Complete the Claude OAuth login in browser
→ Once connected, turn the plugin ON

3. Use Pika inside Claude
→ Run commands like:
/pika:ugc-ad
/pika:podcast
/pika:explainer
