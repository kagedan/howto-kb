I use Claude Code and Obsidian every day. I teach them. they've genuinely changed how I work.

but there's a real danger in that space: every day on Twitter, 50 articles about some new feature, some new tool, some new workflow you "need" to know about.

all of that is a distraction.

the only question that matters is: does this serve what I'm actually trying to build?

if the answer is no, it doesn't matter how powerful the tool is.
