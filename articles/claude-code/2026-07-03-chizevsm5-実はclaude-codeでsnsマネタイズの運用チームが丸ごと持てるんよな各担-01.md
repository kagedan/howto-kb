実はClaude Codeで"SNSマネタイズの運用チーム"が丸ごと持てるんよな。各担当AIが何を自動化できるか一覧にした。保存推奨↓

【戦略責任者】発信方針・方向性の設計
コンセプトをCLAUDE.mdに固定/ターゲットの"言えない悩み"を定義/仮想敵を設定/商品ライン・価格・売る順番を設計/世界観と口調ルールを言語化
→ ここがブレると下の全部がブレる。最初に固めるのが9割。

【リサーチ担当】競合分析・市場調査
競合の"勝ってる要素"を抽出/客の生々しい言葉を収集/満たされてない需要を洗い出す/刺さる切り口の当たりを付ける/自分の伸びた投稿の要因を特定
→ 売れる言葉は自分の頭じゃなく、客とライバルの中に落ちてる。

【企画担当】投稿企画・ネタ出し
1ヶ月分のネタを一気出し/フック型(問い・断言・逆張り)に振り分け/連投・スレッドの構成台本/季節・流行に合わせた企画/伸びたネタの派生案
→ ネタ切れは根性じゃなく、仕組みで消える。
（続く）

【編集担当】投稿作成・改善提案
決めた型で本文を執筆/フックをA・B・Cで複数案/弱い所にダメ出し＆修正/媒体別(X・Threads・note)に最適化/勝ち投稿の口調に寄せて仕上げ
→ 一発で完成させず"添削まで"やらせると精度が段違い。

【教育担当】無料コンテンツ・信頼構築
教育note・コラムの中身を作成/LINEステップ配信(教育シナリオ)を設計/"教育前→教育後"の変容ステップを組む/悩みを言語化した共感コンテンツ/売り込まず必要性を刷り込む
→ 売る前に「この人から買いたい」を作る。ここを飛ばすと一生売れない。

【マーケティング担当】note導線・販売設計
無料→有料の導線をつなぐ/note・LPの構成を設計/松竹梅オファーを作る/ステップ配信(告知→教育→販売)を用意/想定反論と切り返しを準備
→ 集客と販売は"線"でつないで初めて売上になる。
（続く）

【顧客対応担当】DM・フォロー・リピート
DM返信を自分の口調でテンプレ化/購入後フォロー(3日・7日・14日)を設計/リピート・アップセルの案内文/FAQの一次回答を用意
→ 新規を増やすより、一度来た人にリピートしてもらう方が10倍ラク。

【アナリスト担当】数字で改善
集客数×成約率×単価を可視化/ボトルネックがどこか特定/フック違いのA・Bテストを設計/離脱ポイントの改善案を出す/月次の数字レポートを自動化
→ なんとなくの運用を、数字分析する運用に変えるだけで景色が変わる。

【秘書担当】タスク管理・進行管理
やること化して優先順位付け/抜け漏れチェック＆次の一手/逆算スケジュールを作成/再現できる手順として保存/決定事項をlog化して蓄積
→ 動けないのは意志じゃなく、次の一手が見えてないだけ。

普通これ全部やろうとしたら、外注に月何十万、何百万か、チーム雇うしかなかった。

それがClaude Code一つで揃う。個人の限界って"体が1つ"なことだったけど、その常識が破壊された。

ぜひ保存してやってみて。さらに詳しく知りたい人は固定ポストへ。
