経費精算のチェック、やっていますか。

日付の抜け、金額の転記ミス、領収書番号のズレ、上限超えの交通費——。一枚一枚確認して、エラーを見つけて、担当者に差し戻して。月末のあの作業、毎回同じことをやっている気がしませんか。

Claude Codeで、これを終わりにできます。**ExcelデータをClaude Codeに貼り付けるだけで、エラーのある行だけ抜き出して返ってきます。**

## Before / After

**Before：** 経費精算Excelを1行ずつ目で確認 → エラーをメモ → 担当者に連絡（30〜60分）

**After：** データをClaude Codeに貼るだけ → 「ここがおかしい」リストが返ってくる → **3分で完了（90%削減）**

## 準備するもの

- Claude Code（インストール済みであること）   
- チェックしたい経費精算データ（ExcelをテキストでコピーでもCSVでもOK）

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：チェックルールを決める

まずどんなミスを見つけたいかを整理します。完璧でなくていいです。次のステップのプロンプトに直接書くので、メモ程度でOKです。

![](https://assets.st-note.com/img/1776992041-1szU9vyMmSoQnZdt2aNr60Bp.png?width=1200)

### ステップ2：データをClaude Codeに渡す

```
以下の経費精算データをチェックしてください。

チェックルール：
・日付が2026年4月以外のものを抽出
・交通費が10,000円を超えるものを抽出
・領収書番号が空欄の行を抽出
・同じ日・同じ金額が重複している行を抽出

データ：
[Excelからコピーしたデータを貼る]
```

### ステップ3：エラーリストを確認する

Claudeがエラーのある行だけをまとめて返してくれます。

```
チェック結果

エラーあり（要確認）

行5  田中  タクシー    15,000円  交通費上限超え（上限10,000円）
行8  鈴木  外食         3,200円  領収書番号なし
行12 佐藤  電車           500円  日付が3月（対象外）

問題なし：18行
```

### ステップ4：エラー行だけ担当者に差し戻す

リストをコピーしてメールやSlackに貼るだけ。説明する手間が省けます。

## できました。ただ、このままだと1つ問題があります。

毎月、チェックルールを書き直すのは結局手間です。

会社ごとにルールは違う。上限金額も、領収書の要否も、カテゴリの定義も。それを毎回入力するのでは、半分しか解決していません。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、自社のチェックルールを一度書いておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎月の経費チェックが変わります

設定ファイルを一度置けば、毎回のルール入力がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「交通費の上限は1万円で」と説明が必要   
✅ ファイルあり：自動で自社ルールに基づいてチェックされる

❌ 設定なし：毎回「領収書番号が空欄の行を探して」と追加   
✅ ファイルあり：常にルールに沿ったチェックが走る

❌ 設定なし：エラーが出たら差し戻し文を自分で書く   
✅ ファイルあり：エラーリストから差し戻しメールまで1コマンドで完成

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「月末の経費チェック作業」が大幅に減ります。**

## 100円で何が変わるか

経費チェック30〜60分 × 月1回 × 12ヶ月 = **年間最大12時間が経費チェックに消えています。**

設定ファイル1枚で、来月からその時間が大幅に減ります。

**100円 ÷ 10時間削減 = 1時間あたり10円。** ルールが多い会社ほど効果が大きい設定です。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-011（添付）**　←　ダウンロードして置くだけ   
- **よくあるチェックルール8種**　←　日付・上限・重複・カテゴリ整合など、コピーして使える設定例   
- **差し戻しメール自動生成の使い方**　←　エラーリストからそのまま担当者への連絡文が出る

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「交通費の上限金額を教えてください」→ 答える   
- 「チェックしたいルールを教えてください」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたの会社のルールを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
