## 今日やったこと

Claude Codeに音楽ジャンル診断アプリを作ってもらいました。

指示はこれだけです。

```
音楽ジャンル診断アプリをmusic_genre.htmlとして作って。15問の質問に答えるとあなたに合う音楽ジャンルが診断される。ジャンルの特徴・おすすめアーティスト・プレイリストのイメージを表示。見た目はおしゃれに。
```

## できたもの

![](https://assets.st-note.com/img/1779529129-f7PwN6TMID85hkCslrbLAxag.png?width=1200)

**診断できるジャンル（10種）：** 🎤 ポップ / 🎸 ロック / 🎤 ヒップホップR&B / 🔊 エレクトロニック / 🎷 ジャズ / 🎻 クラシック / 🌿 フォーク / 🌸 K-POP / 🤘 メタル / 🎼 R&Bソウル

**結果画面：**

* メインジャンルのマッチ度%
* 音楽パーソナリティの性格分析
* おすすめアーティスト4組（説明付き）
* プレイリストイメージ4シーン
* 他ジャンルとの相性ランキング

## 今回気づいたこと

音楽は「自分を表現するもの」でもあります。

どんな音楽を聴くかで、その人の性格や価値観がわかる気がする。だから「あなたに合う音楽ジャンルは？」という診断は、MBTIや血液型と同じく「自分を知る」コンテンツになります。

診断系は作り続けてきましたが、音楽というテーマは今まで一番身近な気がします。
