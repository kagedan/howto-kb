「Claude Code 何から始めるか分からん」勢に朗報。

Anthropic公式プラグイン「claude-code-setup」が登場。

✅ プロジェクトを自動分析
✅ 推奨自動化（hooks / skills / MCP / subagents）を提案
✅ 設定手順をステップ案内

つまりCLAUDE.md / hooks / agentsの「最初の壁」を公式が引き受ける構造。

注意点として、既存の自前CLAUDE.mdが育ってる場合は提案と衝突しがち。
ゼロから始める人 → 即入れて良し。
育成済みVault勢 → 提案を見て差分だけ採用が無難。
