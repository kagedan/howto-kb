【保存版】Claude Codeのおすすめ機能8選

1. スキル：よく使う指示を「/コマンド」で呼び出し、毎回の入力を省く
2. フック：決めた作業を自動で必ず実行（保存時のチェックなど）
3. サブエージェント：調べ物や下書きを別担当に任せ、会話を散らかさない
4. MCP連携：Slack・スプレッドシート・ブラウザなど外部ツールとつなぐ
5. 定期実行：毎朝の情報収集など、決まった作業を自動で回す
6. プラグイン：便利な機能をまとめて追加できる
7. CLAUDE.md：自分のルールを覚えさせて毎回の説明を不要にする
8. プランモード：いきなり動かず、先に手順を相談できる

エンジニア向けに見えますが、要は「自分専用の仕事アシスタントを育てる道具」です。
特に1と5は、ライターや事務のくりかえし作業をそのまま自動化できます。

気になるものから触ってみましょう。
