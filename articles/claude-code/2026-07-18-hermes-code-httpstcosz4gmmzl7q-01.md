## 1. 一発で当てにいかない——「量」と「往復」で精度を上げる

落合氏の使い方は「たまにAIに聞く」レベルではありません。彼は『猫でもわかる生成AI ー落合陽一に１００のプロンプトを入力してみたー』（扶桑社）という、タイトルからして100本のプロンプトを次々に打ち込む本を出しているほど、とにかく大量に投げる人です。研究者としても2017年頃から画像生成に取り組み、ファッション生成「Deep Yohji」や御朱印生成など、AIで作品を量産してきた"手数の人"でもあります。

その本人が、初心者へのアドバイスとしてこう言っています。「一発で『これだ！』と思える回答はほとんどないので、トライ＆エラーを重ねていくことが大切」。うまくいかなければ「もう少し簡単に説明してくれる？」と重ねればいい。さらに音声認識で「対話のようにテンポよく」「できるだけたくさんしゃべる」ことを勧めています。実際、彼はGoogleの「音声文字変換」アプリを愛用していて、「早口な自分の言葉でもテキパキ文字にしてくれる」と語っているほどで、キーボードで完璧な指示文を組む前に、まず口に出して量をこなす人です。対談動画では「多種多様なAIをたくさん使う」ことの重要性も繰り返しています。ここにある発想はシンプルで、うまいプロンプトを一発で当てにいくのではなく、複数のAIに量と往復で投げて精度を上げる、というものです。
