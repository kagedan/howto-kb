Claudeの使い方25選をまとめた。

25個全部覚えなくていい。
でも1つでも自分の作業にハマれば、
毎日の作業時間がかなり減ると思う。

Claudeって、
「質問するだけのAI」だと思ってたらかなりもったいない。

1つずつざっくり解説すると↓                                                          

① Chat
サクッと聞きたい時に使う。
「この文章どう？」「この投稿案どう？」みたいな軽い相談向き。

② Opus
深く考えさせたい時。
ブログの導線設計、占い商品のコンセプト作りみたいな重めの思考に使う。

③ Sonnet
速く返してほしい時。
要約、リライト、短文作成、誤字チェックに向いてる。

④ Extended Thinking
難しい問題をじっくり考えさせる時。
商品設計、売り方、投稿戦略の壁打ちに使いやすい。

⑤ Cowork
本気で作業する場所。
docx、xlsx、pptxなどを扱いながら、資料作りを進めたい時に便利。

⑥ Claude Code
コードやアプリ制作向き。
ブログの簡単なツール、LP、占い診断ページみたいなものを作る時に使える。

⑦ Claude in Excel
表の中でClaudeを使う。
記事ネタ管理、投稿分析、顧客メモ整理に使えそう。

⑧ Gamma connector
スライド作成向き。
1つのテーマから資料、セミナー台本、講座スライドに広げられる。

⑨ Claude Design
デザイン案を作る。
LPのたたき台、バナー構成、商品ページの見せ方を考える時に使える。

⑩ Claude in Chrome
ブラウザ上で使う。
見ているページを読ませて、要約や改善案を出せるのが強い。

⑪ Computer Use
ClaudeにPC操作を手伝わせる。
クリック、入力、ブラウザ操作みたいな作業の一部を任せるイメージ。

⑫ Dispatch
スマホから指示して、PC側で作業させる。
外出中に「これ調べて」「この作業進めて」に近い使い方。

⑬ Web Search
最新情報を調べる時。
ブログのリサーチや、占い系の暦・イベントネタ探しにも使える。

⑭ Research
複数の情報源をまとめる時。
深い記事、比較記事、講座資料、note作成に向いてる。

⑮ Skills
よく使う作業をスキル化する。
「この形式で投稿案を作る」みたいな作業を型にできる。

⑯ Plugins
機能を追加する。
マーケ、営業、資料作成など、目的別にClaudeを強化するイメージ。

⑰ Projects
案件やテーマごとに保存する。
ブログ用、占い用、LINE配信用みたいに分けると使いやすい。

⑱ Scheduled Tasks
定期作業を自動化する。
毎週のネタ出し、投稿案作成、レポート作成に使えそう。

⑲ Connectors
Gmail、Drive、Notion、カレンダーなどとつなぐ。
自分の情報をもとに作業させられるのが強い。

⑳ Artifacts
文章、表、図解、コードなどを画面横で形にできる。
ただの回答じゃなく、編集できる成果物になる。

㉑ Custom Styles
Claudeの口調を変える。
自分の発信のトーン、占い文体、キスケっぽい言い回しを固定しやすい。

㉒ Team Projects
チームで同じプロジェクトを使う。
外注さんや共同作業者とルールやファイルを共有しやすい。

㉓ Global Instructions
毎回守ってほしいルールを入れる。
「短く」「初心者向け」「煽りすぎない」みたいな指示を固定できる。

㉔ Obsidian Vault
メモや資料を整理する場所とつなげる。
過去のネタ、記事、鑑定文を資産として使いやすくなる。

㉕ About-me file
自分のルール、口調、仕事の前提をまとめるファイル。
毎回説明しなくても、自分らしい出力に近づけやすい。

25個全部やらなくていい。
まずは1つ試すだけでOK。

保存して、あとで見返す用にしよう。
