おはようございます☺  
昨日は朝一からAIの勉強会。  
実際にクロードコードを使ってできることとかを勉強してきました。

誘ってくれた富山県の古井先生、教えてくださった山中様、ありがとうございました☺

正直に言うと、ここ最近で一番「これはすごい」と感じています。

昨日は一日、AIエージェント「Claude Code」と向き合っていました。過去の会議録を横断的に調べたり、いただいたご要望を関係省庁への問い合わせ文にまとめたり、面談で出会った方の情報を整理したり。

これまで秘書業務の中で地道に、時間をかけてやっていた調査・起案作業が、対話しながらどんどん形になっていく。正直、鳥肌が立つ瞬間が何度もありました。

![](https://assets.st-note.com/img/1783022757-ruIWbo7henLwRV43dQpvmcPA.jpg?width=1200)

普通にclaudeを使うだけなら、チャッピーとかジェミニとかといっしょなのですが、この「code」ということがすごい。

要は読み込んだ資料をどんどん吸収して一緒に賢くなっていってくれる。

どういう風に資料を保存しておくのか、それをどのように活用するのかという、動かし方も私が決められる。  
ありとあらゆる資料を読み込ませまくって、それを取り出したい時に取り出せる。クローズドの空間。

今までいただいていた陳情もすべて保存しておいて、どの省庁に該当するのかも全部教えてくれる。

とまあ、こんなもんじゃなく、ここに書くのがもったいないほどの破壊力。

昨日は夜の会合まで、ずっとクロードコードと共闘していました。

まだまだ、賢くない私専用のAIクロードコード。  
これから一日一日賢くなっていきますよ。

これは、最初に始める時がちょっと難しいので、スタートラインに立つところまでは誰かに教えてもらったほうがいいかもしれませんが、そのあとは自分なりに簡単にアレンジし続けることができます。

毎日のSNSの更新の後、AIを育てる日常が追加されました☺子供に勉強を教えるみたいでとても楽しい。  
こんな最高のおもちゃ？を頂いて、昨日はずっとテンションマックスでした。久しぶりにこんなにテンション上がったかも。

子供のころ、こんな感じだったな～って思いだしながら。  
それくらいのわくわく感だったので。

このAIという技術が私たちの生活や仕事をどう変えるのか、自分で使いながら学んでみたいと思います。

本日はここまで。  
それでは本日もよろしくお願いいたします！  
今日もきっと、いいことがある。
