## TL;DR

- **エンジニア** = Claude Code（CLAUDE.md → Skills → MCP の3ステップ設定）
- **バックオフィス** = 5パイプライン（動画リパーパス／リード強化／競合インテル／PDF抽出／サポート返信）

## 1. エンジニアは Claude Code 一択（v0 / Lovable / Bolt は不可）

### なぜ v0・Lovable・Bolt ではダメか

それらが渡してくるのは「プロダクトに見えるプロトタイプ」。深夜2時に auth フローをリファクタしてくれない、テストスイートを回さない、3ヶ月前に放置した機能を「なぜ放置したか」と一緒に拾い直さない。Claude Code はそれをやる。ただしデモのようには動かない。**周囲のワークスペースを自分で組む**必要がある。

### 3ステップ・セットアップ（一度だけ）

#### Step 1: `CLAUDE.md` を書く

リポジトリ直下に Markdown ファイルを置き、テックリードがシニア入社初日に渡す規約をそのまま書く。

```markdown
# CLAUDE.md

## Codebase
- DB は PostgreSQL 16（接続情報は `.env`）
- 認証は Auth.js v5
- PR前に `pnpm test` 必須

## Don't touch without asking
- `migrations/` 配下
- `infra/terraform/` 配下
```

#### Step 2: Skills をビルド

1つのタスクをエンドツーエンドで遂行する短い Markdown を `.claude/skills/` に置く。

- `ship-feature.md` ：PR を出す前のチェックリスト
- `triage-bug.md` ：本番障害時の手順
- `migrate-schema.md` ：データロスを避ける7つの教訓

```markdown
# .claude/skills/ship-feature.md

## Trigger
機能実装が完了したとき

## Steps
1. `pnpm test` を全パスさせる
2. Storybook を更新
3. CHANGELOG.md に追記
4. PR本文を SQUASH 済の意図で書く
```

#### Step 3: MCP（Model Context Protocol）を繋ぐ

```jsonc
// .mcp.json
{
  "mcpServers": {
    "github": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-github"] },
    "postgres": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-postgres", "$DATABASE_URL"] },
    "slack": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-slack"] }
  }
}
```

→ コードベースのコンテキスト・反復可能なスキル・実ツールアクセス、**3点が揃って初めて「採用」になる**。

## 2. バックオフィスは5つのパイプラインで置き換える

| # | パイプライン | 置き換える仕事 |
|---|---|---|
| 1 | YouTube → 10投稿に展開 | コンテンツエディタ |
| 2 | 新CRMリード → 初回メール自動下書き | SDR |
| 3 | 競合サイト巡回 → ブリーフ生成 | ジュニアアナリスト |
| 4 | PDF請求書 → Xero へクリーンレコード | 経理 |
| 5 | 知識ベース参照 → サポート返信ドラフト | カスタマーサポート |

全て週末で組める。**パイプライン = Skills**、**統合 = MCP**、**オーケストレーション = Claude Code**。

## 3. マーケは Higgsfield Marketing Studio に集約

### 6ステップ

```text
1. プロダクトURLを貼る（8秒）
2. AIペルソナを選ぶ／自分の顔をアップロード
3. Hermes Agent が Meta Ads Library と TikTok Creative Center をスクレイピング
4. Seedance 2.0 が広告動画を自動生成（9:16 / 1:1 / 16:9）
5. 1日500本以上のad-readyカット
6. ループが閉じる（勝ちフック自動学習）
```

### リサーチ層を上流に重ねる（中級者向け）

Higgsfield のカテゴリ平均リサーチに頼ると平均的な広告しか出ない。**Hermes Agent (Nous Research)** または **OpenClaw** をローカルで動かしてカスタムリサーチを上流に注入する。

```bash
# Hermes Agent を1コマンドで導入
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

3つのリサーチジョブを設定すれば即戦力：

1. 毎週月曜朝、Meta Ads Library のカテゴリ上位50広告をスクレイプ→フック角度でクラスタリング→Telegramへ
2. X / Reddit で5,000インプレッション以上の自分のニッチ投稿を要約→フック候補に
3. 自分のサポートチケットを毎週読み、顧客が痛みを語る頻出フレーズ3つを抽出
