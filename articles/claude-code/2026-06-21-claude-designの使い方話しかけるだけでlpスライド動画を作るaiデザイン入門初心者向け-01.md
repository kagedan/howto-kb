## まず結論：Claude Designは「完成品」ではなく「たたき台作成」に使うと強い

![](https://assets.st-note.com/img/1782001770-wBCchJOreSQaq4dK78jUW9sv.png?width=1200)

Claude Designを使うときに大事なのは、最初から完璧な完成品を求めないことです。

おすすめは、

**AIで8割のたたき台を作り、人間が最後の2割を整える**

という使い方です。

Claude Designは、ゼロから考える時間を減らすのに向いています。

たとえば、

* LPの構成を作る
* スライドの見た目を整える
* サービス紹介ページを作る
* 動画の流れを作る
* 資料の雰囲気をそろえる
* PowerPointに書き出して修正する

といった作業です。

デザインが苦手な人ほど、まずは「完成品」ではなく「たたき台づくり」と考えると使いやすいです。

## Claude Designでできること｜LP・スライド・資料・動画まで作れる

![](https://assets.st-note.com/img/1782001786-D6Qufx2wPM5RBcOkFCAgb1UJ.png?width=1200)

Claude Designを開くと、作成タイプを選べます。

主な選択肢は次のようなものです。

* Slides
* Product prototype
* Product wireframe
* Document
* Animation
* Blank canvas

普通の会社員向けに言い換えると、こうです。

* **Slides**：説明資料やプレゼン資料を作る
* **Product prototype**：アプリやサービスの試作品を作る
* **Product wireframe**：画面やページの骨組みを作る
* **Document**：レポートやPDF向け資料を作る
* **Animation**：説明動画やアニメーションを作る
* **Blank canvas**：白紙から自由に作る

つまりClaude Designは、文章を作るだけではありません。

**見せるための成果物を作る場所**として使えます。

## Mark upを使うと、画面を見ながら修正指示が出せる

![](https://assets.st-note.com/img/1782001878-Y6c07HtGSKpVxPUfFz9yvWqg.png?width=1200)

Claude Designで便利なのが、**Mark up** です。

画面を見ながら、

* ここを消したい
* ここを短くしたい
* ここを移動したい
* この要素はいらない
* 全体をもっと詰めたい

のように指示できます。

普通のチャットだけだと、どの部分を直したいのか伝えにくいことがあります。

でもMark upなら、見ている場所に対して直接指示できます。

これは、資料やLPを直すときにかなり便利です。

## Claude Designでスライドを作る方法｜8枚構成から始めると簡単

![](https://assets.st-note.com/img/1782001928-UT27vwB0N8my5kiSeOcFfuHA.png?width=1200)

普通の会社員にとって、一番実用的なのは**スライド作成**かもしれません。

Claude Designでは、テーマを伝えるだけで、スライド構成を作れます。

おすすめは、最初から20枚作るのではなく、**8枚構成**で作ることです。

8枚なら、

1. タイトル
2. 背景
3. できること
4. 活用例
5. 使い方
6. 注意点
7. メリット
8. 今日やること

くらいに整理しやすいです。

## Claude Codeで仕上げると、AIっぽさを減らせる

![](https://assets.st-note.com/img/1782001999-mG9jBAKoHzXfsDRNWnI43MQa.png?width=1200)

Claude Designで作ったものは、そのままでも見栄えします。

ただ、実務で使うなら、最後に整える必要があります。

そこでClaude Codeを使うと、

* 差分を見ながら直す
* レイアウトを調整する
* 文言を短くする
* 動きを足す
* スマホ表示を確認する
* 不要な要素を削る

といった仕上げができます。

大事なのは、

**生成しっぱなしにしないこと**

です。

AIで作ったものを、人間の目的に合わせて整えることで、かなり実務レベルに近づきます。

## コピペテンプレ：Claude Codeに仕上げを頼む

```
Claude Designで作成した成果物を、Claude Codeで仕上げたいです。

目的：
AIっぽさを減らして、仕事で使いやすい形に整えること

見てほしい点：
1. 文字量が多すぎる箇所
2. 見出しが抽象的な箇所
3. レイアウトの不自然な箇所
4. スマホ表示で崩れそうな箇所
5. 動きや演出を加えた方がよい箇所
6. 実務向けに直した方がよい表現

ルール：
- いきなり大きく変更しない
- まず改善方針を出す
- 変更箇所と理由を説明する
- 元の構成はできるだけ活かす
```
