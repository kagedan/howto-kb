## 今日の注目AIニュース

### 1. OpenAIがCodex利用調査を公開。AIは“質問相手”から“仕事を任せる相手”へ進む

OpenAIは、Codexの利用データをもとに、AIエージェントが仕事をどう変えているかを公開しました。

ポイントは、Codexが短い質問回答だけではなく、30分以上、1時間以上に相当する長めの作業にも使われ始めていることです。

しかも、開発者だけではありません。法務、採用、財務などの非エンジニア部門でも、調査、整理、自動化、資料化のような仕事に使われている流れが見えます。

AI副業や業務改善で見るなら、これから価値になるのは「いいプロンプト」だけではなさそうです。

どの作業をAIに任せるか。どこで人間が確認するか。ログをどう残すか。ここまで設計できる人が強くなります。

詳しくはこちら:  
<https://diginoteai.com/article/openai-codex-agents-transform-work-delegation>

### 2. Claude Code v2.1.193。自動モードとバックグラウンド作業が、チーム運用向けに整う

Anthropicは、Claude Code v2.1.193を公開しました。

今回の更新では、シェルコマンドを自動モード分類に通す設定、拒否理由の記録、MCP認証の案内、バックグラウンドシェルの整理などが含まれています。

少し地味に見えますが、AIを仕事場で使うならかなり大事です。

AIにファイル編集やコマンド実行を任せるほど、「勝手に進めてよい操作なのか」「なぜ止まったのか」「どの接続先の認証が必要なのか」を見えるようにする必要があります。

AI作業場を作るなら、便利さだけでなく、権限、ログ、拒否理由、認証の設計まで含めて考える時期に入ってきています。

詳しくはこちら:  
<https://diginoteai.com/article/anthropic-claude-code-v2-1-193-auto-mode-background-workflow>

### 3. Genspark DesignがNew表示。アイデアからUI、動画、HTMLアニメまで制作導線が短くなる

Genspark公式トップでは、Genspark Designが案内されています。

Claude Opus 4.7を使い、ラフなアイデアからUIプロトタイプ、動画、HTMLアニメーション、ポスターなどを作れるデザインAIとして紹介されています。

これは、画像を1枚作る話だけではありません。

LPのラフ案、アプリ画面、SNS告知、動画のたたき台、提案資料の見本まで、制作前の「見える化」に使いやすい流れです。

AI副業やWeb制作で見るなら、完成品をいきなり作るというより、顧客と方向性を合わせるためのたたき台作成に使いやすそうです。

詳しくはこちら:  
<https://diginoteai.com/article/genspark-design-ui-prototype-code-workflow>
