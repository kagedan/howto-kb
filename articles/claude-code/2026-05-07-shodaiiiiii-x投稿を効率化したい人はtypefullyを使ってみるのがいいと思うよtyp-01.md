X投稿を効率化したい人は「Typefully」を使ってみるのがいいと思うよ。Typefullyは無料でAPI/MCP使えて、Claude codeやCodexから直接下書きやスケジュール登録ができちゃうし、無料プランでも月15ポストまでできるのでお試しに最適。有料プランも8ドルで良心的なのも良い。
