「またこれか」と思いながらやる作業、ありませんか。

別のシートから数字を拾ってきて、別の表に貼って、並べ替えて、合計を確認して……。間違えると上から怒られる。でも作業自体は頭を使わない。毎月2〜3時間が、これだけで消えている。

そういう作業、Claudeに渡せます。**データを貼り付けるだけで、整形・転記・集計が終わった状態で返ってきます。**

## Before / After

**Before：** 複数のExcelを行き来しながら手でコピペ転記 → 毎月2〜3時間消える → 転記ミスで上から怒られる

**After：** データを貼り付けるだけ → 15〜30分で完了 → **90%削減**

## 準備するもの

- Claude Code（インストール済みであること）   
- 転記したいデータ（ExcelでもCSVでもテキストでもOK）

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：データをテキストでコピーする

ExcelのデータをCtrl+Cでコピー、メモ帳などに貼り付けるとタブ区切りのテキストになります。

```
名前	金額	日付
田中	15000	4/1
鈴木	22000	4/3
佐藤	8000	4/5
```

これをそのままClaude Codeに渡せます。

![](https://assets.st-note.com/img/1776991048-1Gw8FqxrXuRY0sQLTWZc6O9t.png?width=1200)

### ステップ2：「どうしたいか」を一緒に渡す

```
以下のデータを集計してください。
・担当者別の合計金額を出してください
・金額の高い順に並べてください
・合計行を一番下に追加してください

[データをここに貼る]
```

![](https://assets.st-note.com/img/1776991056-5zYK3uL6eSnCNrjZ9iQc2d8R.png?width=1200)

### ステップ3：結果を確認してExcelに貼る

Claudeが整形・集計した結果を出します。それをコピーしてExcelに貼るだけ。

![](https://assets.st-note.com/img/1776991064-7CLE6pDaAThNdIxS2qbJr9Bw.png?width=1200)

### ステップ4：来月も同じ作業が来たら、また貼るだけ

設定なしでも毎回同じように使えます。ただ、毎回「担当者別に」「合計行を」と説明するのは手間です。

## できました。ただ、このままだと1つ問題があります。

毎回「こう整形してください」「担当者別にまとめてください」と説明するのは手間です。来月も、再来月も、毎回同じ説明。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、毎月の転記ルールを覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎月の転記作業が変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「担当者別に集計して」と説明が必要   
✅ ファイルあり：自動で担当者別に集計される

❌ 設定なし：毎回「合計行を追加して」と言わないといけない   
✅ ファイルあり：毎回自動で合計行が入る

❌ 設定なし：毎回「数値はカンマなしで」と指定が必要   
✅ ファイルあり：常にExcel貼り付け用の形式で返ってくる

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「月次転記の憂鬱」が消えます。**

## 100円で何が変わるか

毎月2〜3時間 × 12ヶ月 = **年間最大36時間がコピペ転記に消えています。**

設定ファイル1枚で、来月からその36時間が戻ってきます。

**100円 ÷ 36時間 = 1時間あたり約3円。** これより安く時間を買う方法は、なかなかありません。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-004（添付）**　←　ダウンロードして置くだけ   
- **よくある転記パターン5種の設定例**　←　担当者別集計・日付並べ替え・重複削除・空白補完・リスト照合   
- **「ズレる・消える・増える」エラー対処集**　←　転記ミスの原因はだいたい3パターンに絞られる

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「どんなデータを処理しますか？」→ 答える   
- 「毎月どんな形式で出力していますか？」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたの転記スタイルを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
