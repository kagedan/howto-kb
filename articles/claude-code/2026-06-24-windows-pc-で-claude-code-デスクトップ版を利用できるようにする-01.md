# 💬 概要

本記事では、プログラミング未経験の方でもスムーズに開発を始められるように、**Claude Code デスクトップ版**のセットアップ手順をまとめています。

---

# 🏁 ゴール

> **Windows PC で働く非エンジニアが、Claude Code を使って業務効率化ツールを開発できるように、Claude Code デスクトップ版をセットアップした状態にする。**

---

# 💡 前提知識

## ❓ Claude Code とは

**Claude Code は、あなたの PC 上で実際に作業をしてくれる AI アシスタント**です。ファイルを読み書きしたり、データを整理したり、簡単なツールを作って動かしたりできます。

普段ニュースなどで耳にする「AI に相談する」イメージよりも一歩進んでいて、**「相談相手」ではなく「手を動かしてくれる作業担当」** だと考えるとイメージしやすいです。

非エンジニアの方でも、たとえば次のような業務効率化に使えます。

* Excel の定型集計やレポート作成を自動化する
* 大量のファイルやデータをまとめて整理・名前変更する
* 日々の繰り返し作業を「ツール化」して、ボタンひとつで終わるようにする
* 簡単な入力フォームや変換ツールを作る

## 🆚 ふだんの Claude（claude.ai のチャット）との違い

「Claude」と「Claude Code」は名前が似ていますが、できることが違います。

|  | ふだんの Claude（チャット） | Claude Code |
| --- | --- | --- |
| 主な使い方 | 文章で相談・質問・文章作成 | PC 上で実際に作業（ファイル操作・ツール作成など） |
| あなたの PC のファイル | 触れない | 直接読み書きできる |
| 成果物の使い方 | 自分でコピー＆ペーストして使う | Claude が PC 上に直接作ってくれる |
| たとえると | 物知りな相談相手 | 手を動かしてくれるアシスタント |

## 📌 3つの利用方法と、今回の選択

実は Claude Code を利用する方法は 3 つあります。

| 利用方法 | どんな画面か | 向いている人 |
| --- | --- | --- |
| **CLI（コマンドライン）** | 黒い画面に文字でコマンドを打ち込む「ターミナル」を使う | エンジニア／ターミナルに慣れた人 |
| **VS Code 拡張機能** | プログラマー向けエディタ「VS Code」に組み込んで使う | 普段から VS Code を使う人 |
| **デスクトップ版（専用アプリ）** | ボタン中心の分かりやすい画面。**ターミナル不要** | 初めての人・非エンジニア |

**今回は非エンジニアの初心者の方が対象なので、ターミナルを使わずに直感的に操作できる「デスクトップ版」** を使います。

# 💻 セットアップ手順

## STEP 0. 有料プランとアカウントを確認する

まず、サインインに使う **Anthropic アカウント**と、そのアカウントに**有料プランが紐づいているか**を確認します。

* 会社からアカウントが支給される場合は、**社内の案内に従ってアカウント情報を受け取って**ください。
* 個人で用意する場合は、有料プラン（Pro など）への加入が必要です。

## STEP 1. Claude デスクトップアプリをインストールする

Claude Code は「**Claude**」という名前のデスクトップアプリに含まれています。まずはこのアプリをインストールします。

1. 下記の公式ページを開きます。  
   👉 <https://code.claude.com/docs/ja/desktop-quickstart>
2. ページ内の「**インストール**」セクションにある **Windows 版のダウンロードボタン**をクリックします。
   * 一般的な PC はこちら（x64 用）でOKです。
   * **ARM64 の PC**（Snapdragon 搭載など一部の新しいノート PC）の場合は、ページ内の ARM64 用リンクからダウンロードしてください。
3. ダウンロードした**インストーラー（.exe ファイル）をダブルクリック**し、画面の指示に従ってインストールします。

💡 自分の PC が x64 か ARM64 か分からないとき

「設定」アプリ→「システム」→「バージョン情報」を開き、「**システムの種類**」を確認します。  
「x64 ベース」と書いてあれば x64、「ARM ベース」と書いてあれば ARM64 です。ほとんどの PC は x64 です。

## STEP 2. サインインする

1. インストールが終わったら、**スタートメニューから「Claude」を起動**します。
2. **STEP 0 で確認した Anthropic アカウントでサインイン**します。
   * ブラウザが開いてログインを求められる場合があります。画面の案内に従って完了してください。

## STEP 3. 「Code」タブを開く

Claude アプリには「**Chat**」「**Cowork**」「**Code**」の 3 つのタブがあります。研修で使うのは「**Code**」タブです。

1. 画面**上部中央の「Code」タブ**をクリックします。
2. 正しく開けば、プロンプト（指示）を入力する画面が表示されます。

> 🔍 **このとき表示されるかもしれないメッセージ**
>
> * **「アップグレードしてください」と表示された場合** → 有料プランが必要です。STEP 0 に戻り、プランとアカウントを確認してください。
> * **サインインを促された場合** → サインインを完了し、アプリを再起動してください。
> * **「403」エラーが出た場合** → いったんサインアウト → サインインし直し、アプリを再起動してください。

## STEP 4. Git をインストールする

**Git（ギット）** は、PC 上でファイルの変更履歴を管理するための定番ツールです。**Windows で Claude Code をローカル（自分の PC 上）で使うには Git が必要**なので、ここでインストールします。

> 📝 「Code」タブで自分の PC のフォルダを選ぼうとしたときに「Git が必要です」と案内されることがあります。先に入れておけばスムーズです。

1. 下記の公式ダウンロードページを開きます。  
   👉 <https://gitforwindows.org/>
2. **Windows 用のインストーラー**をダウンロードします。
3. ダウンロードした **.exe をダブルクリック**し、インストールを進めます。
   * 設定項目がいろいろ表示されますが、**基本的にすべて初期設定のまま「Next」を押し続けて OK** です。最後に「Install」「Finish」を押せば完了です。
4. インストールしたら、Claude デスクトップを再起動してください。

## STEP 5. 開発言語（Python / Node.js）をインストールする

業務効率化ツールの多くは、**Python（パイソン）** や **Node.js（ノード）** という「プログラムを動かすための土台（言語の実行環境）」の上で動きます。これらをインストールしておくと、Claude Code が作ったツールを実際に動かせるようになります。

うれしいことに、**このインストール作業自体も Claude Code に手伝ってもらえます**。

1. 「**Code**」タブを開いた状態にします。
2. 画面下のほうで、実行環境として「**Local**（自分の PC 上で動かす）」を選びます。
3. 「**Select folder**」をクリックして、**作業用のフォルダ**を選びます。

   * まだ無ければ、デスクトップなどに「`claude-practice`」のような分かりやすい名前のフォルダを 1 つ作って、それを選んでください。
4. プロンプト（指示入力欄）に、次のように入力して送信します。

   ```
   Python と Node.js をこの PC にインストールして、コマンドで実行できる状態にしてください。
   インストールが終わったら、python --version と node --version でバージョンを確認したいです。
   ```
5. Claude Code が**実行しようとする操作（コマンド）を提案**してきます。内容を確認して、**「Accept（承認）」や「許可」を押して進めます**。

   * 初期設定では、Claude は何かを実行する前に必ずあなたの承認を求めます。安心して進めてください。
6. インストールが完了し、`python --version` と `node --version` でバージョン番号（例：`Python 3.x.x` / `v20.x.x` など）が表示されれば成功です。

💡 うまく確認できないとき

インストール直後はうまく認識されないことがあります。その場合は **Claude アプリ（または PC 本体）を一度再起動**してから、もう一度「Code」タブで `python --version` と `node --version` を確認してください。

再起動しても確認できなかった場合は、公式インストーラーから手動でインストールしてください。

## STEP 6. 動作確認（仕上げ）

最後に、ひととおり動くかを確認します。「Code」タブで、作業用フォルダを選んだ状態で次のように入力してみましょう。

```
このフォルダに「hello.txt」というファイルを作成して、中に「セットアップ完了！」と書いてください。
```

提案された操作を承認し、**フォルダの中に `hello.txt` が作られ、中身に文字が入っていれば、セットアップは無事完了**です。お疲れさまでした 🎉

# 🆘 うまくいかないときは

| 症状 | 対処法 |
| --- | --- |
| 「Code」タブでアップグレードを求められる | 有料プランが必要です。社内の担当者に確認し、有料プランのアカウントでサインインしてください。 |
| サインインできない／403 エラーが出る | サインアウト → サインインし直し → アプリを再起動してください。 |
| フォルダを選ぶと「Git が必要」と出る | STEP 4 に戻って Git をインストールし、アプリを再起動してください。 |
| `python` や `node` のバージョンが表示されない | Claude アプリ、または PC 本体を再起動してから、もう一度確認してください。 |
| ダウンロードしたファイルがどれか分からない | 通常は「ダウンロード」フォルダに保存されています。エクスプローラー（フォルダのアイコン）から探してください。 |
| インストールがブロックされる／管理者権限を求められる | 会社支給 PC の場合は権限制限の可能性があります。社内の担当者または情報システム担当へご相談ください。 |
