はじめに
Xでいいねを押すという行為は、自分の情報感度のログを残すことに近いと思っています。気になった記事、覚えておきたい投稿、あとで読み返したい発言。そういったものが日々タイムラインに流れ、「いいね」として小さく記録されていきます。
ただ、このログはそのままだとけっこう死蔵されがちです。「あのいいね、どこだっけ」と探すことはあっても、体系的に読み返したり、自分の仕事や判断に結びつけて意味付けしたりする機会は意外と少ないのではないでしょうか。1日に数件〜数十件のいいねが、ただフォルダに積まれていくだけの状態になります。
