### 背景

会社でClaude Codeを利用できることになり、1週間ほど利用している。  
データ分析やデータマート作成のコードでAIを利用していた。ただ、最近はベンダーさんと一緒に仕事をすることが多く、ベンダーさんへの依頼や取りまとめをメインに行っている関係上、以前ほど利用していなかった。  
そんな中、Claude Codeを使って開発を進めることとなり自分でも利用してみようと思い、WBSや自分の作業でClaude Codeを使ってみている。

### 実感

とても便利に使えていて、私のような怠け者にはいいツールだと思う。  
今やっていることを全て書き出し、タスクmdファイルを作ってもらう。それに対して、更新をかけながらToDo管理やWBSの更新を行うのが基本的な使い方だ。  
自分のやっている個別タスクについても、個別タスクx.mdのようなファイルを作ってそこでやり取りしている。  
Claude Codeが賢いので、タスクと個別タスクを関連付けて、更新してくれている。  
やる気がなくて、仕事の初速が重い人には仕組みを整えればかなり有用なツールだと実感している。

### これから

これからの自分のバリューは何になるのか。大きくは2つになると考えている。  
①ビジョン  
「大きなビジョンを描きある程度のタスク粒度に落とす」、という従前は役員や部長がやっていたことを一般社員がやる必要が出てくる。なぜその仕事を行うか、何をしたいかという点から始める仕事を突き詰めていく必要がある。  
②細かい部分  
指示をするには設計をしっかり行わないといけない。これは細部を突き詰めていくことから身につけられるようなものであって、細かい仕事をしている人はより価値ある仕事ができると考えている。

うーん、楽しい時代だ
