## はじめに

もう各所で記事が出ていて今さら感はありますが、自分なりにカスタマイズしてみたら意外と楽しかったので、せっかくなので共有してみます。

* 「コンテキストウィンドウ、あとどれくらい残ってるんだろう」
* 「レートリミット、そろそろ引っかかりそう…」
* 「今どのブランチで作業してたっけ」

デフォルトの Claude Code には、ターミナル下部にステータスラインを表示する機能があります。ただし、デフォルト状態では最低限の情報しか出ません。

実は `settings.json` の `statusLine` にカスタムコマンドを指定すると、**自分の好きな情報を自由に表示できます**。今回はこの機能を使って、以下を表示するシェルスクリプトを作りました。

* モデル名
* コンテキストウィンドウ使用率（%）
* セッション経過時間
* セッション名（設定している場合）
* thinking モードの ON/OFF
* 作業ディレクトリ名と Git ブランチ
* Git worktree 名
* **5時間・7日間のレートリミット使用率バー + リセット時刻**
* **Extra Usage の使用状況**（有効な場合）

## 表示している状態はこんな感じです。

## セットアップ

### 1. スクリプトを配置する

後述するスクリプトを `~/.claude/statusline.sh` として保存し、実行権限を付与します。

```
chmod +x ~/.claude/statusline.sh
```

### 2. settings.json に登録する

`~/.claude/settings.json` に以下を追加します。

```
{
  "statusLine": {
    "type": "command",
    "command": "~/.claude/statusline.sh"
  }
}
```

### 3. 依存ツールの確認

スクリプトは以下のコマンドを使用します。macOS なら標準で揃っています。

| ツール | 用途 |
| --- | --- |
| `jq` | JSON パース |
| `curl` | Usage API 呼び出し |
| `git` | ブランチ・worktree 情報 |
| `tput` | ターミナル幅の取得 |
| `security` | macOS Keychain からトークン取得 |

`jq` だけ手動インストールが必要な場合があります。

```
# macOS
brew install jq

# Ubuntu/Debian
sudo apt install jq
```

## コードのポイント解説

### Claude Code から渡される JSON

Claude Code はステータスラインコマンドの **stdin に JSON を渡してきます**。主要なフィールドはこちら。

```
{
  "model": {
    "display_name": "claude-sonnet-4-5"
  },
  "context_window": {
    "context_window_size": 200000,
    "current_usage": {
      "input_tokens": 12345,
      "cache_creation_input_tokens": 5000,
      "cache_read_input_tokens": 8000
    }
  },
  "cost": {
    "total_duration_ms": 2520000
  },
  "session_name": "my-task",
  "cwd": "/Users/you/my-project"
}
```

`jq` でこれを取り出すだけで大半の情報が得られます。

### コンテキストウィンドウ使用率

実際のトークン消費量は 3 種類の合計です。

```
input_tokens=$(echo "$input" | jq -r '.context_window.current_usage.input_tokens // 0')
cache_create=$(echo "$input" | jq -r '.context_window.current_usage.cache_creation_input_tokens // 0')
cache_read=$(echo "$input" | jq -r '.context_window.current_usage.cache_read_input_tokens // 0')
current=$(( input_tokens + cache_create + cache_read ))
pct_used=$(( current * 100 / size ))
```

キャッシュのトークンも含めないと、実際より低い値になるので注意です。

### カラー付きプログレスバー

使用率に応じて色を変えながら `●○` でバーを描画します。

```
build_bar() {
    local pct=$1
    local width=$2
    local filled=$(( pct * width / 100 ))
    local empty=$(( width - filled ))
    local bar_color
    bar_color=$(color_for_pct "$pct")

    local filled_str="" empty_str=""
    for ((i=0; i<filled; i++)); do filled_str+="●"; done
    for ((i=0; i<empty; i++)); do empty_str+="○"; done

    printf "${bar_color}${filled_str}${dim}${empty_str}${reset}"
}
```

### OAuth トークンの取得

Usage API を叩くには Claude Code の OAuth トークンが必要です。スクリプトは以下の順に探索します。

1. 環境変数 `CLAUDE_CODE_OAUTH_TOKEN`
2. macOS Keychain（`security find-generic-password`）
3. `~/.claude/.credentials.json`
4. Linux の `secret-tool`

```
get_oauth_token() {
    # 1. 環境変数
    if [ -n "$CLAUDE_CODE_OAUTH_TOKEN" ]; then
        echo "$CLAUDE_CODE_OAUTH_TOKEN"; return 0
    fi

    # 2. macOS Keychain
    if command -v security >/dev/null 2>&1; then
        local blob
        blob=$(security find-generic-password -s "Claude Code-credentials" -w 2>/dev/null)
        if [ -n "$blob" ]; then
            token=$(echo "$blob" | jq -r '.claudeAiOauth.accessToken // empty' 2>/dev/null)
            [ -n "$token" ] && echo "$token" && return 0
        fi
    fi

    # 3. credentials.json
    local creds_file="${HOME}/.claude/.credentials.json"
    if [ -f "$creds_file" ]; then
        token=$(jq -r '.claudeAiOauth.accessToken // empty' "$creds_file" 2>/dev/null)
        [ -n "$token" ] && echo "$token" && return 0
    fi

    echo ""
}
```

### 60秒キャッシュ

Usage API を毎回叩くとレート制限を消耗するので、`/tmp/claude/statusline-usage-cache.json` にキャッシュします。60秒以内なら API を叩かず、キャッシュを使います。

```
cache_file="/tmp/claude/statusline-usage-cache.json"
cache_max_age=60

if [ -f "$cache_file" ]; then
    cache_mtime=$(stat -c %Y "$cache_file" 2>/dev/null || stat -f %m "$cache_file" 2>/dev/null)
    now=$(date +%s)
    cache_age=$(( now - cache_mtime ))
    if [ "$cache_age" -lt "$cache_max_age" ]; then
        needs_refresh=false
        usage_data=$(cat "$cache_file")
    fi
fi
```

`stat` のオプションが macOS（`-f %m`）と Linux（`-c %Y`）で異なるため、両方試しています。

### macOS / Linux の差異吸収

`date` コマンドの ISO 8601 パースも OS によって異なります。Linux は `-d`、macOS は `-j -f` フラグを使います。スクリプトでは両方を試してどちらかが成功した値を使います。

```
iso_to_epoch() {
    local iso_str="$1"
    # Linux
    local epoch
    epoch=$(date -d "${iso_str}" +%s 2>/dev/null)
    [ -n "$epoch" ] && echo "$epoch" && return 0

    # macOS
    local stripped="${iso_str%%.*}"
    epoch=$(env TZ=UTC date -j -f "%Y-%m-%dT%H:%M:%S" "$stripped" +%s 2>/dev/null)
    [ -n "$epoch" ] && echo "$epoch" && return 0

    return 1
}
```

## スクリプト全文

statusline.sh（クリックで展開）

```
#!/bin/bash
set -f

input=$(cat)

if [ -z "$input" ]; then
    printf "Claude"
    exit 0
fi

# ── Colors ──────────────────────────────────────────────
blue='\033[38;2;0;153;255m'
orange='\033[38;2;255;176;85m'
green='\033[38;2;0;175;80m'
cyan='\033[38;2;86;182;194m'
red='\033[38;2;255;85;85m'
yellow='\033[38;2;230;200;0m'
white='\033[38;2;220;220;220m'
magenta='\033[38;2;180;140;255m'
dim='\033[2m'
reset='\033[0m'

sep=" ${dim}│${reset} "

# ── Helpers ─────────────────────────────────────────────
format_tokens() {
    local num=$1
    if [ "$num" -ge 1000000 ]; then
        awk "BEGIN {printf \"%.1fm\", $num / 1000000}"
    elif [ "$num" -ge 1000 ]; then
        awk "BEGIN {printf \"%.0fk\", $num / 1000}"
    else
        printf "%d" "$num"
    fi
}

color_for_pct() {
    local pct=$1
    if [ "$pct" -ge 90 ]; then printf "$red"
    elif [ "$pct" -ge 70 ]; then printf "$yellow"
    elif [ "$pct" -ge 50 ]; then printf "$orange"
    else printf "$green"
    fi
}

build_bar() {
    local pct=$1
    local width=$2
    [ "$pct" -lt 0 ] 2>/dev/null && pct=0
    [ "$pct" -gt 100 ] 2>/dev/null && pct=100

    local filled=$(( pct * width / 100 ))
    local empty=$(( width - filled ))
    local bar_color
    bar_color=$(color_for_pct "$pct")

    local filled_str="" empty_str=""
    for ((i=0; i<filled; i++)); do filled_str+="●"; done
    for ((i=0; i<empty; i++)); do empty_str+="○"; done

    printf "${bar_color}${filled_str}${dim}${empty_str}${reset}"
}

iso_to_epoch() {
    local iso_str="$1"

    local epoch
    epoch=$(date -d "${iso_str}" +%s 2>/dev/null)
    if [ -n "$epoch" ]; then
        echo "$epoch"
        return 0
    fi

    local stripped="${iso_str%%.*}"
    stripped="${stripped%%Z}"
    stripped="${stripped%%+*}"
    stripped="${stripped%%-[0-9][0-9]:[0-9][0-9]}"

    if [[ "$iso_str" == *"Z"* ]] || [[ "$iso_str" == *"+00:00"* ]] || [[ "$iso_str" == *"-00:00"* ]]; then
        epoch=$(env TZ=UTC date -j -f "%Y-%m-%dT%H:%M:%S" "$stripped" +%s 2>/dev/null)
    else
        epoch=$(date -j -f "%Y-%m-%dT%H:%M:%S" "$stripped" +%s 2>/dev/null)
    fi

    if [ -n "$epoch" ]; then
        echo "$epoch"
        return 0
    fi

    return 1
}

format_reset_time() {
    local iso_str="$1"
    local style="$2"
    [ -z "$iso_str" ] || [ "$iso_str" = "null" ] && return

    local epoch
    epoch=$(iso_to_epoch "$iso_str")
    [ -z "$epoch" ] && return

    case "$style" in
        time)
            LC_TIME=C date -j -r "$epoch" +"%m/%d %l:%M%p" 2>/dev/null | sed 's/^ *//; s/\.//g; s|^0||; s|/0|/|' | tr '[:upper:]' '[:lower:]' || \
            LC_TIME=C date -d "@$epoch" +"%m/%d %l:%M%P" 2>/dev/null | sed 's/^ *//; s/\.//g; s|^0||; s|/0|/|'
            ;;
        datetime)
            LC_TIME=C date -j -r "$epoch" +"%m/%d %l:%M%p" 2>/dev/null | sed 's/^ *//; s/\.//g; s|^0||; s|/0|/|' | tr '[:upper:]' '[:lower:]' || \
            LC_TIME=C date -d "@$epoch" +"%m/%d %l:%M%P" 2>/dev/null | sed 's/^ *//; s/\.//g; s|^0||; s|/0|/|'
            ;;
        *)
            LC_TIME=C date -j -r "$epoch" +"%m/%d" 2>/dev/null | sed 's|^0||; s|/0|/|' || \
            LC_TIME=C date -d "@$epoch" +"%m/%d" 2>/dev/null | sed 's|^0||; s|/0|/|'
            ;;
    esac
}

# ── Extract JSON data ───────────────────────────────────
model_name=$(echo "$input" | jq -r '.model.display_name // "Claude"')

size=$(echo "$input" | jq -r '.context_window.context_window_size // 200000')
[ "$size" -eq 0 ] 2>/dev/null && size=200000

input_tokens=$(echo "$input" | jq -r '.context_window.current_usage.input_tokens // 0')
cache_create=$(echo "$input" | jq -r '.context_window.current_usage.cache_creation_input_tokens // 0')
cache_read=$(echo "$input" | jq -r '.context_window.current_usage.cache_read_input_tokens // 0')
current=$(( input_tokens + cache_create + cache_read ))

used_tokens=$(format_tokens $current)
total_tokens=$(format_tokens $size)

if [ "$size" -gt 0 ]; then
    pct_used=$(( current * 100 / size ))
else
    pct_used=0
fi

thinking_on=false
settings_path="$HOME/.claude/settings.json"
if [ -f "$settings_path" ]; then
    thinking_val=$(jq -r '.alwaysThinkingEnabled // false' "$settings_path" 2>/dev/null)
    [ "$thinking_val" = "true" ] && thinking_on=true
fi

# ── Common ─────────────────────────────────────────────
cwd=$(echo "$input" | jq -r '.cwd // ""')
[ -z "$cwd" ] || [ "$cwd" = "null" ] && cwd=$(pwd)
term_width=$(tput cols 2>/dev/null || echo 80)
compact=false
[ "$term_width" -lt 70 ] && compact=true

# ── LINE 1: Model │ Context % │ Session │ Thinking ────
pct_color=$(color_for_pct "$pct_used")

git_branch=""
git_dirty=""
git_worktree=false
if git -C "$cwd" rev-parse --is-inside-work-tree >/dev/null 2>&1; then
    git_branch=$(git -C "$cwd" symbolic-ref --short HEAD 2>/dev/null)
    if [ -n "$(git -C "$cwd" status --porcelain 2>/dev/null)" ]; then
        git_dirty="*"
    fi
    git_dir=$(git -C "$cwd" rev-parse --git-dir 2>/dev/null)
    git_worktree_name=""
    if [[ "$git_dir" == *"/worktrees/"* ]]; then
        git_worktree=true
        git_worktree_name=$(basename "$git_dir")
    fi
fi

session_duration=""
duration_ms=$(echo "$input" | jq -r '.cost.total_duration_ms // 0')
if [ "$duration_ms" -gt 0 ] 2>/dev/null; then
    elapsed=$(( duration_ms / 1000 ))
    if [ "$elapsed" -ge 3600 ]; then
        session_duration="$(( elapsed / 3600 ))h$(( (elapsed % 3600) / 60 ))m"
    elif [ "$elapsed" -ge 60 ]; then
        session_duration="$(( elapsed / 60 ))m"
    else
        session_duration="${elapsed}s"
    fi
fi

line1="${blue}${model_name}${reset}"
line1+="${sep}"
line1+="✍️ ${pct_color}${pct_used}%${reset}"
if [ -n "$session_duration" ]; then
    line1+="${sep}"
    line1+="${dim}⏱ ${reset}${white}${session_duration}${reset}"
fi
session_name=$(echo "$input" | jq -r '.session_name // empty')
if [ -n "$session_name" ]; then
    line1+="${sep}"
    line1+="🏷️ ${cyan}${session_name}${reset}"
fi
line1+="${sep}"
if $thinking_on; then
    line1+="${magenta}💡 thinking${reset}"
else
    line1+="${dim} thinking${reset}"
fi

# ── LINE 2: dirname (branch) / worktree ────────────────
dirname=$(basename "$cwd")
line2="📁 ${cyan}${dirname}${reset}"
if [ -n "$git_branch" ]; then
    display_branch="$git_branch"
    if $compact; then
        max_branch=$(( term_width - ${#dirname} - 5 ))
        if $git_worktree; then
            max_branch=$(( max_branch - ${#git_worktree_name} - 4 ))
        fi
        [ "$max_branch" -lt 10 ] && max_branch=10
        if [ ${#display_branch} -gt "$max_branch" ]; then
            display_branch="${display_branch:0:$((max_branch - 1))}…"
        fi
    fi
    if $git_worktree; then
        line2+=" ${dim}(${reset}${green}${display_branch}${red}${git_dirty}${reset}${dim})${reset}${sep}${green}🌿 ${git_worktree_name}${reset}"
    else
        line2+=" ${dim}(${reset}${green}${display_branch}${red}${git_dirty}${reset}${dim})${reset}"
    fi
fi

# ── OAuth token resolution ──────────────────────────────
get_oauth_token() {
    local token=""

    if [ -n "$CLAUDE_CODE_OAUTH_TOKEN" ]; then
        echo "$CLAUDE_CODE_OAUTH_TOKEN"
        return 0
    fi

    if command -v security >/dev/null 2>&1; then
        local blob
        blob=$(security find-generic-password -s "Claude Code-credentials" -w 2>/dev/null)
        if [ -n "$blob" ]; then
            token=$(echo "$blob" | jq -r '.claudeAiOauth.accessToken // empty' 2>/dev/null)
            if [ -n "$token" ] && [ "$token" != "null" ]; then
                echo "$token"
                return 0
            fi
        fi
    fi

    local creds_file="${HOME}/.claude/.credentials.json"
    if [ -f "$creds_file" ]; then
        token=$(jq -r '.claudeAiOauth.accessToken // empty' "$creds_file" 2>/dev/null)
        if [ -n "$token" ] && [ "$token" != "null" ]; then
            echo "$token"
            return 0
        fi
    fi

    if command -v secret-tool >/dev/null 2>&1; then
        local blob
        blob=$(timeout 2 secret-tool lookup service "Claude Code-credentials" 2>/dev/null)
        if [ -n "$blob" ]; then
            token=$(echo "$blob" | jq -r '.claudeAiOauth.accessToken // empty' 2>/dev/null)
            if [ -n "$token" ] && [ "$token" != "null" ]; then
                echo "$token"
                return 0
            fi
        fi
    fi

    echo ""
}

# ── Fetch usage data (cached) ──────────────────────────
cache_file="/tmp/claude/statusline-usage-cache.json"
cache_max_age=60
mkdir -p /tmp/claude

needs_refresh=true
usage_data=""

if [ -f "$cache_file" ]; then
    cache_mtime=$(stat -c %Y "$cache_file" 2>/dev/null || stat -f %m "$cache_file" 2>/dev/null)
    now=$(date +%s)
    cache_age=$(( now - cache_mtime ))
    if [ "$cache_age" -lt "$cache_max_age" ]; then
        needs_refresh=false
        usage_data=$(cat "$cache_file" 2>/dev/null)
    fi
fi

if $needs_refresh; then
    token=$(get_oauth_token)
    if [ -n "$token" ] && [ "$token" != "null" ]; then
        response=$(curl -s --max-time 5 \
            -H "Accept: application/json" \
            -H "Content-Type: application/json" \
            -H "Authorization: Bearer $token" \
            -H "anthropic-beta: oauth-2025-04-20" \
            -H "User-Agent: claude-code/2.1.34" \
            "https://api.anthropic.com/api/oauth/usage" 2>/dev/null)
        if [ -n "$response" ] && echo "$response" | jq -e '.five_hour' >/dev/null 2>&1; then
            usage_data="$response"
            echo "$response" > "$cache_file"
        fi
    fi
    if [ -z "$usage_data" ] && [ -f "$cache_file" ]; then
        usage_data=$(cat "$cache_file" 2>/dev/null)
    fi
fi

# ── Rate limit lines ────────────────────────────────────
rate_lines=""

if [ -n "$usage_data" ] && echo "$usage_data" | jq -e . >/dev/null 2>&1; then
    bar_width=10

    five_hour_pct=$(echo "$usage_data" | jq -r '.five_hour.utilization // 0' | awk '{printf "%.0f", $1}')
    five_hour_reset_iso=$(echo "$usage_data" | jq -r '.five_hour.resets_at // empty')
    five_hour_reset=$(format_reset_time "$five_hour_reset_iso" "time")
    five_hour_bar=$(build_bar "$five_hour_pct" "$bar_width")
    five_hour_pct_color=$(color_for_pct "$five_hour_pct")
    five_hour_pct_fmt=$(printf "%3d" "$five_hour_pct")

    seven_day_pct=$(echo "$usage_data" | jq -r '.seven_day.utilization // 0' | awk '{printf "%.0f", $1}')
    seven_day_reset_iso=$(echo "$usage_data" | jq -r '.seven_day.resets_at // empty')
    seven_day_reset=$(format_reset_time "$seven_day_reset_iso" "datetime")
    seven_day_bar=$(build_bar "$seven_day_pct" "$bar_width")
    seven_day_pct_color=$(color_for_pct "$seven_day_pct")
    seven_day_pct_fmt=$(printf "%3d" "$seven_day_pct")

    extra_enabled=$(echo "$usage_data" | jq -r '.extra_usage.is_enabled // false')
    if [ "$extra_enabled" = "true" ]; then
        extra_pct=$(echo "$usage_data" | jq -r '.extra_usage.utilization // 0' | awk '{printf "%.0f", $1}')
        extra_used=$(echo "$usage_data" | jq -r '.extra_usage.used_credits // 0' | awk '{printf "%.2f", $1/100}')
        extra_limit=$(echo "$usage_data" | jq -r '.extra_usage.monthly_limit // 0' | awk '{printf "%.2f", $1/100}')
        extra_bar=$(build_bar "$extra_pct" "$bar_width")
        extra_pct_color=$(color_for_pct "$extra_pct")

        extra_reset=$(LC_TIME=C date -v+1m -v1d +"%m/%d" 2>/dev/null | sed 's|^0||; s|/0|/|')
        if [ -z "$extra_reset" ]; then
            extra_reset=$(LC_TIME=C date -d "$(date +%Y-%m-01) +1 month" +"%m/%d" 2>/dev/null | sed 's|^0||; s|/0|/|')
        fi
    fi

    if $compact; then
        rate_lines+="${white}5h${reset} ${five_hour_pct_color}${five_hour_pct}%${reset}"
        rate_lines+="${sep}"
        rate_lines+="${white}7d${reset} ${seven_day_pct_color}${seven_day_pct}%${reset}"
        if [ "$extra_enabled" = "true" ]; then
            rate_lines+="${sep}"
            rate_lines+="${white}ex${reset} ${extra_pct_color}\$${extra_used}${dim}/${reset}${white}\$${extra_limit}${reset}"
        fi
    else
        if [ "$extra_enabled" = "true" ]; then
            used_width=${#extra_limit}
            extra_used_padded=$(printf "%${used_width}s" "$extra_used")

            rate_lines+="${white}current${reset} ${five_hour_bar} ${five_hour_pct_color}${five_hour_pct_fmt}%${reset} ${dim}⟳${reset} ${white}${five_hour_reset}${reset}"
            rate_lines+="\n${white}weekly${reset}  ${seven_day_bar} ${seven_day_pct_color}${seven_day_pct_fmt}%${reset} ${dim}⟳${reset} ${white}${seven_day_reset}${reset}"
            rate_lines+="\n${white}extra${reset}   ${extra_pct_color}\$${extra_used_padded}${dim}/${reset}${white}\$${extra_limit}${reset} ${dim}⟳${reset} ${white}${extra_reset}${reset}"
        else
            rate_lines+="${white}current${reset} ${five_hour_bar} ${five_hour_pct_color}${five_hour_pct_fmt}%${reset} ${dim}⟳${reset} ${white}${five_hour_reset}${reset}"
            rate_lines+="\n${white}weekly${reset}  ${seven_day_bar} ${seven_day_pct_color}${seven_day_pct_fmt}%${reset} ${dim}⟳${reset} ${white}${seven_day_reset}${reset}"
        fi
    fi
fi

# ── Output ──────────────────────────────────────────────
printf "%b" "$line2"
printf "\n%b" "$line1"
if [ -n "$rate_lines" ]; then
    if $compact; then
        printf "\n%b" "$rate_lines"
    else
        printf "\n\n%b" "$rate_lines"
    fi
fi

exit 0
```

## 注意点・制限事項

### OAuth 認証が必要

Rate Limit 表示は `api.anthropic.com/api/oauth/usage` を叩いています。**Claude.ai の Max プランなどで OAuth 認証している場合のみ動作します**。

API キー（`ANTHROPIC_API_KEY`）で認証している場合は、Rate Limit 部分は表示されません（その他の情報は表示されます）。

### 内部 API の利用について

Usage API（`/api/oauth/usage`）は公式ドキュメント化されていない内部 API です。将来のバージョンで変更・廃止される可能性があります。実際、Claude Code 自身もこの API を使って rate limit 情報を取得しているため、大きく壊れることはないと思いますが、保証はありません。

## カスタマイズのヒント

### バーの幅を変える

`bar_width=10` の数値を変えるだけです。

### バーの文字を変える

```
# ブロック文字を使う場合
for ((i=0; i<filled; i++)); do filled_str+="█"; done
for ((i=0; i<empty; i++)); do empty_str+="░"; done
```

### 表示項目を減らす

例えば thinking 表示が不要なら、`line1` の組み立て部分から該当行を削除するだけです。

### 独自情報を追加する

`line1` や `line2` への追加は自由です。例えば現在時刻を表示するなら:

```
current_time=$(date +"%H:%M")
line1+="${sep}${dim}${current_time}${reset}"
```

## まとめ

Claude Code のステータスライン機能は、標準では最低限の情報しか出ませんが、シェルスクリプト一本でかなり情報量を増やせます。特にレートリミットのリアルタイム表示は、長時間作業するときに「あ、そろそろ切れそうだから作業を調整しよう」という判断をその場でできるようになり、地味に便利です。

ぜひ色んなカスタマイズ方法をおしえてください。
