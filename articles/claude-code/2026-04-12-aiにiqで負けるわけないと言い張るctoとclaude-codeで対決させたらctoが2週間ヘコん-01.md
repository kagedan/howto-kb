うちのCTO（40代）は、AI全般に懐疑的な人だった。

「ChatGPTなんて確率的なオートコンプリートだろ」
「AIに仕事を奪われるとか言ってる奴は本質がわかってない」
「人間の知能とは根本的に違う」

それ自体はいい。AIへの批判的な視点は大事だ。

ただある日、「AIって知能テストでどのくらい取れるんだろ」という話になって、CTOがこう言った。

「そんなもん、人間に勝てるわけないだろ」

それが始まりだった。

https://picoli.site/q-iq-0138a

## 対決の設定

俺が開発したIQテストは30問構成。
CTOとClaude Codeに同じテストを受けさせることにした。

Claude Codeへの問題の渡し方：問題文を一問ずつテキストで渡し、選択肢から回答させる。制限時間は考慮しない（処理速度のアドバンテージを排除するため）。

CTOは自分でブラウザから受けた。

## 結果

| | スコア |
|---|---|
| Claude Code | **127** |
| CTO | **119** |

CTOが負けた。
