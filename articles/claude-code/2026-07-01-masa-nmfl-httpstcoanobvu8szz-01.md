# そもそもClaude Fable 5とは何なのか

Claude Fable 5（クロード・フェイブル5）は、Anthropicが出したばかりの、最新のClaudeです。

覚えることは1つだけでいいです。

これまで一番賢かったモデルOpusのそのさらに上のクラスが初めて一般に開放された、ということ。

今までのClaudeは、Haiku（速い）、Sonnet（バランス）、Opus（最強）の3段でした。Fable 5はこのOpusのもう一段上で、

Anthropicはこの新しい階級を「Mythos（ミトス）級」と呼んでいます。

このMythos級は、公式が「今までで一番性能が高い」と言っているクラスです。

人が横についていなくても、何日も自律で重い仕事を進めていける。

ここが、これまでのClaudeとの一番大きな違いですね。

実はこのクラスのモデル、賢すぎて一回提供が止まりました。

そのあたりの事情は後半の料金のところでまとめて話しますが、今は解禁されていて、僕たちも普通に使えます。

今まで手が届かなかった一番上のクラスが、初めて自分の案件に使える。

それがFable 5です。

# 今までのモデルと何が違うのか

一番分かりやすいのがコーディングの性能です。

細かいベンチマークの数字も公式から出ているんですけど、実際それを見てもピンとこないですよね。

ざっくり言うと、コーディングの実力は、これまで最強だったOpusより頭ひとつ抜けていて、他社の最新モデルもはっきり上回っています。

今あるモデルの中ではトップ、と思ってもらって大丈夫です。

しかもコーディングだけじゃなくて、調べ物や画像の読み取りなんかも含めて、ほとんどの分野で今一番いい結果を出しています。

ただ、ここでもっと大事な公式の一文があります。

「タスクが長く複雑になるほど、他のモデルとの差は大きくなる」
