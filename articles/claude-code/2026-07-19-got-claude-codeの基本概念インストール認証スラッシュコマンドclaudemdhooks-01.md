Claude Codeの基本概念、インストール、認証、スラッシュコマンド、CLAUDE.md、Hooks、MCP、コスト管理までを一つにまとめた技術記事です。Claude Codeはターミナル上で動き、対象のリポジトリを読み、複数ファイルを編集し、シェルコマンドやテストを実行できます。記事では、会話履歴を消す/clear、長い会話を圧縮する/compact、利用モデルの変更、コスト確認など、日常的に使う操作を整理。CLAUDE.mdへプロジェクトの概要、開発規約、テストコマンド、変更禁止領域を記録する方法や、Hooksでファイル編集後にフォーマットやテストを実行する設計も説明されています。MCPを使えば外部サービスとの接続も可能ですが、不要なツールを増やすとコンテキスト消費や権限リスクが高まります。API利用では上限設定と秘密情報の管理が欠かせません。
