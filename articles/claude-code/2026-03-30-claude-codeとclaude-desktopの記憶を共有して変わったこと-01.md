2つの Claude が別人だった
Claude Code で設計方針を詰めた翌日、出先のスマホから Claude に関連する相談をしたくなった。しかし昨日の議論は Claude Code のセッション内にしかない。結局、前提を一から説明し直すか、帰宅まで待つかの二択だった。
