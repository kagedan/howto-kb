🛑Do NOT use GPT-5.4 1M Context Window for coding.

1. Costs more
2. Accuracy drops over 256K context

What that does to your agent?

It's not as smart as soon as it goes above 256K context. 

It's not good for coding...

... Instead you want to reserve 1M Context Window for things where you are planning and need a ton of context to make decisions.

Use it to help you create a spec, but switch back to the default context window (by removing the lines) when you go back to code.
