「とりあえず読んでおいて」と渡されるPDF、ありますか。

分厚い報告書、規約の改定版、提案書……。読まなければいけないのはわかっている。でも100ページを最初から読む時間は正直ない。

Claude Codeなら、**PDFをフォルダに置いて「要点を出して」と頼むだけで、5分で概要が返ってきます。**

## Before / After

**Before：** 100ページPDFを流し読み → メモ取り → 要点整理 → 1〜2時間

**After：** PDFをClaude Codeに渡すだけ → **要点・確認箇所が5分で出てくる（92%削減）**

## 準備するもの

- Claude Code（インストール済みであること）   
- 要点を抽出したいPDFファイル

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：PDFを作業フォルダに置く

任意のフォルダにPDFファイルを入れておくだけです。

```
📁 作業フォルダ
  └── 読みたいファイル.pdf
```

### ステップ2：Claude Codeに依頼する

```
このPDFの要点を抽出してください。

ファイル: 読みたいファイル.pdf
```

これだけです。ファイル名を変えるだけで、どんなPDFにも使えます。

![](https://assets.st-note.com/img/1776991451-bQLk0az1E5DjWhneOptv2wYA.png?width=1200)

### ステップ3：要点まとめを確認する

Claudeがこの形式で返してきます。

```
【全体概要】
〇〇に関する報告書。〜という背景から〜の結論が出ている。（3行以内）

【重要ポイント】
1. 〜（p.12）
2. 〜（p.34）
3. 〜（p.67）

【数値・データ】
- 〜：〇〇%（p.23）
- 〜：〇〇億円（p.45）

【気になる箇所・要確認】
- 〜（p.88）

【結論・次のアクション】
〜
```

ページ番号付きで返ってくるので、気になる箇所だけ原文に飛べます。

![](https://assets.st-note.com/img/1776991459-zFElQLX9KhciCUTa1SqfZkp8.png?width=1200)

### ステップ4：気になる箇所を深掘りする

概要を見て気になる箇所があれば、続けて聞けます。

```
3番の「〜」の部分を詳しく教えてください。
```

わかるまで掘り下げられます。

![](https://assets.st-note.com/img/1776991468-GBSJFOrTVbPM3Dg2l1oYmRQe.png?width=1200)

## できました。ただ、このままだと1つ問題があります。

「要点を出して」だけでは、何を重視すべきかによって結果がブレます。

契約書に必要な情報と、提案書に必要な情報はまったく違う。毎回「義務と禁止事項を優先して」「コストとリスクを中心に」と説明するのは手間です。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、自分がよく読むPDFの種類と優先観点を覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎回のPDF読みが変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「契約書なので義務・禁止事項を優先して」と説明   
✅ ファイルあり：自動で契約書向けの観点で抽出される

❌ 設定なし：毎回「重要ポイントは5個以内で」と指定   
✅ ファイルあり：毎回5個以内で返ってくる

❌ 設定なし：毎回「ページ番号も入れて」と追加   
✅ ファイルあり：常にページ番号付きで出力される

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「PDFの山」が5分で片付くようになります。**

## 100円で何が変わるか

PDF読み1回1時間 × 月5回 × 12ヶ月 = **年間60時間がPDF読みに消えています。**

設定ファイル1枚で、来月からその60時間が戻ってきます。

**100円 ÷ 60時間 = 1時間あたり約2円。** これより安く時間を買う方法は、なかなかありません。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-007（添付）**　←　ダウンロードして置くだけ   
- **用途別プロンプト5種**　←　契約書・報告書・提案書・議事録・論文で使い分け   
- **複数PDFを横断比較する方法**　←　「AとBの違いを教えて」が1分で出てくる

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「どんなPDFを読むことが多いですか？」→ 答える   
- 「抽出結果の使い道はどんなケースが多いですか？」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたのPDF読みスタイルを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
