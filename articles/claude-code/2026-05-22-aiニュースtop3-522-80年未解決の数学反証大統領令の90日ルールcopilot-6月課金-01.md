研究は AI に破られ、政府は規制で追いつき、開発者は固定費を見直す。同じ日に 3 方向の動きが揃いました。

## ① OpenAI の汎用推論モデルが、80 年未解決の数学難問を反証

Erdős が 1946 年に提起した単位距離問題で、代数的整数論を独自に応用。過去の格子最適性の定説を覆しました。数学特化ではない汎用モデルが自力で発見した最初の重大例。外部の数学者、Thomas Bloom らの厳密な検証もクリア済み。

<https://openai.com/index/model-disproves-discrete-geometry-conjecture/>

## ② Trump 大統領、AI サイバーセキュリティ大統領令に署名予定

主要 AI モデルは一般公開の 90 日前に政府提出が必須に。Anthropic の Mythos など強力な新モデルへの懸念が背景。銀行、Pentagon、病院などの重要インフラには事前アクセスを提供。任意ベースのフレームワークですが、Anthropic と OpenAI が協力しています。

<https://www.washingtonpost.com/technology/2026/05/21/white-house-tore-down-ai-rules-now-its-building-new-defenses/>

## ③ GitHub Copilot、6 月 1 日より全プランで使用量課金へ移行

月額固定からトークン消費の「GitHub AI Credits」制に変更。Pro 月 10 ドル + 10 ドル分 Credits、Pro+ 39 ドル + 39 ドル分、Business 19 ドル + 19 ドル分。コード補完と次編集提案は無制限維持。Chat / CLI / Agent / Spaces / Spark はクレジット消費です。

<https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/>

## 感想

PostSupporter と Mapmediary で Claude Code を日常的に使う身からすると、Copilot の 6 月 1 日課金変更はそのまま自分のコスト構造に直結します。「月額固定だと安心して回せる」気分が、6 月以降は崩れる感覚。Chat や Agent を雑に酷使していると、月末に思った以上の請求が来そうな構造です。

OpenAI の数学突破は「AI が独力で発見する」フェーズに入った象徴的な日。汎用モデルが代数的整数論を自力で応用して 80 年問題を反証した、というのが重い。研究分野で AI を使う人にとっては「ChatGPT に聞く」だけでなく「AI に解かせる」段階が現実的になってきた印象。

Trump 大統領令は、AI ラボのリリースサイクルが政府承認を挟む方向に踏み込む流れの始まり。エンドユーザーが直接何かするわけではないですが、Anthropic や OpenAI の新モデル提供サイクルがやや慎重になる可能性を頭に入れておきたいです
