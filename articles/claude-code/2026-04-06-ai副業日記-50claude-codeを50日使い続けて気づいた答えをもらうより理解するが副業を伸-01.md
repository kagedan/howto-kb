<p name="a6112717-53aa-4aae-8a05-5e8cf5cc7cd5" id="a6112717-53aa-4aae-8a05-5e8cf5cc7cd5">Claude Codeを使い始めて50日目。正直に言う。最初の2週間、私はこのツールをまったく使いこなせていなかった。「AIがコードを書いてくれる」と期待していたのに、なぜか作業が全然終わらない。そのギャップに気づいたのは、使い始めて15日目のことだった。</p><p name="10e151a1-0175-4027-a2b5-b454b21c6517" id="10e151a1-0175-4027-a2b5-b454b21c6517">最初の頃、私のやり方はこうだった。エラーが出る→エラーメッセージをそのままコピー→「直して」と貼り付ける。これを繰り返すだけ。1日目に4時間かかった作業が、翌日も3時間半かかった。何も改善されていない。</p><br/><a href='https://note.com/kano6754/n/nb40340079ebd'>続きをみる</a>
