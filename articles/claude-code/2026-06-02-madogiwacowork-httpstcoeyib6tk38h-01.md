## 2. 始める前に確認：プランとOS

セットアップを始める前に、2点だけ確認してください。

## プランの要件

2026年5月時点の私の環境では、Coworkを本格的に使うにはProプラン以上が前提でした。

無料プランでは利用できません。プラン要件は変わる可能性があるため、最終的には公式の表示を確認してください。

ただし、実際に業務で使い倒すには注意点があります。

Coworkはバックグラウンドで自律的にファイルを処理するため、利用枠の消費が速いのです。

大量のファイルを扱う業務では、Proプランの上限にすぐ達してしまうことがあります。

私の環境では、スライドを何十枚も作らせたり、大量のドキュメントを読み込ませたりすると、あっという間に制限に近づきます。

本格的に業務で使うなら、上位プランへの変更を念頭に置いておいた方がいいです。

## OSの要件

WindowsまたはMacのデスクトップアプリが必要です。

スマホ・ブラウザ版では動きません。

この2点を確認できたら、セットアップ開始です。

# 3. STEP 0：Desktopアプリのインストール

**① ダウンロード**

https://claude.com/download にアクセスします。

**② インストール（Windows）**

ダウンロードした .exe ファイルをダブルクリックして起動します。

「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」をクリック。

ウィザードの指示に従って「次へ」→「インストール」で完了します。

**③ ログイン**

アプリが起動したら「Log in」をクリックし、Anthropicアカウントでログインします。

**④ Coworkモードの起動**

左サイドバーの「Cowork」ボタンをクリックします。

「仮想マシンを作成しないと起動できない」というダイアログが出るのでOKをクリック。

仮想マシン作成が完了したら、PCを再起動します。

再起動後にCoworkのホーム画面が表示されれば、起動成功です。
