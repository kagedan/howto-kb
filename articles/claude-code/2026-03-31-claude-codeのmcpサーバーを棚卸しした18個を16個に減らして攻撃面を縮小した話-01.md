MCPサーバーを何個つないでいるか、把握しているだろうか。

うちは18個だった。数えてみて「多いな」と思った。で、全部の権限を洗い出してみたら、思ったよりヤバかった。

## 背景: MCP 66%にセキュリティ指摘

2026年3月、Qualysが公開したレポートによると、監査済みのMCPサーバーの66%にセキュリティ上の指摘があった。「shadow IT for AI」という表現が使われている。

さらにCheck Pointが報告したCVE-2025-59536とCVE-2026-21852。Claude Codeのプロジェクト設定ファイルを経由して、MCPサーバーの悪用でリモートコード実行やAPIトークン窃取が可能になるという内容だった。

つまり、MCPは便利な反面、**つなげばつなぐほど攻撃面が広がる**。当たり前の話なんだが、「便利だから入れよう」で増やしていくと気づいたら18個になっていた。
