まだ知らない人が多いんですが、  

Claude Code に「Skills」という機能があって 
毎回おなじ指示を打ち込む必要がなくなります。  

SKILL.md というファイルに指示を書いておくだけ。 
関連する場面で Claudeが自動的に読み込んでくれる。

 ・コードスタイルの指定 
・出力フォーマットの統一 
・プロジェクト固有のルール  全部ファイルに書いておけば、 もう口頭で説明しなくていい。  

公式の日本語ドキュメントが出てるので 気になる人はリプ欄のURLへ。
