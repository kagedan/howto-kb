## 「AIで記事が書けるって聞くけど、結局コピペとか手直しとか、手間多くない？」

正直、半年前の自分はそう思っていました。  
でも今は違います。

WordPressとClaudeを繋いだら、  
**記事の執筆・下書き保存・タグ設定・カテゴリー分け・公開・データ分析・リライト**まで、全部チャット内で完結するようになりました。

しかもセットアップにかかった時間は約5分。コードも不要。この記事では、その全体像をリアルに書きます。

### ■ プラグイン1つ入れるだけ。仕組みはシンプルです

使うのは「[Easy MCP AI](https://easymcpai.com/)」というWordPressプラグインです。

このプラグインをWordPressに入れると、ClaudeがWordPressを直接操作できるようになります。  
記事の作成・編集・タグ設定・公開まで、Claudeのチャット画面から指示するだけで全部動きます。

## 【セットアップ手順（約5分）】

1. WordPressの管理画面 →「プラグイン」→「新規追加」→「Easy MCP AI」で検索してインストール・有効化

![](https://assets.st-note.com/img/1781873045-KU2fiJw7PWozO1bpZ4Il83v6.png)

Easy MCP AI

2. プラグインの設定画面に表示される「MCP Server URL」をコピー

![](https://assets.st-note.com/img/1781873089-OXnWMirJL2jd5kaP07hqVSBT.png?width=1200)

MCP Server URLのコピー画面

3. Claude.aiの画面から「カスタムコネクタを追加」→ URLを貼り付け

![](https://assets.st-note.com/img/1782173944-NQ2domAh5U7il0GJxs8RIXkg.png)

Claude左上のカスタマイズ

![](https://assets.st-note.com/img/1782174013-Prt35Y09XmanqIGwVcfslE1b.png?width=1200)

右上のプラスボタン

![](https://assets.st-note.com/img/1782174046-u20o9AC8Lfi1Z7eXDvpE6hP3.png)

カスタムコネクタを追加

![](https://assets.st-note.com/img/1782173860-qjvZ4Rok5N2x0yVHTzufOmbK.png)

カスタムコネクタ追加画面

4. OAuth認証画面で「Grant Full Access」を選択して承認

以上です。詰まるポイントはほぼありません。

### ■ ①音声入力で発散するだけで、記事が完成する

連携後の実際の使い方はこうです。

Claudeのチャット画面にスマホの音声入力でざっくり話します。「〇〇というテーマで、こういう内容を書きたくて、読者は30代のビジネスパーソンを想定している」くらいの口語で十分です。

するとClaudeが：  
**・ H2・H3の見出し構成を提案  
・ SEOキーワードを入れながら本文を執筆  
・ アイキャッチ画像のプロンプトを記事内に差し込み  
・ WordPressに直接下書き保存**

「発散する→OK出す→下書き完成」この流れが5〜10分で終わります。

### ■ ②過去記事・ベンチマークを参照して構成を最適化する

さらに強力なのがここです。

Claudeのプロジェクト機能にベンチマークサイトのURL・過去記事・サイトの方針を事前にインプットしておくと、新しい記事を書くときに自動的にそれらを参照します。

**・「競合サイトはこういう構成なので、うちはここで差別化しましょう」  
・「過去に書いた記事と自然につながるように内部リンクを入れておきます」  
・「このサイトのトンマナに合わせて書きます」**

こういう判断を、Claudeが勝手にやってくれます。

### ■ ③タグ・カテゴリー・スラッグまで全部設定される

記事を書いたあとのWordPress設定作業、地味に面倒じゃないですか。

**・ カテゴリーはどれにする？  
・ タグは何をつける？  
・ スラッグ（URL）はどうする？  
・ 抜粋文は？**

これが全部、チャット1回で終わります。「タイトル・抜粋・タグ・カテゴリーを設定して」と言えば、Claudeがサイトの既存設定を見ながら最適なものを自動で設定します。

### ■ ④データを渡せば、分析もリライト提案もやってくれる

もう一歩踏み込んだ使い方として、GA4やサーチコンソールのデータをClaudeに貼り付けると、そのまま分析・改善提案までやってくれます。

たとえばこんな感じです。

**・「CTRが低い記事のタイトルを3パターン改善案出して」  
・「検索順位15位前後の記事、リライトすべきH2を教えて」  
・「PVが伸びてる記事に関連する次の記事テーマを提案して」**

MCP経由でGA4を直接連携する方法も技術的には存在しますが、現状はデータをコピペして渡すだけでも十分実用的です。「分析ツールを開いてうんうん唸る」より、Claudeに投げて提案をもらう方が圧倒的に速い。

## ■ まとめ｜「発散するだけ」で記事が世に出る時代になった

正直、ここまで変わるとは思っていませんでした。

以前は「記事1本書く」だけで数時間かかっていたのが、今は音声入力で話しながら発散して、「OK」を出して、「公開して」の一言で終わります。

ツールを使いこなすというより、「話しかけるだけでブログが動く」感覚に近いです。

・セットアップ：約5分・コード不要  
・使うもの：Easy MCP AIプラグイン ＋ Claude.ai  
・できること：執筆・設定・公開・分析・リライトが全部チャット完結

WordPressでブログをやっているなら、一度試してみる価値は絶対にあります。
