# ■①売る日を先に決めた（3週間後）

最初にやったのは商品作りじゃなくて、
3週間後に売ると日付を決めたことです。

**目標の10万円から逆算して単価は1万円に決定。**

参考にする発信者を決めてその人の販売までの日程を真似ました。

**日付が決まると、発信も商品作りも全部が逆算になります。**

「いつか出そう」は一生出ないです。

当時の僕が一番やりそうだったやつ。

3週間の組み方はこうです。

↓

> **1週目：商品とターゲットを決める。発信を商品の分野に絞り始める**

> **2週目：悩み解決の発信を厚くする。「こういうものを作ってる」と途中経過を見せ始める**

> **3週目：販売日を予告して、当日に売る**

```
あなたはコンテンツ販売のローンチ設計のプロです。
私が「3週間後にnoteを売る」ための日程表を作ってください。

【私の状況】
・売るもの：（決まっていれば書く。無ければ未定でOK）
・価格の想定：約1万円
・目標金額：10万円
・使える作業時間：1日◯時間

【進め方】
STEP1：目標10万円を「価格×部数」に分解して、現実的な販売目標を出す
STEP2：今日から21日後の販売日まで、1週間ごとの日程表を作る
・列は「発信ですること／商品づくりですること／告知ですること」の3つ
・各週のゴールを1行で
・その週にやらなくていいことも1行で（やること以上に大事）
STEP3：一番遅れやすい工程を1つ指摘して、遅れた場合の巻き返し方を添える

【品質条件】
・1日◯時間で本当に回る量にする。無理な計画は立てない
・精神論で埋めない。作業と時間で書く
```
