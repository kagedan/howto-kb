結論から言うと、「誰でも使える期間」は7月1日〜7月7日ではありません。
7月1日からグローバルで再開され、その後も利用できます。ただし、7月7日を境に料金体系だけが変わります。
Anthropic公式の発表によると、スケジュールは以下のとおりです。

6月12日：米国政府の輸出規制により、Claude Fable 5とMythos 5の提供を全ユーザー向けに停止。

6月30日：輸出規制が解除。
7月1日から：世界中のユーザー向けに再提供開始。https://t.co/ASK3gor6Zi
Claude Platform（API）
Claude Code
Claude Cowork
で利用可能。

7月1日〜7月7日：Pro・Max・Team・一部Enterpriseプランでは、週間利用上限の50%までFable 5を追加料金なしで利用可能。

7月8日以降：Fable 5自体は引き続き利用可能。
ただし、usage credits（従量課金・利用クレジット）方式へ移行。無料枠としての50%提供は終了します。

つまり、
利用開始：2026年7月1日～
利用終了：終了日は発表されていません（継続提供予定）
7月7日まで：Proなどの対象プランでは、週間上限の50%まで無料扱い
7月8日以降：追加利用はクレジット消費方式
なお、「誰でも使える」という意味では、無料ユーザー向けにFable 5が提供されるとは公式には書かれていません。公式発表で明記されているのは、Pro・Max・Team・一部Enterpriseプラン向けの利用条件です。無料プランへの提供状況は別扱いとなっています。
