## **何が発表されたのか**

まず、事実を整理する。

OpenAIが発表したGPT-5.6は、3つのモデルファミリーで構成されている。

- **GPT-5.6 Sol**

  - 旗艦モデル

  - コーディング、バイオ、サイバーセキュリティ、長期エージェントタスクに強い

  - `max` reasoning effort と `ultra` mode を備える

  - `ultra` modeではsubagentsを使って複雑な作業を進める

- **GPT-5.6 Terra**

  - 日常業務向けのバランス型

  - OpenAIは、GPT-5.5相当の性能をより低コストで提供する位置づけとして説明している

- **GPT-5.6 Luna**

  - 高速、低コスト型

  - 大量処理や軽めのタスク向けに見える

価格は、1M tokensあたり以下の設定になっている。

- Sol: input $5 / output $30

- Terra: input $2.50 / output $15

- Luna: input $1 / output $6

アクセスは、最初から一般公開ではない。

プレビュー中は、APIとCodexで一部のtrusted partners / organizations向けに提供される。その後、ChatGPT、Codex、APIでより広く提供予定と説明されている。

ここまでは、新モデルの製品情報として読める。

ただ、今回の本題はここからだと思う。

## 今回、普通のモデル発表と違うところ

今回の特徴は、政府との関係が公式ページに明記されていることだ。

OpenAIは、米政府との継続的な関わりの中で、ローンチ前にモデルの計画と能力を共有したと説明している。

さらに、政府の要請により、まずは少数のtrusted partners向け限定プレビューから始めるとも書いている。

一方で、OpenAIはこのような政府アクセスプロセスが長期的なデフォルトになるべきではないとも明記している。

ここで見るべきなのは、OpenAIが限定プレビューを恒久的な標準として書いているわけではないことだ。

OpenAIは、広いアクセスを否定しているわけではない。むしろ、広い提供を目指すための短期的な手段として限定プレビューを置いている。

ただし、結果として、今回のGPT-5.6は「新しいモデルが出ました」というだけの話ではなくなった。

高性能モデルを、誰に、どの順番で、どの条件で渡すのか。

そこまで含めたニュースになっている。

少し前まで、新モデル発表で見るポイントはだいたい決まっていた。

- どれくらい賢くなったか

- 料金はいくらか

- どのプロダクトで使えるか

- 既存モデルと比べて何が変わったか

今回は、そこに別の論点が入っている。

- 誰が先に触れるのか

- その参加情報は誰に共有されるのか

- 政府はどこまでリリースに関わるの
