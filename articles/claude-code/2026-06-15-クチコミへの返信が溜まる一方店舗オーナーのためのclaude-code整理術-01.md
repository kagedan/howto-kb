「クチコミへの返信、先月分がまだ終わっていない」——店舗オーナーや店長なら、一度はこの状況に陥ったことがあるはずです。1件返信するのに10〜20分。丁寧に書こうとすると時間がかかり、後回しにすると溜まる。Googleビジネスプロフィールのクチコミへの返信率・返信速度は、実は検索順位にも影響するとされています。溜めれば溜めるほど、集客に二重のダメージが出ます。

複数店舗を抱えていると、状況はさらに深刻です。5店舗あれば月100件前後のクチコミが来る。全店分を本部担当者が対応しようとすると、それだけで月に何十時間も消えます。

これ、**Claude Codeでクチコミのテキストを渡すだけで返信文が一括生成できます**。高評価・低評価・中間の感情別に返信トーンを分け、店舗ごとのキャラクターに合わせた文体で量産できます。Day2（ECレビュー返信）と同じ「型を固定して一括処理」の発想の、Google口コミ特化版です。

## このワークフローで何ができるか

**ビフォー: クチコミ返信1件に10〜20分**（感情確認・返信内容考案・文章化・確認）×30件=5〜10時間  
**アフター: 30件まとめて20〜30分**（クチコミ一覧貼り付け5分・一括生成10分・確認・微調整15分）

月次で30件分のクチコミ対応が20〜30分に収まれば、空いた時間は接客・商品開発・マーケティング施策に回せます。複数店舗の場合、本部担当者が全店のクチコミを一括処理する体制が作れます。

向いている人

* 月10件以上のクチコミが来るのに返信が追いつかない飲食・美容・整体・小売店のオーナー・店長
* 複数店舗分のクチコミ返信を一括で管理している本部担当者・FC加盟店管理者
* 低評価クチコミへの返信に毎回悩んでいる接客業の担当者

向いていない人

## 手順

### ステップ1: CLAUDE.mdに「店舗プロフィール」「感情別返信ポリシー」「文体ルール」を書く

新規プロジェクトを作り、CLAUDE.mdに店舗情報と返信方針を書きます。感情別のトーンを明確に定義しておくのが精度を上げる最大のポイントです。

```
あなたは「ヘアサロン○○」のGoogleビジネスプロフィール返信担当です。
口コミ一覧を渡されたら、下記のポリシーと文体で返信文を一括生成してください。

【店舗プロフィール】
- 店舗名: ヘアサロン○○（架空）
- 業種: 美容室（カット・カラー・パーマ専門）
- 雰囲気: ナチュラル・落ち着いた大人向け、スタッフとの距離が近いアットホームな雰囲気
- 特徴: 完全予約制・指名料なし・施術中の飲み物無料

【感情別の返信ポリシー】

🟡 高評価（星4〜5）
トーン: 温かく、感謝と次回来店への期待を込める
含めるべき内容:
- 来店への感謝（「お越しいただきありがとうございます」）
- 口コミで触れた内容への具体的な反応（褒めてもらった点を1つ受け取る）
- 次回のご来店への期待・お待ちしている旨

🔴 低評価（星1〜3）
トーン: 誠実な謝罪、言い訳なし。問題を受け止め改善を伝える
含めるべき内容:
- 不快な思いをさせたことへの謝罪（防御・言い訳をしない）
- 具体的に何が問題だったかへの言及（口コミ内容を踏まえる）
- 改善策または直接の問い合わせ先の案内
- 再来店の機会をいただければ、という一言

🟢 中間評価（星3 / テキストのみの短い口コミ）
トーン: 丁寧に受け取る。改善余地があれば軽く触れる
含めるべき内容:
- 来店への感謝
- 口コミ内容への簡潔な返応
- 次回のご来店への案内

【文体ルール】
- 敬語・丁寧語で統一（「です・ます」調）
- 文字数: 高評価100〜150字 / 低評価150〜200字 / 中間評価80〜120字
- 定型のコピペ感が出ないよう、口コミのテキストに触れた1文を必ず入れる
- 店舗名や担当者名の署名は末尾に入れない（Googleビジネスプロフィールには自動で店舗名が表示されるため）

【NG事項】
- 「ご意見ありがとうございます」から始まる定型フレーズは使わない（コピペ感が強い）
- 低評価への反論・言い訳は絶対に書かない
- 「今後ともよろしくお願いします」の機械的な締めは避ける
- 個人を特定できる情報（「○月○日にご来店の方」等）は書かない
```

「低評価クチコミへの返信でNG事項を破らない」ことが特に重要です。言い訳・反論が入ると、他の見込み客がそのやり取りを見て来店を諦めます。

### ステップ2: クチコミ一覧をまとめて渡す

対応したいクチコミを、評価＋テキストのセットで渡します。Googleビジネスプロフィールの管理画面からコピーして貼り付けるだけです。

```
以下の口コミに対して、CLAUDE.mdのポリシーと文体ルールに従って返信文を一括生成してください。
各口コミに番号を付けて対応してください。

No.1 ★★★★★
初めて伺いました。担当の方がカウンセリングをとても丁寧にしてくださり、
なりたいイメージを上手く汲み取ってもらえました。仕上がりも大満足です。
また来月も予約したいと思います。

No.2 ★★★
悪くはないんですが、待ち時間が長かったです。
予約していたのに20分以上待ちました。

No.3 ★★
カラーの色味が希望と全然違いました。写真を見せて伝えたのに
こんなに違うとは思わなかった。もう行きません。

No.4 ★★★★★
スタッフさんみんな感じがよくて、居心地のいいサロンでした。
施術も丁寧で、頭皮マッサージが特に気持ちよかったです。リピします！

No.5 ★★★
普通でした。

（以下、残り25件分）
```

### ステップ3: 返信文を確認・微調整して投稿する

生成された返信文を1件ずつ確認します。修正が必要な場合は、その場でClaude Codeに再生成依頼します。

```
No.3（低評価・カラーの色味）の返信について、
以下の店舗の実情を踏まえて書き直してください。

実情: この口コミのお客様には、カウンセリング時に「写真より明るめにするとブリーチが必要」と
お伝えしており、ブリーチなしで施術した結果として色が薄くなった経緯があります。
ただし、その説明が十分に伝わらなかった可能性があります。
「言い訳にならない範囲で、説明不足だったことを認める」トーンで書き直してください。
```

「言い訳にならない範囲で事実を伝える」という微妙なさじ加減は、Claude Codeへの追加指示で精度を上げられます。1件だけ細かく指定しても、所要時間は数分です。

確認が終わったら、Googleビジネスプロフィールの管理画面に順番に貼り付けるだけです。

### ステップ4: 週1回のバッチ処理に組み込む

クチコミ返信は「溜まってから対応」より「週1バッチで処理」するほうが品質が安定します。

```
週次バッチ処理のルーティン:
- 毎週月曜（または金曜）の作業開始前に Googleビジネスプロフィール管理画面を開く
- 未返信の口コミを一括コピーしてClaude Codeに貼り付け
- 一括生成→確認→投稿を20〜30分で完了させる
- 低評価口コミが含まれる週は、投稿前に店長または責任者が内容を確認する
```

「月まとめて処理」より「週1バッチ」のほうが返信速度が上がり、低評価クチコミへの対応も早くなります。返信速度は来店検討中の潜在顧客への印象に直結します。

### ステップ5: 複数店舗の場合はプロジェクトを店舗別に分ける

FC本部・複数店舗の管理者は、店舗ごとにプロジェクトを作りCLAUDE.mdを書き分けます。

```
プロジェクト構成例:
- salon-shinjuku/    （新宿店・都会的・トレンド重視の文体）
- salon-jiyugaoka/   （自由が丘店・ナチュラル・大人向けの文体）
- salon-kichijoji/   （吉祥寺店・カジュアル・若年層向けの文体）
```

同じチェーンでも店舗の雰囲気・客層が違えばCLAUDE.mdを書き分けます。「どの店舗の返信も同じトーン」になるのを防ぎ、各店舗のブランドを維持できます。

## 応用と注意点

**応用1: 返信データから「改善すべきサービス課題」を抽出する**  
月20〜30件の返信文を生成し終えたあと、低評価クチコミのテキストだけをまとめてClaude Codeに渡すと、「繰り返し指摘されているサービス課題」を一覧化できます。「待ち時間が長い」が3件連続で来ていればオペレーション改善の優先度が上がる、という意思決定の材料になります。

**応用2: 食べログ・じゃらん・TripAdvisorにも横展開する**  
同じCLAUDE.mdとクチコミ返信フォーマットを使って、食べログ・じゃらん・TripAdvisor等他媒体のクチコミ返信にも使えます。媒体ごとに「文字数上限」「NG表現」が違う場合は、CLAUDE.mdに媒体別ルールを追記するだけで対応できます。

**注意点1: 低評価返信は必ず人間が最終確認する**  
Claudeが生成する低評価クチコミへの返信は、稀に「言い訳に読めてしまう表現」が混入します。特に炎上リスクのあるクチコミ（食中毒・接客トラブル・怪我等）の返信は、必ず店長・経営者が全文を読んでから投稿してください。投稿後の取り消しはできますが、スクリーンショットは残ります。

**注意点2: 個人情報・プライバシーへの配慮**  
クチコミには「○月○日に行った」「△△という施術をした」など、特定個人を推定できる情報が含まれることがあります。返信文に個人を特定できる情報を含めると、プライバシー侵害になる可能性があります。Claude Codeへの入力時もこの点を注意し、スクショを撮る際は実際の顧客のクチコミではなく架空のサンプルを使用してください。

## まとめ

* Google口コミ返信は クチ**コミ一覧を渡すだけで30件が20〜30分で完成**する
* 鍵は、CLAUDE.mdに「感情別（高評価・低評価・中間）の返信ポリシー」を書き分けること
* 低評価クチコミでは「言い訳・反論なし」「謝罪＋改善の意思」が鉄則で、NG事項をCLAUDE.mdに明文化しておくと崩れない
* 週1バッチ処理に組み込むと返信速度が上がり、来店検討中の潜在顧客への印象がよくなる

クチコミ返信は「対応が見えるSNS」です。返信の品質・速度が、次のお客様の来店判断に直接影響します。Claude Codeで量産の重さを取り除いたあとは、返信の「内容の質」に集中できます。
