Claude Code artifacts can now call MCP connectors, letting you build dashboards and apps that can fetch information and take actions for each viewer on demand.

Available on Pro, Max, Team, and Enterprise plans. Not available on publicly-shared artifacts. https://t.co/FcvtiOqv0J

Build an artifact once, then it pulls live data via MCP. Artifacts use the viewers’ own MCP connectors, so creators don’t need to re-run sessions to refresh, and viewers can only see the data they have access to.
