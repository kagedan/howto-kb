🔥 How I accidentally learnt to use AI agents  first principles.

Backstory first.

I've had access to AI - Code Suggestions for a while now, IIRC right from GPT2/3 days (~mid 2020) when @tabnine launched one of the first extension in VSCode.

It was brilliant, it complemented ...

1/

my existing workflow that included Intellisense.

Tabnine was better, it would predict what i was trying to do, and suggest next "completion" based on a percentage confidence. But it all started to change when GitHub released Copilot Code Completion.

2/ https://t.co/8Ox2EZ7GsT

All GitHub copilot required was put a "#" comment and it would try to autocomplete the code. This is the workflow i used between 2021 & 2023 till I switched to cursor.

Cursor was different, it was like ChatGPT but inside the code editor. Saved a ton of effort copy pasting into ChatGPT.

3/

Cursor was brilliant because it literally knew what  i was thinking and it would complete my code while I typed. And to make things faster I could chat with it.

The models weren't great, the suggestions were many a times wrong or unnecessarily complex, but it did the heavy lifting.

4/

I used this workflow for a long long time. GitHub copilot, like most Microsoft solution were behind the market even when it came how to use their own product.

Then came claude code, and opencode, and a 100 other CLIs. They were so many and with so many features that it was overwhelming.

5/

I didn't know what a SKILL was, or was a MCP was, how to setup AGENTS..md and it felt too much to learn.

I was happy to build using Cursor & the autocomplete it offered. Like a peasant.

That was till, 3 fortuitous events that occured simultaenouslly
1. I bought $20 version of Claude to work with word documents
2. I created a document that listed a bunch of tests I wanted to perform and gave it to Cursor
3. I needed to create a boring QR code generator that did not want my birth certificate

6/

Sidenote is, how did Microsoft managed to screw this up
