今日はClaude CodeがProプランから使えなくなったという話についてお話しようと思います。

2026年4月21日、エンジニア界隈でちょっとした騒ぎがありました。

Anthropicが**告知なし**で$20のProプランからClaude Codeを削除し、SNSで炎上。その後に撤回という流れです。

何が起きたのかを時系列で整理します。

---

## 何が起きたか（時系列）

| 日付 | 出来事 |
| --- | --- |
| 4月21日 | 料金ページ・公式ドキュメントからProプランのClaude Code記載が消える |
| 4月21〜22日 | SNS・Hacker Newsで発覚・炎上 |
| 4月22日 | Anthropicが「2%のA/Bテスト」と説明 |
| 4月22日中 | 料金ページの表記が元に戻る |

---

## 変更の内容

**変更前：** ProプランにClaude Codeが含まれる  
**変更後（一時的）：** Claude CodeはMaxプラン（$100/月〜）のみに

Anthropicの公式説明は「新規登録者の2%を対象にしたA/Bテストだった」というものです。

* 既存のProプランユーザー：影響なし
* 新規登録者の一部：Claude Codeなしのプランに誘導されていた

炎上後、料金ページは元の表記に戻りましたが、A/Bテスト自体が完全に終了したかどうかは不明確なままです。

---

## なぜ炎上したか

内容への賛否よりも、**「やり方」への批判**が圧倒的でした。

* プレスリリースなし
* 公開の変更ログなし
* ユーザーへのメール通知なし

ひっそりと料金体系を変えようとしたことへの不信感です。

「値上げするなら値上げするって言ってくれ」という反応は、ユーザーとして自然だと思います。

---

## Anthropic側の事情

Anthropicの立場から考えると、経済的な背景は理解できます。

* Proプラン：月$20
* Claude Codeのヘビーユーザーが消費するトークン：数百ドル相当になることも

使えば使うほど赤字になる構造を放置し続けるのは難しいです。

ただ、それをユーザーに伝えず、こっそり変更しようとしたことが問題でした。

---

現時点での状況をまとめます。

| 項目 | 状況 |

| 料金ページの表記 | Claude CodeがProに復活 |
| A/Bテストの継続 | 不明（明確な終了宣言なし） |
| 将来的なProからの削除 | 可能性はゼロではない |

## 個人的な見方

私自身、Claude Codeを日常的に使っています。

今回の件でAnthropicへの信頼が揺らいだという声も多くありましたが、個人的には「収益性の問題はいずれ表面化する」と思っていたので、驚き自体は少なかったです。

問題は**やり方**です。エンジニア向けのツールは、使い続けるかどうかの判断が速い。「こっそり変更」はユーザーの信頼を削ります。

今後もしProからClaude Codeが本格的に外れるなら、そのときは代替ツールも含めて改めて考えます。今のところ、一番使いやすいAI開発支援ツールという評価は変わっていません。
