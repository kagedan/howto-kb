Claude Code使ってるのに、
まだ公式ドキュメント読ませずに実装させてる人、
めちゃくちゃもったいないです。

AIが微妙なコードを出すのは、
プロンプトやskillの精度が低いんじゃなくて、
正しい情報を与えてないからです。

AIは平気で、
❌ 動くけど古い書き方
❌ 非推奨API
❌ 旧バージョンのCLIコマンド
をそれっぽく出しますよね。

しかも厄介なのは、
エラーにはなりにくいから気づけない。

その度にレビューしたり修正指示を出していたら
キリがない。

プロンプトをこねくり回す前に
やるべきことはただ1つ。

最新の公式ドキュメントを参照させること。
これだけで出力の質が圧倒的に上がります。

それをより効率よく行えるのが/find-docsスキル。

使い方を記事で詳しく解説しました。
