📝連日のClaudeアップデート、もはや情報の暴力。一旦落ち着いて深呼吸させてほしい。追いつくのがやっとだ…ｗ

で、今日はAnthropicがAIデザインツール「Claude Design」をリリース。
そしてCanvaとのコラボも普通に感動....
ㅤ
Claudeと会話するだけで
デザイン・プロトタイプ・スライドが作れるようになったから、シンプルなMarkdownビューアを作ってみた

これ、Google StitchとFigmaのかけ合わせみたいになってる

しかもそのままCanvaにエクスポートして編集できる。

「AIで下書き → Canvaで仕上げ」が1クリックで繋がったし、もちろんコード出力してClaude Codeにも引き継いでアプリもリリース可能

Claudeがいよいよ何でも生み出すドラえもん状態になってます

もうAnthropicの中の人は半年以上前からコードをほとんど手で書いてないから、AIの開発スピードがどんどん早くなって人間側が消化しきれませんｗｗ

けど、毎日が刺激的すぎて楽しい
ㅤ
ということで、Claude Designをスレッドで解説していきます↓🧵

まずClaude Designって何かというと、
ㅤ
Claudeに「こんなスライド作って」「プロトタイプ欲しい」と言うだけで
ビジュアルコンテンツが出来上がるツール。
https://t.co/jivIt5Z7e6
ㅤ
最強ビジョンモデル「Claude Opus 4.7」で動いてる。
ㅤ
◆ ブランド自動適用
→ コードベースやデザインファイルを読んで
→ チームのカラー・フォント・コンポーネントを全部自動で使う
→ 毎回ブランドガイドを貼らなくていい
ㅤ
◆ インポートが自由
→ テキストプロンプト、PDF、PPTX、XLSX、コードベース
→ 自社サイトのWebキャプチャも取り込める
ㅤ
◆ 細かい調整もできる
→ インラインコメントで「ここを変えて」
→ スペーシング・カラー・レイアウトのノブをリアルタイムで動かせる
ㅤ
◆ Claude Codeへのハンドオフ
→ デザインが完成したらワンクリックでコード実装に渡せる
ㅤ
Datadog社のPMの言葉が刺さった。
「ラフなアイデアから、会議が終わる前に動くプロトタイプが完成する」

次にCanvaとのコラボが何がエグいかというと、
ㅤ
Claude Designで作ったものを
そのままCanvaに送って、ドラッグ&ドロップで編集できる。
https://t.co/Z3GmZHxjJC
ㅤ
しかも今回、もう1個すごい機能が入ってて。
ㅤ
◆ HTMLインポート（Canva初）
→ AIが生成したHTMLをCanvaのエディタで直接編集できる
→ コードを再生成せずに色・要素・レイアウトを変更
→ Canva Presentationに埋め込みも
→ Canva Websitesとしてそのまま公開も可能
ㅤ
これ何がすごいかというと
「ビジュアル・ドキュメント・インタラクティブコンテンツ」を
1つのエディタで全部扱える世界初のプラットフォームになったということ。
ㅤ
CanvaのCEO、Melanie Perkinsのコメントが象徴的で
「アイデアが生まれる場所にCanvaを届ける」
ㅤ
月2.5億人が使うCanvaと
最強AIのClaudeが本格的に繋がった。

個人的に一番面白いのは
「デザイナーじゃなくてもデザインできる」じゃなくて
「デザイナーがもっと探索できる」という方向性だと思っていて、
ㅤ
Anthropicの発表文に
「経験豊富なデザイナーでも探索の時間を削らざるを得ない」
という一文があった。
ㅤ
つまりこれ、AIが人間の仕事を奪うんじゃなくて
「やりたかったけど時間がなくてできなかったこと」を
取り戻すためのツールとして設計されてる。
ㅤ
Brilliant社の事例が象徴的で
「他ツールで20プロンプト必要だったページが、Claude Designでは2プロンプトで完了」
ㅤ
「AIで下書き → Canvaで仕上げ → Claude Codeで実装」
