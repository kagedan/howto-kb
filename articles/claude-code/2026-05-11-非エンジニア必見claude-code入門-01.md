### 2. コード生成の基本的な考え方

Claudeにコードを生成させる際、特別な機能ボタンがあるわけではありません。普段のチャットと同じように、テキストで「こんなコードを作ってほしい」と伝えるだけでOKなんです！

Claudeは、あなたが与えた指示（これを「プロンプト」と呼びます）を理解し、その内容に沿ったコードを生成してくれます。まるで、優秀なジュニアプログラマーに口頭で指示を出すような感覚に近いですね。
