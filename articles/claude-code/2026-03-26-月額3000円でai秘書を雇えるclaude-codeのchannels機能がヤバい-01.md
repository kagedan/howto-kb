今日もコツコツやってますか？

昨日はClaudeデスクトップアプリの「Cowork」や「Computer Use」について紹介しましたが、思いのほか反響が大きかったです！

やっぱりClaudeに対する関心は高いですね〜。

というわけで、今日はもう一歩踏み込んで、最近ひっそり追加された「超便利だけど上級者向け」の機能を紹介します。

====

**それが「Claude Code 」の「Channels（チャンネルズ）」という機能です。**

これは何かというと、一言でいえば

**「Discordの中に自分専用のAI秘書を作れる」**

機能です。

ぼくのDiscordサーバーには「結ちゃん」というAI秘書が住んでいて、中身はClaude Codeで動いています。

この機能の何がヤバいか。

**それは、外出先からスマホのDiscordで結ちゃんに指示を出すと、「家のパソコンが勝手に動いて仕事をしてくれる」んです！**

スマホでAI秘書に指示を出して、パソコンを遠隔操作。

なんと今、そんなことができるんですよ。

====

たとえば、運転中にちょっとした事業プランを思いついたとき。

休憩がてらパーキングエリアで車を止めて、飲み物を買いつつ、スマホからDiscordを開いて、結ちゃんに指示を出します。

「事業アイデア思いついたから、このテーマでリサーチしておいて」

「リサーチ結果を踏まえて、事業プランを10個考えて」

**すると、自宅にある24時間稼働のMacbookが動き出し、勝手にAIが資料を作り始めるんです。**

いやー、SF映画みたいな世界ですよね……。

ぼくはさらに、毎朝5時に海外や中国のAI・クリプト関連ニュースを巡回してもらっています。

**AI秘書の結ちゃんは、それらを日本語に要約して、Discordの専用チャンネルに投稿してくれます。超優秀。**

![](https://assets.st-note.com/img/1774483928-ymqaLBhEUYSejAxn7Gg8iXN6.png?width=1200)

さらにいうと、ぼくの仕事内容も知っているので、

**「これはイケハヤさんのAIアニメ制作に関係しそうな情報なので、チェックしておくといいですよ」**

みたいな一言や、提案を添えて届けてくれたりもします。すげぇ。

![](https://assets.st-note.com/img/1774483944-5pzZgQY2PyRr3EvVWFqfcsTG.png?width=1200)

あとは、Googleカレンダーを読み込ませて、今日の予定を毎朝通知するようにもしています。

ミーティングすっぽかしがちなので、これは助かってます。。。

![](https://assets.st-note.com/img/1774483912-b7Dd5ghQvjVxCasZGUw6A1rm.png?width=1200)

この日はニワトリ管理の予定が入ってました

**中身は要するにClaude Codeなので、AI秘書の結ちゃんはClaude Codeができることは、基本的にぜんぶできます。**

マーケティング関連だと、Remotionを使った動画編集とかも可能ですし、ブログ記事を作って投稿するとかも超余裕です。

LPも作れるし、なんならスケジュール組んでアクセス解析して、自律的にLPを改善して、その結果をDiscordにレポーティングするところまで作り込めるでしょう。

**「AI秘書を育成する」感覚も楽しくて、キャラクター設定はもちろん、たとえば好感度システムとかも作り込むことができますw**

====

**で、ここまで読むと、わかる人は「それってOpenClawと同じじゃね？」と思うでしょう。**

**まさにそれなんですが、OpenClawとの違いでいうと、こっちはClaudeの月額枠で使えるんですよ！**

OpenClawはここらへん微妙で、基本的にはAPI経由、従量課金で使うことになります。

OpenClawの開発者のピーターさんがOpenAIに移籍した関係で、Codexは月額枠で使ってもいい……ようですが、ちょっと怪しいよなぁと。

「Claude Code Channels」なら、月額20ドル払えば気兼ねなく、その課金枠の中でAI秘書を動かせます。

OpenClawの方が自律性、カスタマイズ性は高いですが、予定管理・ニュース収集・リサーチ・資料作成程度であれば、Claude Code Channelsで十分ではないかな〜と。

====

ただし！！！

**この機能、「設定がとにかく難しい」のが難点です。**

そもそも前提条件が多いです。

**・自分用のDiscordサーバーを作る（これは無料・簡単）**

**・24時間稼働する古いパソコンなどを用意する**

**・そこにClaude Codeをインストールする（まあまあ難しい）**

**・DiscordのBotを開発者ポータルから設定して、Claude Codeと連携させる（かなり難しい）**

特に、非エンジニアの人からすると、後半のBot連携あたりでかなりつまづくはずです。

そもそもClaude Codeのインストールも、普通の人にはむずいんですよね、、、

**いや、もしかしたら、この連携作業すら新機能の「Computer Use」でAIに丸投げできるのか……？**

試してないですが、できるならだいぶ楽ですね！

導入方法はむなかた総理の記事がわかりやすいので、こちらをどうぞ。

これを乗り越えると

**「自分好みのAI秘書を育てられる」**

**「どこでもスマホから家のパソコンを遠隔操作できる」**

という最高の体験が待っています！
