Claude Codeから動画編集ソフトを一切開かずに、目の部分だけをアニメ風に差し替えた事例が話題です。使われていたのは

https://t.co/oA83AF5S5C

Pika社が公開した4K-VFXという新しいMCPスキルです。

Claude CodeにPikaのMCPを接続し、「/4k-vfx」というコマンドで指示を出すと、顔の表情・ジェスチャー・音声・カメラワークはそのままに、指定した部分だけを別のスタイルに変換してくれます。

今回の例では目の部分を大きいアニメ風の目に変換し、瞬きや表情の動きにも自然に追従させていました。裏側ではSeedance 4Kという高解像度モデルが使われているとされています。試すにはまずPika MCPをClaude Codeに接続する必要があります（Pika公式サイトに手順あり）👇
