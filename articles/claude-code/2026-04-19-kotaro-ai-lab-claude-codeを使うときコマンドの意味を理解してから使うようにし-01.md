Claude Codeを使うとき、コマンドの意味を理解してから使うようにしている。

/init でプロジェクト構造を認識させる。
/compact でコンテキストを圧縮して精度を維持する。
/review でコードレビューを依頼する。
/cost でトークン消費を確認する。
/memory でセッションをまたいだ記憶を管理する。

この5つを把握しているだけで、AIとの会話の質が変わる。何となく使っている人と、仕組みを理解して使っている人では、同じツールでもアウトプットに差が出る。

ツールの「使い方」ではなく「仕組み」を覚えると、応用が効くようになる。
