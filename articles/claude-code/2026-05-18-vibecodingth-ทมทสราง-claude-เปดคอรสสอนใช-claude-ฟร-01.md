**ทีมที่สร้าง Claude เปิดคอร์สสอนใช้ Claude ฟรี 13 คอร์ส มี certificate ทุกคอร์ส ไม่ต้องใส่บัตรเครดิต**

ของฟรีระดับนี้มักมาพร้อมคำถามเดียวคือ คอร์สไหนคุ้มเวลาจริง และคอร์สไหนเสียเวลาเปล่า

ช่อง Parker Prompts เรียนครบทั้ง 13 คอร์สภายใน 9 วัน แล้วสรุปว่าบางคอร์สคือการเทรน AI ฟรีที่ดีที่สุดเท่าที่เคยเรียนมา ส่วนบางคอร์สข้ามได้โดยไม่พลาดอะไร

โพสต์นี้สรุปสาระสำคัญจาก Parker Prompts ทั้งเนื้อหาแต่ละคอร์ส ลำดับการเรียน และคอร์สที่ควรข้าม

⎯⎯⎯

แพลตฟอร์มนี้ชื่อ AI Academy ของ Anthropic

สิ่งที่ Parker Prompts ชี้ว่าหาได้ยากจากแหล่งอื่นคือหลักสูตรที่ต่อเนื่องตั้งแต่ระดับพื้นฐานไปจนถึงการ deploy ระดับ production สำหรับโมเดล AI ตัวเดียว และเนื่องจากผู้สอนคือทีมที่ทำ Claude ขึ้นมาเอง เนื้อหาจึงตรงกับการใช้งานจริง

คอร์สแรกที่ผู้จัดทำคลิปบอกว่าจะแนะนำให้คนที่ไม่เคยใช้ Claude คือ **Claude 101** ครอบคลุม projects, artifacts, skills, connectors, enterprise search และ research mode ซึ่งเป็นฟีเจอร์ที่ทำให้ Claude กลายเป็นเครื่องมือสร้าง workflow ไม่ใช่แค่ chatbot ใช้เวลาราว 1 ถึง 2 ชั่วโมง

เหตุผลที่ต้องเรียนก่อนคือคอร์สอื่นเกือบทั้งหมดตั้งสมมติฐานไว้แล้วว่าผู้เรียนรู้จัก projects และเข้าใจ artifacts การข้ามคอร์สนี้จึงทำให้สับสนในคอร์สถัด ๆ ไป

⎯⎯⎯

ตามที่ Parker Prompts นำเสนอ ครึ่งแรกของห้องสมุดคือสาย developer

สาย Claude Code มีสองคอร์สที่ควรเรียนต่อกัน **Claude Code 101** เป็นบทแนะนำสั้นว่าเครื่องมือ command line ตัวนี้อ่าน code base แก้ไฟล์ และรันคำสั่งแทนผู้ใช้อย่างไร ส่วน **Claude Code in Action** เนื้อหาราว 1 ชั่วโมง 15 lecture ซึ่ง Parker Prompts ระบุว่าหลายคนยกให้เป็นคอร์สที่มีคุณค่าสูงที่สุดในทั้ง Academy และเป็นคอร์สเชิงปฏิบัติที่เร็วที่สุด ใช้เวลาราว 1 ชั่วโมงก็ได้ certificate พร้อมความเข้าใจการรัน agentic coding workflow จริง

ขั้นต่อมาคือการสร้างแอปพลิเคชันบน Claude โดยคอร์ส **Building with the Claude API** เป็นคอร์สที่ลึกที่สุดในทั้ง Academy เนื้อหากว่า 8 ชั่วโมง ครอบคลุมการสร้างและประเมิน system prompt การสร้าง custom tool และ external tool และ agent architecture เบื้องหลังแอปพลิเคชันจริง

ผู้จัดทำคลิปเทียบว่าคอร์ส AI engineering ที่เนื้อหาใกล้เคียงกันคิดค่าเรียน
