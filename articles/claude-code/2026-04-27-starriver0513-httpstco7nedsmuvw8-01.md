## 2026 年 4 月大型リデザインで変わったこと

ここが今回の記事を書ききっかけ。

公式ブログ（claude.com/blog/claude-code-desktop-redesign）でアナウンスされた通り、2026 年4月中旬に Claude Code デスクトップ版が大幅リデザインされました。

変化が大きいやつだけ、全部書きます。

**① サイドバーで並列セッション管理**

複数のセッションを並列で走らせて、サイドバーで一元管理できるようになりました。ステータス・プロジェクト・環境でフィルタ可能。リポジトリをまたいで作業しながら、結果が出たセッションから確認する、という使い方ができる。

**② ファイルエディタ内蔵**

アプリ内でファイルを直接開いて編集できます。外部エディタを起動する必要なし。

・1 クリックで Finder / VS Code / Cursor で開ける

・1 クリックで全コピー

・そのままアプリ内でマイクロ修正

触ってみると、ブラウザ版とはまったく別物のレスポンス。Cursor のようなサクサク UI で、編集体験が一段上がっています。

**③ 統合ターミナル**

アプリ内でターミナルが開く。非エンジニアは使わなくて OK。ただし「AI が何のコマンドを実行してるか」がリアルタイムで見えるので、安心感が段違い。

**④ 差分ビューア**

AI が何を変えたかが一目で分かる。リデザインで速度も改善されて、大きな変更でも瞬時に把握できる。

**⑤ プレビューペイン**

HTML・PDF・画像・マークダウン、全部アプリ内の右側タブで即表示。ファイルをダブルクリックして別アプリで開く手間がゼロになる。

**⑥ Routines（クラウド自動化）**

これが一番大きな変化です。

・スケジュール（毎朝 7 時等）

・API 呼び出し（外部システムトリガー）

・GitHub イベント（PR 作成等）

この 3 通りで Claude Code を自動起動できる機能。以前のスケジュールタスクは PC を起動しておく必要がありましたが、Routines はクラウドで走る。PC 閉じてても、寝てても、旅行中でも動き続
