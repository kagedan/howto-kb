フォルダが散らかっていますか。

「最終版」「最終版2」「本当の最終版」が混在している。ファイルを探すたびに5〜10分かかる。「あのファイルどこ？」が毎週発生する——。

Claude Codeで、これを終わりにできます。**「整理して」と一言言うだけで、命名・移動・分類が自動で片付きます。**

## Before / After

**Before：** ファイルが「とりあえずデスクトップ」に積み上がる → 探すたびに5〜10分ロス → 「あのファイルどこ？」が毎週発生

**After：** Claude Codeに「整理して」と頼むだけ → 命名・移動・分類が完了 → **3分で片付く（70%削減）**

## 準備するもの

- Claude Code（インストール済みであること）   
- 整理したいフォルダ（デスクトップでもどこでもOK）

Claude Codeのインストールがまだの方は、[記事001「Claude Codeのインストール方法」](https://note.com/shigoto_dxlab/n/nd9be1b317c43)を先にご覧ください。

## 手順（4ステップ）

### ステップ1：今のフォルダの状態を見せる

Claude Codeを整理したいフォルダで起動して、こう伝えます。

```
このフォルダの中身を確認して、整理してください。
```

Claude Codeがフォルダ内のファイル一覧を読み取ります。

![](https://assets.st-note.com/img/1776992162-6tD7pS5dX8gf0TYmI4u2bCNn.png?width=1200)

### ステップ2：整理方針を確認する

Claudeが現状を分析して提案を出します。

```
現在の問題点：
・命名ルールがバラバラ（日付あり/なし混在）
・「最終版」「最終版2」「本当の最終版」が混在
・年度をまたいだファイルが同じ階層に混在

整理案：
2026/
  01_議事録/
  02_提案書/
  03_報告書/
_archive/（1年以上前のファイル）
```

### ステップ3：「この方針で整理して」と確定する

```
この方針で整理してください。
ファイルの移動と、命名規則の統一もお願いします。
```

実行前に移動するファイル数と移動先を確認してくれます。

![](https://assets.st-note.com/img/1776992170-NyKXd9tlqTxVimIFvSCYpowu.png?width=1200)

### ステップ4：完了確認

```
整理後のフォルダ構成を一覧で出してください。
```

## できました。ただ、このままだと1つ問題があります。

1回きれいにしても、また散らかります。

新しいファイルを置くたびに「このフォルダでいいか」と迷う。毎週同じ場所に「最終版」が増えていく。Claude Codeにルールを覚えさせておかなければ、次回も最初から説明し直しです。

**これでは半分しか楽になっていません。**

## その解決策を、100円でそのままプレゼントします

解決策はあります。   
**設定ファイルに、フォルダ構成・命名規則・アーカイブルールを覚えさせておくだけです。**

でも「自分でファイルを編集する」のが面倒な方のために、この記事では完成版ファイルをそのままプレゼントします。ダウンロードして置くだけ。設定不要です。

## ファイルを1つ置くだけで、毎回の整理が変わります

設定ファイルを一度置けば、毎回の説明がゼロになります。

置くだけで、こうなります：

❌ 設定なし：毎回「年度フォルダで分けて」と説明が必要   
✅ ファイルあり：自動で年度・カテゴリに整理される

❌ 設定なし：毎回「最終版という名前は使わないで」と追加   
✅ ファイルあり：常に日付+バージョン形式で命名される

❌ 設定なし：古いファイルがたまり続ける   
✅ ファイルあり：1年以上前のファイルは自動でアーカイブ対象になる

一度設定すれば、ずっとそのまま使えます。   
**100円の投資で、この先ずっと続く「ファイル探しの時間ロス」が消えます。**

## 100円で何が変わるか

ファイル探し5分 × 1日3回 × 240日 = **年間60時間がファイル探しに消えています。**

設定ファイル1枚で、来月からその時間が大幅に減ります。

**100円 ÷ 40時間削減 = 1時間あたり2.5円。** フォルダが多い人ほど効果が大きい設定です。

有料パートでプレゼントするのは：

- **shigoto\_dxlab-gift-012（添付）**　←　ダウンロードして置くだけ   
- **「最終版」地獄を終わらせる命名規則テンプレ**　←　日付＋バージョン管理の設定例   
- **月次整理を1コマンドにする方法**　←　「先月のファイルを整理して」1行で完結

しかも、ダウンロードしてフォルダに置いたら、あとは何もしなくていいです。

claude を起動すると、Claudeが自動で話しかけてきます。

```
📦 コピペで1記事1改善ラボ のプレゼントファイルを認識しました！
   セットアップを始めます。いくつか質問させてください。
```

あとはClaudeの質問に答えるだけです。

- 「どんなフォルダ構成にしたいですか？」→ 答える   
- 「命名規則はありますか？」→ 答える

**これだけで完了です。設定も書き換えも、一切不要。**   
次回から起動すれば、もうあなたのフォルダルールを覚えた状態で動いています。

考えなくていいです。ダウンロードして、フォルダに置くだけです。
