ChatGPTやClaude Desktopを使い始めたとき、「これで十分じゃないか」と思っていた。

テキストを貼り付けて、「これ直して」「要約して」「メール書いて」。  
返ってきた答えをコピーして、自分のエディタに貼る。  
それがAIの使い方だと思っていた。

でも、それって・・・毎回引越し業者に荷物を運んでもらって、自分の家の中を自分で整理してるようなものだった。

---

Claude DesktopとClaude Code CLIの違い、実際に使って気づいたこと

Claude DesktopはAIとチャットするツール。  
Claude Code CLIはAIと一緒に作業するツール。

この一文で伝わるかどうか、少し不安だけど。。。使ってみたら体感としてわかった。

Claude Desktopでやっていたこと：  
ブラウザでClaude.aiを開く → コードをコピー → チャットに貼り付ける → 「ここが違います」と返ってくる →  
コピーして自分のエディタに戻す。

この「コピー→貼り付け→コピー→貼り付け」の繰り返しが、地味にしんどかった。

Claude Code CLIでやっていること：  
VS Codeを開く → そのまま「このファイルのここを直して」と言う → ファイルが直接書き換わる。

AIが自分のファイルを「見ながら」動いてくれる。  
ファイル全体の文脈を理解した上で提案してくれる。

コードだけじゃなくて、「このプロジェクトの構造はこうなってます」という前提ごと共有できる感覚。

---

正直な話

最初は怖かった。

「AIが勝手にファイルを書き換えるって、大丈夫なの？」

でも実際は、変更前に「こうしますよ」と見せてくれるし、OKしてから実行される。  
運転席はちゃんと自分にある。

---

使い分けはこんな感じ

Claude Desktop  
           向いてること：文章の相談・アイデア出し・調べもの  
　　　使う場所　　：ブラウザ  
　　　感覚　　　　：相談相手  
Claude Code CLI  
　　　向いてること：コードの修正・ファイル操作・作業の自動化  
　　　使う場所　　：VS Code（エディタの中）  
　　　感覚　　　　：隣で作業してる同僚

---

AIを使い始めて気づいたこと

「AIに何かをしてもらう」から「AIと一緒に何かをする」に変わると、できることの幅が変わる。

ツールが変わるだけで、こんなに違うのか——と少し驚いている。

まだ使いこなせているとは言えないけど、それでも毎日少しずつ、前より速く動けている気がする。
