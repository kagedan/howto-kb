## はじめに

Anthropic Academy は、Anthropic が公式に提供している学習プラットフォームだ。

https://anthropic.skilljar.com/

Claude の基本的な使い方から Claude Code、MCP（Model Context Protocol）、Subagents、Claude API 実装、AI Fluency まで、現時点で 17 のコースが提供されている。
また、研修などでよくあるように、最後に問題が出題され、それに合格する必要がある。

コースは個別に履修でき、修了するとオンラインで Certificate が発行される。
また、発行された認定証は LinkedIn にも連携が可能だ。
AI Fluency 系のように、必ずしもエンジニアを前提にしていないコースも揃っているため、「Claude を使う全員」が対象になっているのも特徴だ。

この記事は、

- Anthropic Academy / CCA が気になっている人
- Claude Code や API, MCP, RAG を体系的に学びたい人
- 社内 AI 研修を検討している人
- AI 導入時の教育コストを気にしている人

あたりを主な対象としている。
結論から言うと、Anthropic Academy は単なる「AI ツールの使い方講座」ではなかった。特に印象的だったのは、個人学習だけでなく、**組織で AI を使う際の“共通言語”を揃える教材としてかなり完成度が高い** ことだった。

## CCA（Claude Certified Architect）との違い

少し補足として、Anthropic Academy とは別に、**CCA（Claude Certified Architect）** という認定資格がある。
結論として、認定パートナー企業に所属していて、個人のスキルの証明として扱うなら CCA を受けた方がいいと思う。

自分も CCA が欲しくて、Anthropic に直接問い合わせした。
規約上、認定パートナー周りの詳細は書けないが、フリーランスの受験は現時点ではかなりハードルが高そうだった。
しかし、Academy と CCA は関係しており、CCA に関心がある人にとっても、まず Anthropic Academy 側を一度通しておく価値はかなり大きいと思う。

## 受けるべきか？

結論から言うと、Anthropic Academy はかなりおすすめできる。

特に、

- Claude Code を触り始めた
- AI サービスを構築したい
- MCP を業務導入したい
- 社内 AI 研修を考えている
- AI 活用レベルをチームで揃えたい

あたりに当てはまるなら、受ける価値は大きい。

特に企業研修との相性はかなり良い。
理由はシンプルで、このレベルの教材を社内でゼロから整備するのはかなり重いからだ。

しかも AI 領域は変化が速い。
一度教材を作って終わりではなく、

- モデル更新
- UI 変更
- 運用知識
- ベストプラクティス

まで継続的に追従する必要がある。

Anthropic Academy は、その更新を公式自身がやっている。
これは企業側から見るとかなり大きい。
逆に、「Claude Code で明日コードを書けるようになりたい」だけなら、無理に全冠する必要はない。必要なコースだけ拾う形でも十分価値はある。

## 受けるための前提知識

特別な前提知識はほぼ不要。

AI Fluency 系は技術前提をほとんど要求しないし、Claude Code 系もインストールから順に説明される。
いつもエンジニアが手探りで作成する API Key の作成方法まで説明してくれている。

そのため、「AI を使う側」としての最低限の土台を揃える用途にはかなり向いている。

## 学べること

本記事では詳しく扱わないが、Claude における使い方から、AI サービスの構築、RAG、MCP、AI との接し方まで体系的にすべて学べる。
動画はすべて英語だが、スライド主体のため英語が得意でなくても追いやすかった。

特に、「AI を個人利用する」だけでなく、「組織へ導入する側」にとってはかなり参考になる教材だった。
是非受けて欲しい。

## まとめ

AI を体系的に学べる。
Anthropic Academy は、単なる「公式チュートリアル集」ではなかった。
ただし、Academy を全冠したからといって、「Claude を最大限使いこなせる状態」になるわけではない。

実際の現場では、

- どこまで AI に任せるか
- AI 人材をどう育成するか
- どこから人間レビューへ戻すか
- AI 生成コードをどう監査するか

のような運用判断が大量に発生する。
そこは結局、日々の業務で試しながら積み上げるしかない。

逆に言えば、Anthropic Academy は「AI 活用のスタートラインを揃える教材」としてかなり優秀だった。
