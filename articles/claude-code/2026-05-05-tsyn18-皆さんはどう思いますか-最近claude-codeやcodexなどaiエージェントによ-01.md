皆さんはどう思いますか？

最近、Claude CodeやCodexなど、AIエージェントによる「連携」や「自動化・定期実行」について超盛り上がってるけど、私は全然流行らない気がする。

いろいろ実践してきたけど、とにかく複雑すぎる。

「クラウドとローカルを同期して～」
「MCPやCLIでSaaSとつないで～」
「権限とリポジトリをこうして～」
…みたいな設定がとにかく難解で複雑。一応動くけどさ、内部で何が起きているか全然理解できない。これをセキュリティ含め、組織的に運用できる気がしないんだけど。というか、X上で紹介されている事例って、ほとんどの法人でNGじゃないの？

いずれ、「代表的な連携や自動化」は、GoogleやMicrosoftが出してくると思う。なので、王道の連携や自動化はあんまり手を出さずに冷静に見ておくのが正解だと思う。

例えば、「Claude CodeをGmailとつないで…」みたいな活用は微妙。

逆に、「GPTとElevenlabsをつないで動画生成…」みたいなニッチな使い方は有効だと思う。

要は、最近のAIは「複雑すぎる」んですよね。「もっとシンプル」でいいのでは？と思うわけです。
