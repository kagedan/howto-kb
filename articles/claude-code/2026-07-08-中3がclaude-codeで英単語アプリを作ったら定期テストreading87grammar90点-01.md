# 中3がClaude Codeで英単語アプリを作ったら定期テストReading87・Grammar90点取れた話

## 最初に英語定期テストの点数を発表します

元々は70点くらいだったのが、  
定期テストの英語、こうなりました。

* **Reading: 87点**
* **Grammar: 90点**

英語は得意でも苦手でもない、普通の中学生でした。でも今回は違った。  
理由は1つ。**自分で英単語アプリを作ったから。**  
この記事では、誰でも理解しやすいよう、専門用語などはほぼ無しで説明します。

## Claude Codeで作ることにした

プログラミングはやってたけど、  
**実用レベルで、かつ実際に使うアプリ** は作ったことがなかったが、

**Claude Codeがあった。**

Claude Codeというのは、ご存知の通りAnthropicが提供する、  
AIエージェントに指示することでコードを書いてもらえるというサービス。  
Claude Codeの上位モデルは業界でもトップクラスの推論能力を誇り、  
使用者の指示からその人の意図を正確に読み取り、初心者でも「こういうアプリを作りたい」と説明すると実装してくれる。

最初にやったのは **CLAUDE.md** という仕様書を書くこと。  
EiTan - Apple Watch 英単語学習アプリ  
フラッシュカード形式で英単語を覚える  
英検4級〜1級の6レベル対応  
SwiftUI で実装

これを渡したら、あとはほぼ会話しながら作れた。

## 

## 一番革新的だったこと：3,976語のデータ

アプリの中身は **英検4級〜1級の3,976語**。

さすがに手打ちは無理なので、これもClaude Codeに頼んだ。  
`gen_words.py` というスクリプトを作ってもらい、単語データをJSON形式で自動生成。

| レベル | 語数 |
| --- | --- |
| 4級 | 516語 |
| 3級 | 563語 |
| 準2級 | 648語 |
| 2級 | 676語 |
| 準1級 | 714語 |
| 1級 | 859語 |

コーヒーを飲みながらスペースキーを押すだけで(CCへの許可のアクション)作業は終わり、  
この工程が一番「AIすごい」と思った瞬間だった。

## 完成したアプリ「EiTan」

* Apple Watchの画面でフラッシュカードが表示される
* タップして「知ってる / 知らない」を選ぶだけ
* カラーテーマは4種類（Tokyo Night、ダーク等）
* 背景にはアップロードした写真を適応可能
* 学習した単語は自動で記録される
* **Web版もあるのでiPhoneなどなしでも使える**

よければこちらから試用してみてください。  
👉 **GitHub: [github.com/ESflask/EnglishWord](https://github.com/ESflask/EnglishWord)**

## 勉強法：使い方のコツ

テスト2週間前からこう使った。

1. **通学中（計50分）** にApple WatchやiPhoneで単語帳機能で単語を流し見  
   特にWatch版は、腕を上げるだけで単語が出てくるので、  
   スマホを出す手間がないのがいい。
2. **夜の15分** で間違えた単語だけ復習
3. テスト範囲のレベル（自分の場合は3級~準2級）に絞って集中

ポイントは「完璧に覚えようとしない」こと。  
1単語ごとに5秒を10周、翌日に簡単な復習で回しました。  
流し見を繰り返すうちに自然と頭に入ってくるはず。

## 結果と気づき

### きつかったところ

ただし、Claude Codeは特に17ドルのProプランだと、  
Sonnetを使ってもしばらくすると手一杯になる点が残念。  
親がMax 20x プランに入り始めたのでトークンの問題は解決したが、  
それまではGeminiやCodexも別タブで併用してなんとか耐えた。

### 結果的によかった点

仕方なく使ったGeminiやCodexだったが、性能は十二分によく、費用を抑えて長くAIエージェントを使いたいならこれらのサービスのほうが良いかもしれない。

Reading 87、Grammar 90。自分史上、英語で一番いい点数だった。

勉強時間は前回より少なかったのに。

**自分で作ったアプリだから続けられた**、という実感がある。

市販アプリはどこか他人のもの感がある。  
自分のアプリは、自分の苦手に合わせてある。だから使う。
