# インフロニアHD、データ主導型経営を支える財務会計システムを3カ月で構築*：**導入事例*

インフロニア・ホールディングスは、アクセンチュアとSAPジャパンと協業し、データ主導型経営の基盤となる財務会計システムを3カ月で構築した。

インフロニア・ホールディングスは2026年6月22日、アクセンチュアとSAPジャパンと協業し、データ主導型経営の基盤となる財務会計システムを3カ月で構築したと発表した。経営主導の業務改革先行型アプローチにより、国内最短水準での導入を実現した。

　インフロニアHDは、従来の建設業における請負中心のビジネスモデルからの脱却を目指し、インフラの開発／投資から設計・施工、維持管理／運営までを担う「総合インフラサービス企業」への変革を進めている。実現に向けて、インフラ老朽化や担い手不足、デジタル化の遅れといった業界共通の課題への対応に加え、M&Aによるグループ拡大を背景に、経営データの一元化と可視化による経営管理の高度化に取り組んできた。

　この取り組みの一環として、現場の原価管理から事業別の管理会計、グループ全体の財務会計までを一気通貫でつなぎ、可視化するデータ主導型の次世代基幹システムを構築するプロジェクトを展開。2025年4月にはアクセンチュアとの合弁会社「インフロニア ストラテジー＆イノベーション（ISI）」を設立し、経営判断に必要なデータを集約、可視化する基盤整備を推進してきた。

　今回、集約／可視化したデータに基づき、グループ経営の中核となる財務会計領域でSAPの「SAP Cloud ERP」を3カ月で導入、稼働させた。短期間での導入は、インフロニアHDが事業会社とともに現場や事業特性を踏まえた経営主導の業務改革を長年推進してきたことに加え、アクセンチュアとSAPが2026年4月に発表したSAP Cloud ERPの短期導入アプローチを採用したこと、両社による専任チームを編成してプロジェクトを推進したことなどが寄与したとしている。

　今後は財務会計に加え、調達や原価管理など対象業務を順次拡大する。AIを活用したデータ分析の高度化や予測活用を進め、経営判断の質とスピードの向上を図る。将来は、今回の取り組みをインフラ／建設業界全体へ展開可能なモデルへ発展させ、業界全体の生産性向上につなげる考えだ。
