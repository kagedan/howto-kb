hot take: making AI agents chat with each other to get work done is a dead end.

the only model that's actually stable is treating it like a CI pipeline. you define a task graph, declare the workflow in a config file, and set quality gates.

it's just engineering we're building incredibly fragile systems because we're chasing cool demos instead of reliable architecture
